인원 카운팅 이벤트 설정하기

인원 카운팅 이벤트를 설정하면 지정된 구역에 머무르는 사람의 수를 감지할 수 있습니다. 선택적으로 얼굴 영역을 추가하면 등록된 사용자를 자동으로 신원을 인식합니다.

언제 사용하나요?​

  • 소매점 또는 매장의 특정 구역 방문객 수를 실시간 모니터링하고 싶을 때

  • 혼잡도 관리를 위해 특정 구역의 인원 현황을 파악하고 싶을 때

  • 등록된 직원/VIP 고객의 출입 현황을 추적하고 싶을 때

  • 보안 목적으로 특정 구역의 출입 통계를 관리하고 싶을 때

인원 카운팅 이벤트 생성​

인원 카운팅 이벤트를 생성하면 라이브 메뉴에서 지정된 구역에 머무르는 사람의 수를 감지할 수 있습니다. 또한 얼굴 영역을 설정하면 해당 영역에서 얼굴을 우선적으로 추적할 수 있으며, 등록된 사용자는 신원을 자동 인식합니다.

이벤트 생성 시작​

  1. 화면 왼쪽 상단의 설정 메뉴를 클릭하세요.

  2. 화면 왼쪽 사이드바에서 AI & 규칙 → AI 이벤트를 클릭하세요.

  3. 화면 오른쪽 상단의 + 추가 버튼을 클릭하세요.

  4. AI 이벤트 타입 선택 창에서 인원 카운팅을 클릭하세요.

  5. Vionyx 얼굴 인증 기능 활성화 안내 및 동의 팝업이 표시되면 내용을 확인하고, 동의 체크박스를 클릭한 후 동의하고 얼굴 인증 기능 활성화 버튼을 클릭하세요.

    • 얼굴 인식 기능을 사용하려면 해당 지역의 관련 규제 법규를 준수해야 합니다. 동의하지 않으면 진행할 수 없습니다.

    • Vionyx 얼굴 인증 기능 활성화 안내 및 동의 팝업은 최초 한 번만 표시됩니다. 이후 AI 이벤트 생성 시에는 동의한 것으로 간주합니다.

  6. 오른쪽 인원 카운팅 패널에서 필요한 설정을 완료하세요.

이벤트 이름 설정​

이름 입력 필드에서 AI 이벤트 이름을 입력하세요. 이름은 최대 64자까지 입력할 수 있습니다.

카운팅 영역 설정​

라이브 뷰에서 카운팅 영역을 설정하세요. 카메라에서 설정한 카운팅 영역에 머무르는 사람을 실시간으로 카운팅하고 이벤트를 발생시킵니다. 최소 1개를 설정해야 하며, + 추가 버튼을 클릭해 최대 10개까지 추가할 수 있습니다. 라이브 뷰에 표시되는 카운팅 영역 상자의 모서리 또는 점을 마우스로 드래그하여 원하는 형태로 조정할 수 있습니다. 가운데 영역을 드래그하여 위치를 이동할 수도 있습니다.

  • 이름: 카운팅 영역의 이름을 수정할 수 있습니다. 알기 쉽게 이름을 지정하세요. 카운팅 영역의 이름은 라이브 뷰에서 마우스 오버 시 표시됩니다.

  • 색상: 카운팅 영역의 색상을 선택할 수 있습니다. 라이브 뷰에서 카운팅 영역이 선택한 색상으로 표시됩니다.

  • 조건: 이벤트 발생 조건을 설정하세요.

    • 모든 변화: 사람이 카운팅 영역에 들어오거나 나갈 때마다 이벤트 발생

    • 카운트 제한: 카운팅 영역에 머무르는 사람 수가 지정한 수를 초과할 때 이벤트 발생 (기본값 10명, 최대값 100명)

  • 지속시간: 카운팅 영역에 머무르는 사람의 수가 이벤트 발생 조건을 충족해야 하는 최소 시간을 설정하세요. 예를 들어, Duration을 5초로 설정하면 사람이 카운팅 영역에 5초 이상 머물러야 이벤트가 발생합니다. 최대 60초까지 설정할 수 있습니다.

알아두기

  • 설정한 카운팅 영역의 모양과 크기를 초기화하려면 버튼을 클릭하세요.

  • 추가한 카운팅 영역을 삭제하려면 버튼을 클릭하세요.

얼굴 영역 설정(선택 사항)​

얼굴을 감지할 영역을 설정하세요. 설정하면 해당 영역에서 얼굴을 우선 추적합니다. 얼굴 영역과 카운팅 영역은 겹쳐서 설정할 수 있습니다. + 추가 버튼을 클릭해 최대 10개까지 설정할 수 있습니다. 라이브 뷰에 파란색 상자가 표시되며, 마우스로 모서리 또는 점을 드래그하여 원하는 형태로 조정할 수 있습니다. 가운데 영역을 드래그하여 위치를 이동할 수도 있습니다.

  • 이름: 얼굴 영역의 이름을 수정할 수 있습니다. 알기 쉽게 이름을 지정하세요. 얼굴 영역의 이름은 라이브 뷰에서 마우스 오버 시 표시됩니다.
알아두기

  • 등록된 사용자는 라이브 메뉴에서 신원을 자동 인식해 이름이 표시됩니다. 등록되지 않은 사용자는 미인식 얼굴으로 표시됩니다.

  • 설정한 얼굴 영역의 모양과 크기를 초기화하려면 버튼을 클릭하세요.

  • 추가한 얼굴 영역을 삭제하려면 버튼을 클릭하세요.

카운팅 초기화 설정​

카운팅 영역에 머무르는 사람 수를 초기화하는 조건을 설정하세요.

  • 전체 영역 카운트 초기화: 모든 카운팅 영역의 카운팅을 정해진 시간마다 초기화하려면 켜짐으로 설정하세요. 초기화 시간에서 설정한 시각에 모든 카운팅 영역의 카운팅이 초기화됩니다.

설정 저장​

모든 설정을 완료한 후 오른쪽 상단의 저장 버튼을 클릭하세요. 이전 화면으로 돌아가며, 새로 생성된 AI 이벤트가 목록에 추가됩니다.

라이브에서 확인​

화면 왼쪽 상단의 라이브 메뉴를 클릭하세요. 카메라 영상의 오른쪽 상단의 아이콘이 켜져 있는지 확인하세요. 설정한 카운팅 영역과 얼굴 영역이 영상에 오버레이로 표시됩니다. 카운팅 영역에 머무르는 사람의 수를 카운트 : N 형식으로 표시합니다. 얼굴 영역을 설정했다면 카메라가 해당 영역에서 얼굴을 감지하며, 등록된 사용자는 자동으로 신원이 인식되어 이름을 표시합니다.

알아두기

  • 카운트 값을 수동으로 초기화하려면 → 카운트 초기화 버튼을 클릭하세요.

  • 등록되지 않은 사용자는 미인식 얼굴로 표시됩니다. 등록된 사용자는 이름과 함께 표시합니다.

  • 조명이 부족하거나 역광이 있으면 카메라가 얼굴을 인식하지 못할 수 있습니다. 촬영 공간의 조명을 개선하거나 카메라의 위치와 각도를 조정하여 얼굴이 잘 보이도록 하세요.

  • 얼굴이 카메라로 향해야 인식됩니다. 옆모습 또는 뒷모습은 인식이 어려울 수 있습니다.

  • 너무 많은 이벤트가 발생하면 반복 지연 옵션값을 조정하세요. 이벤트 발생 지연 시간을 늘리면 이벤트 빈도를 줄일 수 있습니다.
