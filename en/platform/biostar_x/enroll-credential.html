    사용자 크리덴셜 등록하기

    Not translated
    This page is currently being translated. We will complete the translation as soon as possible.

    사용자의 크리덴셜을 등록하는 방법을 안내합니다. 사용자의 지문 및 얼굴, 카드 정보, 비밀번호 등을 크리덴셜로 등록할 수 있습니다. 크리덴셜은 사용자를 새로 추가하거나 수정할 때 등록할 수 있습니다.

    → 페이지로 이동해, 아래 두 가지 방식으로 크리덴셜을 등록하거나 추가, 변경할 수 있습니다.

    • 새로운 사용자: 화면 오른쪽 상단의 New User 버튼을 클릭하세요. New User 창의 Credential 섹션에서 크리덴셜을 등록할 수 있습니다.

    • 기존 사용자: 사용자 목록에서 사용자를 선택하세요. 화면 오른쪽에 표시되는 프로필 화면에서 See More 버튼을 클릭하세요. 사용자 정보를 수정할 수 있는 창이 나타나면 Credential 섹션에서 크리덴셜을 등록할 수 있습니다.

    Info

    크리덴셜을 등록하고 Security Level 항목에서 적절한 보안 등급을 설정하세요. 보안 등급이 높을수록 인증률이 낮거나 본인 거부율(FRR)이 높을 수 있습니다.

    Credential: Data used to identify users. Digital signatures, smart cards, biometric data, user names, passwords, etc. are common examples of credentials.

    Security Level​

    장치에서 설정한 바이오메트릭 1:N 보안 등급과 상관없이 사용자에게 별도의 보안 등급을 부여할 수 있습니다. 장치의 1:N 보안 등급을 높게 설정해 자주 인증에 실패하는 사용자가 있다면, Security Level을 낮게 설정하여 인증 실패를 완화할 수 있습니다.

    Info

    • Private Auth Mode를 Biometric으로 설정했다면, 입력된 사용자를 알 수 없기 때문에 1:1 보안 등급을 적용할 수 없으며, 사용자 특정이 가능한 카드 또는 ID로 인증 모드를 설정했을 때만 1:1 보안 등급을 적용할 수 있습니다.

    • 적절한 보안 등급을 설정하세요. 보안 등급이 높을수록 지문 인증률이 낮거나 본인 거부율(FRR)이 높을 수 있습니다.

    지문 등록​

    사용자의 지문을 등록해 출입을 위한 인증 수단으로 사용할 수 있습니다. 지문 인증을 지원하는 장치를 통해 사용자의 지문 정보를 등록하세요. 지문은 지문 스캐너가 설치된 장치에서 스캔할 수 있습니다.

    지문을 등록하기 전에…

    • 사용자의 지문이 깨끗하고 물기가 없는지 확인하세요.

    • 상처가 있거나 지문이 희미한 손가락은 등록하지 마세요.

    Credential 섹션의 Fingerprint 항목에서 버튼을 클릭하세요. Enroll Fingerprint 창이 나타나면 각 항목을 설정하고 Enroll 버튼을 클릭하세요.

    • Enroll Fingerprint: 지문을 등록할 장치를 선택하세요.

    • Fingerprint Quality: 지문 등록 품질을 조절할 수 있습니다. 품질 수준에 미치지 않으면 지문 정보를 등록할 수 없습니다.

    • View Fingerprint Image: 토글 버튼을 클릭하세요. 지문을 스캔했을 때 원본 이미지를 확인할 수 있습니다.

    • 지문은 최대 10개까지 등록할 수 있습니다. 각 번호에 한 개의 지문을 등록할 수 있습니다. 원하는 번호를 선택하고 Scan Fingerprint 버튼을 클릭하세요.

      Info

      번호 버튼을 선택해야 Scan Fingerprint 버튼이 활성화됩니다. 지문을 장치에 스캔하면, 지문 정보가 등록됩니다.

    • Duress: 협박 지문으로 등록하려면 이 옵션을 선택하고 지문을 스캔하세요. 누군가에게 협박을 당하거나 강제로 출입문을 열어야 할 때, 해당 지문을 사용하면 알림을 보낼 수 있습니다.

    • Clear: 등록한 지문을 삭제할 수 있습니다. 지문을 삭제하려면 해당 지문 번호를 선택하고 버튼을 클릭하세요.

    Info

    • 일상에서 출입을 위해 사용하는 지문을 협박 지문으로 사용하지 마세요.

    • View Fingerprint Image 옵션을 활성화할 때, 지문 이미지를 확인할 수 있지만 BioStar에 저장하지 않습니다.

    • 지문 인증률이 낮다면 지문 정보를 삭제하고 새로운 지문 정보를 등록하세요.

    • 좋은 품질의 지문 정보를 얻으려면 지문 인식 센서 표면을 모두 덮을 수 있도록 손가락을 대세요. 검지나 중지의 지문을 등록하세요.

    얼굴 등록​

    얼굴 등록은 사용자의 얼굴을 비주얼 카메라로 촬영한 인증 수단입니다. 적외선 카메라로 촬영한 얼굴 정보와 구분되며, 비주얼 페이스를 지원하는 장치에서만 사용할 수 있습니다. 비주얼 카메라로 촬영한 사용자의 얼굴 정보는 모바일 기기에서 비대면으로 등록할 수도 있습니다.

    얼굴 등록 시 주의 사항

    • 얼굴을 등록할 때 장치와 얼굴의 거리를 60~100cm로 유지하세요.

    • 표정이 변하지 않도록 주의하세요.

    • 마스크, 모자, 안대를 착용하지 마세요.

    • 마스크를 착용한 얼굴을 등록하지 마세요. 마스크를 착용한 얼굴과 착용하지 않은 얼굴을 모두 등록할 경우 오인식률이 높아질 수 있습니다.

    • 고개를 들거나 숙이지 마세요.

    • 진한 화장을 하지 마세요.

    • 눈을 감지 마세요.

    • 화면에 양쪽 어깨가 모두 나타나도록 하세요.

    • 움직이지 말고, 정면으로 서서 얼굴을 등록하세요.

    • 두 명의 얼굴이 화면에 동시에 나타나지 않도록 주의하세요. 한 번에 한 사람씩 등록하세요.

    • 화면의 지시에 따르지 않을 경우 얼굴 등록이 오래 걸리거나 실패할 수 있습니다.

    Credential 섹션의 Face 항목에서 버튼을 클릭하세요. Enroll Face 창이 나타나면 각 항목을 설정하고 Enroll 버튼을 클릭하세요.

    • Enrollment Device: 비주얼 페이스를 등록할 장치를 선택하세요.

    • Add: 비주얼 페이스를 추가하려면 버튼을 클릭하세요. 최대 2개의 비주얼 페이스를 추가할 수 있습니다.

    • Scan: 비주얼 페이스를 스캔하려면 버튼을 클릭하세요. 버튼을 클릭하고 장치 화면의 지시에 따라 비주얼 페이스를 스캔하세요.

    • Upload Images: 비주얼 페이스로 사용할 이미지를 업로드할 수 있습니다.

      Info

      • 지원하는 이미지 파일의 크기는 최대 10Mb입니다.

      • 지원하는 이미지 파일은 JPG, JPEG, PNG 형식입니다.

    • Use as Profile Image: 스캔한 비주얼 페이스를 프로필 이미지로 사용하려면 옵션을 선택하세요.

    • Delete: 등록한 비주얼 페이스를 삭제할 수 있습니다. 삭제하려는 순번 버튼을 선택하고 버튼을 클릭하세요.

    Info

    • 비주얼 페이스를 사용할 수 있는 장치는 아래와 같습니다.

      FaceStation F2, BioStation 3, BioEntry W3

    • 비주얼 페이스를 다른 방식으로 불러오거나 모바일 기기 또는 웹캠으로 등록할 수 있습니다. 자세한 내용은 다음 문서를 참고하세요.

    출입 카드 등록​

    사용자에게 출입 카드를 할당할 수 있습니다. 장치가 지원하는 카드 종류는 장치의 매뉴얼을 참고하세요.

    USB 에이전트를 이용한 카드 등록

    카드 종류CSNWiegand스마트 카드
    EM❌❌❌
    MIFARE✅❌✅
    DESFire✅❌✅
    FeliCa✅❌❌

    CSN 카드​

    1. Credential 섹션의 Card 항목에서 버튼을 클릭하세요.

    2. Enroll Card 창이 나타나면 Card Type 목록에서 CSN을 선택하세요.

    3. Registration Option 항목에서 원하는 카드 등록 방식을 선택하세요.

      • Register by Card Reader: Device 목록에서 카드를 스캔할 장치를 선택하고 Read Card 버튼을 클릭하세요.

      • Assign Card: 목록에서 사용자에게 할당하려는 카드를 선택하세요. 입력 필드를 통해 검색할 수도 있습니다.

      • Enter Manually: 카드 번호를 직접 입력하거나 Use User ID 버튼을 클릭하세요.

    4. Enroll 버튼을 클릭하세요.

    Wiegand 카드​

    1. Credential 섹션의 Card 항목에서 버튼을 클릭하세요.

    2. Enroll Card 창이 나타나면 Card Type 목록에서 Wiegand를 선택하세요.

    3. Card Data Format 항목에서 카드 데이터 형식을 선택하세요.

    4. Registration Option 항목에서 원하는 카드 등록 방식을 선택하세요.

      • Register by Card Reader: Device 목록에서 카드를 스캔할 장치를 선택하고 Read Card 버튼을 클릭하세요.

      • Assign Card: 목록에서 사용자에게 할당하려는 카드를 선택하세요. 입력 필드를 통해 검색할 수도 있습니다.

      • Enter Manually: 식별 코드(Facility Code)와 카드 ID를 직접 입력하세요.

    5. Enroll 버튼을 클릭하세요.

    Info

    Card Data Format 항목에 원하는 카드 데이터 형식이 없다면 다음 문서를 참고하세요.

    스마트 카드​

    액세스 온 카드 또는 보안 크리덴셜 카드, 커스텀 스마트 카드를 등록할 수 있습니다.

    1. Credential 섹션의 Card 항목에서 버튼을 클릭하세요.

    2. Enroll Card 창이 나타나면 Card Type 목록에서 Smart Card를 선택하세요.

    3. Device 항목에서 카드를 등록할 장치를 선택하세요.

    4. 장치를 선택하면 Card Layout Format 항목에 설정된 카드 레이아웃 형식이 표시됩니다.

    5. Card Type에서 원하는 카드 종류를 선택하세요.

      • Secure Credential Card: 사용자의 정보(카드 ID, PIN, 출입 그룹, 기간, 지문 템플릿, 얼굴 템플릿, 개인 인증 모드)를 카드에 저장할 수 있습니다.

      • Access on Card: 사용자의 정보(카드 ID, PIN, 지문 템플릿, 얼굴 템플릿)를 카드에 저장할 수 있습니다. BioStar X에 저장된 사용자 정보를 사용하려면 서버 매칭 옵션이 활성화되어 있어야 합니다.

      • Custom Smart Card: 타사에서 발급한 스마트 카드를 등록할 수 있습니다. Read Card 버튼을 클릭해 카드 ID를 등록하세요.

    6. Write Smart Card 버튼을 클릭하세요.

    Info

    • 스마트 카드에 저장하는 사용자 정보는 BioStar X에 저장된 정보를 사용합니다. 만약 변경한 사용자 정보를 저장하지 않으면, 스마트 카드에 잘못된 사용자 정보를 저장할 수 있습니다. 또한, 변경한 사용자 정보를 장치와 동기화하지 않으면 장치가 인증을 수행하지 못할 수 있습니다.

    • 장치에 스마트 카드 레이아웃을 설정하려면 다음 문서를 참고하세요.

    • Secure Credential Card는 카드 ID를 직접 설정할 수 있습니다.

    • Custom Smart Card를 사용할 수 있는 장치와 펌웨어 버전은 아래 목록을 참고하세요.

      • XPass D2 펌웨어 1.7.1 이상

      • BioEntry P2 펌웨어 1.5.1 이상

      • BioEntry W2 펌웨어 1.8.0 이상

      • BioStation 2a 펌웨어 1.1.0 이상

      • X-Station 2 펌웨어 1.3.0 이상

      • BioStation 3 펌웨어 1.3.0 이상

      • BioEntry W3 펌웨어 1.0.0 이상

      • BioLite N2 펌웨어 1.6.2 이상

    • 새로운 스마트 카드 레이아웃을 설정하려면 다음 문서를 참고하세요.

    • 스마트 카드를 포맷하여 정보를 다시 기록하려면 다음 문서를 참고하세요.

    카드 읽기​

    스마트 카드를 포맷하고 정보를 다시 기록할 수 있습니다.

    1. Credential 섹션의 Card 항목에서 버튼을 클릭하세요.

    2. Enroll Card 창이 나타나면 Card Type 목록에서 Read Card를 선택하세요.

    3. Device 항목에서 카드를 등록할 장치를 선택하세요.

    4. Smart Card Type을 선택하세요.

    5. Read Card 버튼을 선택하세요.

    6. 카드 정보를 확인하고 Format Card 버튼을 클릭하세요.

    Info

    • Device 항목의 장치 목록은 스마트 카드 레이아웃이 설정되어야 표시되며, 설정에 대한 자세한 내용은 다음 문서를 참고하세요.

    • 커스텀 스마트 카드는 포맷할 수 없습니다.

    모바일 인증 등록​

    슈프리마의 에어팝 포탈(Airfob Portal)과 연동하여 모바일 출입 카드를 사용하면, 사용자에게 모바일 출입 카드를 발급할 수 있습니다. 모바일 출입 카드는 각 사용자가 개별 등록하거나 CSV 불러오기 기능을 이용해 다수의 사용자를 한 번에 등록할 수도 있습니다.

    에어팝 포탈에서 설정한 모바일 출입 카드 발송 방법에 따라 사용자의 이메일 또는 전화번호를 입력해야 합니다.

    Info

    • 모바일 출입 카드는 CSN 모바일 카드와 템플릿 온 모바일 중 하나만 사용할 수 있습니다.

    • 에어팝 포탈 및 모바일 출입 카드 사용에 대한 자세한 내용은 다음 문서를 참고하세요.

    템플릿 온 모바일​

    템플릿 온 모바일은 사용자의 바이오 메트릭 템플릿을 모바일 기기에 저장하는 모바일 액세스 카드입니다. BioStar X 서버, 에어팝 포탈(Airfob Portal)과 장치에 사용자 데이터를 저장하지 않고도 바이오 메트릭으로 인증할 수 있습니다.

    이 기능은 바이오 메트릭을 크리덴셜로 사용하고 싶으나 개인정보 보호로 인해 서버 및 장치에 생체 정보를 저장할 수 없는 환경에서 유용하게 사용할 수 있습니다. 템플릿 온 모바일의 발급과 바이오 메트릭 템플릿 등록 과정이 따로 진행되기 때문에, 사용자는 관리자와 대면하지 않고도 템플릿 온 모바일 지원 장치에서 직접 비주얼 페이스를 등록할 수 있습니다.

    1. Credential 섹션의 Mobile Access 항목에서 버튼을 클릭하세요.

    2. Enroll Mobile 창이 나타나면 Card Type에서 Template on Mobile을 선택하세요.

    3. Smart Card Type에서 원하는 카드 종류를 선택하세요.

      • Access on Card: 사용자의 정보(카드 ID, PIN, 출입 그룹, 기간, 개인 인증 모드)를 카드에 저장할 수 있습니다.

      • UID: 보안 크리덴셜 카드입니다. 사용자의 카드 ID, PIN 데이터를 카드에 저장할 수 있습니다. BioStar X에 저장된 사용자 정보를 사용하려면 서버 매칭이 켜져 있어야 합니다.

    4. Card Information 섹션의 각 항목을 설정하세요.

    5. Information 섹션에서 원하는 옵션을 활성화하세요.

    6. Enroll 버튼을 클릭하세요.

    등록한 사용자의 메일 주소로 발급 메일을 발송합니다. 메일의 링크를 통해 에어팝 패스(Airfob Pass) 애플리케이션을 설치하고, 모바일 기기에 템플릿 온 모바일 발급을 완료해야 사용할 수 있습니다.

    Tip

    템플릿 온 모바일 인증 방법

    사용자는 템플릿 온 모바일을 발급받은 모바일 기기를 장치에 태그하고, 화면의 안내에 따라 얼굴을 인증하세요.

    Info

    • 유효기간은 다이내믹 사이트를 사용할 때 활성화됩니다. 사용자의 모바일 출입 카드의 만료 기간 및 사용 기간을 설정할 수 있습니다.

    • Information 섹션의 사진, 부서, 직함 옵션을 활성화했다면, 해당 정보를 사용자의 모바일 출입 카드에 표시할 수 있습니다.

    • 템플릿 온 모바일을 사용할 수 있는 장치와 펌웨어 버전은 아래 목록을 참고하세요.

      • BioStation 3 펌웨어 1.2.0 이상

      • BioEntry W3 펌웨어 1.0.0 이상

    • 모바일 출입 카드는 CSN 모바일 카드와 템플릿 온 모바일 중 하나만 사용할 수 있습니다.

    • 템플릿 온 모바일은 사용자당 하나만 발급할 수 있습니다.

    CSN 모바일 카드​

    1. Credential 섹션의 Mobile Access 항목에서 버튼을 클릭하세요.

    2. Enroll Mobile 창이 나타나면 Card Type에서 CSN을 선택하세요.

    3. Smart Card Type에서 원하는 카드 종류를 선택하세요.

      • Access on Card: 사용자의 정보(카드 ID, PIN, 출입 그룹, 기간, 개인 인증 모드)를 카드에 저장할 수 있습니다.

      • UID: 보안 크리덴셜 카드입니다. 사용자의 카드 ID, PIN 데이터를 카드에 저장할 수 있습니다. BioStar X에 저장된 사용자 정보를 사용하려면 서버 매칭이 켜져 있어야 합니다.

    4. Card Information 섹션의 각 항목을 설정하세요.

    5. Information 섹션에서 원하는 옵션을 활성화하세요.

    6. Enroll 버튼을 클릭하세요.

    Info

    • 유효기간은 다이내믹 사이트를 사용할 때 활성화됩니다. 사용자의 모바일 출입 카드의 만료 기간 및 사용 기간을 설정할 수 있습니다.

    • Information 섹션의 사진, 부서, 직함 옵션을 활성화했다면, 해당 정보를 사용자의 모바일 출입 카드에 표시할 수 있습니다.

    • 이메일 또는 텍스트 메시지를 통해 전송한 활성화 코드를 사용자가 분실하거나 삭제하여 활성화 코드를 다시 발급하려면 재전송 버튼을 클릭하세요.

    • CSN 모바일 애플리케이션에 대한 자세한 내용은 다음 링크를 참고하세요.

    QR/Barcode 등록​

    QR 코드 또는 바코드를 사용자 인증 수단으로 사용할 수 있습니다.

    Info

    • QR/Barcode 인증을 사용할 수 있는 스캐너를 포함하는 장치는 아래를 참고하세요.

      X-Station 2 (XS2-QDPB, XS2-QAPB)

    • QR/Barcode 인증을 사용할 수 있는 카메라를 포함하는 장치는 아래 목록을 참고하세요.

      • X-Station 2 (XS2-ODPB, XS2-OAPB, XS2-DPB, XS2-APB) 펌웨어 1.2.0 이상

      • BioStation 3 (BS3-DB, BS3-APWB) 펌웨어 1.1.0 이상

      • 카메라로 QR/Barcode 인증을 사용하려면 별도의 장치 라이선스가 필요합니다. 자세한 내용은 다음 문서를 참고하세요.

    BioStar 2 QR​

    BioStar X에서 암호화된 PIN을 포함한 QR 코드를 사용자에게 발급할 수 있습니다.

    1. Credential 섹션의 QR/Barcode 항목에서 버튼을 클릭하세요.

    2. Enroll QR/Barcode 창이 나타나면 QR/Barcode Type 항목에서 BioStar QR를 선택하세요.

    3. 카드 ID를 직접 입력하거나 카드 ID를 자동 생성하려면 Use Random ID 옵션을 활성화하세요.

    4. Enroll 버튼을 클릭하세요.

    사용자에게 발급한 QR 코드를 사용자 정보에 등록된 이메일 주소로 전송합니다.

    Info

    • 중복된 카드 ID 생성을 방지하기 위해 Input Type 항목에서 Use Random ID 옵션을 활성화하기를 권장합니다.

    • BioStar QR를 사용하기 전에 SMTP 설정을 포함한 이메일 정보 설정을 완료하세요. 자세한 내용은 다음 문서를 참고하세요.

    • BioStar QR를 발급하려면 반드시 사용자 정보에 이메일 주소를 등록해야 합니다.

    QR/Barcode​

    외부의 기타 발급처에서 발급된 QR/바코드를 사용자에게 등록할 수 있습니다.

    1. Credential 섹션의 QR/Barcode 항목에서 버튼을 클릭하세요.

    2. Enroll QR/Barcode 창이 나타나면 QR/Barcode Type 항목에서 QR/Barcode를 선택하세요.

    3. 카드 ID를 입력하세요.

    4. Enroll 버튼을 클릭하세요.

    Info

    카드 ID는 영문, 숫자, 특수 기호를 포함하여 최대 32자까지 입력할 수 있습니다.

    패스코드 등록​

    사용자가 장치에 비밀번호를 입력해 출입할 수 있도록 비밀번호를 할당할 수 있습니다.

    1. Credential 섹션의 Passcode 항목에서 버튼을 클릭하세요.

    2. Enroll Passcode 창이 나타나면 패스코드를 입력하세요.

    3. Confirm Passcode 항목에 한 번 더 패스코드를 입력하세요.

    4. Enroll 버튼을 클릭하세요.

    Info

    CSV 불러오기 기능을 이용해 다수의 사용자 PIN을 한 번에 등록할 수 있습니다. 자세한 내용은 다음 문서를 참고하세요.

    Access PIN 등록​

    사용자가 장치에 비밀번호를 입력해 출입할 수 있도록 비밀번호를 할당할 수 있습니다.

    1. Credential 섹션의 Access PIN 항목에서 버튼을 클릭하세요.

    2. Enroll Access PIN 창이 나타나면 PIN을 입력하세요.

    3. Confirm Access PIN 항목에 한 번 더 PIN을 입력하세요.

    4. Enroll 버튼을 클릭하세요.

    Info

    CSV 불러오기 기능을 이용해 다수의 사용자 PIN을 한 번에 등록할 수 있습니다. 자세한 내용은 다음 문서를 참고하세요.
