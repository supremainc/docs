Card Printer

BioStar X and cardPresso can be linked to print a card with a design the user wants from BioStar X.

Installing and configuring the cardPresso​

To use the card printer feature of BioStar X, install and configure the cardPresso program as below.

  1. Download the latest version of cardPresso to the PC where you want to print cards using cardPreso. Refer to the following link for the download path.

  2. Install the program and activate the license issued by cardPresso.

    • License type: cardPresso XXL edition

    • Element IDs supported by BioStar X are as below. Make sure to enter XML item ID name as element ID when creating a card template.

    6User GroupGROUP
    7Profile PhotoPHOTO

    9Custom Field 2CUSTOM2

  3. After setting the save path of Card Template to be used in the cardPresso Web Print Server, click Save.

  4. Right-click on the cardPresso icon created on the desktop and run Create Shortcut.

  5. Right-click on the copied shortcut icon and select Properties. On the Shortcut tab, add a space at the end of the Target, add /PRINTSERVER and click OK. This shortcut will act as a Web Print Server shortcut.

  6. Double-click the modified cardPresso Web Print Server shortcut icon to run it, then set each item.

    • Address: Enter the IP address of the computer acting as the Web Print Server.

    • Port: Port number on which cardPresso receive print operations.

    • Allowed user groups: Select a user group that has access to the Web Print Server.

  7. Click Save when the configuration is complete.

  8. Click at the bottom right to start the service. If properly set, the icon changes to and a listen OK message appears.

    • Do not close the window until card printing is complete.

Connecting cardPresso to BioStar X​

To connect cardPresso to BioStar X, set BioStar X as below.

  1. Click Settings on the Launcher page.

  2. In the left sidebar, click Card Printer.

  3. Activate the cardPresso Setting to Use, then enter each item.

    • ID: Enter your cardPresso login ID.

    • Password: Enter your cardPresso login password.

    • IP Address: Enter the IP address of the PC that runs the cardPresso web print server.

    • Port: Enter the port number used by cardPresso to receive print operations.

    • Printer Name: Enter the name of the printer to be used as a card printer. It can be found in the Windows Control Panel → Devices and printers.

    • Card Template: Click + Add to add a Card Template. For Name, enter the full path including the card template file name and extension.

      • The file name of the card template can be entered using alphanumeric, and special characters.

      • Up to 20 card templates are supported.

      • Example path: C:\template\example.card

    • Card Template Print Test: Test print the card template.

  4. Click Apply when the configuration is complete.

Test printing the card template​

Test print a card template with Administrator information.

  1. Click Settings on the Launcher page.

  2. In the left sidebar, click Card Printer.

  3. Select the card template to print, then click Test Print.

  4. Card template with Administrator information is printed.

Print card templates by user​

Print cards with per-user card templates.

  1. Click User on the Launcher page.

  2. Click the New User button at the top right of the screen. The New User window appears.

  3. Enter each item and click Print Card.

    • Each item can be entered with alphanumeric and special characters.

  4. Select the card template to print and click Print.

  5. When a pop-up window asking whether to print the card appears, click OK.

  6. New user will be created and a card is printed as a card template with user information.

Add custom user field​

Print a card with a card template with custom user field added.

  1. Click Settings on the Launcher page.

  2. In the left sidebar, click the Server.

  3. In the Custom User Field, click + Add to enter a Name and select a Type. The order of the Custom User Field adds decides the element ID of the card template.

    • Based on the BioStar X element ID, Order 1 is the same as CUSTOM1. For more information about BioStar X element ID, refer to the following.

  4. Click Apply when the configuration is complete.

  5. Open the card template file by clicking File → Open Template in the cardPresso program. Add a new element and enter the ID as the XML item ID name in the BioStar X element ID order.

  6. Click File → Save when the configuration is complete.

  7. In the User menu of BioStar X, select the user to apply the card template. Items added from Custom User Field are appears. Enter each item and click Print Card.

  8. Select the card template to print and click Print.

  9. When a pop-up window asking whether to print the card appears, click OK.

  10. User information is saved and the card is printed as a card template with custom user fields added.
