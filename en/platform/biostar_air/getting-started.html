Getting Started

The BioStar Air portal provides a unified interface across the web and mobile apps, delivering a consistent user experience. Through the BioStar Air portal, administrators can manage access control settings, monitor site usage, and seamlessly configure users and devices.

Customers who began using BioStar Air access control before May 2025 may still be using the Airfob Space app, which has since been rebranded as the Suprema Pass app. Both apps offer identical functionality and are fully compatible. There is no need for existing users to switch â€” mobile credentials work seamlessly across both.

Access the portalâ€‹

Open a web browser and go to https://admin.biostarair.com to access the BioStar Air portal. When the login page appears, select Global Server or EU Server based on your location. Obtain an administrator account and have another administrator grant you permissions to sign in.

Review the dashboardâ€‹

After logging in, the Dashboard appears and shows key site information.

  • Tagged Status: View the number of users who accessed the site today.

  • Active Users: View the total number of registered and active users. Click the card to open User Management.

  • Devices: View the number of registered devices. Click the card to open Device Management.

  • Credit Status: View the remaining account credits. Click the card to open Settings â†’ Credit Center.

  • Usage: Sort usage by device group, frequency (daily, weekly, monthly), or custom date range.

  • Recent Actions: Shows an activity log of administrator actions such as sign-ins, device updates, and user creation.

Site setup guideâ€‹

Follow these instructions to set up a new site.

Add administratorsâ€‹

Invite administrators.

  1. Click Settings â†’ Admin in the left sidebar.

  2. Click Invite at the top right of the screen.

  3. When the Admin Invitation window appears, enter the administratorâ€™s email address and assign a role.

  4. Click Send.

Register devicesâ€‹

Review registered devices or add new ones. Click Device Management â†’ Registered Devices in the left sidebar.

  • If the device is not registered, ask another administrator to register it on-site with the BioStar Air mobile app.

  • In Registered Devices, delete devices, update firmware, and manage device groups.

Create and manage doorsâ€‹

Add doors and assign devices.

  1. Click Door Management â†’ Registered Doors in the left sidebar.

  2. If no doors are listed, click Add Door in the upper-right corner.

  3. When the Add New Door panel appears, enter a name and description.

  4. Assign registered devices to Entry Device and Exit Device.

In Door Management, edit or delete doors, configure door schedules, and manage them by group.

Categorize doors into groupsâ€‹

Categorize doors into groups.

  1. Click Door Management â†’ Door Groups in the left sidebar.

  2. Click Add Door Group at the top right of the screen.

  3. When the Add Door Group window appears, enter a group name such as Indoor, Outdoor, or Trap Doors, and select the desired doors.

  4. After selecting all doors, click OK.

Configure access rightsâ€‹

Follow these steps to manage access rights.

Set up holiday groupsâ€‹

  1. In the left sidebar, click Access Management â†’ Holiday Groups.

  2. Click Add Holiday Group at the top right of the screen.

  3. When the Add Holiday Group panel appears, enter a name (for example, "Korea Holidays").

  4. In the Add Holidays section, set the holidays on which to restrict access, and click Add.

  5. After adding all holidays, click Save.

Create schedulesâ€‹

  1. Click Access Management â†’ Schedules in the left sidebar.

  2. Click Add Schedules at the top right of the screen.

  3. When the Add Schedule panel appears, enter a name.

  4. Select the days to activate and link a holiday group.

  5. When the schedule is complete, click Save.

Define access levelsâ€‹

  1. Click Access Management â†’ Access Levels in the left sidebar.

  2. Click Add Access Level at the top right of the screen.

  3. When the Access Level panel appears, enter a name, and assign door groups, individual doors, and schedules.

  4. When the access level is configured, click Add.

Create user groups and add usersâ€‹

Create a user group and add users to the new group.

  1. Click User Management â†’ User Groups in the left sidebar.

  2. Click Add User Group at the top right of the screen.

  3. When the Add User Group window appears, enter the user group name.

  4. Click Save to save the user group.

  5. Click User Management â†’ Users in the left sidebar.

  6. Click Add User â†’ Add User at the top right of the screen.

  7. When the Add New User(s) window appears, enter the user ID, name, email, and so on in the User Detail section.

  8. In the Credential Detail section, enable one or more credential types.

  9. In the Access Detail section, select an access level.

  10. To finish configuring the userâ€™s access, click Activate Credential in the lower-right corner.

Info

In the User Management â†’ User Groups menu, bulk-manage users, reissue, suspend, or delete them, export data, and upload profile images in bulk.

Tip

Checklist

To summarize, follow these steps.

  1. Add administrators.

  2. Register devices.

  3. Create doors and assign devices.

  4. Categorize doors into groups.

  5. Configure access rights (holidays, schedules, access levels).

  6. Create user groups and add users.

Following these steps sets up the BioStar Air site smoothly. Refer to the individual video tutorials in the user guide for more details.
