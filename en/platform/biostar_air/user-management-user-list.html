Manage Users

Customers who began using BioStar Air access control before May 2025 may still be using the Airfob Space app, which has since been rebranded as the Suprema Pass app. Both apps offer identical functionality and are fully compatible. There is no need for existing users to switch — mobile credentials work seamlessly across both.

User list​

The user list shows all users registered in BioStar Air at a glance. View each user's details and use the filter and sort features to quickly find specific users. Edit, delete, and assign groups to users directly from the user list.

Click User Management → Users in the left sidebar.

User Information​

The table in the user list provides the following information.

  • User Type: Indicates whether the user is a regular or temporary user. For more information on regular and temporary users, refer to the following.

  • Name: The user's name.

  • ID: The user's unique ID.

  • Credential Type: Shows the credentials issued to the user. For more information on credential types, refer to the following.

  • Valid Through: The validity period of the credentials issued to the user.

  • User Status: Indicates whether the user is active or inactive.

  • User Group: The group to which the user belongs. For more information on user groups, refer to the following.

  • Access Level: The access level assigned to the user. For more information on access levels, refer to the following.

  • Floor level: The user's permissions for floors they can access. For more information on floor levels, refer to the following.

Filter​

Use the filter feature in the top header of the user list. Click in the header to set the desired conditions.

Info

Click at the top of the user list to reset the applied filter conditions.

Additional features​

The features available at the top of the user list are as follows.

  • : Assign, remove, or move the selected users to a group. For more information on user group management, refer to the following.

  • : Change the access level of the selected users. For more information on managing access levels, refer to the following.

  • : Allow the selected users to tag mobile cards consecutively without waiting time. The Suprema Pass app supports this starting from version 2.9.0.

  • : Apply a user template to the selected users. Add any other information needed in addition to the default user information.

  • : Export the selected user information to a PDF or CSV file.

  • : Upload user images in bulk. The name of each image file to upload must match the user's ID.

    For example, the file 601.jpg corresponds to user ID 601.

    File format requirements

    • Supported formats: .jpg, .jpeg, .png

    • File size limit: 1 MB per image

  • : Send a face enrollment link to the selected users by email or text message. When a user clicks the link and enrolls their face on the webpage, they are granted access via facial authentication.

    The link cannot be sent to users without a registered email address or phone number.

  • : Reset all filters applied to the user list. This feature is available only when filters are applied. For more information on applying filters, refer to the following.

  • : Configure user reports. After a report is generated, it is sent to the administrator's email. For more information on report settings, refer to the following.

Edit users​

Follow these steps to edit a user's details or change their credentials.

  1. Click User Management → Users in the left sidebar.

  2. Click the user you want to edit in the list.

  3. Click in the top right of the user details.

  4. Edit the required details when the Edit User window appears.

  5. After editing, click Save at the bottom right.

Manage user status​

Activate or suspend a user's access permissions. Reissue credentials as well.

Individual user​

  1. Click User Management → Users in the left sidebar.

  2. Click the user in the list to change their status.

  3. Select the desired action at the top of the user details.

  • : Restore access permissions to suspended users. Use this feature when the user is suspended.

  • : Reissue credentials. Use this feature when the user's credentials have expired or are unavailable.

  • : Block the user's access. Use this feature when the user is active.

Batch actions​

  1. Click User Management → Users in the left sidebar.

  2. Click the checkbox of each user in the list to change their status. You can select more than one user.

  3. Click at the top left of the screen and select the desired action from the pop-up menu.

  • Activate: Restore access permissions to suspended users. Use this feature when the user is suspended.

  • Reissue: Reissue credentials. Use this feature when the user's credentials have expired or are unavailable.

  • Suspend: Block the user's access. Use this feature when the user is active.

Delete users​

Caution

Deleted users cannot be recovered. Be sure to verify before deleting.

Individual user​

  1. Click User Management → Users in the left sidebar.

  2. Click the user you want to delete in the list.

  3. Click in the user details.

  4. When a warning message appears, click OK.

Delete multiple users​

  1. Click User Management → Users in the left sidebar.

  2. Click the checkbox of each user you want to delete in the list. You can select more than one user.

  3. Click → Delete at the top left of the screen.

  4. When a warning message appears, click OK.

Configure user reports​

Create a user report and send it to the administrator's email.

  1. Click at the top of the user list.

  2. When the Report Settings window appears, enter a report name.

  3. Select the items to include in the report from each filter option.

  4. Click Add Report to save the report.

  5. A Report List is created and a download link is sent to the administrator's email.

Info

In Report List, search for, edit, download, or delete generated reports.

Search users​

Use the search field in the top right of the user list to quickly find specific users. Search by user name, ID, or phone number.
