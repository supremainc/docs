Authentication

Describes the supported authentication methods and authentication procedures.

Info

If you use Suprema CLUe mode, you can set the authentication mode only on the Suprema CLUe server, not on the device.

Authentication mode​

Set the authentication mode. You can combine Fingerprint, PIN, Card, and ID for authentication mode, and you can also set schedules for each authentication mode.

Fingerprint authentication mode​

You can set schedules for each authentication method that uses fingerprints.

  1. Press ESC, enter the password, and access the menu.

  2. Press 6 after selecting AUTHENTICATION → Auth Mode → Fingerprint Mode.

  3. Select the desired item and press 4 or 6 to set the schedule.

    • : This mode uses only fingerprints.

    • + : This mode authenticates with a fingerprint followed by entering a PIN.

  4. Press OK to save the settings.

Info

Schedule can be set on the Suprema CLUe server. If you have no scheduled settings, you can only select Never and Always.

Card Authentication Mode​

You can set schedules for each authentication method that uses cards.

  1. Press ESC, enter the password, and access the menu.

  2. Press 6 after selecting AUTHENTICATION → Auth Mode → Card Mode.

  3. Select the desired item and press 4 or 6 to set the schedule.

    • : This mode uses only cards.

    • + : This mode authenticates with a card followed by fingerprint authentication.

    • + : This mode uses a card for authentication followed by entering a PIN.

    • + / : This mode authenticates with a card followed by either fingerprint or PIN entry.

    • + + : This mode uses a card followed by both fingerprint authentication and PIN entry.

  4. Press OK to save the settings.

Info

Schedule can be set on the Suprema CLUe server. If you have no scheduled settings, you can only select Never and Always.

ID Authentication Mode​

You can set schedules for each authentication method that uses IDs.

  1. Press ESC, enter the password, and access the menu.

  2. Press 6 after selecting AUTHENTICATION → Auth Mode → ID Mode.

  3. Select the desired item and press 4 or 6 to set the schedule.

    • + : This mode authenticates with an ID followed by a fingerprint.

    • + : This mode uses an ID followed by entering a PIN.

    • + / : This mode uses an ID followed by either fingerprint authentication or entering a PIN.

    • + + : This mode uses an ID followed by both fingerprint authentication and entering a PIN.

  4. Press OK to save the settings.

Info

Schedule can be set on the Suprema CLUe server. If you have no scheduled settings, you can only select Never and Always.

Attendance Mode​

You can set how to register T&A mode.

  1. Press ESC, enter the password, and access the menu.

  2. Select AUTHENTICATION → T&A, then press 6.

  3. Select the desired item and press 4 or 6 to set the schedule.

    • T&A Event: You can check attendance events.

    • T&A Mode: Set how to use attendance mode.

    • T&A Required: Set to require a user to select a T&A event when authenticating. Setting Enabled allows you to require selecting a time and attendance event when authenticating.

    • T&A Fixed: You can configure to use only time and attendance events selected by an administrator. Available when T&A Mode is set to Fixed.

    • Job Code: Set whether to use the job code.

  4. Press OK to save the settings.

Info

Set Job Code to Enabled to change the user's action code. Long press OK then authenticate to select the desired work code.

Fingerprint Settings​

You can change the settings for fingerprint authentication.

  1. Press ESC, enter the password, and access the menu.

  2. Press 6 after selecting AUTHENTICATION → Fingerprint.

  3. Select the desired item and press 4 or 6 to set the schedule.

    • Security Level: You can set the security level for 1:N authentication.

    • Matching Timeout: Set the wait time for fingerprint matching. If authentication is not completed within the specified time, it will fail.

    • View Image: The original image of the scanned fingerprint can be viewed.

    • Sensor Sensitivity: Set the sensitivity of the fingerprint recognition sensor. Set the sensitivity higher to obtain precise fingerprint information.

    • Live Finger Detection: Set the fake fingerprint detection level. Higher levels for fake fingerprint detection may also increase rejection rates for actual people’s fingerprints.

    • 1:N Fast Mode: Set the speed of fingerprint authentication. When you select Auto, authentication speed is set according to the total number of fingerprint templates enrolled in the device.

    • Template Format: Set the fingerprint template format. SUPREMA is set as default. Changing the template format will make all previously saved fingerprints unusable, so take care when changing.

      Info

      Delete all user fingerprint information and then change the template format. If user fingerprint information is enrolled, you cannot change the template format.

    • Sensor Mode: Set to Auto On to turn on the fingerprint sensor when it detects the user’s finger. When set to Always On, the sensor remains on even when not scanning fingerprints.

    • Advanced Enrollment: Assess the quality of scanned fingerprints to save high-quality fingerprint information. Enable to notify the user when fingerprint quality is low, helping them scan fingerprints correctly.

    • Duplicate Check: Check for duplicate fingerprints when enrolling.

  4. Press OK to save the settings.

Server matching​

When you enable server matching, the device authenticates users or Suprema CLUe users according to the set authentication mode. It can be used when the device has too much user information or when the device is installed outside.

  1. Press ESC, enter the password, and access the menu.

  2. Press 6 after selecting AUTHENTICATION → Server Matching.

  3. Press 4 or 6 to change the settings.

  4. Press OK to save the settings.

Info

  • If you use Server Matching, enable server matching on both the device and the Suprema CLUe server.

  • If you use Suprema CLUe mode, Server Matching is enabled automatically.
