라인 통과 감지 이벤트 설정하기

Not translated
This page is currently being translated. We will complete the translation as soon as possible.

라인 통과 감지 이벤트를 설정하면 설정한 가상 라인을 통과하는 사람을 감지하여 실시간으로 이벤트를 발생합니다. 통과 방향(A→B, B→A, 양방향)을 지정하여 특정 방향의 출입만 모니터링할 수 있으며, 선택적으로 얼굴 영역을 추가하면 등록된 사용자를 자동으로 신원을 인식합니다.

언제 사용하나요?​

  • 출입구의 통행량 모니터링 및 카운팅

  • 특정 방향으로의 출입만 감시 (한 방향 통행로 감지)

  • 업무시간 외 특정 영역으로의 출입 감지

  • 라인을 기준으로 좌측/우측 구역 이동 추적

라인 통과 감지 이벤트 생성​

라인 통과 감지 이벤트를 생성하면 Live 메뉴에서 특정 구역에 사람이 라인을 통과하는 것을 감지하여 이벤트를 발생시킬 수 있습니다. 또한 얼굴 영역을 설정하면 해당 영역에서 얼굴을 우선적으로 추적할 수 있으며, 등록된 사용자는 신원을 자동 인식합니다.

이벤트 생성 시작​

  1. 화면 왼쪽 상단의 Settings 메뉴를 클릭하세요.

  2. 화면 왼쪽 사이드바에서 AI & Rule → AI Analysis를 클릭하세요.

  3. 화면 오른쪽 상단의 + Add 버튼을 클릭하세요.

  4. Select AI Analysis Type 창에서 Line Crossing를 선택하고 Next 버튼을 클릭하세요.

  5. Face Recognition Usage Consent 팝업이 표시되면 내용을 확인하고 Agree 버튼을 클릭하세요.

    • 얼굴 인식 기능을 사용하려면 해당 지역의 관련 규제 법규를 준수해야 합니다. 동의하지 않으면 진행할 수 없습니다.

    • Face Recognition Usage Consent 팝업은 최초 한 번만 표시됩니다. 이후 AI 이벤트 생성 시에는 동의한 것으로 간주합니다.

  6. 오른쪽 Line Crossing 패널에서 필요한 설정을 완료하세요.

이벤트 이름 설정​

Name 입력란에서 AI 이벤트 이름을 입력하세요. 이름은 최대 64자까지 입력할 수 있습니다.

가상 라인 설정​

라이브 뷰에서 가상 라인을 적절한 위치에 설정하세요. 카메라에 설정한 가상 라인을 통과하는 사람을 감지하여 이벤트를 발생시킵니다. 최소 1개를 설정해야 하며, + Add 버튼을 클릭해 최대 10개까지 추가할 수 있습니다. 라이브 뷰에 주황색 라인이 표시되며, 마우스로 모서리 또는 점을 드래그하여 원하는 형태로 조정할 수 있습니다. 가운데 영역을 드래그하여 위치를 이동할 수도 있습니다.

  • Name: 가상 라인의 이름을 수정할 수 있습니다. 알기 쉽게 이름을 지정하세요. 가상 라인의 이름은 라이브 뷰에서 마우스 오버 시 표시됩니다.

  • Condition: 가상 라인을 통과하는 조건을 설정하세요. 아래 3가지 방향 옵션 중 하나를 선택할 수 있습니다.

    • A -> B: A에서 B 방향으로 통과 시 이벤트 발생

    • B -> A: B에서 A 방향으로 통과 시 이벤트 발생

    • A <-> B: 양방향으로 통과 시 이벤트 발생

  • Face Matching: 가상 라인을 통과하는 사람의 신원을 자동으로 확인하려면 On으로 설정하세요.

情報

  • 설정한 가상 라인의 모양과 크기를 초기화하려면 버튼을 클릭하세요.

  • 추가한 가상 라인을 삭제하려면 버튼을 클릭하세요.

얼굴 영역 설정(선택 사항)​

얼굴을 감지할 영역을 설정하세요. 설정하면 해당 영역에서 얼굴을 우선 추적합니다. 얼굴 영역과 가상 라인은 겹쳐서 설정할 수 있습니다. + Add 버튼을 클릭해 최대 10개까지 설정할 수 있습니다. 라이브 뷰에 파란색 상자가 표시되며, 마우스로 모서리 또는 점을 드래그하여 원하는 형태로 조정할 수 있습니다. 가운데 영역을 드래그하여 위치를 이동할 수도 있습니다.

  • Name: 얼굴 영역의 이름을 수정할 수 있습니다. 알기 쉽게 이름을 지정하세요. 얼굴 영역의 이름은 라이브 뷰에서 마우스 오버 시 표시됩니다.
情報

  • 등록된 사용자는 Live 메뉴에서 신원을 자동 인식해 이름이 표시됩니다. 등록되지 않은 사용자는 Unknown Face으로 표시됩니다.

  • 설정한 얼굴 영역의 모양과 크기를 초기화하려면 버튼을 클릭하세요.

  • 추가한 얼굴 영역을 삭제하려면 버튼을 클릭하세요.

이벤트 발생 조건 설정​

이벤트 발생 조건을 설정하세요.

  • Every Delay: 이벤트 발생 지연 시간을 설정하세요. 설정한 시간을 초과했을 때만 이벤트가 발생하며, 최대 60초까지 설정할 수 있습니다.

설정 저장​

모든 설정을 완료한 후 오른쪽 상단의 Create 버튼을 클릭하세요. 이전 화면으로 돌아가며, 새로 생성된 AI 이벤트가 목록에 추가됩니다.

라이브에서 확인​

화면 왼쪽 상단의 Live 메뉴를 클릭하세요. 카메라 영상의 오른쪽 상단의 아이콘이 켜져 있는지 확인하세요. 설정한 가상 라인과 얼굴 영역이 영상에 오버레이로 표시됩니다. 사람이 가상 라인을 통과할 때마다 이벤트가 발생합니다. 얼굴 영역을 설정했다면 카메라가 해당 영역에서 얼굴을 감지하며, 등록된 사용자는 자동으로 신원이 인식되어 이름을 표시합니다.

情報

  • 등록되지 않은 사용자는 Unknown Face로 표시됩니다. 등록된 사용자는 이름과 함께 표시합니다.

  • 조명이 부족하거나 역광이 있으면 카메라가 얼굴을 인식하지 못할 수 있습니다. 촬영 공간의 조명을 개선하거나 카메라의 위치와 각도를 조정하여 얼굴이 잘 보이도록 하세요.

  • 얼굴이 카메라로 향해야 인식됩니다. 옆모습 또는 뒷모습은 인식이 어려울 수 있습니다.

  • 너무 많은 이벤트가 발생하면 Every Delay 옵션값을 조정하세요. 이벤트 발생 지연 시간을 늘리면 이벤트 빈도를 줄일 수 있습니다.
