May 2026

2026年5月28日

This May, Suprema Docs expanded in both scope and scale. The new device BioStation 3 Max has joined the lineup, and BioStar X and BioStar Air have received significant feature enhancements. Don't miss the highlight of this month—a brand-new Integration category covering third-party product connectivity.

Platform​

BioStar X​

  • Read the v1.0.2 Release Notes

  • Change the Main Server IP Address: Added instructions for changing the BioStar X main server IP address

  • Directory Integration Settings: Added Add/Update/Delete modes to directory service integration settings

  • System Restore: Updated detailed procedures for using the restore tool

  • Delete Event Logs from Device: Added the feature to delete event logs stored on the device

  • Integrate VMS: Added support for Nx Witness v6.x.x

  • Set Up the Camera: Added instructions for configuring camera group permission settings

  • Added support specifications for BioStation 3 Max and XPass Q2 devices

BioStar Air​

  • Read the SW v2.12 Release Notes

  • Install Airfob Patch: Added instructions for installing the Airfob patch
