January 2026

2026年1月2日

Suprema Docs is a knowledge hub that brings together all product manuals and technical documentation in one place. It includes the following key documentation:

  • Product manuals: Installation and user manuals for Suprema products

  • Solution manuals

    • User manual for BioStar X

    • User manual for BioStar Air (coming soon)

  • Technical documents: APIs and other technical resources

  • Release Notes

Below is a summary of the key updates for January 2026.

BioStar X​

  • Added BioStar X v1.0.1 Release Notes

  • Added guide for IR Face credentials

  • Added guide for Remote Access configuration and usage

  • Added guide for integrating event logs using the Virtual Device feature

  • Updated guides for the ability to enter a reason when exporting or printing user, data, and audit trail menus

    • Export/Import User Information

    • Generate Report

    • Manage Audit Trail

  • Updated License Policy details

    • Added description that devices cannot be added without a Base License

    • Added description of limit on the number of Ethernet-connected devices

    • Added description of maximum administrator count

  • Added BioStar X API documentation

  • Added BioStar X License Calculator page to help with complex licensing requirements

What’s next and expected benefits​

Suprema Docs will continue to evolve. We will keep updating the content to improve completeness and quality, so it better meets the needs of our customers and partners. Internally, it will also help you quickly share accurate information needed for product development, technical support, and sales activities. We also plan to expand multilingual support in the near future. Documentation in Japanese and Spanish will be released soon.
