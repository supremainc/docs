March 2026

2026年4月8日

This March update improves your Suprema Docs experience with a more intuitive interface and enhanced search functionality. Meanwhile, the BioStar Air Administrator Guide has been completely restructured to help you understand all features—from user management to access control and video surveillance—at a glance.

Docs​

  • Redesigned home page

  • Enhanced search functionality

    Added product-specific search and keyboard shortcut (Ctrl + K) to open the search bar

Platform​

BioStar X​

  • Updated documentation for VMS Server Integration and Certificate Installation

BioStar Air​

  • Completed comprehensive restructuring and optimization of the BioStar Air Administrator Guide

  • Expanded multilingual support for BioStar Air: English, Japanese, and Spanish translations added

    • Added documentation for key features including user management, access rights management, and video management

    • Added English interface images and user interface screenshots

  • Improved BioStar Air table of contents structure and reorganized sections

Devices​

Added new release notes and updated existing device documentation.

Device Documentation Updates​

  • XPass 2: Added v1.5.1 release notes

  • Device Manager: Established new app release notes category and added v1.2.2 release notes

  • BioEntry P2: Added v1.6.0 release notes

  • Added instructional images for device factory reset button location

Developer​

  • API documentation updated: Create BioStar X QR

  • Device SDK: Added v2.9.12 release notes

Multilingual Support​

  • Improved documentation quality for Japanese and Spanish versions

    • Enhanced consistency of document titles and content

    • Corrected mistranslations and standardized terminology

    • Fixed spacing and grammar issues

  • Significantly updated English translation

    • Standardized technical terminology

    • Improved sentence structure and phrasing

Open Source Software Licenses​
