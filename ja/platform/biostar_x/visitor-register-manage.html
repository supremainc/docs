来訪申請を管理する

来訪申請を承認または修正できます。 管理者が直接来訪者を追加または削除することもできます。

来訪承認​

来訪申請履歴を確認し、来訪者の入退室を承認する方法を説明します。

  1. ランチャー ページで 訪問者 をクリックします。

  2. 来訪者一覧で、承認する来訪者をクリックします。

  3. 来訪者の来訪情報を確認して修正が必要な項目を修正した後、編集 をクリックします。

  4. 画面右下の ﾁｪｯｸｲﾝ をクリックします。

  5. 登録情報 の内容を確認し、来訪を承認するには、画面右下の 承認 をクリックします。

情報

  • 来訪申請時に来訪者入退室利用規約に同意していない申請者の場合、承認 は無効化されます。 条件を表示 をクリックして来訪者に入退室利用規約を提示し、規約への同意を取得します。 来訪者が入退室利用規約に同意しない場合、来訪が制限されます。

  • 来訪PCにカード登録デバイスが設定されている場合、承認 および 登録ｶｰﾄﾞ が有効になります。 来訪を承認して入退室カードを発行するには、承認 および 登録ｶｰﾄﾞ をクリックします。 詳細については次のドキュメントを参照してください。

  • 来訪PCでカード利用を設定するには、設定 → 訪問者 メニューで以下の最小設定が必要です。 詳細については次のドキュメントを参照してください。

    • ｻｲﾄ 設定 → ｶｰﾄﾞ 使用 / ｶｰﾄﾞ種別

    • 訪問PC 訪問PC設定 → ｶｰﾄﾞ 端末名称

承認とカード登録​

来訪申請を承認して入退室カードを発行する方法を説明します。

  1. 登録情報 画面で、画面右下の 承認 および 登録ｶｰﾄﾞ をクリックします。

  2. カード登録ウィンドウが表示されたら、希望する登録方法を選択します。

    • ｶｰﾄﾞﾘｰﾀﾞｰによる登録
    • 手動入力

ｶｰﾄﾞﾘｰﾀﾞｰによる登録

来訪PCに接続されたデバイスでカード情報をスキャンして登録できます。

  1. 登録方法 を ｶｰﾄﾞﾘｰﾀﾞｰによる登録 に選択します。

  2. 登録するカードをデバイスにスキャンするには、ｶｰﾄﾞ読出し をクリックします。

  3. カードを登録するには、登録 をクリックします。

手動入力​

カード番号を直接入力して登録できます。

  1. 登録方法 を 手動入力 に選択します。

  2. ｶｰﾄﾞ ID を直接入力します。

  3. カードを登録するには、登録 をクリックします。

指紋検索による来訪者承認​

指紋を登録済みの来訪者を指紋で検索し、承認できます。

  1. ランチャー ページで 訪問者 をクリックします。

  2. 来訪申請一覧で左上の 訪問者を検索 をクリックします。

  3. 指紋スキャナに来訪者の指紋をスキャンします。

  4. 指紋と一致する来訪者が検索されると、確認メッセージが表示されます。 登録情報 をクリックします。

  5. 来訪者の来訪情報を確認して修正が必要な項目を修正した後、編集 をクリックします。

  6. 画面右下の ﾁｪｯｸｲﾝ をクリックします。

  7. 登録情報 の内容を確認し、来訪を承認するには、画面右下の 承認 をクリックします。

来訪者追加​

管理者が直接来訪者を追加する方法を説明します。

  1. ランチャー ページで 訪問者 をクリックします。

  2. 来訪者一覧で左上の 訪問者追加 をクリックします。

  3. 訪問者 セクションで来訪者の氏名と電話番号を入力します。

  4. ﾎｽﾄ セクションで訪問先担当者の氏名と電話番号を入力します。

    氏名または電話番号を入力すると、一致するユーザーの一覧が表示されます。 希望するユーザーを指定するには、選択 をクリックします。

  5. ﾛｸﾞｲﾝ情報 セクションで入退室グループと訪問期間を設定します。

    訪問PCに割り当てた訪問先の入退室グループのみ選択できます。 詳細は次のドキュメントを参照してください。

  6. 資格 セクションで入退室手段を設定します。

    • ｶｰﾄﾞ：カード認証を使用する場合は ﾘｸｴｽﾄ に設定します。

    • 指紋：指紋認証を使用する場合は、+ 指紋 をクリックして来訪者の指紋を登録します。

  7. 来訪者追加を完了するには、画面右下の 登録 をクリックします。

情報

  • 名前は最大48文字まで入力できます。

  • 訪問者情報入力にカスタムフィールドを追加する場合は次のドキュメントを参照してください。

来訪者削除​

  1. ランチャー ページで 訪問者 をクリックします。

  2. 来訪申請一覧で削除する来訪者のチェックボックスをクリックします。

  3. 画面右上の 訪問者削除 をクリックします。

  4. 確認メッセージが表示されたら、はい ボタンをクリックします。

情報

  • 訪問者削除 は、来訪者のチェックボックスをクリックすると有効になります。

  • 来訪者は来訪申請一覧からのみ削除できます。
