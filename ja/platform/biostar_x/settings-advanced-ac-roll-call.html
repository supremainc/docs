人数点検（ロールコール）

人員点検（Roll Call）は、特定の時点と場所での入退室記録を基に人員リストを迅速に生成し、現在位置を把握する機能です。 緊急時に、ユーザーがあらかじめ指定された場所の入室または退室デバイスで認証し、人員点検場所に到着したかどうかを把握できます。

緊急対応、安全確認、救助隊への情報提供などに使用できます。 これにより、入退室記録を効率的に活用して人員状況を管理できます。

情報

人員点検設定は アドバンス（Advanced）ライセンス以上で追加オプションとして利用できます。 ライセンスポリシーの詳細は次のドキュメントを参照してください。

人員点検の登録​

  1. ランチャーページで設定をクリックします。

  2. 画面左側のサイドバーで 高度なAC をクリックします。

  3. ｿﾞｰﾝ追加 ボタンをクリックします。

  4. 出席確認 を選択し、適用 ボタンをクリックします。

  5. 情報 セクションで追加する人員点検設定の名前と説明を入力します。

  6. 設定 セクションで詳細項目を設定します。 詳しくは 次のドキュメント を参照してください。

  7. すべての設定が完了したら、画面下部の 適用 ボタンをクリックします。

情報

名称 は最大48文字まで入力でき、他の設定と同じ名前にはできません。

設定オプションの案内​

人員点検設定の詳細項目は以下のとおりです。

  • ﾓｰﾄﾞ：適用範囲を確認できます。 人員点検設定は ｸﾞﾛｰﾊﾞﾙ モードのみサポートしており、BioStar X に登録されているすべてのデバイスで設定できます。

  • 有効/無効：設定を有効化または一時的に無効化できます。

  • 手動ステータス変更：手動で状態を変更できるように設定できます。

  • 入室端末：入室時に使用するデバイスを選択します。 登録済みデバイスの一覧から選択できます。

  • 退室端末：退室時に使用するデバイスを選択します。 登録済みデバイスの一覧から選択できます。

  • ﾕｰｻﾞｰ：入室可能なユーザーグループまたは個別ユーザーを選択できます。

情報

  • 同じデバイスを 入室端末 と 退室端末 に同時に設定することはできません。

  • 入室端末 と 退室端末 は最大 10 個まで選択できます。

  • 二重認証 を設定したデバイスは、入室端末 または 退室端末 に設定できません。

  • デバイス登録の詳細は、以下のドキュメントを参照してください。

    • デバイスを登録する
    • Wiegandデバイスを登録する
    • スレーブ装置を登録する

有効化/無効化​

無効化された人員点検設定を有効化したり、有効化されている設定を無効化できます。

  1. ランチャーページで設定をクリックします。

  2. 画面左側のサイドバーで 高度なAC をクリックします。

  3. 一覧で 出席確認 をクリックします。

  4. 人員点検設定リストの中から有効化または無効化する項目のチェックボックスをクリックします。

  5. 画面右上の 有効/無効 または 無効 ボタンをクリックします。

削除​

  1. ランチャーページで設定をクリックします。

  2. 画面左側のサイドバーで 高度なAC をクリックします。

  3. 一覧で 出席確認 をクリックします。

  4. 人員点検設定リストの中から削除する項目のチェックボックスをクリックします。

  5. 画面右上の ｿﾞｰﾝ削除 ボタンをクリックします。

  6. 警告メッセージが表示されたら 継続 ボタンをクリックします。

一覧ページをナビゲートする​

ページを移動したり、1ページに表示する一覧の件数を設定できます。 画面右上のページ移動ツールを使用します。

  • ：最初のページに移動します。

  • ：前のページに移動します。

  • ：次のページに移動します。

  • ：最後のページに移動します。

  • 目的のページに移動するには、数字入力フィールドにページ番号を入力します。

  • 1ページに表示する一覧の件数を設定するには、行選択ボックスをクリックします。

人員点検を開始する​

人員点検機能は BioStar X モバイルアプリで利用できます。

情報

BioStar X モバイルアプリは次のリンクからインストールできます。

  • iOS：Apple App Store

  • Android：Google Play

モバイルアプリで人員点検を開始する​

モバイルアプリで人員点検機能を開始するには、次の手順に従います。

  1. モバイル端末で BioStar X モバイルアプリを起動します。

  2. アプリの初期画面で Roll Call ボタンをタップします。

  3. Roll Call リストで人員点検を実行する項目の ボタンをタップします。

  4. 確認メッセージが表示されたら Yes ボタンをタップします。

人員点検が開始され、進行中の人員点検項目をタップすると人員点検状況を確認できます。

進行中の人員点検の管理​

  • 入室済みのユーザーには アイコンが表示されます。 すべてのユーザーが入室すると、人員点検終了メッセージが表示されます。 人員点検を終了するには Yes ボタンをタップします。

  • 管理者は手動でユーザーの入室または状態を変更できます。 ユーザーリストで状態を変更するユーザーの一番右にある丸いボタンを長押しします。

  • 入室ユーザーを初期化するには Reset ボタンをタップします。 Accounted ユーザーを Unacounted 状態に変更します。

  • 別の管理者によって人員点検が終了すると、通知メッセージが表示されます。

  • 進行中の人員点検を終了するには、人員点検リスト画面で ボタンをタップするか、人員点検状況画面で END ボタンをタップします。

  • 進行中の人員点検は BioStar X でも終了できます。 設定 → 高度なAC → イベント → 出席確認 メニューに移動し、進行中の人員点検項目を選択した後、画面右上の 終了 ボタンをクリックします。

  • 人員点検を完了した後、データ → レポートを生成 → 出席 で関連レポートを確認できます。 詳細は 次のドキュメント を参照します。

人員点検中のユーザー情報確認​

モバイルアプリで、人員点検中に各ユーザーの情報を確認できます。 ユーザーに緊急連絡が必要な場合や現在のユーザーの位置を把握する際に役立ちます。

人員点検中に詳細を確認するユーザー項目を左にスワイプします。 その後 ボタンをタップします。

人員点検中にユーザーへメモを残す​

モバイルアプリで人員点検中に特定のユーザーが入室していない場合、そのユーザーにメモを残すことができます。

点呼中にメモを入力するユーザー項目を左へスワイプしてください。 その後、 ボタンをタップしてください。 メモ入力画面で内容を入力してから Save ボタンをタップしてください。

メモを入力したユーザーには、点呼一覧で アイコンが表示されます。

点呼の詳細を確認​

モバイルアプリで点呼設定の詳細を確認できます。

  1. モバイル端末で BioStar X モバイルアプリを起動します。

  2. アプリの初期画面で Roll Call ボタンをタップします。

  3. 一覧で詳細を確認する項目を左へスワイプし、 ボタンをタップしてください。

点呼レポートを確認​

点呼レポートを確認するには、次の手順に従ってください。

  1. ランチャー ページで データ をクリックしてください。

  2. 画面左のサイドバーで イベント → 出席 をクリックしてください。

  3. 出席を選択 ウィンドウが表示されたら、レポートで確認する点呼項目を選択し、次 ボタンをクリックしてください。

    目的の点呼項目が表示されない場合は、画面上部の期間を調整してください。

  4. オプション設定ウィンドウでレポートに含めるデータを選択し、生成 ボタンをクリックしてください。

レポートを生成します。 レポートでは、各ユーザーの情報や入室時間、退室時間、最後の場所、メモ内容などを確認できます。
