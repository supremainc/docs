출입 권한 관리

Not translated
This page is currently being translated. We will complete the translation as soon as possible.

출입 권한 관리는 보안 시스템의 핵심입니다. 휴일 그룹, 스케줄, 출입 레벨, 층 레벨을 조합하여 사용자가 언제, 어디서 출입할 수 있는지를 세밀하게 제어하세요.

ヒント

출입 권한 관리에 대한 동영상 가이드를 시청하려면 다음 영상을 참고하세요.

2025년 5월 이전에 BioStar Air 출입 통제 서비스를 사용하기 시작한 고객은 현재 Suprema Pass 앱으로 새로 설계된 Airfob Space 앱을 계속 사용할 수 있습니다. 두 앱 모두 동일한 기능을 제공하며 완벽하게 호환됩니다. 기존 사용자는 전환할 필요가 없으며, 모바일 인증 정보가 두 앱 모두에서 원활하게 작동합니다.

휴일을 설정하고, 출입 레벨 생성, 사용자에게 출입 레벨을 할당하는 방법에 대한 자세한 내용은 다음 절차를 참고하세요.

휴일 그룹 설정​

출입을 제한할 수 있는 기간 또는 날짜를 휴일 그룹을 설정하세요.

  1. 화면 왼쪽 사이드바에서 Access Management → Holiday Groups를 클릭하세요.

  2. 화면 오른쪽 상단의 Add Holiday Group 버튼을 클릭하세요.

  3. Add Holiday Group 패널이 나타나면 그룹 이름을 입력하세요.

  4. Add Holidays 섹션에서 휴일 이름과 날짜를 지정하고 Add 버튼을 클릭하세요.

    • 날짜가 고정된 공휴일을 추가하려면 이름과 날짜를 설정하고, Repeat Annually 옵션을 선택하세요.

      예, 1월 1일 신정, 5월 5일 어린이날

    • 매년 반복되지만 일정한 기간을 갖는 휴일이라면 이름과 날짜를 설정하고, Date Range 옵션을 선택하세요.

      예, 10월 5일 ~ 10월 8일 추석 연휴

  5. 같은 방식으로 다른 휴일을 추가하세요.

  6. 휴일 그룹 설정을 저장하려면 Save 버튼을 클릭하세요.

情報

휴일 그룹 관리에 대한 자세한 내용은 다음 문서를 참고하세요.

스케줄 추가​

스케줄은 사용자가 출입문에 출입할 수 있는 시간입니다. 스케줄을 추가해 출입 시간을 정의하세요.

  1. 화면 왼쪽 사이드바에서 Access Management → Schedules을 클릭하세요.

  2. 화면 오른쪽 상단의 Add Schedules 버튼을 클릭하세요.

  3. Add Schedule 패널이 나타나면 Name과 Description(선택 사항)을 입력하세요.

  4. Time Intervals * 섹션에서 요일과 시간을 선택하세요.

    예를 들어, 근무 시간을 설정하려면 월요일부터 금요일까지, 오전 9시부터 오후 6시까지 선택하세요. 반대로 근무 시간이 아닌 스케줄을 설정하려면 토요일과 일요일, 또는 매일 오후 6시부터 다음 날 오전 9시까지 선택하세요.

  5. 선택한 일정을 적용하려면 Add 버튼을 클릭하세요.

  6. Holiday Group Time 섹션에서 스케줄에 적용할 휴일 그룹을 선택하고 시간을 설정하세요.

  7. 휴일 그룹을 적용하려면 Add 버튼을 클릭하세요.

  8. 스케줄을 추가하려면 Save 버튼을 클릭하세요.

情報

스케줄 관리에 대한 자세한 내용은 다음 문서를 참고하세요.

출입 레벨 추가​

출입 레벨을 통해 출입문, 스케줄 및 휴일에 따라 사용자의 권한을 설정할 수 있습니다.

  1. 화면 왼쪽 사이드바에서 Access Management → Access Levels을 클릭하세요.

  2. 화면 오른쪽 상단의 Add Access Level 버튼을 클릭하세요.

  3. Access Level 패널이 나타나면 Name 및 Description(선택 사항)을 입력하세요.

  4. Add Door Groups 또는 Add Individual Doors 섹션에서 출입문 그룹 또는 개별 출입문을 선택하세요.

  5. Schedule 섹션에서 출입 레벨에 적용할 스케줄을 선택하세요. 제한 없는 출입을 허용하려면 Always를 선택하세요.

  6. 출입 레벨을 추가하려면 Save 버튼을 클릭하세요.

情報

출입 레벨 관리에 대한 자세한 내용은 다음 문서를 참고하세요.

사용자에게 출입 레벨 할당​

사용자에게 출입 레벨을 할당하세요.

  1. 화면 왼쪽 사이드바에서 User Management → Users를 클릭하세요.

  2. 목록에서 출입 레벨을 할당할 사용자의 체크박스를 클릭하세요. 1명 이상의 사용자를 선택할 수 있습니다.

  3. 목록 상단의 버튼을 클릭하세요.

  4. 팝업 메뉴에서 원하는 출입 레벨을 선택하세요.

  5. OK 버튼을 클릭하세요.

情報

사용자 관리에 대한 자세한 내용은 다음 문서를 참고하세요.

층 레벨 설정​

층 레벨을 사용하면 특정 층에 대한 출입 권한을 관리할 수 있습니다.
