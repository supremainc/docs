등록 출입문 관리하기

Not translated
This page is currently being translated. We will complete the translation as soon as possible.

BioStar Air의 출입문 관리 기능은 관리자가 출입 및 퇴출 장치 할당, 릴레이 구성, 일정 설정을 통한 출입 지점 구성 방법을 안내합니다. 출입문은 보안, 화재 대피 및 근태 관리(T&A)를 위해 출입을 관리하고 사용자 이동을 추적하는 체계를 제공합니다.

ヒント

출입문 등록 및 관리에 대한 동영상 가이드를 시청하려면 다음 영상을 참고하세요.

2025년 5월 이전에 BioStar Air 출입 통제 서비스를 사용하기 시작한 고객은 현재 Suprema Pass 앱으로 새로 설계된 Airfob Space 앱을 계속 사용할 수 있습니다. 두 앱 모두 동일한 기능을 제공하며 완벽하게 호환됩니다. 기존 사용자는 전환할 필요가 없으며, 모바일 인증 정보가 두 앱 모두에서 원활하게 작동합니다.

등록된 출입문 확인​

출입문은 장치 등록 시 자동으로 생성됩니다. 장치 등록 시 마지막 단계에서 이 장치로 출입문 만들기 옵션을 선택하면 출입문이 생성됩니다.

등록된 출입문을 확인하려면 화면 왼쪽 사이드바에서 Door Management → Registered Doors을 클릭하세요.

출입문 목록은 다음과 같은 정보를 제공합니다.

  • Doors: 출입문 이름입니다. 일반적으로 장치 등록 시 자동 지정되며, 관리자는 출입문 이름을 변경할 수 있습니다.

  • Entry Device: 출입문으로 들어오는 사용자를 인증하는 장치입니다.

  • Exit Device: 출입문에서 나가는 사용자를 인증하는 장치입니다.

  • Door Group: 출입문에 할당된 그룹 이름입니다.

  • Description: 출입문의 위치에 대한 설명입니다.

  • Camera: 통합 CCTV 기능이 지원되는 경우 연결된 카메라 정보가 표시됩니다.

情報

  • 출입문 목록에서 필터링 옵셥은 Door Group 열만 제공합니다. 버튼을 클릭하고 원하는 조건을 설정하세요.

  • BioStar Air 모바일 앱에서 장치 등록에 대한 자세한 내용은 다음 문서를 참고하세요.

출입문 관리 도구​

출입문 목록에서 하나 또는 여러 개의 출입문을 선택하면 관리자는 다음 도구를 통해 출입문을 관리할 수 있습니다.

  • : 출입문 정보를 수정하거나 할당된 장치의 설정을 변경할 수 있습니다. 한 개의 출입문만 선택해야 이 기능을 사용할 수 있습니다. 자세한 내용은 다음 문서를 참고하세요.

  • : 출입문 목록에서 선택한 출입문의 상태를 업데이트할 수 있습니다.

  • : 출입문 목록에서 선택한 출입문을 사이트에서 제거합니다.

  • : 출입문 열림 및 잠김 상태를 위한 스케줄을 설정합니다. 자세한 내용은 다음 문서를 참고하세요.

  • : 출입문 목록에서 선택한 출입문을 그룹으로 구성할 수 있습니다.

  • : 출입문 리포트를 생성할 수 있습니다.

출입문 추가​

새 출입문을 추가하려면 다음 단계를 따르세요.

  1. 화면 왼쪽 사이드바에서 Door Management → Registered Doors을 클릭하세요.

  2. 화면 오른쪽 상단의 Add New Door 버튼을 클릭하세요.

  3. Add New Door 패널이 나타나면 Door Name을 입력하세요.

  4. Entry Device와 Exit Device를 선택하세요.

  5. 설치한 카메라가 있다면 Camera를 할당하세요.

  6. 출입문 설정을 저장하고 추가하려면 Save 버튼을 클릭하세요.

출입문의 장치 할당 규칙

  • 각 장치는 하나의 출입문에만 할당할 수 있습니다.

  • 장치가 이미 등록되어 있지만 사용할 수 있는 장치 목록에 나타나지 않는다면, 해당 장치가 다른 출입문에 할당되어 있는지 확인하세요. 다른 장치에 할당되어 있다면 해당 출입문에서 제거한 후 다시 시도하세요.

출입문 수정​

출입문의 세부 정보 및 설정을 수정하려면 다음 단계를 따르세요.

  1. 화면 왼쪽 사이드바에서 Door Management → Registered Doors을 클릭하세요.

  2. 목록에서 수정할 출입문을 클릭하세요. 또는 출입문 왼쪽의 체크박스를 클릭한 후 상단의 화면 왼쪽 사이드바에서 버튼을 클릭하세요.

  3. Edit Door 패널이 나타나면 Door Name 및 Description을 수정하세요.

  4. Entry and Exit Device 섹션에서 출입문의 사용자 인증을 위한 장치를 선택하세요.

    • Entry Device: 출입문으로 들어오는 사용자를 인증하는 장치를 선택하세요.

    • Exit Device: 출입문에서 나가는 사용자를 인증하는 장치를 선택하세요.

    • Camera: 통합 CCTV 기능이 지원되는 경우 이벤트 로그를 위한 CCTV 카메라를 연결할 수 있습니다.

  5. Auxiliary Device 섹션에서 릴레이 장치를 설정하세요.

    • Door Relay: 출입문 잠금을 제어하는 릴레이 장치를 선택하세요.

    • Relay Ignoring Door Sensor: 문제 해결을 위해 이 옵션을 활성화하세요.

  6. 추가 설정을 구성하세요.

    • 설치된 경우 Exit Button으로 설정할 포트를 선택하세요.

    • Door Sensor로 설정할 포트를 선택하세요.

    • Open Time에서 출입문이 열린 상태로 유지되는 시간을 설정하세요. (기본값: 3초)

  7. 수정 사항을 적용하려면 Save 버튼을 클릭하세요.

情報

  • 보안 구역 내에 있는 릴레이를 Exit Device로 할당할 것을 권장합니다.

  • Entry Device 및 Exit Device는 별도의 네트워크에 연결되어야 합니다. 마스터-슬레이브 구성에서는 Wiegand 또는 RS-485를 통해 연결할 수 없습니다.

  • SIO2 보안 모듈 구성에 대한 자세한 내용은 다음 문서를 참고하세요.

모바일 기기에서 출입문 관리​

관리자는 BioStar Air 모바일 앱에서 직접 출입문을 관리할 수 있습니다.

  1. 모바일 기기에서 BioStar Air 앱을 실행하고 로그인하세요.

  2. 초기 화면에서 하단의 Management를 탭하세요.

  3. Door 카드를 탭하세요.

  4. 등록된 모든 출입문을 확인하세요.

출입문 추가​

새 출입문을 추가하려면 오른쪽 상단의 버튼을 탭하고 필요한 세부정보를 입력하세요.

출입문 수정​

출입문 목록에서 출입문을 탭하여 세부 정보를 수정하거나 다음 설정을 구성할 수 있습니다.

  • 입구 및 출구 장치 할당

  • 릴레이 설정

  • 퇴출 버튼 및 출입문 센서 구성

출입문 리포트 생성​

출입문 리포트를 생성하여 관리자의 이메일로 전송할 수 있습니다. 출입문 리포트는 출입문을 통해 출입한 사용자에 대한 정보를 제공합니다.

  1. 화면 왼쪽 사이드바에서 Door Management → Registered Doors을 클릭하세요.

  2. 출입문 목록 상단의 버튼을 클릭하세요.

  3. Report Settings 창이 나타나면 리포트 이름을 입력하세요.

  4. 리포트에 포함할 항목을 선택하세요.

  5. 생성한 리포트를 저장하려면 Add Report 버튼을 클릭하세요.

  6. Report List이 생성되며 관리자의 이메일로 다운로드 링크가 전송됩니다.

情報

Report List에서 생성된 리포트를 검색하거나, 수정, 다운로드, 삭제할 수 있습니다.
