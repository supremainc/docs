Device Information and Configuration Options

Administrators can select a registered device to view its basic information and configure operational settings such as network, authentication, and time and attendance management. Configuration options vary by device model. Expand the device entry to view the available options.

AirFob Edge​

詳細​

The 詳細 section provides basic information — such as the device name, model, serial number, and firmware version — and includes an action menu for controlling features and default settings.

Basic Information​

View the device's basic information. This information cannot be edited.

  • 製品: The product name.

  • モデル名: The device's model name.

  • シリアルナンバー: The device's serial number.

  • ファームウェアバージョン: The firmware version installed on the device.

Actions​

The Action (액션) feature lets you control the device's functions.

  • ファームウェアアップデート: Update the device firmware to the latest version.

  • 再起動: Reboot the device remotely when an issue or anomaly occurs.

  • デバイスのリセット: Initialize all settings stored on the device. This action deletes all settings and data on the device.

  • デバイスのリセット ネットワーク設定を除く: Initialize all settings except the network settings. This action deletes all settings and data but keeps the network settings.

Initial Settings​

Modify the device's default settings.

  • ニックネーム: The alias shown in the device list and logs. You can edit this field directly.

  • タイムゾーン: Set the device time zone to keep logs consistent when the site and device clocks differ. Activate the 夏時間を使用 option to apply daylight saving time (DST).

  • ハンズフリ: Extend the BLE range.

  • QRリーダー: Confirm that an RS-485 QR reader is connected to the device.

    • When a QR reader is connected to the device, you cannot use another RS-485 device.

    • When using an SIO2, you cannot use a QR reader on the device connected to that door.

  • NFC/RFカード: Activate or deactivate card authentication. DESFire and Felica card formats are supported.
  • ビープ音の設定: Configure the success beep for access authentication.

ネットワーク​

Modify the device's network settings.

  • DHCP使用: Configure the device to obtain an IP address automatically.

    To set the network manually, disable DHCP使用 and fill in the following fields.

  • IPアドレス
  • ゲートウェイ
  • DNSアドレス
  • サブネットマスク

T&A (Time & Attendance, T&A)​

When this feature is enabled, 勤怠イベント is logged in Event Log and used in T&Aレポート.

  • T&Aを使用: Click the toggle switch to turn attendance management on or off.

  • 勤怠管理モード: Select the desired attendance mode.

    • スケジュール: Attendance events change automatically according to the set schedule. You can select a schedule to apply to attendance events.

    • 固定: Only the selected attendance event is available.

    • 携帯: When presenting a mobile credential, the Suprema Pass app lets the user select the attendance event.

情報

For more information on configuring time and attendance, see Time & Attendance (T&A).

ウィーガンド出力​

This feature integrates with external systems (e.g., alarm panels). Activate the ウィーガンド出力を使用する option to use this feature.

  • 出力モード: Configure the Wiegand signal output mode.

    • 通常: Scan cards using the configured Wiegand format.

      • フェイルコードを使用する: You can set error codes and choose the value to send when Wiegand card authentication fails.

    • バイパス: Send the CSN regardless of Wiegand authentication. Set this if you are using it as a device without door control functionality.

  • 出力情報: Select the information output during authentication.

  • パルス幅: Set the pulse width for the Wiegand signal.

  • パルス間隔: Set the pulse interval for the Wiegand signal.

  • モバイル連続タグ: Tag mobile cards consecutively without wait time.

  • 連続タグの制限: Set the maximum number of consecutive mobile card tags.

情報

The モバイル連続タグ feature is supported in Suprema Pass app 2.9.0 or later.

感度​

Adjust the RSSI value to set how close the mobile device must be to unlock the door.

  • Lower values help block unintended activation.

  • Higher values improve detection in high-interference environments.

情報

To reset the settings, click リセット.

モジュール​

Option to detect an OM-120 module connected to the device. Click 追加 to detect the module.

カード​

In the カード section, you can modify card authentication settings.

NFC/RFカード​

  • NFC/RFカード: Click the toggle switch to enable or disable card access authentication. Select the card types supported for card authentication.

    • EM4100 (125 KHz)
    • Mifare Classic/Mifare DESFire/FeliCa
  • バイトオーダー

    • MSB: Processes card data in order from large byte units to small byte units, sequentially storing the card serial number (CSN) from left to right.

    • LSB: Processes card data in order from small byte units to large byte units, storing the card serial number (CSN).

  • データ読み取り方法: Select CSN (default) or Wiegand format.

    Wiegand supports 5 default formats. Add more formats in 設定 → カードフォーマット. For more information about custom card format settings, see Custom Wiegand card format.

専用カード​

For Wiegand cards, you can select iClass (SE / SEOS) and HID Prox. Click the toggle to enable the feature, then select the desired card format in 専用カード.

情報

  • Support may vary by device.

  • Wiegand supports 5 default formats. Add more formats in 設定 → カードフォーマット. For more information about custom card format settings, see Custom Wiegand card format.

ネットワーク​

Modify the device's network settings.

  • DHCP使用: Configure the device to obtain an IP address automatically.

    To set the network manually, disable DHCP使用 and fill in the following fields.

  • IPアドレス
  • ゲートウェイ
  • DNSアドレス
  • サブネットマスク

認証​

  • 認証モード: Add authentication methods. Click 追加 to add an authentication method.

    • XP2-MDPB / XP2-GDPB: Only RF cards can be selected as the authentication method.

    • XP2-GKDPB: You can use a combination of an RF card and a PIN as the authentication method.

  • 認証タイムアウト: Authentication fails if the credential is not authenticated within the set time.

    This feature is supported only on the XP2-GKDPB model.

情報

You can configure different authentication methods for the selected schedule. For example, you can allow card-only authentication during work hours and allow card-and-PIN authentication outside work hours.

顔​

安全​

  • 偽造検出: Prevent user authentication with spoofed faces such as photos.

  • 重複チェック: Check for duplicate registrations when enrolling facea.

検出​

  • モーション センサーの感度: Set the sensitivity for detecting movement around the device.

  • 最大ヘッド ポーズ角度: Set the maximum rotation angle allowed for a face.

  • 広範囲検索: Expand the area in the camera view where faces are recognized.

  • 検出距離 (cm): Set the minimum and maximum recognition distance.

  • 登録時間 秒: If a face is not enrolled within the set time during enrollment, the process is canceled.

マスク検出​

Activate the マスク検出 option to configure mask detection.

  • マスク検出レベル: Select mask detection sensitivity.

  • マスク検出に失敗した場合

    • アクセスを拒否: Users not wearing a mask are denied authentication, and a mask-non-compliance event log is recorded.

    • ログイン後にアクセスを許可: Users not wearing a mask can still authenticate, but a mask-non-compliance event log is recorded.

  • マスク チェック

    • アクセス後: After authentication, check mask usage or measure temperature.

    • アクセス前: Check mask usage or measure temperature before authentication. In this case, users not wearing a mask or exceeding the threshold temperature cannot authenticate.

    • アクセスなし: Use the device only to check mask usage or measure temperature. Users wearing a mask or having a temperature below the threshold can enter regardless of authentication.

  • デバイスのカメラでQRコードまたはバーコードを認証: Set whether to allow QR/barcode authentication via the device's camera.

  • QRコードでカード番号を認証: Set authentication to accept a QR code containing the same data as the CSN card or Wiegand card issued to the user. This feature is available when the デバイスのカメラでQRコードまたはバーコードを認証 option is enabled.

情報

When using non-face credentials, the timing of "before" and "after" is usually important.

  • マスク検出レベル: Select mask detection sensitivity.

  • マスク検出に失敗した場合

    • アクセスを拒否: Users not wearing a mask are denied authentication, and a mask-non-compliance event log is recorded.

    • ログイン後にアクセスを許可: Users not wearing a mask can still authenticate, but a mask-non-compliance event log is recorded.

  • マスク チェック

    • アクセス後: After authentication, check mask usage or measure temperature.

    • アクセス前: Check mask usage or measure temperature before authentication. In this case, users not wearing a mask or exceeding the threshold temperature cannot authenticate.

    • アクセスなし: Use the device only to check mask usage or measure temperature. Users wearing a mask or having a temperature below the threshold can enter regardless of authentication.

  • デバイスのカメラでQRコードまたはバーコードを認証: Set whether to allow QR/barcode authentication via the device's camera.

  • QRコードでカード番号を認証: Set authentication to accept a QR code containing the same data as the CSN card or Wiegand card issued to the user. This feature is available when the デバイスのカメラでQRコードまたはバーコードを認証 option is enabled.

情報

When using non-face credentials, the timing of "before" and "after" is usually important.

T&A (Time & Attendance, T&A)​

When this feature is enabled, 勤怠イベント is logged in Event Log and used in T&Aレポート.

  • T&Aを使用: Click the toggle switch to turn attendance management on or off.

  • 勤怠管理モード: Select the desired attendance mode.

    • 手動: When the attendance key appears on the screen, the user selects the attendance event during authentication.
    • スケジュール: Attendance events change automatically according to the set schedule. You can select a schedule to apply to attendance events.
    • 手動固定: The user continues to use the last selected attendance event.

    • 固定: Only the selected attendance event is available.

    • 携帯: When presenting a mobile credential, the Suprema Pass app lets the user select the attendance event.

情報

For more information on configuring time and attendance, see Time & Attendance (T&A).

ディスプレイとサウンド​

ディスプレイ​

  • スクリーンセーバーを使用: Reduce LCD brightness when the device is not in use to lower unnecessary power consumption.

  • 常時点灯バックライト: Keep the backlight always on or dim it after inactivity.

  • バックライトタイムアウト: Set the time before the backlight turns off automatically.

  • 言語: Set the language shown on the device screen.

  • ホーム画面: Configure the items displayed on the device’s home screen background.

    • 通常表示: Display the default image on the home screen.

    • ロゴ: Display a user-registered image on the home screen.

    • お知らせ: Display administrator input on the home screen.

    After clicking アップデート and saving, the image or notice no longer appears in the edit screen but remains on the device.

サウンド​

  • 音声指示を使用: Use voice guidance instead of notification tones.

  • デバイス音量: Adjust the volume of sounds generated by the device.

  • インターホンスピーカー音量 / インターホンマイク音量: Adjust the output or input volume when using the IP intercom feature.

ウィーガンド出力​

This feature integrates with external systems (e.g., alarm panels). Activate the ウィーガンド出力を使用する option to use this feature.

  • 出力モード: Configure the Wiegand signal output mode.

    • 通常: Scan cards using the configured Wiegand format.

      • フェイルコードを使用する: You can set error codes and choose the value to send when Wiegand card authentication fails.

    • バイパス: Send the CSN regardless of Wiegand authentication. Set this if you are using it as a device without door control functionality.

  • 出力情報: Select the information output during authentication.

  • パルス幅: Set the pulse width for the Wiegand signal.

  • パルス間隔: Set the pulse interval for the Wiegand signal.

  • モバイル連続タグ: Tag mobile cards consecutively without wait time.

  • 連続タグの制限: Set the maximum number of consecutive mobile card tags.

情報

The モバイル連続タグ feature is supported in Suprema Pass app 2.9.0 or later.

インターコム​

  • インターコムを使用する: Activate or deactivate the IP intercom feature.

  • SIPサーバートランスポート: Choose the SIP transport method when configuring the intercom’s SIP server.

  • SIPサーバーアドレス: Enter the SIP server address.

  • SIPサーバーポート: Enter the SIP server port.

  • 登録期間: Enter the registration duration. The device attempts to register with the SIP server every time the registration duration expires.

  • SIPユーザー名: Enter the SIP account username.

  • パスワード: Enter the SIP account password.

  • 認証ID: Enter the SIP account authorization ID.

  • 発信プロキシサーバー: Use a separate outbound proxy server for SIP services if required.

  • 内線番号: Add or edit extension numbers.

  • ビデオ解像度: Select the video resolution when using the intercom.

  • ドア開閉ボタン: Specify a button to unlock the door during a call.

セキュアタンパー​

If a tamper event occurs, configure the device to delete all user information, all logs, and security keys stored on the device.

RTSP​

Configure details for Real Time Streaming Protocol (RTSP) streaming. The device's camera can stream live video.

  • RTSPを使用: Turn RTSP streaming on or off.

  • アドレス: RTSP server address.

  • ポート: Enter the RTSP server port.

  • ID / パスワード: Enter the RTSP server ID and password.

  • RTSPビデオ解像度: Select the video resolution when using RTSP.

情報

The RTSP streaming feature is supported only on BioStation 3.

感度​

Adjust the RSSI value to set how close the mobile device must be to unlock the door.

  • Lower values help block unintended activation.

  • Higher values improve detection in high-interference environments.

情報

To reset the settings, click リセット.
