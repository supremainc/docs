파트너 포털에 기술자 초대하기

Not translated
This page is currently being translated. We will complete the translation as soon as possible.

기술자 계정을 통해 공유 관리자 계정 없이 파트너와 설치 업체에 제한된 사이트 접근 권한을 부여할 수 있습니다. 기술자는 파트너 포털에서 중앙 집중식으로 관리되며, 각 사이트별로 개별 할당할 수 있습니다.

기술자 계정 사용의 주요 이점

  • 보안 강화: 공유 관리자 계정 제거로 부정 접근 위험 감소

  • 규정 준수: 개인정보 보호 규정(GDPR 등) 준수 가능, 사용자 데이터 접근 제외 옵션 제공

  • 접근 제어: 기기 접근부터 완전한 관리자 권한까지 세분화된 권한 설정 가능

  • 감사 추적성: 감사 로그에 개별 기술자 활동 기록으로 책임 추적 가능

  • 효율성 증대: 로그인 충돌 없이 여러 사이트에서 기술자 배정 및 관리 가능

  • 빠른 해제: 기술자 접근 권한을 언제든 즉시 취소 가능

기술자 초대​

사이트에 대한 접근 권한을 부여하기 전에 파트너는 기술자를 초대하여 기술자 계정을 생성해야 합니다. 기술자 계정은 이메일 주소를 기반으로 생성되며, 초대된 기술자는 이메일을 통해 초대를 수락하여 계정을 활성화할 수 있습니다.

  1. 파트너 포털에 로그인하세요.

  2. 화면 왼쪽 사이드바에서 Admins & Technicians를 클릭하세요.

  3. Technicians 탭을 클릭하세요.

  4. 화면 오른쪽 상단의 Invite 버튼을 클릭하세요.

  5. Invite Technicians 창이 나타나면 초대할 기술자의 이메일 주소를 입력하세요.

  6. Add 버튼이 활성화되면 클릭하세요.

  7. 추가된 기술자는 메일 주소 입력란 하단에 표시됩니다. 기술자를 추가로 초대하려면 5~6단계를 반복하세요.

  8. 기술자를 모두 추가했다면 OK 버튼을 클릭하세요.

  9. 전송 확인 메시지가 나타나면 Confirm 버튼을 클릭하세요.

情報

  • 파트너 포털에 기술자 계정을 초대했더라도 각 사이트에 대한 초대는 별도로 진행해야 합니다. 사이트에 기술자를 초대하는 방법에 대한 자세한 내용은 사이트에 기술자 배정하기를 참고하세요.

  • 초대 이메일을 통해 가입한 기술자는 파트너 포털에 로그인할 수 있지만, 모든 메뉴에 대한 접근을 제한합니다. 기술자 본인 계정의 비밀번호 변경 및 계정 삭제, 언어 설정만 사용할 수 있습니다.

초대 상태​

파트너는 기술자를 초대한 후 기술자 목록에서 기술자의 초대 상태를 확인할 수 있습니다.

가입 대기 상태​

초대 받은 기술자가 이메일을 수락하기 전까지는 초대 상태가 Pending로 표시됩니다. 초대 받은 기술자는 최대 7일까지 초대를 수락할 수 있으며, 7일이 지나면 초대가 만료되어 Expired 상태로 변경됩니다.

가입 완료 상태​

기술자가 이메일 초대를 수락하고 가입을 완료하면 초대 상태가 Accepted로 변경됩니다.

가입 취소 상태​

초대된 기술자의 이메일 주소를 잘못 입력했거나 문제가 해결되어 더 이상 기술자 초대가 필요하지 않을 때, 초대를 취소하면 초대 상태가 Revoked 상태로 변경됩니다. 기술자 초대 취소에 대한 자세한 내용은 #revokeInvite를 참고하세요.

초대 취소​

파트너가 초대한 기술자의 이메일 주소를 잘못 입력했거나 문제가 해결되어 더 이상 기술자 초대가 필요하지 않을 때, 초대를 취소할 수 있습니다.

  1. 기술자 목록에서 초대 상태가 Pending 상태인 기술자의 체크박스를 클릭하세요. 1명 이상의 기술자를 선택할 수 있습니다.

  2. 기술자 목록 왼쪽 상단의 버튼을 클릭하세요.

    Pending 상태인 기술자만 초대 취소할 수 있습니다.

  3. 팝업 메시지 창이 나타나면 내용을 확인하고 Yes 버튼을 클릭하세요.

  4. 초대 취소 성공 메시지가 나타나면 Close 버튼을 클릭하세요.

기술자 목록에서 초대 취소한 기술자의 초대 상태가 Revoked 상태로 변경됩니다.

초대 링크 재전송​

초대된 기술자가 초대 이메일을 받지 못했거나 초대 이메일을 분실했을 때, 또는 초대 링크가 만료되었을 때 초대 링크를 재전송할 수 있습니다.

  1. 기술자 목록에서 초대 상태가 Pending 또는 Expired 상태인 기술자의 체크박스를 클릭하세요. 1명 이상의 기술자를 선택할 수 있습니다.

  2. 기술자 목록 왼쪽 상단의 버튼을 클릭하세요.

    Revoked 또는 Expired 상태인 기술자만 초대 링크를 재전송할 수 있습니다.

  3. 팝업 메시지 창이 나타나면 내용을 확인하고 Yes 버튼을 클릭하세요.

  4. 초대 링크 재전송 성공 메시지가 나타나면 Close 버튼을 클릭하세요.

기술자 목록에서 초대 링크를 재전송한 기술자의 초대 상태가 Pending 상태로 변경됩니다.

기술자 관리​

초대된 기술자가 이메일 초대를 수락하여 가입을 완료하면, 기술자 목록에서 해당 기술자를 관리할 수 있습니다. 기술자의 계정을 잠금 또는 해제하거나 사이트에 대한 접근 권한을 수정할 수 있습니다. 더이상 기술자의 지원이 필요하지 않을 때는 기술자 계정을 삭제할 수도 있습니다.

기술자 계정 잠금/해제​

이메일 초대를 수락하여 가입 완료한 기술자는 Lock Account 옵션을 통해 계정을 잠그거나 해제할 수 있습니다. 계정을 잠그면 기술자는 파트너 포털에 로그인할 수 없으며, 계정 정보는 유지되어 나중에 필요할 때 계정을 다시 활성화할 수 있습니다.

잠금 상태의 기술자는 Lock Account 열이 Locked으로 표시되고, 해제된 기술자는 Unlocked로 표시됩니다.

情報

  • 가입 완료한 기술자의 계정은 기본적으로 잠금 해제된 상태입니다.

  • 관리자이면서 기술자로 초대된 계정은 잠그거나 해제할 수 없습니다.

개별 기술자 계정 잠금/해제​

기술자 목록에서 Lock Account 열의 토글 버튼을 클릭해 개별 기술자 계정을 잠그거나 해제할 수 있습니다.

  1. 기술자 목록에서 Lock Account 열의 토글 버튼을 클릭하세요.

  2. 팝업 메시지 창이 나타나면 내용을 확인하고 Yes 버튼을 클릭하세요.

  3. 기술자 계정 잠금 또는 해제 성공 메시지가 나타나면 Close 버튼을 클릭하세요.

일괄 기술자 계정 잠금/해제​

기술자 목록에서 여러 기술자 계정을 선택해 일괄적으로 잠그거나 해제할 수 있습니다.

  1. 기술자 목록에서 잠그거나 해제할 기술자의 체크박스를 클릭하세요. 1명 이상의 기술자를 선택할 수 있습니다.

  2. 화면 왼쪽 상단의 또는 버튼을 클릭하세요.

  3. 팝업 메시지 창이 나타나면 내용을 확인하고 Yes 버튼을 클릭하세요.

  4. 기술자 계정 잠금 또는 해제 성공 메시지가 나타나면 Close 버튼을 클릭하세요.

사이트 접근 권한 수정​

사이트에 접근 권한을 부여 받은 기술자의 사이트 접근 권한을 수정할 수 있습니다.

  1. 기술자 목록의 Invited Sites 열에서 1개 이상인 기술자를 클릭하세요.

  2. Edit Technician 창이 나타나면 Invited Sites에서 기술자가 초대된 사이트 목록을 확인하세요.

  3. Lock Account 열의 토글 버튼을 클릭해 사이트 접근을 허용하거나 차단하세요.

  4. 수정 사항을 저장하려면 Save 버튼을 클릭하세요.

  5. 사이트 접근 권한 수정 성공 메시지가 나타나면 Close 버튼을 클릭하세요.

情報

기술자에게 사이트 접근 권한을 부여하는 방법에 대한 자세한 내용은 사이트에 기술자 배정하기를 참고하세요.

기술자 계정 삭제​

더이상 기술자의 지원이 필요하지 않을 때, 기술자 계정을 삭제할 수 있습니다. 기술자 계정을 삭제하면 해당 기술자는 파트너 포털에 로그인할 수 없으며, 기술자 계정과 관련된 모든 정보가 시스템에서 완전히 제거됩니다.

  1. 기술자 목록에서 삭제할 기술자의 체크박스를 클릭하세요. 1명 이상의 기술자를 선택할 수 있습니다.

  2. 화면 왼쪽 상단의 버튼을 클릭하세요.

  3. 팝업 메시지 창이 나타나면 내용을 확인하고 Confirm 버튼을 클릭하세요.

  4. 기술자 계정 삭제 성공 메시지가 나타나면 Close 버튼을 클릭하세요.

삭제한 기술자 계정은 기술자 목록에서 더이상 표시되지 않습니다.

사이트별 기술자 관리​

사이트에 배정된 기술자의 계정을 잠그거나 해제, 삭제할 수 있습니다.

  1. 화면 왼쪽 사이드바에서 Site List을 클릭하세요.

  2. 사이트 목록에서 기술자의 접근을 허용할 사이트를 클릭하세요.

  3. Edit Site 패널이 나타나면 Admin & Technicians → Technicians List 탭을 차례로 클릭하세요.

  4. Technicians List에서 아래의 기능을 수행하세요.

    • 기술자 계정을 잠그거나 해제하려면 Lock Account 열에서 토글 버튼을 클릭하세요.

    • 기술자 계정을 삭제하려면 → Delete를 클릭하세요.

  5. 수정한 내용을 적용하려면 하단의 Apply 버튼을 클릭하세요.

情報

최초 등록된 기술자 계정은 사이트에 대한 접근 권한이 Locked 상태입니다. Locked 상태를 해제하려면 BioStar Air 포털에서 해당 사이트의 관리자 권한을 가진 계정으로 해제할 수 있습니다. 사이트에 기술자를 배정하고, 계정 잠금 해제에 대한 자세한 내용은 사이트에 기술자 배정하기를 참고하세요.
