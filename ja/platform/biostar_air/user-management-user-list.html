ユーザー管理

ユーザー一覧は、組織の出入管理を担当する管理者にとって必須のツールです。 ユーザーのグループ化による組織管理、リアルタイムの有効化／一時停止による入社や退職の社員への即時対応、さまざまなクレデンシャルの統合管理、有効期限の監視によるセキュリティ強化、フィルタリングやレポート作成による報告・監査対応まで、あらゆるユーザー管理業務を行えます。

ユーザー一覧の表示​

ユーザー一覧では、BioStar Air に登録されたすべてのユーザーを一目で確認できます。 各ユーザーの詳細を確認し、フィルターや並べ替え機能を活用して特定のユーザーを簡単に見つけることができます。 また、ユーザー一覧から直接ユーザーの編集、削除、グループ割り当てなどの操作を行うことができます。

  • Web
  • Mobile

画面左のサイドバーで ユーザー管理 → ユーザー をクリックします。

BioStar Airアプリでユーザー管理機能を利用できます。

  1. モバイル機器で BioStar Air アプリを起動してください。

  2. 画面下部のナビゲーションバーで 管理 をタップします。

  3. ユーザー をタップします。

ユーザーリストが表示されます。 新しいユーザーを追加したり、既存ユーザーの詳細情報を編集できます。

ユーザー情報​

ユーザー一覧のテーブルで提供される情報は以下のとおりです。

  • ユーザータイプ：ユーザーが一般ユーザーか一時ユーザーかを示します。 一般および一時ユーザーの詳細は ユーザーのカードとクレデンシャルの種類 を参照してください。

  • 名前：ユーザーの名前です。

  • ID：ユーザーの固有 ID です。

  • 認証情報の種類：ユーザーに付与されたクレデンシャルを示します。 クレデンシャルの種類の詳細は ユーザーのカードとクレデンシャルの種類 を参照してください。

  • 有効期間：ユーザーに付与されたクレデンシャルの有効期限です。

  • ユーザーステータス：該当ユーザーの有効化または無効化の状態を示します。

  • ユーザーグループ：ユーザーが所属するグループです。 ユーザーグループの詳細は ユーザーグループを管理する を参照してください。

  • アクセスレベル：ユーザーに割り当てた出入レベルです。 出入レベルの詳細は 出入レベルの管理性行する を参照してください。

  • 床面高さ：ユーザーが出入できるフロアに関する権限です。 フロアレベルの詳細は フロアレベルを管理する を参照してください。

フィルタリング​

  • Web
  • Mobile

ユーザー一覧の上部ヘッダー領域でフィルター機能を利用できます。 ヘッダー領域の ボタンをクリックすると、条件を設定できます。

情報

適用したフィルター条件を初期化するには、ユーザー一覧上部の ボタンをクリックしてください。

モバイルアプリでは、ユーザー一覧を任意の基準でフィルタリングして表示できます。 ユーザー一覧の右上にある フィルター をタップします。 ユーザーは以下の基準でフィルタリングできます。

  • ユーザータイプ：通常または一時ユーザーでフィルタリングします。

  • 認証情報の種類：クレデンシャルの種類でフィルタリングします。

  • ユーザーステータス：有効状態でフィルタリングします。

  • ユーザーグループ：ユーザーグループでフィルタリングします。

追加機能​

ユーザー一覧上部で提供される機能は以下のとおりです。

  • ：選択したユーザーにグループを割り当てたり、除外・移動したりできます。 詳細は #manageUserGroups を参照してください。

  • ：選択したユーザーの出入レベルを変更できます。 アクセスレベル管理の詳細は 出入レベルの管理性行する を参照してください。

  • ：選択したユーザーが待ち時間なしで連続してモバイルカードをタップできるように設定できます。 Suprema Pass アプリは 2.9.0 以上でサポートされています。

  • ：選択したユーザーにユーザーテンプレートを適用できます。 既定のユーザー情報に加えて必要な情報を追加できます。

  • ：選択したユーザー情報を PDF または CSV ファイルにエクスポートできます。

  • ：ユーザー画像を一括アップロードできます。 アップロードする画像ファイルの名前はユーザーIDと一致している必要があります。

    例：601.jpg ファイルはユーザーID 601 に対応します。

    ファイル形式の要件

    • 対応形式: .jpg, .jpeg, .png

    • ファイルサイズ制限: 画像ごとに 1MB

  • ：選択したユーザーに顔登録リンクをメールまたはSMSで送信できます。 ユーザーがリンクをクリックしてウェブページで顔を登録すると、顔認証で出入できる権限が付与されます。

    メールまたは携帯電話番号が登録されていないユーザーにはリンクを送信できません。

  • ：ユーザー一覧に適用されているフィルターをリセットします。 この機能はフィルターが適用されている状態で利用できます。 フィルタリングの適用方法の詳細は、#userFilteringを参照してください。

  • ：ユーザーのレポートを設定できます。 リポートの作成が完了すると、管理者のメールに送信されます。 レポート設定の詳細については、#userReportを参照してください。

  • ：ユーザー一覧で個人情報保護のために氏名やIDなどをマスク（***）できます。 この機能は、個人情報保護が必要な環境で役立ちます。 トグルボタンで機能を有効にし、保存をクリックしてください。

ユーザー情報の編集​

  • Web
  • Mobile

ユーザーの詳細を編集するか、クレデンシャルを変更するには、次の手順に従ってください。

  1. 画面左のサイドバーで ユーザー管理 → ユーザー をクリックします。

  2. 一覧で編集するユーザーをクリックします。

  3. ユーザー詳細の右上にある ボタンをクリックします。

  4. ユーザーの編集 ウィンドウが表示されたら、必要な詳細を編集します。

  5. 編集が完了したら、右下の 保存 ボタンをクリックします。

モバイルアプリでユーザーの詳細情報を編集するには、次の手順に従ってください。

  1. ユーザー一覧で編集するユーザーをタップします。

  2. 右上の をタップします。

  3. ユーザーの編集 画面で必要な情報を編集します。

  4. 変更を保存するには 保存 をタップします。

    クレデンシャルを発行するには 認証情報を有効化 を使用します。

情報

ユーザー情報の詳細については、ユーザーの追加を参照してください。

ユーザーグループの管理​

ユーザーをユーザーグループに割り当てる、移動する、またはグループから除外する方法をご案内します。 ユーザーグループの作成と管理の詳細については、ユーザーグループを管理するを参照してください。

  • Web
  • Mobile

グループへのユーザー割り当て​

ユーザーグループを作成した後、ユーザー一覧からユーザーを追加できます。

  1. 画面左のサイドバーで ユーザー管理 → ユーザー をクリックします。

  2. ユーザー一覧で、先に作成したグループに割り当てるユーザーのチェックボックスをクリックします。

  3. ユーザー一覧の左上にある をクリックしてください。

  4. ユーザーグループ管理 ウィンドウが表示されたら、ユーザーを割り当てる親グループまたはサブグループを選択します。

  5. 選択を完了してグループ設定を保存するには、OKをクリックしてください。

ユーザー一覧の ユーザーグループ 列で、ユーザーが新しいグループに割り当てられていることを確認できます。

ユーザーグループ に戻ると、該当グループの ユーザーの合計 が更新されていることを確認できます。

グループの解除または移動​

ユーザーをグループから除外するか、別のグループに移動できます。

  1. 画面左のサイドバーで ユーザー管理 → ユーザー をクリックします。

  2. ユーザー一覧で、グループから解除または移動するユーザーのチェックボックスをクリックします。

    グループを移動する場合は、同じグループのユーザーを選択してから操作を進めます。

  3. ユーザー一覧の左上にある をクリックします。

  4. ユーザーグループ管理 ウィンドウが表示されたら、次のいずれかを行います。

    • グループを解除する場合は、選択したグループのチェックボックスをオフにします。

    • グループを移動する場合は、選択したグループのチェックボックスをオンにし、移動先のグループを選択します。

  5. グループ設定を保存するには、OKをクリックしてください。

ユーザー一覧の ユーザーグループ 列でユーザーのグループ変更を確認できます。

モバイルアプリでユーザーグループを管理できます。 ユーザー一覧の上部にある をタップします。 ユーザーグループの一覧を確認できます。

  • グループに所属するユーザーを表示するには、対象グループの右側にある をタップします。

  • 新しいグループを追加するには、右上の をタップしてください。

  • グループ名を編集するには、対象のグループを長押しし、下部の をタップしてください。

  • グループを削除するには、対象のグループを長押ししてから下部の をタップしてください。 1つ以上のグループを選択して削除できます。

情報

ユーザーグループ管理の詳細については、ユーザーグループを管理するを参照してください。

ユーザー状態管理​

ユーザーの出入権限を有効化または一時停止できます。 また、クレデンシャルを再発行することもできます。

個別ユーザー​

  • Web
  • Mobile

  1. 画面左のサイドバーで ユーザー管理 → ユーザー をクリックします。

  2. ユーザー一覧で状態を変更するユーザーをクリックします。

  3. ユーザー詳細の上部で希望する機能を選択します。

  • ：ユーザーの出入権限を有効にします。 新規ユーザーの初期有効化または一時停止されたユーザーの再有効化時に使用します。

  • ：クレデンシャルを再発行します。 この機能は、ユーザーのクレデンシャルが期限切れになった場合やクレデンシャルが使用できず新しく発行する場合に使用できます。

  • ：ユーザーの出入をブロックします。 この機能はユーザーが有効化されている状態で使用できます。

モバイルアプリのユーザー一覧で、ユーザーをタップします。 ユーザーの詳細 画面で をタップします。 ユーザーアクション メニューが表示されます。

  • 活性化 / 再活性化: ユーザーのアクセス権を有効にします。 新規ユーザーの初期有効化または一時停止されたユーザーの再有効化時に使用します。

  • 再発行：ユーザーのクレデンシャルを再発行します。

  • 一時停止：ユーザーを一時的に無効にします。 一時停止されたユーザーは入退室できません。

  • 削除：ユーザーをシステムから削除します。

情報

再活性化 オプションは一時停止されたユーザーにのみ表示されます。

一括操作​

  • Web
  • Mobile

  1. 画面左のサイドバーで ユーザー管理 → ユーザー をクリックします。

  2. ユーザー一覧で状態を変更するユーザーのチェックボックスをクリックします。 1 人以上のユーザーを選択できます。

  3. 画面左上の ボタンをクリックし、ポップアップメニューで希望する操作を選択します。

  • 活性化 / 再活性化: ユーザーのアクセス権を有効にします。 新規ユーザーの初期有効化または一時停止されたユーザーの再有効化時に使用します。

  • 再発行：クレデンシャルを再発行します。 この機能は、ユーザーのクレデンシャルが期限切れになった場合やクレデンシャルが使用できず新しく発行する場合に使用できます。

  • 一時停止：ユーザーの出入をブロックします。 この機能はユーザーが有効化されている状態で使用できます。

モバイルアプリのユーザー一覧で、状態を変更するユーザーを長押しします。 追加で状態を変更するユーザーを選択します。 画面下部のユーザーアクションメニューで、希望する機能をタップします。

  • 活性化 / 再活性化: ユーザーのアクセス権を有効にします。 新規ユーザーの初期有効化または一時停止されたユーザーの再有効化時に使用します。

  • 再発行：ユーザーのクレデンシャルを再発行します。

  • 一時停止：ユーザーを一時的に無効にします。 一時停止されたユーザーは入退室できません。

  • 削除：ユーザーをシステムから削除します。

情報

再活性化 オプションは一時停止されたユーザーにのみ表示されます。

ユーザー削除​

注意

削除されたユーザーは復元できません。 削除する前に必ず確認してください。

個別ユーザー​

  1. 画面左のサイドバーで ユーザー管理 → ユーザー をクリックします。

  2. 一覧で削除するユーザーをクリックします。

  3. ユーザー詳細で ボタンをクリックします。

  4. 警告メッセージが表示されたら、はい ボタンをクリックします。

一括削除​

  1. 画面左のサイドバーで ユーザー管理 → ユーザー をクリックします。

  2. 一覧で削除するユーザーのチェックボックスをクリックします。 1 人以上のユーザーを選択できます。

  3. 画面左上の → 削除 をクリックします。

  4. 警告メッセージが表示されたら、はい ボタンをクリックします。

ユーザーリポートの作成​

ユーザーリポートを生成して管理者のメールに送信できます。

  1. ユーザー一覧上部の ボタンをクリックします。

  2. レポート設定 ウィンドウが表示されたら、レポート名を入力します。

  3. 各フィルターオプションでレポートに含める項目を選択します。

  4. 作成したレポートを保存するには、レポートの追加 をクリックします。

  5. レポートリスト が作成され、管理者のメールにダウンロードリンクが送信されます。

情報

レポートリスト で作成されたリポートを検索、編集、ダウンロード、削除できます。

ユーザー検索​

ユーザー一覧右上の検索入力フィールドで特定のユーザーを素早く見つけることができます。 ユーザー名、ID、電話番号で検索できます。
