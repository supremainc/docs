등록 장치 관리하기

Not translated
This page is currently being translated. We will complete the translation as soon as possible.

등록된 장치의 연결 상태를 한눈에 확인하고, 생체 인증과 카드 설정을 변경하거나 펌웨어를 업데이트합니다. 소수의 장치는 개별로, 다수의 장치는 일괄로 관리할 수 있어 보안 시스템을 효율적으로 운영할 수 있습니다.

情報

  • 장치는 iOS 또는 안드로이드의 BioStar Air 모바일 앱에서만 등록할 수 있습니다. 웹 포털에서는 등록할 수 없습니다. 모바일 앱에서 장치 등록 방법에 대한 자세한 내용은 모바일 앱에서 장치 관리하기를 참고하세요.

  • 원격 접속, 생체 인증, QR 코드 출입, 펌웨어 업데이트와 같은 일부 기능을 사용하려면 장치는 인터넷 연결이 필요합니다.

ヒント

장치 등록 및 관리에 대한 동영상 가이드를 시청하려면 다음 영상을 참고하세요.

장치 목록 조회​

장치 목록에서는 BioStar Air에 등록된 모든 장치를 확인할 수 있습니다. 각 장치의 연결 상태, 모델명, 펌웨어 버전 등 주요 정보를 한눈에 볼 수 있습니다. 또한 개별 장치를 클릭하여 세부 정보를 확인하고 설정을 수정할 수 있습니다.

화면 왼쪽 사이드바에서 Device Management → Registered Devices를 클릭하세요.

장치 연결 상태​

장치 목록에서 상태 아이콘을 통해 각 장치의 연결 상태를 확인할 수 있습니다.

  • 녹색: 온라인 상태로, 해당 장치는 원격으로 제어할 수 있으며, QR 인증을 수락하고 업데이트할 수 있습니다.

  • 빨간색: 오프라인 상태의 장치입니다.

  • 주황색: 펌웨어 업데이트 진행 중인 상태입니다.

  • ⚪ 연결 없음: 네트워크 연결이 없는 장치입니다. (예, Airfob Patch)

  • 주의: 서버와 장치 간의 동기화가 필요한 상태입니다.

장치 정보​

장치 목록의 각 열은 아래 정보를 포함하고 있습니다.

  • Product: 슈프리마 데이터베이스로부터 가져온 장치 이름과 모델명입니다.

  • Nick Name: 쉽게 식별할 수 있도록 사용자가 수정할 수 있는 항목입니다. 장치 목록에서 장치를 클릭하면 표시되는 Edit Device 창에서 변경할 수 있습니다.

  • Device ID: 장치에 할당된 고유 식별자입니다.

  • Device Group: 장치가 속한 그룹입니다.

  • Battery: 배터리로 작동하는 장치에서 확인할 수 있습니다. 에어팝 태그(AirFob Tag) 장치만 지원합니다.

  • Firmware: 현재 설치된 펌웨어 버전입니다.

  • Timezone: 장치에 설정된 시간대입니다.

필터링​

장치 목록의 상단 헤더 영역에서 필터링 기능을 이용할 수 있습니다. 헤더 영역의 버튼을 클릭하면 원하는 조건을 설정할 수 있습니다.

情報

적용한 필터 조건을 초기화하려면 버튼을 클릭하고 Reset 버튼을 선택하세요.

개별 장치 정보 및 설정 수정​

  1. 화면 왼쪽 사이드바에서 Device Management → Registered Devices를 클릭하세요.

  2. 목록에서 수정할 장치를 클릭하세요.

  3. Edit Device 창이 나타납니다.

  4. 장치의 세부 정보 또는 설정을 수정하세요.

  5. 변경 사항을 저장하려면 하단의 Save 버튼을 클릭하세요.

각 섹션별 세부 정보 수정 및 설정 옵션에 대한 자세한 내용은 장치 정보 및 설정 옵션 안내를 참고하세요.

장치 일괄 작업​

장치 목록 상단에서 제공하는 기능을 통해 여러 장치를 한 번에 관리할 수 있습니다.

  • : 장치 목록 및 장치 상태를 업데이트합니다.

  • : 사이트에 등록된 장치를 삭제할 수 있습니다.

    警告

    온라인 상태의 장치만 영구적으로 삭제할 수 있입니다. 장치가 오프라인이라면, 다른 사이트에 추가하기 전에 BioStar Air 모바일 앱에서 먼저 삭제하세요. 모바일 앱에서 장치 삭제 방법에 대한 자세한 내용은 모바일 앱에서 장치 관리하기를 참고하세요.

  • : 장치 목록에서 선택한 장치의 펌웨어를 업데이트할 수 있습니다. 펌웨어 업데이트에 대한 자세한 내용은 펌웨어 업그레이드를 참고하세요.

  • : 장치 목록에서 선택한 장치에서 서버로 생체 인증 템플릿을 동기화할 수 있습니다.

  • : 장치 목록에서 선택한 장치를 그룹에 할당할 수 있습니다. 장치 그룹 관리에 대한 자세한 내용은 #device_group_in_device_list를 참고하세요.

  • : 장치 목록에서 선택한 장치의 RF 카드 바이트 순서를 MSB 또는 LSB로 전환할 수 있습니다.

    • MSB: 카드 데이터를 처리할 때 큰 바이트 단위에서 작은 바이트 단위 순서로 처리하여 카드 일련 번호(CSN)를 왼쪽에서 오른쪽으로 순차 저장합니다.

    • LSB: 카드 데이터를 처리할 때 작은 바이트 단위에서 큰 바이트 단위 순서로 처리하여 카드 일련 번호(CSN)를 저장합니다.

  • : 독립 실행 모드에서 원래 설정된 장치의 설정을 가져올 수 있습니다.

  • : 장치 목록에서 선택한 장치의 상태를 업데이트할 수 있습니다.

장치 그룹 관리​

장치 목록에서 개별 장치를 선택하여 장치 그룹에 추가하거나, 기존 그룹에서 제거할 수 있습니다.

  1. 화면 왼쪽 사이드바에서 Device Management → Registered Devices을 클릭하세요.

  2. 장치 그룹 목록에서 그룹에 추가하거나 제거하려는 장치의 체크박스를 클릭하세요.

  3. 장치 그룹 목록 상단의 버튼을 클릭하세요.

  4. Manage Device Groups 창이 나타나면 선택한 장치를 그룹에 추가하거나 제거하세요.

    Device Groups의 체크박스 상태는 아래와 같은 의미를 가집니다.

    • None: 선택한 장치를 그룹에서 제거합니다.

    • Some: 선택한 장치 중 일부가 그룹에 포함된 상태입니다.

    • All: 선택한 장치를 그룹에 추가합니다.

  5. 변경한 내용을 저장하려면 OK 버튼을 클릭하세요.

情報

  • Manage Device Groups 창에서 다른 그룹을 선택하면 선택한 장치는 해당 그룹으로 이동합니다.

  • 장치는 그룹에 중복으로 속할 수 있습니다. 하나의 장치를 여러 그룹에 할당할 수 있습니다.

  • 변경한 내용을 초기화하려면 Reset 버튼을 클릭하세요.
