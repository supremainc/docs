Roll Call genera rápidamente una lista de personal según los registros de entrada en un tiempo y lugar específicos. Cuando ocurre una emergencia, el sistema puede determinar si un usuario llegó al punto de lista de asistencia al autenticarse en el dispositivo de entrada o salida en una ubicación predefinida.

Puede utilizarse para la respuesta a emergencias, verificaciones de seguridad y para proporcionar información al equipo de rescate. Esto permite la gestión eficiente del estado del personal utilizando registros de entrada y salida.

:::info

* Antes de iniciar Roll Call, complete la setting de Roll Call en BioStar X. Para obtener más información sobre cómo configurar Roll Call, consulte <DocLink docId='platform/biostar_x/settings-advanced-ac-roll-call' />.

* Las configuraciones de lista de asistencia pueden utilizarse mediante opciones adicionales disponibles con una licencia **Avanzada** o superior. Para obtener más información sobre las políticas de licencia, consulte <DocLink docId='platform/biostar_x/licensing#features-add-ons' />.

:::
