Use la app móvil BioStar X para administrar el sistema de control de acceso BioStar X desde un dispositivo móvil. Realizar tareas como registro de usuarios, control remoto de puertas, monitorización de eventos en tiempo real y manejo de alertas.

![BioStar X Mobile Connection Architecture](/img/common/biostarx-mobile-connection-architecture.png)

## Funciones clave

### Gestión Usuarios

* Agregar, eliminar y editar información del usuario

* Administración de credenciales: PIN, Fingerprint, Face IR, Face, Card (CSN, Wiegand, Smart Card: SCC, AoC)

### Control de Puerta

* Bloquear, desbloquear, abrir y liberar puertas de forma remota

* Libere el anti-passback (APB) y restablezca las alertas de puerta

### Monitoreo en tiempo real

* Revise los registros de eventos de acceso en tiempo real para la administración continua

### Administrar alertas

* Maneje alertas de puerta, dispositivo y control de acceso avanzado

* Administre las alertas de forma eficiente con listas separadas para eventos nuevos y reconocidos

### Administre el control de acceso sobre la marcha

Use la app móvil BioStar X para realizar las siguientes tareas sobre la marcha:

* Administre usuarios y agregue credenciales

* Monitoreo de eventos en tiempo real

* Responda rápidamente a las alertas

* Notificaciones de puerta: solicitud de apertura, apertura forzada, puerta abierta demasiado tiempo

* Notificaciones de dispositivo: manipulación, reinicio, desconexión RS485

* Alertas de área: anti-passback, incendio y más

## Protección de la privacidad

Suprema no recopila, almacena ni procesa datos personales a través de la app móvil BioStar X. Todos los datos relacionados con el usuario, incluidas las imágenes faciales y las credenciales, se envían directamente al sistema interno del cliente y no se envían a los servidores de Suprema.
