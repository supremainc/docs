Cuando la lista de usuarios es larga y difícil de buscar, use la búsqueda y los filtros para encontrar usuarios rápidamente.

## Búsqueda de usuarios

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-user-list-search.png' alone />

</Column>
<Column className='padding--l40'>

Puede buscar por ID de usuario, nombre o dirección de correo electrónico.

1. En la pantalla de inicio de la app, toque <Cmd>USER</Cmd>.

   :::note

   También puedes acceder a ella desde el menú lateral. Toque <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>User</Cmd>.

   :::

2. Toque <IcSearch title='Search' width='20px' height='20px' /> en la parte superior de la lista de usuarios.

3. Ingrese un término de búsqueda en el campo de búsqueda. Puede buscar combinando el ID de usuario, el nombre y la dirección de correo electrónico.

4. Después de escribir el término de búsqueda, toca <IcSearch title='Search' width='20px' height='20px' />.

Los usuarios que coinciden con el término de búsqueda aparecen en los resultados. En los resultados de búsqueda, toque al usuario que desea abrir para ver los detalles del usuario.

:::info

* El historial de búsquedas recientes aparece debajo del campo de búsqueda. Toca una entrada reciente de búsqueda para buscar de nuevo con ese término.

* Para cerrar el campo de búsqueda, toca <IcClose title='Close' width='20px' height='20px' />.

:::

</Column>
</Columns>

## Filtrar la lista de usuarios

<Column className='width--23'>

</Column>
<Column className='padding--l40'>

Filtre la lista de usuarios por diversas condiciones para encontrar usuarios rápidamente.

1. En la pantalla de inicio de la app, toque <Cmd>USER</Cmd>.

   :::note

   También puedes acceder a ella desde el menú lateral. Toque <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>User</Cmd>.

   :::

2. Toque <IcAppFilter title='Filter' width='20px' height='20px' /> en la parte superior de la lista de usuarios.

3. Establezca las condiciones deseadas en <Cmd>User Filter</Cmd>.

   * <Cmd>Sort By</Cmd>: Seleccione el orden ascendente o descendente por ID de usuario, nombre o grupo.

   * <Cmd>User Group</Cmd>: Seleccione un grupo de usuarios específico para mostrar solo a los usuarios de ese grupo.

   * <Cmd>Status</Cmd>: Para mostrar solo a los usuarios activos, seleccione <Cmd>Active</Cmd>. Para mostrar solo a los usuarios inactivos, seleccione <Cmd>Inactive</Cmd>.

4. Para filtrar la lista de usuarios con las condiciones seleccionadas, toque <Cmd>Apply</Cmd>.

:::info

Para restablecer las condiciones del filtro, toca <IcAppLogFiltered title='Filtered' width='20px' height='20px' /> → <IcAppReset title='Reset' width='20px' height='20px' /> en la esquina superior derecha de la pantalla.

:::

</Column>
</Columns>
