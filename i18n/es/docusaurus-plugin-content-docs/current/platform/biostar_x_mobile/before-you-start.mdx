Antes de usar la app móvil BioStar X, revise los siguientes requisitos e instale la app.

## Entorno compatible

La app móvil BioStar X puede usarse bajo las siguientes condiciones.

* **Sistema compatible**: BioStar X

* **Requisitos del servidor**: Se requiere un servidor BioStar X con acceso a la API.

* **Usuarios**: Los privilegios compatibles incluyen todos los niveles de cuenta predefinidos, y más adelante se agregará compatibilidad con niveles de cuenta personalizados.

* **Dispositivo móvil**: iOS 13.0 o posterior, o Android 7.0 o posterior.

:::info

La app móvil BioStar X no puede operar de forma independiente. Conéctela a un servidor BioStar X que se ejecute en la red del cliente.

* Para obtener más información sobre los requisitos mínimos para instalar BioStar X, consulte <DocLink docId='platform/biostar_x/system-requirements' target='_blank' />.

* Para obtener más información sobre cómo instalar BioStar X, consulte <DocLink docId='platform/biostar_x/express-install' target='_blank' />.

:::

## Requisitos de licencia y red

Se requiere una licencia de BioStar X para usar la app móvil BioStar X. Compre una opción de licencia adicional para usarla. Para obtener más información sobre las políticas de licencias, consulte <DocLink docId='platform/biostar_x/licensing#features-add-ons' target='_blank' />.

| Entorno de acceso | Licencia requerida |
| --- | --- |
| Misma red | Licencia de la app móvil |
| Red externa (IP pública) | Licencia de la app móvil |
| Red externa (sin IP pública) | Licencia de la app móvil + licencia de acceso remoto |

* Se requiere una licencia de **app móvil** para integrar el servidor BioStar X con la app móvil.

* De forma predeterminada, use la app móvil conectándose al servidor BioStar X en la misma red.

* El acceso desde una red externa requiere una dirección IP pública. De lo contrario, compre una licencia adicional de **Acceso remoto**.

:::info

* Para obtener más información sobre cómo aplicar licencias en BioStar X, consulte <DocLink docId='platform/biostar_x/settings-license-biostar-x-license' target='_blank' />.

* Para obtener más información sobre la configuración del servidor BioStar X, consulte <DocLink docId='platform/biostar_x/settings-server-server' target='_blank' />.

* Para obtener más información sobre la función de Acceso remoto en BioStar X, consulte <DocLink docId='platform/biostar_x/settings-remote-access' target='_blank' />.

:::
