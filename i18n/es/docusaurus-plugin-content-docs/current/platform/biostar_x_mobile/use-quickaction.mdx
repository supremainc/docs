---

**Acciones rápidas** es una función que automatiza procedimientos de control complejos durante emergencias. Ejecuta al instante las acciones de varios dispositivos predefinidos con un solo botón para responder con rapidez.

:::info

* Para obtener más información sobre la función **Acciones rápidas**, consulta <DocLink docId='platform/biostar_x/settings-custom-interface' />.

* En la app móvil, puedes ejecutar la función Acciones rápidas o ver información detallada sobre las entradas de Acciones rápidas. Para crear o editar una entrada de Acciones rápidas, inicia sesión en la interfaz web de BioStar X y configúrala.

:::

## Ejecutar una Acción Rápida

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-quickaction-list.png' alone />

</Column>
<Column className='padding--l40'>

Debes crear Acciones rápidas en la interfaz web de BioStar X antes de usarlas en la app móvil.

1. Toca <Cmd>QUICK ACTION</Cmd> en la pantalla de inicio de la app.

2. En la lista de Acciones rápidas, toca el botón <Cmd>Activate</Cmd> de la entrada de Acción rápida que quieres ejecutar.

Cuando la Acción rápida se ejecuta correctamente, aparece un mensaje de éxito.

:::info

* Según la configuración de Acciones rápidas, puede aparecer una ventana emergente para confirmar la ejecución.

* Cada entrada de Acciones rápidas muestra la fecha y hora de activación más recientes.

:::

</Column>
</Columns>

## Detalles de Acción Rápida

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-quickaction-details.png' alone />

</Column>
<Column className='padding--l40'>

Para ver la información detallada configurada para una entrada de Acciones rápidas, toca la entrada en la lista de Acciones rápidas.

* <Cmd>Last Activation</Cmd>: La fecha y hora en que se ejecutó por última vez la Acción rápida.

* <Cmd>Confirm before run</Cmd>: Indica si aparece una ventana emergente para confirmar la ejecución de la Acción rápida.

* <Cmd>Activate</Cmd>: El botón que ejecuta la Acción rápida.

* <Cmd>Description and Operator Level</Cmd>: La descripción de la Acción rápida y el nivel de administrador requerido para ejecutarla.

* <Cmd>Trigger and Actions</Cmd>: Las puertas que controla la Acción rápida cuando se ejecuta y las acciones aplicadas a esas puertas.

:::info

Para obtener más información sobre las condiciones de activación y las acciones, consulta <DocLink docId='platform/biostar_x/settings-triger-and-action' />.

:::

</Column>
</Columns>
