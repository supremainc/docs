Gestione la configuración básica necesaria para operar BioStar Air. Configure el sistema para cumplir con los requisitos de la organización administrando las cuentas de administrador con privilegios operativos, estableciendo la información básica de la organización, configurando las zonas horarias y las alertas, personalizando las plantillas de correo electrónico y definiendo los formatos de tarjeta.
