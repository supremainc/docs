Esta guía explica cómo configurar el sitio de demostración de BioStar Air, conectar y registrar dispositivos compatibles y probar la integración de la API en la nube. Si has trabajado con BioStar 2 antes, muchos conceptos te resultarán familiares. BioStar Air está completamente basado en la nube.

## Requisitos previos {#pre-requisites}

Antes de comenzar, comprueba lo siguiente.

* Prepara un **dispositivo compatible con BioStar Air** con el firmware en estado de fábrica.

  * XP2-AIR, XS2-AIR, BEW3-AIR, BS3-AIR, BLN2-AIR

  * Los dispositivos pueden comprarse a través de distribuidores autorizados de Suprema.

* El dispositivo debe estar conectado a Internet mediante Ethernet.

* Permite el acceso saliente en la red.

  * Puerto 443 (HTTPS)

  * Puerto 5671 (MQTT sobre TLS)

## Configura el entorno de demostración y la clave de API {#demo-setup}

<Steps>
<Step>

### Regístrate e inicia sesión

* Ve a [BioStar Air Developer](https://developers.biostarair.com/login).

* Haz clic en **Regístrate**, completa el formulario y espera la aprobación.

* Después de la aprobación, inicia sesión con tus credenciales.

</Step>
<Step>

### Crea una aplicación de demostración

1. Ve a **API** → **Gestión**.

2. Haz clic en **+ Registrar**.

3. Selecciona **Demo** como tipo de aplicación.

4. Introduce un nombre para la aplicación y haz clic en **Registrar**.

5. Haz clic en **Descargar** para obtener el archivo Excel con las credenciales de acceso al sitio de demostración (correo electrónico y contraseña).

</Step>
<Step>

### Generar una clave API

1. En **Gestión de aplicaciones**, haz clic en la aplicación que registraste.

2. Desplázate hasta **Gestión de API**.

3. Haz clic en **Agregar**.

4. Introduce un nombre y confírmalo. Guarda la clave generada.

</Step>
<Step>

### Inicia sesión en el portal de demostración

1. Ve al [Portal de demostración](https://demo.airfobspace.com/login).

2. Inicia sesión con el correo electrónico y la contraseña del archivo Excel.

3. Introduce la clave API que acabas de generar.

</Step>
</Steps>

## Registrar dispositivos {#registerDevices}

Usa la aplicación BioStar Air Demo (solo Android) en tu dispositivo móvil.

1. Haz clic en el siguiente enlace para descargar la aplicación móvil. [DeviceRegistration.zip](https://moca-public-file-share.s3.ap-northeast-2.amazonaws.com/DeviceRegistration.zip)

2. Inicia sesión con las credenciales del archivo Excel.

3. Acerca tu dispositivo Android al lector (dentro del alcance BLE).

   * En **Menú principal** → **Dispositivos**, pulsa el botón <IcAdd /> en la esquina superior derecha.

   * Busca el dispositivo y pulsa **Registrar**.

4. El dispositivo emite un pitido, se reinicia y aparece en la lista de dispositivos registrados.

:::info

Los dispositivos solo se pueden registrar a través de la aplicación móvil. No es posible registrar dispositivos desde el portal web. El registro mediante API requiere un certificado de cifrado que no se proporciona a los socios.

:::

## Gestión Dispositivos

Gestiona los dispositivos desde la app Demo o desde el portal web de demostración.

* **BioStar Air Demo App**

* **Portal web de demostración**: [https://demo.airfobspace.com/login](https://demo.airfobspace.com/login)

## Demo vs producción {#demo-vs-production}

* Un sitio **Demo** para pruebas se crea a través del [Portal de desarrolladores](https://developers.biostarair.com/login).

* Un sitio de producción se crea a través del [Portal de socios](https://partner.biostarair.com/) y requiere un ID de sitio, correo electrónico de usuario y contraseña. Solo los revendedores autorizados o las sucursales de Suprema pueden crear un sitio de producción.

:::info

* Los ID de usuario y los ID de cuenta son diferentes. No los confundas.

* Si administra varios sitios, guarde las contraseñas de forma segura.

* Utilice siempre el token Bearer más reciente.

* Un token Bearer copiado del navegador puede reutilizarse en Postman.

:::
