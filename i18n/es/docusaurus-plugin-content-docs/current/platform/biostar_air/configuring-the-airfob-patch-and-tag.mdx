## Registra el Airfob Patch en la aplicación BioStar Air {#registerApp}

### Antes de comenzar

* Instale la app BioStar Air en su dispositivo móvil. Instale la app móvil mediante el siguiente enlace.

  * [iOS](https://apps.apple.com/us/app/airfob-pro/id1659959609)

  * [Android](https://play.google.com/store/apps/details?id=com.mocainc.afs2pro)

* Inicie sesión con el mismo ID y contraseña que usa para el portal de administración de BioStar Air.

* Asegúrese de que BLE esté habilitado en su dispositivo móvil.

* Fije el Airfob Patch al lector de forma permanente o temporal. Para obtener más información, consulte <DocLink docId='platform/biostar_air/install-airfob-patch' />.

:::info

Para obtener más información sobre el uso de la app móvil BioStar Air, consulte <DocLink docId='platform/biostar_air/managing-sites-with-app' />.

:::

### Registrar Airfob Patch

1. Abra la aplicación BioStar Air en su dispositivo móvil e inicie sesión.

2. En la pantalla principal de BioStar Air, toque <Cmd sid='site_setting_tab_management' product='air' /> en la parte inferior.

3. Acerca su dispositivo móvil a menos de 10 cm del Airfob Patch y luego toque el botón <IcAdd /> en la esquina superior derecha de la pantalla.

4. En la pantalla de búsqueda de dispositivos, toque <Cmd sid="button.register" product="2" /> en el Airfob Patch para enrolarlo.

5. Cuando aparezca la ventana de registro del dispositivo, toque <Cmd sid="device_btn_ok" product="air" />.

Vuelva a la lista de dispositivos y confirme el Airfob Patch agregado.

### Configura el método de salida del ID de tarjeta {#set-card-id-output-method}

* Desplázate hasta **Método de salida del ID de tarjeta** y toca **Editar**.

* Toca **Seleccionar** junto a **ID de tarjeta de usuario**.

</Step>
<Step>

### Selecciona el método de salida {#choose-the-output-method}

* **Ninguno**: Seleccione esta opción cuando no desee enviar un ID de tarjeta. Útil para el control de tiempo y asistencia, ya que prolonga la vida útil de la batería.

* **Dispositivo fijo**: Envía el ID de tarjeta fijo preprogramado. Útil para cerraduras o intercomunicadores que requieren programar cada tarjeta RFID de forma individual. Solo este ID fijo aparece en los registros del dispositivo.

* **ID de tarjeta de usuario**: Envía un ID de tarjeta único para cada usuario que coincide con el número de tarjeta RFID registrado en el sistema de control de acceso. Esto permite una integración sin necesidad de emitir nuevas tarjetas físicas.

* **Entrada directa**: Especifique manualmente el ID de la tarjeta que se enviará. Útil para configuraciones personalizadas en las que un mismo ID debe utilizarse en varias unidades.

</Step>
<Step>

### Guardar la configuración {#save-your-settings}

* Mantenga el dispositivo móvil cerca de la unidad, toque <Cmd sid="save_btn" product="air" /> y confirme las selecciones.

* Se muestra el mensaje 'Actualización completada'.

</Step>
</Steps>
