Los administradores pueden seleccionar un dispositivo registrado para ver su información básica y configurar los parámetros operativos, como la red, la autenticación y la gestión del tiempo y el control de asistencia. Las opciones de configuración varían según el modelo de dispositivo. Despliegue la entrada del dispositivo para ver las opciones disponibles.
