Este documento es una referencia para los ajustes de dispositivos y paneles en BioStar Air. Los administradores pueden ver información básica del hardware y ejecutar acciones como actualizar el firmware, reiniciar o restablecer. Pueden configurar ajustes de operación como red, autenticación, formato de tarjeta, registro de tiempo y asistencia, Wiegand, sensibilidad y módulos conectados. Los ajustes disponibles varían según el modelo de dispositivo y el tipo de panel.
