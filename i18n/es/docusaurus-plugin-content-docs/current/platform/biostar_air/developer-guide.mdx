BioStar Air ofrece una API RESTful en la nube que puedes utilizar para integrarla con sistemas externos, automatizar el ciclo de vida de los usuarios y gestionar las credenciales. Configura un entorno de demostración y comprende el flujo de autenticación; a continuación, implementa la integración con la API de gestión de usuarios.
