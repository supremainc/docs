El Portal de Socios de BioStar Air es una plataforma dedicada para revendedores de valor agregado (VAR) con acuerdos de distribución. Crea sitios de clientes, asigna créditos o configura sitios de federación para gestionar varias ubicaciones de forma centralizada. 

Instala hardware en las instalaciones, como intercomunicadores SIP, cámaras IP y escáneres de códigos QR, y configura la red.
