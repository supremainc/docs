## Plataforma de control de acceso 100% en la nube

BioStar Air es una solución nativa en la nube desarrollada por Suprema, líder mundial en seguridad biométrica. Diseñada para eliminar la necesidad de infraestructura local, BioStar Air ofrece un control de acceso seguro, flexible y escalable que se puede gestionar desde cualquier lugar. Con **lectores inteligentes** que se conectan directamente a la nube, las organizaciones pueden controlar fácilmente el acceso en múltiples sitios usando únicamente un teléfono o un navegador.

## Flexibilidad de credenciales sin igual

BioStar Air ofrece la gama más amplia de opciones de autenticación, lo que permite a las organizaciones gestionar el acceso a sus instalaciones de forma segura y práctica.

* **Autenticación facial**

* **Credenciales móviles personalizadas para iOS y Android** (gratuitas)

* **Múltiples formatos RFID**

* **QR dinámico y LinkPass sin aplicación**

## No se requiere infraestructura local

BioStar Air elimina la necesidad de servidores en las instalaciones, máquinas virtuales, VPN o controladores de puertas. No se requiere mantenimiento de TI. Instale un **lector inteligente** y conéctelo a la nube para comenzar a gestionar el acceso de inmediato. Los lectores funcionan en múltiples sitios sin importar la red o la ubicación física.

## Gestión multisede sencilla

BioStar Air ofrece una solución completa para controlar el acceso en cada ubicación—**oficinas**, **almacenes**, **tiendas** o **sitios remotos**—todo desde un único sistema.

## Escalabilidad preparada para el futuro con arquitectura edge

La **arquitectura edge** de BioStar Air integra el control de acceso directamente en el lector inteligente, eliminando la necesidad de controladores de puertas independientes. Esto permite a las organizaciones **escalar con facilidad**: comprar solo lo que necesitan hoy y expandirse fácilmente sin actualizaciones complejas.

## Optimizado para empresas centradas en la nube

BioStar Air está diseñado para **integradores de sistemas** y **clientes** que buscan una solución de control de acceso totalmente nativa en la nube con biometría líder en la industria. Ofrece un enfoque moderno, flexible y altamente seguro para la gestión de accesos en empresas centradas en la nube.

## Funciones de BioStar Air

| Función | Descripción |
| --- | --- |
| **Solución completa de control de acceso** | Soportado |
| **Compatible con el hardware de Suprema** | XPass 2, X-Station 2, BioEntry W3, BioStation 3, Airfob Patch, Airfob Tag, Output Module(OM-120), SIO2<br/>- Soporte planificado: XPass D2, CoreStation 20 |
| **Alta seguridad** | ISO 27001 / 27701, TLS 1.2, AES-256 |
| **Biometría: reconocimiento facial y huella dactilar** | Ofrece soporte nativo para los lectores de Suprema e incluye una base de datos biométrica integrada. |
| **Tarjetas RFID** | MIFARE Classic, DESFire, FeliCa, EM4100, formatos Wiegand personalizados, iCLASS, Seos. Admite hasta ocho tarjetas por usuario. |
| **Credenciales móviles** | Gratuitas e ilimitadas. Incluyen credenciales seguras BLE/NFC (iOS, Android), LinkPass sin aplicación y autenticación mediante QR. |
| **Despliegue** | Nativo en la nube |
| **Interfaz** | Navegador, aplicación iOS, aplicación Android |
| **Modelo de precios** | Por suscripción |
| **Integración** | APIs, webhooks y SDKs extensos |
| **Arquitectura Edge** | Los lectores incluyen un controlador integrado y se conectan directamente a la nube vía Ethernet. No se requieren nodos en la nube adicionales ni paneles de control de acceso. |
| **inicio de sesión** | Accesible desde cualquier lugar. |
| **Ubicación del servidor** | Corea (global) y Alemania (Europa) |
| **Control de tiempo y asistencia** | Los botones de control de tiempo y asistencia del lector se registran como eventos y pueden integrarse en los informes de horas trabajadas. |
| **Video** | VMS integrado con soporte para ONVIF y Uniview. No se requiere NVR; el lector actúa como puerta de enlace. |
| **Alarmas** | Permite notificar a los administradores sobre intrusiones en zonas de alarma. |
| **Partición de datos** | Estándar |
| **Objetivo** | Franquicias, multisitio, multi-tenant, ubicaciones remotas y más |
