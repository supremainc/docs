Esta es una guía de instalación y configuración de hardware para configurar correctamente el sistema BioStar Air. Estos documentos explican cómo conectar y configurar cada dispositivo, incluyendo interfaces SIP, cámaras IP, Los escáneres QR y Airfop Patch, así como cómo configurar los puertos de red y la configuración del módulo de seguridad.
