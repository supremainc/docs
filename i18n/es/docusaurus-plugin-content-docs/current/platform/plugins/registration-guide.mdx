Guía sobre cómo registrar y administrar conectores desarrollados en **BioStar X Service Manager**.

## Preparativos antes del registro

1. **Ejecutar el servidor de conector**

   El servidor de conectors debe estar en ejecución antes de registrar el conector. **BioStar X** verifica la accesibilidad del servidor de conector durante el proceso de registro.

2. **Verificar el acceso a la red**

   Asegura que el servidor de **BioStar X** pueda acceder al servidor de conector.

   ```bash
   # Test from BioStar X Server
   curl -H "X-BioStar-Ping: test123" http://localhost:8000/bsx
   ```
