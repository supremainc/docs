Guía sobre cómo registrar y administrar complementos desarrollados en **BioStar X Service Manager**.

## Preparativos antes del registro

1. **Ejecutar el servidor de complementos**

   El servidor de complementos debe estar en ejecución antes de registrar el complemento. **BioStar X** verifica la accesibilidad del servidor de complementos durante el proceso de registro.

2. **Verificar el acceso a la red**

   Asegura que el servidor de **BioStar X** pueda acceder al servidor de complementos.

   ```bash
   # Test from BioStar X Server
   curl -H "X-BioStar-Ping: test123" http://localhost:8000/bsx
   ```
