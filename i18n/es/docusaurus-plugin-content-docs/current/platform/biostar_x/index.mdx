## Bienvenido a la guía del usuario de BioStar X

**BioStar X** es la plataforma de seguridad integrada de próxima generación de Suprema que combina el control de acceso basado en IA con el análisis de video inteligente. Una plataforma innovadora que puede integrar y gestionar de manera eficaz las operaciones de seguridad generales, más allá del simple control de acceso. **BioStar X**, la culminación del control de acceso basado en IA y el análisis de video inteligente, está diseñada para utilizar la tecnología de IA para realizar la gestión de acceso en tiempo real, el monitoreo de video y la detección de comportamientos anómalos en una sola plataforma. Además del control de acceso, está equipada con varias funciones de análisis de video basadas en IA, como la detección de intrusiones, la detección de merodeo y la detección de caídas, lo que permite respuestas proactivas.

Este documento presenta diversas características y métodos de configuración de **BioStar X**. Brindamos orientación extensa y diversa para ayudarle a aprovechar al máximo todas las funciones de **BioStar X**. Siga la guía paso a paso para implementar **BioStar X** de manera fácil y rápida.
