Este documento proporciona orientación sobre cómo inscribir credenciales de usuario. Los usuarios pueden inscribir sus fingerprints, faces, información de card y PINs como credenciales. Las credenciales se pueden añadir o modificar al registrar o actualizar un usuario.

Vaya a la página <Cmd sid="shell.header.launcher" /> → <Cmd sid="shell.header.user" /> y inscriba, añada o modifique credenciales de las dos siguientes maneras:

import EnterUserPage from '../../common/_enter-user-page.mdx';

<EnterUserPage />

En la sección <Cmd sid="team.user.credential.title" />, se pueden configurar los siguientes elementos. Consulte las descripciones de cada elemento para configurar los permisos de usuario adecuados.

<Image src="/img/biostarx-new-user-credential.png" />

Inscriba credenciales y establezca el nivel de seguridad apropiado en <Cmd sid="user.fingerSecurityLevel" product="2" />. Un nivel de seguridad más alto puede resultar en una menor tasa de autenticación o una mayor tasa de rechazo falso (FRR).
