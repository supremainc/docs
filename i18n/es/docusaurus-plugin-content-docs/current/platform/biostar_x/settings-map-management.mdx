Los ajustes de mapa permiten configurar y administrar pisos, instalaciones y zonas en conjunto con el mapa. Utilice esta función para ver un mapa segmentado por zonas, instalaciones y pisos en la página <Cmd sid="shell.header.arena" />.

<DocCardList />

:::info

* Para obtener más información sobre el monitoreo de mapas, consulta [lo siguiente](monitoring-map).

* Las funciones de configuración y monitoreo del mapa solo están disponibles con una licencia **Advanced** o superior.

:::
