En la página <Cmd sid="shell.header.setting" />, administra varios ajustes de **BioStar X**. Los componentes de la interfaz de usuario de esta página son los siguientes:

<Image src="/img/biostarx-ui-settings.png" className="none" />

<Calloutlists className="svg-none-style">
  1. Revisa los distintos menús de ajustes que se ofrecen en el menú <Cmd sid="shell.header.setting" />. Para obtener más información, consulta [lo siguiente](how-to-guide-settings).

  2. Una herramienta para navegar por la lista de páginas. Desplázate entre páginas o ve a la página deseada.

     <Image src="/img/biostarx-settings-access-control-status-pagenation.png" />

     * <IcFirst2 />: Ir a la primera página.

     * <IcFw2 />: Ir a la página anterior.

     * <IcBw2 />: Ir a la página siguiente.

     * <IcLast2 />: Ir a la última página.

     * Ingresa el número de página en el campo de entrada para ir a la página deseada.

     * Haz clic en el cuadro de selección de filas para establecer la cantidad de elementos que se muestran en cada página.

     :::info

     La compatibilidad puede variar según el menú de ajustes.

     :::

  3. Haz clic en <IcMoreW /> para acceder a opciones adicionales.

     :::info

     Las funciones compatibles pueden variar según el menú de ajustes.

     :::
</Calloutlists>
