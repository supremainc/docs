Guía para registrar códigos QR o códigos de barras como métodos de autenticación de usuario. Inscriba los códigos QR generados directamente por **BioStar X** y los códigos QR/códigos de barras emitidos externamente.

:::info

* A continuación se enumeran los dispositivos que incluyen escáneres capaces de autenticación mediante QR/código de barras.

  * X-Station 2 (XS2-QDPB, XS2-QAPB)

* Los dispositivos que incluyen cámaras capaces de autenticación mediante QR/código de barras se incluyen en la siguiente lista.

  * X-Station 2 (XS2-ODPB, XS2-OAPB, XS2-DPB, XS2-APB) versión de firmware 1.2.0 o superior

  * BioStation 3 (BS3-DB, BS3-APWB) versión de firmware 1.1.0 o superior

  * Se requiere una licencia de dispositivo independiente para utilizar la autenticación mediante QR/código de barras con una cámara. Para más información, consulta [el siguiente](settings-license-device-license).

:::

## BioStar X QR {#biostarxqr}

Emita un código QR que contenga un PIN cifrado para permitir el acceso del usuario.

:::note

**Antes de comenzar**

* Complete la configuración de la información de correo electrónico, incluida la configuración SMTP, antes de usar **BioStar QR**. Para más información, consulta [lo siguiente](settings-email-setting).

* Para emitir **BioStar QR**, debe registrar una dirección de correo electrónico en la información del usuario. Para obtener más información, consulte [lo siguiente](add-user-basic-info).

:::

1. Haga clic en <Cmd sid="button.addQr" product="2" /> en la sección <Cmd sid="common.credential" product="2" />.

2. Cuando aparezca la ventana <Cmd sid="user.header.qr" product="2" />, seleccione <Cmd sid="team.user.biostarQr" /> en <Cmd sid="user.qrType" product="2" />.

   <Image src="/img/biostarx-new-user-credential-biostarqr.png" />

3. Seleccione <Cmd sid="zone.intrusion.inputType" product="2" />.

   * <Cmd sid="user.random.cardid" product="2" />: Genera automáticamente un ID de tarjeta.

   * <Cmd sid="user.manual.input" product="2" />: Ingrese el ID de la tarjeta manualmente.

4. Haga clic en <Cmd sid="button.enroll" product="2" />.

Luego, cuando un usuario se autentica, se emiten códigos QR y se envían a la dirección de correo electrónico registrada en la información del usuario. El usuario puede escanear el código QR incluido en el correo electrónico en el dispositivo para obtener acceso.

:::info

Para evitar la creación de ID de tarjeta duplicados, se recomienda configurar <Cmd sid="zone.intrusion.inputType" product="2" /> en <Cmd sid="user.random.cardid" product="2" />.

:::

## QR/código de barras {#qrbarcode}

Registre un código QR/código de barras emitido por emisores externos para los usuarios.

1. Haga clic en <Cmd sid="button.addQr" product="2" /> en la sección <Cmd sid="common.credential" product="2" />.

2. Cuando aparezca la ventana <Cmd sid="user.header.qr" product="2" />, seleccione <Cmd sid="team.user.credential.qrBarcode.title" /> en las opciones de <Cmd sid="user.qrType" product="2" />.

   <Image src="/img/biostarx-new-user-credential-qrbarcode.png" />

3. Seleccione <Cmd sid="common.enterManually" product="2" /> en las opciones de <Cmd sid="common.registrationOption" product="2" />.

4. Introduzca directamente el ID del código QR/código de barras emitido previamente en la opción <Cmd sid="user.cardId" product="2" />.

5. Haga clic en <Cmd sid="button.enroll" product="2" />.

:::info

En la opción <Cmd sid="user.cardId" product="2" />, puede ingresar hasta 32 caracteres, incluidas letras en inglés, números y caracteres especiales.

:::
