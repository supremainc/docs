La interfaz de usuario (UI) de **BioStar X** está diseñada para permitir que los usuarios naveguen y utilicen el sistema con facilidad. Este documento presenta los componentes comunes de la UI de **BioStar X**. Comprenda la UI de **BioStar X** y utilice el sistema con mayor eficiencia.

El área de encabezado de la pantalla se utiliza en todas las páginas y ofrece botones de enlace para acceder a varias funciones de **BioStar X**.

<Image src="/img/biostarx-header.png" />

<Calloutlists>
  1. **Logo de BioStar X**: Botón de enlace que permite a los usuarios regresar a la página <Cmd sid="shell.header.launcher" /> de **BioStar X**.

  2. **Accesos directos**: Proporciona enlaces a las páginas principales que ofrece **BioStar X**.

  3. <Cmd sid="shell.header.action" />: Ejecuta la función **action** configurada por el usuario del servidor. Al hacer clic en el botón se muestra la lista de acciones. Para obtener más información, consulte [lo siguiente](settings-custom-interface).

  4. <IcSvTime width="26" height="26" /> **Hora del servidor**: Compruebe la hora actual del servidor en ejecución de **BioStar X**. Al pasar el mouse sobre el botón se muestra la hora actual del servidor.

  5. <IcInfo width="26" height="26" /> **Información**: Verifique la información de versión de **BioStar X** instalado.

  6. <IcHelp width="26" height="26" /> **Ayuda**: Botón de enlace que dirige a la página de ayuda de **BioStar X**.

  7. <IcAccount width="26" height="26" /> **Cuenta**: Ve a <Cmd sid="setting.menu.preferences.root" /> o cambia la contraseña del usuario que inició sesión actualmente.

     * Para obtener más información sobre <Cmd sid="setting.menu.preferences.root" />, consulta [lo siguiente](settings-preferences).

     * Para obtener más información sobre cambiar la contraseña del usuario que inició sesión actualmente, consulta [lo siguiente](set-permission#changeLoginUserPassword).

  8. <IcLogout width="26" height="26" /> **Cerrar sesión**: Finaliza la sesión del usuario conectado y cierra la sesión.
</Calloutlists>
