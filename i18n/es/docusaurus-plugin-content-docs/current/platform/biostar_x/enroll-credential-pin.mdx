Guíe al usuario sobre cómo inscribir un PIN para que pueda acceder ingresándolo en el dispositivo. Inscriba un PIN para usuarios individuales o inscriba a varios usuarios en lote mediante la importación CSV.

## Registrar PIN

1. Haga clic en <Cmd sid="user.pin" product="2" /> en la sección <Cmd sid="common.credential" product="2" />.

   <Image src="/img/biostarx-new-user-credential-pin.png" />

2. Cuando se active el campo de entrada, ingrese la contraseña.

3. Ingrese la misma contraseña en <Cmd sid="user.confirmPin" product="2" />.

4. Haga clic en <Cmd sid="button.enroll" product="2" />.

:::info

* El PIN debe tener al menos 8 caracteres y solo puede contener números.

* Inscriba como máximo una credencial de PIN por usuario.

:::

## Inscribir PIN mediante importación CSV

Use la función de importación CSV para inscribir a varios usuarios a la vez.

1. Agregue una columna `pin` al archivo CSV que se importará.

2. Ingrese el PIN del usuario en la columna `pin` y guarde el archivo CSV.

3. Vaya a la página <Cmd sid="shell.header.user" />.

4. Haz clic en <IcMoreW /> en la parte superior derecha de la pantalla y selecciona <Cmd sid="team.menu.import.title" /> → <Cmd sid="team.menu.import.csv" />.

5. Siga las indicaciones en pantalla para seleccionar el archivo CSV y configurar las opciones de importación.

:::info

* Inscriba como máximo una credencial de PIN por usuario.

* Para obtener más información sobre la función de importación CSV, consulte [lo siguiente](export-import-user-info#import-csv).

:::
