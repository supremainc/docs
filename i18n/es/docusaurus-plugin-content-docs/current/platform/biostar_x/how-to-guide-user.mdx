Esta guía abarca las funciones disponibles en la página <Cmd sid="shell.header.user" />, una de las funciones fundamentales de BioStar X. Esta página presenta varias funciones para gestionar eficazmente a los usuarios mediante la gestión de grupos de usuarios, la gestión de usuarios, la gestión de plantillas de usuarios y la verificación de usuarios por permisos de acceso, así como métodos para mejorar la seguridad mediante permisos. Utilice las distintas funciones para sacar el máximo partido a **BioStar X**.

Haga clic en <Cmd sid="shell.header.user" /> en la página <Cmd sid="shell.header.launcher" /> o seleccione <Cmd sid="shell.header.user" /> en la lista de accesos directos de la esquina superior izquierda de la pantalla.
