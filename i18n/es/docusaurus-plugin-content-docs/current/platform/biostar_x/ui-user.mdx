El menú <Cmd sid="team.sidebar.user" /> de la página <Cmd sid="shell.header.user" /> permite una gestión eficiente de usuarios mediante varias funciones, como la administración de grupos de usuarios, la administración de usuarios y la consulta de usuarios por derechos de acceso, lo que mejora la seguridad mediante permisos. Los componentes de la IU de la página <Cmd sid="shell.header.user" /> son los siguientes:

<Image src="/img/biostarx-ui-user.png" className="none" />

<Calloutlists>
  1. Compruebe los grupos de usuarios o los grupos de acceso.

     * Para obtener más información sobre los grupos de usuarios, consulte [lo siguiente](user-group-manage).

     * Para obtener más información sobre los grupos de acceso, consulte [lo siguiente](view-user-by-access).

  2. Vea los grupos de usuarios o grupos de acceso en formato de estructura de árbol. Expanda o contraiga la estructura de árbol y haga clic en cada grupo para verificar los usuarios que pertenecen a ese grupo.

  3. Verifique la cantidad de usuarios en el grupo de usuarios seleccionado desde el menú lateral.

  4. Haga clic en <Cmd sid="common.selectAll" /> para seleccionar todos los usuarios mostrados en la lista de usuarios. La cantidad de usuarios seleccionados aparece en <Cmd sid="team.user.selected" />.

  5. Ingrese palabras clave para buscar usuarios. Para obtener más información sobre la búsqueda de usuarios, consulte [lo siguiente](view-users#user-search).

  6. Haga clic en el botón de navegación de página ubicado en la parte superior derecha de la pantalla para ver las listas de usuarios siguiente o anterior. Especifique también la página a la que desea moverse. Para obtener más información, consulte [lo siguiente](view-users#select-view-options).

  7. Haga clic en <Image src="/img/menus/ico-more-white.svg" ico alone /> para acceder a varias funciones relacionadas con los usuarios.

     * <Cmd sid="team.menu.export.title" />: Guarde la lista de usuarios en formato de archivo CSV o de datos. Para obtener más información, consulte [lo siguiente](export-import-user-info).

     * <Cmd sid="team.menu.import.title" />: Importe listas de usuarios en formato de archivo CSV o de datos, o importe información de autenticación facial. Para obtener más información, consulte [lo siguiente](export-import-user-info).

     * <Cmd sid="team.menu.transferToDevice" />: Función para enviar la información de usuarios registrados al dispositivo. Para obtener más información, consulte [lo siguiente](transfer-user-info-to-device).

     * <Cmd sid="team.menu.faceMigration" />: Mejore el rendimiento de autenticación actualizando los rostros inscritos en versiones anteriores de **BioStar X** con el algoritmo más reciente. Para obtener más información, consulte [lo siguiente](migration-face).

     * <Cmd sid="team.menu.columnLayout" />: Cambie los elementos de encabezado de las columnas en la lista de usuarios. Para obtener más información al respecto, consulte [lo siguiente](view-users#user-list-column-settings).

     * <Cmd sid="team.menu.print" />: Imprima la lista de usuarios.

  8. Utilice las funciones para el registro de usuarios y la modificación o eliminación por lotes.

     * <Cmd sid="common.batchEdit" />: Modifique información o derechos de acceso para varios usuarios a la vez. Para obtener más información, consulte [lo siguiente](edit-user#user-information-collective-modification).

     * <Cmd sid="common.delete" />: Elimine los usuarios seleccionados. Para obtener más información, consulte [lo siguiente](delete-user).

     * <Cmd sid="team.user.newUser" />: Registre nuevos usuarios. Para obtener más información, consulte [lo siguiente](add-user).
</Calloutlists>
