**BioStar X** es el software de control de acceso de próxima generación desarrollado como sucesor de **BioStar 2**. Admite diversos entornos de implementación actuales y futuros gracias a una arquitectura de sistema mejorada, funciones ampliadas y un modelo de licenciamiento más flexible.

**BioStar X** ofrece un total de cinco niveles de licencia escalables que van desde **Starter** hasta **Elite**, lo que permite elegir según distintos entornos, desde pequeñas empresas hasta grandes corporaciones. Diseñado con una estructura modular, puede añadir diversas funciones opcionales como integración de video, acceso móvil y soporte para múltiples servidores.

## Licencia base {#base-license}

**La licencia básica** es un componente esencial para activar y operar el sistema. Esta licencia sirve como base para aplicar funciones adicionales o actualizaciones.

:::note

* La licencia **Device Manager** es necesaria para los clientes que utilizan software o sistemas de terceros y desean administrar de manera eficiente los dispositivos Suprema a través de **BioStar X**.

* **Número máximo de operadores** se refiere al número de usuarios a quienes se les otorgan permisos de acceso al sistema.

* Si el número máximo de dispositivos IP conectados supera 1,000, se recomienda utilizar el servidor **Multi Communication**.

:::

## Ampliación de capacidad {#capacity-upgrade}

Las actualizaciones de licencia están disponibles para los titulares de licencias **Essential**, **Advanced**, **Enterprise** y **Elite**. Los titulares de la licencia **Device Manager** solo pueden comprar actualizaciones de usuario.

| &nbsp; | Cantidad |
| :-: | :-: |
| Puerta | 32 |
| Usuario | 5,000 |
| Administrador | 10 |

## Paquete {#package}

El siguiente elemento se puede adquirir como complemento a partir del nivel de licencia **Advanced**, y está incluido de forma predeterminada en los niveles de licencia **Enterprise** y **Elite**.

* **Advanced Access Control**: Un paquete de funciones avanzadas de control de acceso: Global APB, Global Fire Alarm, Intrusion Alarm, Mustering, Occupancy Limit, Elevator Control e Interlock.

## Licencia de dispositivo

Las licencias de dispositivos se pueden comprar individualmente y se pueden agregar independientemente de la licencia básica. Las licencias de dispositivos se emiten según el número de serie (S/N). Esta función solo se puede activar en dispositivos que requieran licencia.

| Función | Tipo de Licencia | Modelos compatibles | Observaciones |
| --- | --- | --- | --- |
| **Face Server Matching** | Dispositivo | - | Se requiere un contrato de mantenimiento anual para soporte avanzado. |
| **Camera QR** | Dispositivo | X-Station 2 (sin sensor de lectura de QR), BioStation 3 | - |
| **Cerradura inalámbrica** | Dispositivo | CoreStation 40 | - |
