# Version 1.0.2 (Build No. 1.0.2.179)

Release<Separator/>2026-06-18

## New Features and Improvements

---

1. Supports new device.

   - BioStation 3 MAX

   - XPass Q2

1. Support for deleting device event logs in BioStar X.

1. Added support for changing the main server IP address.

1. Added support for two user synchronization mode options when integrating with a directory service.

   - **Add/Update Only** (Default)

   - **Add/Update/Delete**

1. Improved Backup and Restore

   - Improved to support restoration even when the server IP address has changed.

   - Improved backups to include data from user-defined paths, such as encryption keys and image logs.

   - Improved the stability of the backup and restore process.

1. Improved to allow administrators to be assigned camera group-based access permissions, restricting access to cameras within designated groups only.

1. Improved system security vulnerabilities

   - Application of HTTP security headers for the web server

   - Enhancement of the encryption method for database connection passwords

   - Improved XSS security vulnerabilities

   - Improved SQL Injection security vulnerabilities

   - Improved security vulnerabilities related to access permissions on the user list page

   - Improved security vulnerabilities related to device secure communication and encryption keys

   - Improved security vulnerabilities related to PDF export

1. Security improvements for Time & Attendance

   - Updated the Minimist library to address security vulnerabilities

   - Improved security vulnerabilities in script files

   - Enhanced the database password encryption method

   - Upgraded web libraries (AngularJS, jQuery, and Moment.js)

1. Updated Open JDK version.

1. Supported **Nx Witness v6** for VMS integration.

1. Added a popup message recommending the deletion of unnecessary Redis for when upgrading to BioStar X.
