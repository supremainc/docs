Cambie la configuración de la base de datos.

1. Haz clic en <Start /> → **BioStar X** → **BioStar X Service Manager** en Windows.

2. Cuando aparezca la ventana **BioStar X Service Manager** en el navegador web, haga clic en el menú <Cmd>SERVICES</Cmd>.

3. Haga clic en <Cmd>Stop</Cmd> para todos los servicios.

   <Image src="/img/common/biostarx-tutoria-service-manager-services-stop.png" alone />

   :::note

   Haga clic en <Cmd>Stop All</Cmd> para detener todos los servicios a la vez.

   :::

4. Haz clic en el menú <Cmd>DATABASE</Cmd> en la barra lateral izquierda.

   <Image src="/img/common/biostarx-tutoria-service-manager-database.png" alone />

5. Seleccione en la lista de bases de datos la base cuya configuración desea cambiar.

6. En la sección <Cmd>Database Server Configuration</Cmd>, cambie los siguientes ajustes:

   <Image src="/img/common/biostarx-tutoria-service-manager-database-configuration.png" alone />

   * <Cmd>Name</Cmd>: Ingresa el nombre de la base de datos.

   * <Cmd>Description</Cmd>: Ingresa la descripción de la base de datos.

   * <Cmd>DB Type</Cmd>: Selecciona el tipo de base de datos. (MariaDB, MS SQL)

   * <Cmd>Host</Cmd>: Ingresa el nombre de host o la dirección IP del servidor de bases de datos.

   * <Cmd>Port</Cmd>: Ingresa el número de puerto del servidor de bases de datos.

   * <Cmd>AC</Cmd> / <Cmd>TA</Cmd> / <Cmd>Video</Cmd>: Ingrese el nombre, el usuario y la contraseña de las bases de datos AC, TA y Video.

7. Haz clic en el botón <Cmd>Test Connection</Cmd> en la esquina superior derecha para comprobar si la base de datos está correctamente conectada.

8. Haz clic en <Cmd>Save</Cmd> en la parte superior derecha de la pantalla para guardar la configuración.
