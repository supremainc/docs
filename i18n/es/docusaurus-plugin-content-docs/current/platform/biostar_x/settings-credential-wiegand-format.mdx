Configure el formato de lectura de datos de la tarjeta. Los datos de la tarjeta se procesan según el formato Wiegand configurado.

## Configurar formato de tarjeta Wiegand

1. Haz clic en <Cmd sid="shell.header.setting" /> en la página <Cmd sid="shell.header.launcher" />.

2. Haz clic en <Cmd sid="setting.menu.credential.root" /> → <Cmd sid="setting.menu.credential.cardFormat" /> en la barra lateral izquierda.

3. Haga clic en <Cmd sid="common.wiegand" product="2" /> en la lista de formatos de tarjeta de la izquierda.

   <Image src="/img/biostarx-settings-credentail-card-wiegand-list.png" />

4. Haga clic en el botón <IcEdit /> del elemento sin nombre en la lista de formatos Wiegand.

5. Cuando aparezca la pantalla <Cmd sid="setting.cardFormat.addNewWiegand" product="2" />, configure cada elemento. Para obtener más información, consulta [lo siguiente](#setting).

6. Haga clic en <Cmd sid="button.apply" product="2" /> para agregar el formato Wiegand configurado.

:::info

* Se pueden usar hasta 15 formatos Wiegand.

* Los formatos predefinidos no se pueden modificar ni eliminar.

:::

## Guía de opciones de configuración {#setting}

Consulte los siguientes elementos para establecer la configuración de bits al agregar o editar un formato de tarjeta Wiegand.

<Image src="/img/biostarx-settings-credential-wiegand-format.png" />

* <Cmd sid="common.name" product="2" />: Ingrese el nombre del formato Wiegand.

* <Cmd sid="common.description" product="2" />: Ingrese una descripción breve del formato Wiegand.

* <Cmd sid="setting.cardFormat.totalBits" product="2" />: Ingrese el número total de bits.

* <Cmd sid="setting.cardFormat.fcField" product="2" />: Establezca si se usará el código de identificación. Para usar el código de identificación, marque la casilla y escriba <Cmd sid="setting.cardFormat.startBit" product="2" /> y <Cmd sid="setting.cardFormat.endBit" product="2" />.

* <Cmd sid="setting.cardFormat.idField" product="2" />: Ingrese <Cmd sid="setting.cardFormat.startBit" product="2" /> y <Cmd sid="setting.cardFormat.endBit" product="2" /> para la ID que desea usar. Haga clic en <Cmd sid="button.plusAdd" product="2" /> para agregar un campo de ID.

* <Cmd sid="setting.cardFormat.parityBits" product="2" />: Haga clic en <Cmd sid="button.plusAdd" product="2" /> para ingresar el número de bits de paridad. Ingrese la posición de los bits de paridad y los bits inicial y final.

:::info

Para agregar bits de paridad, se debe ingresar la opción <Cmd sid="setting.cardFormat.totalBits" product="2" />.

:::
