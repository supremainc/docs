Esta guía describe la interfaz de usuario (UI) de la página <Cmd sid="shell.header.arena" />. Puede monitorear el estado de seguridad en tiempo real según puertas, mapas y dispositivos en la página <Cmd sid="shell.header.arena" />. También puede revisar eventos en tiempo real y tomar las acciones necesarias. Los componentes de la UI de la página <Cmd sid="shell.header.arena" /> son los siguientes.

<Image src="/img/biostarx-ui-monitoring.png" className="none" />

<Calloutlists>
  1. Ingrese palabras clave en el campo de búsqueda para obtener dispositivos, puertas, cámaras y otros elementos deseados.

  2. Supervise la estructura en árbol de los grupos revisando <Cmd sid="arena.tree.door.title" />, <Cmd sid="arena.tree.map.title" /> y <Cmd sid="arena.tree.device.title" />.

     * Para obtener más información sobre la supervisión de <Cmd sid="arena.tree.door.title" />, consulte [lo siguiente](monitoring-door).

     * Para obtener más información sobre la supervisión de <Cmd sid="arena.tree.map.title" />, consulte [lo siguiente](monitoring-map).

     * Para obtener más información sobre la supervisión de <Cmd sid="arena.tree.device.title" />, consulte [lo siguiente](monitoring-device).

  3. Haga clic en el botón <Image src="/img/menus/ico-more-black.svg" ico alone /> para expandir o contraer la estructura en árbol de la lista.

     * <Cmd sid="arena.tree.expandAll" />: Expande todas las listas dentro del grupo en el menú lateral para mostrarlas.

     * <Cmd sid="arena.tree.collapseAll" />: Contrae todas las listas dentro del grupo en el menú lateral para ocultarlas.

  4. El **mosaico de video** le permite revisar video en tiempo real o mapas de puertas, cámaras y áreas. Puede supervisar múltiples transmisiones de cámaras simultáneamente.

     * Para obtener más información sobre la supervisión de transmisiones de cámaras, consulte [lo siguiente](monitoring-video).

     * Para obtener más información sobre el monitoreo de mapas, consulta [lo siguiente](monitoring-map).

  5. El **panel de control** le permite controlar las funciones de la puerta o dispositivo seleccionados. Para obtener más información, consulte [lo siguiente](control-door).

  6. La **lista de eventos** le permite revisar eventos en tiempo real en orden cronológico. Puede ver información detallada, estado, usuarios relacionados e información del dispositivo para cada evento y tomar medidas inmediatas si es necesario. Para obtener más información, consulte [lo siguiente](monitoring-event).
</Calloutlists>
