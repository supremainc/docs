La página <Cmd sid="shell.header.launcher" /> es la página de inicio inicial de **BioStar X** que los usuarios encuentran después de iniciar sesión y sirve como centro para acceder a las funciones principales de **BioStar X**. Esta página está diseñada para que los usuarios naveguen y utilicen fácilmente todas las funciones de **BioStar X**. Los componentes de la IU de la página Lanzador son los siguientes.

<Image src="/img/biostarx-launcher.png" className="none" />

La página <Cmd sid="shell.header.launcher" /> proporciona vínculos a las páginas <Cmd sid="shell.header.user" />, <Cmd sid="shell.header.arena" />, <Cmd sid="shell.header.data" />, <Cmd sid="shell.header.dashboard" /> y <Cmd sid="shell.header.setting" />.

* <Cmd sid="shell.header.user" />: Administre usuarios de forma eficiente mediante funciones como gestión de grupos de usuarios, gestión de usuarios, gestión de plantillas de usuarios y verificación de usuarios por permisos de acceso, lo que refuerza la seguridad a través de los permisos. Para obtener más información, consulte [lo siguiente](how-to-guide-user).

* <Cmd sid="shell.header.arena" />: Supervise puertas, dispositivos, video y eventos de diversas maneras. Controle funciones relacionadas y supervise a través de video en tiempo real. Para obtener más información, consulte [lo siguiente](how-to-guide-monitoring).

* <Cmd sid="shell.header.data" />: Consulte información de usuarios o eventos según los criterios deseados y genere informes en una programación establecida. Para obtener más información, consulte [lo siguiente](how-to-guide-data).

* <Cmd sid="shell.header.dashboard" />: Los usuarios pueden agregar y organizar widgets basados en datos seleccionados de los diversos datos que proporciona **BioStar X** para crear el panel que deseen. Para obtener más información, consulte [lo siguiente](how-to-guide-dashboard).

* <Cmd sid="shell.header.setting" />: Configure **BioStar X**, optimizado para los entornos de los usuarios, mediante diversos ajustes para dispositivos, control de acceso, permisos de usuario, idioma y hora. Para obtener más información, consulta [lo siguiente](how-to-guide-settings).

:::info
