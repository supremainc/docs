Guía la interfaz de usuario (IU) de la página <Cmd sid="shell.header.dashboard" />.

<Image src="/img/biostarx-dashboard.png" className="none" caption />

<Calloutlists>
  1. **Área de widgets**: Zona donde se muestran los widgets añadidos por el usuario.

     * El tamaño de cada widget se puede ajustar libremente dentro de los límites de tamaño de cada widget.

     * Los widgets solo se pueden colocar en áreas vacías y no pueden superponerse entre sí.

  2. **Actualizar**: Actualiza la página del panel para actualizar los datos del widget modificado.

     Establece el ciclo de actualización en <Cmd sid="reportApp.autoRefreshInterval" product="2" /> en <Cmd sid="reportApp.dashboardSettings" product="2" />.

  3. <Cmd sid="reportApp.dashboardSettings" product="2" />: Agrega nuevos widgets o configura los datos y la apariencia de los widgets añadidos.
</Calloutlists>
