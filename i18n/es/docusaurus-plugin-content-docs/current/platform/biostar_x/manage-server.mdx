Consulta cómo administrar los servicios del servidor BioStar X mediante **BioStar X Service Manager**. Inicia o detén servicios individuales.

## Administrar el servicio de BioStar X {#biostar-x-setting}

Proporciona una herramienta para administrar el estado del servidor **BioStar X**. Haz clic en <Start /> → **BioStar X** → **BioStar X Service Manager** en Windows. La ventana <Cmd>BioStar X Service Manager</Cmd> aparece.

<Image src="/img/common/biostarx-tutoria-service-manager-main.png" alone />

:::info

Si no se reinicia, **BioStar X** podría no funcionar correctamente. Si cambiaste la configuración de hora del servidor **BioStar X**, detén y reinicia **BioStar X Core Service**.

:::

### Detener servicio {#end-service}

Haz clic en el botón <Cmd>Stop</Cmd> de los servicios individuales en el menú <Cmd>SERVICES</Cmd> para apagar **BioStar X**.

<Image src="/img/common/biostarx-tutoria-service-manager-main.png" alone />

{/* 모든 서비스를 종료하려면 화면 오른쪽 상단의 <Cmd>Stop All</Cmd> 버튼을 클릭하세요. */}

### Iniciar el servidor {#start-the-server}

Haz clic en <Cmd>Start</Cmd> del servicio en el menú <Cmd>SERVICES</Cmd> con estado <Cmd>Stopped</Cmd> en la columna <Cmd>Status</Cmd> para reiniciar **BioStar X**.

<Image src="/img/common/biostarx-tutoria-service-manager-main-restart.png" alone />

{/* 모든 서비스를 시작하려면 화면 오른쪽 상단의 <Cmd>Start All</Cmd> 버튼을 클릭하세요. */}
