Cambie el formato de datos de tarjeta Wiegand en uso de forma simultánea.

:::caution

El cambio de formato de datos de las tarjetas ya asignadas a usuarios también se realizará, así que tenga precaución.

:::

1. Haz clic en <Cmd sid="shell.header.setting" /> en la página <Cmd sid="shell.header.launcher" />.

2. Haga clic en <Cmd sid="setting.menu.credential.root" /> → <Cmd sid="setting.menu.credential.card" /> en la barra lateral izquierda.

3. Haga clic en <IcMoreW /> en la parte superior derecha de la lista de tarjetas y seleccione <Cmd sid="setting.card.more.changeDataFormat" product="2" />.

4. Cuando aparezca la ventana <Cmd sid="setting.card.changeDataFormat.title" product="2" />, seleccione el formato de datos de tarjeta que desea cambiar en la lista <Cmd sid="setting.card.changeDataFormat.currentType" product="2" />.

   <Image src="/img/biostarx-settings-credentail-card-change-data-format.png" />

5. Seleccione el formato de datos de tarjeta deseado en la lista <Cmd sid="setting.card.changeDataFormat.targetType" product="2" />.

6. Haga clic en <Cmd sid="button.apply" product="2" /> para cambiar el formato de datos de tarjeta.
