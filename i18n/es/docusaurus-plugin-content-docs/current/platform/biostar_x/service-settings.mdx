Configure y gestione los servicios centrales del sistema en el menú <Cmd>Configuración del servicio</Cmd> de <Cmd>BioStar X Service Manager</Cmd>. Configure cada servicio de forma independiente y verifique los puertos de red y la información de versión.

La <Cmd>Configuración del servicio</Cmd> incluye los siguientes servicios principales.

* <Cmd>Servicio web principal de BioStar X</Cmd>: Servicio de interfaz web

* <Cmd>Servicio principal de BioStar X</Cmd>: Funcionalidades centrales del sistema y servicio de comunicación API

* <Cmd>Servicio de puerta de enlace unificada</Cmd>: Servicio de puerta de enlace integrada basado en proxy inverso

* <Cmd>Servicio coordinador de BioStar X</Cmd>: Administración de la información de configuración del sistema distribuido y sincronización de servicios

* <Cmd>Servidor BioStar X (Principal)</Cmd>: Servicio de servidor principal

* <Cmd>Servicio de caché de BioStar X</Cmd>: Almacenamiento en caché de datos y mejora del rendimiento del sistema

## Situaciones en las que se necesitan cambios de configuración {#when-to-change}

Cambie la configuración del servicio en las siguientes situaciones.

* **Resolver conflictos de puertos**: cuando otra aplicación utiliza el mismo puerto y provoca conflictos

* **Cumplimiento de políticas de seguridad**: cuando la política de seguridad de la red de la organización solo permite puertos específicos

* **Configuración del firewall**: cuando sea necesario cambiar a puertos permitidos por el firewall corporativo

* **Cambios en el entorno del servidor**: al cambiar la dirección IP del servidor o la configuración de red

* **Optimización del rendimiento**: cuando se necesita separar puertos para distribuir el tráfico o equilibrar la carga

## Cómo cambiar la configuración {#how-to-change}

1. Haga clic en el menú <Cmd>CONFIGURACIÓN DEL SERVICIO</Cmd> en <Cmd>BioStar X Service Manager</Cmd>.

2. Vaya a la sección del servicio que desea cambiar.

3. Modifique el número de puerto o la dirección necesarios.

4. Haga clic en el botón <Cmd>Aplicar</Cmd> en la esquina superior derecha de la pantalla para aplicar los cambios.

:::note

* Reinicie el servicio correspondiente para aplicar los cambios tras modificar la configuración.

* La compatibilidad para cambiar el puerto varía según el servicio.

:::

:::caution

Al cambiar los números de puerto, asegúrese de que no existan conflictos con otros servicios o aplicaciones.

:::

## Composición del servicio {#service-composition}

### Servicio web principal de BioStar X {#biostar-x-core-web-service}

Este servicio es responsable de la interfaz de usuario basada en web.

<Image src="/img/common/biostarx-sm-cws.png" alone />

* <Cmd>Puerto HTTPS</Cmd>: puerto de acceso a la interfaz web (predeterminado: 5002)

* <Cmd>Puerto WebServerThrift</Cmd>: puerto de comunicación del servidor web basado en el protocolo Thrift (predeterminado: 9310)

### Servicio principal de BioStar X {#biostar-x-core-service}

Este es el servicio principal que gestiona las funcionalidades centrales del sistema y las comunicaciones API externas.

<Image src="/img/common/biostarx-sm-core-s.png" alone />

* <Cmd>Puerto WebSocket</Cmd>: puerto WebSocket para comunicación bidireccional en tiempo real (predeterminado: 9002)

* <Cmd>Puerto API</Cmd>: puerto de comunicación de la API REST (predeterminado: 9010)

* <Cmd>Puerto WebServerFastCgi</Cmd>: puerto de comunicación del servidor web basado en protocolo FastCGI (predeterminado: 9000)

* <Cmd>Puerto RPC</Cmd>: puerto de comunicación de llamada a procedimiento remoto (RPC) (predeterminado: 51218)

### Servicio de puerta de enlace unificada {#unified-gateway-service}

Procese eficientemente las solicitudes al servidor **BioStar X** mediante proxy inverso, mejore las vulnerabilidades de seguridad de los iframes y minimice los errores de certificado SSL.

<Image src="/img/common/biostarx-sm-ugs.png" alone />

* <Cmd>Puerto HTTPS</Cmd>: puerto de comunicación HTTPS de la puerta de enlace integrada (predeterminado: 443)

### Servicio coordinador de BioStar X {#biostar-x-coordinator-service}

Este servicio se encarga de administrar la información de configuración del sistema distribuido, supervisar el estado del servicio y sincronizar los servicios.

<Image src="/img/common/biostarx-sm-coordinator-s.png" alone />

* <Cmd>Puerto de cliente</Cmd>: puerto de comunicación para conexiones de cliente (predeterminado: 21810)

:::info

* Cambie el valor del <Cmd>Puerto de cliente</Cmd> y, luego, reinicie manualmente el **Servicio coordinador de BioStar X** en **Servicios de Windows** (*services.msc*).

* Después de reiniciar el **Servicio coordinador de BioStar X**, vaya a **BioStar X Service Manager** → **SERVICIOS** y reinicie todos los servicios.

  Haga clic en <Cmd>Detener todo</Cmd> en la esquina superior derecha de la pantalla. Cuando todos los servicios estén detenidos y el botón <Cmd>Iniciar todo</Cmd> esté habilitado, haga clic en él.

:::

### Servidor BioStar X (principal) {#biostar-x-server-(main)}

Este es el servicio central que funciona como servidor principal del sistema.

<Image src="/img/common/biostarx-sm-server.png" alone />

* <Cmd>Dirección del servidor</Cmd>: dirección IP del servidor principal

* <Cmd>Puerto del servidor</Cmd>: puerto de comunicación general del servidor (predeterminado: 51212)

* <Cmd>Puerto SSL del servidor</Cmd>: puerto de comunicación segura cifrada SSL (predeterminado: 51213)

* <Cmd>Puerto gRPC del servidor</Cmd>: puerto de comunicación de alto rendimiento basado en el protocolo gRPC (predeterminado: 51219)

### Servicio de caché de BioStar X {#biostar-x-cache-service}

Este servicio mejora la velocidad de procesamiento de datos y el rendimiento del sistema al almacenar en memoria los datos utilizados con frecuencia.

<Image src="/img/common/biostarx-sm-cache-s.png" alone />

* <Cmd>Puerto de cliente</Cmd>: puerto de conexión de cliente del servicio de caché (predeterminado: 10800)

* <Cmd>Puerto de comunicación</Cmd>: puerto de comunicación interna entre nodos de caché (predeterminado: 47500)

* <Cmd>Puerto de descubrimiento</Cmd>: puerto para el descubrimiento automático de nodos de caché distribuidos (predeterminado: 47100)
