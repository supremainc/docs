---
id: register-license-key
title: Registrar clave de licencia
description: Activa la licencia de BioStar X para usar funciones adicionales.
keywords:
  - Licencia
---

Si adquiriste la licencia de BioStar X, registra la clave de licencia para usar más funciones.

Ve a <Cmd sid="shell.header.setting" /> → <Cmd sid="setting.menu.license.root" /> → <Cmd sid="setting.menu.license.biostarXLicense" />.

El método para activar <Cmd sid="setting.menu.license.biostarXLicense" /> varía según el entorno de red. Verifique el estado de la red y active la licencia según las instrucciones proporcionadas.

:::info

* Para más información sobre el menú <Cmd sid="setting.menu.license.root" />, consulta [lo siguiente](settings-license).

* Para obtener más información sobre la política de licencias, consulta [lo siguiente](licensing).

* Para obtener información sobre los códigos de error de licencias, consulta [el siguiente enlace](https://reprisesoftware.com/docs/isv/appendix/appendix-b-rlm-status-values.html).

:::
