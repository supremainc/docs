Utilice la función <Cmd sid="shell.header.visitor" /> para crear una página de solicitud de visitantes en la PC de solicitudes y gestione el acceso de visitantes en la PC de administración.

:::info

* La función <Cmd sid="shell.header.visitor" /> está disponible con opciones adicionales en la licencia **Advanced** o superior. Para más información sobre la política de licencias, consulta [lo siguiente](licensing#features-add-ons).

* Después de activar la función de visitantes mediante la licencia, consulte [lo siguiente](settings-visitor) para obtener instrucciones detalladas de configuración.

:::

## Aplicar la licencia de visitantes

Esta guía explica cómo aplicar la licencia de visitantes. Después de comprar la licencia <Cmd sid="shell.header.visitor" className="normal" /> al distribuidor de **BioStar X**, siga las instrucciones a continuación para aplicar la licencia.

import LicenseApply from '../../common/_bsx-license-apply.mdx'

:::tip

Aplicar la licencia de visitantes activa el menú <Cmd sid="shell.header.setting" /> → <Cmd sid="common.visitor" product="2" />, y agrega un enlace a la página de gestión de <Cmd sid="shell.header.visitor" /> en la lista de accesos directos en la esquina superior izquierda de la pantalla.

:::

## Configuración de la página de solicitud de visitantes

La página de solicitud de visitantes puede usarse después de priorizar la configuración de visitantes. Consulte [lo siguiente](settings-visitor) para obtener instrucciones detalladas para configurar la página de solicitud de visitantes.

Se puede acceder a la página de solicitud de visitantes desde una PC dedicada a solicitudes, lo que permite a los visitantes enviar solicitudes de acceso. Para obtener más información sobre las solicitudes de visitantes, consulte [lo siguiente](visitor-register).

## Gestión de acceso de visitantes

Verifique el historial de solicitudes de visitantes y gestione el acceso de visitantes. Para obtener más información sobre la gestión de acceso de visitantes, consulte [lo siguiente](visitor-register-manage).
