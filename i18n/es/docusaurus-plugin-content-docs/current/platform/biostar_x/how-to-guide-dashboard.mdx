El panel se puede personalizar para adaptarse a las preferencias individuales, lo que permite que cada usuario seleccione la información que desee, agregue widgets y configure y organice libremente los widgets.

:::info

* Cada usuario puede configurar <Cmd sid="shell.header.dashboard" /> de forma individual.

* La información disponible en <Cmd sid="shell.header.dashboard" /> puede variar según <Cmd sid="user.bioStarOperator" product="2" />.

:::
