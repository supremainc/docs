Esta guía describe cómo configurar el sistema de control de acceso. Establezca puntos de entrada y horarios de entrada para crear niveles de acceso y configure grupos de acceso usando los niveles de acceso y la información de grupos de usuarios. Use los grupos de acceso configurados para la configuración del control de acceso.
