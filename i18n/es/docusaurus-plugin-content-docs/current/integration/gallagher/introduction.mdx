## Características

Integración de Suprema con Gallagher: comunicación entre Gallagher Command Centre y Suprema BioStar 2. Así, se pueden generar plantillas biométricas para los dispositivos Suprema desde Gallagher Command Centre, lo que ofrece una solución de vanguardia para todos los usuarios finales.

**A continuación se presentan las características que ofrece la integración de Suprema con Gallagher:**

- El enrolamiento masivo de usuarios y la gestión desde Gallagher Command Centre se reflejan en Suprema BioStar 2.

- Con la integración de Suprema, los usuarios pueden enrolar datos de rostro visual biométrico y huella dactilar mediante escaneo en diversos dispositivos biométricos Suprema.

- Detección de CSN y Wiegand, e integración con BioStar para el enriquecimiento automático de datos.

- Con la integración de Suprema, los usuarios pueden enrolar datos de tarjeta mediante escaneo desde dispositivos biométricos como X-Station 2, BioLite N2, BioStation 3, FaceStation F2, XPass 2, X-Station 2, BioEntry W3 y BioEntry W2 (estos son los dispositivos probados).

- Permite administrar hasta 1,000 dispositivos biométricos.

- Todas las settings de los dispositivos Suprema se pueden realizar en la misma ventana de Command Centre.

- Compatible con una amplia gama de tipos de tarjeta, incluidas tarjetas TOC, tarjetas QR e incluso configuraciones de tarjeta personalizadas adaptadas a las necesidades específicas del cliente.

- La instalación de la integración se simplifica mediante un paquete EXE incluido que contiene todos los requisitos previos necesarios y garantiza un proceso de instalación fluido y eficiente.

- El middleware incluye soporte integrado para settings relacionadas con la red, junto con una guía completa de solución de problemas para ayudar con errores de configuración y escenarios de tolerancia a fallos.
