La integración de Suprema con Gallagher es un middleware que permite al sistema Gallagher Command Centre 8.8 o posterior comunicarse con dispositivos biométricos Suprema, que pueden inscribir credenciales faciales biométricas o de huella dactilar para los usuarios desde Command Centre y administrar la configuración de los dispositivos conectados. Con la integración de Suprema con Gallagher, puede configurar y crear fácilmente el sistema de administración biométrica para Command Centre con dispositivos biométricos Suprema.

:::info

Para obtener más detalles sobre la funcionalidad del sistema de control de acceso Gallagher Command Centre, consulte los manuales de usuario de Command Centre.

:::
