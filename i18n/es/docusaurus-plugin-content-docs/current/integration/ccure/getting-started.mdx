Antes de usar la integración Suprema - C•CURE para implementar Suprema Biometrics en la plataforma C•CURE 9000 para un sistema de control de acceso, deben instalarse en el mismo servidor o estación de trabajo donde está instalado el servidor C•CURE tanto C•CURE 9000 v2.90, v3.0, v3.10 (o posterior) Server como el servidor Suprema BioStar 2, junto con este servicio de instalación de la integración. Además, debe instalarse C•CURE 9000 v2.90, v3.0, v3.10 (o posterior) Client en la(s) estación(es) de trabajo donde se usará, junto con este cliente de instalación de la integración.

El servidor recibe registros de eventos, información de usuarios, etc. de los dispositivos conectados y los almacena. La integración se puede instalar fácilmente. Antes de la instalación, revise los Requisitos mínimos del sistema.

El paquete de instalación de la integración Suprema - C•CURE para servidor y cliente se puede proporcionar a solicitud enviada al [equipo de Suprema](https://support.supremainc.com).

## Abreviaturas

- Software House C•CURE 9000 (C•CURE)

- BioStar 2 de Suprema (BioStar 2)

- Integración Suprema - C•CURE ("Integración" para abreviar)

Esta integración incluye los siguientes componentes:

- Servicio de integración Suprema - C•CURE (Servicio de integración)

- Cliente de integración Suprema - C•CURE (Cliente de integración)

- Herramienta de configuración de la integración Suprema - C•CURE (Integration CT)

  - Integration CT es la herramienta de interfaz gráfica web para configurar el Servicio de integración (sin GUI) y ajustar también los parámetros del Cliente de integración.

## Dispositivos Compatibles

Dispositivos RFID:

- familia XPass 2

- familia X-Station 2

Dispositivos faciales:

- FaceStation F2

- BioStation 3

- BioEntry W3

Dispositivos de huella:

- Familia BioEntry (P2, W2)

- BioLite N2

- BioStation 2, BioStation A2

- X-Station 2 (modelo de huella)

- BioStation L2

Dispositivos NO compatibles:

- BioEntry R2

- CoreStation

- FaceStation 2 / FaceLite

:::info

Los lectores USB como BioMini Plus 2 / Slim 2 pueden ser compatibles mediante personalización. Para ello, escriba a `latam@supremainc.com`.

:::
