title: Endurecimiento de BioStar X
description: "Refuerza la seguridad de BioStar X en dos capas críticas: protección a nivel de la aplicación mediante autenticación y control de acceso, y protección a nivel de los datos mediante cifrado y almacenamiento mínimo de datos biométricos."
---

Asegurar BioStar X requiere proteger dos capas interconectadas: el sistema y los datos personales que almacena.

* **Seguridad del sistema** refuerza la aplicación mediante cambios en la cuenta de administrador predeterminada, control de acceso basado en roles, autenticación sólida y comunicación cifrada del dispositivo.

* **Seguridad de los datos personales** protege las credenciales de usuario y los datos biométricos sensibles mediante cifrado de la base de datos y prácticas de almacenamiento mínimo.

Implementar ambas capas garantiza el cumplimiento normativo (GDPR, ISO 27001, etc.) y minimiza el riesgo de brechas.
