title: Guía de Endurecimiento del Servidor
description: Recomendaciones para proteger el sistema operativo, la red y el entorno físico que alojan BioStar X.
---

Esta guía ofrece prácticas recomendadas para proteger entornos Windows que alojan BioStar X. Cubre controles a nivel del sistema, de la red y de la aplicación, diseñados para minimizar vulnerabilidades, reducir la superficie de ataque y garantizar el cumplimiento de las políticas de seguridad de la organización.

* Se aplica a todos los Windows Server (hardware físico o máquina virtual) que ejecutan BioStar X.

* Cubre la configuración del sistema, la protección de la red, los controles de acceso, la supervisión y la aplicación de parches.

* Está dirigido a administradores de sistemas, ingenieros de seguridad y equipos de implementación.
