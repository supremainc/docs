CoreStation Setup Manager es un servidor web que gestiona los ajustes de red de CoreStation o verifica el estado de conexión de los dispositivos esclavos, los puertos de entrada y salida y los puertos Wiegand. Usa CoreStation Setup Manager para verificar in situ el estado de la red y del cableado sin tener que desplazarte hasta la ubicación del servidor BioStar X mientras configuras el sistema de control de acceso con CoreStation.

CoreStation Setup Manager ofrece las siguientes funciones.

* Ajuste de dirección IP

* Consultar la información del dispositivo CoreStation

* Ajustes de administrador

* Consultar el estado de conexión de los dispositivos esclavos

* Supervisar entradas y salidas y el estado de los puertos Wiegand

* Inicializar el dispositivo sin afectar los ajustes de red

* Restablecimiento de fábrica

## Requisitos mínimos

Verifica los dispositivos y las versiones de firmware compatibles.

* Firmware de CoreStation versión 1.3.1 o superior

* Firmware de CoreStation 20 versión 1.0.0 o superior

:::info

* CoreStation Setup Manager es compatible con el firmware de CoreStation versión 1.3.1 o superior y con el firmware de CoreStation 20 versión 1.0.0 o superior. Si utilizas una versión de firmware inferior, conéctate a BioStar X para actualizar el firmware a la versión más reciente.

* Para obtener más detalles sobre cómo conectar e instalar cada dispositivo, consulta la guía del producto correspondiente.

  * [**CoreStation**](../corestation_40/index.mdx)

  * [**CoreStation 20**](../corestation_20/index.mdx)

* Para obtener más información sobre CoreStation y CoreStation Setup Manager, comunícate con el equipo de soporte de Suprema (CS@suprema.co.kr).

:::
