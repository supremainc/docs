CoreStation Setup Manager es un servidor web que gestiona los ajustes de red de CoreStation o verifica el estado de conexión de los dispositivos esclavos, los puertos de entrada y salida y los puertos Wiegand. Usa CoreStation Setup Manager para verificar in situ el estado de la red y del cableado sin tener que desplazarte hasta la ubicación del servidor BioStar X mientras configuras el sistema de control de acceso con CoreStation.
