Guía para configurar la cuenta de administrador y los ajustes de red del servidor web.

## Editar cuenta de administrador

Edita la cuenta de administrador de CoreStation Setup Manager.

1. Haz clic en **SETTINGS**.

2. Haz clic en **Admin Account** → <IcEdit />**Edit**.

   <Image src="/img/device/settings-admin-accounts.png" alone />

3. Consulta [Initial Setup of the Device](initial-setup), edita cada elemento y luego haz clic en **OK**.

   <Image src="/img/device/settings-edit-user.png" alone />

4. Haz clic en **Apply** para guardar los ajustes.

## Red del servidor web

Cambia la información de red de CoreStation Setup Manager.

1. Haz clic en **SETTINGS**.

2. Edite los campos necesarios.

   <Image src="/img/device/settings-webserver-network.png" alone />

   <div className="overflow-x col2--20-80">
     | Nombre | Descripción de la función |
     | --- | --- |
     | **Dirección IP** | Edita la dirección IP de CoreStation Setup Manager.<br />- Solo se puede usar una IP estática. No se admite DHCP. |
     | **Puerta de enlace** | Edita la puerta de enlace de CoreStation Setup Manager. |
     | **Máscara de subred** | Edita la máscara de subred de CoreStation Setup Manager. |
   </div>

3. Haz clic en **Apply** para guardar los ajustes. El dispositivo se reiniciará.
