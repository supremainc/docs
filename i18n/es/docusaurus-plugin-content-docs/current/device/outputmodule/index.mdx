El Módulo de Salida es un módulo de control de múltiples salidas que puede usar 12 salidas de relé. Cuando se configura con BioStar X, puede utilizarse para el control de acceso por piso. Se puede conectar un máximo de 31 dispositivos a un solo dispositivo maestro mediante la cadena RS-485, y también se pueden usar entradas AUX. Además, puede utilizarse de diversas maneras junto con áreas antipassback o zonas de alarma contra incendios.
