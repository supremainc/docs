## Nombre y función de cada parte

### Configuración de puertos internos

<Image src='/img/camera/name-function-each-part-vionyx.svg' className='none' />

* **Entrada de alarma, relé (5 pines)**: Conecte el cable para la entrada de alarma y la salida de relé.

* **Entrada/salida de audio (4 pines)**: Conecte el cable para la entrada y salida de audio.

* **Ethernet (PoE+)**: Conecte el cable Ethernet. La alimentación puede suministrarse mediante PoE+.

* **Alimentación, RS-485 (5 pines)**: Conecte el cable para la alimentación y la comunicación RS-485.

* **Botón de reinicio**: Restaura todos los ajustes a los valores de fábrica. Para más información sobre la inicialización de fábrica, consulte [Restablecimiento de fábrica](installation#factory-reset).

* **Ranura para tarjeta Micro SD**: Inserte la tarjeta Micro SD.

* **USB (tipo A)**: Conecte accesorios USB, como Wi-Fi y BLE.

* **Interruptor del micrófono**: Utilice el interruptor físico del micrófono para controlar la grabación de audio.

### Indicador de estado LED

El LED en la parte frontal de la cubierta de la cámara indica el estado de funcionamiento y de autenticación mediante colores y patrones de parpadeo.

<Image src='/img/camera/led-status-indicator-vionyx.svg' className='none' alone />

| Color del LED | Estado |
| :-: | --- |
| Verde | Autenticación exitosa |
| Rojo | Autenticación fallida |
| Azul | Funcionamiento normal |
| Azul/verde intermitente | Al configurar <Cmd sid='ui_012' product='dev' /> para usar, no se puede obtener la dirección IP. |

### Configuración del interruptor del micrófono

Utilice el interruptor del micrófono para desactivar físicamente el micrófono de la cámara y evitar la grabación de audio.

<Image src='/img/camera/microphone-switch-vionyx.svg' className='none' alone />

* La grabación de audio está activada por defecto.

* Deslice el interruptor del micrófono hacia la izquierda para desactivarlo y detener la grabación de audio.

* La grabación de audio también se puede desactivar desde la cuenta de Command.

:::caution

Si el interruptor del micrófono está desactivado físicamente, no se grabará audio aunque se cambien los ajustes en la cuenta de Command.

:::
