## Nombre y función de cada parte

### Configuración de puertos internos

<ProdImg src='/img/camera/name-function-each-part-vionyx.svg' className='none' width={700} />

<div className='overflow-x col2-2-8'>
| Nombre | Función |
| -- | -- |
| **Entrada de alarma, Relé (5 pines)** | Conecte el cable para la entrada de alarma y la salida de relé. |
| **Entrada/Salida de audio (4 pines)** | Conecte el cable para la entrada/salida de audio. |
| **Ethernet (PoE+)** | Conecte el cable Ethernet. La alimentación puede suministrarse mediante PoE+. |
| **Alimentación, RS-485 (5 pines)** | Conecte el cable para alimentación y comunicación RS-485. |
| **Botón de reinicio** | Restablece todos los ajustes a los valores predeterminados de fábrica. Para obtener más información sobre la inicialización de fábrica, consulte [Restablecimiento de fábrica](maintenance#factory-reset). |
| **Ranura para tarjeta Micro SD** | Inserte una tarjeta microSD. |
| **USB (Tipo A)** | Conecte accesorios basados en USB. |
| **Interruptor de MIC** | Use el interruptor del micrófono para habilitar o deshabilitar físicamente la grabación de audio. |
</div>

### Indicador de estado LED

El LED en la parte frontal de la cubierta de la cámara indica el estado de funcionamiento y de autenticación mediante colores y patrones de parpadeo.

<Image src='/img/camera/led-status-indicator-vionyx.svg' className='none' width={350} alone />

| Color del LED | Estado |
| :-: | --- |
| Verde | Autenticación exitosa |
| Rojo | Autenticación fallida |
| Azul | Funcionamiento normal |
| Azul/verde intermitente | Al configurar <Cmd sid='ui_012' product='dev' /> para usar, no se puede obtener la dirección IP. |

<PageBreak />

### Configuración del interruptor del micrófono (próximamente)

Utilice el interruptor del micrófono para desactivar físicamente el micrófono de la cámara y evitar la grabación de audio.

<Image src='/img/camera/microphone-switch-vionyx.svg' className='none' width={450} alone />

* La grabación de audio está deshabilitada por defecto.

* Deslice el interruptor del micrófono hacia la derecha para habilitar la grabación de audio.
