Este tema explica cómo establecer una contraseña e iniciar sesión la primera vez que acceda al servidor web de ViOnyx.

## Acceso al servidor web de la cámara {#access-web-server}

:::info

* Use una computadora conectada a la misma red que la cámara.

* Si el dispositivo no obtiene una dirección IP después de un restablecimiento de fábrica, cambia a la dirección IP predeterminada (`https://192.168.1.100`), así que conéctese usando esa dirección.

:::

1. Póngase en contacto con [Ventas de Suprema](https://www.supremainc.com/es/support/main.asp) para solicitar el paquete de instalación (*VionyxTool.zip*).

1. Extraiga el archivo de instalación y ejecute *vionyx_tool.exe*.

   <Image src="/img/camera/web-server-settings1.png" width='80%' alone />

3. Haga clic en Buscar cámaras.

4. En la lista de cámaras detectadas, busque el número de serie de la cámara (**DEVICE ID**) y verifique la **IP ADDRESS**.

   <Image src="/img/camera/web-server-settings2.png" width='80%' alone />

5. Inicie el navegador web.

6. Ingrese la **IP ADDRESS** de la cámara, incluida `https://`, en la barra de direcciones y luego presione **Enter**.

   > Ejemplo: `https://192.168.1.100/`

7. Aparecerá la página del servidor web de ViOnyx.

## Establecer la contraseña

La primera vez que acceda al servidor web de ViOnyx, establezca una contraseña.

1. Consulte [Acceso al servidor web de la cámara](#access-web-server), luego abra un navegador web y vaya al servidor web de ViOnyx en `https://IP_ADDRESS`.

   > Ejemplo: `https://192.168.1.100/`

   <Image src="/img/camera/password-setting1-vionyx.png" />

2. Ingrese la contraseña en el campo <Cmd sid='password' product='cam' />.

3. Vuelva a ingresar la misma contraseña en el campo <Cmd sid='confirm_password' product='cam' />.

4. Asegúrese de que la contraseña que introdujo cumpla con todos los siguientes requisitos.

   :::note

   **Requisitos de la contraseña**

   * **Confirmación de contraseña**: Los valores en los campos <Cmd sid='password' product='cam' /> y <Cmd sid='confirm_password' product='cam' /> deben coincidir exactamente.

   * **Longitud**: Introduzca entre 12 y 64 caracteres.

   * **Composición de caracteres**: Debe incluir al menos 1 de cada uno de los siguientes: letras mayúsculas en inglés (A-Z), letras minúsculas en inglés (a-z), dígitos (0-9) y caracteres especiales (`"` `!` `@` `#` `%` `^` `&` `*` `(` `)` `-` `_` `=` `+` `[` `]` `{` `}`).

     * No se permiten espacios, Hangul ni otros caracteres Unicode.

   * **Reglas de seguridad**: Para evitar vulnerabilidades de seguridad, no se permiten los siguientes patrones.

     * Secuencias consecutivas de 4 o más letras o números (p. ej., abcd, 1234, 4321)

     * Caracteres o dígitos idénticos repetidos 4 o más veces (p. ej., aaaa, 1111)

     * Contiene el ID de usuario (no distingue entre mayúsculas y minúsculas)

     * Palabras o patrones comunes que son fáciles de adivinar (p. ej., password, qwerty)

   :::

5. Cuando se cumplan todas las condiciones, el botón <Cmd sid='submit' product='cam' /> se activa. Haga clic en el botón para finalizar la configuración de la contraseña.

:::caution

Tenga cuidado de no perder la contraseña. Si la pierde, deberá restablecer la cámara.

:::

## Cómo Iniciar Sesión

Una vez establecida la contraseña, se le redirigirá automáticamente a la página de inicio de sesión. Introduzca el ID de administrador y la contraseña que configuró para acceder al servidor web de ViOnyx.

<Image src="/img/camera/password-setting2-vionyx.png" />

1. Ingrese **admin** en el campo <Cmd sid='id' product='cam' />.

   * Puede cambiar el ID de administrador (**admin**) en <Cmd sid='system' product='cam' /> → <Cmd sid='account_management' product='cam' />.

2. Ingrese la contraseña que configuró en el campo <Cmd sid='password' product='cam' />.

3. Haga clic en el botón <Cmd sid='sign_in' product='cam' />. Después de iniciar sesión correctamente, aparecerá el visor en vivo de la cámara del servidor web de ViOnyx.

4. Consulte la [Guía del usuario del ViOnyx Web Server](../vionyx_webserver) para configurar la cámara.
