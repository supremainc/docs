## TABLERO

1. Presiona <Image src="/img/common/ico-menu.svg" ico alone /> y autentícate con la credencial de nivel de administrador.<br />Consulta el estado y la información del dispositivo en <Cmd sid="ui_109" product="dev" />. Como alternativa, presiona <Cmd sid="ui_174" product="dev" /> para inscribir directamente a un usuario.

2. Seleccione el menú deseado en la parte inferior del panel de control para registrar un usuario o configurar un dispositivo.
