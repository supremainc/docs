# Application Version 1.2.0

Release<Separator/>2024-10-22

:::caution

Device Manager v1.1.8 supports Android 5.0 or higher, iOS 9.0 or higher.

:::

## New Features and Improvements (Common)

---

1. Supports new device.

   * XPass 2 (XP2-MAPB)

   * BioEntry W3 (BEW3-DB, BEW3-APB)

	:::info

   Biometric device only supports network and server settings.

   :::

1. Supports network settings of the **IP device**.

1. Supports XPass D2 smart card key encryption.

1. Supports device firmware upgrade.

   * XPass 2 (XP2-MDPB, XP2-MAPB, XP2-GDPB, XP2-GKDPB)

1. Supports BLE device firmware upgrade.

   * XPass D2 (XPD2-MDB, XPD2-GDB, XPD2-GKDB)

1. Added an option to enable or disable the automatic device connection.

1. Added option to export or import a templates for XPass D2 settings into the app.

1. Discontinued support for the **Scan Card** feature.

## New Features and Improvements (Android)

---

1. Improved the area of the **Load Template** button on the XPass D2 settings screen.

## New Features and Improvements (iOS)

---

1. Improved to allow deletion and modification when an invalid DNS server address is entered.
