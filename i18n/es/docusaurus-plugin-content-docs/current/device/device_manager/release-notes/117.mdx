# Application Version 1.1.7

Release<Separator/>2022-06-13

:::caution

Improvements in Device Manager v1.1.7 are supported on Android 5.0 Lollipop OS or later, and iOS will be supported in the future. Device Manager v1.1.3 and earlier versions are supported on Android 5.0 Lollipop OS or later, and iOS 9.0 or later.

:::
