El Módulo de Entradas admite entradas supervisadas (12 canales), relé (2 canales), entrada AUX (2 canales) y entrada de sabotaje (1 canal), y puede conectarse a un único dispositivo maestro en una cadena RS-485 de hasta 31 dispositivos. Puede utilizarse junto con sensores de detección o puertas de acceso y puede integrarse fácilmente con otros sistemas mediante el método de contacto seco.
