FaceStation 2 es el dispositivo de autenticación facial de próxima generación de Suprema, equipado con el algoritmo de autenticación facial de mayor rendimiento del mundo. Ofrece un rendimiento de autenticación ultrarrápido, capaz de verificar hasta 4,000 individuos por segundo mediante un potente procesador de cuatro núcleos.
