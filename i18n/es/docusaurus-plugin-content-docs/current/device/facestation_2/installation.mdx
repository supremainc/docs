Ofrece los procedimientos completos de instalación y ejemplos de conexión necesarios para el dispositivo.

## Fijar el soporte y el producto

### Soporte general

1. Determine la posición correcta para instalar el soporte usando la plantilla de perforación proporcionada.

   <Image src="/img/device/installation-recommended-fs2.svg" className="none" alone />

   <InstallationBracketNotes height={110} />

2. Asegura firmemente el soporte con los tornillos de fijación en el lugar donde se montará el producto.

   <Image src="/img/device/installation-device1-fs2.svg" className="none" alone />

   :::info

   * Si instalas el producto en una pared de concreto, perfora un agujero, inserta un anclaje de PVC y sujétalo con un tornillo de fijación.

   * Para evitar interferencias de RF, se debe mantener una distancia mínima de separación.

   <Columns>
     <Column className="text--center">
       <Image src="/img/device/installation-wall1-fs2.png" className="none" width="60%" />

       <div className="overflow-x">
         | Espesor de pared | Distancia |
         | :-: | :-: |
         | 100 mm | 283 mm |
         | 120 mm | 275 mm |
         | 150 mm | 260 mm |
       </div>
     </Column>

     <Column className="text--center">
       <Image src="/img/device/installation-wall2-fs2.png" className="none" width="60%" />
     </Column>
   </Columns>

   * Al utilizar una credencial de acceso móvil, instala los dispositivos manteniendo una distancia mínima de 1 m entre ellos para evitar interferencias BLE.

   :::

3. Monta el producto en el soporte fijado.

   <Image src="/img/device/installation-device2-fs2.svg" className="none" alone />

4. Gira los tornillos de fijación para ensamblar el producto con el soporte.

   <Image src="/img/device/installation-device3-fs2.svg" className="none" alone />

   :::info

   Al ensamblar el producto con el soporte, puedes usar el tornillo de fijación del soporte incluido (en forma de estrella) en lugar del tornillo de fijación del producto para aumentar la seguridad.

   :::

### Soporte inclinable

Este producto puede instalarse para permitir ajustes de ángulo durante la certificación usando un soporte inclinable dedicado. El soporte inclinable dedicado se vende por separado.

1. Determine la posición correcta para instalar el soporte usando la plantilla de perforación proporcionada.

   <Image src="/img/device/installation-recommended-fs2.svg" className="none" alone />

   <InstallationBracketNotes height={110} />

2. Fije firmemente el soporte inclinable en el lugar donde se montará el producto utilizando los tornillos diseñados para el soporte inclinable.

   <Image src="/img/device/installation-device-tilt_bracket1-fs2.svg" className="none" alone />

   :::info

   * Si instalas el producto en una pared de concreto, perfora un agujero, inserta un anclaje de PVC y sujétalo con un tornillo de fijación.

   * Para evitar interferencias de RF, se debe mantener una distancia mínima de separación.

   <Columns>
     <Column className="text--center">
       <Image src="/img/device/installation-wall1-fs2.png" className="none" width="60%" />

       <div className="overflow-x">
         | Espesor de pared | Distancia |
         | :-: | :-: |
         | 100 mm | 283 mm |
         | 120 mm | 275 mm |
         | 150 mm | 260 mm |
       </div>
     </Column>

     <Column className="text--center">
       <Image src="/img/device/installation-wall2-fs2.png" className="none" width="60%" />
     </Column>
   </Columns>

   * Al utilizar una credencial de acceso móvil, instala los dispositivos manteniendo una distancia mínima de 1 m entre ellos para evitar interferencias BLE.

   :::

3. Fije firmemente el soporte encima del soporte inclinable.

   <Image src="/img/device/installation-device-tilt_bracket2-fs2.svg" className="none" alone />

4. Monta el producto en el soporte fijado.

   <Image src="/img/device/installation-device-tilt_bracket3-fs2.svg" className="none" alone />

5. Gira los tornillos de fijación para ensamblar el producto con el soporte.

   <Image src="/img/device/installation-device-tilt_bracket4-fs2.svg" className="none" alone />

   :::info

   Al ensamblar el producto con el soporte, puedes usar el tornillo de fijación del soporte incluido (en forma de estrella) en lugar del tornillo de fijación del producto para aumentar la seguridad.

   :::
