## Nombre y función de cada parte

<Image src="/img/device/name-function-each-part-xpd2.png" width="80%" className="none" />

* **Indicador LED**: Indica el estado operativo del dispositivo mediante el color del LED.

  * Verde: autenticación exitosa.

  * Rojo: autenticación fallida.

  * Azul/cian intermitente: funcionamiento normal.

  * Parpadeo azul/verde: durante la inicialización de la configuración de red

  * Verde intermitente: esperando entrada.

  * Parpadeo verde/blanco: conexión a la aplicación Suprema AirFob Pass (BLE)

  * Parpadeo blanco: conexión a la aplicación Device Manager

* **Unidad de autenticación de tarjeta RF y tarjeta de acceso móvil**: parte para escanear una tarjeta RFID o una tarjeta de acceso móvil para la entrada.

* **Área de reconocimiento de tarjeta de acceso móvil**: esta es la parte que reconoce tarjetas de acceso móvil para la entrada.

* **Botón de restablecimiento**: elimina toda la información y los certificados almacenados e inicializa la configuración. Para obtener más información sobre la inicialización de fábrica, consulta [lo siguiente](installation#factory-reset).

* **Cable**

  Alimentación / RS-485 / Salida Wiegand / Entrada LED / Entrada de beep / Tamper

* **Teclado**: <Badge only>XPD2-GKDB</Badge> Se usa para autenticar ingresando directamente la ID de la tarjeta. Cambia la configuración del método de autenticación en BioStar X para autenticar mediante el modo de entrada directa.

  * **0-9**: se utiliza al ingresar números.

  * <Image src="/img/common/ico-star.svg" ico alone />: se utiliza para distinguir entre el código FC y el ID al ingresar el ID de tarjeta Wiegand.

  * <Image src="/img/common/ico-pound.svg" ico alone />: se utiliza para finalizar la entrada.
