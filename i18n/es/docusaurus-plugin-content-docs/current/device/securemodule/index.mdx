Secure Module es un controlador independiente para gestionar un relé de puerta externo y ampliar la entrada/salida. Se puede usar con dispositivos de control de acceso IP de Suprema para comunicación cifrada, mejorando la seguridad y proporcionando una solución optimizada para el entorno de uso.
