## Nombre y función de cada parte

<Image src="/img/device/name-function-each-part-bs3.png" className="none" width="60%" />

* **Micrófono**: Transmite la voz del usuario cuando el interfono está conectado.

  * Cubrir u obstruir el micrófono puede interferir con la transmisión de la voz.

* **Cámara infrarroja**: Captura la imagen facial infrarroja.

* **Cámara visual**: Captura una imagen facial real.

* **Pantalla**: Proporciona una interfaz de usuario para la operación.

* **Unidad de autenticación de tarjeta RF y tarjeta de acceso móvil**: parte para escanear una tarjeta RFID o una tarjeta de acceso móvil para la entrada.

* **Indicador LED**: Indica el estado operativo del dispositivo mediante el color del LED.

  * Verde: autenticación exitosa.

  * Rojo: autenticación fallida.

  * Azul: Operación normal.

  * Azul/verde intermitente:

    * Al configurar <Cmd sid="ui_012" product="dev" /> para su uso, no se puede obtener la dirección IP.

    * Cuando está en modo <Cmd sid="ui_205" product="dev" />, la conexión con el dispositivo maestro se desconecta.

  * Parpadeo rojo/morado: El dispositivo está bloqueado.

  * Parpadeo amarillo: Esperando entrada.

* **Altavoz**: Emite sonido.

* **Ethernet**: Conecta el cable Ethernet.

* **Entrada (5 pines)**: Conecta el cable de entrada.

* **Puerto de expansión USB (5 pines)**: Conecta otros productos accesorios.

* **Salida Wiegand (4 pines)**: Conecta el cable de salida Wiegand.

* **Entrada Wiegand (4 pines)**: Conecta el cable de entrada Wiegand.

* **RS-485 (4 pines)**: Conecta el cable RS-485.

* **Relé (3 pines)**: Conecta el cable del relé.

* **Alimentación (2 pines)**: Conecta el cable de alimentación.

* **Ranura para memoria USB (tipo C)**: Conecta la memoria USB.

## Nota sobre el uso de códigos QR y la licencia del dispositivo

### Cómo escanear un código QR

Emite un código QR a un usuario y úsalo como método de autenticación. Escanea correctamente el código QR siguiendo las instrucciones en la pantalla para aumentar la tasa de reconocimiento.

<Image src="/img/device/qr-scanning-bs3.svg" className="none" alone />

* Mantén la distancia entre la cámara del dispositivo y el código QR a 30~40 cm.

* Coloca el código QR en el centro de la pantalla de escaneo del dispositivo.

* Al autenticar con el código QR descargado en tu smartphone, ajusta el brillo de tu smartphone para que el dispositivo pueda leer el código QR con claridad.

### Notas sobre la licencia del dispositivo

* El uso de la autenticación mediante código QR requiere una licencia de dispositivo aparte. Contacta al lugar de compra para obtener una licencia de dispositivo.

* El archivo de licencia del dispositivo está cifrado y no puede modificarse de forma arbitraria.

* Las licencias de dispositivo se emiten según el ID del dispositivo. Si el ID del dispositivo se cambia de manera inusual, no se proporciona servicio de garantía para la licencia.
