BioStation 3 establece un nuevo estándar en el control de acceso y ofrece una nueva experiencia de acceso a puertas que mejora la seguridad en todas las puertas para organizaciones de todos los tamaños. La terminal es más pequeña que el modelo anterior, pero ofrece la mayor variedad de funciones.
