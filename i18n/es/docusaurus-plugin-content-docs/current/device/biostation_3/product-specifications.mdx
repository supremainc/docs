:::note

<Anno id="anno1">1)</Anno> La distancia mínima de autenticación facial indicada es la configuración predeterminada del producto y la distancia recomendada de autenticación facial es de 0.6 a 1.0 m.

<Anno id="anno2">2)</Anno> Las tarjetas DESFire EV2/EV3 son compatibles con las tarjetas DESFire EV1 y admiten las funciones CSN y de tarjeta inteligente, incluidas las de la tarjeta DESFire EV1.

<Anno id="anno3">3)</Anno> El alcance de lectura RF variará según el entorno de instalación.

<Anno id="anno4">4)</Anno> Admite códigos QR/códigos de barras compuestos por cadenas de hasta 32 caracteres ASCII.

<Anno id="anno5">5)</Anno> Requiere una licencia aparte.

<Anno id="anno6">6)</Anno> Número de usuarios registrados sin datos de credenciales.

<Anno id="anno7">7)</Anno> Requiere alimentación PoE+ (IEEE 802.3at, 30 W) o PoE++ (IEEE 802.3bt, 60-100 W). No se admite PoE estándar (IEEE 802.3af, 15.4 W). El uso de PoE estándar puede provocar que los dispositivos funcionen de forma inestable o sufran daños permanentes. Antes de conectar el dispositivo, asegúrese de que su switch de red o inyector PoE sea compatible con PoE+ o PoE++.

:::

:::info

* Las especificaciones del producto están sujetas a cambios sin previo aviso para mejorar el rendimiento.

* Cuando este producto funciona como dispositivo maestro, conectar un dispositivo con sensor de huellas como esclavo permite autenticar huellas mediante el dispositivo esclavo.

  * Máx. Credenciales de huellas digitales (1:N): 100,000

  * Máx. Credenciales de huella digital (1:1): 100,000

:::
