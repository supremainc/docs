Proporciona orientación sobre los menús principales y las funciones accesibles con privilegios de administrador.

## TABLERO

1. Presione <Image src="/img/common/ico-menu.svg" ico alone /> y autentíquese con la credencial de nivel Administrador.<br />Verifique el estado y la información del dispositivo en <Cmd sid="ui_109" product="dev" />. Como alternativa, presione <Cmd sid="ui_174" product="dev" /> para registrar directamente a un usuario.

2. Seleccione el menú deseado en la parte inferior del panel de control para registrar un usuario o configurar un dispositivo.

   <Image src="/img/device/bs3-menu.png" />

:::info

* Si no se ha designado un administrador, al presionar <Image src="/img/common/ico-menu.svg" ico alone /> se mostrará la pantalla de <Cmd sid="ui_109" product="dev" />.

* <Cmd sid="ui_142" product="dev" /> solo se puede confirmar en el modelo BS3-APWB.

:::
