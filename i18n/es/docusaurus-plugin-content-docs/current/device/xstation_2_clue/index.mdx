El X-Station 2 es un port inteligente integrado para control de acceso y gestión de asistencia, diseñado para ofrecer amplia compatibilidad con tarjetas RFID y autenticación Fingerprint rápida y precisa. Usa una cámara integrada para leer códigos QR y códigos de barras, y además admite de forma opcional tarjetas de acceso móviles. Secure Boot, la comunicación OSDP sobre RS-485 y los registros de imágenes mejoran la seguridad y la confiabilidad operativa, mientras que Power over Ethernet (PoE), una pantalla táctil capacitiva de 4 pulgadas y un diseño con clasificación IP65 simplifican la instalación.
