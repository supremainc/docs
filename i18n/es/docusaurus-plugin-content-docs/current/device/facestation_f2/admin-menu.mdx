## Menú completo

1. Presione <Image src="/img/common/ico-menu.svg" ico alone /> y autentíquese con la credencial de nivel de administrador.

2. Selecciona el menú deseado.

Si no se designa un administrador, haz clic en <Image src="/img/common/ico-menu.svg" ico alone /> para mostrar la pantalla del menú.

:::
