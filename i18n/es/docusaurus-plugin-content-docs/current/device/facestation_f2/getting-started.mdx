## Nombre y función de cada parte

<Image src="/img/device/name-function-each-part-fsf2.png" className="none" width="80%" />

* **Lámpara LED**: La lámpara LED se enciende cuando detecta a un usuario en la oscuridad.

* **Cámara infrarroja**: Captura la imagen facial infrarroja.

* **Cámara visual**: Captura una imagen facial real.

* **Pantalla**: Proporciona una interfaz de usuario para la operación.

* **Unidad de autenticación de tarjeta RF y tarjeta de acceso móvil**: parte para escanear una tarjeta RFID o una tarjeta de acceso móvil para la entrada.

* **Unidad de reconocimiento de huellas**: <Badge only>FSF2-ODB</Badge> La parte que reconoce la huella para el acceso.

* **Ethernet**: Conecta el cable Ethernet.

* **Entrada (4 pines)**: Conecta el cable de entrada.

* **Entrada/salida Wiegand (4 pines)**: Conecta el cable Wiegand de entrada/salida.

* **RS-485 (4 pines)**: Conecta el cable RS-485.

* **Altavoz**: Emite sonido.

* **Relé (3 pines)**: Conecta el cable del relé.

* **Alimentación (2 pines)**: Conecta el cable de alimentación.

* **Ranura para tarjeta SAM**: Se puede usar una tarjeta SAM.

* **Ranura para memoria USB (tipo C)**: Conecta la memoria USB.
