Proporciona instrucciones para configurar los ajustes del sistema y del dispositivo.

## Pantalla y sonido

Cambie los ajustes de pantalla y sonido del dispositivo.

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Presione <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_366' product='dev' />.

3. Cambie el elemento deseado.

   <Image src='/img/device/bs3-displaysound-clue.png' />

   * <Cmd sid='ui_364' product='dev' />: Permite elegir los elementos que se mostrarán en el fondo de la pantalla de inicio.

   * <Cmd sid='ui_388' product='dev' />: Indica el estado del modo CLUe.

   * <Cmd sid='ui_389' product='dev' />: Seleccione si usar el botón de entrada.

   * <Cmd sid='ui_227' product='dev' />: Permite seleccionar el idioma que se mostrará en la pantalla.

   * <Cmd sid='ui_284' product='dev' />: Configure si se utiliza un mensaje privado que se mostrará en la pantalla cuando el usuario se autentique. <Cmd sid='ui_284' product='dev' /> se puede configurar mediante el SDK de dispositivos BioStar. Si no se configura, no se mostrará ningún mensaje cuando la autenticación sea correcta, incluso si <Cmd sid='ui_284' product='dev' /> está habilitado en el dispositivo.

   * <Cmd sid='ui_184' product='dev' />: Configure si se utiliza un mensaje privado que se mostrará en la pantalla cuando el usuario se autentique. <Cmd sid='ui_184' product='dev' /> se puede configurar mediante la API de BioStar X. Si no se configura, no se mostrará ningún mensaje cuando la autenticación sea correcta, incluso si <Cmd sid='ui_184' product='dev' /> está habilitado en el dispositivo.

   * <Cmd sid='ui_365' product='dev' />: Configure si el protector de pantalla está habilitado. Reduce el consumo de energía innecesario al disminuir el brillo de la pantalla cuando el dispositivo no se utiliza.

   * <Cmd sid='ui_131' product='dev' />: Configure el tiempo (s) para que la pantalla de menú desaparezca automáticamente. Si no se pulsa ningún botón durante el tiempo establecido, la pantalla regresará a la pantalla de inicio.

   * <Cmd sid='ui_135' product='dev' />: Configure el tiempo (s) para que un mensaje de configuración completada o de notificación desaparezca automáticamente.

   * <Cmd sid='ui_148' product='dev' />: Configure el tiempo (s) para apagar la iluminación de la pantalla.

   * <Cmd sid='ui_251' product='dev' />: Permite usar guía por voz en lugar de tonos de notificación.

   * <Cmd sid='ui_153' product='dev' />: Configure el volumen.

   * <Cmd sid='ui_028' product='dev' />: Ajuste el volumen de salida del altavoz cuando use el intercomunicador.

   * <Cmd sid='ui_026' product='dev' />: Ajuste el volumen de entrada del micrófono cuando use el intercomunicador.

:::info

Para obtener más información sobre el SDK de dispositivos BioStar, consulte la [página del SDK de dispositivos BioStar](https://kb.supremainc.com/bs2sdk/doku.php) en la base de conocimientos de Suprema.

:::

## Red

Cambie la configuración de comunicación de dispositivos como Ethernet, servidor y comunicación serial.

### Ethernet

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Seleccione <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_344' product='dev' /> → <Cmd sid='ui_252' product='dev' />.

3. Cambie el elemento deseado.

   <Image src='/img/device/bs3-network-ethernet-clue.png' />

   * <Cmd sid='ui_012' product='dev' />: Configure si se utiliza DHCP. Si la configuración DHCP está deshabilitada, se pueden modificar <Cmd sid='ui_029' product='dev' />, <Cmd sid='ui_068' product='dev' /> y <Cmd sid='ui_192' product='dev' />.

   * <Cmd sid='ui_290' product='dev' />: Configure el puerto del dispositivo.

   * <Cmd sid='ui_029' product='dev' />: Compruebe la dirección IP del dispositivo. Para modificarla, desactive la configuración de DHCP.

   * <Cmd sid='ui_068' product='dev' />: Compruebe la puerta de enlace del dispositivo. Para modificarla, desactive la configuración de DHCP.

   * <Cmd sid='ui_192' product='dev' />: Compruebe la máscara de subred del dispositivo. Para modificarla, desactive la configuración de DHCP.

   * <Cmd sid='ui_111' product='dev' />: Configure la dirección del servidor DNS.

   * <Cmd sid='ui_391' product='dev' />: Ingrese la dirección para la prueba de ping.

   * <Cmd sid='ui_392' product='dev' />: Ingrese el tiempo máximo de espera para la prueba de ping.

   * <Cmd sid='ui_390' product='dev' />: Inicia la prueba de Ping.

      :::info

      Seleccione un servidor DNS abajo para ejecutar la prueba de Ping.

      :::

4. Pulsa <Cmd sid='ui_037' product='dev' /> para guardar la configuración.

:::info

<Cmd sid='ui_252' product='dev' /> no se puede usar con <Cmd sid='ui_141' product='dev' />.

:::

### Inalámbrico

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Selecciona <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_344' product='dev' /> → <Cmd sid='ui_141' product='dev' />.

3. Se mostrará la lista de redes disponibles para conectarse.

   <Image src='/img/device/bs3-network-wifi-clue.png' />

4. Seleccione la red a la que desea conectarse e ingrese la contraseña. Al presionar <Cmd sid='ui_037' product='dev' />, se establecerá la conexión a la red inalámbrica.

5. Presiona <Image src='/img/common/ico-wireless-info.svg' ico alone /> en la red inalámbrica que deseas usar y luego desactiva la configuración DHCP para permitir que los usuarios modifiquen <Cmd sid='ui_029' product='dev' />, <Cmd sid='ui_068' product='dev' /> y <Cmd sid='ui_192' product='dev' />.

:::info

* <Cmd sid='ui_141' product='dev' /> solo está disponible para el modelo BS3-APWB.

* <Cmd sid='ui_141' product='dev' /> no se puede usar con <Cmd sid='ui_252' product='dev' />.

* Se requiere un enrutador inalámbrico para conectar <Cmd sid='ui_141' product='dev' />. Para obtener información sobre la instalación y configuración del enrutador inalámbrico, consulte el manual del usuario del enrutador.

:::

### Servidor

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Seleccione <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_344' product='dev' /> → <Cmd sid='ui_180' product='dev' />.

3. Cambie el elemento deseado.

   <Image src='/img/device/bs3-network-server.png' />

   * <Cmd sid='ui_283' product='dev' />: Envíe directamente una señal de conexión desde el dispositivo a un servidor con la información ingresada. Cuando <Cmd sid='ui_283' product='dev' /> está habilitado, el usuario puede ingresar <Cmd sid='ui_182' product='dev' />, <Cmd sid='ui_183' product='dev' /> y <Cmd sid='ui_189' product='dev' />.

      :::info

      Cuando usa el modo Suprema CLUe, el setting <Cmd sid='ui_283' product='dev' /> queda fijo.

      :::

   * <Cmd sid='ui_182' product='dev' />: Para conectarse a Suprema CLUe, ingrese `127.0.0.1`.

   * <Cmd sid='ui_183' product='dev' />: Ingrese <Cmd sid='ui_183' product='dev' /> en lugar de <Cmd sid='ui_182' product='dev' />.

   * <Cmd sid='ui_189' product='dev' />: Para conectarse a Suprema CLUe, ingrese `51212`.

4. Pulsa <Cmd sid='ui_037' product='dev' /> para guardar la configuración.

### Configuración de comunicación serial

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Selecciona <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_344' product='dev' />.

3. Establezca la opción deseada en <Cmd sid='ui_211' product='dev' />.

   <Image src='/img/device/bs3-network-osdp-clue.png' />

   * <Cmd sid='ui_049' product='dev' />: Seleccione el modo RS-485.

   * <Cmd sid='ui_297' product='dev' />: Seleccione la velocidad en baudios deseada.

   * <Cmd sid='ui_205' product='dev' />: Muestra una lista de dispositivos conectados como esclavos. Al hacer clic en <Image src='/img/common/ico-add.svg' ico alone /> se busca un nuevo dispositivo esclavo y se agrega automáticamente a la lista.

      :::info

      El menú <Cmd sid='ui_205' product='dev' /> solo está disponible en el dispositivo maestro.

      :::

## Dispositivo

### Intercomunicador IP

Configura la cuenta y las extensiones del intercomunicador.

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Seleccione <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_025' product='dev' />.

   <Image src='/img/device/bs3-network-intercom-clue.png' />

   * <Cmd sid='ui_025' product='dev' />: Configure si el intercomunicador está habilitado. Al habilitar la configuración del intercomunicador, puede configurar <Cmd sid='ui_055' product='dev' /> y <Cmd sid='ui_101' product='dev' />.

   * <Cmd sid='ui_055' product='dev' />: Ingrese cada dato para que el dispositivo se registre en el servidor SIP.

   * <Cmd sid='ui_101' product='dev' />: Agregue, elimine o edite números de extensión.

   * <Cmd sid='ui_102' product='dev' />: Configure si se mostrarán los números de extensión.

   :::info

   * Si <Cmd sid='ui_356' product='dev' /> no está configurado, no será posible distinguir los receptores.

   * Los números de extensión solo pueden incluir caracteres alfanuméricos (distingue mayúsculas y minúsculas) y los caracteres especiales `+` `-` `@` `.`.

   :::

3. Pulsa <Cmd sid='ui_037' product='dev' /> para guardar la configuración.

### Relé

Configure los puertos de entrada de <Cmd sid='ui_144' product='dev' /> y <Cmd sid='ui_346' product='dev' /> en el dispositivo.

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Seleccione <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_122' product='dev' />.

   <Image src='/img/device/bs3-network-relay.png' />

   * <Cmd sid='ui_122' product='dev' />: Configure si el relé está habilitado. Para configurar <Cmd sid='ui_144' product='dev' /> y <Cmd sid='ui_346' product='dev' />, active esta opción.

   * <Cmd sid='ui_144' product='dev' />: Establece la duración que la puerta permanece abierta cuando se ha realizado la autenticación de usuario estándar.

   * <Cmd sid='ui_346' product='dev' />: Seleccione el puerto de entrada al que está conectado el botón de salida.

   * <Cmd sid='ui_274' product='dev' />: Seleccione el tipo de interruptor (<Cmd sid='ui_036' product='dev' /> o <Cmd sid='ui_035' product='dev' />) para el puerto de entrada del botón de salida.

3. Cambie el elemento deseado.

   <Image src='/img/device/bs3-datetime.png' />

   * <Cmd sid='ui_098' product='dev' />: Compruebe la fecha y hora actuales. Para modificarla manualmente, desactive <Cmd sid='ui_208' product='dev' />.

   * <Cmd sid='ui_358' product='dev' />: Ajuste la referencia horaria de la ubicación actual.

   * <Cmd sid='ui_208' product='dev' />: Sincronice el servidor y la hora. Si desea sincronizar el servidor y la hora, habilite <Cmd sid='ui_208' product='dev' />.

   * <Cmd sid='ui_097' product='dev' />: Configure el formato de fecha. Seleccione <Cmd sid='ui_062' product='dev' />, <Cmd sid='ui_033' product='dev' /> y <Cmd sid='ui_009' product='dev' />.

   <Image src='/img/device/bs3-dst.png' />

   * <Cmd sid='ui_217' product='dev' /> / <Cmd sid='ui_218' product='dev' /> / <Cmd sid='ui_216' product='dev' /> / <Cmd sid='ui_215' product='dev' />: Configure los detalles sobre cuándo comienza el horario de verano.

   * <Cmd sid='ui_094' product='dev' /> / <Cmd sid='ui_095' product='dev' /> / <Cmd sid='ui_093' product='dev' /> / <Cmd sid='ui_092' product='dev' />: Configure los detalles sobre cuándo termina el horario de verano.

### Memoria USB

Conecta una memoria USB e importa la información de usuarios al dispositivo o exporta registros e información de usuarios desde el dispositivo. También puede actualizar el firmware.

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Seleccione <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_061' product='dev' />.

3. Selecciona el elemento deseado y cambia el ajuste.

   <Image src='/img/device/bs3-usb.png' />

   * <Cmd sid='ui_063' product='dev' />: Seleccione los elementos que desea importar desde la memoria USB conectada.

     * <Cmd sid='ui_166' product='dev' />: Importe la información de usuarios.

     * <Cmd sid='ui_393' product='dev' />: Copie archivos de imagen al dispositivo.

        :::info

        Puede copiar hasta 5 archivos de imagen a la vez.

        :::

   * <Cmd sid='ui_100' product='dev' />: Seleccione la información que desea exportar a la memoria USB conectada.

     * <Cmd sid='ui_137' product='dev' />: Exporte toda la información.

     * <Cmd sid='ui_166' product='dev' />: Exporte la información de usuarios.

     * <Cmd sid='ui_394' product='dev' />: Exporte la información de registros.

   * <Cmd sid='ui_353' product='dev' />: Actualice el firmware si los archivos de firmware están guardados en la memoria USB conectada.

:::caution

No desconecte la fuente de alimentación mientras se actualiza el firmware del dispositivo. El dispositivo puede funcionar incorrectamente.

:::

:::info

* Se recomienda usar una unidad flash USB formateada en exFAT. Las unidades flash USB en formato FAT32 pueden funcionar incorrectamente cuando se almacenan más de 50,000 credenciales faciales.

* Verifica la cantidad de espacio libre en la unidad flash USB. Se requieren al menos 15 GB de espacio libre para guardar 100,000 credenciales faciales.

:::

### Sabotaje

Configura si se utiliza el tamper de seguridad. Cuando el tamper está activado, establece la eliminación de todos los usuarios, todos los registros y la clave de seguridad.

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Selecciona <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_280' product='dev' />.

3. Active <Cmd sid='ui_150' product='dev' /> para eliminar los datos de todos los usuarios registrados, todos los registros y las claves seguras cuando ocurra un evento de manipulación.

### RTSP

Conecta un sistema de gestión de video (VMS) de terceros para monitorear los videos filmados con la cámara visual del producto.

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Seleccione <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_051' product='dev' />.

3. Selecciona el elemento deseado y cambia el ajuste.

   <Image src='/img/device/bs3-rtsp-clue.png' />

   * <Cmd sid='ui_051' product='dev' />: Configure si RTSP está habilitado o no. Cuando el ajuste RTSP está activado, se pueden configurar <Cmd sid='ui_020' product='dev' />, <Cmd sid='ui_155' product='dev' />, <Cmd sid='ui_053' product='dev' /> y <Cmd sid='ui_054' product='dev' />.

   * <Cmd sid='ui_020' product='dev' />: Ingrese el ID. El ID puede tener hasta 32 caracteres, en una combinación de caracteres alfanuméricos (distingue mayúsculas y minúsculas) y los símbolos `-` `_`.

   * <Cmd sid='ui_155' product='dev' />: Ingrese la contraseña. La contraseña puede tener de 8 a 32 caracteres en una combinación de letras (mayúsculas y minúsculas) y números.

   * <Cmd sid='ui_053' product='dev' />: Ingrese la dirección RTSP.

   * <Cmd sid='ui_054' product='dev' />: Ingrese el puerto RTSP.

:::info

Cuando el dispositivo está configurado como dispositivo esclavo, es posible que la función RTSP no esté disponible.

:::

### Reiniciar dispositivo

Reinicia el dispositivo.

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Seleccione <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_286' product='dev' />.

3. Presiona <Cmd sid='ui_367' product='dev' /> para reiniciar el dispositivo. Presiona <Cmd sid='ui_321' product='dev' /> para regresar a la pantalla anterior.

### Restaurar valores predeterminados

Los ajustes del dispositivo, los ajustes de red y los niveles de operador se restablecerán.

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

2. Seleccione <Cmd sid='ui_193' product='dev' /> → <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_089' product='dev' />.

   * <Cmd sid='ui_304' product='dev' />: Inicializa todos los ajustes almacenados en el dispositivo. Presione <Cmd sid='ui_367' product='dev' /> para inicializar todos los ajustes del dispositivo. Presiona <Cmd sid='ui_321' product='dev' /> para regresar a la pantalla anterior.

   * <Cmd sid='ui_104' product='dev' />: Inicializa todos los ajustes excepto los de red. Presione <Cmd sid='ui_367' product='dev' /> para inicializar todos los ajustes excepto los de red. Presiona <Cmd sid='ui_321' product='dev' /> para regresar a la pantalla anterior.

   * <Cmd sid='ui_072' product='dev' />: Elimina toda la información guardada en el dispositivo y el certificado raíz, y restaura los ajustes predeterminados.

   * <Cmd sid='ui_121' product='dev' />: Elimina el certificado raíz guardado en el dispositivo.

3. Si continúas con <Cmd sid='ui_089' product='dev' />, el dispositivo se reiniciará.

:::info

* Al restablecer, el nivel de operador también se restablecerá. Después de restablecer, vuelve a configurar el nivel de operador.

* El ajuste de idioma no cambiará después de restablecer.

* El menú <Cmd sid='ui_072' product='dev' /> puede utilizarse cuando el certificado raíz está guardado en el dispositivo.

* El menú <Cmd sid='ui_121' product='dev' /> puede utilizarse solo cuando el certificado raíz está guardado en el dispositivo y se ha designado un administrador.

:::
