## Tablero

:::info

Cuando se usa el modo Suprema CLUe, el enroll, la edición y la eliminación de usuarios están disponibles solo en el servidor Suprema CLUe, no en el dispositivo.

:::

1. Presione <Image src='/img/common/ico-menu.svg' ico alone />, ingrese la contraseña y entre al menú.

   Verifique el estado y la información del dispositivo en <Cmd sid='ui_109' product='dev' />. Como alternativa, presione <Cmd sid='ui_174' product='dev' /> para registrar directamente a un usuario.

2. Seleccione el menú deseado en la parte inferior del panel de control para registrar un usuario o configurar un dispositivo.

   <Image src='/img/device/bs3-menu.png' />

:::info

* Si <Cmd sid='ui_387' product='dev' /> no está configurado, al presionar <Image src='/img/common/ico-menu.svg' ico alone /> se abre la pantalla <Cmd sid='ui_109' product='dev' />. Para obtener detalles sobre cómo configurar la contraseña, consulta [Contraseña del menú](settings#menu-password).

* <Cmd sid='ui_142' product='dev' /> solo se puede confirmar en el modelo BS3-APWB.

:::
