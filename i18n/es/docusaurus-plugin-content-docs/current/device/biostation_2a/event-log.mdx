Describe cómo ver los registros de eventos generados por el dispositivo.

## Busca registros de eventos

Configura condiciones para buscar registros.

1. Presione <Image src="/img/common/ico-menu1.svg" ico alone /> y autentíquese con la credencial de administrador.

2. Seleccione <Cmd sid="ui_255" product="dev" /> → <Cmd sid="ui_066" product="dev" /> y luego presione **OK**.

   <Image src="/img/device/bs2a-eventlog.png" />

3. Seleccione el elemento deseado y presione <IcFw />/<IcBw /> para cambiar la configuración. Al presionar **OK**, los registros que coinciden con las condiciones aparecen en la pantalla.

4. Presione **ESC** para regresar a la pantalla anterior.

## Eliminar todos los registros

Elimina todos los registros guardados.

1. Presione <Image src="/img/common/ico-menu1.svg" ico alone /> y autentíquese con la credencial de administrador.

2. Seleccione <Cmd sid="ui_255" product="dev" /> → <Cmd sid="ui_303" product="dev" /> y luego presione **OK**.

   <Image src="/img/device/bs2a-eventlog1.png" />

3. Pulsa **OK** para eliminar todos los registros. Presione **ESC** para regresar a la pantalla anterior.

## Ver uso de registros

Consulta el estado del uso de registros.

1. Presione <Image src="/img/common/ico-menu1.svg" ico alone /> y autentíquese con la credencial de administrador.

2. Seleccione <Cmd sid="ui_255" product="dev" /> → <Cmd sid="ui_120" product="dev" /> y luego presione **OK**. Consulta el uso de registros del dispositivo.

   <Image src="/img/device/bs2a-eventlog2.png" />
