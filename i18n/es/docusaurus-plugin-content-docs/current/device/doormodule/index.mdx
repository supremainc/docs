El Módulo de Puerta controla hasta 2 puntos de acceso mediante comunicación cifrada y proporciona 2 interfaces de lector Wiegand para integrarse con lectores de terceros. El Módulo de Puerta se puede utilizar en varias configuraciones de sistema mediante múltiples interfaces de E/S.
