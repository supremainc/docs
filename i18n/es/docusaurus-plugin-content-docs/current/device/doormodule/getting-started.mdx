## Nombre y función de cada parte

<Magnify src="/img/en/device/name-function-each-part-dm20.svg" width={800} zoomFactor={.4} mgWidth={500} mgHeight={500} />

:::info

* Presione el botón **INIT** durante más de 2 segundos, después de inicializar el producto vinculado al dispositivo, para conectarlo a otro dispositivo.

* Presione el botón **RESET** para reiniciar el producto.

* Consulta el estado del LED que aparece a continuación al utilizar Entrada Supervisada.

  * Naranja: cortocircuito

  * Rojo: corto

  * Verde: encendido

  * Apagado: apagado

:::

### Ejemplo de instalación

Este producto admite varias entradas y salidas, así como conexiones Wiegand, lo que mejora la escalabilidad con dispositivos Suprema y de terceros.

También está disponible la Entrada Supervisada, que puede verificar el estado de cortocircuito, corto, encendido y apagado.
