## Nombre y función de cada parte

<Image src="/img/device/name-function-each-part-bep2.png" className="none" width="60%" />

* **Indicador LED**: Indica el estado operativo del dispositivo mediante el color del LED.

  * Verde: autenticación exitosa.

  * Rojo: autenticación fallida.

  * Azul/cian intermitente: funcionamiento normal.

  * Azul/verde intermitente:

    * Al configurar <Cmd sid="ui_012" product="dev" /> para su uso, no se puede obtener la dirección IP.

    * Cuando está en modo <Cmd sid="ui_205" product="dev" />, la conexión con el dispositivo maestro se desconecta.

    * Al inicializar la configuración de red.

  * Verde intermitente: esperando entrada.

* **Sección de reconocimiento de tarjetas RF**: Esta es la parte que reconoce tarjetas RF para acceso.

* **Unidad de reconocimiento de huella dactilar**: La parte que reconoce la huella dactilar para acceso.

* **Alimentación (2 pines)**: Conecta el cable de alimentación.

* **Relé (3 pines)**: Conecta el cable del relé.

* **Ethernet (4 pines)**: Conecta el cable Ethernet.

* **RS-485 (4 pines)**: Conecta el cable RS-485.

* **Entrada (4 pines)**: Conecta el cable de entrada.

* **Entrada/salida Wiegand (4 pines)**: Conecta el cable Wiegand de entrada/salida.

* **Botón de reinicio**

  * Inicializa la configuración de red. Para obtener más información sobre la inicialización de la configuración de red, consulta [lo siguiente](installation#reset-network-settings).

  * Elimina toda la información y los certificados almacenados en el dispositivo e inicializa la configuración. Para obtener más información sobre la inicialización de fábrica, consulta [lo siguiente](installation#factory-reset).
