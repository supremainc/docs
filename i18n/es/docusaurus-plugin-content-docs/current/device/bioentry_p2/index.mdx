BioEntry P2 es un dispositivo de control de acceso compacto basado en la tecnología biométrica de próxima generación y la plataforma de seguridad de Suprema. Equipado con una CPU de 1.0 GHz líder en su clase, ofrece un rendimiento de autenticación ultrarrápido y un inicio veloz, mientras aplica el último algoritmo de reconocimiento de huellas dactilares de Suprema para mejorar el rendimiento. Además, mediante la tecnología de tarjetas multi-RFID que puede leer frecuencias de 125 kHz y 13.56 MHz, puede reconocer diversos estándares como MIFARE, HID iCLASS, DESFire, FeliCa, HID Prox y EM con un solo dispositivo. Diseñado en un formato tipo montante, el BioEntry P2 puede instalarse perfectamente en espacios estrechos como los marcos de las puertas.
