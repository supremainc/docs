# Firmware Version 1.4.1 (Build No. 1.4.1_260416)

Release<Separator/>2026-04-17

:::caution

- After upgrading the firmware to v1.4.0, it is not able to downgrade to an earlier version of the firmware.

- XS2-APB, XS2-QAPB and XS2-OAPB types of X-Station 2 model support HID iClass card type through SE processor.<br/>For devices with the new SE processor, after upgrading the firmware to v1.1.4 or higher, you cannot downgrade to a lower version and downgrade the firmware customized in the lower version.<br/>Devices with existing SE processor can be downgraded to a lower version.

- For more information, check the serial number of the device and contact the [Suprema](https://supremainc.com).

:::

## New Features and Improvements

---

1. Supported an option to select QR authentication using the device camera upon **Camera QR** license activation on XS2-QDPB and XS2-QAPB models.

1. Supported EV2 Secure Messaging authentication and issuance for DESFire EV2 and EV3 cards.

1. Improved log display for unsupported events and errors on the device.

1. Provided **global** firmware for non-CE(EU) regions

   - Disabled the **Master Admin** and **Admin 2-step Authentication** features

   - Reduced the minimum length for PIN enrollment to 4 digits

1. Supported connecting the XPass Q2 model as a slave device.
