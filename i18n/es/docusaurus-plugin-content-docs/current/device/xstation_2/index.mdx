X-Station 2 es un terminal de acceso inteligente que admite varios métodos de autenticación, desde tarjetas de acceso móviles hasta códigos QR, códigos de barras, tarjetas RFID y huellas dactilares. Equipado con funciones de arranque seguro, OSDP y una cámara para el registro de imágenes, X-Station 2 mejora la seguridad de acceso mientras protege los datos de los usuarios. X-Station 2 es fácil de instalar, usar y mantener, ya que cuenta con PoE, una interfaz de usuario intuitiva con pantalla táctil de 4 pulgadas y clasificación IP65 para resistencia al agua y al polvo.
