## Nombre y función de cada parte

<Image src="/img/device/name-function-each-part-extm.png" className="none" width="50%" />

* **Ranura para tarjeta SAM**: Use tarjetas SAM (Módulo de Acceso Seguro).

* **Puerto de expansión USB (5 pines)**: conecta el dispositivo al módulo de expansión.

* **RS-485 (4 pines)**: Conecta el cable RS-485.

* **RS-232C (4 pines)**: conecta el cable RS-232C.

## Ejemplo de instalación

:::caution

La conexión del módulo de expansión no es compatible con el firmware estándar, por lo que se requiere un firmware personalizado aparte.

:::

Al montar el módulo de expansión en el Suprema se permite usar puertos RS-485 y RS-232C adicionales, así como tarjetas SAM.

Los dispositivos Suprema que pueden admitir el módulo de expansión son los siguientes.

* [**BioStation 3**](../biostation_3/index.mdx)

* [**BioStation 2a**](../biostation_2a/index.mdx)
