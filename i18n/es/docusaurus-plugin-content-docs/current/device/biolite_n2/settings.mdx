Proporciona instrucciones para configurar los ajustes del sistema y del dispositivo.

## Pantalla y sonido

Cambie los ajustes de pantalla y sonido del dispositivo.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_366" product="dev" /> y presione **6**.

3. Selecciona el elemento deseado y pulsa **4** o **6** para configurar el horario.

   <Image src="/img/device/bln2-displaysound.png" />

   * <Cmd sid="ui_364" product="dev" />: Permite elegir los elementos que se mostrarán en el fondo de la pantalla de inicio.

   * <Cmd sid="ui_148" product="dev" />: Configure el tiempo (s) para apagar la iluminación de la pantalla.

   * <Cmd sid="ui_131" product="dev" />: Configure el tiempo (s) para que la pantalla de menú desaparezca automáticamente. Si no se pulsa ningún botón durante el tiempo establecido, la pantalla regresará a la pantalla de inicio.

   * <Cmd sid="ui_135" product="dev" />: Configure el tiempo (s) para que un mensaje de configuración completada o de notificación desaparezca automáticamente.

   * <Cmd sid="ui_227" product="dev" />: Permite seleccionar el idioma que se mostrará en la pantalla.

   * <Cmd sid="ui_251" product="dev" />: Permite usar guía por voz en lugar de tonos de notificación.

   * <Cmd sid="ui_153" product="dev" />: Configure el volumen.

4. Pulsa **OK** para guardar la configuración.

## Dispositivo

### Fecha y hora {#date-time}

Configura la fecha y la hora. Configura la fecha y la hora con precisión para recopilar datos de registro precisos.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_098" product="dev" /> y presione **6**.

3. Selecciona el elemento deseado y pulsa **4** o **6** para configurar el horario.

   <Image src="/img/device/bln2-datetime.png" />

   * <Cmd sid="ui_096" product="dev" />: Verifique la fecha actual. Desactiva <Cmd sid="ui_208" product="dev" /> para modificarla manualmente.

   * <Cmd sid="ui_207" product="dev" />: Verifique la hora actual. Desactiva <Cmd sid="ui_208" product="dev" /> para modificarla manualmente.

   * <Cmd sid="ui_358" product="dev" />: Establece la referencia de hora de la ubicación actual.

   * <Cmd sid="ui_208" product="dev" />: Sincroniza el servidor y la hora. Habilita <Cmd sid="ui_208" product="dev" /> para sincronizar el servidor y la hora.

   * <Cmd sid="ui_097" product="dev" />: Establece el formato de fecha. Selecciona <Cmd sid="ui_062" product="dev" />, <Cmd sid="ui_033" product="dev" /> y <Cmd sid="ui_009" product="dev" />.

   * <Cmd sid="ui_209" product="dev" />: Establece el formato de hora. Selecciona <Cmd sid="ui_005" product="dev" /> y <Cmd sid="ui_238" product="dev" />.

4. Pulsa **OK** para guardar la configuración.

### Horario de verano

Aplica el horario de verano para usar el dispositivo. Configura correctamente la hora de inicio y fin.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_270" product="dev" /> y presione **6**.

3. Selecciona el elemento deseado y pulsa **4** o **6** para cambiar los ajustes.

   * <Cmd sid="ui_253" product="dev" />: Establece el nombre del horario de verano.

   * <Cmd sid="ui_217" product="dev" /> / <Cmd sid="ui_218" product="dev" /> / <Cmd sid="ui_216" product="dev" /> / <Cmd sid="ui_215" product="dev" />: Define los detalles de cuándo inicia el horario de verano.

   * <Cmd sid="ui_094" product="dev" /> / <Cmd sid="ui_095" product="dev" /> / <Cmd sid="ui_093" product="dev" /> / <Cmd sid="ui_092" product="dev" />: Define los detalles de cuándo finaliza el horario de verano.

4. Pulsa **OK** para guardar la configuración.

### Relé

Configure los puertos de entrada de <Cmd sid="ui_144" product="dev" /> y <Cmd sid="ui_015" product="dev" /> en el dispositivo. Útil cuando se utiliza como dispositivo independiente.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_122" product="dev" /> y presione **6**.

   * <Cmd sid="ui_122" product="dev" />: Configure si el relé está habilitado. Habilitar el ajuste de relé permite configurar <Cmd sid="ui_144" product="dev" /> y <Cmd sid="ui_015" product="dev" />.

   * <Cmd sid="ui_144" product="dev" />: Establece la duración que la puerta permanece abierta cuando se ha realizado la autenticación de usuario estándar.

   * <Cmd sid="ui_015" product="dev" />: Selecciona el puerto de entrada conectado al botón de salida.

   * <Cmd sid="ui_274" product="dev" />: Selecciona el tipo de interruptor (<Cmd sid="ui_036" product="dev" /> o <Cmd sid="ui_035" product="dev" />) para el puerto de entrada del botón de salida.

3. Pulsa **OK** para guardar la configuración.

### Información del dispositivo

Consulta el nombre del modelo, el ID del dispositivo, el número de serie, la versión del dispositivo y del firmware, el kernel y la dirección MAC.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_289" product="dev" /> y presione **6**. Verifica la información de <Cmd sid="ui_136" product="dev" />, <Cmd sid="ui_282" product="dev" />, <Cmd sid="ui_210" product="dev" />, <Cmd sid="ui_019" product="dev" />, <Cmd sid="ui_017" product="dev" />, <Cmd sid="ui_332" product="dev" />, <Cmd sid="ui_031" product="dev" />.

3. Presione **ESC** para regresar a la pantalla anterior.

### Uso de memoria

Consulta el estado de uso de la memoria.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_133" product="dev" /> y presione **6**. Consulta el estado de uso de memoria del dispositivo.

3. Presione **ESC** para regresar a la pantalla anterior.

### Sabotaje

Configura si se utiliza el tamper de seguridad. Cuando el tamper está activado, establece la eliminación de todos los usuarios, todos los registros y la clave de seguridad.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_150" product="dev" /> y presione **4** o **6** para cambiar la configuración.

3. Pulsa **OK** para guardar la configuración.

### Reiniciar dispositivo

Reinicia el dispositivo.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_286" product="dev" /> y presione **6**.

3. Presione **OK** para reiniciar el dispositivo. Pulsa **ESC** para cancelar.

### Restaurar valores predeterminados

Los ajustes del dispositivo, los ajustes de red y los niveles de operador se restablecerán.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_089" product="dev" /> y presione **6**.

   * <Cmd sid="ui_305" product="dev" />: Inicializa todos los ajustes guardados en el dispositivo. Presione **OK** para inicializar todas las configuraciones del dispositivo. Presione **ESC** para regresar a la pantalla anterior.

   * <Cmd sid="ui_160" product="dev" />: Inicializa todos los ajustes excepto los de red. Presione **OK** para inicializar todas las configuraciones excepto las de red. Presione **ESC** para regresar a la pantalla anterior.

   * <Cmd sid="ui_072" product="dev" />: Elimina toda la información guardada en el dispositivo y el certificado raíz, y restaura los ajustes a los valores predeterminados.

   * <Cmd sid="ui_121" product="dev" />: Elimina el certificado raíz guardado en el dispositivo.

3. Si continúas con <Cmd sid="ui_089" product="dev" />, el dispositivo se reiniciará.

:::info

* Al restablecer, el nivel de operador también se restablecerá. Después de restablecer, vuelve a configurar el nivel de operador.

* El ajuste de idioma no cambiará después de restablecer.

* El menú <Cmd sid="ui_072" product="dev" /> puede usarse cuando el certificado raíz está guardado en el dispositivo.

* El menú <Cmd sid="ui_121" product="dev" /> puede usarse solo cuando el certificado raíz está guardado en el dispositivo y se ha designado un administrador.

:::

### Administrador Principal

Cambie el método de autenticación del administrador maestro.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_128" product="dev" />.

3. Configure los métodos de autenticación deseados.

   <div className="grid-container">
     <Image src="/img/device/bln2-master-admin.png" width="100%" />

     <Image src="/img/device/bln2-master-admin-edit.png" width="100%" />
   </div>

:::info

* Debe enrolar al menos dos tipos de credenciales.

* Disponible solo en dispositivos nuevos con firmware v1.7.0 o posterior.

:::

### Autenticación en dos pasos del administrador {#admin-2-step-authentication}

Los dispositivos con firmware actualizado no ofrecen la configuración <Cmd sid="ui_128" product="dev" />; mejore la seguridad del dispositivo elevando los privilegios de administrador completo mediante la opción <Cmd sid="ui_073" product="dev" />.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_073" product="dev" />.

   <Image src="/img/device/bln2-administrator-two-step-auth.png" />

:::info

* <Cmd sid="ui_073" product="dev" /> está disponible después de actualizar el firmware a la versión v1.7.0 o posterior, y el firmware no puede degradarse a una versión anterior después de la actualización.

* El primer paso de autenticación es el predeterminado a menos que la opción <Cmd sid="ui_073" product="dev" /> esté habilitada.

* Si no hay dos tipos de credenciales enroladas para todos los administradores, la activación falla y se muestra un mensaje de error. Enrole dos tipos de credenciales para todos los administradores e inténtelo nuevamente.

  <Image src="/img/device/bln2-master-admin-error-message.png" />

:::

:::caution

Si elimina las credenciales de todos los administradores por debajo del tipo 2 después de habilitar <Cmd sid="ui_073" product="dev" />, ya no tendrá acceso al menú de administrador ni podrá conectarse a BioStar X. Por lo tanto, tenga especial cuidado al eliminar las credenciales de administrador.

:::

### Tipo de ID de usuario

Selecciona el tipo de ID de usuario que se va a registrar en el dispositivo.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_168" product="dev" /> y presione **4** o **6** para cambiar la configuración.

   * <Cmd sid="ui_200" product="dev" />: Permite introducir un ID de usuario solo con números.

   * <Cmd sid="ui_237" product="dev" />: Permite introducir un ID de usuario con caracteres alfanuméricos y ciertos símbolos `_` `-`.

3. Pulsa **OK** para guardar la configuración.

## Red

Cambie la configuración de comunicación de dispositivos como Ethernet, servidor y comunicación serial.

### TCP/IP

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_103" product="dev" /> → <Cmd sid="ui_059" product="dev" /> y presione **6**.

3. Selecciona el elemento deseado y pulsa **4** o **6** para cambiar los ajustes.

   <Image src="/img/device/bln2-network-ethernet.png" />

   * <Cmd sid="ui_355" product="dev" />: Configure los puertos del dispositivo.

   * <Cmd sid="ui_012" product="dev" />: Configure si se utiliza DHCP. Si la configuración DHCP está deshabilitada, el usuario puede modificar <Cmd sid="ui_029" product="dev" />, <Cmd sid="ui_068" product="dev" /> y <Cmd sid="ui_192" product="dev" />.

   * <Cmd sid="ui_029" product="dev" />: Compruebe la dirección IP del dispositivo. Para modificarla, desactive la configuración de DHCP.

   * <Cmd sid="ui_068" product="dev" />: Compruebe la puerta de enlace del dispositivo. Para modificarla, desactive la configuración de DHCP.

   * <Cmd sid="ui_192" product="dev" />: Compruebe la máscara de subred del dispositivo. Para modificarla, desactive la configuración de DHCP.

   * <Cmd sid="ui_013" product="dev" />: Establece la dirección del servidor de nombres de dominio.

4. Pulsa **OK** para guardar la configuración.

### Servidor

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_103" product="dev" /> → <Cmd sid="ui_180" product="dev" /> y presione **6**.

3. Selecciona el elemento deseado y pulsa **4** o **6** para cambiar los ajustes.

   <Image src="/img/device/bln2-network-server.png" />

   * <Cmd sid="ui_234" product="dev" />: Al seleccionar <Cmd sid="ui_283" product="dev" /> se envía una señal de conexión directa al servidor con la información ingresada en el dispositivo. Se pueden ingresar <Cmd sid="ui_183" product="dev" />, <Cmd sid="ui_182" product="dev" /> y <Cmd sid="ui_189" product="dev" />.

   * <Cmd sid="ui_183" product="dev" />: Introduzca <Cmd sid="ui_183" product="dev" /> en lugar de <Cmd sid="ui_182" product="dev" />.

   * <Cmd sid="ui_182" product="dev" />: Ingrese la dirección IP de la PC en la que está instalado BioStar X.

   * <Cmd sid="ui_189" product="dev" />: Ingrese el puerto de la PC en la que está instalado BioStar X.

4. Pulsa **OK** para guardar la configuración.

### Configuración de comunicación serial

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_103" product="dev" /> → <Cmd sid="ui_049" product="dev" /> y presione **6**.

3. Selecciona el elemento deseado y pulsa **4** o **6** para cambiar los ajustes.

   <Image src="/img/device/bln2-network-rs485.png" />

   * <Cmd sid="ui_112" product="dev" />: Seleccione el modo de operación RS-485.

   * <Cmd sid="ui_113" product="dev" />: Seleccione el rendimiento de comunicación deseado.

   * <Cmd sid="ui_257" product="dev" />: Cuando utilice <Cmd sid="ui_269" product="dev" />, seleccione esta opción para mostrar en la pantalla los resultados de autenticación de controladores o dispositivos de terceros. Esta opción puede utilizarse cuando el modo de <Cmd sid="ui_049" product="dev" /> está configurado en <Cmd sid="ui_088" product="dev" /> o <Cmd sid="ui_205" product="dev" />.

4. Pulsa **OK** para guardar la configuración.
