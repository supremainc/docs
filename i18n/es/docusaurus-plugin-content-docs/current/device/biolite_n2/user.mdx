Describe cómo registrar, editar, eliminar y gestionar usuarios.

## Registrar información de usuario {#registering-user-information}

Registre información de usuario que incluya datos de autenticación biométrica.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_166" product="dev" /> → <Cmd sid="ui_173" product="dev" /> y luego presione **6**.

   <Image src="/img/device/bln2-user-add.png" />

3. Seleccione el elemento deseado y presione **4** o **6** para configurarlo. Presione **OK** para registrar la información del usuario.

   * <Cmd sid="ui_020" product="dev" />: Ingrese un número entre 1 y 4294967294 para registrar un ID de usuario. Si <Cmd sid="ui_168" product="dev" /> está configurado en <Cmd sid="ui_237" product="dev" />, puede usar una combinación de caracteres alfanuméricos y los símbolos `_` `-` como ID, con un máximo de 32 caracteres.

   * <Cmd sid="ui_253" product="dev" />: Puede ingresar un nombre de usuario usando los botones numéricos y de función.

   * <Cmd sid="ui_312" product="dev" />: Inscriba una huella digital para la autenticación del usuario. Después de escanear la huella digital del dedo que se va a inscribir, vuelva a escanear la huella del mismo dedo. Seleccione la huella digital que desea eliminar y presione **4** para eliminarla, o presione **6** para escanearla nuevamente. Para agregar más huellas digitales, seleccione <Cmd sid="ui_001" product="dev" /> y luego presione **6**.

   * <Cmd sid="ui_322" product="dev" />: Inscriba una tarjeta para la autenticación del usuario. Escanee la tarjeta que se asignará al usuario en la unidad de reconocimiento de tarjetas RF.

   * <Cmd sid="ui_040" product="dev" />: Ingrese el PIN que desea usar. Después de ingresar el PIN deseado, ingréselo nuevamente para confirmación. Ingrese un número de entre 8 y 16 dígitos para evitar filtraciones.

   * <Cmd sid="ui_172" product="dev" />: Seleccione los permisos que desea asignar al usuario.

     :::info

     Los menús disponibles varían según el nivel de usuario establecido.

     * <Cmd sid="ui_273" product="dev" />: Este es un nivel de usuario regular y no puede acceder al menú.

     * <Cmd sid="ui_300" product="dev" />: Puede usar todos los menús.

     * <Cmd sid="ui_194" product="dev" />: Puede usar los menús <Cmd sid="ui_256" product="dev" />, <Cmd sid="ui_366" product="dev" />, <Cmd sid="ui_280" product="dev" />, <Cmd sid="ui_344" product="dev" />, <Cmd sid="ui_255" product="dev" />.

     * <Cmd sid="ui_171" product="dev" />: Puede usar los menús <Cmd sid="ui_166" product="dev" />, <Cmd sid="ui_255" product="dev" />.

     :::

   * <Cmd sid="ui_213" product="dev" />: Configure la fecha de inicio de uso de la cuenta de usuario.

   * <Cmd sid="ui_129" product="dev" />: Configure la fecha de finalización de uso de la cuenta de usuario.

   * <Cmd sid="ui_002" product="dev" />: Puede establecer el nivel de seguridad para la autenticación 1:1.

   * <Cmd sid="ui_363" product="dev" />: Puede seleccionar el número de huella digital que se utilizará como huella digital de amenaza. Debe inscribir al menos 2 huellas digitales para usar esta función.

   * <Cmd sid="ui_064" product="dev" />: Puede cambiar el método de autenticación para cada usuario.

4. Pulsa **OK** para guardar la configuración.

## Modificar información de usuario

El gestor de usuarios o el administrador pueden modificar la información de usuario registrada. Agregue la huella digital o la tarjeta del usuario y modifique el PIN y los permisos.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_166" product="dev" /> → <Cmd sid="ui_170" product="dev" /> y luego presione **6**.

3. Seleccione los términos de búsqueda. Puede buscar usuarios usando <Cmd sid="ui_137" product="dev" />, <Cmd sid="ui_020" product="dev" />, <Cmd sid="ui_253" product="dev" />, <Cmd sid="ui_312" product="dev" />, <Cmd sid="ui_322" product="dev" />.

   <Image src="/img/device/bln2-user-search.png" />

4. Seleccione al usuario que desea modificar y luego presione **6**. Consulte [Registrar información de usuario](#registering-user-information) para modificar la información.

   * Para eliminar un usuario, seleccione **4** y luego presione **OK**.

:::info

<Cmd sid="ui_320" product="dev" /> se puede registrar en BioStar X. Para obtener más información sobre el registro de grupos de acceso, consulte la [Guía del administrador de BioStar X](../../platform/biostar_x/settings-access-control-manage-access-group#addAccessGroup).

:::

## Eliminar todos los usuarios

Elimine todos los usuarios registrados a la vez.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_166" product="dev" /> → <Cmd sid="ui_302" product="dev" /> y luego presione **6**.

3. Presione **OK** para eliminar a todos los usuarios registrados.

## Ver uso de usuarios

Vea de un vistazo la cantidad de usuarios, huellas digitales y tarjetas inscritas.

1. Pulsa **ESC** y autentícate como administrador.

2. Seleccione <Cmd sid="ui_166" product="dev" /> → <Cmd sid="ui_175" product="dev" /> y luego presione **6**. La información de uso de usuarios aparece.
