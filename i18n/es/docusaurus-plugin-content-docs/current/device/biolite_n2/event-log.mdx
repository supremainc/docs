Describe cómo ver los registros de eventos generados por el dispositivo.

## Busca registros de eventos

Configura condiciones para buscar registros.

1. Pulsa **ESC** y autentícate como administrador.

2. Selecciona <Cmd sid="ui_255" product="dev" /> → <Cmd sid="ui_066" product="dev" /> y pulsa **6**.

3. Selecciona el elemento deseado y pulsa **6** para cambiar la configuración.

   <Image src="/img/device/bln2-eventlog.png" />

4. Configura las condiciones de búsqueda y pulsa **OK** para mostrar los registros que cumplen dichas condiciones.

5. Presione **ESC** para regresar a la pantalla anterior.

## Eliminar todos los registros

Elimina todos los registros guardados.

1. Pulsa **ESC** y autentícate como administrador.

2. Selecciona <Cmd sid="ui_255" product="dev" /> → <Cmd sid="ui_301" product="dev" /> y pulsa **6**.

3. Pulsa **OK** para eliminar todos los registros. Pulsa **ESC** para cancelar.

## Ver uso de registros

Consulta el estado del uso de registros.

1. Pulsa **ESC** y autentícate como administrador.

2. Selecciona <Cmd sid="ui_255" product="dev" /> → <Cmd sid="ui_120" product="dev" /> y pulsa **6**.

3. Consulta el uso de registros del dispositivo.

4. Presione **ESC** para regresar a la pantalla anterior.
