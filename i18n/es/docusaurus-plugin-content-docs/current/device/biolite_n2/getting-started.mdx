## Nombre y función de cada parte

<Image src="/img/device/name-function-each-part-bln2.png" className="none" width="30%" />

* **Pantalla**: Proporciona una interfaz de usuario para la operación.

* **Botón de funciones**: Se utiliza para la tecla de asistencia, la modificación de texto, etc.

* **Teclado**

  * **0-9**: Se utiliza para ingresar números o caracteres.

  * **2**, **4**, **6**, **8**: Se utilizan para navegar al elemento deseado.

* **Botón OK**: Se utiliza para seleccionar el modo de asistencia y guardar los ajustes. Cuando el código de trabajo esté configurado, mantén presionado para cambiar el código de trabajo del usuario.

* **Botón ESC**: Se utiliza para abrir el menú, volver a la pantalla anterior y cancelar la entrada.

* **Unidad de reconocimiento de huella dactilar**: La parte que reconoce la huella dactilar para acceso.

* **Unidad de autenticación de tarjeta RF y tarjeta de acceso móvil**: parte para escanear una tarjeta RFID o una tarjeta de acceso móvil para la entrada.

* **Cable**

  Alimentación / RS-485 / Ethernet / Relé / Entrada/Salida Wiegand / Entrada

* **Altavoz**: Emite sonido.
