Proporciona orientación sobre los menús principales y las funciones accesibles con privilegios de administrador.

## Menú completo

1. Pulsa **ESC** y autentícate como administrador.

2. Selecciona el menú deseado.

   <Image src="/img/device/bln2-menu.png" />

:::info

Si no se especifica ningún administrador, presiona **ESC** para mostrar la pantalla de menú.

:::
