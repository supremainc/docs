Use la configuración de la cámara para mantener una calidad de imagen óptima para el entorno de captura, configurar la red y aplicar políticas de seguridad y privacidad. Configure los perfiles de video, la imagen de la cámara, la privacidad y superposición, la fecha y hora, y los ajustes de red para operar la cámara de forma eficaz.
