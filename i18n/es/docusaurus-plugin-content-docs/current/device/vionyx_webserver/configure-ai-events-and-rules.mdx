Configure eventos de IA para supervisar situaciones específicas detectadas por la cámara. Combine eventos de IA —seguimiento de personas, detección de intrusos, cruce de línea y conteo de personas— para cumplir con los requisitos de seguridad y operación. Las reglas de IA ejecutan acciones automáticamente cuando ocurren eventos, como la reproducción de audio o el envío de webhooks.

:::info

Para usar eventos de IA, configure al menos un área facial que se aplique a todos los eventos.

:::
