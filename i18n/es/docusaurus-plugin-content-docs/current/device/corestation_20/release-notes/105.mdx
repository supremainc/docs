# Firmware Version 1.0.5 (Build No. 1.0.5_260616)

Release<Separator/>2026-06-24

:::caution

- After upgrading the firmware to v1.0.3, it is not able to downgrade to an earlier version of the firmware.

- For more information, check the serial number of the device and contact the [Suprema](https://supremainc.com).

:::

## New Features and Improvements

---

1. Supports new device.

   - XPass Q2

1. Improved CoreStation Setup Manager features

   - Supported manual configuration of SCB Keys.

   - Supported backup and restoration of network information (Device ID, SCB Keys, etc.).

   - Improved the ability to transfer existing settings when replacing the CoreStation.

   - Improved the overall font sizes and UI.

1. Improved the UI to not display the 'Activated Count' when applying a Camera QR license.
