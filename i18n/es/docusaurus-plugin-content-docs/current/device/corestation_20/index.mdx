CoreStation 20 es un controlador de acceso de dos puertas diseñado para seguridad de nivel empresarial. Admite hasta 500,000 usuarios y ofrece funciones de gestión de asistencia integradas con BioStar X. Las funciones de seguridad mejoradas con soporte PoE+ y una ranura SAM están disponibles solo en el modelo CS-20P, lo que permite una instalación flexible y autenticación segura con tarjetas inteligentes.
