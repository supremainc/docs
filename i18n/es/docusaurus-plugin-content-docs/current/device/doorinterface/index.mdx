Door Interface es una robusta interfaz para 2 puertas y 4 lectores, diseñada para ofrecer un control de acceso confiable. Se integra fácilmente con la serie CoreStation, lo que permite un control de puertas flexible a través de comunicación RS-485 con cableado mínimo. Además, brinda un control de acceso confiable incluso en modo sin conexión.
