:::info

* El dispositivo puede usarse como controlador de múltiples puertas junto con dispositivos esclavos mediante el cable RS485. Los dispositivos esclavos se utilizan como lectores ficticios y la autenticación se realiza en el dispositivo maestro.

<Include product="bioentry_w3">
  - Si se conecta XPass o XPass 2 al dispositivo maestro, solo se puede usar la autenticación con tarjeta.
</Include>

* El número máximo de dispositivos esclavos que se pueden conectar varía según el método de autenticación, el número de usuarios y el número de dispositivos. La cantidad de dispositivos esclavos también afecta la velocidad de autenticación por huella.

* Un dispositivo maestro puede controlar 31 dispositivos esclavos. El ancho de banda de RS-485 permite conectar hasta 7 dispositivos de autenticación por huella.

* Para obtener más información, ponte en contacto con el [equipo de soporte técnico de Suprema](https://support.supremainc.com).

:::
