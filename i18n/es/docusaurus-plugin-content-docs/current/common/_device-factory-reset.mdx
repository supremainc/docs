## Restablecimiento de fábrica {#factory-reset}

Elimina toda la información almacenada en el dispositivo e inicializa la configuración.

1. Encienda el dispositivo.

2. Presione rápidamente el botón de reinicio tres veces.

   <Include product='bioentry_w3'><Image src='/img/device/restoring-factory-defaults-bew3.svg' className='none' width={200} alone /></Include>

3. Presione el botón de reinicio una vez más cuando el LED del dispositivo parpadee {props.name}.

:::info

Utilice Factory Default solo cuando el certificado raíz esté almacenado en el dispositivo.

:::
