## Sensor de puerta y botón de salida

Los puertos SUPERVISED INPUT Nº 0 - Nº 5 se pueden configurar para ser usados para Entrada normal o Entrada Supervisada.

<Image src="/img/device/installation-door-senser-rte-connection-cs20.png" width="60%" className="none" />

:::info

El puerto **Alarm** de SUPERVISED INPUT puede usarse conectando resistencias de 1 kΩ, 2.2 kΩ, 4.7 kΩ o 10 kΩ. Después de conectar la resistencia correspondiente al dispositivo de entrada de la conexión, configure el mismo valor de resistencia en BioStar 2.

:::
