:::info

* En el nombre del archivo descargado, `X.Y.Z` es la información de la versión y `BB` es el número de compilación.

* La contraseña de la cuenta de administrador se usa al iniciar sesión por primera vez después de instalar **BioStar X**.

* La contraseña de la cuenta root de la base de datos se utiliza como contraseña inicial para AC, TA y Video DB.

* La ruta de almacenamiento de la clave de cifrado se puede cambiar. Si modificas o mueves el archivo de la clave de cifrado después de cambiar la ruta, puede ocurrir un error del sistema.

* Cuando se elimina **BioStar X**, el archivo de la clave de cifrado también se elimina.

* El certificado proporcionado del Agente de Dispositivo USB solo se puede aplicar a la red local.

* Si otro programa está usando el puerto 443, el programa BioStar X Setting se ejecuta automáticamente y permite cambiar el número de puerto. Para obtener más detalles sobre cómo cambiar el número de puerto, consulta [lo siguiente](change-port).

* Para más información sobre cómo cambiar la configuración de la base de datos, consulta [lo siguiente](change-db).

:::

:::warning

* Ten cuidado de no perder la contraseña de la cuenta de administrador o root de la base de datos.

* Si pierdes la contraseña, puede que no sea posible actualizar la versión ni hacer copias de seguridad o restaurar la BD.

:::
