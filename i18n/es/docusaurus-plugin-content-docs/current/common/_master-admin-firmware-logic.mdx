:::info

Lo siguiente aplica solo a BioStation 3, X-Station 2, BioStation 2a y BioLite N2. Los criterios de versión difieren según el modelo, pero el comportamiento es el mismo.

:::

### Diferencias de versión por producto

| BioStation 3 | Versión 1.3.1 o anterior | Versión 1.4.0 a 1.4.1 | Versión 1.4.2 o posterior |
| X-Station 2 | Versión 1.3.2 o anterior | Versión 1.4.0 | Versión 1.4.1 o posterior |
| BioStation 2a | Versión 1.1.1 o anterior | Versión 1.2.0 | Versión 1.2.1 o posterior |
| BioLite N2 | Versión 1.6.2 o anterior | Versión 1.7.0 | Versión 1.7.1 o posterior |
