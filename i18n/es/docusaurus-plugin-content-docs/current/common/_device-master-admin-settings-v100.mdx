## Configurar administrador maestro

La función <Cmd sid='ui_128' product='dev' /> mejora los privilegios administrativos generales del dispositivo para aumentar la seguridad del dispositivo y evitar el acceso y los cambios de configuración no autorizados. Inscriba un administrador maestro mediante esta función. Solo el administrador maestro inscrito puede acceder al menú de administración para cambiar la configuración del dispositivo.

:::note

¿Qué es un administrador maestro?

* Si se utiliza el dispositivo sin configurar un administrador, cualquiera podrá acceder al menú de administración y cambiar la configuración del dispositivo. La función de administrador maestro aplica la configuración administrativa general para solucionar estas vulnerabilidades de seguridad.

* El administrador maestro no puede eliminarse directamente y solo se eliminará durante <Cmd sid='ui_072' product='dev' />.

:::

### Registra al administrador maestro

En la pantalla de registro de <Cmd sid="ui_128" product="dev" /> que aparece en el primer inicio, registra <Cmd sid="ui_128" product="dev" />. Si no lo registras, el dispositivo queda restringido y no es posible avanzar a la siguiente pantalla.

:::info

Registre <Cmd sid='ui_128' product='dev' /> para usar todas las funciones, incluso si conecta BioStar X y asigna un administrador global del dispositivo.

:::

1. Enciende el producto por primera vez para que se muestre la pantalla de registro de <Cmd sid="ui_128" product="dev" />.

### Información adicional

* **Cambio manual de la clave hash del dispositivo**

   Al cambiar manualmente la clave hash del dispositivo, aparecerá un mensaje de advertencia que indicará que los PIN de Master Admin se eliminarán. Confirme el mensaje antes de continuar.

* **Restricción de transmisión de imágenes biométricas RS-485**

   Las imágenes de credenciales biométricas no se transmiten por comunicación RS-485, por lo que aparecen imágenes sustitutas al ver el rostro del Master Admin registrado en dispositivos esclavos.
