  ## Sensor de puerta y botón de salida

  Los puertos SUPERVISED INPUT Nº 0 - Nº 5 se pueden configurar para ser usados para Entrada normal o Entrada Supervisada.

  <Image src="/img/device/installation-door-senser-rte-connection-cs20.png" width="45%" className="none" />

  :::info

  El puerto **Alarm** de SUPERVISED INPUT puede usarse conectando resistencias de 1 kΩ, 2.2 kΩ, 4.7 kΩ o 10 kΩ. Después de conectar la resistencia adecuada al dispositivo de entrada, configure el mismo valor de resistencia en BioStar X.

  :::
</Include>

<Include product="doormodule">
  ## Botón de salida

  Los puertos IN0(SP), IN1(SP), IN4(SP) e IN5(SP) admiten la función Supervised Input y operan solo como N/C. Conecte a los puertos IN2, IN3, IN6 e IN7 para evitar usar Supervised Input.

  <Image src="/img/device/installation-door-button-connection-dm20.png" width="65%" className="none" />
