#### Puertos y servicios de red

Este dispositivo usa los siguientes puertos para la comunicación de red y el funcionamiento estable del servicio.

<div className='overflow-x col2--20-80'>

| Protocolo | Servicio |
| :-: | --- |
| **TCP** | Se utiliza para los servicios de comunicación entre el servidor y el dispositivo, así como para el servicio de cambio de estado operativo del dispositivo. |
| **UDP** | Se usa para el servicio de detección del dispositivo, que permite buscar dispositivos en la red. |

</div>

:::info

Estos puertos se usan para proporcionar las funciones normales de red del producto. Al configurar las reglas del firewall o los ajustes de seguridad de red, permita el uso de estos puertos.

:::
