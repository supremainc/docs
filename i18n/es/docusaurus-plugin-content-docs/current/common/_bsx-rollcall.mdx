## Iniciar lista de asistencia {#mobileAppRollCall}

La función de lista de asistencia está disponible en la aplicación móvil **BioStar X**.

<Include pages='settings-advanced-ac-roll-call'>

:::info

Puedes instalar la aplicación móvil **BioStar X** mediante los siguientes enlaces.

* iOS: [Apple App Store](https://apps.apple.com/kr/app/biostar-x-mobile/id6741415253)

* Android: [Google Play](https://play.google.com/store/apps/details?id=com.suprema.biostarmobile)

:::

</Include>

### Iniciar lista de asistencia en la aplicación móvil {#useRollCallMobileApp}

Para iniciar la función de lista de asistencia en la aplicación móvil, sigue estos pasos.

1. Abre la aplicación móvil **BioStar X** en tu dispositivo móvil.

2. Desde la pantalla principal de la app, toca <Cmd>ROLL CALL</Cmd>.

   <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-01.png' width={253} alone />

   :::note

   También puedes acceder a ella desde el menú lateral. Toca <IcHamberg title='Hamburger icon' width={20} /> → <Cmd>Monitoring</Cmd> → <Cmd>Roll Call</Cmd>.

   :::

3. En la lista de Roll Call, toca <IcRCplay title='Start' width={20} height={20} /> del elemento para ejecutar la lista de asistencia.

   <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-02.png' width={253} alone />

4. Cuando aparezca el mensaje de confirmación, toca <Cmd>Yes</Cmd>.

La lista de asistencia se iniciará y, al tocar el elemento de lista de asistencia en curso, podrás comprobar su estado.

<Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-03.png' width={595} alone className='none' />

### Control de lista de asistencia en curso

* Los usuarios que ingresen se marcarán con el ícono <IcRCdone title='Done' width={20} height={20} />. Cuando todos los usuarios hayan ingresado, aparecerá un mensaje que indica que la lista de asistencia ha finalizado. Para finalizar la lista de asistencia, toca <Cmd>Yes</Cmd>.

* El administrador puede cambiar manualmente la entrada o el estado del usuario. Mantén presionado el botón redondo en el extremo derecho del usuario cuyo estado deseas cambiar en la lista de usuarios.

  <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-change-status.png' alone className='none' width={595} />

* Para reiniciar la lista de asistencia, toca <Cmd>Reset</Cmd>. Todo el progreso se descarta y la lista de asistencia se reinicia desde el inicio.

* Si otro administrador finaliza la lista de asistencia, aparecerá un mensaje de notificación.

  <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-stopped.png' width={253} alone />

* Para finalizar una lista de asistencia en curso, toca el botón <IcRCstop title='Stop' width={20} height={20} /> en la pantalla de lista de asistencia o toca el botón <Cmd>END</Cmd> en la pantalla de estado de la lista de asistencia.

* También puedes finalizar la lista de asistencia en curso en **BioStar X**. Ve a <Cmd sid='shell.header.setting' /> → <Cmd sid='setting.menu.advancedAc.root' /> → <Cmd sid='data.sidebar.events' /> → menú <Cmd sid='zone.rollCall' product='2' />, selecciona la lista de asistencia en curso y haz clic en el botón <Cmd sid='zone.pause.end' /> en la esquina superior derecha de la pantalla.

  <Image src='/img/biostarx-settings-adv-ac-rollcoll-ended.png' />

<Include pages='settings-advanced-ac-roll-call'>

* Después de completar la lista de asistencia, puedes consultar los informes relacionados en <Cmd sid='shell.header.data' /> → <Cmd sid='data.mainMenu.generate' /> → <Cmd sid='data.sidebar.rollCall' />. Para obtener más información, consulte <DocLink inner='#viewRollCallReport' />.

</Include>

### Información del usuario durante la lista de asistencia

Durante la lista de asistencia en la aplicación móvil, puedes consultar la información de cada usuario. Consulta detalles como la información de contacto del contacto de emergencia y la última ubicación conocida del usuario.

Desliza hacia la izquierda sobre el elemento de usuario del que quieras ver detalles durante la lista de asistencia. Luego, toca <IcRCinfo title='Information' width={20} height={20} />.

<Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-user-info.png' className='none' width={595} alone />

### Dejar una nota para usuarios durante la lista de asistencia

Si un usuario específico no ha ingresado durante la lista de asistencia en la aplicación móvil, puedes dejarle una nota.

Desliza hacia la izquierda sobre el elemento de usuario donde quieras ingresar una nota durante la lista de asistencia. Luego, toca <IcRCmemo title='Memo' width={20} height={20} />. En la pantalla de ingreso de notas, escribe el contenido y toca <Cmd>Save</Cmd>.

<Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-user-memo.png' className='none' width={595} alone />

Los usuarios que hayan ingresado notas se marcarán con el ícono <IcRCmemo title='Memo' width={20} height={20} /> en la lista de asistencia.

<Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-user-status.png' width={253} alone />

### Ver detalles de la lista de asistencia {#viewRollCallinfo}

Consulta los detalles de la configuración del pase de lista en la aplicación móvil.

1. Abre la aplicación móvil **BioStar X** en tu dispositivo móvil.

2. Desde la pantalla principal de la app, toca <Cmd>ROLL CALL</Cmd>.

   <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-01.png' width={253} alone />

   :::note

   También puedes acceder a ella desde el menú lateral. Toca <IcHamberg title='Hamburger icon' width={20} /> → <Cmd>Monitoring</Cmd> → <Cmd>Roll Call</Cmd>.

   :::

3. Desliza el elemento hacia la izquierda para consultar los detalles en la lista y, luego, toca <IcRCinfo title='Information' width={20} height={20} />.

   <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-info.png' width={595} alone className='none' />
