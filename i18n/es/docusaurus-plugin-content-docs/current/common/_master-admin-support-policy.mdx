### Política de soporte del Master Administrator <Badge only>Productos CE vs productos Global (No CE)</Badge> {#master-admin-support-policy}

</Xclude>

La feature <Cmd sid="ui_128" product='dev' /> se activa solo en <Cmd>Productos CE con una etiqueta que tiene la marca CE</Cmd>.

:::note

Use la etiqueta adhesiva en el rótulo posterior para identificar **Productos CE** y **productos Global (No CE)**. La feature funciona de manera diferente, como se indica a continuación.

* **Productos CE**: marca CE presente

* **productos Global (No CE)**: marca CE ausente

:::

:::note

Cuando se conecta a BioStar X (o BioStar 2), el setting de Master Admin cambia según la compatibilidad del dispositivo, y la activación del menú de autenticación en dos pasos del administrador se sincroniza automáticamente.
