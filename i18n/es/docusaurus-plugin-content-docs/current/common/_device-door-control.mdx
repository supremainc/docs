## Control de Puerta

Puede abrir, bloquear o desbloquear manualmente la puerta conectada.

:::info

* Solo puede controlar la puerta cuando el dispositivo esté configurado como dispositivo maestro.

* Utilice <Cmd sid='ui_300' product='dev' /> o <Cmd sid='ui_194' product='dev' /> para controlar la puerta.

:::

1. Seleccione <Cmd sid='ui_380' product='dev' /> en la pantalla principal.

   <Include product='biostation_3'><Image src='/img/device/bs3-door-control1.png' /></Include>
   <Include product='biostation_3_max'><Image src='/img/device/bs3m-door-control1.png' /></Include>

2. Autentica usando el método de privilegio de administrador.

3. Seleccione el método de control de la puerta que desee.

   <Include product='biostation_3'><Image src='/img/device/bs3-door-control2.png' /></Include>
   <Include product='biostation_3_max'><Image src='/img/device/bs3m-door-control2.png' /></Include>

   * <Cmd sid='ui_382' product='dev' />: Establezca el tiempo para la apertura inmediata de la puerta o para mantenerla abierta.

   * <Cmd sid='ui_383' product='dev' />: Establezca el tiempo para el cierre inmediato de la puerta o para mantenerla bloqueada.

   * <Cmd sid='ui_384' product='dev' />: Cancele la apertura manual o la retención manual del bloqueo.

:::info

Establezca los tiempos de apertura manual y de retención manual del bloqueo entre 1 y 86,400 segundos.

:::
