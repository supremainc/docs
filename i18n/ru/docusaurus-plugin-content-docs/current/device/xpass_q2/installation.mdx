## Крепление кронштейна и устройства

1. Плотно закрепите кронштейн с помощью крепёжных винтов в месте установки устройства.

   <Image src='/img/device/installation-device1-xpq2.svg' className='none' width='20%' alone />

   :::info

   - Если установка производится на бетонную стену, просверлите отверстие, вставьте пластиковый дюбель и закрепите крепёжным винтом.

   - Для предотвращения радиочастотных помех необходимо соблюдать минимальное расстояние между устройствами.

   <Columns>
   <Column className='text--center'>

   <Image src='/img/device/installation-wall1-xpq2.png' className='none' width='60%' />

   <div className='overflow-x'>

   | Толщина стены | Расстояние |
   | :-----------: | :--------: |
   |     100 мм    |   270 мм   |
   |     120 мм    |   250 мм   |
   |     150 мм    |   170 мм   |

   </div>

   </Column>

   <Column className='text--center'>

   <Image src='/img/device/installation-wall2-xpq2.png' className='none' width='60%' />

   </Column>
   </Columns>

   - При использовании мобильных карт доступа устанавливайте устройства с расстоянием не менее 1 м между ними, чтобы избежать помех BLE.

   :::

   <PageBreak />

2. Затяните четыре винта, фиксирующих крышку кабеля.

   <Image src='/img/device/installation-device2-xpq2.svg' className='none' width='20%' alone />

   :::info

   Убедитесь, что крышка кабеля полностью закрыта после подключения к устройству, чтобы обеспечить водо- и пылезащиту (степень защиты IP65).

   :::

3. Установите устройство на закреплённый кронштейн.

   <Image src='/img/device/installation-device3-xpq2.svg' className='none' width='35%' alone />

4. Поверните фиксирующие винты для соединения устройства с кронштейном.

   <Image src='/img/device/installation-device4-xpq2.svg' className='none' width='15%' alone />

   :::info

   При сборке устройства с кронштейном вы можете использовать прилагаемый фиксирующий винт кронштейна (звездообразный) вместо стандартного винта крепления устройства для большей надёжности.

   :::

### Максимальная длина удлинителя по спецификациям кабеля

Неправильное подключение кабеля может привести к сбоям в работе устройства. Устройство поддерживает DC 12V, DC 24V и PoE. ## Обновление прошивки

Вы можете подключить USB-накопитель для обновления прошивки.

1. Сохраните файл прошивки на USB-накопитель для обновления.

   :::info

   Рекомендуется сохранять на USB-накопителе только один файл прошивки для обновления.

   :::

2. Подключите USB-накопитель к устройству.

   :::info

   Во время чтения USB-накопителя аутентификация временно приостанавливается.

   :::

3. Когда светодиод или зуммер запрашивает аутентификацию, выполните вход с правами администратора.

   :::info

   - Зарегистрируйте администратора в BioStar X для устройства. Обновление прошивки невозможно, если в устройстве не зарегистрирован администратор. Для подробных инструкций по регистрации администратора обратитесь к <Linkto to='platform/biostar_x/set-permission'>Руководству администратора BioStar X</Linkto>.

   - При каждом подключении USB-накопителя допускается только одна попытка аутентификации.

   - Если аутентификация не удалась, светодиод или зуммер укажут на ошибку, и чтение USB завершится. Отключите USB-накопитель и подключите его снова к устройству.

   :::

4. После успешной аутентификации администратора устройство начнет обновление прошивки и уведомит об этом с помощью светодиода.

   :::info

   Если файл прошивки отсутствует или поврежден, обновление не начнется, и светодиод или зуммер укажут на ошибку.

   :::

5. Устройство автоматически перезагрузится после завершения обновления прошивки.

:::caution

Не отключайте питание во время обновления прошивки устройства. Устройство может работать неправильно.

:::
