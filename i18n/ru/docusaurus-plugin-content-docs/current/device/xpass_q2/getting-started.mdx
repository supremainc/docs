## Название и функция каждой части

<Image src='/img/device/name-function-each-part-xpq2.svg' className='none' width='70%' />

- **Светодиодный индикатор**: отображает состояние работы устройства с помощью цвета светодиода.

  - Зелёный: аутентификация прошла успешно.

  - Красный: аутентификация не удалась.

  - Мигание синим или голубым: нормальная работа.

  - Мигание синим и зелёным:

    - При включенном <Cmd sid='ui_012' product='dev' /> невозможно получить IP-адрес.

    - В режиме <Cmd sid='ui_205' product='dev' /> связь с основным устройством прервана.

  - Мигание красным и пурпурным: устройство заблокировано.

  - Мигание зелёным: ожидание ввода.

- **Светодиод**: автоматически загорается при обнаружении движения рядом со сканером QR/штрих-кодов.

- **Сканер QR/штрих-кодов**: считывает данные QR- или штрих-кодов для доступа.

- **Блок аутентификации RF-карт и мобильных карт доступа**: сканирует RFID- или мобильные карты доступа для входа.

- **Разъём USB (Type-C) для флеш-накопителя**: подключите USB-накопитель.

- **Кнопка сброса**

  - Инициализирует сетевые настройки. За подробностями по инициализации сетевых настроек обращайтесь к разделу [Initialize Network Settings](installation#reset-network-settings).

  - Удаляет всю информацию и сертификаты, сохранённые на устройстве, и сбрасывает настройки. Подробнее о сбросе к заводским настройкам смотрите в разделе [Factory default](installation#factory-reset).

- **Ethernet**: подключите Ethernet-кабель.

- **RS-485 (4-контактный)**: подключите кабель RS-485.

- **Вход (5-контактный)**: подключите входной кабель.

- **Реле (3-контактный)**: подключите кабель реле.

- **Питание (2-контактный)**: подключите кабель питания.

- **Расширительный USB-порт (5-контактный)**: подключите другие аксессуары.

- **Wiegand вход/выход (4-контактный)**: подключите кабель Wiegand вход/выход.

- **Выход питания считывателя (3-контактный)**: подключите кабель питания считывателя.

- **Выход питания замка (3-контактный)**: подключите кабель питания замка.

## Примечания по QR/штрихкоду и лицензии устройства

### Правильный метод аутентификации QR/штрихкода {#how-to-scan-qrbarcode}

Выдайте пользователю QR-/штрихкод для использования в качестве средства аутентификации. Сканируйте QR-/штрихкод согласно инструкции для улучшения распознавания.

<Image src='/img/device/qr-scanning-xpq2.svg' className='none' width={300} alone />

- Держите расстояние 5–15 см между сканером устройства и QR-/штрихкодом.

- Расположите QR-/штрихкод в центре сканера.

- Если аутентификация происходит с помощью QR-/штрихкода на мобильном устройстве, отрегулируйте яркость экрана для лучшей видимости.

- Сканирование QR-/штрихкода может занять больше времени или не пройти успешно при неправильном положении, расстоянии или освещении.

### Примечания к лицензии устройства

- Использование дополнительных методов аутентификации QR/Barcode требует отдельной лицензии устройства. Обратитесь в место покупки для оформления лицензии устройства.

- Файл лицензии устройства зашифрован и не может быть изменён произвольно.

- Лицензии устройства выдаются на основе ID устройства. Если ID устройства изменён нестандартным способом, гарантийное обслуживание лицензии не предоставляется.

- Для подробной информации о лицензиях устройств обратитесь к <Linkto to='platform/biostar_x/settings-license-device-license'>Руководству администратора BioStar X</Linkto>.
