Это руководство объясняет, как сбросить камеру к заводским настройкам и заменить модуль осушителя.

## Заводские настройки по умолчанию {/* #factory-reset */}

Удаляет всю информацию, хранящуюся на камере, и сбрасывает все настройки.

1. После подключения камеры к источнику питания найдите кнопку сброса.

   <Image src='/img/camera/factory-reset-vionyx.svg' className='none' width={350} alone />

2. Нажмите и удерживайте кнопку сброса не менее 5 секунд, затем отпустите ее.

3. После завершения сброса к заводским настройкам камера автоматически перезагрузится.

:::caution

Выполнение сброса к заводским настройкам удалит все пользовательские данные и настройки, включая сетевые настройки.

:::

:::info

- По умолчанию IP-адрес настраивается через DHCP.

- Если после сброса к заводским настройкам устройство не получает IP-адрес, оно переходит на IP-адрес по умолчанию (`https://192.168.1.100`), поэтому подключайтесь, используя этот адрес.

:::

## Замена модуля осушителя

Модуль осушителя установлен внутри камеры для предотвращения конденсации на объективе. Чтобы поддерживать эффективность осушения, рекомендуется заменять модуль осушителя не реже одного раза в год.

1. Проверьте место установки модуля осушителя.

2. Ослабьте винт, повернув его против часовой стрелки, затем снимите модуль осушителя с крышки.

   <Image src='/img/camera/change-silicagel-vionyx.svg' className='none' width={684} alone />

3. Поместите новый модуль осушителя на крышку, затем поверните винт по часовой стрелке, чтобы закрепить его.

:::caution

Интервалы замены могут различаться в зависимости от условий установки.

:::

:::info

- Запишите дату замены, чтобы отслеживать следующую замену.

- Запасные модули осушителя можно приобрести у продавца, у которого вы купили продукт.

:::
