## Название и функция каждой части

### Конфигурация внутренних портов

<Image src='/img/camera/name-function-each-part-vionyx.svg' className='none' width={700} />

<div className='overflow-x col2-2-8'>
| Name | Function |
| -- | -- |
| **Alarm Input, Relay (5-pin)** | Подключите кабель для входа тревоги и выхода реле. |
| **Audio Input/Output (4-pin)** | Подключите кабель для аудиовхода/аудиовыхода. |
| **Ethernet (PoE+)** | Подключите Ethernet-кабель. Питание может подаваться через PoE+. |
| **Power, RS-485 (5-pin)** | Подключите кабель питания и связи RS-485. |
| **Reset Button** | Сбрасывает все настройки до заводских значений. Подробнее о инициализации до заводских настроек смотрите в разделе [Factory default](maintenance#factory-reset). |
| **Micro SD Card Slot** | Вставьте карту microSD. |
| **USB (Type-A)** | Подключайте USB-аксессуары. |
| **MIC Switch** | Используйте переключатель микрофона, чтобы физически включать или отключать аудиозапись. |
</div>

### Световой индикатор LED

Светодиод на передней части крышки камеры показывает состояние работы устройства и аутентификации с помощью цветов и мигающих сигналов.

<Image src='/img/camera/led-status-indicator-vionyx.svg' className='none' width={350} alone />

|     Цвет светодиода     | Статус                                                                          |
| :---------------------: | ------------------------------------------------------------------------------- |
|         Зелёный         | Аутентификация прошла успешно                                                   |
|         Красный         | Ошибка аутентификации                                                           |
|          Синий          | Нормальная работа                                                               |
| Мигание синим и зелёным | При включенном <Cmd sid='ui_012' product='dev' /> невозможно получить IP-адрес. |

<PageBreak />

### Настройки переключателя микрофона (скоро будет доступно)

Используйте переключатель микрофона, чтобы физически отключить микрофон камеры и предотвратить запись звука.

<Image src='/img/camera/microphone-switch-vionyx.svg' className='none' width={450} alone />

- По умолчанию аудиозапись отключена.

- Переведите переключатель микрофона вправо, чтобы включить аудиозапись.
