## Реле подключения

### Защитный блокирующий замок

Для использования защитного блокирующего замка подключите нормально замкнутое (N/C) реле, как показано на рисунке ниже. Для защитного блокирующего замка через реле обычно протекает ток. При срабатывании реле, прерывающем ток, дверь откроется. Если питание устройства отключится из-за перебоя в электроснабжении или внешних причин, дверь откроется.

:::caution

- Установите диод с обеих сторон провода дверного замка, как показано на рисунке, чтобы защитить реле от обратного тока, возникающего при работе замка.

- Используйте отдельный источник питания для устройства и дверного замка.

- Отдельные интеллектуальные считыватели Suprema содержат встроенные реле, которые могут непосредственно блокировать и разблокировать двери без внешних контроллеров для удобства. Однако для систем контроля доступа с требованиями безопасности НЕ рекомендуется использовать встроенное реле считывателя, чтобы предотвратить возможные атаки с вмешательством, которые могут спровоцировать разблокировку двери. Для таких систем настоятельно рекомендуется использовать отдельный блок реле для управления замком, например Suprema Secure I/O 2, DM-20 или CoreStation, установленный с защищённой стороны двери.

:::

:::info

Обратите внимание на направление установки диода. Установите диод как можно ближе к дверному замку.

:::
