:::caution

Автономные интеллектуальные считыватели Suprema содержат встроенные реле, которые могут напрямую блокировать и разблокировать двери без внешних контроллеров для удобства. Однако для систем контроля доступа с повышенными требованиями безопасности НЕ рекомендуется использовать внутреннее реле считывателя, чтобы предотвратить возможные атаки и манипуляции, которые могут привести к нежелательному открытию двери. Для таких систем настоятельно рекомендуется использовать отдельное реле для управления замком, например, Suprema Secure I/O 2, DM-20 или CoreStation, установленные со стороны, защищённой от несанкционированного доступа.

:::
