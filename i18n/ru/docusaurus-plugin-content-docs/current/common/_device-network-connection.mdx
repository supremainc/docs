#### Подключение LAN (прямое подключение к ПК)

<Include product='bioentry_w3,facestation_f2,bioentry_w2,biolite_n2,biostation_2a,biostation_3,facestation_2,xstation_2'>

Устройство оснащено функцией автоматического MDI/MDIX, позволяющей подключаться к ПК напрямую, используя обычный прямой кабель CAT-5 или кроссовый кабель.

</Include>

<Include product='biostation_3_max'>

Устройство оснащено функцией автоматического MDI/MDIX, поэтому подключение к ПК возможно напрямую с помощью кроссового или стандартного прямого кабеля CAT-5e.

</Include>

<Include product='xpass_s2'>

Устройство поддерживает функцию автоматического MDI/MDIX, что позволяет подключаться напрямую к ПК с помощью стандартного кабеля CAT-5.

</Include>

<Include product='bioentry_p2,xpass_2,corestation_40,corestation_20'>

Устройство оснащено функцией автоматического MDI/MDIX, поэтому подключайте его напрямую к ПК с помощью стандартного кабеля CAT-5e.

</Include>

<Include product='corestation_20'>

<Badge only>CS-20P</Badge> Поддерживает PoE+, питание по Ethernet-кабелю. При использовании PoE+ поддерживается только одно подключение Lock.

</Include>

<Include product='xpass_q2'>

- Устройство оснащено функцией автоматического MDI/MDIX, поэтому подключение к ПК возможно напрямую с помощью кроссового или стандартного прямого кабеля CAT-5e.

  :::caution

  Некоторые нестандартные концентраторы и коммутаторы могут не поддерживать автоматический MDI/MDIX.

  :::

- <Badge only>XPQ2-APB</Badge> Поддерживает PoE — питание по Ethernet-кабелю.

</Include>

#### Сетевые порты и сервисы

Это устройство использует следующие порты для сетевой связи и стабильной работы сервисов.

<div className='overflow-x col2--20-80'>

| Протокол | Описание сервиса                                                                                                 |
| :------: | ---------------------------------------------------------------------------------------------------------------- |
|  **TCP** | Используется для служб связи между сервером и устройством, а также для службы переключения состояния устройства. |
|  **UDP** | Используется для службы обнаружения устройств в сети.                                                            |

</div>

:::info

Эти порты обеспечивают нормальное функционирование сетевых возможностей продукта. При настройке правил межсетевого экрана или параметров сетевой безопасности разрешите использование этих портов.

:::
