## Инициализация сетевых параметров {#reset-network-settings}

Инициализируйте сетевые параметры устройства.

1. Включите устройство.

2. Нажмите кнопку сброса и удерживайте её, пока устройство не выполнит автоматическую перезагрузку.

3. Подключите устройство, используя инициализированные сетевые параметры.

   - **TCP/IP-адрес**: назначение адреса через DHCP (если назначение адреса через DHCP не удаётся, будет установлен адрес 169.254.x.x).

   - **Режим сервера**: отключён

   - **RS-485**: по умолчанию, 115200 бод

4. Измените параметры TCP/IP или RS-485.

5. Проверьте корректность сетевых параметров после выключения и повторного включения питания.
