## Заводские настройки по умолчанию {#factory-reset}

Удалите всю информацию, хранящуюся на устройстве, и инициализируйте настройки.

1. Включите питание.

2. Быстро нажмите кнопку сброса три раза.

3. Нажмите кнопку сброса ещё раз, когда светодиод устройства начнёт мигать {props.name}.

:::info

Использовать заводские настройки можно только если корневой сертификат сохранён на устройстве.

:::
