title: BioStar X Hardening
description: Strengthen BioStar X security at two critical layers — application-level protection through authentication and access control, and data-level protection through encryption and minimal biometric storage.
---

Securing BioStar X requires protecting two interconnected layers: the system itself and the personal data it stores.

* **System Security** reinforces the application through default admin account changes, role-based access control, strong authentication, and encrypted device communication.

* **Personal Data Security** protects sensitive user credentials and biometric data through database encryption and minimal storage practices.

Implementing both layers ensures regulatory compliance (GDPR, ISO 27001, etc.) and minimizes breach risk.
