### 6.1. Default BioStar X Network Ports

| Service | Purpose | Default Port(s) | Notes / Recommendations |
|---------|---------|-----------------|------------------------|
| Unified Gateway Service | HTTPS (system access) | 443 | Default for admin/operator login. Changeable via **Service Manager → Service Settings**. |
| Core Web Service | HTTPS (internal UI) | 5002 | Prefer proxied to 443; restrict direct access. |
|  | CloudNgrok | 52000 | Disable if not required. |
|  | Thrift RPC | 9310 | Restrict to trusted services. |
| Core Service | WebSocket | 9002 | Internal only. |
|  | FastCGI | 9000 | Should be bound locally. |
|  | API | 9010 | Secure with TLS & authentication. |
| Coordinator Service | Client Communication | 21810 | Internal only. |
| Main Server | TCP Server | 51212 | **Plaintext** device comms — block if possible. |
|  | SSL Server | 51213 | **Preferred** device comms (TLS). |
|  | gRPC | 51219 | Internal service comms only. |
| Cache Service | Client | 10800 | Equivalent to Redis — do not expose externally. |
|  | Communication | 47500 | Internal only. |
|  | Discovery | 47100 | Internal only. |

### 6.2. Default Security Features in BioStar X

| Feature | Location in UI | Default | Recommendation |
|---------|---------------|---------|----------------|
| Change Default Admin Login ID & Password | User Module → Edit Default Admin (UID:1) | admin / installer PW | Change immediately after install. |
| Access Separation (roles) | Settings → User → Account Levels | Predefined: Admin, Operator, Monitoring | Create custom roles as needed. |
| Password Strength Policy | Settings → System → Security | Medium | Set to Strong, enforce MFA. |
| Session Security (Simultaneous Connection) | Settings → System → Security | Allow | Set to **Inactive** to prevent shared logins. |
| Multi-Factor Authentication | Settings → System → Security → MFA | Not Use | Enable for Admin/Operator/Auditor accounts. |
| Secure Communication with Device | Settings → System → Security → Advanced | Not Use | Enable + use SSL port (51213). |
| Device Hashkey Management | Settings → System → Security → Advanced | Not Use | Enable for tamper protection in high-security deployments. |
| System Backup | Settings → System → System Backup | Manual | Enable Automatic Daily backups. |
| Directory Integration | Settings → Directory Integration | Not Configured | Integrate with Entra ID or AD where available. |
| Log Settings (Retention & Levels) | Settings → Server → Server | 60 days, Info | Adjust per compliance. Forward to SIEM. |
| Encrypt Personal Data on Database | Settings → System → Security | Not Use | Enable in production. |
| Store Visual Face Image | Settings → Server → Expert Settings | Use | Disable unless justified. |
