### LED Status Indicator

You can check the product operating status by the color of the LED status indicator.

| Item              | LED             | Status                                                                         |
| ----------------- | --------------- | ------------------------------------------------------------------------------ |
| **POWER**         | Red Lit         | Power On                                                                       |
| **STATUS**        | Green Lit       | Connected status via secure session                                            |
| ^                 | Blue Lit        | Not connected to the master device                                             |
| ^                 | Pink Lit        | In the process of upgrading firmware                                           |
| ^                 | Yellow Lit      | Communication error due to different key or packet loss in OSDP causing RS-485 |
| ^                 | Sky Blue Lit    | Connected status without secure session                                        |
| **RELAY (0~11)**  | Red Lit         | Relay Operation                                                                |
| **RS-485 TX**     | Orange blinking | In the process of transmitting RS-485 data                                     |
| **RS-485 RX**     | Green blinking  | In the process of receiving RS-485 data                                        |
| **AUX IN (0, 1)** | Orange Lit      | In the process of receiving AUX signal                                         |

### Installation example

This product is an extension module that controls access to building floors. By connecting to Suprema devices and BioStar X, one module can control 12 floors.

When connected in an RS-485 daisy chain, up to 192 floors can be controlled per elevator.
