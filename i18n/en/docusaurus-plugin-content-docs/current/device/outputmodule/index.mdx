The Output Module is a multi-output control module that can use 12 relay outputs. When configured with BioStar X, it can be used for floor access control. A maximum of 31 devices can be connected to a single master device via RS-485 chain, and AUX inputs can also be used. Additionally, it can be utilized in various ways in conjunction with anti-passback areas or fire alarm zones.
