# Application Version 1.2.1

Release<Separator/>2025-05-19

:::caution

Device Manager v1.1.8 supports Android 5.0 or higher, iOS 9.0 or higher.

:::

## New Features and Improvements (Common)

---

1. Supports new device.

   * XPass 2 (XP2-MAPB-H)

1. Added the option to select **Encryption Type** (Crypto-1, AES) for MIFARE Plus smart cards (Suprema/Custom) in the smart card layout settings.

## Bug Fixes (iOS)

---

<BugLists>

1. Improved the application to display a “Device Reboot” popup when restarting the device. <Badge>1.1.0</Badge>

1. The application freezes when the **End Bit** is set lower than the **Start Bit** while adding a new Wiegand format. <Badge>1.1.0</Badge>

1. The **Skip Bytes** option can be set up to 255 in the **Custom Smart Card Layout** settings. <Badge>1.1.8</Badge>

1. The **File Read Access Key Index** option of **DESFire Advanced** can be set up to 31 in the smart card layout settings. <Badge>1.1.0</Badge>

</BugLists>
