Suprema Device Manager is a mobile application that uses BLE communication to configure Suprema’s XPass D2, XPass 2, BioEntry W3, and XPass Q2 devices.

<Image src='/img/device/device-manager-system-diagram1.svg' className='none' width={650} alone />

Administrators can connect to the server from a PC or set the device's network, server, RS-485 communication, card format, keypad settings, PIN, LED, and buzzer from a mobile device without physically disconnecting it, and can also use functions such as device restart, firmware upgrade, and device setting initialization.

For XPass D2, values set once can be saved as a template for quick and easy application to multiple devices.

:::info

Using Suprema Device Manager allows on-site modification of the settings of XPass D2 configured with third-party controllers.

<Image src='/img/device/device-manager-system-diagram2.svg' className='none' width={900} alone />

:::

## Minimum requirements

### Mobile Device

Check if the mobile device supports BLE communication.

* Android 5.0 Lollipop or later

* iOS 9.0 or later

### Device and Firmware

Check compatible devices and firmware versions.

* **<Linkto to='device/xpass_d2'>XPass D2</Linkto>** firmware version 1.1.0 or later

* **<Linkto to='device/xpass_2'>XPass 2</Linkto>**

  * XP2-MDPB, XP2-GDPB, XP2-GKDPB: firmware version 1.0.0 or later

  * XP2-MAPB: firmware version 1.4.1 or later

  * XP2-MAPB-H: firmware version 1.4.3 or later

* **<Linkto to='device/bioentry_w3'>BioEntry W3</Linkto>** firmware version 1.0.1 or later

* **<Linkto to='device/xpass_q2'>XPass Q2</Linkto>** firmware version 1.0.0 or later

:::info

* Compatible devices and firmware are subject to change.

* If the device firmware version is lower than the versions listed above, upgrade the device firmware in BioStar X. When using the device as a slave, the connected master's firmware must also be the latest version compatible with BioStar 2 2.7.0 or later.

* For more details on upgrading device firmware, please refer to [IP Device (Upgrade Device Firmware)](ip-device-settings#device-firmware-upgrade), [XPass D2 (Upgrade Device Firmware)](xpd2-settings#device-firmware-upgrade), or the <Linkto to='platform/biostar_x/settings-device-upgrade-firmware'>BioStar X Administrator Guide</Linkto>.

:::
