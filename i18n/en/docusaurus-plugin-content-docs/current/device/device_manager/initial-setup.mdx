Guidance on how to select a device and manage app settings.

## Select device

Select the device to configure using Suprema Device Manager.

You can select an IP device or XPass D2, and the options available for configuration will vary depending on the selected device.

1. Install the **Suprema Device Manager** application on the mobile device of the device manager. Suprema Device Manager is available for download from the **App Store** and **Play Store**.

2. Run the Suprema Device Manager on your mobile device.

3. Select the device to configure.

   <Image src='/img/device/device-manager-main.png' width={250} />

## Configuring app settings {#app-settings}

You can configure the automatic device connection feature or check the app version information.

1. Run the Suprema Device Manager on your mobile device.

2. Select <IcSet /> on the main screen.

   <Image src='/img/device/device-manager-app-setting.png' className='none' width={540} />

   * <Cmd sid='ui_276' product='dev' />: You can enable or disable the automatic device connection feature.

   * <Cmd sid='ui_226' product='dev' />: You can check the current app version and license information.
