## Full menu

1. Press **ESC**, enter the password, and access the menu.

2. Select your desired menu.

   <Image src='/img/device/bln2-menu.png' />

:::info

If <Cmd sid='ui_387' product='dev' /> is not set, press **ESC** to open the menu screen. For details on how to set the password, refer to [Menu password](settings#menu-password).

:::
