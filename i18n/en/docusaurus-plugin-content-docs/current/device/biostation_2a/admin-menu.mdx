Provides guidance on the main menus and features accessible with administrator privileges.

## Full menu

1. Press <Image src='/img/common/ico-menu1.svg' ico alone /> and authenticate with the administrator credential.

2. Select your desired menu.

   <Image src='/img/device/bs2a-all-menu.png' />

:::info

If no administrator is designated, pressing <Image src='/img/common/ico-menu1.svg' ico alone /> will display the menu screen.

:::

## Quick Menu

1. Press and hold <Image src='/img/common/ico-menu1.svg' ico alone /> for more than 1 second, then release it. Authenticate using administrator privilege method.

2. Select your desired menu.

   <Image src='/img/device/bs2a-quick-menu.png' />

:::info

If no administrator is designated, pressing and holding <Image src='/img/common/ico-menu1.svg' ico alone /> will show the quick menu screen.

:::

## Key Menu

Press the **OK** button to display the key menu screen. You can enter by pressing <Cmd sid='ui_334' product='dev' /> or the directional button corresponding to the desired menu on the home screen.

<Image src='/img/device/bs2a-key-menu.png' />

* <IcFw /> <Cmd sid='ui_020' product='dev' />: ID authentication can be performed as set in <Cmd sid='ui_023' product='dev' />.

* <IcDown/> <Cmd sid='ui_083' product='dev' />: You can select attendance events to authenticate.

* <IcBw /> <Cmd sid='ui_279' product='dev' />: You can select the user's work code.

* <IcUp/> <Cmd sid='ui_069' product='dev' />: You can start/stop security.

:::info

If in the <Cmd sid='ui_069' product='dev' /> state, you cannot enter <Cmd sid='ui_334' product='dev' />. You can disable security by pressing the <IcUp/> button on the home screen.

:::
