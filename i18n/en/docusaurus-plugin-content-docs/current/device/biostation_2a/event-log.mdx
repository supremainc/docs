Describes how to view event logs generated by the device.

## Search Event Logs

Set conditions to search logs.

1. Press <Image src='/img/common/ico-menu.svg' ico alone /> and authenticate with the Admin level credential.

2. On the <Cmd>DASHBOARD</Cmd>, select <Image src='/img/common/ico-eventlog.png' ico alone />. All event logs appear.

3. Press <Image src='/img/common/ico-search.svg' ico alone /> to search a specific log and change the condition.

   <Image src='/img/device/device-eventlog.png' />

4. Set the condition and press <Cmd>OK</Cmd> to log that matches the condition will be displayed on the screen.

## Delete All Logs

You can delete all saved logs.

1. Press <Image src='/img/common/ico-menu.svg' ico alone /> and authenticate with the Admin level credential.

2. On the <Cmd>DASHBOARD</Cmd>, select <Image src='/img/common/ico-eventlog.png' ico alone />. All event logs appear.

3. To delete all logs, select <Image src='/img/common/ico-delete.svg' ico alone /> and then press <Cmd>OK</Cmd>. To return to the previous screen, press <Cmd>Cancel</Cmd>.
