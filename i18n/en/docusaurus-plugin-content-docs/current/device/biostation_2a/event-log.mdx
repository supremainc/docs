Describes how to view event logs generated by the device.

## Search event logs

Set conditions to search logs.

1. Press <Image src='/img/common/ico-menu1.svg' ico alone /> and authenticate with the administrator credential.

2. Select <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_066' product='dev' /> and then press **OK**.

   <Image src='/img/device/bs2a-eventlog.png' />

3. Select the desired item and press <IcFw />/<IcBw /> to change the settings. When you press **OK**, logs matching the conditions appear on the screen.

4. Press **ESC** to return to the previous screen.

## Delete all logs

You can delete all saved logs.

1. Press <Image src='/img/common/ico-menu1.svg' ico alone /> and authenticate with the administrator credential.

2. Select <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_303' product='dev' /> and then press **OK**.

   <Image src='/img/device/bs2a-eventlog1.png' />

3. Press **OK** to delete all logs. Press **ESC** to return to the previous screen.

## View log usage

You can check the status of log usage.

1. Press <Image src='/img/common/ico-menu1.svg' ico alone /> and authenticate with the administrator credential.

2. Select <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_120' product='dev' /> and then press **OK**. You can check the log usage of the device.

   <Image src='/img/device/bs2a-eventlog2.png' />
