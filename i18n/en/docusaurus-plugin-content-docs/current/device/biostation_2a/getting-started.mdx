* **LCD 화면**: 동작을 위한 UI를 제공합니다.

* **숫자/메뉴/ESC 버튼**: 

* **1 ~ 9**: 숫자/문자 입력, 메뉴 선택

* <Image src='/img/common/' ico alone />: 메뉴 진입

ESC: 이전 화면으로 이동, 입력 취소

기능 버튼	근태 기능 키로 사용하거나 하위 메뉴를 선택할 때 사용합니다.

방향/OK 버튼	 : 문자 종류 변경, 경비 시작/해제 설정

 : 문자 종류 변경, 근태 이벤트를 선택하여 근태 등록

 : 숫자/문자 삭제, ID를 사용하여 인증

 : 기호 삽입, 항목 설정, 작업 코드 선택

OK: 항목 선택/설정 저장, 키 메뉴 활성화

LED 상태 표시등	제품의 작동 상태가 표시됩니다.

Green: Authentication success.

Red: Authentication failure.

Blue: Normal operation.

Blue/Green blinking:

DHCP를 사용으로 설정 시 IP 주소를 받아오지 못하는 상태

슬레이브 모드일 때 마스터 장치와 연결이 끊어진 상태

Red/Purple blinking: Device is locked.

노랑색 점멸: 입력 대기 상태

지문 인식부	출입을 위해 지문을 인식시키는 부분입니다.

RF 카드 및 모바일 출입카드 인식부	출입을 위해 RF 카드 또는 모바일 출입카드를 인식시키는 부분입니다.

USB (Type-C) 메모리 슬롯	USB 메모리를 연결합니다.

스피커	소리를 들을 수 있습니다.

Wiegand 입력 (4핀)	Wiegand 입력 케이블을 연결합니다.

Wiegand 출력 (4핀)	Wiegand 출력 케이블을 연결합니다.

RS-485 (4핀)	RS-485 케이블을 연결합니다.

이더넷	이더넷 케이블을 연결합니다.

Input (5핀)	Input 케이블을 연결합니다.

릴레이 (3핀)	릴레이 케이블을 연결합니다.

USB 확장 포트 (5핀)	기타 액세서리 제품을 연결할 수 있습니다.

전원 (2핀)	전원 케이블을 연결합니다.
