# Firmware Version 1.1.0 (Build No. 1.1.0_240104)

Release<Separator/>2024-01-04

## New Features and Improvements

---

1. Supports **Custom Smart Card Layout**.

1. Applied Open SSL 3.0.10.

    - After upgrading the firmware to v1.1.0, it is not able to downgrade to an earlier version of the firmware.

1. Extended the user expiration period to a maximum of '**2037-12-31**'.

1. Improved pop-up message to be displayed when **Timed Anti PassBack** occurs.

1. Improved **Key Menu** screen UI.

1. Improved fingerprint algorithm.

## Bug Fixes

---

<BugLists>

1. When **Relay** is set in the device menu, the door does not open with a mobile card. <Badge>1.0.0</Badge>

1. The device abnormally restarts when changing the device's Network settings in BioStar 2. <Badge>1.0.0</Badge>

1. Cannot connect to a wireless network by manually entering the SSID in **Wireless LAN**. <Badge>1.0.0</Badge>

1. Authentication fails when authenticating with a DESFire AoC while the device is connected as a slave. <Badge>1.0.0</Badge>

1. When authenticating with ID + PIN on a device connected as a slave, the user ID is displayed but the user name is not displayed. <Badge>1.0.0</Badge>

1. The device restarts when formatting and reissuing the user's iClass AoC in BioStar 2. <Badge>1.0.0</Badge>

1. When pressing the OK button consecutively and then pressing the directional key, a beep sound plays, resulting in an authentication failure. <Badge>1.0.0</Badge>

</BugLists>
