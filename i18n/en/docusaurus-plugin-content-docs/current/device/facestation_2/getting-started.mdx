## Name and function of each part

<Image src='/img/device/name-function-each-part-fs2.png' className='none' width='60%' />

* **Microphone**: Transmits the voice of the user when the interphone is connected.

  * Covering or blocking the microphone may interfere with voice transmission.

* **Display**: Provides a UI for operation.

* **Infrared Camera**: Capture the infrared face image.

* **Visual Camera**: Capture an actual face image.

* **RF card and mobile access card authentication unit**: Part to scan a RFID card or mobile access card for entrance.

* **Speaker**: Delivers sound.

* **USB Memory Slot**: Connect the USB memory.

  * The Suprema thermal camera (TCM10-FS2) cannot be used simultaneously with the USB memory.

* **Mini USB Cable Slot**: Support coming soon.

* **Input (4-pin)**: Connect the input cable.

* **RS-485 (4-pin)**: Connect the RS-485 cable.

* **Relay (3-pin)**: Connect the Relay cable.

* **Power Supply (2-pin)**: Connects the power supply cable.

* **Ethernet**: Connect the Ethernet cable.

* **Wiegand input (4-pin)**: Connect the Wiegand Input cable.

* **Wiegand output (4-pin)**: Connect the Wiegand Output cable.
