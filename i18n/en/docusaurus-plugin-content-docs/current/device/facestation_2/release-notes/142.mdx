# Firmware Version 1.4.2 (Build No. 1.4.2_210409)

Release<Separator/>2021-04-12

## New Features and Improvements

---

1. Replaced with a new LCD screen

    - After upgrading a firmware to v1.4.2, you cannot downgrade the firmware to the previous version that includes the kernel and U-Boot.

1. Added minimum and maximum threshold temperature functions.

    - Temperature input range: 1–45℃

    - Default value of minimum threshold temperature: 32℃
