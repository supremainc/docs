# Firmware Version 1.1.0 (Build No. 1.1.0_180304)

Release<Separator/>2018-03-26

## New Features and Improvements

---

1. Support the intrusion alarm zone, Interlock zone and Muster zone.

1. Support the Ethernet zone.

    - Ethernet zone: The zone master role is performed by a master device, not the BioStar 2 server, and establishes the zone using Ethernet communication between the devices.

    - Except Fire Alarm Zone.

1. Support Daylight Saving Time setting.

1. Improves Trigger & Action for duress finger.

1. Improves Private Authentication on AoC.

1. Add and change the function that import & export USB logs and users to be compatible with BioStar 2 V2.6.

    - Before: Provided a folder such as Device Type_Device ID_YYMMDD_HHMM.

    - After: Provides a compressed file such as BioStar2_YYYYMMDD_HHMMSS.tgz.

    - Beginning with V2.6.1 on BioStar 2 platform.

1. Improves to handle the encryption key of the important information stored in database differently from server to server.

1. Support One Device Mode(Legacy).

1. Improves the ability to set the user ID type in SIP server

    - Support number and alphanumeric options for entering up to 32 characters.

1. Support simultaneous use of Wiegand input and output.

1. Support the secure tamper.

1. Support fingerprint enrollment on slave device.

1. Improves user transfer speed.

1. Support SEOS smart cards (Elite Key).

1. Improves performance of SEOS smart card RF reading.

1. Performs the AoC matching when connected to a master device as a slave device.

1. Support Enhanced fake face enrollment block.

1. Support for connecting new devices.

    - BioLite N2, XPass D2, BioEntry P2, BioEntry R2
