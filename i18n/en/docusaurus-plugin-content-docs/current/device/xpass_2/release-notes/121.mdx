# Firmware Version 1.2.1 (Build No. 1.2.1_200612)

Release<Separator/>2020-06-26

## New Features and Improvements

---

1. Support encryption for the blacklist and user ID set as an administrator when migrating user information.

1. Improved not to initialize the network and SSL settings when the device ID is changed.

## Bug Fixes

---

<BugLists>

1. Issue that the value is not sent to the server since corrupted logs occur.

1. Issue that the connection of the slave device is disconnected when the slave device is connected.

1. Issue that the option for card types is deactivated when the device is connected as a slave after the factory reset.

1. Issue that logs of devices are not sent to the server after the factory reset.

</BugLists>
