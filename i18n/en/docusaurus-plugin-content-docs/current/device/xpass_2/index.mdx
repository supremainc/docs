XPass 2 is a compact outdoor RFID device based on Suprema's latest RFID technology and multi-hardware platform. It supports mullion type that can be installed on door frames, gangbox type that can be installed on switch boxes, and keypad type, enabling perfect adaptation to various installation environments.

Additionally, it applies Dual-Frequency (125kHz / 13.56MHz) RFID technology, supporting various standards of cards including EM, MIFARE, DESFire EV1, as well as mobile cards using NFC and BLE, optimized for outdoor installation with IP67 rated waterproof and dustproof performance and IK08 rated vandal-proof structure, allowing installation anywhere users need.
