# Firmware Version 1.3.0 (Build No. 1.3.0_190626)

Release<Separator/>2019-07-12

## New Features and Improvements

1. OSDP Standardization

    - Improved to comply with OSDP V2.1.7 protocol when connecting with 3rd-party controllers.

1. Supports options for selection by card type.

1. Supports the latest version of I/O module Micom (V1.3.1).

1. Firmware upgrade status is added to LED status indicator.
