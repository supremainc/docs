## Name and function of each part

### Internal port configuration

<Image src='/img/camera/name-function-each-part-vionyx.svg' className='none' />

* **Alarm input, relay (5-pin)**: Connect the cable for alarm input and relay output.

* **Audio I/O (4-pin)**: Connect the cable for audio input and output.

* **Ethernet (PoE+)**: Connect the Ethernet cable. Power can be supplied via PoE+.

* **Power, RS-485 (5-pin)**: Connect the cable for power supply and RS-485 communication.

* **Reset button**: Restores all settings to factory defaults. For more information on factory initialization, refer to [Factory default](installation#factory-reset).

* **Micro SD card slot**: Insert the Micro SD card.

* **USB (Type-A)**: Connect USB-based accessories such as Wi-Fi and BLE.

* **Microphone switch**: Use the physical microphone switch to control audio recording.

### LED Status Indicator

The LED on the front of the camera cover shows the device's operating and authentication status using colors and blinking patterns.

<Image src='/img/camera/led-status-indicator-vionyx.svg' className='none' alone />

| LED color | Status |
| :-: | --- |
| Green | Authentication successful |
| Red | Authentication failed |
| Blue | Normal operation |
| Blue/Green blinking | When setting <Cmd sid='ui_012' product='dev' /> to use, unable to retrieve IP address. |

### Microphone switch settings

Use the microphone switch to physically disable the camera's microphone and prevent audio recording.

<Image src='/img/camera/microphone-switch-vionyx.svg' className='none' alone />

* Audio recording is enabled by default.

* Slide the microphone switch to the left to disable the microphone and stop audio recording.

* Audio recording can also be disabled from the Command account.

:::caution

If the microphone switch is physically disabled, audio will not be recorded even if the settings are changed in the Command account.

:::
