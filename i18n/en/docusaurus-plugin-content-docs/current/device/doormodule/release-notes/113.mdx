# Firmware Version 1.1.3 (Build No. 1.1.3_171229)

Release<Separator/>2018-04-13


## New Features and Improvements

---

1. When there is the response data with OSDP Secure Session, it responds with 0x16 for SEC_BLK_TYPE value and responds with 0x18 when there is no data.

1. Verified OSDP compatibility with the latest version of the master device.

    - FS2 AWB/D (V1.1.0)

    - BSA2-OEPW/OHPW/OIPW/OMPW (V1.5.0)

    - BS2-OEPW/OHPW/OIPW/OMPW (V1.6.0)

    - CS40 (V1.1.0)

    - BEW2-OAP/ODP/OHP/OAP(UL) (V1.2.0)

    - BSL2-OE/OM (V1.3.0)

    - BEP2-OA/OD/ODS (V1.1.0)
