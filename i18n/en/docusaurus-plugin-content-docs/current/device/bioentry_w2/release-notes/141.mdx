# Firmware Version 1.4.1 (Build No. 1.4.1_190911)

Release<Separator/>2019-09-18

## Bug Fixes

---

<BugLists>

1. After a user scans and registers a card on a device set as Wiegand Out device, if an existing user authenticates with a credential other than the card, the Wiegand output will behave abnormally.

1. The device restarts if a user authenticates a fingerprint on the device set as below.

    - Byte Order: LSB

    - Wiegand Out: User ID

1. When using firmware V1.4.0 the connection to the I/O device that using the firmware version below is lost.

    - DM-20 FW V1.1.2

    - OM-120 FW V1.0.0

    - Secure I/O 2 FW V1.2.1

1. The master device intermittently reboots when upgrading the firmware of the slave device.

</BugLists>
