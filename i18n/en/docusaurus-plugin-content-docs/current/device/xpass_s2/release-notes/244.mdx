# Firmware Version 2.4.4 (Build No. 2.4.4_180724)

Release<Separator/>2018-09-04

## Bug Fixes

---

1. When formatting MIFARE Plus card to use it as a smart card, the site key of the card gets written differently than the value set on the device.

1. When using as a slave device, the RS-485 connection gets disconnected after changing the device settings and restarting the device.

1. Trigger & Action does not work normally when Bypass mode is set in Wiegand output.

1. Device ID changes to zero and does not work normally.

1. The device fails to boot because the secure chip authentication failed repeatedly.

1. The device cannot receive an IP address from the DHCP server in a specific environment.
