CoreStation 20 is a 2-door access control controller designed for enterprise-grade security. It supports up to 500,000 users and provides attendance management features integrated with BioStar X. Enhanced security features with PoE+ support and a SAM slot are available only in the CS-20P model, supporting flexible installation and secure smart card authentication.
