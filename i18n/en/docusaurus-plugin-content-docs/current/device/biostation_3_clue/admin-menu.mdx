## Dashboard

:::info

When using Suprema CLUe mode, user enrollment, editing, and deletion are available only on the Suprema CLUe server, not on the device.

:::

1. Press <Image src='/img/common/ico-menu.svg' ico alone />, enter the password, and enter the menu.

   You can check the status and information of the device at <Cmd sid='ui_109' product='dev' />. Alternatively, you can press <Cmd sid='ui_174' product='dev' /> to directly enroll a user.

2. To register a user or set up a device, select the desired menu from the bottom of the dashboard.

   <Image src='/img/device/bs3-menu.png' />

:::info

* If <Cmd sid='ui_387' product='dev' /> is not set, pressing <Image src='/img/common/ico-menu.svg' ico alone /> opens the <Cmd sid='ui_109' product='dev' /> screen. For details on how to set the password, refer to [Menu password](settings#menu-password).

* <Cmd sid='ui_142' product='dev' /> can only be confirmed on the BS3-APWB model.

:::
