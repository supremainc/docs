Describes how to view event logs generated by the device.

## Search event logs

Set conditions to search logs.

1. Press <Image src='/img/common/ico-menu.svg' ico alone /> and authenticate with the Admin level credential.

2. Select <Image src='/img/common/ico-eventlog.png' ico alone /> in <Cmd sid='ui_109' product='dev' />. All event logs appear.

3. Press <Image src='/img/common/ico-search.svg' ico alone /> to search a specific log and change the condition.

   <Image src='/img/device/bs3-eventlog.png' />

4. Set the search criteria and click <Cmd sid='ui_037' product='dev' /> to display logs that match the criteria.

## Delete all logs

You can delete all saved logs.

1. Press <Image src='/img/common/ico-menu.svg' ico alone /> and authenticate with the Admin level credential.

2. Select <Image src='/img/common/ico-eventlog.png' ico alone /> in <Cmd sid='ui_109' product='dev' />. All event logs appear.

3. To delete all logs, select <Image src='/img/common/ico-delete.svg' ico alone /> and click <Cmd sid='ui_367' product='dev' />. Press <Cmd sid='ui_321' product='dev' /> to return to the previous screen.
