Provides guidance on the main menus and features accessible with administrator privileges.

## DASHBOARD

1. Press <Image src='/img/common/ico-menu.svg' ico alone /> and authenticate with the Admin level credential.<br/>You can check the status and information of the device at <Cmd sid='ui_109' product='dev' />. Alternatively, you can press <Cmd sid='ui_174' product='dev' /> to directly enroll a user.

2. To register a user or set up a device, select the desired menu from the bottom of the dashboard.

   <Image src='/img/device/bs3-menu.png' />

:::info

* If an administrator is not designated, pressing <Image src='/img/common/ico-menu.svg' ico alone /> will display the <Cmd sid='ui_109' product='dev' /> screen.

* <Cmd sid='ui_142' product='dev' /> can only be confirmed on the BS3-APWB model.

:::
