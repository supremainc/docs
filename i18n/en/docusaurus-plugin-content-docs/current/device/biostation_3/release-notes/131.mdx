# 펌웨어 버전 1.3.1 (빌드 번호 1.3.1_240902)

릴리즈<Separator/>2024-09-03

## 신규 기능 및 개선

---

1. 저장된 내선 번호의 **표시 이름**이 긴 경우 발생하는 문제 개선

    - 내선 번호 목록에서 긴 내선 번호에 말줄임표가 표시되도록 개선

    - 통화 연결 화면에서 **표시 이름**의 중간 부분만 표시되는 문제 개선

    - 통화 중 화면에서 긴 표시 이름이 줄넘김 되도록 개선

1. 얼굴 인증 내 **고급 설정** 화면 UI 개선

1. **RTSP** 로그인 비밀번호 설정 시 필요한 조건에 대한 취약점 강화

1. **비주얼 페이스 이미지 저장하기** 기능 추가

    - 비주얼 페이스에 실제 이미지와 템플릿을 모두 저장하도록 하거나 실제 이미지는 저장하지 않고 템플릿만 저장하도록 설정할 수 있는 기능 지원

1. PIN을 입력할 때 숫자 키의 위치가 무작위로 변경되는 **스크램블 키패드**를 활성화하거나 비활성화할 수 있는 옵션 추가

    - **인증** → **스크램블 키패드**

1. 근거리에서 얼굴 인증 시 발생하는 오인증을 줄이기 위해 근거리에서 얼굴 인증을 방지하는 기능 추가

1. 위조 얼굴 검출 성능 개선

    :::info

    - 위조 얼굴 검출 성능을 개선하여 iBeta Level 1 ISO 30107-3 제시형 공격 탐지 시험에 통과하였습니다.

    - 햇빛이 강한 환경에서는 **위조 얼굴 검출** 기능을 **보통**으로 사용하는 것을 권장합니다.

    :::

1. **장치 정보** 내에 위조 얼굴 검출의 알고리즘 버전을 확인할 수 있는 **FAS_IR_SP** 항목 추가

1. 얼굴 인증 시 마스크를 착용한 사용자는 인증을 거부하도록 설정할 수 있는 **마스크 미착용** 옵션 추가

1. 장치에서 네트워크 정보를 직접 입력하여 무선 네트워크를 추가하고 연결할 수 있는 **무선 네트워크 수동 연결** 기능 추가

1. 연결한 무선 네트워크의 Wi-Fi 신호 강도를 주기적으로 확인하여 홈 화면의 Wi-Fi 신호 강도 아이콘이 갱신되도록 개선
