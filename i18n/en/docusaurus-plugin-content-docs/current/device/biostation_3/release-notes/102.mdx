# 펌웨어 버전 1.0.2 (빌드 번호 1.0.2_230111)

릴리즈<Separator/>2023-01-20

## 신규 기능 및 개선

---

1. 신규 SE 프로세서 및 신규 펌웨어 지원

    - 하위 버전의 펌웨어로 다운그레이드 지원

1. 서버에는 있고 장치에는 없는 사용자를 서버에서 수정했을 때 장치의 이벤트 로그에 **사용자 부분 수정 실패 (서버)** 로그가 간결하게 출력되도록 수정

## 버그 수정

---

<BugLists>

1. 마스터 장치의 릴레이를 출입문에 사용하고 있을 때 슬레이브로 연결된 장치에 인증하면 출입문이 열리지 않는 문제 <Badge>1.0.0</Badge>

1. **시리얼 통신 설정**에서 낮은 **전송 속도**를 설정했을 때 얼굴 인증이 동작하지 않는 문제 <Badge>1.0.0</Badge>

1. 영어, 한국어 외 커스텀 폰트 파일 누락 문제 <Badge>1.0.0</Badge>

1. 얼굴 인증 시 Wiegand 출력값이 이전에 카드로 인증 성공된 다른 사용자 정보로 출력되는 문제 <Badge>1.0.0</Badge>

1. 입력 신호에 의한 경비 시작/해제를 설정한 후 입력 신호가 발생하면 경비/해제 로그와 도어 잠김 로그가 생성되는 문제 <Badge>1.0.0</Badge>

1. 인증 모드를 수정하면 기존에 설정해놓았던 다른 인증 모드가 사라지는 문제 <Badge>1.0.0</Badge>

1. 내선 번호 목록 화면에서 목록 번호와 목록이 겹쳐보이는 문제 <Badge>1.0.0</Badge>

1. SDK를 사용하여 스마트 카드를 발급할 때 비주얼 페이스 템플릿을 추가하면 카드 스캔 화면이 갑자기 사라지는 문제 <Badge>1.0.0</Badge>

</BugLists>
