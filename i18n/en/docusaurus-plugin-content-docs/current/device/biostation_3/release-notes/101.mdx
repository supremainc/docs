# Firmware Version 1.0.1 (Build No. 1.0.1_221020)

Release<Separator/>2022-10-21

## New Features and Improvements

---

1. Improved motion detection performance for low-light environments.

1. Improved Visual Face matching algorithm.

1. Improved all pop-up messages displayed on the device to operate according to the time set in **Message Timeout**.

1. Fixed Korean and English strings in the **RTSP** menu.
