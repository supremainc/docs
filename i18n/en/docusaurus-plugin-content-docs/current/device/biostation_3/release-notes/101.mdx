# 펌웨어 버전 1.0.1 (빌드 번호 1.0.1_221020)

릴리즈<Separator/>2022-10-21

## 신규 기능 및 개선

---

1. 저조도 사용 환경에 대한 행동 감지 성능 개선

1. 비주얼 페이스 매칭 알고리즘 개선

1. 장치에 표시되는 모든 팝업 메시지가 **메시지 표시 시간**에 설정된 시간에 따라 동작하도록 개선

1. **릴레이** 메뉴의 한국어 용어 일부 수정

1. **RTSP** 메뉴의 한국어, 영어 용어 일부 수정
