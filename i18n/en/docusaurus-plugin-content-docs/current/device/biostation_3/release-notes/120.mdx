# Firmware Version 1.2.0 (Build No. 1.2.0_231005)

Release<Separator/>2023-10-05

## New Features and Improvements

---

1. Supports **Template on Mobile**.

1. Improved face anti-spoofing performance when **Operation Mode** is set to **Fusion Matching** and **Fake Detection** is set to **Normal**.

1. Fixed strings in the **Use QR as Card** menu.

1. Separated event code for QR authentication.

1. Added Voice Instruction.

1. Updated Japanese regulatory certification electronic notation.
