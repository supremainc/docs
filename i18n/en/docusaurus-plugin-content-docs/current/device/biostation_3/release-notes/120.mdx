# 펌웨어 버전 1.2.0 (빌드 번호 1.2.0_231005)

릴리즈<Separator/>2023-10-05

## 신규 기능 및 개선

---

1. **템플릿 온 모바일** 지원

1. **작동 모드**가 **퓨전 매칭**으로 설정되어 있고 **위조 얼굴 검출** 기능이 **보통**으로 설정되어 있을 때 위조 얼굴 검출 성능 개선

1. **카드를 QR로 인증하기** 메뉴 용어 수정

1. QR 인증 관련 이벤트 코드 분리

1. 음성 안내 추가

1. 일본 규제 인증 전자 표기 업데이트

## 버그 수정

---

<BugLists>

1. BioStar 2의 **장치에서 서버 연결**에서 설정한 서버 URL이 장치 설정에서 나타나지 않는 문제 <Badge>1.0.1</Badge>

1. 관리자로 메뉴 진입 후 홈 화면으로 이동하여 관리자 ID로 인증할 경우 인증 결과가 출력되지 않는 문제 <Badge>1.0.0</Badge>

1. 이벤트 로그 목록 화면에서 스크롤을 이동하면 화면이 겹쳐 보이는 문제 <Badge>1.0.0</Badge>

1. 슬레이브로 연결된 장치에서 ID + PIN으로 인증할 경우 사용자 ID만 표시되고 이름은 출력되지 않는 문제 <Badge>1.1.1</Badge>

1. 유효하지 않은 리소스 파일로 펌웨어 업그레이드에 성공하는 문제 <Badge>1.0.0</Badge>

1. 펌웨어 업그레이드 시 펌웨어에 포함된 리소스 파일이 업데이트되지 않는 문제 <Badge>1.0.0</Badge>

1. BioStar 2에서 1:N 인증 모드가 없는 마스터 장치로 사용자 정보를 전송하면 슬레이브로 연결된 장치에서 1:N 인증이 실패하는 문제 <Badge>1.0.0</Badge>

1. 마스터 장치를 재시작했을 때 장치의 **이벤트 로그**에 재시작 로그가 생성되지 않는 문제 <Badge>1.0.1</Badge>

1. BioStar 2에서 사용자의 **카드 포맷** 시 장치가 꺼지는 문제 <Badge>1.0.0</Badge>

1. EM 카드를 연속으로 인증하면 장치가 멈추거나 재시작되는 문제 <Badge>1.0.0</Badge>

1. 마스터-슬레이브로 장치가 연결된 상태에서 출입문을 삭제 후 다시 추가했을 때 슬레이브로 연결된 장치에 인증 성공해도 출입문이 열리지 않는 문제 <Badge>1.0.0</Badge>

1. 스케줄 잠금 구역 위반이 발생했을 때 이벤트 로그가 비정상적으로 출력되는 문제 <Badge>1.0.0</Badge>

1. 스마트 카드 바이트 순서 설정 수정 <Badge>1.1.0</Badge>

    - Wiegand 또는 OSDP 출력 사용 시 Wiegand 출력 정보를 사용자 ID로 설정한 경우에도 스마트 카드 바이트 순서에 영향을 받도록 수정

</BugLists>
