Describes how to register, edit, delete, and manage users.

## Registering user information {#registering-user-information}

You can register user information that includes biometric authentication data.

1. Click <Image src='/img/common/ico-menu2.svg' ico alone /> and authenticate with administrator privileges.

2. Select <Cmd sid='ui_166' product='dev' /> → <Image src='/img/common/ico-add2.svg' ico alone /> and set the desired items. User usage information appears.

   <Image src='/img/device/xs2-user-add.png' className='none' />

   * <Image src='/img/common/ico-photo1.svg' ico alone />: You can register a user photo.

   * **Name**: Enter the user name.

   * **ID**: Enter a number between 1 and 4294967295 to register a user ID. If <Cmd sid='ui_168' product='dev' /> is set to <Cmd sid='ui_237' product='dev' />, you can use a combination of alphanumeric characters and symbols `_` `-` as the ID, with a maximum of 32 characters.

   * <Cmd sid='ui_312' product='dev' />: <Badge only>XS2-ODPB, XS2-OAPB</Badge> You can enroll a fingerprint for user authentication. Press <Cmd sid='ui_312' product='dev' /> → <Cmd sid='ui_108' product='dev' /> and follow the on-screen instructions to enroll a fingerprint. To enroll more fingerprints, press <Image src='/img/common/ico-add2.svg' ico alone />. For more information on fingerprint registration, refer to [User fingerprint enrollment](#enrolling-user-fingerprint).

   * <Cmd sid='ui_323' product='dev' />: You can enroll a card for user authentication. Press <Cmd sid='ui_323' product='dev' /> → <Cmd sid='ui_108' product='dev' />, then scan the card to assign it to the user at the RF card authentication unit. To enroll more cards, click <Image src='/img/common/ico-add2.svg' ico alone />.

   * <Cmd sid='ui_040' product='dev' />: You can enter the PIN you want to use. Press <Cmd sid='ui_040' product='dev' /> → <Cmd sid='ui_108' product='dev' /> and enter the PIN you want to use. For confirmation, enter the PIN once more and then press <Cmd sid='ui_037' product='dev' />. To prevent leaks, input a PIN of 4 to 16 digits.

   * <Cmd sid='ui_172' product='dev' />: You can select permissions to assign to the user.

     :::info

     Available menus vary according to the set user level.

     * <Cmd sid='ui_075' product='dev' />: This is a general user level, and the menu is not available.

     * <Cmd sid='ui_300' product='dev' />: You can use all menus.

     * <Cmd sid='ui_194' product='dev' />: You can use the menus <Cmd sid='ui_256' product='dev' />, <Cmd sid='ui_366' product='dev' />, <Cmd sid='ui_280' product='dev' />, <Cmd sid='ui_344' product='dev' />, <Cmd sid='ui_255' product='dev' />.

     * <Cmd sid='ui_171' product='dev' />: You can use the menus <Cmd sid='ui_166' product='dev' />, <Cmd sid='ui_255' product='dev' />.

     :::

   * <Cmd sid='ui_250' product='dev' />: You can set the duration for using the user account.

   * <Cmd sid='ui_363' product='dev' />: <Badge only>XS2-ODPB, XS2-OAPB</Badge> Select to enroll a duress fingerprint. If you are coerced by someone or need to open the door forcefully, authenticating with this fingerprint can send an alarm signal to BioStar X.

   * <Cmd sid='ui_064' product='dev' />: You can change the authentication method for each user.

3. Press <Cmd sid='ui_295' product='dev' /> to complete the registration of user information.

## User fingerprint enrollment <Badge only>XS2-ODPB, XS2-OAPB</Badge> {#enrolling-user-fingerprint}

1. Click <Image src='/img/common/ico-menu2.svg' ico alone /> and authenticate with administrator privileges.

2. Click <Cmd sid='ui_166' product='dev' /> → <Image src='/img/common/ico-add2.svg' ico alone />.

3. Select <Image src='/img/common/ico-add-finger-xs2.png' ico alone /> <Cmd sid='ui_312' product='dev' /> → <Cmd sid='ui_108' product='dev' />.

4. After scanning the fingerprint of the finger to be enrolled, scan the same finger's fingerprint again.

:::info

* Fingerprint enrollment is available only for XS2-ODPB and XS2-OAPB products.

* Place the finger deep enough to contact with the sensor completely.

* Place the center of the fingerprint in the center of the sensor.

* If a finger has a cut or the fingerprint is blurry, select another finger for the fingerprint.

* Enter accurately according to the on-screen instructions when authenticating fingerprints.

* If you make the finger upright so that the contact area with the sensor is decreased or the angle of finger is warped, fingerprint authentication may not be performed.

:::

## Modifying User Information

User Management or Administrator can modify the registered user information. You can add the user's fingerprint or card and modify the PIN and permissions.

1. Click <Image src='/img/common/ico-menu2.svg' ico alone /> and authenticate with administrator privileges.

2. Click <Cmd sid='ui_166' product='dev' /> → <Image src='/img/common/ico-search.svg' ico alone />.

3. Select your search terms. You can search for users with <Cmd sid='ui_299' product='dev' />, <Cmd sid='ui_167' product='dev' />, <Cmd sid='ui_253' product='dev' />, <Cmd sid='ui_312' product='dev' />, <Cmd sid='ui_323' product='dev' />.

   :::info

   Fingerprints can only be searched for on XS2-ODPB and XS2-OAPB products.

   :::

4. Select the user to modify.

5. Modify the information by referring to [Registering user information](#registering-user-information) and press <Cmd sid='ui_037' product='dev' />.

   * To delete a user, select <Image src='/img/common/ico-delete.svg' ico alone /> and press <Cmd sid='ui_037' product='dev' />.

:::info

<Cmd sid='ui_320' product='dev' /> can be registered in BioStar X. For more information on registering access groups, refer to the [BioStar X Administrator Guide](../../platform/biostar_x/settings-access-control-manage-access-group#addAccessGroup).

:::

## Delete All Users

You can delete all registered users at once.

1. Click <Image src='/img/common/ico-menu2.svg' ico alone /> and authenticate with administrator privileges.

2. Select <Cmd sid='ui_166' product='dev' /> → <Image src='/img/common/ico-delete.svg' ico alone />.

3. Select <Cmd sid='ui_303' product='dev' /> to check, then press <Cmd sid='ui_037' product='dev' />.

4. To delete all users, press <Cmd sid='ui_367' product='dev' />.

## View User Usage

You can see registered users and card counts at a glance.

1. Click <Image src='/img/common/ico-menu2.svg' ico alone /> and authenticate with administrator privileges.

2. Click <Cmd sid='ui_166' product='dev' /> → <Image src='/img/common/ico-info2.svg' ico alone />. User usage information appears.

   <Image src='/img/device/xs2-user-usage.png' />

:::info

The number of fingerprints can only be verified on XS2-ODPB and XS2-OAPB products.

:::
