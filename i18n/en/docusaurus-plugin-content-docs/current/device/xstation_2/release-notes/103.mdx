# Firmware Version 1.0.3 (Build No. 1.0.3_210705)

Release<Separator/>2021-07-12

## Bug Fixes

---

<BugLists>

1. It was able to configure the HID Prox card settings for devices (XS2-QDPB, XS2-DPB) that do not support the HID Prox card from the server. <Badge>1.0.0</Badge>

1. The QR/Barcode sensor did not work properly. <Badge>1.0.2</Badge>

1. The user ID was output differently from the registered user ID when authenticating a QR/Barcode set as a blacklist. <Badge>1.0.2</Badge>

1. The existing QR/Barcode authentication option settings were not maintained after restarting the device. <Badge>1.0.2</Badge>

</BugLists>
