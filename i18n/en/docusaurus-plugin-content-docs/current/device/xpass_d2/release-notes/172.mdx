# Firmware Version 1.7.2 (Build No. 1.7.2_231102)

Release<Separator/>2023-11-03

:::caution

Devices with the new BLE chip cannot downgrade to a lower version after upgrading the firmware to v1.7.0 or higher. Devices with existing BLE chips can be downgraded to a lower version.

For more information, check the serial number of the device and contact the [Suprema](https://supremainc.com).

:::

## Bug Fixes

---

<BugLists>

1. When running the Device Manager application v1.1.8 on iOS, certain devices do not connect. <Badge>1.7.0</Badge>

1. Custom smart card byte order is affected by the CSN card byte order setting. <Badge>1.7.1</Badge>

1. In the Device Manager application, there is an issue where the default **Encryption Type** is set to 'DES/3DES' instead of 'AES' when configuring a **DESFire** card in the **Suprema Smart Card Layout** tab. <Badge>1.0.0</Badge>
