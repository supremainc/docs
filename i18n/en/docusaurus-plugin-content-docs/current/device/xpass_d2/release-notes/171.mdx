# Firmware Version 1.7.1 (Build No. 1.7.1_230405)

Release<Separator/>2023-04-05

:::caution

Devices with the new BLE chip cannot downgrade to a lower version after upgrading the firmware to v1.7.0 or higher. Devices with existing BLE chips can be downgraded to a lower version.

For more information, check the serial number of the device and contact the [Suprema](https://supremainc.com).

:::

## New Features and Improvements

---

1. Supports **Custom Smart Card Layout**.

    - XPass D2 firmware v1.7.1 or higher.

1. Supports **Keypad Backlight** configuration.

    - XPass D2 (XPD2-GKDB) firmware v1.7.1 or higher.
