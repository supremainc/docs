Guidance on how to set up the admin account and web server network settings.

## Edit admin account

You can edit the admin account of the CoreStation Setup Manager.

1. Click **SETTINGS**.

2. Click **Admin Account** â†’ <IcEdit />**Edit**.

   <Image src='/img/device/settings-admin-accounts.png' alone />

3. Refer to [Initial Setup of the Device](initial-setup), edit each item, and then click **OK**.

   <Image src='/img/device/settings-edit-user.png' alone />

4. Click **Apply** to save the settings.

## Web server network

You can change the network information of the CoreStation Setup Manager.

1. Click **SETTINGS**.

2. Edit the necessary fields.

   <Image src='/img/device/settings-webserver-network.png' alone />

   <div className='overflow-x col2--20-80'>

   | Name            | Feature description                                                                                            |
   | --------------- | -------------------------------------------------------------------------------------------------------------- |
   | **IP Address**  | Edit the IP address of the CoreStation Setup Manager.<br/>- Only static IP can be used. DHCP is not supported. |
   | **Gateway**     | Edit the gateway of the CoreStation Setup Manager.                                                             |
   | **Subnet Mask** | Edit the subnet mask of the CoreStation Setup Manager.                                                         |

   </div>

3. Click **Apply** to save the settings. The device will restart.
