CoreStation Setup Manager is a web server that manages the network settings of CoreStation or checks the connection status of slave devices, input and output ports, and Wiegand ports. With CoreStation Setup Manager, you can check the status of the network and wiring on-site without having to go to the BioStar X server location while configuring the access control system using CoreStation.

The following features are available in CoreStation Setup Manager.

* IP address setting

* Check CoreStation device information

* Admin settings

* Check the connection status of slave devices

* Monitor input and output, Wiegand port status

* Initialize device, excluding network settings

* Factory reset

## Minimum requirements

Check compatible devices and firmware versions.

* CoreStation firmware version 1.3.1 or higher

* CoreStation 20 firmware version 1.0.0 or higher

:::info

* CoreStation Setup Manager is supported on CoreStation firmware version 1.3.1 or higher, and CoreStation 20 firmware version 1.0.0 or higher. If using a lower version of firmware, connect to BioStar X to upgrade the firmware to the latest version.

* For more details on how to connect and install each device, refer to the guide for the respective product.

  * [**CoreStation**](../corestation_40/index.mdx)

  * [**CoreStation 20**](../corestation_20/index.mdx)

* For more information on CoreStation and CoreStation Setup Manager, contact Suprema support team (CS@suprema.co.kr).

:::
