CoreStation Setup Manager is a web server that manages the network settings of CoreStation or checks the connection status of slave devices, input and output ports, and Wiegand ports. With CoreStation Setup Manager, you can check the status of the network and wiring on-site without having to go to the BioStar X server location while configuring the access control system using CoreStation.
