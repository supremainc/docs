Change the administrator account password to enhance camera access security. Change the password regularly as a security best practice.

Change the password in the following situations.

* When setting up the administrator account for the first time to enhance security

* When renewing the password to comply with routine security policy

* If the password is compromised and requires immediate change

* When replacing the administrator, set a new password

## Check password rules

Check the following rules before setting the password. All rules must be met.

* The password must be at least 12 characters long. Maximum length: 64 characters.

* Include at least one lowercase letter (`a-z`) and one uppercase letter (`A-Z`).

* Include at least one digit (`0-9`) and one special character.

  > Example special characters: `"` `!` `@` `#` `$` `%` `^` `&` `*` `(` `)` `-` `_` `=` `+` `[` `]` `{` `}`

* Do not include sequences of 4 or more consecutive characters (e.g., `1234`, `abcd`) or 4 or more identical characters.

* The username (case-insensitive) must not be included in the password.

  > admin, Admin, ADMIN, Admin123! etc.

* Common weak passwords such as `password`, `qwerty`, `123456` are not allowed.

## Change administrator password

1. Click the <Cmd sid='system' product='cam' /> menu in the upper-left corner of the screen.

2. Click <Cmd sid='account_management' product='cam' /> in the left sidebar.

   <ProdImg src='/img/cam-server/cam-webserver-system-account-manage.png' />

3. In the <Cmd sid='administrator' product='cam' /> section, click <Cmd sid='update' product='cam' />.

4. When the <Cmd sid='change_password' product='cam' /> window appears, enter the current administrator password in the <Cmd sid='current_password' product='cam' /> input field.

   <ProdImg src='/img/cam-server/cam-webserver-system-admin-change-pw.png' />

5. Enter the new password in the <Cmd sid='password' product='cam' /> and <Cmd sid='confirm_password' product='cam' /> input fields, in that order.

6. Click <Cmd sid='ok' product='cam' /> to change the new password.

When a success message appears, the password change is complete.

:::info

* After changing the password, you may need to log in again.

* Password change events are recorded in the system log.

:::

## If you forget the password

Recover a lost administrator password using the following methods.

* **Factory reset**: Running a factory reset restores the administrator account to the default values. Warning: Factory Reset deletes all data. For more information about how to perform a factory reset, see the <Linkto to='docs/device/vionyx/maintenance'>ViOnyx Installation Guide</Linkto>.

* **Contact technical support**: Contact <Support /> for help.
