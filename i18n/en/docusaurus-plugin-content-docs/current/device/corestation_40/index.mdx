CoreStation is a controller that surpasses the limitations of a typical ACU access control system and provides the advantages of biometric authentication. It offers enterprise-level hardware performance capable of storing up to 500,000 users and performing 400,000 fingerprint matches per second. It also supports all access control and attendance management features of the BioStar X platform.
