Describes how to view event logs generated by the device.

## Search event logs

Set conditions to search logs.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_066' product='dev' /> and press **6**.

3. Select the desired item and press **6** to change the setting.

   <Image src='/img/device/bln2-eventlog.png' />

4. Set the search conditions and press **OK** to display the logs that match the conditions.

5. Press **ESC** to return to the previous screen.

## Delete all logs

You can delete all saved logs.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_301' product='dev' /> and press **6**.

3. Press **OK** to delete all logs. Press **ESC** to cancel.

## View log usage

You can check the status of log usage.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_120' product='dev' /> and press **6**.

3. You can check the log usage of the device.

4. Press **ESC** to return to the previous screen.
