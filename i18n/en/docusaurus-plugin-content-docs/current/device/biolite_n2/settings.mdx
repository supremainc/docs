Provides instructions on how to set system and device configurations.

## Display & Sound

You can change the display and sound settings of the device.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_366' product='dev' /> and press **6**.

3. Select the desired item and press **4** or **6** to set the schedule.

   <Image src='/img/device/bln2-displaysound.png' />

   * <Cmd sid='ui_364' product='dev' />: You can choose items to display in the background of the home screen.

   * <Cmd sid='ui_148' product='dev' />: Set the time (sec) to turn off the lighting of the screen.

   * <Cmd sid='ui_131' product='dev' />: Set the time (sec) for the menu screen to disappear automatically. If there is no button input during a set time, the screen will return to the home screen.

   * <Cmd sid='ui_135' product='dev' />: Set the time (sec) for a setting complete message or notification message to disappear automatically.

   * <Cmd sid='ui_227' product='dev' />: You can select the language that will be displayed on the screen.

   * <Cmd sid='ui_251' product='dev' />: You can use voice guidance instead of notification tones.

   * <Cmd sid='ui_153' product='dev' />: Set the volume.

4. Press **OK** to save the settings.

## Device

### Date & Time {#date-time}

You can set date and time. Set the date and time accurately in order to collect accurate log data.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_098' product='dev' /> and press **6**.

3. Select the desired item and press **4** or **6** to set the schedule.

   <Image src='/img/device/bln2-datetime.png' />

   * <Cmd sid='ui_096' product='dev' />: You can check the current date. To modify it manually, disable <Cmd sid='ui_208' product='dev' />.

   * <Cmd sid='ui_207' product='dev' />: You can check the current time. To modify it manually, disable <Cmd sid='ui_208' product='dev' />.

   * <Cmd sid='ui_358' product='dev' />: Set the time reference of the current location.

   * <Cmd sid='ui_208' product='dev' />: Synchronize the server and the time. If you wish to synchronize the server and the time, enable <Cmd sid='ui_208' product='dev' />.

   * <Cmd sid='ui_097' product='dev' />: You can set the date format. You can select <Cmd sid='ui_062' product='dev' />, <Cmd sid='ui_033' product='dev' />, and <Cmd sid='ui_009' product='dev' />.

   * <Cmd sid='ui_209' product='dev' />: You can set the time format. You can select <Cmd sid='ui_005' product='dev' /> and <Cmd sid='ui_238' product='dev' />.

4. Press **OK** to save the settings.

### Daylight saving time

You can use the device by applying daylight saving time. Set the start and end time correctly.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_270' product='dev' /> and press **6**.

3. Select the desired item and press **4** or **6** to change settings.

   * <Cmd sid='ui_253' product='dev' />: Set the name of daylight-saving time.

   * <Cmd sid='ui_217' product='dev' /> / <Cmd sid='ui_218' product='dev' /> / <Cmd sid='ui_216' product='dev' /> / <Cmd sid='ui_215' product='dev' />: Set details of when daylight-saving time begins.

   * <Cmd sid='ui_094' product='dev' /> / <Cmd sid='ui_095' product='dev' /> / <Cmd sid='ui_093' product='dev' /> / <Cmd sid='ui_092' product='dev' />: Set details of when daylight-saving time ends.

4. Press **OK** to save the settings.

### Relay

You can set the input ports of <Cmd sid='ui_144' product='dev' /> and <Cmd sid='ui_015' product='dev' /> on the device. Useful when used as a standalone.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_122' product='dev' /> and press **6**.

   * <Cmd sid='ui_122' product='dev' />: You can set whether relay is enabled or not. Enabling relay settings will allow you to configure <Cmd sid='ui_144' product='dev' /> and <Cmd sid='ui_015' product='dev' />.

   * <Cmd sid='ui_144' product='dev' />: Set the duration for the door to remain open when standard user authentication has been carried out.

   * <Cmd sid='ui_015' product='dev' />: Select the input port connected to the exit button.

   * Switch: Select the type of switch (<Cmd sid='ui_274' product='dev' />N/O or <Cmd sid='ui_035' product='dev' />N/C<Cmd sid='ui_036' product='dev' />) for the Exit Button's input port.

3. Press **OK** to save the settings.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_289' product='dev' /> and press **6**. You can check the information of <Cmd sid='ui_136' product='dev' />, <Cmd sid='ui_282' product='dev' />, <Cmd sid='ui_210' product='dev' />, <Cmd sid='ui_019' product='dev' />, <Cmd sid='ui_017' product='dev' />, <Cmd sid='ui_332' product='dev' />, <Cmd sid='ui_031' product='dev' />.

3. Press **ESC** to return to the previous screen.

### Memory usage

View the status of memory usage.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_133' product='dev' /> and press **6**. You can view the memory usage status of the device.

3. Press **ESC** to return to the previous screen.

### Secure Tamper

You can set whether to use security tamper. When the tamper is turned on, you can set to delete the entire user, the entire log, and the security key.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_150' product='dev' /> and press **4** or **6** to change settings.

3. Press **OK** to save the settings.

### Restart device

The user can restart the device.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_286' product='dev' /> and press **6**.

3. Press **OK** to restart the device. Press **ESC** to cancel.

### Restore default

Device settings, network settings, and operator levels will be reset.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_089' product='dev' /> and press **6**.

   * <Cmd sid='ui_305' product='dev' />: Initializes all settings saved on the device. Press **OK** to initialize all device settings. Press **ESC** to return to the previous screen.

   * <Cmd sid='ui_160' product='dev' />: Initializes all settings except for network settings. Press **OK** to initialize all settings except for network settings. Press **ESC** to return to the previous screen.

   * <Cmd sid='ui_072' product='dev' />: Deletes all information saved in the device and the root certificate, restoring settings to default.

   * <Cmd sid='ui_121' product='dev' />: Deletes the root certificate saved in the device.

3. If you proceed with <Cmd sid='ui_089' product='dev' />, the device will restart.

:::info

* When you reset, the operator level will be reset as well. After resetting, make sure to set the operator level again.

* Language setting will not change after resetting.

* The <Cmd sid='ui_072' product='dev' /> menu can be used when the root certificate is saved in the device.

* The <Cmd sid='ui_121' product='dev' /> menu can be used only when the root certificate is saved in the device and an administrator has been designated.

:::

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_128' product='dev' />.

3. Configure the desired authentication methods.

   <Image src='/img/device/bln2-master-admin-edit.png' width='100%' />

### Administrator two-step authentication {#admin-2-step-authentication}

Devices upgraded to firmware version 1.4.0 do not provide <Cmd sid='ui_128' product='dev' /> settings, but you can enhance the security of the device by strengthening the administrator's permissions through the <Cmd sid='ui_073' product='dev' /> option.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_280' product='dev' /> → <Cmd sid='ui_073' product='dev' />.

   <Image src='/img/device/bln2-administrator-two-step-auth.png' />

:::info

* <Cmd sid='ui_073' product='dev' /> is available after upgrading the firmware to v1.7.0 or later, and the firmware cannot be downgraded to an earlier version after the upgrade.

* The first step of authentication is default unless the <Cmd sid='ui_073' product='dev' /> option is enabled.

* If there are not two types of credentials enrolled for all administrators, activation fails and an error message is displayed. Enroll two types of credentials for all administrators and try again.

  <Image src='/img/device/bln2-master-admin-error-message.png' />

:::

:::caution

If you delete all administrators' credentials to below type 2 after enabling <Cmd sid='ui_073' product='dev' />, you will no longer have access to the administrator menu without being able to connect to BioStar X. Therefore, you need to be particularly careful when deleting administrator credentials.

:::

## Network

You can change the communication settings of devices such as Ethernet, server, and serial communication.

### TCP/IP

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_103' product='dev' /> → <Cmd sid='ui_059' product='dev' /> and press **6**.

3. Select the desired item and press **4** or **6** to change settings.

   <Image src='/img/device/bln2-network-ethernet.png' />

   * <Cmd sid='ui_355' product='dev' />: You can set the ports of the device.

   * <Cmd sid='ui_012' product='dev' />: Set whether to use DHCP. If DHCP setting is disabled, the user can modify <Cmd sid='ui_029' product='dev' />, <Cmd sid='ui_068' product='dev' />, and <Cmd sid='ui_192' product='dev' />.

   * <Cmd sid='ui_029' product='dev' />: You can check the device's IP address. To modify, disable DHCP setting.

   * <Cmd sid='ui_068' product='dev' />: You can check the device's gateway. To modify, disable DHCP setting.

   * <Cmd sid='ui_192' product='dev' />: You can check the device's subnet mask. To modify, disable DHCP setting.

   * <Cmd sid='ui_013' product='dev' />: You can set the domain name server address.

4. Press **OK** to save the settings.

### Server

2. Select <Cmd sid='ui_103' product='dev' /> → <Cmd sid='ui_180' product='dev' /> and press **6**.

3. Select the desired item and press **4** or **6** to change settings.

Users can input <Cmd sid='ui_183' product='dev' />, <Cmd sid='ui_182' product='dev' />, and <Cmd sid='ui_189' product='dev' />.
