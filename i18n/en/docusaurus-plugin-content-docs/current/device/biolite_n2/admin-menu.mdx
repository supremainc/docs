Provides guidance on the main menus and features accessible with administrator privileges.

## Full menu

1. Press **ESC** and authenticate as an administrator.

2. Select your desired menu.

   <Image src='/img/device/bln2-menu.png' />

:::info

If no administrator is specified, press **ESC** to display the menu screen.

:::
