## Name and function of each part

<Image src='/img/device/name-function-each-part-bln2.png' className='none' width='30%' />

* **Display**: Provides a UI for operation.

* **Feature Button**: Used for attendance key, text modification, etc.

* **Keypad**

  * **0-9**: Used to input numbers or characters.

  * **2**, **4**, **6**, **8**: Used to navigate to the desired item.

* **OK Button**: Used to select attendance mode and save settings. If the work code is set, you can press and hold to change the user's work code.

* **ESC Button**: Used to open the menu, go back to the previous screen, and cancel input.

* **Fingerprint Recognition Unit**: The part that recognizes the fingerprint for access.

* **RF card and mobile access card authentication unit**: Part to scan a RFID card or mobile access card for entrance.

* **Cable**

  Power / RS-485 / Ethernet / Relay / Wiegand Input/Output / Input

* **Speaker**: Delivers sound.
