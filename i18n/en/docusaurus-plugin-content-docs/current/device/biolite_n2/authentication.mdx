Describes the supported authentication methods and authentication procedures.

## Authentication mode

Set the authentication mode. You can combine <Cmd sid='ui_312' product='dev' />, <Cmd sid='ui_040' product='dev' />, <Cmd sid='ui_322' product='dev' />, and <Cmd sid='ui_020' product='dev' /> for authentication mode, and you can also set schedules for each authentication mode.

### Fingerprint authentication mode

You can set schedules for each authentication method that uses fingerprints.

1. Press **ESC** and authenticate as an administrator.

2. Press **6** after selecting <Cmd sid='ui_256' product='dev' /> → <Cmd sid='ui_261' product='dev' /> → <Cmd sid='ui_313' product='dev' />.

3. Select the desired item and press **4** or **6** to set the schedule.

   <Image src='/img/device/bln2-auth-authmode.png' />

   * <Image src='/img/common/ico-authmode-finger-bln2.png' ico alone />: This mode uses only fingerprints.

   * <Image src='/img/common/ico-authmode-finger-bln2.png' ico alone /> + <Image src='/img/common/ico-authmode-pin-bln2.png' ico alone />: This mode authenticates with a fingerprint followed by entering a PIN.

4. Press **OK** to save the settings.

:::info

* <Cmd sid='ui_203' product='dev' /> can be set in BioStar X. If you have no scheduled settings, you can only select <Cmd sid='ui_162' product='dev' /> and <Cmd sid='ui_361' product='dev' />.

* For more information about <Cmd sid='ui_203' product='dev' /> settings, refer to the [BioStar X Administrator Guide](../../platform/biostar_x/settings-schedule).

:::

## Attendance Mode

You can set how to register T&A mode.

1. Press **ESC** and authenticate as an administrator.

2. Select <Cmd sid='ui_256' product='dev' /> → <Cmd sid='ui_083_title' product='dev' />, then press **6**.

3. Select the desired item and press **4** or **6** to set the schedule.

   <Image src='/img/device/bln2-auth2.png' />

   * <Cmd sid='ui_084' product='dev' />: You can check attendance events.

   * <Cmd sid='ui_083_list' product='dev' />: Set how to use attendance mode.

   * <Cmd sid='ui_082' product='dev' />: Set to require a user to select a T&A event when authenticating. Setting <Cmd sid='ui_372' product='dev' /> allows you to require selecting a time and attendance event when authenticating.

   * <Cmd sid='ui_080' product='dev' />: You can configure to use only time and attendance events selected by an administrator. Available when <Cmd sid='ui_083_list' product='dev' /> is set to <Cmd sid='ui_071' product='dev' />.

   * <Cmd sid='ui_279' product='dev' />: Set whether to use the job code.

4. Press **OK** to save the settings.

:::info

Set <Cmd sid='ui_279' product='dev' /> to <Cmd sid='ui_372' product='dev' /> to change the user's action code. Long press **OK** then authenticate to select the desired work code.

:::

## Fingerprint Settings

You can change the settings for fingerprint authentication.

1. Press **ESC** and authenticate as an administrator.

2. Press **6** after selecting <Cmd sid='ui_256' product='dev' /> → <Cmd sid='ui_312' product='dev' />.

3. Select the desired item and press **4** or **6** to set the schedule.

   <Image src='/img/device/bln2-auth3.png' />

   * <Cmd sid='ui_149' product='dev' />: You can set the security level for 1:N authentication.

   * <Cmd sid='ui_130' product='dev' />: Set the wait time for fingerprint matching. If authentication is not completed within the specified time, it will fail.

   * <Cmd sid='ui_254' product='dev' />: The original image of the scanned fingerprint can be viewed.

   * <Cmd sid='ui_197' product='dev' />: Set the sensitivity of the fingerprint recognition sensor. Set the sensitivity higher to obtain precise fingerprint information.

   * <Cmd sid='ui_249' product='dev' />: Set the fake fingerprint detection level. Higher levels for fake fingerprint detection may also increase rejection rates for actual people’s fingerprints.

   * <Cmd sid='ui_003' product='dev' />: Set the speed of fingerprint authentication. When you select <Cmd sid='ui_275' product='dev' />, authentication speed is set according to the total number of fingerprint templates enrolled in the device.

   * <Cmd sid='ui_342' product='dev' />: Set the fingerprint template format. <Cmd sid='ui_057' product='dev' /> is set as default. Changing the template format will make all previously saved fingerprints unusable, so take care when changing.

      :::info

      Delete all user fingerprint information and then change the template format. If user fingerprint information is enrolled, you cannot change the template format. 

      :::

   * <Cmd sid='ui_198' product='dev' />: Set to <Cmd sid='ui_277' product='dev' /> to turn on the fingerprint sensor when it detects the user’s finger. When set to <Cmd sid='ui_362' product='dev' />, the sensor remains on even when not scanning fingerprints.

   * <Cmd sid='ui_116' product='dev' />: Assess the quality of scanned fingerprints to save high-quality fingerprint information. Enable to notify the user when fingerprint quality is low, helping them scan fingerprints correctly.

   * <Cmd sid='ui_118' product='dev' />: Check for duplicate fingerprints when enrolling.

4. Press **OK** to save the settings.

## Server matching

You can set up server matching. When server matching is used, authentication is performed on BioStar X without authentication on the device. It can be used when the device has too much user information or when the device is installed outside.

1. Press **ESC** and authenticate as an administrator.

2. Press **6** after selecting <Cmd sid='ui_256' product='dev' /> → <Cmd sid='ui_185' product='dev' />.

   <Image src='/img/device/bln2-auth4.png' />

3. Press **4** or **6** to change the settings.

4. Press **OK** to save the settings.

:::info

* <Cmd sid='ui_185' product='dev' /> can only use <Cmd sid='ui_312' product='dev' />, <Cmd sid='ui_322' product='dev' />, and <Cmd sid='ui_020' product='dev' />.

* If you use <Cmd sid='ui_185' product='dev' />, you must enable server matching on both the device and BioStar X. For more information about BioStar X server matching settings, refer to the [BioStar X Administrator Guide](../../platform/biostar_x/settings-server-server#servermatching).

:::

## Auth result display option

You can set the auth result display option.

1. Press **ESC** and authenticate as an administrator.

2. Press **6** after selecting <Cmd sid='ui_256' product='dev' />.

3. Select the desired item and press **4** or **6** to change settings.

   <Image src='/img/device/bln2-auth5.png' />

   * <Cmd sid='ui_169' product='dev' />: You can select how the user ID is displayed on the authentication result screen of the device.

      * <Cmd sid='ui_138' product='dev' />: Displays all user IDs.

      * <Cmd sid='ui_316' product='dev' />: Displays only the first letter of the user ID.

      * <Cmd sid='ui_357' product='dev' />: Do not display the user ID.

   * <Cmd sid='ui_177' product='dev' />: You can select how the user name is displayed on the authentication result screen of the device.

      * <Cmd sid='ui_138' product='dev' />: Displays all user names.

      * <Cmd sid='ui_316' product='dev' />: Displays only the first letter of the user name.

      * <Cmd sid='ui_357' product='dev' />: Do not display the user name.
