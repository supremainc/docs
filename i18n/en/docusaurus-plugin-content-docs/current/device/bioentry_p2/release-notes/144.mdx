# Firmware Version 1.4.4 (Build No. 1.4.4_220914)

Release<Separator/>2022-09-14

:::caution

- BEP2-OA type of BioEntry P2 model support HID iCLASS card type through SE processor.<br/>For devices with the new SE processor, after upgrading the firmware to v1.4.4 or later, you cannot downgrade to a lower version and downgrade the firmware customized in the lower version. Devices with existing SE processor can be downgraded to a lower version.

- For more information, check the serial number of the device and contact the [Suprema](https://supremainc.com).

:::

## New Features and Improvements

---

1. Supports the dual SE processor.

## Bug Fixes

---

<BugLists>

1. Incorrect Bit Count and card data were output when the Suprema device was connected to a 3rd-party controller via OSDP and the CSN card was registered in Wiegand format and authenticated. <Badge>1.4.3</Badge>

1. Device does not recognize some HID iCLASS Seos cards. <Badge>1.0.0</Badge>

1. Event logs and real-time logs are not output due to corrupted log records. <Badge>1.0.0</Badge>

1. Improved the structure to prevent authentication fail caused by broken database or cache memory. <Badge>1.0.0</Badge>

</BugLists>
