## Name and function of each part

<Image src='/img/device/name-function-each-part-fsf2.png' className='none' width='80%' />

* **LED Lamp**: The LED lamp turns on when it detects a user in the dark.

* **Infrared camera**: Capture the infrared face image.

* **Visual camera**: Capture an actual face image.

* **Display**: Provides a UI for operation.

* **RF card and mobile access card authentication unit**: Part to scan a RFID card or mobile access card for entrance.

* **Speaker**: Delivers sound.

* **Ethernet**: Connect the Ethernet cable.

* **Input (4-pin)**: Connect the input cable.

* **Relay (3-pin)**: Connect the Relay cable.

* **Wiegand Input/Output (4-pin)**: Connect the Wiegand input/output cable.

* **Power supply (2-pin)**: Connects the power supply cable.

* **RS-485 (4-pin)**: Connect the RS-485 cable.

* **SAM Card Slot**: A SAM card can be used.

* **USB (Type-C) memory slot**: Connects the USB memory.
