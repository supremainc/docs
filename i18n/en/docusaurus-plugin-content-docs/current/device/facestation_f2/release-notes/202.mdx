# Firmware Version 2.0.2 (Build No. 2.0.2_220502)

Release<Separator/>2022-05-10

## New Features and Improvements

---

1. Visual Face algorithm v2 support.

1. Improved to detect a mask when enrolling a face so that the face cannot be enrolled with a mask on or covering the mouth.

1. Improved Dynamic ROI (Region of Interest) performance.

1. Added a progress bar to screen during migration.

1. Improved to check for the duplication of cards registered with users when transmitting user data from the server to devices.

1. Improved the Arm/Disarm status is maintained if the device loses power.

## Bug Fixes

---

<BugLists>

1. When upgrading the firmware from v1.x.x to v2.x.x using a USB memory, the upgrade fails, and when the device is rebooted after removing the USB memory, the device stops in the loading state. <Badge>1.0.0</Badge>

    :::info

    When upgrading the firmware using a USB memory inevitably, first set **Menu Timeout** to **Always On** in the **Display & Sound** menu before proceeding with the upgrade. Also, even if a failure message is displayed during the upgrade, DO NOT remove the USB memory and wait until a success message is displayed. The device will restart automatically when the upgrade is complete.

    :::
