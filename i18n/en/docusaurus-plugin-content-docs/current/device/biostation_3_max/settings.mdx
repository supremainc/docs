Provides instructions on how to set system and device configurations.

## Display & Sound

You can change the display and sound settings of the device.

1. Press <Image src="/img/common/ico-menu.svg" ico alone /> and authenticate with the Admin level credential.

2. Press <Cmd sid="ui_193" product="dev" /> → <Cmd sid="ui_366" product="dev" />.

3. Change the desired item.

   <Image src="/img/device/bs3m-display-sound.png" />

   * <Cmd sid="ui_364" product="dev" />: You can choose items to display in the background of the home screen.

   * <Cmd sid="ui_227" product="dev" />: You can select the language that will be displayed on the screen.

   * <Cmd sid="ui_284" product="dev" />: Set whether to use a Private Message, which will be displayed on the screen when the user authenticates. The <Cmd sid="ui_284" product="dev" /> can be set using the BioStar Device SDK. If not set, does not display a message when authentication is successful even if <Cmd sid="ui_284" product="dev" /> is enabled on the device.

   * <Cmd sid="ui_184" product="dev" />: Set whether to use a Private Message, which will be displayed on the screen when the user authenticates. <Cmd sid="ui_184" product="dev" /> can be set using the BioStar X API. If not set, does not display a message when authentication is successful even if <Cmd sid="ui_184" product="dev" /> is enabled on the device.

   * <Cmd sid="ui_365" product="dev" />: Set whether the screen saver is enabled or not. It reduces unnecessary energy consumption by lowering the brightness of the screen when the device is not in use.

   * <Cmd sid="ui_131" product="dev" />: Set the time (sec) for the menu screen to disappear automatically. If there is no button input during a set time, the screen will return to the home screen.

   * <Cmd sid="ui_135" product="dev" />: Set the time (sec) for a setting complete message or notification message to disappear automatically.

   * <Cmd sid="ui_148" product="dev" />: Set the time (sec) to turn off the lighting of the screen.

   * <Cmd sid="ui_251" product="dev" />: You can use voice guidance instead of notification tones.

   * <Cmd sid="ui_153" product="dev" />: Set the volume.

   * <Cmd sid="ui_028" product="dev" />: Adjust the speaker's output volume when using intercom.

   * <Cmd sid="ui_026" product="dev" />: Adjust the microphone's input volume when using intercom.

:::info

For more information on the BioStar Device SDK, refer to the [BioStar Device SDK page](https://kb.supremainc.com/bs2sdk/doku.php) on the Suprema Knowledge Base.

:::

## Network

You can change the network settings of the device, such as Ethernet, server, OSDP, etc.

### Ethernet

1. Press <Image src="/img/common/ico-menu.svg" ico alone /> and authenticate with the Admin level credential.

2. Select <Cmd sid="ui_193" product="dev" /> → <Cmd sid="ui_344" product="dev" /> → <Cmd sid="ui_252" product="dev" />.

3. Change the desired item.

   <Image src="/img/device/bs3m-ethernet.png" />

   * <Cmd sid="ui_012" product="dev" />: Set whether to use DHCP. If DHCP setting is disabled, the user can modify <Cmd sid="ui_029" product="dev" />, <Cmd sid="ui_068" product="dev" />, and <Cmd sid="ui_192" product="dev" />.

   * <Cmd sid="ui_290" product="dev" />: Set the device's port.

   * <Cmd sid="ui_029" product="dev" />: You can check the device's IP address. To modify, disable DHCP setting.

   * <Cmd sid="ui_068" product="dev" />: You can check the device's gateway. To modify, disable DHCP setting.

   * <Cmd sid="ui_192" product="dev" />: You can check the device's subnet mask. To modify, disable DHCP setting.

   * <Cmd sid="ui_111" product="dev" />: Set the DNS server address.

4. To save the settings, press <Cmd sid="ui_037" product="dev" />.

### Server

1. Press <Image src="/img/common/ico-menu.svg" ico alone /> and authenticate with the Admin level credential.

2. Select <Cmd sid="ui_193" product="dev" /> → <Cmd sid="ui_344" product="dev" /> → <Cmd sid="ui_180" product="dev" />.

3. Change the desired item.

   <Image src="/img/device/bs3m-edit-server.png" />

   * <Cmd sid="ui_283" product="dev" />: You can send a connection signal from the device to a server with the input information directly. When <Cmd sid="ui_283" product="dev" /> is enabled, the user can input <Cmd sid="ui_182" product="dev" />, <Cmd sid="ui_183" product="dev" />, and <Cmd sid="ui_189" product="dev" />.

   * <Cmd sid="ui_182" product="dev" />: Enter the IP address of the PC on which BioStar X is installed.

   * <Cmd sid="ui_183" product="dev" />: You can enter <Cmd sid="ui_183" product="dev" /> instead of <Cmd sid="ui_182" product="dev" />.

   * <Cmd sid="ui_189" product="dev" />: Enter the port of the PC on which BioStar X is installed.

4. To save the settings, press <Cmd sid="ui_037" product="dev" />.

### OSDP settings

1. Press <Image src="/img/common/ico-menu.svg" ico alone /> and authenticate with the Admin level credential.

2. Select <Cmd sid="ui_193" product="dev" /> → <Cmd sid="ui_344" product="dev" />.

3. Set the desired item in <Cmd sid="ui_039" product="dev" />.

   <Image src="/img/device/bs3m-osdp-setting.png" />

   * <Cmd sid="ui_049" product="dev" />: Select the RS-485 mode.

This option can be used when <Cmd sid="ui_049" product="dev" /> mode is set to <Cmd sid="ui_088" product="dev" /> or <Cmd sid="ui_205" product="dev" />.

## Device

### IP Intercom

Set the account and extensions of intercom.

1. Press <Image src="/img/common/ico-menu.svg" ico alone /> and authenticate with the Admin level credential.

2. Select <Cmd sid="ui_193" product="dev" /> → <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_025" product="dev" />.

   <Image src="/img/device/bs3m-intercom.png" />

   * <Cmd sid="ui_025" product="dev" />: Set whether the intercom is enabled or not. Enabling intercom settings allows you to configure <Cmd sid="ui_055" product="dev" /> and <Cmd sid="ui_101" product="dev" />.

   * <Cmd sid="ui_055" product="dev" />: Enter each item so that the device can be registered on the SIP server.

   * <Cmd sid="ui_027" product="dev" />: Select the resolution of the intercom video output from the device.

   * <Cmd sid="ui_101" product="dev" />: Add, delete, or edit extension numbers.

   * <Cmd sid="ui_102" product="dev" />: Set whether to display extension numbers.

   :::info

   * If <Cmd sid="ui_356" product="dev" /> is not set, it will not be possible to distinguish receivers.

   * Extension numbers can only enter alphanumeric (case sensitive), and special characters `+` `-` `@` `.`.

   :::

3. To save the settings, press <Cmd sid="ui_037" product="dev" />.

   <Image src="/img/device/bs3m-date-time.png" />

### USB memory

Connect USB Memory and import user information to the device or export log and user information from the device. You can also upgrade the firmware.

1. Press <Image src="/img/common/ico-menu.svg" ico alone /> and authenticate with the Admin level credential.

2. Select <Cmd sid="ui_193" product="dev" /> → <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_061" product="dev" />.

3. Select the desired item and change the setting.

   <Image src="/img/device/bs3m-usb-memory.png" />

   * <Cmd sid="ui_063" product="dev" />: Import user information from the connected USB memory.

   * <Cmd sid="ui_100" product="dev" />: Select information you wish to export to the connected USB memory.

   * <Cmd sid="ui_353" product="dev" />: Upgrade the firmware if the firmware files are saved in the connected USB memory.

:::caution

Do not disconnect the power supply while upgrading the firmware of the device. The device may malfunction.

:::

:::info

* It is recommended to use a USB flash drive formatted in exFAT. USB flash drives in FAT32 format may malfunction when more than 50,000 face credentials are stored.

* Check the amount of free space on your USB flash drive. To save 100,000 face credentials, a minimum of 15 GB of free space is required.

:::

1. Press <Image src="/img/common/ico-menu.svg" ico alone /> and authenticate with the Admin level credential.

2. Select <Cmd sid="ui_193" product="dev" /> → <Cmd sid="ui_280" product="dev" />.

3. Activate <Cmd sid="ui_150" product="dev" /> to delete data of all registered users, all logs, and secure keys when a temper event occurs.

### RTSP

Connect a third-party video management system (VMS) to monitor videos filmed with the product's visual camera.

1. Press <Image src="/img/common/ico-menu.svg" ico alone /> and authenticate with the Admin level credential.

2. Select <Cmd sid="ui_193" product="dev" /> → <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_051" product="dev" />.

3. Select the desired item and change the setting.

   <Image src="/img/device/bs3m-rtsp.png" />

   * <Cmd sid="ui_051" product="dev" />: You can set whether RTSP is enabled or not. When RTSP setting is enabled, the user can configure <Cmd sid="ui_020" product="dev" />, <Cmd sid="ui_155" product="dev" />, <Cmd sid="ui_053" product="dev" />, <Cmd sid="ui_054" product="dev" />, and <Cmd sid="ui_052" product="dev" />.

   * <Cmd sid="ui_054" product="dev" />: Enter the RTSP port number.

When the device is configured as a slave device, the RTSP function may not be available.

### Restart device

The user can restart the device.

1. Press <Image src="/img/common/ico-menu.svg" ico alone /> and authenticate with the Admin level credential.

2. Select <Cmd sid="ui_193" product="dev" /> → <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_286" product="dev" />.

3. Press <Cmd sid="ui_367" product="dev" /> to restart the device. Press <Cmd sid="ui_321" product="dev" /> to return to the previous screen.

### Restore default

Device settings, network settings, and operator levels will be reset.

1. Press <Image src="/img/common/ico-menu.svg" ico alone /> and authenticate with the Admin level credential.

2. Select <Cmd sid="ui_193" product="dev" /> → <Cmd sid="ui_280" product="dev" /> → <Cmd sid="ui_089" product="dev" />.

   * <Cmd sid="ui_304" product="dev" />: Initializes all settings stored on the device. Press <Cmd sid="ui_367" product="dev" /> to initialize all device settings. Press <Cmd sid="ui_321" product="dev" /> to return to the previous screen.

   * <Cmd sid="ui_104" product="dev" />: Initializes all settings except for network settings. Press <Cmd sid="ui_367" product="dev" /> to initialize all settings except for network settings. Press <Cmd sid="ui_321" product="dev" /> to return to the previous screen.

   * <Cmd sid="ui_072" product="dev" />: Deletes all information saved in the device and the root certificate, restoring settings to default.

   * <Cmd sid="ui_121" product="dev" />: Deletes the root certificate saved in the device.

3. If you proceed with <Cmd sid="ui_089" product="dev" />, the device will restart.

:::info

* When you reset, the operator level will be reset as well. After resetting, make sure to set the operator level again.

* Language setting will not change after resetting.

* The <Cmd sid="ui_072" product="dev" /> menu can be used when the root certificate is saved in the device.

* The <Cmd sid="ui_121" product="dev" /> menu can be used only when the root certificate is saved in the device and an administrator has been designated.

:::
