The <Cmd sid='shell.header.launcher' /> page serves as the starting page of **BioStar X**, acting as a central hub for users to access the main features of **BioStar X**. This page is designed to allow users to easily navigate and use all features of **BioStar X**. The UI components of the launcher page are as follows.

<Image src='/img/biostarx-launcher.png' className='none' />

The <Cmd sid='shell.header.launcher' /> page provides links to the <Cmd sid='shell.header.user' />, <Cmd sid='shell.header.arena' />, <Cmd sid='shell.header.data' />, <Cmd sid='shell.header.dashboard' />, and <Cmd sid='shell.header.setting' /> pages.

* <Cmd sid='shell.header.user' />: Efficiently manage users through various features such as user group management, user management, user template management, and checking users by access permissions, thereby enhancing security through permissions. For more information, refer to the [following](how-to-guide-user).

* <Cmd sid='shell.header.arena' />: Monitor devices, elevators, and zones related to access points. Control related features and monitor through real-time video. For more information, refer to the [following](how-to-guide-monitoring).

* <Cmd sid='shell.header.data' />: Check various data and statistical information related to users. For more information, refer to [this document](#).

* <Cmd sid='shell.header.dashboard' />: Users can add and arrange widgets based on data selected from the various data provided by **BioStar X** to create their desired dashboard. For more information, refer to [this document](#).

* <Cmd sid='shell.header.setting' />: Configure **BioStar X** optimized for users' environments through various settings such as device and zone permissions, user permissions, language, and time. For more information, refer to [this document](#).

:::info

For more information about the header area at the top of the screen, refer to the [following](ui-common).

:::
