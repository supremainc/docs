Guidelines for registering new users. You can enter information such as the user's photo, name, email, and phone number, and set the user's access permissions and biometric information.

Proceed in the order of entering user information, setting access permissions, and enrolling credentials.

1. Click <Cmd sid='shell.header.user' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click the <Cmd sid='team.user.newUser' /> button in the upper right corner of the <Cmd sid='shell.header.user' /> page.

3. Please enter user information when the <Cmd sid='team.user.newUser' /> window appears.

4. Enter the user's basic information in the <Cmd sid='team.user.information' /> section.

5. Enroll the user's biometric information in the <Cmd sid='team.user.credential.title' /> section.

6. Set the user's access permissions in the <Cmd sid='team.user.permission.title' /> section.

7. Enter the user's personal message in the <Cmd sid='common.advanced' product='2' /> section.

8. Enter all information and click the <Cmd sid='common.save' /> button.

The registered user appears in the user list. For detailed information on input and settings by section, please refer to the below.

:::info

* Access permissions and biometric information are optional. You can modify or add after saving the basic information.

* To print user information as a card using the card template, click the <Cmd>Print Card</Cmd> button at the top right of the screen. For more information, refer to [this document](#).

* To cancel user registration, click the <Cmd>Cancel</Cmd> button at the top right of the screen.

:::
