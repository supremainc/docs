You can print cards with the desired design by integrating **BioStar X** and cardPresso. cardPresso is professional software for card design and printing. Using cardPresso, you can create various card templates and arrange user information accordingly.

## Before start

To use the card printer feature of **BioStar X**, install the cardPresso program and connect it to **BioStar X**. To use the web printing server function of the cardPresso program, a cardPresso XXL edition license is required.

:::info

* For more information on the license types of the cardPresso program, refer to [the following link](https://www.cardpresso.com/cardpresso-software/editions/).

* The web printing server function is supported only on Windows operating systems.

:::

### Installing and configuring the cardPresso {#installing-configuring-cardPresso}

Follow the steps below to install the cardPresso program and set up the web printing server. 

:::note

The menu path or screen may vary depending on the installation version.

:::

1. Download the latest version of cardPresso compatible with the operating system of the PC to be operated as a web printing server from [cardPresso Download](https://www.cardpresso.com/downloads-cp). 

2. Install the program and activate the license issued by cardPresso.

3. After running cardPresso, create a card template.

   <Image src='/img/common/cardpresso-new-card-template.png' alone caption />

4. Save the created card template to a desired location on the server where **BioStar X** is installed. (ex. *C:\template\example.card*)

5. Right-click on the desktop of the PC where cardPresso is installed, then click **New** → **Shortcut** from the popup menu. {/* New → Shortcut */}

6. Enter the following in the **Item Location** field, then click the **Next** button.

   ```
   "C:\Program Files (x86)\cardPresso\cardPresso.exe" /PRINTSERVER
   ```

   <Image src='/img/cardpresso-print-server-short-cut.png' width={400} />

   :::note

   The path where cardPresso is installed may differ based on the installation environment. After confirming the path where cardPresso is installed, modify it appropriately according to the above path.

   :::

7. Enter a name for the shortcut, then click the **Finish** button.

8. Double-click the cardPresso web printing server shortcut icon created on the desktop to run it.

9. When the cardPresso web printing server window appears, click the **Settings** icon.

   <Image src='/img/biostarx-settings-card-printer-installation2.png' />

   * **Address**: Enter the IP address of the computer acting as the Web Print Server.

   * **Port**: Port number on which cardPresso receive print operations.

   * **Allowed user groups**: Select a user group that has access to the Web Print Server.

10. Click **Save** when the configuration is complete.

11. Click the <Image src='/img/common/ico-card-printer-play.png' ico alone /> in the bottom right corner to start the service. If properly set, the icon changes to <Image src='/img/common/ico-card-printer-stop.png' ico alone /> and a **listen OK** message appears.

    <Image src='/img/biostarx-settings-card-printer-installation3.png' />

    :::note

    Do not close the window until card printing is complete.

    :::

## Connecting cardPresso to BioStar X

To connect cardPresso to **BioStar X**, set it up as shown below.

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. In the left sidebar, click <Cmd sid="common.cardPrinter" product='2' />.

3. Activate the <Cmd sid="common.cardPrinter.cardpressoSetting" product='2' /> to <Cmd sid="common.use" product='2' />, then enter each item.

   <Image src='/img/biostarx-settings-card-printer.png' className='none' />

   * <Cmd sid="common.cardPrinter.id" product='2' />: Enter your cardPresso login ID. (default: ADMIN)

   * <Cmd sid="common.cardPrinter.password" product='2' />: Enter your cardPresso login password. (default: admin)

   * <Cmd sid="common.cardPrinter.ipAddress" product='2' />: Enter the IP address of the PC that runs the cardPresso web print server.

   * <Cmd sid="common.cardPrinter.ipAddressPort" product='2' />: Enter the port number used by cardPresso to receive print operations.

   * <Cmd sid="common.cardPrinter.printerName" product='2' />: Enter the name of the printer to be used as a card printer. It can be found in the Windows **Control Panel** → **Devices and printers**.

   * <Cmd sid="common.cardPrinter.cardTemplate" product='2' />: Click <Cmd sid="button.plusAdd" product='2' /> to add a <Cmd sid="common.cardPrinter.cardTemplate" product='2' />. In the <Cmd sid="setting.server.card_template.configure.th.name" product='2' /> item, enter the full path including the filename and extension of the card template.

     * The file name of the card template can be entered using alphanumeric, and special characters.

     * Up to 20 card templates are supported.

     * Example path: C:\template\example.card

   * <Cmd sid="common.cardPrinter.printTestLabel" product='2' />: Test print the card template.

4. Once you complete the settings, click <Cmd sid="button.apply" product='2' />.

## Test printing the card template

Test print a card template with Administrator information.

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. In the left sidebar, click <Cmd sid="common.cardPrinter" product='2' />.

3. Select the card template you want to print, then click the <Cmd sid="common.cardPrinter.testPrint" product='2' /> button.

   <Image src='/img/biostarx-settings-card-printer1.png' className='none' />

4. Card template with Administrator information is printed.

## Print card templates by user {#printing-card-template-per-user}

Print cards with per-user card templates.

1. Click <Cmd sid='shell.header.user' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click the <Cmd sid='team.user.dialog.newUser' /> button at the top right of the screen. The <Cmd sid='team.user.dialog.newUser' /> window appears.

3. Fill out the form and click the <Cmd sid="button.printCard" product='2' /> button.

   * Each item can be entered with alphanumeric and special characters.

   <Image src='/img/biostarx-settings-card-printer-user.png' />

4. Select the card template you want to print and click the <Cmd sid="common.cardPrinter.print" product='2' /> button.

   <Image src='/img/biostarx-settings-card-printer-user1.png' className='none' />

5. When the popup window asking whether to print the card appears, click the <Cmd sid="common.cardPrinter.ok" product='2' /> button.

   <Image src='/img/biostarx-settings-card-printer-user2.png' className='none' />

New user will be created and a card is printed as a card template with user information.

## Add custom user fields {#adding-custom-user-fields}

Print a card with a card template with custom user field added.

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click <Cmd sid='setting.menu.server.root' /> → <Cmd sid='setting.menu.server.server' /> in the left sidebar.

3. In the <Cmd sid='setting.server.userDeviceManagement' product='2' /> section, click the <Cmd sid="button.plusAdd" product='2' /> button to enter <Cmd sid="setting.server.custom_field.configure.th.name" product='2' /> and select <Cmd sid="setting.server.custom_field.configure.th.type" product='2' />. The order of the <Cmd sid="setting.server.custom_field.title" product='2' /> adds decides the element **ID** of the card template.

   <Image src='/img/biostarx-settings-card-printer-custom-user.png' />

   Based on the BioStar X element ID, **Order 1** is the same as **CUSTOM1**. For more information on the BioStar X attribute ID, refer to [the following](#card-template-attribute-ids).

4. Once you complete the settings, click the <Cmd sid="button.apply" product='2' /> button at the very bottom of the screen.

5. Open the card template file by clicking **File** → **Open Template** in the cardPresso program. Add a new element and enter the **ID** as the **XML item ID name** in the BioStar X element ID order.

   <Image src='/img/biostarx-settings-card-printer-template.png' />

6. Click **File** → **Save** when the configuration is complete.

7. Return to **BioStar X** and select the user to apply the card template on the <Cmd sid='shell.header.user' /> page.

8. You can check the custom user fields you added. Fill in accordance with the form and click the <Cmd sid="button.printCard" product='2' /> button.

   <Image src='/img/biostarx-settings-card-printer-custom-user1.png' />

9. Select the card template you want to print and click the <Cmd sid="common.cardPrinter.print" product='2' /> button.

   <Image src='/img/biostarx-settings-card-printer-user1.png' className='none' />

10. When the popup window asking whether to print the card appears, click the <Cmd sid="common.cardPrinter.ok" product='2' /> button.

    <Image src='/img/biostarx-settings-card-printer-user2.png' className='none' />

User information is saved and the card is printed as a card template with custom user fields added.
