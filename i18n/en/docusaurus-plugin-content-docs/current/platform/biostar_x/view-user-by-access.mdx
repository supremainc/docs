This guide describes how to check users registered for each access permission group. You can check the user's access permissions and edit the user's information.

1. Click <Cmd sid='shell.header.user' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click the <Cmd sid='team.tree.accessExplorer' /> tab in the tree structure menu.

   <Image src='/img/biostarx-user-access-explorer.png' />

3. Click on the desired access permission group in <Cmd>All Access Groups</Cmd>.

You can view users belonging to the selected access permission group in the user list.

:::info

The tree structure menu is composed of the following structure.

<Columns>
<Column><Image src='/img/biostarx-user-access-explorer-tree-level.png' /></Column>

<Column className='padding--l10'>

**All Access Groups**

* Access Group: Access Group

  * Access Level: Access Level

    * Door: Entrance door

    * Schedule: Schedule

  * Floor Level: Floor Level

    * Floor: Floor

    * Schedule: Schedule

</Column>
</Columns>

:::
