This guide covers the features available on the <Cmd sid='shell.header.user' /> page, one of BioStar X's core features. This page introduces various features to effectively manage users through user group management, user management, user template management, and user verification by access permissions, as well as methods to enhance security through permissions. Use the various features to make the most of **BioStar X**.

Click <Cmd sid='shell.header.user' /> on the <Cmd sid='shell.header.launcher' /> page or select <Cmd sid='shell.header.user' /> from the shortcut list at the top left of the screen.
