QR codes or barcodes can be used as user authentication methods.

:::info

* Devices that include scanners capable of QR/Barcode authentication are as follows:

  * X-Station 2 (XS2-QDPB, XS2-QAPB)

* Devices that include cameras capable of QR/Barcode authentication are as follows:

  * X-Station 2 (XS2-ODPB, XS2-OAPB, XS2-DPB, XS2-APB) firmware version 1.2.0 or higher

  * BioStation 3 (BS3-DB, BS3-APWB) firmware version 1.1.0 or higher

  * A separate device license is required to use QR/Barcode authentication with the camera. For more information, refer to [this document](#).

:::

## <Cmd sid='team.user.biostarQr' className='normal' /> {#biostarxqr}

You can issue a QR code containing an encrypted PIN to allow user access.

1. In the <Cmd sid='team.user.credential.title' /> section, click the <IcAdd/> button in the <Cmd sid='team.user.credential.qrBarcode.title' /> field.

2. When the <Cmd sid='team.user.credential.qrBarcode.enrollQrBarcode' /> window appears, select <Cmd sid='team.user.credential.qrBarcode.qrBarcodeType' /> as <Cmd sid='team.user.biostarQr' />.

   <Image src='/img/biostarx-new-user-credential-biostarqr.png' />

3. To enter a card ID directly or automatically generate a card ID, activate the <Cmd sid='team.user.credential.qrBarcode.useRandomId' /> option.

4. Click the <Cmd sid='button.apply' product='2' /> button.

A QR code issued to the user will be sent to the email address registered with their information. The user can scan the QR code included in the email with the device to gain access.

:::info

* To prevent duplicate card ID generation, it is recommended to activate the <Cmd sid='team.user.credential.qrBarcode.useRandomId' /> option in the <Cmd sid='team.user.credential.qrBarcode.inputType' />.

* Before using <Cmd sid='team.user.biostarQr' />, complete the email configuration including SMTP settings. For more information, refer to [this document](#).

* To issue <Cmd sid='team.user.biostarQr' />, the user's email address must be registered in their information.

:::

## <Cmd sid='team.user.credential.qrBarcode.title' className='normal' /> {#qrbarcode}

You can register QR/barcodes issued by other entities for users.

1. In the <Cmd sid='team.user.credential.title' /> section, click the <IcAdd/> button in the <Cmd sid='team.user.credential.qrBarcode.title' /> field.

2. When the <Cmd sid='team.user.credential.qrBarcode.enrollQrBarcode' /> window appears, select <Cmd sid='team.user.credential.qrBarcode.qrBarcodeType' /> as <Cmd sid='team.user.credential.qrBarcode.title' />.

   <Image src='/img/biostarx-new-user-credential-qrbarcode.png' />

3. Input the previously issued QR/barcode in the <Cmd sid='team.user.cardId' /> field.

4. Click the <Cmd sid='button.apply' product='2' /> button.

:::info

The card ID can be up to 32 characters long and can include letters, numbers, and special characters.

:::
