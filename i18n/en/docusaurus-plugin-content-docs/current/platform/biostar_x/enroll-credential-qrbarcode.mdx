QR codes or barcodes can be used as user authentication methods.

:::info

* Devices that include scanners capable of QR/Barcode authentication are listed below.

  * X-Station 2 (XS2-QDPB, XS2-QAPB)

* Devices that include cameras capable of QR/Barcode authentication are included in the list below.

  * X-Station 2 (XS2-ODPB, XS2-OAPB, XS2-DPB, XS2-APB) firmware version 1.2.0 or higher

  * BioStation 3 (BS3-DB, BS3-APWB) firmware version 1.1.0 or higher

  * A separate device license is required to use QR/Barcode authentication with a camera. For more information, refer to [this document](#).

:::

## <Cmd sid='team.user.biostarQr' className='normal' /> {#biostarxqr}

You can issue a QR code containing an encrypted PIN to allow user access.

:::note

**Before you start**

* Complete the email information settings, including SMTP configuration, before using **BioStar QR**. For more information, refer to the [following](settings-email-setting).

* To issue **BioStar QR**, you must register an email address in the user information. For more information, refer to the [following](add-user-basic-info).

:::

1. Click <Cmd sid='button.addQr' product='2' /> in the <Cmd sid='common.credential' product='2' /> section.

2. When the <Cmd sid='user.header.qr' product='2' /> window appears, select <Cmd sid='team.user.biostarQr' /> from the <Cmd sid='user.qrType' product='2' />.

   <Image src='/img/biostarx-new-user-credential-biostarqr.png' />

3. Select <Cmd sid='zone.intrusion.inputType' product='2' />.

   * <Cmd sid='user.random.cardid' product='2' />: Automatically generate a card ID.

   * <Cmd sid='user.manual.input' product='2' />: Enter the card ID manually.

4. Click the <Cmd sid='button.enroll' product='2' /> button.

Send the issued QR code to the email address registered in the user information. The user can scan the QR code included in the email on the device to gain access.

:::info

To prevent duplicate card ID creation, it is recommended to set <Cmd sid='zone.intrusion.inputType' product='2' /> to <Cmd sid='user.random.cardid' product='2' />.

:::

## <Cmd sid='team.user.credential.qrBarcode.title' className='normal' /> {#qrbarcode}

Register a QR/barcode issued by other external issuers for users.

1. Click <Cmd sid='button.addQr' product='2' /> in the <Cmd sid='common.credential' product='2' /> section.

2. When the <Cmd sid='user.header.qr' product='2' /> window appears, select <Cmd sid='team.user.credential.qrBarcode.title' /> from the <Cmd sid='user.qrType' product='2' /> options.

   <Image src='/img/biostarx-new-user-credential-qrbarcode.png' />

3. Select <Cmd sid='common.enterManually' product='2' /> from the <Cmd sid='common.registrationOption' product='2' /> options.

4. Enter the previously issued QR/barcode ID directly in the <Cmd sid='user.cardId' product='2' /> option.

5. Click the <Cmd sid='button.enroll' product='2' /> button.

:::info

In the <Cmd sid='user.cardId' product='2' /> option, you can input up to 32 characters, including English letters, numbers, and special characters.

:::
