Using the <Cmd sid='shell.header.data' /> menu, you can query user information registered in **BioStar X** or events that meet specific criteria, and generate reports on a regular schedule. You can automatically generate the desired report on a regular schedule if needed, and the generated report can be exported as CSV or PDF files or printed.

Click <Cmd sid='shell.header.data' /> or select <Cmd sid='shell.header.data' /> from the shortcut list at the top left of the screen on the <Cmd sid='shell.header.launcher' /> page.
