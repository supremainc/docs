Set layout of smart cards such as MIFARE, iCLASS, DESFire, iCLASS Seos.

## Add smart card {#addSmartformat}

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click <Cmd sid='setting.menu.credential.root' /> â†’ <Cmd sid='setting.menu.credential.cardFormat' /> on the left sidebar.

3. Click <Cmd sid='common.smartCard' product='2' /> in the left card format list.

   <Image src='/img/biostarx-settings-credentail-card-smart.png' />

4. Click <Cmd sid='button.func.addSmartCard' product='2' /> in the top left of the screen.

5. When the <Cmd sid='setting.cardFormat.addNewSmartCard' product='2' /> screen appears, configure each item. For more information, refer to [the following](#smartCardType).

   <Image src='/img/biostarx-settings-credentail-smart-format-add.png' />

6. Once all settings are completed, click <Cmd sid='button.apply' product='2' />.

## Smart card types {#smartCardType}

You can set the desired smart card type in the <Cmd sid='setting.cardFormat.smartCardType' product='2' /> option.

* <Cmd sid='common.supremaSmartCard' product='2' />

  <Image src='/img/biostarx-settings-credentail-smart-format-type.png' />

* <Cmd sid='common.customSmartCard' product='2' />

  <Image src='/img/biostarx-settings-credentail-custom-smart-format-type.png' />

:::info

* Custom smart cards only support MIFARE, DESFire, and FeliCa. 

* FeliCa is only supported on BioStation 3 firmware version 1.3.0 or higher.

:::

## Setting options guide {#setting}

Set the card layout in the <Cmd sid='common.info' product='2' /> section.

<Image src='/img/biostarx-settings-credentail-smart-format-info.png' />

:::info

* The <Cmd sid='setting.cardFormat.primaryKey' product='2' /> and <Cmd sid='setting.cardFormat.secondaryKey' product='2' /> support only hexadecimal. Enter the key values to be used in the right field and click <Cmd sid='setting.cardFormat.convertToHex' product='2' />. Use the converted value as the site key.

* <Cmd sid='setting.cardFormat.convertToHex' product='2' /> is only available when the <Cmd sid='setting.cardFormat.smartCardType' product='2' /> option is <Cmd sid='common.supremaSmartCard' product='2' />.

:::

### Set basic information

* <Cmd sid='common.name' product='2' />: Enter the smart card name.

* <Cmd sid='setting.cardFormat.secondaryKey' product='2' />: Configure whether to use a secondary site key. Set to <Cmd sid='common.active' product='2' /> to enable secondary site key configuration. When a secondary site key is set, it will be used for authentication when the card's primary site key does not match.

  <Image src='/img/biostarx-settings-credential-cart-secondary-key.png' />

##### Layout {#layout}

You can modify the layout that records user information, facial, and fingerprint information.

* <Cmd sid='setting.cardFormat.numberOfTemplate' product='2' />: Set the number of fingerprint templates to include in the layout.

* <Cmd sid='setting.cardFormat.templateSize' product='2' />: Set the number of bytes used by each fingerprint template.

* <Cmd sid='setting.cardFormat.useFace' product='2' />: Choose whether to use the face template.

* <Cmd sid='setting.cardFormat.faceTemplateSize' product='2' />: Set the number of bytes used by the face template.

:::info

<Cmd sid='setting.cardFormat.useFace' product='2' /> is available on FaceStation F2, BioStation 3, and BioEntry W3 models.

:::

#### Custom smart card

Custom smart cards only support MIFARE, DESFire, and FeliCa.

:::info

FeliCa is only supported on BioStation 3 firmware version 1.3.0 or higher.

:::
