You can change the database settings.

1. Click <Start/> → **BioStar X** → **BioStar X Service Manager** on Windows.

2. When the **BioStar X Service Manager** window appears in the web browser, click the <Cmd>SERVICES</Cmd> menu.

3. Click the <Cmd>Stop</Cmd> button for all services.

   <Image src='/img/common/biostarx-tutoria-service-manager-services-stop.png' alone />

   :::note

   Click <Cmd>Stop All</Cmd> to stop all services at once.

   :::

4. Click the <Cmd>DATABASE</Cmd> menu in the left sidebar.

   <Image src='/img/common/biostarx-tutoria-service-manager-database.png' alone />

5. Select the database desired to change the settings for from the database list.

6. In the <Cmd>Database Server Configuration</Cmd> section, you can change the following settings:

   <Image src='/img/common/biostarx-tutoria-service-manager-database-configuration.png' alone />

   * <Cmd>Name</Cmd>: Enter the database name.

   * <Cmd>Description</Cmd>: Enter the database description.

   * <Cmd>DB Type</Cmd>: Select the database type. (MariaDB, MS SQL)

   * <Cmd>Host</Cmd>: Enter the hostname or IP address of the database server.

   * <Cmd>Port</Cmd>: Enter the port number of the database server.

   * <Cmd>AC</Cmd> / <Cmd>TA</Cmd> / <Cmd>Video</Cmd>: Enter the name, user, and password for the AC, TA, and Video databases.

7. Click the <Cmd>Test Connection</Cmd> button in the upper-right corner to check if the database is connected properly.

8. Click <Cmd>Save</Cmd> at the top right of the screen to save settings.
