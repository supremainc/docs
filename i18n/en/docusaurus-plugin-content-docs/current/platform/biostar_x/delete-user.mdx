If you need to delete a user registered in **BioStar X** due to resignation, contract termination, or security violation, follow the steps below.

1. Click <Cmd sid='shell.header.user' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click the checkbox to the left of the user you want to delete from the user list. You can select more than one user.

3. Click the <Cmd sid='common.delete' /> button at the top right of the screen.

4. When the confirmation message appears, click the <Cmd sid='common.acknowledge' /> button.

Confirm that the selected user has been deleted from the user list.

:::info

Delete the user information from the device before deleting the user from the server. For more information on how to delete user information from the device, refer to the [following](delete-user-info-from-device).

:::
