If you have completed the installation of **BioStar X** through the installation package, please access it via a web browser to check if it is functioning properly. **BioStar X** provides a web-based service that allows access anytime and anywhere.

## Log in from a Web Browser {#log-in-from-a-web-browser}

1. Please launch the web browser.

2. Enter `https://127.0.0.1` in the address bar of your web browser.

   :::info

   * If you installed **BioStar X** on another PC, please enter the IP address of that PC. e.g. `https://192.168.0.1`

   * Do not use 'localhost' as the **BioStar X** access address.

   :::

3. When the login screen appears, log in with the administrator account.

   :::info

   * The admin ID is **admin**. 

   * When you first log in, a **Not Secure** warning will be displayed in the address bar of your web browser. To apply the HTTPS protocol, please refer to [this document](#using-https).

   :::

After logging in, you can view the <Cmd sid='shell.header.launcher' /> screen.

<Image src='/img/biostarx-main-launcher.png' alone />

:::info

* It is recommended to use a browser version of Chrome 100 or higher.

* The external IP address of the PC where **BioStar X** is installed can be checked by visiting [this link](https://whatismyipaddress.com/).

* **BioStar X** uses **port 443** by default. If there is a program using port **443**, please close it and try connecting again. If you cannot close the program, please run **BioStar Setting** to change the port number. For more information, refer to [this document](change-port).

:::
