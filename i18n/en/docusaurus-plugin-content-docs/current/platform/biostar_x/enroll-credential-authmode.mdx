Users can flexibly set authentication methods. You can use the default authentication method set on the device or designate individual authentication modes tailored to each user. Additionally, the extended authentication mode, which includes both facial and fingerprint authentication, enhances security levels. 

## Device defaults

Setting the <Cmd sid='common.authMode' product='2' /> option to <Cmd sid='user.deviceDefault' product='2' /> allows for authentication based on the method configured on the device.

<Image src='/img/biostarx-new-user-credential-authmod-default.png' />

## Private authentication mode

Setting the <Cmd sid='common.authMode' product='2' /> option to <Cmd sid='user.privateMode' product='2' /> enables different authentication methods for each user.

<Image src='/img/biostarx-new-user-credential-authmod-private.png' />

Click the <Cmd sid='button.addplus' product='2' /> button to open the <Cmd sid='device.addNewAuthMode' product='2' /> window. Configure the desired authentication methods.

<Image src='/img/biostarx-new-user-credential-authmod-private-add.png' />

* <Cmd sid='user.authMode.toggle' product='2' />: Set whether to use the extended authentication mode. The extended authentication mode allows for a combination of facial and fingerprint authentication methods.

* <Cmd sid='common.authMode' product='2' />: Set the authentication methods using drag and drop.

  <Image src='/img/biostarx-new-user-credential-authmod-private-join.png' />

To enroll the configured authentication methods, click the <Cmd sid='button.apply' product='2' /> button.

:::info

* The <Cmd sid='user.authMode.toggle' product='2' /> option is supported on the FaceStation F2, BioStation 3, and BioEntry W3.

* Setting <Cmd sid='user.includeDeviceDefaultAuthMode' product='2' /> allows the use of both the authentication modes set on the device and the private authentication modes configured in **BioStar X**.

  <Image src='/img/biostarx-new-user-credential-authmod-note-image.png' />

* Setting <Cmd sid='user.excludeDeviceDefaultAuthMode' product='2' /> allows only the private authentication modes set in **BioStar X** to be used.

:::
