If you have purchased a **BioStar X** license, you can register the license key and use more features. Go to <Cmd>Settings</Cmd> → <IcLicense/> <Cmd>License</Cmd> → <Cmd>BioStar X License</Cmd> menu.

The method for activating the **BioStar X** license varies depending on the user's network environment. Check your network status and activate your license according to the provided instructions.

:::info

* For more information about <Cmd>License</Cmd>, refer to the [following](settings-license).

* For more information on the license policy, refer to [this document](licensing).

:::

## Registering in online {#registering-in-an-online-state}

To activate the **BioStar X** license online while connected to the internet, please enter your name and the activation key you received, then click the <Cmd sid='setting.server.license.button.activate' product='2' /> button.

<Image src='/img/biostarx-tutorial-license-online.png' />

## Registering in offline {#registering-offline}

To activate the **BioStar X** license in a closed network environment or in an offline state with limited internet access, please follow the instructions below.

1. Click <Cmd sid='setting.server.license.button.requestOfflineKey' product='2' /> in the <Cmd>Activation</Cmd> section.

   <Image src='/img/biostarx-tutorial-license-offline.png' />

2. When the <Cmd sid='setting.server.license.title.activateLicenseOffline' product='2' /> window appears, click <Cmd sid='setting.server.license.button.requestOfflineKey' product='2' />.

   <Image src='/img/biostarx-tutorial-license-offline-popup.png' />

3. When the dialog box appears, enter the company name (<Cmd>Request by</Cmd>) and the activation key (<Cmd>License Key</Cmd>).

   <Image src='/img/biostarx-tutorial-license-offline-popup-filed.png' />

4. Click the <Cmd sid='setting.server.license.button.download' product='2' /> button to download the license request file (*\*.req*).

5. Please send it to the regional dealer.

When you receive the license file (*\*.lic*) from the regional dealer, click the <Cmd sid='setting.server.license.button.activate' product='2' /> button to upload the license file.

:::info

If you do not have an activation key, enter only the company name (<Cmd>Request by</Cmd>).

:::
