If you have purchased a **BioStar X** license, you can register the license key and use more features. Go to <Cmd>Settings</Cmd> → <IcLicense/> <Cmd>License</Cmd> → <Cmd>BioStar X</Cmd> menu.

The method for activating the **BioStar X** license varies depending on the user's network environment. Check your network status and activate your license according to the provided instructions.

## Registering in Online {#registering-in-an-online-state}

To activate the **BioStar X** license online while connected to the internet, please enter your name and the activation key you received, then click the <Cmd sid='setting.server.license.button.activate' product='2' /> button.

## Registering in Offline {#registering-offline}

To activate the **BioStar X** license in a closed network environment or in an offline state with limited internet access, please follow the instructions below.

1. In the <Cmd sid='setting.server.license.visitor' product='2' /> section, click the <Cmd sid='setting.server.license.button.requestOfflineKey' product='2' /> button.

2. When the input window appears, enter the company name and activation key.

3. Click the <Cmd sid='setting.server.license.button.download' product='2' /> button to download the license request file (*\*.req*).

4. Please send it to the regional dealer.

When you receive the license file (*\*.lic*) from the regional dealer, click the <Cmd sid='setting.server.license.button.activate' product='2' /> button to upload the license file.

:::info

If you do not have an activation key, just enter the company name.

:::
