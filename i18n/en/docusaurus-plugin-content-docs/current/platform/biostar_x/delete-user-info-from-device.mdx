Provides a feature to delete user information registered on the **BioStar X** server from the device for access control. User information can be deleted from each device.

1. Click <Cmd sid='shell.header.user' /> on the <Cmd sid='shell.header.launcher' /> page.

2. In the user list, click the checkbox to the left of the user you want to delete from the device.

3. Select the <IcMoreW/> button at the top right of the screen and click the <Cmd sid='user.header.removeFromDevice' product='2' /> button.

4. When the <Cmd sid='user.header.removeFromDevice' product='2' /> window appears, select the device from which to delete the selected user information.

5. Click the <Cmd sid='common.delete' /> button.

:::info

* You can also search for devices by clicking the search icon.

* The <Cmd sid='user.header.removeFromDevice' product='2' /> button is enabled only when the <Cmd sid='setting.server.automaticUserSync' product='2' /> option is set to <Cmd sid='device.Input.unsupervised' product='2' />.

* User information is deleted only from the device and remains on the **BioStar X** server.

:::
