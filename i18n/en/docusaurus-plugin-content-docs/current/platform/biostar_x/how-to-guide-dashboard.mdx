The dashboard can be customized to suit individual preferences by allowing each user to select the information they want, add widgets, and freely configure and arrange the widgets.

:::info

* For each user, the <Cmd>Dashboard</Cmd> can be configured for them personally.

* Information available on the <Cmd>Dashboard</Cmd> may vary depending on <Cmd sid='user.bioStarOperator' product='2' />.

:::
