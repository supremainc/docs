The dashboard can be customized to suit individual preferences by allowing each user to select the information they want, add widgets, and freely configure and arrange the widgets.

:::info

* Each user can configure <Cmd sid='shell.header.dashboard' /> individually.

* The information available on <Cmd sid='shell.header.dashboard' /> may vary depending on <Cmd sid='user.bioStarOperator' product='2' />.

:::
