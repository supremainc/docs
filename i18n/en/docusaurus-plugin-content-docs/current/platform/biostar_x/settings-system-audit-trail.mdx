Audit trail tracks user access information as well as all the information changed in the system. You can set a filter for each item for sorting.

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. In the left sidebar, click the <Cmd sid="monitoring.auditTrail" product='2' />.

3. Edit the necessary fields.

<Image src='/img/biostarx-settings-system-audit-trail.png' className='none' />

<Calloutlists>

1. **Period**: You can set the previous 1 month or 3 months conveniently for the period.

2. **Filter**: You can set conditions for each filter item. Click <Cmd sid="ta.report.condition.saveFilter.button" product='2' /> to save the filter.

3. **Page Navigation Buttons and Number of List Rows**: You can move a page or set the number of list rows to be displayed on one page.

   * <Image src='/img/common/ico-audit-trail-first-page.png' ico alone />: Go to the first page.

   * <Image src='/img/common/ico-audit-trail-previous-page.png' ico alone />: Go to the previous page.

   * <Image src='/img/common/ico-audit-trail-nav-page.png' ico alone />: Enter the page number to go to.

   * <Image src='/img/common/ico-audit-trail-next-page.png' ico alone />: Go to the next page.

   * <Image src='/img/common/ico-audit-trail-last-page.png' ico alone />: Go to the last page.

   * <Image src='/img/ico-audit-trail-rows-page.png' ico />: Set the number of list rows to be displayed on one page.

4. **Function Buttons (CSV Export, Column Setting)**: You can save the list of audit list as a CSV file or changes the column setting.

5. **Audit List**: Shows the audit list.

</Calloutlists>
