<Cmd sid='common.dst' product='2' /> (DST) is a function that adjusts the time to better utilize natural daylight.

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. In the left sidebar, click the <Cmd sid="setting.systemSecurity" product='2' />.

3. Click <Cmd sid="button.plusAdd" product='2' />.

4. You can set the <Cmd sid="setting.systemSecurity.allowPasswordChange" product='2' /> up to 10 times.

   <Image src='/img/biostarx-settings-system-daylight-saving-time.png' className='none' />

5. Click <Cmd sid="button.apply" product='2' /> to save the settings.

:::info

You cannot edit or delete a daylight saving time that is already in use.

:::
