**BioStar X** displays various information such as user lists and event lists in a table format. Users can click on the column header area to sort the list, resize the columns, and select or hide columns displayed in the list.

## Show/Hide Columns {#columnvisibility}

You can select only the desired items from the various information displayed in the table on the list-type page to show on the screen. For example, the <Cmd sid='shell.header.user' /> page displays the user ID, name, email address, and the status of enrolled credentials in columns. Users can use the <Cmd sid='common.dialog.columnSetting.title' /> feature to select and view only the information they need, and they can save the settings as a template for future use.

1. Click the <IcMoreW /> button at the top of the list.

2. Click <Cmd sid='common.dialog.columnSetting.title' />.

3. Select the desired number from <Cmd sid='common.dialog.columnSetting.layout' /> when the <Cmd sid='common.dialog.columnSetting.title' /> window appears.

   <Image src='/img/biostarx-common-ui-column-layout.png' />

4. Select or deselect the desired items in <Cmd sid='common.dialog.columnSetting.dialogSettings' />.

5. Click the <Cmd sid='common.apply' /> button.

:::info

* The configuration of the <Cmd sid='common.dialog.columnSetting.dialogSettings' /> may vary depending on the page.

* The number selected in <Cmd sid='common.dialog.columnSetting.layout' /> represents a template. For example, selecting number 1 in <Cmd sid='common.dialog.columnSetting.layout' /> will apply the column layout of template number 1 to the list. Users can select and save templates from 1 to 8.

* To cancel the set content and revert to the default value, click the <Cmd sid='common.dialog.columnSetting.reset' /> button.

:::

## Load Column Layout {#columnlayoutload}

If you have set a custom column layout on the list-type page, you can load the saved layout as needed.

1. Click the <IcMoreW /> button at the top of the list.

2. Click <Cmd sid='common.dialog.columnSetting.title' />.

3. Select the desired number in <Cmd sid='common.dialog.columnSetting.layout' />.

4. Click the <Cmd sid='common.apply' /> button.

The column layout of the selected template will be applied to the list.
