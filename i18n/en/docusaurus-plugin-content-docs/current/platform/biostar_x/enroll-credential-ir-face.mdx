You can enroll the user's face as a means of authentication for access control. IR face enrollment is an authentication method that captures the user's face using an infrared camera.

:::note

**Precautions when enrolling IR face**

import EnrollingFace from '../../common/_enrolling-face-warning.mdx';

<EnrollingFace />

:::

## Enroll by device {#enrolldevice}

Click <Cmd sid="button.addIRFace" product="2" /> in the <Cmd sid="common.credential" product="2" /> section. When the <Cmd sid="userCredential.enroll.IRfaceEnroll" product="2" /> window appears, set each item and click <Cmd sid="button.enroll" product="2" />.

<Image src="/img/biostarx-new-user-credential-ir-face.png" />

* <Cmd sid="common.device" product="2" />: Select the device for enrolling the IR face.

* <Cmd sid="userCredential.enroll.poseSensitivity" product="2" />: Set sensitivity to the face's position, angle, and distance when enrolling the IR face. Set it high to obtain a precise IR face template.

* <Cmd sid="button.addplus" product="2" />: Click to enroll the IR face. Add up to five IR faces.

* <Cmd sid="button.scan" product="2" />: Click the added <Cmd>Nth</Cmd> button to scan the IR face. Follow the on-screen instructions to scan the face.

  <Image src="/img/biostarx-new-user-credential-ir-face-nth.png" />

* <Cmd sid="userCredential.enroll.useProfile" product="2" />: Check this option to use the enrolled IR face image as the profile image.

* <Cmd sid="button.delete" product="2" />: Delete the IR face credential. Select the number to delete and click the button.

:::info

* Devices that can enroll IR face are as follows:

  * FaceStation F2

  * FaceLite

  * BioStation A2

  * BioStation 2

  * BioStation L2

  * Thermal Camera

* If the face authentication rate is low, delete face information and enroll a new face.

:::
