You can import the logs of the device to the **BioStar X** server to check events. This feature is available when the <Cmd sid='setting.server.logUploadType' product='2' /> option is set to <Cmd sid='common.manual' product='2' /> in the <Cmd sid='setting.server' product='2' /> settings.

:::info

For more information on the <Cmd sid='setting.server.logUploadType' product='2' /> option, refer to [the following](settings-server-server#basic-setting).

:::

## Import manually

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click <Cmd sid='setting.menu.event.root' /> → <Cmd sid='setting.menu.event.eventLogImport' /> in the left sidebar of the screen.

3. Select the period for fetching logs in <Cmd sid='setting.eventLogImport.UpdateLog.label' />.

   <Image src='/img/biostarx-settings-event-log-from-device.png' />

   * <Cmd sid='setting.eventLogImport.UpdateLog.dropdown.SinceLast' product='2' />: Imports logs since the date of the last log saved to the **BioStar X** server.

   * <Cmd sid='setting.eventLogImport.UpdateLog.dropdown.All' product='2' />: Imports all logs from the device.

   * <Cmd sid='setting.eventLogImport.UpdateLog.dropdown.UserDefined' product='2' />: You can select the start and end dates to import logs.

4. Click the <Cmd sid='button.updateLog' product='2' /> button to import the logs.

## Import from external storage

You can load event logs stored on an external storage device (USB) into **BioStar X**.

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click <Cmd sid='setting.menu.event.root' /> → <Cmd sid='setting.menu.event.eventLogImport' /> in the left sidebar of the screen.

3. Click the <Cmd sid='button.browse' product='2' /> button for the <Cmd sid='menu.dataImport' product='2' /> item.

   <Image src='/img/biostarx-settings-event-log-import-data.png' />

4. Select the data file (*.tgz*) saved locally.

A message will appear on the screen if the data file is successfully imported.

:::info

* Data exported from devices using an outdated firmware version cannot be imported into **BioStar X**. Always use the latest version of the firmware.

* Only data files exported from FaceStation F2, FaceStation 2, FaceLite, BioStation A2, BioStation 2, X-Station 2, BioStation 3 models can be imported.

* If the imported event logs are not from access points, elevators, or zones set in the **BioStar X** system, some information may appear blank.

:::
