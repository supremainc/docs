This guide explains how to check the list of doors accessible by users. By checking the access permission status, you can manage the doors a user can access and the permissions for those doors. Use the filter feature to sort the desired results, and you can save the list as a CSV file. 

## Check access permission status

Access permissions can be viewed by door or floor, or by user.

<Image src='/img/biostarx-settings-access-control-status.png' />

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click <Cmd sid='setting.menu.accessControl.root' /> in the left sidebar.

3. Click the <Cmd sid='common.status' product='2' /> tab.

4. Click the desired item from the list of statuses under the <Cmd sid='common.status' product='2' /> tab.

   * <Cmd sid='access_control.status.access_control_status_by_door.list_name' product='2' />

   * <Cmd sid='access_control.status.access_control_status_by_user.list_name' product='2' />

   * <Cmd sid='ac.status.elevatorbyFloor.title' product='2' />

   * <Cmd sid='ac.status.elevatorbyUser.title' product='2' />

You can check the access permission status for the selected item.

## Save filter

To sort the desired results, click the <IcFilter2 /> button in the header of the access permission list table. You can add filter conditions.

After setting the filter, you can save it. Click <Cmd sid='monitoring.saveFilter' product='2' />. The saved filter can be checked in the status list under the <Cmd sid='common.status' product='2' /> tab.

<Image src='/img/biostarx-settings-access-control-status-filter.png' />

* Click <IcEditUL /> to rename the filter.

* Click <IcTrash2 /> to delete the filter.

## Export to CSV

You can export the current access permission status list as a CSV file. Click <IcMoreW /> at the top right of the screen and select <Cmd sid='button.downloadToCsv' product='2' />.

## Set columns

You can hide or display desired columns in the current access permission status list table. Click <IcMoreW /> at the top right of the screen and select <Cmd sid='columnSetting.dialog.header' product='2' />.

<Image src='/img/biostarx-settings-access-control-status-column-setting.png' />

* Select the columns to display from <Cmd sid='columnSetting.columnList' product='2' />. The selected columns will appear in the list table. The unselected columns will be hidden from the list table.

* Click <Cmd sid='columnSetting.defaultColumn' product='2' /> to initialize the settings.

Once all settings are complete, click <Cmd sid='button.apply' product='2' />.
