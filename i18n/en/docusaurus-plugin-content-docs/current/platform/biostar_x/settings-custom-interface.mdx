By adding the Quick Action button in the <Cmd sid="common.customInterface" product='2' /> menu, you gain the ability to conveniently control multiple doors simultaneously. Simply click the Quick Action button on the BioStar X main screen to initiate the command.

To add a <Cmd sid="common.triggerAndAction" product='2' /> for Quick Action, refer to the [following](settings-triger-and-action).

## Adding the Quick Action button to the BioStar X main screen

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. In the left sidebar, click the <Cmd sid="common.customInterface" product='2' />.

3. In the <Cmd sid="common.quickActionLayout" product='2' />, click <Cmd sid="button.plusAdd" product='2' /> to enter and select each item.

   <Image src='/img/settings-custom-interface-add-quick-action.png' className='none' />

   * <Cmd sid="settings.customInterface.quickAction.operatorLevel" product='2' />: Select the <Cmd sid="common.quickAction" product='2' /> that can run the added <Cmd sid="settings.customInterface.quickAction.operatorLevel" product='2' />. You can select multiple options.

   * <Cmd sid="settings.customInterface.quickAction.triggerAction" product='2' />: Select the <Cmd sid="settings.customInterface.quickAction.triggerAction" product='2' /> to run. You can select multiple options.

   * <Cmd sid="settings.customInterface.quickAction.confirmBeforeRun" product='2' />: Enabling this option prompts a confirmation pop-up when pressing the <Cmd sid="common.quickAction" product='2' /> button on the main screen, asking whether to proceed with running the <Cmd sid="common.quickAction" product='2' />.

4. Click <Cmd sid="button.apply" product='2' /> to save the settings.

5. Click <Cmd sid="button.apply" product='2' /> on the <Cmd sid="common.customInterface" product='2' /> screen. The <Cmd sid="common.quickAction" product='2' /> button added to the top bar of BioStar X will be created.

   <Image src='/img/settings-custom-interface-main-quick-action.png' className='none' />

   ```
   The <Cmd sid="common.quickAction" product='2' /> button that appears may vary depending on the <Cmd sid="user.bioStarOperator" product='2' /> set for the user.
   ```

6. Clicking the <Cmd sid="common.quickAction" product='2' /> button, created on the top bar of the main screen, will execute the predefined actions set in <Cmd sid="common.triggerAndAction" product='2' />.

## Editing the Quick Action buttons

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. In the left sidebar, click the <Cmd sid="common.customInterface" product='2' />.

3. Click the <IcEditUL /> icon in the right column of the Quick Action button to edit, then edit the desired part and click <Cmd sid="button.apply" product='2' />.

   <Image src='/img/settings-custom-interface-quick-action-layout.png' />

4. Click <Cmd sid="button.apply" product='2' /> on the <Cmd sid="common.customInterface" product='2' /> screen to apply the changes.

## Deleting the Quick Action button

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. In the left sidebar, click the <Cmd sid="common.customInterface" product='2' />.

3. Click the <IcTrash2 /> icon in the right column of the Quick Action button to delete.

   <Image src='/img/settings-custom-interface-confirm.png' className='none' />

4. Check the deletion confirmation pop-up and click <Cmd sid="button.yes" product='2' />.

5. Click <Cmd sid="button.apply" product='2' /> on the <Cmd sid="common.customInterface" product='2' /> screen to apply the deletion.
