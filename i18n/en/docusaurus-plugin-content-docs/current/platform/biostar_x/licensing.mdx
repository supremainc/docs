**BioStar X** is the next generation access control software developed as a successor to **BioStar 2**. Supports various deployment environments of today and the future through an enhanced system architecture, expanded features, and a more flexible licensing model.

**BioStar X** offers a total of five scalable license levels ranging from **Starter** to **Elite**, allowing you to choose according to various environments from small businesses to large corporate settings. Designed in a modular structure, it can add a variety of optional features such as video integration, mobile access, and multi-server support.

## Package {#package}

The following items can be purchased individually from **Advance**, **Enterprise**, and **Elite** license tiers.

* **Advanced Access Control**: This is a package that allows you to set zones such as Anti-Passback, Fire Alarm, Intrusion Alarm, Mustering, Occupancy Limit, Elevator, and Interlock, and use advanced access control features.

## Notes {#reference}

* **Controlling Door Schedules**: Available at no additional cost for licenses that support the door access. In **BioStar 2**, the **Zone** feature is only available when the Advanced or higher license supports it.

* **Plugin/SDK License**: If this feature is implemented, a license will be granted separately.

* **Single Sign-On (SSO) and Multi-Factor Authentication (MFA)**

  * Implementing SSO or multi-factor authentication using external providers requires a separate license.  

  * Multi-factor authentication using devices by Suprema is offered free of charge.

* **Advanced CSV Data Import**: This feature may be offered for a fee if implemented.

* **Other Premium Support Request**: Requests for Suprema's premium support may incur additional charges. Please refer to the below.

  * Secured communication using a custom certificate

  * Applying a custom certificate for HTTPS

  * System upgrade

* **Lists that may incur additional costs (TSS)**

  * Custom system integration or API development

  * On-site support service

  * Specialized training or certification

  * Performance tuning and Optimization

  * System backup and Disaster recovery configuration

  * Change settings after installation
