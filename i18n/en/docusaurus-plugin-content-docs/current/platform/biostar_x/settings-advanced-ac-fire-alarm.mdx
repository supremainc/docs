Guidelines on how to set up fire alarms. The fire alarm is a feature that allows you to set all access doors or elevators to open or lock when a fire occurs.

## Register fire alarm {#addFireAlarm}

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click <Cmd sid='setting.menu.advancedAc.root' /> in the left sidebar.

3. Click <Cmd sid='button.func.addZone' product='2' />.

4. Select <Cmd sid='zone.fireAlarm' product='2' /> and click <Cmd sid='button.apply' product='2' />.

5. Enter the name of the fire alarm to add in the <Cmd sid='common.info' product='2' /> section.

   <Image src='/img/biostarx-settings-adv-ac-add-fire-alarm.png' />

6. Set the details of the fire alarm in the <Cmd sid='common.setting' product='2' /> section. For more information, refer to [the following](#setting).

7. Set the actions to be performed when a fire alarm occurs in the <Cmd sid='common.alarm' product='2' /> section.

8. After completing all settings, click <Cmd sid='button.apply' product='2' /> at the bottom of the screen.

:::info

The <Cmd sid='common.alarm' product='2' /> section must set <Cmd sid='zone.pause.doorselected' product='2' /> to be usable.

:::

## Setting options guide {#setting}

The details for setting the fire alarm are as follows.

* <Cmd sid='zone.globalZone' product='2' />: Set the application range.

  * <Cmd sid='zone.globalZone.local' product='2' />: Can only be set to devices connected via RS-485 with entering devices.

  * <Cmd sid='zone.globalZone.global' product='2' />: Can be set to all devices registered in **BioStar X**.

* <Cmd sid='zone.pause.activatedAndDeactivated' product='2' />: You can activate or temporarily deactivate the settings.

* <Cmd sid='zone.pause.doorselected' product='2' />: Select the doors to designate for the fire alarm rule.

* <Cmd sid='zone.pause.liftselected' product='2' />: Select the elevator to designate the fire alarm rules.

* Click <Cmd sid='button.addplus' product='2' /> and set the devices to trigger fire alarm signals with <Cmd sid='zone.deviceInput' product='2' />.

:::info

* Setting the <Cmd sid='zone.globalZone' product='2' /> option to <Cmd sid='zone.globalZone.local' product='2' /> allows you to set the fire alarm function on only one access door or elevator.

* Setting the <Cmd sid='zone.globalZone' product='2' /> option to <Cmd sid='zone.globalZone.global' product='2' /> allows you to set the fire alarm function on both access doors and elevators simultaneously.
