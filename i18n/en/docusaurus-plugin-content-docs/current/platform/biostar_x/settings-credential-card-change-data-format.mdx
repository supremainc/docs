You can change the Wiegand card data format in use all at once.

:::caution

Changing the data format of cards already assigned to users will also occur, so be cautious.

:::

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click <Cmd sid='setting.menu.credential.root' /> â†’ <Cmd sid='setting.menu.credential.card' /> in the left sidebar.

3. Click <IcMoreW /> at the top right of the card list and select <Cmd sid='setting.card.more.changeDataFormat' product='2' />.

4. When the <Cmd sid='setting.card.changeDataFormat.title' product='2' /> window appears, select the card data format you want to change from the <Cmd sid='setting.card.changeDataFormat.currentType' product='2' /> list.

   <Image src='/img/biostarx-settings-credentail-card-change-data-format.png' />

5. Select the desired card data format from the <Cmd sid='setting.card.changeDataFormat.targetType' product='2' /> list.

6. Click <Cmd sid='button.apply' product='2' /> to change the card data format.
