You can change language/time zone, date/time format, and upload a sound file to be used as an alert.

1. Click on <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

1. Click <Cmd sid='common.preference' product='2' /> in the left sidebar of the screen.

1. Edit the necessary fields.

    <Image src='/img/biostarx-settings-preferences.png' />

    <Calloutlists>

    1. <Cmd sid='preference.languageTimezone' product='2' />: You can configure the BioStar X language and time zone settings.

        * <Cmd sid='common.language' product='2' />: Select a language to use.

        * <Cmd sid='common.timeZone' product='2' />: Select a time zone to use.

        * <Cmd sid='common.dst' product='2' />: Select the daylight saving time to apply to BioStar X server. If no registered daylight saving time, see [this document](#).

    1. <Cmd sid="preference.dateTimeFormat" product='2' />: You can configure the date and time format to use in BioStar X.

        * <Cmd sid="common.dateFormat" product='2' />: Changes the date format.

        * <Cmd sid="common.timeFormat" product='2' />: Changes the time format.

    1. <Cmd sid="device.info.sound" product='2' />: You can upload a sound file to use in BioStar X.

    </Calloutlists>

        1. Click <Cmd sid="button.plusAdd" product='2' />.

            <Image src='/img/biostarx-settings-preferences-sound.png' className='none' />

        1. Click <Cmd sid="setting.preference.sound.alert.addSound.browse" product='2' /> to select a file.

        1. Select .wav file or .mp3 file and then click **Open**.

        1. Click <Cmd sid="button.add" product='2' /> to upload.

        :::info

        * Sound files must be .wav or .mp3 format.

        * A maximum file size is 10MB.

        :::

1. Click <Cmd sid="button.apply" product='2' /> to save the settings.
