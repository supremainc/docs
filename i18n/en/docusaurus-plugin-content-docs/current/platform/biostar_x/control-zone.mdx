This guides how to control doors or devices set with advanced access control. The features provided enable quick responses and efficient management in case of issues.

1. Click <Cmd sid='shell.header.arena' /> on the <Cmd sid='shell.header.launcher' /> page or select <Cmd sid='shell.header.arena' /> from the shortcut list at the top left of the screen.

2. Move to <Cmd sid='arena.tree.door.title' /> tab under the left sidebar on the screen and select <Cmd sid='arena.tree.door.allZones' />. 

3. Select the configured advanced access control and right-click. You can select the desired feature from the popup menu.

<Image src='/img/biostarx-monitoring-door-zone-popup.png' />

:::info

The features offered vary depending on the type of advanced access control. The types of advanced access control are as follows.

* **Anti-Passback**

* Fire Alarm

* Scheduled Lock/Unlock

* Intrusion Alarm

* Interlock

* Muster

* Occupancy Limit Zone

* Roll Call

For more information on each advanced access control, refer to [the following](settings-advanced-ac).

:::

## Clear anti pass back {#apb-release}

You can clear the APB alarm for the door or device set to **Anti-Passback**. Click <Cmd sid='arena.tree.contextMenu.clearApb' /> in the popup menu. A completion message appears at the top of the screen.

:::info

* The <Cmd sid='arena.tree.contextMenu.clearApb' /> feature is only available for **Anti-Passback**.

* For more information about **Anti-Passback** settings, refer to [the following](settings-advanced-ac-apb).

* For more information about **Anti-Passback**, refer to [the following](../../reference/glossary#apb).

:::

## Clear alarm {#alarm}

You can clear the alarm that occurred at the door or device. Click <Cmd sid='arena.tree.contextMenu.clearAlarm' /> in the popup menu. A completion message appears at the top of the screen.

:::info

<Cmd sid='arena.tree.contextMenu.clearAlarm' /> cannot be used in the occupancy limit feature.

:::

## Activate/Deactivate {#zone-revitalizationdisabled}

You can activate advanced access control or disable unused features. Deactivated advanced access control displays the <IcDisable width='20px' height='20px' /> icon. Click <Cmd sid='arena.tree.contextMenu.enable' /> or <Cmd sid='arena.tree.contextMenu.disable' /> in the popup menu.

## Security/Disarm {#security-zone-control}

You can set or clear the security for the door or device. Click <Cmd sid='arena.tree.contextMenu.arm' /> or <Cmd sid='arena.tree.contextMenu.disarm' /> in the popup menu.

:::info

* <Cmd sid='arena.tree.contextMenu.arm' /> or <Cmd sid='arena.tree.contextMenu.disarm' /> can only be used on advanced access control set for security.

* For more information on security settings, refer to [the following](settings-advanced-ac-intrusion-alarm).

:::

## View muster report {#confirmation-of-muster-reports}

To check the report on musters, click <Cmd sid='arena.tree.contextMenu.musterReport' />. In the new window, you can check user entry and exit times and whether the stay time has exceeded through the user list and devices set with the muster feature.

:::info

* The <Cmd sid='arena.tree.contextMenu.musterReport' /> feature is only available in musters.

* For more information about the muster feature, refer to [the following](settings-advanced-ac-muster).

:::

## Monitoring occupancy limit {#monitoring-of-restrictions-on-the-number-of-people}

You can check the status of devices set with the occupancy limit feature. Click <Cmd sid='arena.tree.contextMenu.occupancyMonitoring' /> in the popup menu. In the new window, go to <Cmd sid='shell.header.setting' /> â†’ <Cmd sid='setting.menu.advancedAc.root' /> menu. You can check the status in the <Cmd sid='zone.occupancy.title' product='2' /> list.

:::info

* The <Cmd sid='arena.tree.contextMenu.occupancyMonitoring' /> feature is only available for items set with the occupancy limit feature.

* For more information on the occupancy limit settings, refer to [the following](settings-advanced-ac-occupancy).

:::

## View details {#detail-view}

You can check detailed information and event occurrence history of advanced access control. Click <Cmd sid='arena.tree.contextMenu.viewDetail' /> in the popup menu. Detailed information and event occurrence history are displayed in the right panel of the screen.

* **Detailed Information**: You can check the name and status of the advanced access control feature, device information, etc.

* <Cmd sid='arena.recentEvent' />: You can check the list of recent events occurred in the selected advanced access control.

:::info

For more information on each item in the advanced access control details section, refer to [the following](settings-advanced-ac).

:::
