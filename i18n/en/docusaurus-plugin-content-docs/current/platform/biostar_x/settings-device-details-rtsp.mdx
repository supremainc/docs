In the <Cmd sid='device.info.rtsp' product='2' /> section, you can configure the details for **Real Time Streaming Protocol** (RTSP) streaming. The device's camera can stream live video.

<Image src='/img/biostarx-setting-device-edit-rtsp.png' />

* <Cmd sid='device.info.rtsp' product='2' />: Enable RTSP streaming by setting <Cmd sid='device.info.enabled' product='2' />.

* <Cmd sid='device.info.rtsp.address' product='2' />: The RTSP server address is fixed. Click the <Cmd sid='device.info.rtsp.copy' product='2' /> button to copy the RTSP address.

* <Cmd sid='device.info.rtsp.port' product='2' />: Set the RTSP port.

* <Cmd sid='device.info.rtsp.id' product='2' />: Enter the RTSP server ID.

* <Cmd sid='device.info.rtsp.password' product='2' />: Enter the RTSP server password.

* <Cmd sid='device.info.rtsp.resolution' product='2' />: Select the resolution of the video output when using RTSP. The default is <Cmd>180 x 320</Cmd>.

:::info

* This feature supports BioStation 3 and BioEntry W3 models.

* The <Cmd sid='device.info.rtsp.resolution' product='2' /> option is supported on BioStation 3 firmware version 1.3.0 or higher.

* If the user has made any arbitrary changes, click the <Cmd sid='button.apply' product='2' /> button. Changes will not be applied to the device if not saved.

:::
