Set the types of alert and messages to appear on the screen when events occur in the device, door, or area. Additionally, you can configure it to play a sound file uploaded by the user when the alert occurs.

## Alert Settings

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click <Cmd>Event</Cmd> â†’ <Cmd>Alert</Cmd> in the left sidebar.

3. Select the event to display on the screen from the event list in each category.

4. Once all settings are completed, click the <Cmd sid='button.apply' product='2' /> button.

<Image src='/img/biostarx-settings-alert.png' />

## Edit Alert Message

Click <IcEdit2 /> in the event list to edit the alert message. When the <Cmd sid='setting.alert.alertMessage' product='2' /> window appears, set each item and click <Cmd sid='button.apply' product='2' />.

<Image src='/img/biostarx-settings-alert-edit-message.png' />

* <Cmd sid='common.name' product='2' />: Change the alert message name.

* <Cmd sid='common.message' product='2' />: Enter the message to display on the screen.

* <Cmd sid='setting.alert.config.soundFile' product='2' />: If you have uploaded a sound file to play for the event, you can select it from the list.

* <Cmd sid='setting.alert.config.playOption' product='2' />: If you have selected a sound file, choose the number of times to play it.

:::info

If there are no sound files to play, refer to the [following](settings-preferences) to upload one.

:::
