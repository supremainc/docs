On the <Cmd sid='shell.header.setting' /> page, manage various settings of **BioStar X**. The UI components of this page are as follows:

<Image src='/img/biostarx-ui-settings.png' className='none' />

<Calloutlists className='svg-none-style'>

1. Check various settings menus provided in the <Cmd sid='shell.header.setting' /> menu. For more information, refer to [the following](how-to-guide-settings).

2. A tool to navigate the page list. Move between pages or go to the desired page.

   <Image src='/img/biostarx-settings-access-control-status-pagenation.png' />

   * <IcFirst2 />: Move to the first page.

   * <IcFw2 />: Move to the previous page.

   * <IcBw2 />: Move to the next page.

   * <IcLast2 />: Move to the last page.

   * Enter the page number in the input field to move to the desired page.

   * Click the row selection box to set the number of items displayed on each page.

   :::info

   Support may vary depending on the settings menu.

   :::

3. Click the <IcMoreW /> button to access additional options.

   :::info

   Supported features may vary depending on the settings menu.

   :::

</Calloutlists>
