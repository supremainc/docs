On the <Cmd sid='shell.header.setting' /> page, manage various settings of **BioStar X**. The UI components of this page are as follows:

<Image src='/img/biostarx-ui-settings.png' />

<Calloutlists>

1. Enter keywords in the search input field to get settings items.

2. Check various settings menus provided in the <Cmd sid='shell.header.setting' /> menu. For more information, refer to the [following](how-to-guide-settings).

3. A tool to navigate the page list. Move between pages or go to the desired page.

    :::info

    Support may vary depending on the settings menu.

    :::

4. Click the <IcMoreW /> button to access additional options.

    :::info

    Supported features may vary depending on the settings menu.

    :::

</Calloutlists>
