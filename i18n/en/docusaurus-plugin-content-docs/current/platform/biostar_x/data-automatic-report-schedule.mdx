Set a schedule to automatically generate <Cmd sid='reportApp.customReport' product='2' /> created by setting the DYNAMIC period.

1. Click <Cmd sid='reportApp.mainMenu.schedule' product='2' /> â†’ <Cmd>Add Schedule</Cmd>.

1. Set each field as needed.

    <Image src='/img/biostarx-data-automatic-report-schedule.png' />

    * <Cmd sid='reportApp.information' product='2' / >: Set basic information of Automatic Report Schedule.

        * <Cmd sid='reportApp.scheduleName' product='2' / >: Enter the schedule name.

    * <Cmd sid='reportApp.reportAndSchedule' product='2' />: Set the schedule to be automatically generated for each report.

        * <Cmd sid='reportApp.report' product='2' / >: Select a custom report to automatically generate. Only custom reports set to DYNAMIC period will appear.

        * <Cmd sid='reportApp.frequency' product='2' / >: Set the frequency to automatically generate reports.

        * <Cmd sid='reportApp.generateTime' product='2' / >: Set the time to automatically generate reports.

    * <Cmd sid='reportApp.reportFormat' product='2' />: Set the format for each report.

        * <Cmd sid='reportApp.outputType' product='2' />: Set the automatically generate method of reports.

        * <Cmd sid='reportApp.reportTitle' product='2' />: Select <Cmd sid='reportApp.showTitle' product='2' /> to display the report name as the title on every page.

        * <Cmd sid='reportApp.header' product='2' />: If <Cmd sid='reportApp.showHeader' product='2' /> is selected, the header is displayed when the report is created. Select <Cmd sid='reportApp.onEveryPage' product='2' /> to display the header on every page.

            :::info

            The header may vary depending on the reports.

            :::

        * <Cmd sid='reportApp.footer' product='2' />: Set whether to display page numbers.

        * <Cmd sid='reportApp.fileFormat' product='2' />: Set the file format for exporting reports.

1. Click <Cmd sid="reportApp.apply" product='2' /> to save the settings.
