This guides you through the various features that can be set on the <Cmd sid='shell.header.setting' /> page of **BioStar X**. You can add devices and explore various options for customizing the user environment, such as permissions, language, date and time, and access card management. 

Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page or select <Cmd sid='shell.header.setting' /> from the shortcut list at the top left of the screen.

:::info

The features that can be configured may differ based on the user's permissions. Some features can only be used by users with administrator permission. For more information about user permission-based accessible menus, refer to the [following](set-permission#operatorLevel).

:::
