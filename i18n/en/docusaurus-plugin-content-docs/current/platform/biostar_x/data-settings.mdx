Set the save path for reports with <Cmd sid='reportApp.mainMenu.schedule' product='2' /> configured.

1. Click <Cmd sid='reportApp.mainMenu.settings' product='2' />.

2. Enter the path where you want to save the report.

   <Image src='/img/biostarx-data-settings.png' className='none' />

   * ```
     <Cmd sid='reportApp.exportReportPath' product='2' />: The report will be saved to the specified path. If you do not enter a path, it will be automatically saved in the "Documents\BioStarX" folder on your PC.
     ```

3. ```
   Click <Cmd sid="reportApp.apply" product='2' /> to save the settings.
   ```
