Set the path where the report will be saved if you have set <Cmd sid='reportApp.mainMenu.schedule' product='2' />.

1. Click <Cmd sid='reportApp.mainMenu.settings' product='2' />.

2. Enter the path where you want to save the report.

   <Image src='/img/biostarx-data-settings.png' className='none' />

   * <Cmd sid='reportApp.exportReportPath' product='2' />: The report will be saved to the specified path, and if no path is entered, it will be automatically saved in the *Documents\BioStarX* folder on the user's PC.

3. Click <Cmd sid="reportApp.apply" product='2' /> to save the settings.
