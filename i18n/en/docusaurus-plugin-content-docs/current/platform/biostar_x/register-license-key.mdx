If you purchased the BioStar X license, you can register the license key and use more features.

Navigate to <Cmd sid='shell.header.setting' /> → <Cmd sid='setting.menu.license.root' /> → <Cmd sid='setting.menu.license.biostarXLicense' />.

The method to activate <Cmd sid='setting.menu.license.biostarXLicense' /> varies depending on your network environment. Check your network status and activate your license according to the provided instructions.

:::info

* Certain features such as adding users, doors, and ports are restricted when the license is not registered.

* For more information the <Cmd sid='setting.menu.license.root' /> menu, see <DocLink docId='platform/biostar_x/settings-license' />.

* For more information on the licensing policy, see <DocLink docId='platform/biostar_x/licensing' />.

* For more information about license error codes, refer to [the following link](https://reprisesoftware.com/docs/isv/appendix/appendix-b-rlm-status-values.html).

:::
