Check the status of each elevator. This helps administrators quickly grasp the operating status of each elevator through the indicated icons and detailed descriptions, and respond swiftly in case of issues.

Click <Cmd sid='shell.header.arena' /> on the <Cmd sid='shell.header.launcher' /> page or select <Cmd sid='shell.header.arena' /> from the shortcut list at the top left of the screen. In the <Cmd sid='arena.tree.door.title' /> tab on the left sidebar, go to <Cmd sid='arena.tree.door.allElevatorGroups' />.

:::info

For more information about setting up the elevator, refer to [this document](#).

:::

## Checking Elevator Alarms {#check-the-elevator-alarm}

When an event occurs in each elevator, the <IcErr width='20px' height='20px' /> icon will be displayed. Click the alarm icon to check the event details.
