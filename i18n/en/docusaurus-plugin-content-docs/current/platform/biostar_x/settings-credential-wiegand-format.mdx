Set the card data reading format. Card data is processed according to the configured Wiegand format.

## Set Wiegand card format

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click <Cmd sid='setting.menu.credential.root' /> â†’ <Cmd sid='setting.menu.credential.cardFormat' /> on the left sidebar.

3. Click <Cmd sid='common.wiegand' product='2' /> in the left card format list.

   <Image src='/img/biostarx-settings-credentail-card-wiegand-list.png' />

4. Click the <IcEdit /> button for the unnamed item in the Wiegand format list.

5. When the <Cmd sid='setting.cardFormat.addNewWiegand' product='2' /> screen appears, set each item. For more information, refer to [the following](#setting).

6. Click <Cmd sid='button.apply' product='2' /> to add the configured Wiegand format.

:::info

* Up to 15 Wiegand formats can be used.

* Predefined formats cannot be modified or deleted.

:::

## Setting options guide {#setting}

Reference the items below to set the bit configuration when adding or editing a Wiegand card format.

<Image src='/img/biostarx-settings-credential-wiegand-format.png' />

* <Cmd sid='common.name' product='2' />: Enter the Wiegand format name.

* <Cmd sid='common.description' product='2' />: Enter a brief description of the Wiegand format.

* <Cmd sid='setting.cardFormat.totalBits' product='2' />: Enter the total number of bits.

* <Cmd sid='setting.cardFormat.fcField' product='2' />: Set whether to use the identification code. To use the identification code, click the checkbox and enter <Cmd sid='setting.cardFormat.startBit' product='2' /> and <Cmd sid='setting.cardFormat.endBit' product='2' />.

* <Cmd sid='setting.cardFormat.idField' product='2' />: Enter the <Cmd sid='setting.cardFormat.startBit' product='2' /> and <Cmd sid='setting.cardFormat.endBit' product='2' /> for the ID you want to use. Click <Cmd sid='button.plusAdd' product='2' /> to add an ID field.

* <Cmd sid='setting.cardFormat.parityBits' product='2' />: Click <Cmd sid='button.plusAdd' product='2' /> to enter the number of parity bits. Enter the position of the parity bits and the start and end bits.

:::info

To add parity bits, the <Cmd sid='setting.cardFormat.totalBits' product='2' /> option must be entered.

:::
