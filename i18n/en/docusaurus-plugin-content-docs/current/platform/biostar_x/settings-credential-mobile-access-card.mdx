Integrate the [Airfob Portal](https://www.airfob.com/) to issue or delete mobile access cards for users in **BioStar X**.

Users can receive mobile access cards via a link sent to their email without signing up for the Airfob Portal or separately registering for mobile access cards.

:::info

* Only one of the **CSN Mobile** card or **Template on Mobile** can be used for mobile access cards.

* Refer to the following for devices and firmware versions that support the **CSN Mobile** card.

  <details>
  <summary>View devices and firmware versions supporting the **CSN Mobile** card</summary>

  * XPass 2 firmware version 1.1.0 or higher

  * XPass D2 (Rev 2) firmware version 1.4.0 or higher

  * BioLite N2 firmware version 1.3.0 or higher

  * BioEntry W2 (Rev 2) firmware version 1.6.0 or higher

  * FaceStation 2 firmware version 1.4.0 or higher

  * FaceStation F2 firmware 1.0.0 or higher

  * BioStation 2 firmware version 1.9.0 or higher (NFC supported model)

  * BioStation A2 firmware version 1.8.0 or higher (NFC supported model)

  * FaceLite firmware version 1.2.0 or higher

  * X-Station 2 firmware 1.0.0 or higher

  * BioStation 3 firmware 1.0.0 or higher

  * BioEntry W3 firmware version 1.0.0 or higher

  </details>

* Refer to the following for devices and firmware versions that support **Template on Mobile**.

  * BioStation 3 firmware version 1.2.0 or higher

  * BioEntry W3 firmware version 1.0.0 or higher

:::

Set whether to use mobile access cards and manage settings related to Airfob Portal integration. You can also register devices that will use mobile access cards. After signing up for the Airfob Portal, you can manage mobile access cards in **BioStar X**.

<Steps>
<Step>

## Sign up for Airfob Portal and create Site {#airfob-portal-setup}

Set up mobile access cards and registered devices in the Airfob Portal, and manage your site and credits.

1. Access the [Airfob Portal](https://portal.airfob.com/).

2. Click **Get Started** to proceed with membership registration and site creation.

3. Enter the Airfob Portal administrator's email address in the email input field, then click **Get Started**. A verification code will be sent to the entered email address.

4. Enter the verification code received via email into the verification code input field, then click **Confirm**.

   :::note

   The verification code is a 6-digit number.

   :::

5. Review the privacy policy and terms, then click **Agree**.

6. Set a password and nickname for the Airfob Portal, then click **Create Account**. Account creation will be completed.

7. Once account creation is completed, click **Sign In**.

8. Enter your email and password, then click **Sign In**.

9. Click **Create Site** to proceed with site creation.

   :::note

   **Site** refers to the organization or company using mobile access cards.

   :::

10. Set the site name and country, then click **Next**.

11. Select the site type.

    <SiteType />

12. Click **Create**. Site creation will be completed.

13. Click the site name to access the Airfob Portal for that site.

:::info

For more information about using the Airfob Portal, refer to the [following link](https://airfob.zohodesk.com/portal/home).

:::

</Step>
<Step>

## Configure mobile access card in BioStar X {#biostar-x-mobile-access-setup}

After signing up for the Airfob Portal, you can manage mobile access cards in **BioStar X**. To use Suprema mobile access cards, sign up for the Airfob Portal and create a site.

1. Click <Cmd sid='shell.header.setting' /> on the <Cmd sid='shell.header.launcher' /> page.

2. In the left sidebar, click <Cmd sid='setting.menu.credential.root' /> → <Cmd sid='setting.menu.credential.mobileAccess' />.

3. Edit the necessary fields.

4. Once all settings are completed, click the <Cmd sid='button.close' product='2' /> button.

<Image src='/img/biostarx-settings-credentail-mobile-access.png' />

### Setting options guide {#setting}

* <Cmd sid='setting.mobileCredential.setting' product='2' />: You can set whether to use mobile access cards. Set to <Cmd sid='common.use' product='2' /> to issue mobile access cards to users.

* <Cmd sid='setting.mobileCredential.siteType' product='2' />: Check the type of site.

  <SiteType />

* <Cmd sid='setting.mobileCredential.mobileCredentialServerAddress' product='2' />: Check the domain address of the Airfob Portal.

* <Cmd sid='setting.mobileCredential.mobileCredentialPort' product='2' />: Check the port number of the Airfob Portal.

* <Cmd sid='setting.mobileCredential.siteId' product='2' />: Enter the site ID created in the Airfob Portal. The site ID can be found in the Airfob Portal pathway **Settings** → **Site**.

* <Cmd sid='setting.mobileCredential.email' product='2' />: Enter the email address of the mobile access card administrator account.

* <Cmd sid='setting.mobileCredential.password' product='2' />: Enter the password of the mobile access card administrator account.

:::info

To use mobile access cards in **BioStar X**, complete the sign-up and initial setup of the Airfob Portal first. For more information, refer to the [following](#airfob-portal-setup).

:::

</Step>
<Step>

## Register device {#registerDevice}

You can register devices that will use mobile access cards directly from the Airfob Pass application or **BioStar X**.

<Image src='/img/biostarx-settings-credentail-mobile-access-connection.png' />

Once the settings for integration with the Airfob Portal are completed, click <Cmd sid='setting.mobileCredential.connect' product='2' />. The <Cmd sid='setting.mobileCredential.registerDeviceTable' product='2' /> option will be available.

1. To add a device that will use mobile access cards, click <Cmd sid='button.addplus' product='2' />.

   <Image src='/img/biostarx-settings-credentail-mobile-access-register-device.png' />

2. In the device list of the <Cmd sid='setting.mobileCredential.registerDeviceTable' product='2' /> window, click the checkbox for the device that will use the mobile access card.

   <Image src='/img/biostarx-settings-credentail-mobile-access-register-device-popup.png' />

3. After selecting all devices, click <Cmd sid='button.add' product='2' />.

You can verify the added devices in the device list. 

* To resend the mobile access card certificate, click <IcRefresh />.

* To delete a registered device, click <IcTrash2 />.

:::info

* Refer to the following for devices and firmware versions that can use mobile access cards.

  <details>
  <summary>View devices and firmware versions that can use mobile access cards</summary>

  * XPass 2 firmware version 1.1.0 or higher

  * XPass D2 (Rev 2) firmware version 1.4.0 or higher

  * BioLite N2 firmware version 1.3.0 or higher

  * BioEntry W2 (Rev 2) firmware version 1.6.0 or higher

  * FaceStation 2 firmware version 1.4.0 or higher

  * FaceStation F2 firmware 1.0.0 or higher

  * BioStation 2 firmware version 1.9.0 or higher (NFC supported model)

  * BioStation A2 firmware version 1.8.0 or higher (NFC supported model)

  * FaceLite firmware version 1.2.0 or higher

  * X-Station 2 firmware 1.0.0 or higher

  * BioStation 3 firmware 1.0.0 or higher

  * BioEntry W3 firmware version 1.0.0 or higher

  </details>

* You can also register devices using the Airfob Pass application.

* Deleting a registered device will remove the mobile access card certificate sent to that device.

:::

</Step>

## Issue mobile access card {#issueMobileAccessCard}

You can issue mobile access cards to users registered in **BioStar X**. To issue a mobile access card to a user, essential user information based on the method of sending the mobile access card must be entered. For more information, refer to the following documents.

* [User Information Registration](add-user)

* [Mobile Access Card Registration](enroll-credential#enroll-mobile)
