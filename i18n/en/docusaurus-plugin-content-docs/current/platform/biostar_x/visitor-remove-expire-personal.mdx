You can delete visitor information that has exceeded the personal data retention period.

1. Click <Cmd sid='shell.header.visitor' /> on the <Cmd sid='shell.header.launcher' /> page.

2. Click the <Cmd sid='visitor.expired.personalData' product='2' /> tab at the top of the screen.

3. When the list of visitors whose personal data retention period has expired is displayed, click the checkbox for the visitor you want to delete. You can select more than one.

   <Image src='/img/biostarx-visitor-expire-personal-data-lists.png' />

4. Click the <Cmd sid='visitor.button.deletePersonalData' product='2' /> button in the top right of the visitor list.

5. When the confirmation message appears, click <Cmd sid='button.yes' product='2' />.

:::info

* Only users with <Cmd sid='setting.account.administrator' product='2' className='bold' /> permissions can view the list of expired personal data visitors. For more information on user permissions, refer to [the following](add-user).

* For more information on setting the personal data retention period, refer to [the following](settings-visitor#settings-visitor-terms).

:::
