This guide provides instructions on how to manage the status of the **BioStar X** server. You can stop or restart the running services.

## Manage BioStar X service {#biostar-x-setting}

Provides a tool to manage the status of the **BioStar X** server. Click <Start/> → **BioStar X** → **BioStar X Service Manager**. The <Cmd>BioStar X Service Manager</Cmd> window appears.

<Image src='/img/common/biostarx-tutoria-service-manager-main.png' alone />

:::info

If you do not restart, **BioStar X** may not operate properly. If you have changed the time settings of the **BioStar X** server, stop and restart the **BioStar X Core Service**.

:::

### End service {#end-service}

To shut down **BioStar X**, click the <Cmd>Stop</Cmd> button for individual services in the <Cmd>SEVICES</Cmd> menu.

<Image src='/img/common/biostarx-tutoria-service-manager-main.png' alone />

{/* 모든 서비스를 종료하려면 화면 오른쪽 상단의 <Cmd>Stop All</Cmd> 버튼을 클릭하세요. */}

### Start the server {#start-the-server}

To restart **BioStar X**, click <Cmd>Start</Cmd> for the service in the <Cmd>SEVICES</Cmd> menu with a <Cmd>Stopped</Cmd> status in the <Cmd>Status</Cmd> column.

<Image src='/img/common/biostarx-tutoria-service-manager-main-restart.png' alone />

{/* 모든 서비스를 시작하려면 화면 오른쪽 상단의 <Cmd>Start All</Cmd> 버튼을 클릭하세요. */}
