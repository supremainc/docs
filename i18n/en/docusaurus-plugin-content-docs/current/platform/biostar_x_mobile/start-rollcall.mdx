Roll Call quickly generates a personnel list based on entry records within a specific time and place. When an emergency occurs, the system can determine whether a user has arrived at the roll call location by authenticating through the entry or exit device at a predesignated location.

Can be used for emergency response, safety checks, and providing rescue team information. This allows for efficient management of personnel status utilizing entry and exit records.

:::info

* Before starting Roll Call, complete the Roll Call settings in BioStar X. For more information on how to set up Roll Call, refer to <DocLink docId='platform/biostar_x/settings-advanced-ac-roll-call' />.

* Roll call settings can be used through additional options available with an **Advanced** license or above. For more information on licensing policies, see <DocLink docId='platform/biostar_x/licensing#features-add-ons' />.

:::
