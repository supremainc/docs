In the BioStar X Mobile App settings menu, manage the app's main features, including login, time zone, and security. To access the app settings menu, tap <IcSet title='Settings' /> in the upper right corner of the Home screen.

<Image src='/img/mobile/bsx-mobile-app-setting.png' width='252px' alone />

## Check app version information

In <Cmd>BioStar X Mobile App</Cmd>, check the version information for the currently installed BioStar X Mobile App. Use the app version information when contacting support for troubleshooting.

## Login settings

### Automatic login settings

Users can enable the automatic login feature instead of entering login information each time they open the app or log in again after the session expires. 

1. To enable the automatic login feature, turn on the <Cmd>Auto Login</Cmd> option.

   <Image src='/img/mobile/bsx-mobile-app-setting-auto-login.png' width='252px' alone />

2. To save changes, tap <Cmd>OK</Cmd> in the upper right corner of the screen.

3. When the confirmation message appears, tap <Cmd>Yes</Cmd>.

:::info

* Use the automatic login feature only when <Cmd>Remember username and password</Cmd> is enabled on the login screen. If this option is not enabled, the automatic login feature is unavailable.

  <Image src='/img/mobile/bsx-mobile-login-check.png' width='252px' alone />

* If the automatic login feature is enabled, logout is unavailable. To log out, disable the automatic login feature first, and then log out.

:::

### Biometric login settings

On mobile devices with biometric authentication, enable the biometric login feature. When this feature is enabled, administrators can log in with the fingerprint enrolled on the mobile device instead of a user ID and password.

1. To enable the biometric login feature, tap <Cmd>Biometrics Information</Cmd> → <Cmd>Biometrics Login</Cmd>.

   <Image src='/img/mobile/bsx-mobile-app-setting-biometrics.png' width='252px' alone />

2. To save changes, tap <Cmd>OK</Cmd> in the upper right corner of the screen.

3. When the confirmation message appears, tap <Cmd>Yes</Cmd>.

Use biometric authentication on the login screen.

<Image src='/img/mobile/bsx-mobile-app-setting-biometrics-login.png' width='252px' alone />

:::info

This feature requires biometric authentication on the mobile device and the <Cmd>Remember username and password</Cmd> option enabled on the login screen.

:::

## Standard time zone settings

Set the time zone for the location where the BioStar X server is installed. A correct time zone setting is important for accurate timestamps in event logs and notifications.

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-setting-timezone.png' alone />

</Column>
<Column className='padding--l40'>

1. Tap <Cmd>Time Zone</Cmd>.

2. Select the time zone for the location where the BioStar X server is installed from the time zone list.

3. Tap <Cmd>Yes</Cmd>.

4. To save changes, tap <Cmd>OK</Cmd> in the upper right corner of the screen.

5. When the confirmation message appears, tap <Cmd>Yes</Cmd>.

:::info

The changed setting also applies to the BioStar X server, and the timestamps in event logs and notifications shown in the app reflect the updated time zone.

:::

</Column>
</Columns>

## Date and time display format settings

You can set the date and time display format to be used across the system.

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-setting-date-format.png' className='none' alone />

</Column>
<Column className='padding--l40'>

### Date display format

1. Tap <Cmd>Date Format</Cmd>.

2. Select the desired date display format from the <Cmd>Date Format</Cmd> list.

   * <Cmd>MM/dd/yyyy</Cmd>: 01/31/2025

   * <Cmd>yyyy/MM/dd</Cmd>: 2025/01/31

   * <Cmd>dd/MM/yyyy</Cmd>: 31/01/2025

3. Tap <Cmd>Yes</Cmd>.

4. To save changes, tap <Cmd>OK</Cmd> in the upper right corner of the screen.

5. When the confirmation message appears, tap <Cmd>Yes</Cmd>.

</Column>
</Columns>

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-setting-time-format.png' className='none' alone />

</Column>
<Column className='padding--l40'>

### Time display format

1. Tap <Cmd>Time Format</Cmd>.

2. Select the desired time display format from the <Cmd>Time Format</Cmd> list.

   * <Cmd>hh:mm</Cmd>: 23:59, 24-hour format

   * <Cmd>hh:mm a</Cmd>: 11:59 PM, 12-hour format

   * <Cmd>a hh:mm</Cmd>: PM 11:59, 12-hour format

3. Tap <Cmd>Yes</Cmd>.

4. To save changes, tap <Cmd>OK</Cmd> in the upper right corner of the screen.

5. When the confirmation message appears, tap <Cmd>Yes</Cmd>.

</Column>
</Columns>

## Confirm changes with biometrics

When making important system changes, you can require additional administrator authentication with biometrics. This helps prevent unauthorized changes by other users.

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-setting-biometrics-change-confirm.png' width='252px' alone />

</Column>
<Column className='padding--l40'>

1. Tap <Cmd>Biometrics Information</Cmd> → <Cmd>Changes Confirmation</Cmd>.

2. To save changes, tap <Cmd>OK</Cmd> in the upper right corner of the screen.

3. When the confirmation message appears, tap <Cmd>Yes</Cmd>.

The following changes require biometric authentication.

* Change administrator password

* Change app settings

* Enroll a new user

* Edit user information

* Delete users

* Change door status

* Change advanced access control status

* Delete alarm notifications

</Column>
</Columns>
