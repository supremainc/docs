Before using the BioStar X mobile app, check the following requirements and install the app.

## Supported environment

The BioStar X mobile app can be used under the following conditions.

* **Supported system**: BioStar X

* **Server requirements**: A BioStar X server with API access is required

* **Users**: Only users pre-enrolled in the BioStar X system with administrator privileges can use it

* **Mobile device**: iOS 13.0 or later, or Android 7.0 or later

:::info

The BioStar X mobile app cannot operate independently. Connect it to a BioStar X server running on the customer network.

* For more information on the minimum requirements for installing BioStar X, see <DocLink docId='platform/biostar_x/system-requirements' target='_blank' />.

* For more information on installing BioStar X, see <DocLink docId='platform/biostar_x/express-install' target='_blank' />.

:::

## License and network requirements

A BioStar X license is required to use the BioStar X mobile app. Purchase an add-on license option to use it. For more information on licensing policies, see <DocLink docId='platform/biostar_x/licensing#features-add-ons' target='_blank' />.

| Access environment | Required license |
| --- | --- |
| Same network | Mobile app license |
| External network (public IP) | Mobile app license |
| External network (no public IP) | Mobile app license + Remote Access license |

* A **mobile app** license is required to integrate the BioStar X server with the mobile app.

* By default, use the mobile app by connecting to the BioStar X server on the same network.

* Access from an external network requires a public IP address. Otherwise, purchase an additional **Remote Access** license.

:::info

* For more information on applying licenses in BioStar X, see <DocLink docId='platform/biostar_x/settings-license-biostar-x-license' target='_blank' />.

* For more information on BioStar X server settings, see <DocLink docId='platform/biostar_x/settings-server-server' target='_blank' />.

* For more information on the Remote Access feature in BioStar X, see <DocLink docId='platform/biostar_x/settings-remote-access' target='_blank' />.

:::
