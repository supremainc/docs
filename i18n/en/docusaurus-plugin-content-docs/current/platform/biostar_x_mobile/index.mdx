Use the BioStar X mobile app to manage the BioStar X access control system from a mobile device. Perform tasks such as user enrollment, remote door control, real-time event monitoring, and alarm handling.

![BioStar X Mobile Connection Architecture](/img/common/biostarx-mobile-connection-architecture.png)

## Key features

### User Management

* Add, delete, and edit user information

* Credential management: Card, Face, Facial Authentication, Fingerprint

### Door Control

* Lock, unlock, open, and release doors remotely

* Release anti-passback (APB) and reset door alarms

### Real-time monitoring

* Check real-time access event logs for ongoing management

### Manage alarms

* Handle door, device, and advanced access control alarms

* Manage alarms efficiently with separate lists for new and acknowledged events

### Manage access control on the go

Use the BioStar X mobile app to perform the following tasks on the go:

* Manage users and add credentials

* Real-time event monitoring

* Respond quickly to alarms

* Door notifications: open request, forced open, hold open

* Device notifications: tamper, reboot, RS485 disconnection

* Area alarms: anti-passback, fire, and more

## Privacy protection

Suprema does not collect, store, or process any personal data through the BioStar X mobile app. All user-related data, including facial images and credentials, are sent directly to the customer’s internal system and are not sent to Suprema servers.
