Edit user information when a user's affiliation changes or permissions need to be adjusted. Learn how to delete users when they leave the company or their contracts end to enhance security.

## Edit user

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-user-details.png' alone />

</Column>
<Column className='padding--l40'>

You can edit user information, permissions, and credentials from the User Details screen.

1. On the app home screen, tap <Cmd>USER</Cmd>.

   :::note

   You can also access it through the side menu. Tap <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>User</Cmd>.

   :::

2. Tap the user to edit in the user list.

3. In User Details, tap <IcAppEdit title='Edit' width='20px' height='20px' /> at the top right of the screen.

4. In <Cmd>Edit User</Cmd>, you can edit basic user information, change permissions, and manage credentials.

5. To save changes, tap <IcSave title='Save' width='20px' height='20px' /> at the top right of the screen.

:::info

* For more information on user information, permissions, and credentials, refer to <DocLink docId='platform/biostar_x_mobile/add-user' />.

* To temporarily deactivate a user, tap the toggle switch in <Cmd>Status</Cmd> and set it to <Cmd>Inactive</Cmd>. Inactive users are blocked from access.

* To delete a registered credential, tap the credential to delete, then tap <IcAppDelete title='Delete' /> at the top right of the screen. Select the credential to delete, then tap <Cmd>OK</Cmd>.

  <Image src='/img/mobile/bsx-mobile-app-add-user-credential-remove.png' width='70%' className='none' alone />

:::

</Column>
</Columns>

## Delete users

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-user-delete.png' alone />

</Column>
<Column className='padding--l40'>

If you need to delete a user due to resignation, contract termination, or a security violation, follow these instructions.

1. On the app home screen, tap <Cmd>USER</Cmd>.

   :::note

   You can also access it through the side menu. Tap <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>User</Cmd>.

   :::

2. Tap <IcAppDelete title='Delete' width='20px' height='20px' /> at the top right of the user list.

3. Select the user to delete. You can select more than one user.

4. Tap <Cmd>OK</Cmd> at the top right of the screen.

5. When the confirmation message appears, tap <Cmd>Yes</Cmd>.

The selected user is deleted. Deleted users cannot be recovered.

:::info

The administrator account with ID `1` cannot be deleted.

:::

</Column>
</Columns>
