In the mobile app, check alerts from BioStar X and resolve them as needed. When an alert event such as a forced door open occurs, check the alert in the app and record the action taken.

<Image src='/img/mobile/bsx-mobile-app-alert.png' width='500px' className='none' alone />

:::info

* For more information on checking and resolving alerts in BioStar X, see <DocLink docId='platform/biostar_x/monitoring-event#acknowledge-alarm-and-record-action' />.

* The date and time shown in the alert list follow the BioStar X server's time settings. For more information on date and time display format settings in BioStar X, refer to <DocLink docId='platform/biostar_x/settings-preferences' />.

:::

Tap <Cmd>ALERT</Cmd> on the app home screen. View the <Cmd>Alert List</Cmd>. 

In the alert list, view alert events such as forced door opens in reverse chronological order. When a new alert event occurs, it is added to the list in real time.

### View alert details

In the alert list, tap a specific alert to view its details. Alert details show the event type, user, occurrence time, and the door/device.

* To record the action taken after checking an alert, enter the details in the <Cmd>Comment (optional)</Cmd> field. The entered details are saved on the BioStar X server and appear in the alert event details.

* To enter the action taken and resolve the alert, tap <Cmd>ACKNOWLEDGE</Cmd>. The alert is resolved, and the event disappears from <Cmd>Alert List</Cmd>.
