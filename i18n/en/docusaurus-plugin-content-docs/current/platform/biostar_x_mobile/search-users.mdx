When the user list is long and hard to search, use search and filtering to find users quickly.

## User search

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-user-list-search.png' alone />

</Column>
<Column className='padding--l40'>

You can search by user ID, name, or email address.

1. On the app home screen, tap <Cmd>USER</Cmd>.

   :::note

   You can also access it through the side menu. Tap <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>User</Cmd>.

   :::

2. Tap <IcSearch title='Search' width='20px' height='20px' /> at the top of the user list.

3. Enter a search term in the search field. You can search by combining the user ID, name, and email address.

4. After you finish entering the search term, tap <IcSearch title='Search' width='20px' height='20px' />.

Users that match the search term appear in the search results. In the search results, tap the user you want to open the user details.

:::info

* Recent search history appears below the search input field. Tap a recent search entry to search again with that term.

* To close the search input field, tap <IcClose title='Close' width='20px' height='20px' />.

:::

</Column>
</Columns>

## Filter the user list

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-user-list-filter.png' alone />

</Column>
<Column className='padding--l40'>

Filter the user list by various conditions to find users quickly.

1. On the app home screen, tap <Cmd>USER</Cmd>.

   :::note

   You can also access it through the side menu. Tap <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>User</Cmd>.

   :::

2. Tap <IcAppFilter title='Filter' width='20px' height='20px' /> at the top of the user list.

3. Set the desired conditions in <Cmd>User Filter</Cmd>.

   * <Cmd>Sort By</Cmd>: Select ascending or descending order by user ID, name, or group.

   * <Cmd>User Group</Cmd>: Select a specific user group to show only users in that group.

   * <Cmd>Status</Cmd>: To show only active users, select <Cmd>Active</Cmd>. To show only inactive users, select <Cmd>Inactive</Cmd>.

4. To filter the user list with the selected conditions, tap <Cmd>Apply</Cmd>.

:::info

To reset the filter conditions, tap <IcAppLogFiltered title='Filtered' width='20px' height='20px' /> → <IcAppReset title='Reset' width='20px' height='20px' /> in the upper right corner of the screen.

:::

</Column>
</Columns>
