View a user's access history to check for abnormal access or access outside authorized hours. Filter log events to easily find history for a desired period and location.

## View user access history

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-user-log-event.png' alone />

</Column>
<Column className='padding--l40'>

1. On the app home screen, tap <Cmd>USER</Cmd>.

   :::note

   You can also access it through the side menu. Tap <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>User</Cmd>.

   :::

2. Tap the user whose log events you want to view in the user list.

3. Tap <IcAppLog title='Log' width='20px' height='20px' /> in the upper right in user details.

4. In <Cmd>Event Log</Cmd>, you can view the user's access history.

   <Image src='/img/mobile/bsx-mobile-app-user-log-filter.png' width='252px' alone />

The latest log events are displayed first, and you can review the access location and event details.

</Column>
</Columns>

## Filter log events

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-user-log-event-filter.png' alone />

</Column>
<Column className='padding--l40'>

You can filter a user's log events by various conditions, such as date and time, event type, and access location.

1. Tap <IcAppLogFiltered title='Filter' width='20px' height='20px' /> in the upper right of Event Log.

2. When the filter criteria appear, set the desired criteria.

   * <Cmd>Period</Cmd>: Set the period when the log event occurred.

   * <Cmd>Time</Cmd>: Set the time when the log event occurred.

   * <Cmd>Event</Cmd>: Select the log event type. For example, select access authentication success or access authentication failure.

   * <Cmd>User</Cmd>: View log events for users other than the current user.

   * <Cmd>Device</Cmd>: Select the device where the log event occurred.

   * <Cmd>Door</Cmd>: Select the door where the log event occurred.

3. To filter log events with the selected criteria, tap <Cmd>Apply</Cmd>.

:::info

To reset the filter conditions, tap <IcAppLogFiltered title='Filtered' width='20px' height='20px' /> → <IcAppReset title='Reset' width='20px' height='20px' /> in the upper right corner of the screen.

:::

</Column>
</Columns>
