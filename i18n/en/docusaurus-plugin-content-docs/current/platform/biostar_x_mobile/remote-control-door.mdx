In the door list, check each door's status in real time and control it as needed. During normal operation, manage doors with basic control features. Use additional control options for stronger security or emergencies.

Tap <Cmd>DOOR</Cmd> on the app home screen. The list of doors enrolled in BioStar X appears.

:::info

You can also access it through the side menu. Tap <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>Door</Cmd>.

:::

## Open/Lock door

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-door-control.png' alone />

</Column>
<Column className='padding--l40'>

The remote door control feature lets you open or lock door to allow or block access.

* <Cmd>Open</Cmd>: Temporarily opens the door to allow access. The door closes automatically after the set time.

* <Cmd>Release</Cmd>: Sets the door to allow access only for authenticated users. The door status changes according to the schedule set for the door.

:::info

For more information on how to temporarily open a door and set the auto-close time, see <Cmd sid='common.open' product='2' /> → <Cmd sid='door.openTime' product='2' /> in <DocLink docId='platform/biostar_x/settings-door-add#option' />.

:::

</Column>

## Additional control

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-door-control-more.png' alone />

</Column>
<Column className='padding--l40'>

When stronger security is needed or access must be controlled for a long period, manually lock or unlock the door without a time limit. You can also clear the triggered alarm.

Tap <IcAirMore title='More' /> at the far right of the door to use additional control options.

* <Cmd>Manual Lock</Cmd>: Manually locks the door to block access. This feature locks the door without a time limit. In this state, access is blocked for all users.

* <Cmd>Manual Unlock</Cmd>: Manually unlocks the door to allow access. This feature unlocks the door without a time limit. In this state, access is allowed for all users.

* <Cmd>Clear Alarm</Cmd>: Clear the alarm that occurred at the door.

* <Cmd>Clear APB</Cmd>: Clear APB status for all users at the selected door.

:::info

After you manually lock or unlock a door, the door status does not change automatically. For security, use <Cmd>Release</Cmd> to make the door operate according to the set schedule.

:::

</Column>
</Columns>
