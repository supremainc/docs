This article explains how to change BioStar Air account information, configure alert reception methods, manage SIP intercom, leave a site, and delete accounts.

## Access account settings {#accessAccountSettings}

1. Log in to the [BioStar Air portal](https://admin.biostarair.com).

2. Click the user profile icon at the top right of the screen.

3. Click <Cmd sid="settings_menu" product="air" /> in the drop-down menu.

   <Image src="/img/air/bsair-device-bs3-sip-account-settings.png" />

4. When the <Cmd sid="settings_menu" product="air" /> window appears, set your desired options.

   * <Cmd sid="account" product="air" />: Edit the user name, password, and phone number. For more information on details, refer to [the following](#changeAccountInfo).

   * <Cmd sid="alerts.alerts" product="air" />: Set how alerts are received. For more information on details, refer to [the following](#setAlertPreferences).

   * <Cmd sid="device_intercom_title" product="air" />: Manage SIP intercom settings. For more information on details, refer to [the following](sip-intercom-for-biostation3).

   * <Cmd sid="site_setting_tab_management" product="air" />: Leave the site or delete the account. For more information on details, refer to [the following](#manageAccount).

5. Click <Cmd sid="save_btn" product="air" /> to save the changes.

## Change account information {#changeAccountInfo}

In the <Cmd sid="account" product="air" /> tab, edit information for the currently logged-in user.

<Image src="/img/air/bsair-account-settings-account.png" />

* Edit the user name, password, and phone number. Click <IcAirEdit title="Edit" /> for each item.

* <Cmd sid="session_timeout_label" product="air" />: Automatically logs out after the selected time elapses. Select the desired time.

## Set alerts {#setAlertPreferences}

In the <Cmd sid="alerts.alerts" product="air" /> tab, set how alerts are received. Activate the <Cmd sid="all_service_alerts" product="air" /> option to receive all alerts. Disable unwanted alerts individually.

<Image src="/img/air/bsair-account-settings-alert-report.png" />

:::info

* If you agreed to receive app alerts but none appear, check the mobile settings. App alerts must be allowed on iOS or Android devices.

* Users may refuse to receive service alerts, and doing so does not restrict service use.

:::

## IP intercom settings

In the <Cmd sid="device_intercom_title" product="air" /> tab, manage SIP intercom settings.

<Image src="/img/air/bsair-device-bs3-sip-account-settings-intercom.png" />

:::info

For more information on SIP intercom settings, refer to [the following](sip-intercom-for-biostation3).

:::

## Leave or delete account {#manageAccount}

In the <Cmd sid="site_setting_tab_management" product="air" /> tab, leave the site or delete the account.

<Image src="/img/air/bsair-account-settings-management.png" />

* <Cmd sid="leave_site" product="air" />: Leave the site. After leaving a site, you must receive an invitation from the site administrator to join again.

* <Cmd sid="account_settings_option_3" product="air" />: Permanently delete the account. Deleting the account permanently deletes all personal information and settings and cannot be recovered.
