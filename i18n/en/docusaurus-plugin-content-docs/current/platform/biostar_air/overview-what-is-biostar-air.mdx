## 100% cloud access control platform

BioStar Air is a cloud-native access control solution developed by Suprema, a global leader in biometric security. Designed to eliminate the need for on-premises infrastructure, BioStar Air provides secure, flexible, and scalable access control that can be managed from anywhere. With smart readers that connect directly to the cloud, organizations can easily control access at multiple sites using only a phone or a browser.

## Unparalleled credential flexibility

BioStar Air provides the most comprehensive range of authentication options, enabling organizations to manage facility access in multiple secure and convenient ways.

* **Facial authentication**

* **Fingerprint** (coming soon)

* **Custom mobile credentials for iOS and Android** (free)

* **Apple Wallet** (coming soon)

* **Multiple RFID formats**

* **App-free Dynamic QR and LinkPass**

## No on-prem infrastructure required

BioStar Air eliminates the need for on-site servers, virtual machines, VPNs, or door controllers. No IT maintenance is required. Install a **smart reader** and connect it to the cloud to start managing access immediately. Readers operate across multiple sites regardless of network or physical location.

## Easy multi-site management

BioStar Air provides a complete solution for controlling access to every location—**offices**, **warehouses**, **retail stores**, or **remote sites**—all within a single system.

## Future-proof scalability with edge architecture

BioStar Air’s **edge architecture** embeds access control directly in the smart reader, removing the need for separate door controllers. This lets organizations **scale effortlessly**. Purchase only what is needed today and expand easily without complex upgrades.

## Optimized for cloud-centric businesses

BioStar Air is designed for **system integrators** and **customers** seeking a fully cloud-native access control solution with industry-leading biometrics. It offers a modern, flexible, and highly secure approach to access management for cloud-centric enterprises.
