BioStar Air is a cloud-native access control solution developed by Suprema, a global leader in biometric security. Designed to eliminate the need for on-premises infrastructure, BioStar Air provides secure, flexible, and scalable access control that can be managed from anywhere. With smart readers that connect directly to the cloud, organizations can easily control access at multiple locations using only a mobile phone or a browser.

## User guide overview

The user guide is divided into four role-based sections. Choose the guide that corresponds to your role.

* <DocLink docId='platform/biostar_air/site-admin-guide' />: Responsible for the day-to-day operation of a BioStar Air site, including registering users and credentials, setting up devices, doors, and elevators, configuring access permissions, monitoring events, and managing site and service settings.

* <DocLink docId='platform/biostar_air/partner-install-guide' />: Responsible for creating and managing sites via the Partner Portal, installing hardware such as readers, modules, and cameras, and configuring networks.

* <DocLink docId='platform/biostar_air/credential-users-guide' />: End users who unlock doors with mobile credentials via the Suprema Pass app and configure authentication methods and Bluetooth sensitivity.

* <DocLink docId='platform/biostar_air/developer-guide' />: Documentation for developers who want to automate user management and credential issuance via the BioStar Air Cloud API.
