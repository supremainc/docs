BioStar Air records shift and break times to provide basic **T&A** tracking and integrates with external T&A systems or exports the data. BioStar Air is not a complete T&A system, but it supports event logging and API integration with payroll and T&A systems. It also generates basic **working hours** reports.

## T&A events {#tna_events}

The following T&A events can be recorded. T&A events are logged and can be exported as a CSV file or retrieved through the API.

* **Shift start**

* **Shift end**

* **Break start**

* **Break end**

## Configure T&A {#settingTna}

Follow these steps to enable and configure T&A settings.

1. Click <Cmd sid="settings_menu" product="air" /> → <Cmd sid="site_submenu" product="air" /> in the left sidebar.

2. Click <Cmd sid="site_setting_tab_setting" product="air" /> at the top of the screen.

   <Image src="/img/air/bsair-settings-site-tab-settings.png" />

3. Activate <Cmd sid="site_profile_input8" product="air" />.

4. When the message window appears, click <Cmd sid="btn_ok" product="air" />Yes.

5. In <Cmd sid="site_profile_input8" product="air" />, click <Cmd sid="tna_setting" product="air" />.

   <Image src="/img/air/bsair-settings-site-tab-settings-enable-tna.png" />

6. When the <Cmd sid="site_profile_input8" product="air" /> window appears, set the label for <Cmd sid="device_edit_table_head1" product="air" />.

   > For example, Shift start, Shift end, Break start, Break end

   <Image src="/img/air/bsair-settings-site-tab-settings-tna-mng.png" />

7. Select whether the event type for <Cmd sid="device_edit_table_head1" product="air" /> is **Clock-In** or **Clock-Out**.

8. Click <Cmd sid="save_btn" product="air" /> to save the settings.

## Configure T&A on devices {#configuringTnaDevices}

After activating <Cmd sid="site_profile_input8" product="air" />, specify the devices that will use it.

1. Click <Cmd sid="device_menu" product="air" /> → <Cmd sid="device_submenu" product="air" /> in the left sidebar.

2. In the device list, click the device to configure for T&A.

3. When the <Cmd sid="device_edit_title" product="air" /> window appears, go to the <Cmd sid="new_device_edit_tna_title" product="air" /> section.

   <Image src="/img/air/bsair-device-mng-device-set-tna.png" />

4. Click the toggle for <Cmd sid="new_device_edit_tna_toogle" product="air" /> to enable T&A.

5. Click <Cmd sid="device_edit_tna_mobile_required_label" product="air" /> to require T&A selection before authentication.

6. Click <Cmd sid="save_btn" product="air" /> to save the settings.

:::note

T&A event labels are managed at the **site** level and cannot be changed per device.

:::

### <Cmd sid="device_edit_tna_title" product="air" className="normal" /> {#device_edit_tna_title}

A device records T&A events in several ways.

* <Cmd sid="device_edit_tna_mode_option1" product="air" />: The device does not record T&A data.

* <Cmd sid="device_edit_tna_mode_option3" product="air" />: Records clock-in and clock-out times based on the defined schedule. For example, events between 8:00 AM and 12:00 PM are recorded as **Clock-In**, while events between 1:00 PM and 5:00 PM are recorded as **Clock-Out**.

* <Cmd sid="device_edit_tna_mode_option5" product="air" />: Assigns a fixed T&A code to the device. Each authentication on that device records the same T&A event. Use this option to designate the device for clock-in or clock-out.

* <Cmd sid="device_edit_tna_mode_option6" product="air" />: Requires the user to select a T&A code before authentication. The selected T&A event is recorded with the access log.

## View and export T&A logs {#viewing-and-exporting-t-and-a-logs}

T&A events are saved to the event log and can be reviewed or exported to a CSV file.

1. Click <Cmd sid="monitoring_menu" product="air" /> → <Cmd sid="event_logs_submenu" product="air" /> in the left sidebar.

2. Locate the **T&A** column in the log table.

   <Image src="/img/air/bsair-monitoring-event-log-list.png" />

3. Click <IcAirColumn /> at the top of the log table to customize the view and move the **T&A** column for better visibility.

   <Image src="/img/air/bsair-monitoring-event-log-list-column-set.png" />

4. Click <IcAirDown /> and select <Cmd sid="report_download_csv" product="air" /> to download the data as a CSV file.
