Floor levels are used in elevator access control settings. Floor levels enable fine-grained control over access permissions for each floor. For example, configure certain user groups so they can access only specific floors.

## Add floor levels {#addFloorLevel}

Assign users or groups that can access specific floors.

1. Click <Cmd sid="access_menu" product="air" /> → <Cmd sid="floor_levels_submenu" product="air" /> in the left sidebar.

2. Click <Cmd sid="floor_levels_title_add_button" product="air" /> at the top right of the screen.

   <Image src="/img/air/bsair-access-mng-floor.png" />

3. When the <Cmd sid="add_floor_levels_add_floors_title" product="air" /> panel appears, enter <Cmd sid="schedule_name" product="air" /> and <Cmd sid="schedule_description" product="air" /> (optional).

   <Image src="/img/air/bsair-access-mng-floor-add.png" />

4. Select the floors to apply floor permissions in the <Cmd sid="add_floor_levels_add_floors" product="air" /> section.

5. Select the schedule to apply to the floor level in the <Cmd sid="floor_levels_list_column_schedule_name" product="air" /> section. Select <Cmd sid="always_schedule" product="air" /> to allow unrestricted access.

6. Click <Cmd sid="save_btn" product="air" /> to add the floor level.

## Edit floor levels {#editFloorLevel}

1. Click <Cmd sid="access_menu" product="air" /> → <Cmd sid="floor_levels_submenu" product="air" /> in the left sidebar.

2. Click the floor level you want to edit from the list. Alternatively, select the checkbox of the floor level you want to edit, then click <IcAirEdit /> at the top of the list.

   <Image src="/img/air/bsair-access-mng-floor-edit.png" />

3. When the <Cmd sid="add_floor_levels_edit_floors_title" product="air" /> panel appears, modify the necessary information or change the settings.

4. Click <Cmd sid="save_btn" product="air" /> to save the changes.

## Delete floor levels {#deleteFloorLevel}

1. Click <Cmd sid="access_menu" product="air" /> → <Cmd sid="floor_levels_submenu" product="air" /> in the left sidebar.

2. Select the checkbox of the floor level you want to delete from the list. More than one floor level can be selected.

3. Click <IcAirRemove /> at the top of the list.

4. When a warning message appears, click <Cmd sid="btn_yes" product="air" />.
