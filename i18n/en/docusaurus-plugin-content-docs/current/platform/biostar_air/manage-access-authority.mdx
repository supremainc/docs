## Assign access levels to users

Assign access levels to users.

1. Click <Cmd sid="user_menu" product="air" /> → <Cmd sid="user_submenu" product="air" /> in the left sidebar.

2. Select the checkbox next to the users in the list to assign an access level. You can select more than one user.

3. Click the <IcAirLock /> button at the top of the list.

4. Select the desired access level from the pop-up menu.

   <Image src='/img/air/bsair-access-authority-set.png' />

5. Click <Cmd sid="set_access_done_btn" product="air" />.

:::info

For more information about user management, see <DocLink docId='platform/biostar_air/user-management-user-list' />.

:::

</Step>
