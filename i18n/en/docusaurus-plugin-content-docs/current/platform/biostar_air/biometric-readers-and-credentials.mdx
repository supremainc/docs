BioStar Air now supports biometric readers and biometric credentials to enhance secure access control. Register and configure compatible biometric devices, such as BioStation 3 (BS3) and BioEntry W3 (W3), to authenticate users via facial authentication. The feature supports both single and two
factor authentication, enhancing flexibility and security.

## Overview {#overview}

* **Register readers**: Use the BioStar Air mobile app to register biometric readers.

* **Device setting**: Go to the [BioStar Air Portal](https://admin.biostarair.com) → <Cmd sid="device_menu" product="air" /> → <Cmd sid="device_submenu" product="air" /> to configure the device facial authentication setting.

* **Face credential setting**: Go to the [BioStar Air Portal](https://admin.biostarair.com) → <Cmd sid="user_menu" product="air" /> → <Cmd sid="user_submenu" product="air" /> to add or edit users. Activate the facial credential in the <Cmd sid="credential_detail" product="air" /> section.

## Step-by-step instructions {#step-by-step-instructions}

<Steps>
<Step>

### Register biometric device {#registerBiometricReader}

Verify that BioStar Air-compatible firmware is installed on the BioEntry W3 or BioStation 3. Models whose names end with `-AIR` already include the default firmware supplied by Suprema.

Launch the BioStar Air app on the mobile device and sign in with the administrator ID and password. Register the device in the same way as any other reader.

:::info

For more information on device registration, refer to [the following](register-device).

:::

</Step>
<Step>

### Configure reader {#configureDevice}

1. Log in to the [BioStar Air Portal](https://admin.biostarair.com) with an administrator account.

2. Click <Cmd sid="device_menu" product="air" /> → <Cmd sid="device_submenu" product="air" /> in the left sidebar.

3. Click the biometric reader in the device list.

4. When the <Cmd sid="device_edit_title" product="air" /> window appears, go to the <Cmd sid="new_layout_nav_title_authentication" product="air" /> section.

   <Image src="/img/air/bsair-device-bs3-info-edit.png" />

5. Set <Cmd sid="new_layout_nav_title_authentication_txt1" product="air" />.

   * **Single-factor**: Authenticate with either an RF card or facial authentication.

   * **Two-factor**: Delete either the RF card or facial authentication factor, then re-enroll both to enable two-factor authentication. Click <Cmd sid="add_btn" product="air" /> and combine desired authentication methods.

   <Image src="/img/air/bsair-device-two-fator-auth.png" width="30%" />

6. Set <Cmd sid="new_layout_nav_title_authentication_auth_timeout" product="air" />. If the credential is not authenticated within the specified time, authentication will fail.

7. Configure the following options in the <Cmd sid="new_layout_nav_title_face" product="air" /> section.

   * <Cmd sid="new_layout_device_bew3_block_7_title" product="air" />

   * <Cmd sid="new_layout_device_bew3_block_2_title" product="air" />

   * <Cmd sid="new_layout_device_bew3_block_3_title" product="air" />

   * <Cmd sid="new_layout_device_bew3_block_4_title" product="air" />

8. Configure additional options.

   * <Cmd sid="new_layout_device_bew3_block_7_text2" product="air" />

   * <Cmd sid="new_layout_device_bew3_block_8_title" product="air" />

   * <Cmd sid="new_layout_device_bew3_block_8_1_text" product="air" />

:::info

For more information on the settings in the <Cmd sid="new_layout_nav_title_authentication" product="air" /> section, refer to [the following](managing-registered-devices#faceAuthentication).

:::

</Step>
<Step>

### Enroll user faces {#add-or-edit-user-and-enroll-face}

1. Click <Cmd sid="user_menu" product="air" /> → <Cmd sid="user_submenu" product="air" /> in the left sidebar.

2. Open the window to add a new user or edit an existing user.

   * **Add a new user**: Click <Cmd sid="add_user_btn" product="air" /> in the upper right of the user list.

   * **Edit an existing user**: Select the user in the list, then click <IcAirEdit />.

3. From the user information window, go to the <Cmd sid="credential_detail" product="air" /> section and enable the <Cmd sid="card_type_option5" product="air" /> credential.

   <Image src="/img/air/bsair-user-info-credential-face.png" />

4. Click <IcAdd /> and add the facial template.

   <Image src="/img/air/bsair-user-info-credential-face-add.png" />

   * <Cmd sid="select_file" product="air" />: Select a photo file stored locally

   * <Cmd sid="register_on_device" product="air" />: Use the connected biometric reader (BioEntry W3 or BioStation 3)

   * <Cmd sid="register_on_mobile" product="air" />: Scan the QR code sent to the user’s mobile device and enroll the face from the mobile device

   :::note

   Only two face photos can be uploaded per user. Generally, one photo is sufficient.

   :::

5. Click <Cmd sid="save_btn" product="air" /> to complete face enrollment.

:::info

For more information on uploading user face photos in bulk, refer to [the following](adding-users-in-bulk#uploadProfileImages).

:::

</Step>
</Steps>

## Notes {#notes}

* BioStar Air-compatible firmware must be installed for the biometric reader to operate normally.

* Up to two face templates can be used per user.

* For advanced configuration, refer to the BioEntry W3 or BioStation 3 user guide.

* Enroll at least two credentials to use two-factor authentication. For example, enroll both RF card and face credentials.
