Set XPass 2 as the primary device to connect XPass D2 as a secondary device. Manage all access control logic on XPass 2. Configure a door with entry and exit devices without requiring two network connections.

This document provides step-by-step instructions to set XPass 2 as the primary device and connect XPass D2 as a secondary device. It also explains the features supported and the limitations of the primary-secondary configuration.

## Before you start

Update the firmware to version 2.12 or later before setting XPass 2 as the primary device. For more information about firmware upgrades, see <DocLink docId='platform/biostar_air/how-to-upgrade-firmware' />.

## Set XPass 2 as the primary device

1. Click <Cmd sid="device_menu" product="air" /> → <Cmd sid="device_submenu" product="air" /> in the left sidebar.

2. In the device list, click XPass 2.

3. When the <Cmd sid='device_edit_title' product='air' /> window appears, go to the <Cmd sid='new_layout_nav_title_network' product='air' /> section.

4. Change the <Cmd sid='device_xpass2_rs485' product='air' /> setting to <Cmd sid='device_xpass2_rs485_master' product='air' />.

   <Image src='/img/air/bsair-device-xpass2-rs485-set-master.png' />

5. Click <Cmd sid="save_btn" product="air" /> at the bottom to save the changes.

## Add a secondary device

Add a secondary device such as XPass D2 or Secure Module (SIO2). 

1. In the device list, click <IcAirMore title='More' /> at the far right of the XPass 2 set as the primary device.

   <Image src='/img/air/bsair-device-xpass2-add-slave.png' />

2. Click <Cmd sid='add_slave_devices' product='air' /> to open the <Cmd sid='add_slave_devices' product='air' /> window.

   <Image src='/img/air/bsair-device-xpass2-add-slave-find.png' />

   :::note

   If no secondary devices are found, click <Cmd sid='find_devices' product='air' /> to search again for devices connected to XPass 2.

   :::

3. When connected secondary devices appear in the list, click the checkbox next to each device you want to add.

4. Once your selections are complete, click <Cmd sid='add_btn' product='air' />.

Connected secondary devices appear hierarchically under XPass 2. See all devices connected to XPass 2 at a glance in the panel list.

:::info

* For more information on XPass D2, see the [XPass D2 installation guide](../../device/xpass_d2).

* An XPass 2 set as the primary device can have up to 31 secondary devices assigned.

* For more information about the Secure Module (SIO2), see the [Secure Module installation guide](../../device/securemodule).

:::
