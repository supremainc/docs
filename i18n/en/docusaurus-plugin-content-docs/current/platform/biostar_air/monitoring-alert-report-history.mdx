Learn how to check the history of alerts and reports generated in BioStar Air. Administrators can view past alerts and generated reports through the web portal and mobile app.

## Before start

The alert and report history shows only items generated by the alert and report rules configured in <Cmd sid="settings_menu" product="air" /> → <Cmd sid="alerts.alert_sub_menu_title" product="air" />. For more information on setting rules for creating alerts and reports, refer to [the following](settings-alert-report).

## Check alert and report history

Click <Cmd sid="monitoring_menu" product="air" /> → <Cmd sid="alerts.alert_sub_menu_title" product="air" /> in the left sidebar. The alert and report history appears.

<Image src="/img/air/bsair-monitoring-alert-report-history.png" />

* View the time, type, and details of each generated alert or report in the list.

* Click the date range area in the top-right corner of the screen, set the desired period, and filter the alert and report history within that period.
