With card templates in BioStar Air, administrators design and customize mobile cards for users. The templates enhance branding by including corporate colors, logos, and personalized details. Card templates apply to mobile cards used in the Suprema Pass app.

## Use default card templates {#usingDefaultCardTemplates}

Every site in BioStar Air includes default templates. Click <Cmd sid="user_menu" product="air" /> → <Cmd sid="card_template_submenu" product="air" /> in the left sidebar.

<Image src="/img/air/bsair-user-mobile-card-template.png" />

View the default templates. Administrators apply mobile card templates in one of the following three ways.

* Apply the default card template to a user's mobile card

* Modify parameters of an existing template to apply a customized mobile card

* Create a new custom template and apply it instead of the existing template

## Create a new card template {#creatingNewCardTemplate}

Create a new card template.

1. Click <Cmd sid="user_menu" product="air" /> → <Cmd sid="card_template_submenu" product="air" /> in the left sidebar.

2. Click <Cmd sid="add_new_card_template" product="air" />.

   <Image src="/img/air/bsair-user-mobile-card-template-new.png" />

3. When the <Cmd sid="card_templates_header" product="air" /> panel appears, set the card design and layout.

   <Image src="/img/air/bsair-user-mobile-card-template-new-card.png" />

4. After applying the desired settings, click <Cmd sid="save_btn" product="air" /> to save the mobile card template.

### <Cmd sid="card_styles_header" product="air" className="normal" /> {#cardStyles}

* <Cmd sid="card_styles_1" product="air" />: Set the background color of the mobile card.

* <Cmd sid="card_styles_2" product="air" />: Upload a background image for the mobile card. Click <Cmd sid="background_img_upload_btn" product="air" />, and then select a background image file stored locally. When the <Cmd sid="crop_image_header" product="air" /> window appears, click the <Cmd sid="fit_this_image" product="air" /> option or manually adjust the size. After completing the image adjustments, click <Cmd sid="save_btn" product="air" />.

  <Image src="/img/air/bsair-user-mobile-card-template-adjust-image.png" width="50%" />

### <Cmd sid="card_properties" product="air" className="normal" /> {#cardProperties}

## Set the default card template

Set a newly added mobile card template as the default.

1. Click <Cmd sid="user_menu" product="air" /> → <Cmd sid="card_template_submenu" product="air" /> in the left sidebar.

2. Click the checkbox in the upper right corner of the desired template.

3. Click <IcAirDef /> at the top left of the template list.

   <Image src="/img/air/bsair-user-mobile-card-template-set-default.png" />

4. When the message window appears, click <Cmd sid="card_template_yes_btn" product="air" />.

The default mobile card template is automatically applied when a new user is added.

:::info

Only one template can be set as default.

:::

## Edit card templates

1. Click <Cmd sid="user_menu" product="air" /> → <Cmd sid="card_template_submenu" product="air" /> in the left sidebar.

2. Click the template you want to edit in the template list. Alternatively, click the checkbox in the upper right of the template, and then click <IcAirEdit />.

3. When the <Cmd sid="card_templates_header" product="air" /> panel appears, modify the card design and layout.

4. Click <Cmd sid="save_btn" product="air" /> to save the changes.

5. When the message window appears, click <Cmd sid="card_template_replace_btn" product="air" /> or <Cmd sid="card_template_save_new_btn" product="air" />.

   <Image src="/img/air/bsair-user-mobile-card-template-edit-confirm.png" />

   * <Cmd sid="card_template_replace_btn" product="air" />: Save the changes to the existing template.

   * <Cmd sid="card_template_save_new_btn" product="air" />: Save the changes as a new template.

## Delete card templates

1. Click <Cmd sid="user_menu" product="air" /> → <Cmd sid="card_template_submenu" product="air" /> in the left sidebar.

2. In the template list, click the checkbox in the upper right of the template you want to delete. Select one or more templates.

3. Click <IcAirRemove /> at the top left of the template list.

   <Image src="/img/air/bsair-user-mobile-card-template-remove.png" />

4. When the message window appears, click <Cmd sid="btn_ok" product="air" />Yes.

:::info

Default templates cannot be deleted.

:::

## Apply templates to all users

1. Click <Cmd sid="user_menu" product="air" /> → <Cmd sid="card_template_submenu" product="air" /> in the left sidebar.

2. In the template list, click the checkbox in the upper right of the template you want to apply.

3. Click <Cmd sid="update_all_users1" product="air" /> at the top right of the screen.

   <Image src="/img/air/bsair-user-mobile-card-template-apply-all.png" />

4. When the message window appears, click <Cmd sid="btn_ok" product="air" />Yes.

:::info

* Applying the selected card template to all users replaces any template previously set.

* Only one card template can be applied to all users.

:::

## Change a user's card template {#applying-card-templates-to-users}

Follow these steps to change an individual user's card template.

1. Click <Cmd sid="user_menu" product="air" /> → <Cmd sid="user_submenu" product="air" /> in the left sidebar.

2. Click the user in the user list to apply a card template.

3. Click <IcAirEdit /> in the top right of the user details.

   <Image src="/img/air/bsair-user-mobile-card-template-apply-user-edit.png" width="70%" />

4. When the <Cmd sid="user_edit_header" product="air" /> window appears, click <IcAirL /> or <IcAirR /> under Card Template in the right panel to select the desired card template.

   <Image src="/img/air/bsair-user-mobile-card-template-apply-user-details.png" width="70%" />

5. Click <Cmd sid="save_btn" product="air" /> to apply the changes.
