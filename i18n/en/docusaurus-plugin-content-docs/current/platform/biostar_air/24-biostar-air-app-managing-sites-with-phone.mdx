## Feature Overview {#feature-overview}

The BioStar Air app provides full functionality equivalent to the BioStar Air web portal. Users can manage access control, monitor event logs, control devices, and perform administrative tasks directly from their mobile devices.

<CustomerNote />


## Navigating to the Feature {#navigating-to-the-feature}

To start using the BioStar Air app:

* Download and install the BioStar Air app on your mobile device.

  * [iOS](https://apps.apple.com/us/app/airfob-pro/id1659959609)

  * [Android](https://play.google.com/store/apps/details?id=com.mocainc.afs2pro)

* Log in using the same ID and password as the web version.


## Step-by-Step Instructions {#step-by-step-instructions}

<Steps>

<Step>

### Dashboard Overview {#dashboard-overview}

After logging in, you will land on the **Dashboard**, which displays:

* Connected cameras (if any).

* Daily user tag-in count.

* Total registered active users.

* Number of connected devices.

* Credit status.

</Step>
<Step>

### User Management {#user-management}

1. Navigate to the **User** tab.

1. Select a user to view their details.

1. You can modify user status (Reissue, Suspend, Delete, or Activate).

1. To add a new user, tap the **+** button and complete the necessary details.

</Step>
<Step>

### Monitoring {#monitoring}

The **Monitoring** tab provides:

* Event logs.

* Door status.

* Live video (if cameras are connected).

</Step>
<Step>

### Accessing More Features {#accessing-more-features}

Tap the **hamburger menu** for a full list of features, including:

#### Devices {#devices}

1. View all registered devices.

1. Select a device to edit, update firmware, or reboot.

1. Use **BLE mode** for device updates when not connected to a network (proximity required).

1. To add a new device, tap the **+** button.

#### Doors {#doors}

1. View door status and control settings.

1. Tap once to edit a door or tap and hold to select multiple doors for bulk actions.

1. Delete doors if needed.

#### Event Logs {#event-logs}

1. Filter logs based on specific criteria.

1. Check door status and remotely control doors.

1. Set doors to **Unlock Mode** or **Lock Mode**, overriding the relay settings or schedule.

1. Start a **Roll Call**. For more information, refer to [the following](managing-roll-call-fire-muster).

</Step>
</Steps>

## Common Issues & Considerations {#common-issues-considerations}

* Ensure you log in with the same credentials as the web version.

* For BLE mode updates, be in close proximity to the device.

* Some features, such as live video monitoring, require connected cameras.
