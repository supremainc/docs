The Suprema Pass app serves as the user interface for the BioStar Air system and primarily acts as a digital wallet for mobile credentials. Users issue and store mobile access cards in the app to enter secure areas with their mobile devices.

* Download and install Suprema Pass.

  * [iOS](https://apps.apple.com/kr/app/suprema-pass/id6745057639)

  * [Android](https://play.google.com/store/apps/details?id=com.supremainc.supremapass)

* Log in on the mobile device using the credentials provided by the administrator.

:::info

* Bluetooth must be enabled to authenticate with the mobile card.

* Enabling two-step authentication disables <Cmd>Background Mode</Cmd>.

* The administrator must enable QR authentication and attendance management in the BioStar Air portal.

* Verify that the site supports the hands-free access method.

:::
