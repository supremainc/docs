## Go to user management {#accessingUserManage}

Use the user management feature in the BioStar Air app.

1. Run BioStar Air on your mobile device.

2. Tap <Cmd sid="site_setting_tab_management" product="air" /> in the bottom navigation bar.

3. Tap <Cmd sid="user_submenu" product="air" />.

A list of users appears. Add new users or edit existing user details.

## Add users {#accessingUserManage}

The mobile app also allows adding new users.

1. Tap <IcAdd /> at the top of the user list.

2. Enter the required details in <Cmd sid="add_user_form_header" product="air" />.

3. Select whether the user is regular or temporary in <Cmd sid="select_user_card_type_label" product="air" />.

4. Issue credentials for access in <Cmd sid="credential_detail" product="air" />.

5. Enter the user's email address or mobile phone number in <Cmd sid="user_form_input8" product="air" />.

6. Set the validity period and access level.

7. To issue the card later, tap <Cmd sid="save_btn" product="air" />, or tap <Cmd sid="add_user_card_activate_btn" product="air" /> to issue the credentials immediately.

:::info

For more information about adding users, refer to [the following](adding-individual-users).

## User actions {#userActions}

Tap a user in the user list. In the <Cmd sid="user_detail" product="air" /> screen, tap <span className="rect green"><IcAirMore /></span>. The **User actions** menu appears.

* <Cmd sid="after_suspend_user_reactivate_action" product="air" />: Reactivates suspended users.

* <Cmd sid="user_reactivate_action" product="air" />: Reissues the user's credentials.

* <Cmd sid="suspend_action" product="air" />: Temporarily deactivates the user. Suspended users cannot access.

* <Cmd sid="delete_action" product="air" />: Deletes the user from the system.

:::info

The <Cmd sid="after_suspend_user_reactivate_action" product="air" /> option appears only for suspended users.

:::

## Edit user information

Edit user details.

1. Tap the user to edit in the user list.

2. Tap <IcAirEdit /> in the top right.

3. Edit the necessary information on the <Cmd sid="user_edit_header" product="air" /> screen.

4. To save the changes, tap <Cmd sid="save_btn" product="air" />.

   Use <Cmd sid="add_user_card_activate_btn" product="air" /> to issue credentials.

## Filter users {#filteringUsers}

Filter the user list by the desired criteria. Tap <Cmd sid="logs_filters_action_btn" product="air" /> at the top right of the user list. Users can be filtered by the following criteria.

* <Cmd sid="user_table_column8" product="air" />: Filters by user type (regular or temporary).

* <Cmd sid="user_table_column2" product="air" />: Filters by credential type.

* <Cmd sid="user_table_column4" product="air" />: Filters by activation status.

* <Cmd sid="user_table_column5" product="air" />: Filters by user group.

## Perform batch actions {#performing-batch-actions}

Select multiple users to perform user actions in batch.

1. Long-press a user in the user list.

2. Tap additional users to select them.

3. Tap one of the **User actions** provided at the bottom of the screen.

:::info

For more information on the **User actions** feature, refer to [the following](#userActions).

## Manage user groups {#manageUserGroups}

Manage user groups. Tap <IcAirGroup /> at the top of the user list. The user group list appears.

* To view users in a group, tap <IcBw /> to the right of the desired group.

* To add a new group, tap <IcAdd /> in the top right.

* To edit a group name, long-press the desired group and tap <IcAirEdit /> at the bottom.

* To delete a group, long-press the desired group and tap <IcAirRemove /> at the bottom. Select one or more groups to delete them.

:::info

For more information on user group management, refer to [the following](managing-user-groups).
