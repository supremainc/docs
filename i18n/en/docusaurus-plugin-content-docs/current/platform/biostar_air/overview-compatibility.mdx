BioStar Air (formerly Airfob Space) uses an edge architecture that connects readers directly to the cloud, so it does not require on-premise servers, virtual machines (VMs), VPNs, or door controllers. Because no IT maintenance is required, install a **Smart Reader** and connect it to the cloud to start access control. Readers operate across multiple sites without sharing the same network or physical location.

## Supported Devices

BioStar Air is compatible with various access control devices and seamlessly integrates with devices in Suprema and MOCA systems to enable cloud-based access control.

### Readers

* **XPass 2** (MDPB, GDPB, GKDPB)

* **X-Station 2** (DPB)

* **BioEntry W3**

* **BioStation 3**

* **XPass D2** (coming soon)

### Patch devices

* **Airfob Patch** (AP-H)

* **Airfob Tag** (AP-B)

### Other compatible devices

* **Output Module** (OM-120)

* **SIO2**

* **CoreStation 20** (CS-20, coming soon)

* **Door Interface** (DI-24, coming soon)
