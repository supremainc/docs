## Firmware Version 2.4.0 (Build No. 2.4.0_240603)

Release<Separator/>2024-06-03

This release introduces new authentication features and improves system behavior during initialization and API processing.

### Key Features and Improvements

1. Added video gateway functionality.

1. Improved API error counting to only count actual API response errors, not timeouts

1. Improved resetting of device time and language settings during initialization
