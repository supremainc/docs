## Firmware Version 2.10.0 (Build No. 2.10.0_250813)

Release<Separator/>2025-08-13

This release significantly expands device capabilities, including video and communication, and credential support.

### New Features and Improvements

1. Supports intrusion alarm.

1. Supports Wiegand format.

1. Supports RF card MSB/LSB option.

1. Supports user PIN credential.

1. Added SIP intercom functionality.

1. Supports RTSP video streaming.

1. Supports QR code authentication.
