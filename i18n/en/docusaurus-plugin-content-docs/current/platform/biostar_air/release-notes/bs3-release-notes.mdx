## Firmware Version 2.10.2 (Build No. 2.10.2_251024)

Release<Separator/>2025-10-31

:::caution

Because of the enhanced encryption, downgrading to an earlier firmware version will require re-registering the device.

:::

### New Features and Improvements

1. Improved reliability of automatic user synchronization with the server

1. Enhanced encryption for configuration data

1. Added support for Video Gateway integration

### Bug Fixes

1. Fixed occasional boot-up failures

1. Fixed an issue where the Time & Attendance confirmation screen did not close automatically when **T&A required** was enabled but facial authentication was disabled
