title: Purpose
description: Purpose and scope of Suprema Integration with Gallagher middleware, enabling biometric device communication with Gallagher Command Centre.
keywords: [ Middleware, Command Centre, Biometric Devices ]
---

Suprema Integration with Gallagher is a middleware that allows the Gallagher Command Centre 8.8 or later System to communicate with the Suprema Biometric devices, which can register Biometric Face or Finger credentials to users from the Command Centre, and manage connected device settings. With Suprema Integration with Gallagher, you can easily set up and build the Biometric Management System for the Command Centre using Suprema biometric devices.

:::info

For more details on the functionality of the Gallagher Command Centre access control system, see the user manuals for the Command Centre.

:::
