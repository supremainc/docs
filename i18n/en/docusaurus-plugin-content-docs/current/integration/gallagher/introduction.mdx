## Features

Suprema integration with Gallagher communication between Gallagher Command Centre and Suprema BioStar 2. So, you can generate biometric templates for the suprema Devices from the Gallagher Command Centre, which then provides a futuristic solution for all end users.

**The following are the features provided by Suprema integration with Gallagher:**

* Bulk user enrollment and Management from Gallagher Command Centre will be reflected in Suprema BioStar 2.

* From the Suprema Integration, users can register Biometric Visual Face and Fingerprint data through scanning from various Suprema Biometric devices.

* CSN and Wiegand detection and integration with Biostar for automatic data enrichment.

* From the Suprema Integration, users can enroll card data through scanning from biometric devices like X-Station 2, BioLite N2, BioStation 3, FaceStation F2, XPass 2, X-Station 2, BioEntry W3, and BioEntry W2 (These are tested devices).

* Allows up to 1,000 Biometric Devices to Manage.

* All Suprema Device configurations can be done within the same window of the Command Centre.

* Supports a wide range of card types, including TOC cards, QR cards, and even custom card configurations tailored for specific customer needs.

* Integration setup is simplified through a bundled EXE package that includes all necessary prerequisites, ensuring a smooth and efficient installation process.

* Middleware includes built-in support for network-related configurations, along with a comprehensive troubleshooting guide to assist with misconfigurations and fault tolerance scenarios.
