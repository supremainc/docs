Before using the Suprema - C•CURE Integration to implement Suprema Biometrics to the C•CURE 9000 Platform for an access control system, both C•CURE 9000 v2.90, v3.0, v3.10 (or later) Server and Suprema BioStar 2 server along with this Integration Setup service must be installed on the same server/workstation where C•CURE server is installed, and C•CURE 9000 v2.90, v3.0, v3.10 (or later) Client for the workstation(s) where it is going to be used along with this integration setup client.

The Server receives event logs, user information, etc. from connected devices and stores them. The integration can be installed easily. Before installation, please check the Minimum System Requirements.

The Suprema - C•CURE Integration installation setup for server and client can be provided upon request made to the [Suprema Team](https://support.supremainc.com).

## Abbreviations

* Software House C•CURE 9000 (C•CURE)

* Suprema’s BioStar 2 (BioStar 2)

* Suprema - C•CURE Integration ("Integration" for short)

This integration comprises the following components:

* Suprema - C•CURE Integration Service (Integration Service)

* Suprema - C•CURE Integration Client (Integration Client)

* Suprema - C•CURE Integration Configuration Tool (Integration CT)

   * The Integration CT is the web-based GUI tool to configure the Integration Service (with no GUI) and to adjust parameters regarding the Integration Client as well.

* Integration CT is optimized for MS Edge, Brave Browser or Google Chrome.

* Please do not use older versions of Windows, that can lead to system instability due to OS libraries incompatibility as Microsoft has made substantial changes in OS libraries compared to older versions of those specified in the previous table.

## Supported Devices

RFID Devices:

* XPass 2 family

* X-Station 2 family

Facial Devices:

* FaceStation F2

* BioStation 3

* BioEntry W3

Fingerprint Devices:

* BioEntry Family (P2, W2)

* BioLite N2

* BioStation 2, BioStation A2

* X-Station 2 (Finger model)

* BioStation L2

NOT Supported Devices:

* BioEntry R2

* CoreStation

* FaceStation 2 / FaceLite

:::info

USB readers as the BioMini Plus 2 / Slim 2 can be supported through customization. For such please contact `latam@supremainc.com`.

:::
