* **New User**: Click the <Cmd sid='team.user.dialog.newUser' /> button at the top right of the screen. The <Cmd sid='team.user.dialog.newUser' /> window appears.

* **Existing User**: Double-click a user in the user list. Or, click the user and click the <Cmd sid='team.user.preview.button.seeMore' /> button in the preview screen displayed on the right side of the screen. A window for editing user information appears.
