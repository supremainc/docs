* <Cmd sid='setting.account.administrator' product='2' />: This is an administrator level that can use all menus.

* <Cmd sid='setting.account.userOperator' product='2' />: <Cmd sid='shell.header.user' /> menu has **read** and **write** permissions.

* <Cmd sid='setting.account.monitoringOperator' product='2' />: <Cmd sid='shell.header.arena' />, <Cmd sid='shell.header.data' />, and <Cmd sid='shell.header.dashboard' /> menus have **read** and **write** permissions.

* <Cmd sid='setting.account.taOperator' product='2' />: <Cmd sid='common.gnb.time_attendance' product='2' /> menu has **read** and **write** permissions.

* <Cmd sid='setting.account.visitorOperator' product='2' />: <Cmd sid='common.gnb.visitor' product='2' /> menu has **read** and **write** permissions.

:::info

* Select <Cmd sid='common.selectNone' /> if you do not want to set an admin level. In this case, the user cannot log in to **BioStar X**.

</Include>
