:::info

The following applies only to BioStation 3, X-Station 2, BioStation 2a, and BioLite N2. The version criteria differ by model, but the behavior is the same.

:::

### Version differences by product

| Product name | Factory version (older version) | Master administrator add version | New version (CE/global distinction) |
| --- | --- | --- | --- |
| BioStation 3 | Version 1.3.1 or lower | Version 1.4.0 to 1.4.1 | Version 1.4.2 or higher |
| X-Station 2 | Version 1.3.2 or lower | Version 1.4.0 | Version 1.4.1 or higher |
| BioStation 2a | Version 1.1.1 or lower | Version 1.2.0 | Version 1.2.1 or higher |
| BioLite N2 | Version 1.6.2 or lower | Version 1.7.0 | Version 1.7.1 or higher |
