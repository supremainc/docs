1. Click <Cmd sid="access_menu" product="air" /> → <Cmd sid="holiday_groups_submenu" product="air" /> in the left sidebar.

2. Click <Cmd sid="holiday_group_add_btn" product="air" /> at the top right of the screen.

   <Image src="/img/air/bsair-access-mng-holiday.png" />

3. Enter the group name when the <Cmd sid="holiday_group_add_title" product="air" /> panel appears.

   <Image src="/img/air/bsair-access-mng-holiday-add.png" />

4. In the <Cmd sid="add_holidays_title" product="air" /> section, specify the holiday name and date, and click <Cmd sid="add_btn" product="air" />.

   * Set the name and date, and select the <Cmd sid="holiday_group_checkbox_annually" product="air" /> option to add a fixed-date public holiday.

     > e.g., January 1 New Year’s Day, May 5 Children’s Day

   * Set the name and date, and select the <Cmd sid="holiday_group_checkbox_duration" product="air" /> option for holidays that recur annually but span a specific period.

     > e.g., October 5–8 Chuseok holidays

5. Add other holidays in the same way.

   <Image src="/img/air/bsair-access-mng-holiday-add-new-day.png" />

6. Click <Cmd sid="save_btn" product="air" /> to save the holiday group settings.
