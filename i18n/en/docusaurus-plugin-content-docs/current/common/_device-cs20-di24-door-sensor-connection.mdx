## Door sensor and exit button

SUPERVISED INPUT ports No. 0 - No. 5 can be set to be used for normal Input or Supervised Input.

<Image src='/img/device/installation-door-senser-rte-connection-cs20.png' width='60%' className='none' />

:::info

The SUPERVISED INPUT **Alarm** port can be used by connecting 1 k立, 2.2 k立, 4.7 k立, or 10 k立 resistors. After connecting the resistor corresponding to the input device of the connection, set the same resistance value in BioStar 2.

:::
