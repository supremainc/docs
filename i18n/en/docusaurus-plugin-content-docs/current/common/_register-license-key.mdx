## Registering in an online state {#registering-online}

To activate the **BioStar X** license while online with an internet connection, enter your name and the received license key, then click <Cmd sid='setting.server.license.button.activate' product='2' />.

<Image src='/img/biostarx-tutorial-license-online.png' />

## Registering in offline {#registering-offline}

To activate the **BioStar X** license in a closed network environment or in an offline state with limited internet access, please follow the instructions below.

1. In the <Cmd sid='setting.xlicense.activation' product='2' /> section, click <Cmd sid='setting.xlicense.offlineActivation' product='2' />.

   <Image src='/img/biostarx-tutorial-license-offline.png' />

2. When the <Cmd sid='setting.server.license.title.activateLicenseOffline' product='2' /> window appears, click <Cmd sid='setting.xlicense.button.requestOfflineKey' product='2' />.

   <Image src='/img/biostarx-tutorial-license-offline-popup.png' />

3. When the dialog appears, enter <Cmd sid='setting.xlicense.offlineActivation.requestBy' product='2' /> and <Cmd sid='setting.xlicense.offlineActivation.activationKey' product='2' />.

   <Image src='/img/biostarx-tutorial-license-offline-popup-filed.png' className='none' />

4. Click the <Cmd sid='setting.server.license.button.download' product='2' /> button to download the license request file (*\*.req*).

5. Send it to the purchase location.

Once you receive the license file (*\*.lic*) from the purchase location, click the <Cmd sid='setting.xlicense.offlineActivationTitle' product='2' /> button to upload the license file.

:::info

If you do not have a license key, only enter <Cmd sid='setting.xlicense.offlineActivation.requestBy' product='2' />.

:::
