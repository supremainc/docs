## Registering in an online state {#registering-online}

To activate the <Cmd>BioStar X License</Cmd> online while connected to the internet, please enter your name and the activation key you received, then click the <Cmd sid='setting.server.license.button.activate' product='2' /> button.

<Image src='/img/biostarx-tutorial-license-online.png' />

## Registering in offline {#registering-offline}

To activate the <Cmd>BioStar X License</Cmd> in a closed network environment or in an offline state with limited internet access, follow the instructions below.

1. Click <Cmd>Offline Activation</Cmd> in the <Cmd>Activation</Cmd> section.

   <Image src='/img/biostarx-tutorial-license-offline.png' />

2. When the <Cmd sid='setting.server.license.title.activateLicenseOffline' product='2' /> window appears, click <Cmd sid='setting.server.license.button.requestOfflineKey' product='2' />.

   <Image src='/img/biostarx-tutorial-license-offline-popup.png' />

3. When the dialog box appears, enter the company name (<Cmd>Request by</Cmd>) and the activation key (<Cmd>License Key</Cmd>).

   <Image src='/img/biostarx-tutorial-license-offline-popup-filed.png' className='none' />

4. Click the <Cmd sid='setting.server.license.button.download' product='2' /> button to download the license request file (*\*.req*).

5. Please send it to the regional dealer.

When you receive the license file (*\*.lic*) from the regional dealer, click the <Cmd sid='setting.server.license.button.activate' product='2' /> button to upload the license file.

:::info

If you do not have an activation key, enter only the company name (<Cmd>Request by</Cmd>).

:::
