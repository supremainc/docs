You can configure the <strong>{props.name}</strong> field to be mapped with the user field of **BioStar 2**.

Select the field of <strong>{props.name}</strong> to use as the user field of **BioStar 2** in <Cmd sid='settings.activeDirectory.FieldConfiguration' product='2' />.

<Image src={`/img/${props.img01}`} />

:::info

Each user field of **BioStar 2** is set by default to map to the corresponding user information in <strong>{props.name}</strong>.

To select a field value other than the default, click the field in <Cmd sid={props.sid} product='2' /> and select the desired field value.

:::
