You can set the field for <strong>{props.name}</strong> to map to the user fields of **BioStar X**. Select the field for <strong>{props.name}</strong> to use as the user field for **BioStar X** in the <Cmd sid='settings.activeDirectory.FieldConfiguration' product='2' /> section.

<Image src={`/img/${props.img01}`} />

:::info

Each user field of **BioStar X** is set by default to map to items that match the user information of <strong>{props.name}</strong>. To select a field value other than the default, click the field in <Cmd sid={props.sid} product='2' /> and select the desired field value.
