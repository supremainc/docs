## Installation Guide

1. Go to the [Suprema Download Center](https://download.supremainc.com/), log in, and download the installation package (*BioStar 2 Setup(x64).X.Y.Z.BB.exe*).

1. Run the downloaded installation file.

1. Select the language to use and select the <Cmd code='biostar2.install.ok' className='bold'/> button.

    <Image src='/img/biostar2-install-select-language.png' />

1. To continue the installation, select <Cmd code='biostar2.install.agree' className='bold'/> and click the <Cmd code='biostar2.install.next' className='bold'/> button.

    <Image src='/img/biostar2-install-agree.png' />

1. Enter the administrator account password and click the <Cmd code='biostar2.install.next' className='bold'/> button.

    <Image src='/img/biostar2-install-input-admin-pw.png' />

<Include pages='tutorial-express-install'>

6. Select <Cmd code='biostar2.install.expressInstall' className='bold'/> and click the <Cmd code='biostar2.install.next' className='bold'/> button.

    <Image src='/img/biostar2-install-select-quick.png' />

7. Enter the root account password for the database and click the <Cmd code='biostar2.install.next' className='bold'/> button.

    <Image src='/img/biostar2-install-input-db-pw.png' />

</Include>

<Include pages='tutorial-custom-install'>

6. Select <Cmd code='biostar2.install.customInstall' className='bold'/> and click the <Cmd code='biostar2.install.next' className='bold'/> button.

    <Image src='/img/biostar2-install-select-custom.png' />

7. Enter the details of the pre-configured database and click the <Cmd code='biostar2.install.next' className='bold'/> button.

    <Image src='/img/biostar2-install-input-db-details.png' />

</Include>

8. Set the installation path for **BioStar X** and click the <Cmd code='biostar2.install.next' className='bold'/> button.

    <Image src='/img/biostar2-install-set-path.png' />

1. Set the path to store the encryption key and click the <Cmd code='biostar2.install.next' className='bold'/> button.

    <Image src='/img/biostar2-install-set-encrypt-path.png' />

1. Read the information about the management and responsibility of personal information stored in the database, and click the <Cmd code='biostar2.install.next' className='bold'/> button to continue the installation.

    <Image src='/img/biostar2-install-read-covenant.png' />

1. Set the port for **BioStar X** communication and click the <Cmd code='biostar2.install.next' className='bold'/> button.

    <Image src='/img/biostar2-install-set-port.png' />

1. Select the components of **BioStar X** and click the <Cmd code='biostar2.install.next' className='bold'/> button. If you select **USB Device Agent**, the USB Agent and driver for using BioMini, BioMini Plus 2, BioMini Slim 2, and DUALi DE-620 will be installed together.

    <Image src='/img/biostar2-install-additional.png' />

1. When all preparations for installation are complete, click the <Cmd code='biostar2.install.btnInstall' className='bold'/> button.

    <Image src='/img/biostar2-install-complete.png' />

1. Select whether to install additional programs and click the <Cmd code='biostar2.install.complete' className='bold'/> button.

    <Image src='/img/biostar2-install-additional-fin.png' />

1. Complete the installation of **USB Device Agent** by following the instructions on the installation screen.

:::info

* In the downloaded file name, `X.Y.Z` is the version information and `BB` is the build number.

* The administrator account password is used when logging in for the first time after installing **BioStar X**.

* The root account password for the database is used as the initial password for AC, TA, and Video DB.

* The storage path for the encryption key can be changed. If you modify or move the encryption key file after changing the path, a system error may occur.

* When **BioStar X** is deleted, the encryption key file is also deleted.

* The provided USB Device Agent certificate can only be applied to the local network.

* If another program is using port 443, the BioStar X Setting program runs automatically and allows you to change the port number. For details on changing the port number, refer to [this document](change-port).

* For more information about changing database settings, refer to [this document](change-db).

:::

:::warning

* Be careful not to lose the administrator or root account password for the database.

* If you lose the password, version upgrade and DB backup/restore may not be possible.

:::
