## Installation guide

1. Go to the [Suprema Download Center](https://download.supremainc.com/), log in, and download the installation package (*BioStar X Setup.X.Y.Z.BB.exe*).

2. Run the downloaded installation file.

3. Select the language to use and select the <Cmd code='biostar2.install.ok' className='bold'/> button.

   <Image src='/img/biostarx-install-select-language.png' />

4. To continue the installation, select <Cmd code='biostar2.install.agree' className='bold'/> and click the <Cmd code='biostar2.install.next' className='bold'/> button.

   <Image src='/img/biostarx-install-agree.png' />

5. Enter the administrator account password and click the <Cmd code='biostar2.install.next' className='bold'/> button.

   <Image src='/img/biostarx-install-input-admin-pw.png' />

<Include pages='express-install'>

6. Select <Cmd code='biostar2.install.expressInstall' className='bold'/> and click the <Cmd code='biostar2.install.next' className='bold'/> button.

   <Image src='/img/biostarx-install-select-quick.png' />

7. Enter the root account password for the database and click the <Cmd code='biostar2.install.next' className='bold'/> button.

   <Image src='/img/biostarx-install-input-db-pw.png' />

</Include>

<Include pages='custom-install'>

6. Select <Cmd code='biostar2.install.customInstall' className='bold'/> and click the <Cmd code='biostar2.install.next' className='bold'/> button.

   <Image src='/img/biostarx-install-select-custom.png' />

7. Enter the details of the pre-configured database and click the <Cmd code='biostar2.install.next' className='bold'/> button.

   <Image src='/img/biostarx-install-input-db-details.png' />

</Include>

8. Set the installation path for **BioStar X** and click the <Cmd code='biostar2.install.next' className='bold'/> button.

   <Image src='/img/biostarx-install-set-path.png' />

9. Set the path to store the encryption key and click the <Cmd code='biostar2.install.next' className='bold'/> button.

   <Image src='/img/biostarx-install-set-encrypt-path.png' />

10. Check the contents of the security warning message and click <Cmd code='biostar2.install.ok' className='bold' />.

    <Image src='/img/biostarx-install-set-encrypt-path-confirm.png' />

11. Read the information about the management and responsibility of personal information stored in the database, and click the <Cmd code='biostar2.install.next' className='bold'/> button to continue the installation.

    <Image src='/img/biostarx-install-read-covenant.png' />

12. Set the port for **BioStar X** communication and click the <Cmd code='biostar2.install.next' className='bold'/> button.

    <Image src='/img/biostarx-install-set-port.png' />

13. Select the components of **BioStar X** and click the <Cmd code='biostar2.install.next' className='bold'/> button. If you select **USB Device Agent**, the USB Agent and driver for using BioMini, BioMini Plus 2, BioMini Slim 2, and DUALi DE-620 will be installed together.

    <Image src='/img/biostarx-install-additional.png' />

14. When all preparations for installation are complete, click the <Cmd code='biostar2.install.btnInstall' className='bold'/> button.

    <Image src='/img/biostarx-install-complete.png' />

15. Select whether to install additional programs and click the **Finish** button.

    <Image src='/img/biostarx-install-additional-fin.png' />

16. Complete the installation of **USB Device Agent** by following the instructions on the installation screen.
