## Door Control

You can manually open, lock, or unlock the connected door.

:::info

* You can control the door only when the device is configured as the master device.

* Use <Cmd sid='ui_300' product='dev' /> or <Cmd sid='ui_194' product='dev' /> to control the door.

:::

1. Select <Cmd sid='ui_380' product='dev' /> on the main screen.

   <Include product='biostation_3'><Image src='/img/device/bs3-door-control1.png' /></Include>
   <Include product='biostation_3_max'><Image src='/img/device/bs3m-door-control1.png' /></Include>

2. Authenticate using administrator privilege method.

3. Choose the desired door control method.

   <Include product='biostation_3'><Image src='/img/device/bs3-door-control2.png' /></Include>
   <Include product='biostation_3_max'><Image src='/img/device/bs3m-door-control2.png' /></Include>

   * <Cmd sid='ui_382' product='dev' />: Set the time for immediate door opening or to keep the door open.

   * <Cmd sid='ui_383' product='dev' />: Set the time for immediate locking or to keep the door locked.

   * <Cmd sid='ui_384' product='dev' />: Cancel manual opening or manual lock hold.

:::info

Set the manual open and manual lock-hold times from 1 to 86,400 seconds.

:::
