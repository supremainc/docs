### <Cmd sid='new_layout_nav_title_detail' product='air' className='normal' /> {#deviceInfo}

The <Cmd sid='new_layout_nav_title_detail' product='air' /> section provides basic information — such as the device name, model, serial number, and firmware version — and includes an action menu for controlling features and default settings.

#### Basic Information

View the device's basic information. This information cannot be edited.

* <Cmd sid='device_non_editable_fields_product' product='air' />: The product name.

* <Cmd sid='device_non_editable_fields_model' product='air' />: The device's model name.

* <Cmd sid='device_non_editable_fields_serial' product='air' />: The device's serial number.

* <Cmd sid='device_non_editable_fields_firmware' product='air' />: The firmware version installed on the device.

#### Actions

The **Action** feature lets you control the device's functions.

<Image src='/img/air/bsair-device-mng-edit-actions.png' />

* <Cmd sid='firmware_update_btn' product='air' />: Update the device firmware to the latest version.

* <Cmd sid='device_edit_actions_reboot' product='air' />: Reboot the device remotely when an issue or anomaly occurs.

* <Cmd sid='device_edit_actions_factory_reset' product='air' />: Initialize all settings stored on the device. This action deletes all settings and data on the device.

* <Cmd sid='device_edit_actions_factory_reset' product='air' /> <Cmd sid='device_edit_actions_excluding_network' product='air' />: Initialize all settings except the network settings. This action deletes all settings and data but keeps the network settings.

#### Initial Settings

Modify the device's default settings.

* <Cmd sid='device_editable_fields_nick_name' product='air' />: The alias shown in the device list and logs. You can edit this field directly.

* <Cmd sid='device_editable_fields_timezone' product='air' />: Set the device time zone to keep logs consistent when the site and device clocks differ. Activate the <Cmd sid='device_field_dst_usage' product='air' /> option to apply daylight saving time (DST).

<Include type={props.type} ref='xpass2'>

* <Cmd sid='keypad_use_rf_card_number' product='air' />: Enter the RF card number using the keypad to use it.

   :::note

   This feature is supported only on the **XP2-GKDPB** model, which has a keypad.

   :::

</Include>

* <Cmd sid='device_editable_fields_hands_free' product='air' />: Extend the BLE range.

<Include type={props.type} ref='ae-mc,xpass2,xs2'>

* <Cmd sid='device_editable_fields_qr_reader' product='air' />: Confirm that an RS-485 QR reader is connected to the device.

   :::note

  * When a QR reader is connected to the device, you cannot use another RS-485 device.

  * When using an SIO2, you cannot use a QR reader on the device connected to that door.

   :::

</Include>

<Include type={props.type} ref='ae-mc,xs2'>

* <Cmd sid='device_editable_fields_card' product='air' />: Activate or deactivate card authentication. DESFire and Felica card formats are supported.

</Include>

<Include type={props.type} ref='ae-mc,xpass2'>

* <Cmd sid='device_editable_fields_beep_setting' product='air' />: Configure the success beep for access authentication.

</Include>

<Include type={props.type} ref='xpass2,bew3,bs3'>

### <Cmd sid='device_card_title' product='air' className='normal' /> {#cardSettings}

In the <Cmd sid='device_card_title' product='air' /> section, you can modify card authentication settings.

#### <Cmd sid='device_editable_fields_card' product='air' className='normal' /> {#nfcRfcard}

* <Cmd sid='device_editable_fields_card' product='air' />: Click the toggle switch to enable or disable card access authentication. Select the card types supported for card authentication.

  * <Cmd sid='device_editable_fields_card_em4100' product='air' />

  * <Cmd sid='device_editable_fields_card_mifare_felica' product='air' />

* <Cmd sid='device_editable_fields_card_byte_order' product='air' />

  * <Cmd>MSB</Cmd>: Processes card data in order from large byte units to small byte units, sequentially storing the card serial number (CSN) from left to right.

  * <Cmd>LSB</Cmd>: Processes card data in order from small byte units to large byte units, storing the card serial number (CSN).

* <Cmd sid='device_card_field_data_reading_method' product='air' />: Select CSN (default) or Wiegand format.

   :::note

   Wiegand supports 5 default formats. Add more formats in <Cmd sid='settings_menu' product='air' /> → <Cmd sid='card_format_submenu' product='air' />. For more information about custom card format settings, see <DocLink docId='platform/biostar_air/custom-wiegand-card-formats' />.

   :::

#### <Cmd sid='device_card_wiegand_card' product='air' className='normal' /> {#device_card_wiegand_card}

For Wiegand cards, you can select <Cmd sid='device_card_wiegand_card_iclass' product='air' /> and <Cmd sid='device_card_wiegand_card_hid_prox' product='air' />. Click the toggle to enable the feature, then select the desired card format in <Cmd sid='device_card_wiegand_card' product='air' />.

:::info

* Support may vary by device.

* Wiegand supports 5 default formats. Add more formats in <Cmd sid='settings_menu' product='air' /> → <Cmd sid='card_format_submenu' product='air' />. For more information about custom card format settings, see <DocLink docId='platform/biostar_air/custom-wiegand-card-formats' />.

:::

</Include>

* <Cmd sid="new_layout_device_bew3_block_8_2_text" product="air" />: Select mask detection sensitivity.
