## Start roll call {#mobileAppRollCall}

The roll call feature is available in the BioStar X mobile app.

<Include pages='settings-advanced-ac-roll-call'>

:::info

You can install the BioStar X mobile app via the following links.

* iOS: [Apple App Store](https://apps.apple.com/kr/app/biostar-x-mobile/id6741415253)

* Android: [Google Play](https://play.google.com/store/apps/details?id=com.suprema.biostarmobile)

:::

</Include>

### Start roll call in the mobile app {#useRollCallMobileApp}

To start the roll call feature in the mobile app, follow these steps.

1. Launch the BioStar X mobile app on your mobile device.

2. From the app home screen, tap <Cmd>ROLL CALL</Cmd>.

   <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-01.png' width={253} alone />

   :::note

   You can also access it through the side menu. Tap <IcHamberg title='Hamburger icon' width={20} /> → <Cmd>Monitoring</Cmd> → <Cmd>Roll Call</Cmd>.

   :::

3. In the Roll Call list, tap <IcRCplay title='Start' width={20} height={20} /> for the item you want to execute roll call.

   <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-02.png' width={253} alone />

4. When the confirmation message appears, tap <Cmd>Yes</Cmd>.

Roll call will start, and by tapping on the ongoing roll call item, you can check the status of the roll call.

<Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-03.png' width={595} alone className='none' />

### Ongoing roll call control

* Users entering will be marked with the <IcRCdone title='Done' width={20} height={20} /> icon. When all users have entered, a message indicating that roll call has ended will appear. To end the roll call, tap <Cmd>Yes</Cmd>.

* The administrator can manually change the user's entry or status. Long press the round button on the far right of the user whose status you wish to change from the user list.

  <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-change-status.png' alone className='none' width={595} />

* To reset the roll call, tap <Cmd>Reset</Cmd>. All progress is discarded and the roll call restarts from the beginning.

* If roll call is ended by another administrator, a notification message will appear.

  <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-stopped.png' width={253} alone />

* To end an ongoing roll call, tap the <IcRCstop title='Stop' width={20} height={20} /> button in the roll call list screen, or tap <Cmd>END</Cmd> button in the roll call status screen.

* You can also end the ongoing roll call in **BioStar X**. Go to the <Cmd sid='shell.header.setting' /> → <Cmd sid='setting.menu.advancedAc.root' /> → <Cmd sid='data.sidebar.events' /> → <Cmd sid='zone.rollCall' product='2' /> menu, select the ongoing roll call item, and click the <Cmd sid='zone.pause.end' product='2' /> button at the top right of the screen.

  <Image src='/img/biostarx-settings-adv-ac-rollcoll-ended.png' />

<Include pages='settings-advanced-ac-roll-call'>

* After completing the roll call, you can check related reports from <Cmd sid='shell.header.data' /> → <Cmd sid='data.mainMenu.generate' /> → <Cmd sid='data.sidebar.rollCall' />. For more information, see <DocLink inner='#viewRollCallReport' />.

</Include>

### User information during roll call

During roll call in the mobile app, you can check each user's information. View details such as contact information for emergency contact and the user's last known location.

Swipe left on the user item for which you want to check details during roll call. Then tap <IcRCinfo title='Information' width={20} height={20} />.

<Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-user-info.png' className='none' width={595} alone />

### Leave a note for users during roll call

If a specific user has not entered during roll call in the mobile app, you can leave a note for that user.

Swipe left on the user item where you want to input a note during roll call. Then tap <IcRCmemo title='Memo' width={20} height={20} />. On the note input screen, enter the content and tap <Cmd>Save</Cmd>.

<Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-user-memo.png' className='none' width={595} alone />

Users who have entered notes will be marked with the <IcRCmemo title='Memo' width={20} height={20} /> icon in the roll call list.

<Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-user-status.png' width={253} alone />

### Check roll call details {#viewRollCallinfo}

You can check the details of the roll call settings in the mobile app.

1. Launch the BioStar X mobile app on your mobile device.

2. From the app home screen, tap <Cmd>ROLL CALL</Cmd>.

   <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-01.png' width={253} alone />

   :::note

   You can also access it through the side menu. Tap <IcHamberg title='Hamburger icon' width={20} /> → <Cmd>Monitoring</Cmd> → <Cmd>Roll Call</Cmd>.

   :::

3. Swipe left on the item to check details from the list, then tap <IcRCinfo title='Information' width={20} height={20} />.

   <Image src='/img/common/biostarx-settings-adv-ac-mobile-app-rollcall-info.png' width={595} alone className='none' />
