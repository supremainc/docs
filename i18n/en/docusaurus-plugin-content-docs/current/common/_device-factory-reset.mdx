## Factory default {#factory-reset}

Delete all information stored on the device and initialize the settings.

1. Power on.

2. Quickly press the reset button three times.

   <Include product='xpass_q2'><Image src='/img/device/restoring-factory-defaults-xpq2.svg' className='none' width={200} alone /></Include>
   <Include product='bioentry_w3'><Image src='/img/device/restoring-factory-defaults-bew3.svg' className='none' width={200} alone /></Include>
   <Include product='bioentry_w2'><Image src='/img/device/restoring-factory-defaults-bew2.svg' className='none' width={200} alone /></Include>
   <Include product='xpass_d2'><Image src='/img/device/restoring-factory-defaults-xpd2.svg' className='none' width={200} alone /></Include>
   <Include product='xpass_2'><Image src='/img/device/restoring-factory-defaults-xpd2.svg' className='none' width={200} alone /></Include>
   <Include product='bioentry_p2'><Image src='/img/device/restoring-factory-defaults-bep2.svg' className='none' width={200} alone /></Include>
   <Include product='bioentry_r2'><Image src='/img/device/restoring-factory-defaults-ber2.svg' className='none' width={200} alone /></Include>
   <Include product='xpass_s2'><Image src='/img/device/restoring-factory-defaults-xps2.svg' className='none' width={300} alone /></Include>

3. Press the reset button once more when the device's LED blinks {props.name}.

:::info

You can only use Factory Default when the root certificate is stored on the device.

:::
