入退室管理システムに Suprema バイオメトリクスを C•CURE 9000 プラットフォームへ実装するために Suprema - C•CURE Integration を使用する前に、C•CURE サーバーがインストールされている同じサーバー/ワークステーションに、C•CURE 9000 v2.90、v3.0、v3.10（以降）サーバー、Suprema BioStar 2 サーバー、およびこの Integration Setup サービスをインストールする必要があります。さらに、使用するワークステーションには、C•CURE 9000 v2.90、v3.0、v3.10（以降）クライアントと、この Integration Setup クライアントをインストールする必要があります。

サーバーは、接続されたデバイスからイベントログやユーザー情報などを受信して保存します。 この統合は簡単にインストールできます。 インストール前に、最小システム要件を確認してください。

Suprema - C•CURE Integration のサーバーおよびクライアント用セットアップは、[Suprema Team](https://support.supremainc.com) への依頼により提供されます。

## 略語

- Software House C•CURE 9000（C•CURE）

- Suprema の BioStar 2（BioStar 2）

- Suprema - C•CURE Integration（以下「Integration」）

この統合には、次のコンポーネントが含まれます。

- Suprema - C•CURE Integration Service（Integration Service）

- Suprema - C•CURE Integration Client（Integration Client）

- Suprema - C•CURE Integration Configuration Tool（Integration CT）

  - Integration CT は、Integration Service（GUI なし）を設定し、Integration Client 関連のパラメーターも調整するための Web ベースの GUI ツールです。

- 最適な性能を得るには、64 ビット オペレーティング システムのみを使用してください。

- Integration CT は、MS Edge、Brave Browser、または Google Chrome に最適化されています。

- 旧バージョンの Windows は使用しないでください。前の表で示した旧バージョンと比べて、Microsoft が OS ライブラリを大幅に変更しているため、OS ライブラリの非互換性によりシステムが不安定になる可能性があります。

- MS SQL Server と Suprema BioStar 2 が別の PC にインストールされている場合は、BioStar 2 をインストールした PC に Native Client をインストールしてください。

  > https://www.microsoft.com/en-us/download/details.aspx?id=50402

## 対応デバイス

RFID デバイス:

- XPass 2 ファミリー

- X-Station 2 ファミリー

顔認証デバイス:

- FaceStation F2

- BioStation 3

- BioEntry W3

指紋デバイス:

- BioEntry ファミリー（P2、W2）

- BioLite N2

- BioStation 2, BioStation A2

- X-Station 2（指紋モデル）

- BioStation L2

非対応デバイス:

- BioEntry R2

- CoreStation

- FaceStation 2 / FaceLite

:::info

BioMini Plus 2 / Slim 2 などの USB リーダーは、カスタマイズにより対応できます。 該当する場合は、`latam@supremainc.com` に連絡してください。

:::
