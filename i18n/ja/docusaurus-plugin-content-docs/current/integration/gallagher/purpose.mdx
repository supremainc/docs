  - ミドルウェア

---

Suprema Integration with Gallagher は、Gallagher Command Centre 8.8 以降のシステムが Suprema バイオメトリクスデバイスと通信するためのミドルウェアです。Command Centre からユーザーに対してバイオメトリクス顔面認証または指紋認証の Credential を登録し、接続されたデバイスの設定を管理できます。 Suprema Integration with Gallagher を使用すると、Suprema バイオメトリクスデバイスを使って、Command Centre 用のバイオメトリクス管理システムを簡単にセットアップし、構築できます。

:::info

Gallagher Command Centre アクセス制御システムの機能の詳細については、Command Centre のユーザーマニュアルを参照してください。

:::
