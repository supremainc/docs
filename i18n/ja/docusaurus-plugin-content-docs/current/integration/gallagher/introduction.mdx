## 機能

Gallagher Command Centre と Suprema BioStar 2 の間で行う Suprema と Gallagher の通信。 そのため、Gallagher Command Centre から Suprema デバイス用の生体認証テンプレートを生成でき、すべてのエンドユーザーに先進的なソリューションを提供します。

**Suprema と Gallagher のインテグレーションで提供される機能は次のとおりです：**

- Gallagher Command Centre での一括ユーザー登録と管理が Suprema BioStar 2 に反映されます。

- Suprema インテグレーションにより、ユーザーは各種 Suprema 生体認証デバイスでのスキャンを通じて Visual Face と Fingerprint の生体認証データを登録できます。

- CSN と Wiegand の検出、および BioStar とのインテグレーションによるデータの自動拡充。

- Suprema インテグレーションにより、ユーザーは X-Station 2、BioLite N2、BioStation 3、FaceStation F2、XPass 2、X-Station 2、BioEntry W3、BioEntry W2 などの生体認証デバイスのスキャンを通じてカードデータを登録できます（これらはテスト済みのデバイスです）。

- 最大 1,000 台の生体認証デバイスを管理できます。

- すべての Suprema デバイスの設定は、Command Centre の同じウィンドウ内で行えます。

- TOC カード、QR カード、さらに特定の顧客ニーズに合わせたカスタムカード設定を含む、幅広いカード種別をサポートします。

- 必要な前提条件をすべて含むバンドル EXE パッケージによりインテグレーションのセットアップを簡素化し、スムーズで効率的なインストールを実現します。

- ミドルウェアには、ネットワーク関連の設定に対応する組み込み機能と、設定ミスやフォールトトレランスのシナリオを支援する包括的なトラブルシューティングガイドが含まれます。
