title: BioStar Xの強化
description: 強力な認証、役割ベースのアクセス制御、暗号化、定期バックアップで、BioStar Xシステムと個人データを保護する方法を案内します。
---

BioStar Xを安全に運用するには、2つのセキュリティ層を併せて強化する必要があります。

* **システムセキュリティ**は、デフォルトの管理者アカウントの変更、役割ベースのアクセス制御、強力な認証、デバイスの暗号化通信でアプリケーション自体を保護します。

* **個人データセキュリティ**は、ユーザー情報と生体認証データをデータベースの暗号化と最小保存の原則で保護します。

この2つの領域を両方構成すると、コンプライアンス（GDPR、ISO 27001など）を満たし、データ侵害のリスクを最小限に抑えられます。
