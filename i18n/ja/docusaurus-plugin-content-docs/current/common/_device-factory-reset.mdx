## 工場初期化 {#factory-reset}

デバイスに保存されているすべての情報とルート証明書を削除し、設定を初期化します。

1. 電源を入れます。

2. 初期化ボタンをすばやく3回押してください。

   <Include product='bioentry_w3'><Image src='/img/device/restoring-factory-defaults-bew3.svg' className='none' width={200} alone /></Include>

3. デバイスの LED が {props.name} で点滅したら、初期化ボタンをもう一度押してください。

:::info

デバイスにルート証明書が保存されていない場合、工場初期化を実行できません。

:::
