BioStar X のユーザーフィールドにマッピングする <strong>{props.name}</strong> フィールドを設定できます。 <Cmd sid='settings.activeDirectory.FieldConfiguration' product='2' /> セクションで、BioStar X のユーザーフィールドとして使用する <strong>{props.name}</strong> のフィールドを選択します。

<Image src={`/img/${props.img01}`} />

:::info

BioStar X の各ユーザーフィールドは、<strong>{props.name}</strong> のユーザー情報と一致する項目にマッピングできるようデフォルト設定されています。 デフォルト設定されたフィールド値ではなく別のフィールド値を選択するには、<Cmd sid={props.sid} product='2' /> 列の項目をクリックして、目的のフィールド値を選択します。

<Image src={`/img/${props.img02}`} />

:::
