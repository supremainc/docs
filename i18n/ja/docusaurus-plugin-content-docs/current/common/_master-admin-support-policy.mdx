### マスター管理者対応ポリシー <Badge only>CE 製品 vs グローバル製品（Non-CE）</Badge> {/* #master-admin-support-policy */}

</Xclude>

<Cmd sid="ui_128" product='dev' /> 機能は、<Cmd>CE マークの付いたラベルが貼付された CE 製品のみで有効化</Cmd>されます。

:::note

製品背面のラベルシールで **CE 製品** / **グローバル製品**（Non-CE）を識別でき、これに応じて機能は次のように区別して動作します。

* **CE 製品**：CE マークあり

* **グローバル製品**（Non-CE）：CE マークなし

:::
