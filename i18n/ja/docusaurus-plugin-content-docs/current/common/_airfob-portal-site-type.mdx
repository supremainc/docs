:::note

サイトの種類に応じて、カードタイプを **ダイナミック** または **レギュラー** に選択できます。 

* **ダイナミック**：デバイスおよび使用期間に応じてクレジットを差し引くタイプで、カード発行後も回収、停止、再発行、有効期限の設定などの管理が可能です。 そのため、フィットネスクラブや自習室などの会員制施設、またはシェアオフィスで利用できます。

* **レギュラー**：発行したカード数に応じてクレジットを差し引くタイプで、管理者が **BioStar X** でカードを削除するまで永久に使用できます。 そのため、一般企業や団体で社員証や入館証として利用できます。

:::
