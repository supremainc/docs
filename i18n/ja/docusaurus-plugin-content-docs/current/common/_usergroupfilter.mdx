<Cmd sid='settings.activeDirectory.ADServerConfiguration' product='2' /> セクションで設定を完了し、<Cmd sid='settings.activeDirectory.Connect' product='2' /> ボタンをクリックすると、<Cmd sid='settings.activeDirectory.userGroup' product='2' /> セクションで<strong>{props.name}</strong>からユーザー グループ情報を取得します。 **BioStar X** に同期しないユーザー グループを選択解除します。

<Image src='/img/biostarx-setting-directory-usergroup-filter.png' />

* <Cmd sid='settings.activeDirectory.Refresh' product='2' />：ユーザー グループ情報を再読み込みするにはクリックします。

* <IcSearch /> ボタンをクリックして目的のユーザー グループを検索できます。
