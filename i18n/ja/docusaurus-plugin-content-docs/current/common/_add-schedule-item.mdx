1. 画面左のサイドバーで <Cmd sid='access_menu' product='air' /> → <Cmd sid='schedules_submenu' product='air' /> をクリックします。

2. 画面右上の <Cmd sid='schedules_add_btn' product='air' /> ボタンをクリックします。

   <Image src='/img/air/bsair-access-mng-schedule.png' />

3. <Cmd sid='schedule_add_title' product='air' /> パネルが表示されたら、<Cmd sid='schedule_name' product='air' /> と <Cmd sid='schedule_description' product='air' />(オプション) を入力します。

   <Image src='/img/air/bsair-access-mng-schedule-add.png' />

4. <Cmd sid='schedule_section_title_time_intervals' product='air' /> セクションで曜日と時間を選択します。

   > たとえば、勤務時間を設定するには、月曜日から金曜日まで、午前9時から午後6時まで選択します。 逆に勤務時間外のスケジュールを設定するには、土曜日と日曜日、または毎日午後6時から翌朝午前9時まで選択します。

5. 選択したスケジュールを適用するには、<Cmd sid='add_btn' product='air' /> ボタンをクリックします。

6. <Cmd sid='schedule_section_title_holiday_time' product='air' /> セクションでスケジュールに適用する休日グループを選択し、時間を設定します。

   <Image src='/img/air/bsair-access-mng-schedule-add-holiday-group.png' />

7. 休日グループを適用するには、<Cmd sid='add_btn' product='air' /> ボタンをクリックします。

8. スケジュールを追加するには、<Cmd sid='save_btn' product='air' /> ボタンをクリックします。
