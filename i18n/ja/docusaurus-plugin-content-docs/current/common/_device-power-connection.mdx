<Include product='bioentry_w2'>

* 装置の電源は外部12 VDCのListed Class 2電源装置またはUL294B PoE電源装置を使用する必要があります。

* ULC S319を満たすために「PSC-100A」電源供給装置と「Casil CA 1270」バッテリーを接続する必要があります。

</Include>

<Include product='bioentry_w3,biostation_2a,biostation_3,xstation_2,bioentry_w2,xpass_q2'>

* 電源装置（またはアダプター）とPoE電源装置を同時に接続しないでください。

</Include>

<Include product='corestation_20,corestation_40,doormodule,inputmodule,outputmodule,securemodule'>

* 出入管理装置と製品に接続する電源は必ず分けて使用してください。

</Include>

<Include product='doorinterface'>

* リーダーと製品に接続する電源は必ず分けて使用してください。

</Include>

* 電源アダプターはIEC\/EN 62368-1の認証を受け、端末の消費電力を上回る容量のものを使用してください。 電源アダプターに他の機器も接続する場合は、端末の消費電力と他機器の消費電力を合算した電流容量以上のアダプターを使用する必要があります。

  * 最大消費電流の規格は製品仕様の[電源](product-specifications#specs.power)を参照してください。

<Xclude product='corestation_20,corestation_40,doorinterface,inputmodule,outputmodule,securemodule'>

* Secure I/O 2、電気錠（電気式ロック）、製品は必ず個別の電源を使用してください。 電源を共用すると装置が誤動作する可能性があります。

</Xclude>

<Include product='bioentry_w2'>

* アラーム信号はリレー接点方式でAC OKおよびバッテリー低電圧状態ピンを介して送信されます。

  <div className='overflow-x'>

  |     機能     |         説明        | リレー状態 |
  | :--------: | :---------------: | :---: |
  |   AC電源確認   |       AC電源オン      |   短絡  |
  |      ^     |       AC電源オフ      |   断線  |
  | バッテリー低電圧状態 | バッテリー電圧が11 V以下のとき |   短絡  |
  |      ^     | バッテリー電圧が11 V以上のとき |   断線  |

  </div>

* アラーム信号機能を使用するには外部電源が必要です。 最大印加電圧は30 V、最大シンク電流は1 Aです。

* PoE電源はNational Electrical Code、ANSI\/NFPA 70、Article 725.121のクラス2およびクラス3回路の電源要件に準拠しなければなりません。

* 5eケーブル接続は推奨される最小性能カテゴリです。 性能カテゴリは設置現場で要求される伝送速度と一致しなければなりません。

* PSEまたは電源インジェクターとPDの間を接続できる最小導体ゲージは、パッチコードの場合26 AWG（0.13 mm²）です。 水平またはライザケーブルの場合24 AWG（0.21 mm²）です。

* PoEで電源を供給する場合、UL294BまたはUL294 8th Editionを遵守する必要があります。

* IEEE 802.3（atまたはaf）仕様への準拠は、この要件の一部とは見なされません。

</Include>
