## オンラインで登録する {#registering-online}

インターネットに接続されたオンライン環境で **BioStar X** ライセンスをアクティベートするには、名前と受け取ったライセンスキーを入力し、<Cmd sid='setting.server.license.button.activate' product='2' /> ボタンをクリックします。

<Image src='/img/biostarx-tutorial-license-online.png' />

## オフラインで登録する {#registering-offline}

閉鎖ネットワーク環境やインターネット接続が制限されたオフライン環境で **BioStar X** ライセンスをアクティベートするには、以下の手順に従ってください。

1. <Cmd sid='setting.xlicense.activation' product='2' /> セクションで <Cmd sid='setting.xlicense.offlineActivation' product='2' /> ボタンをクリックします。

   <Image src='/img/biostarx-tutorial-license-offline.png' />

2. <Cmd sid='setting.server.license.title.activateLicenseOffline' product='2' /> ウィンドウが表示されたら、<Cmd sid='setting.xlicense.button.requestOfflineKey' product='2' /> ボタンをクリックします。

   <Image src='/img/biostarx-tutorial-license-offline-popup.png' />

3. ダイアログが表示されたら、<Cmd sid='setting.xlicense.offlineActivation.requestBy' product='2' /> と <Cmd sid='setting.xlicense.offlineActivation.activationKey' product='2' /> を入力します。

   <Image src='/img/biostarx-tutorial-license-offline-popup-filed.png' className='none' />

4. <Cmd sid='setting.server.license.button.download' product='2' /> ボタンをクリックして、ライセンス要求ファイル(*\*.req*)をダウンロードします。

5. 購入先に送信します。

購入先からライセンスファイル(*\*.lic*)を受け取ったら、<Cmd sid='setting.xlicense.offlineActivationTitle' product='2' /> ボタンをクリックしてライセンスファイルをアップロードします。
:::info

ライセンスキーを持っていない場合は、<Cmd sid='setting.xlicense.offlineActivation.requestBy' product='2' /> のみ入力します。

:::
