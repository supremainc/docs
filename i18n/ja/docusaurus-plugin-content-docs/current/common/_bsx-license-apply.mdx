1. **BioStar X** の管理者アカウントでログインしてください。

2. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

3. 画面左のサイドバーで <Cmd sid='setting.menu.license.root' /> → <Cmd sid='setting.menu.license.biostarXLicense' /> をクリックします。

4. <Cmd sid='setting.xlicense.activation' product='2' />で、<Cmd sid='setting.xlicense.activatedBy' product='2' />と<Cmd sid='setting.xlicense.activationKey' product='2' />の項目を順に入力します。

   <Image src='/img/biostarx-tutorial-license-online.png' />

5. <Cmd sid='setting.server.license.button.activate' product='2' /> ボタンをクリックします。

登録したライセンスは <Cmd sid='setting.xlicense.myLicense' product='2' /> で確認できます。

:::info

インターネット接続が制限されているオフライン環境や閉域ネットワーク環境でライセンスを有効化するには、<DocLink docId='platform/biostar_x/settings-license-biostar-x-license#registering-offline' />を参照してください。

:::
