1. 画面左のサイドバーで <Cmd sid='access_menu' product='air' /> → <Cmd sid='access_levels_submenu' product='air' /> をクリックしてください。

2. 画面右上の <Cmd sid='add_access_level' product='air' /> ボタンをクリックします。

   <Image src='/img/air/bsair-access-mng-ac-level.png' />

3. <Cmd sid='add_access_level' product='air' />パネルが表示されたら、<Cmd sid='schedule_name' product='air' />と<Cmd sid='schedule_description' product='air' />(任意)を入力してください。

   <Image src='/img/air/bsair-access-mng-ac-level-add.png' />

4. <Cmd sid='access_level_drawler_title_add_door_group' product='air' /> または <Cmd sid='access_level_drawler_title_individual_door' product='air' /> セクションでドアグループまたは個別のドアを選択します。

5. <Cmd sid='schedules_title' product='air' />セクションで、アクセスレベルに適用するスケジュールを選択してください。 <Cmd sid='always_schedule' product='air' />を選択して、制限のないアクセスを許可します。

6. アクセスレベルを追加するには、<Cmd sid='save_btn' product='air' /> ボタンをクリックします。
