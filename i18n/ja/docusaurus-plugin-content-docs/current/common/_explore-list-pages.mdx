## 一覧ページをナビゲートする

ページを移動したり、1ページに表示する一覧の件数を設定できます。 画面右上のページ移動ツールを使用します。

<Image src='/img/biostarx-settings-access-control-status-pagenation.png' />

* <IcFirst2 />：最初のページに移動します。

* <IcFw2 />：前のページに移動します。

* <IcBw2 />：次のページに移動します。

* <IcLast2 />：最後のページに移動します。

* 目的のページに移動するには、数字入力フィールドにページ番号を入力します。

* 1ページに表示する一覧の件数を設定するには、行選択ボックスをクリックします。
