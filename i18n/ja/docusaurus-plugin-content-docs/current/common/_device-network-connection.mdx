#### LAN接続（PCと直接接続）

<Include product='bioentry_w3,facestation_f2,bioentry_w2,biolite_n2,biostation_2a,biostation_3,facestation_2,xstation_2,biostation_3_max'>

本製品はオートMDI/MDIX機能を備えているため、クロスケーブルまたは標準ストレートタイプのCAT-5ケーブルでPCに直接接続できます。

</Include>

<Include product='xpass_s2'>

本製品はオートMDI/MDIX機能を備えているため、標準タイプのCAT-5ケーブルでPCに直接接続できます。

</Include>

<Include product='bioentry_p2,xpass_2,corestation_40,corestation_20'>

本製品はオートMDI/MDIX機能を備えているため、標準タイプのCAT-5eケーブルでPCに直接接続できます。

</Include>

<Include product='corestation_20'>

<Badge only>CS-20P</Badge> はイーサネットケーブルを介した電源供給に対応するPoE+をサポートします。 PoE+を使用する場合、Lock接続は1つのみサポートされます。

</Include>

<Include product='xpass_q2'>

* 本製品はオートMDI/MDIX機能を備えているため、クロスケーブルまたは標準ストレートタイプのCAT-5eケーブルでPCに直接接続できます。

  :::caution

  一部の非標準ハブ／スイッチでは、オートMDI/MDIX機能がサポートされない場合があります。

  :::

* <Badge only>XPQ2-APB</Badge> はイーサネットケーブルを介した電源供給に対応するPoEをサポートします。

</Include>

* ネットワークケーブルの一部の対や線を別用途に使用しないでください。

:::
