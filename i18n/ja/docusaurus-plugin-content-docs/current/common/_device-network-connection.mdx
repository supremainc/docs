#### LAN接続（PC と直接接続）

<Include product='bioentry_w3,facestation_f2,bioentry_w2,biolite_n2,biolite_n2_clue,biostation_2a,biostation_3,biostation_3_clue,facestation_2,xstation_2,xstation_2_clue'>

#### ネットワークポートおよびサービス

本製品はネットワーク通信および円滑なサービス運用のため、以下のポートを使用します。

<div className='overflow-x col2--20-80'>

| プロトコル | サービス |
| :-: | --- |
| **TCP** | サーバーとデバイス間の通信サービスおよびデバイスの運用状態切り替えサービスに使用されます。 |
| **UDP** | ネットワーク上のデバイス検索サービスに使用されます。 |

</div>

:::info

上記のポートは製品が正常にネットワーク機能を提供するために使用されます。ファイアウォールやネットワークセキュリティ設定の際には、これらのポートの使用許可が必要です。

:::
