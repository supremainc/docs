## 出入口ドア制御

接続された出入口ドアを手動で開けたり、施錠・解錠したりできます。

:::info

* デバイスをマスターとして構成している場合のみ、出入口ドアを制御できます。

* <Cmd sid='ui_300' product='dev' /> または <Cmd sid='ui_194' product='dev' /> で出入口ドアを制御できます。

:::

1. ホーム画面で <Cmd sid='ui_380' product='dev' /> を選択してください。

   <Include product='biostation_3'><Image src='/img/device/bs3-door-control1.png' /></Include>
   <Include product='biostation_3_max'><Image src='/img/device/bs3m-door-control1.png' /></Include>

2. 管理者権限の認証手段で認証してください。

3. 希望する出入口ドアの制御方法を選択してください。

   <Include product='biostation_3'><Image src='/img/device/bs3-door-control2.png' /></Include>
   <Include product='biostation_3_max'><Image src='/img/device/bs3m-door-control2.png' /></Include>

   * <Cmd sid='ui_382' product='dev' />：出入口ドアをすぐに開けるか、開放状態を維持する時間を設定できます。

   * <Cmd sid='ui_383' product='dev' />：出入口ドアをすぐに施錠するか、施錠状態を維持する時間を設定できます。

   * <Cmd sid='ui_384' product='dev' />：手動で開放または解錠を行います。

:::info

手動開放および施錠の保持時間は1～86,400秒に設定できます。

:::
