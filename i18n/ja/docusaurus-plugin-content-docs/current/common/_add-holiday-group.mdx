1. 画面左のサイドバーで <Cmd sid='access_menu' product='air' /> → <Cmd sid='holiday_groups_submenu' product='air' /> をクリックしてください。

2. 画面右上の <Cmd sid='holiday_group_add_btn' product='air' /> ボタンをクリックします。

   <Image src='/img/air/bsair-access-mng-holiday.png' />

3. <Cmd sid='holiday_group_add_title' product='air' /> パネルが表示されたら、グループ名を入力します。

   <Image src='/img/air/bsair-access-mng-holiday-add.png' />

4. <Cmd sid='add_holidays_title' product='air' /> セクションで休日名と日付を指定し、<Cmd sid='add_btn' product='air' /> ボタンをクリックします。

   * 日付が固定された祝日を追加するには、名前と日付を設定し、<Cmd sid='holiday_group_checkbox_annually' product='air' /> オプションを選択します。

   * 毎年繰り返されるが、一定期間を持つ休日の場合は、名前と日付を設定し、<Cmd sid='holiday_group_checkbox_duration' product='air' /> オプションを選択します。

5. 同じ方法で他の休日を追加します。

   <Image src='/img/air/bsair-access-mng-holiday-add-new-day.png' />

6. 休日グループの設定を保存するには、<Cmd sid='save_btn' product='air' /> ボタンをクリックします。
