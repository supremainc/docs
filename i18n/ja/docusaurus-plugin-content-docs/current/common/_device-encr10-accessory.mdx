## アクセサリー

<Include product='corestation_40'>

この製品ではアクセサリとして筐体（ENCR-10）を併せて購入できます。

</Include>

<Include product='inputmodule,outputmodule'>

この製品ではアクセサリとして筐体（ENCR-10）を併せて購入でき、1つの筐体に2台の製品を設置できます。

</Include>

エンクロージャには、電源状態LEDボード、電源分配ボード、電源供給装置、タンパースイッチが含まれています。 製品を筐体に設置する方法は[筐体の設置](installation#install-enclosure)を参照してください。
