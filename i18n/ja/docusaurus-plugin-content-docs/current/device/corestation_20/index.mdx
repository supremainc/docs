CoreStation 20は、エンタープライズレベルのセキュリティのために設計された2ドア用アクセス制御コントローラーです。 最大50万人のユーザーをサポートでき、BioStar Xと統合して勤怠管理機能も提供します。 PoE+対応とSAMスロットによる強化されたセキュリティ機能はCS-20Pモデルのみで提供され、柔軟な設置と安全なスマートカード認証をサポートします。
