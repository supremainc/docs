Door Interfaceは、安定した入退室制御のために設計された高性能な2ドア・4リーダーインターフェースです。 CoreStationシリーズと簡単に連携でき、最小限の配線でRS-485通信による柔軟なドア制御が行えます。 また、オフライン状態でも安定した入退室制御を提供します。
