# Firmware Version 1.5.2 (Build No. 1.5.2_260407)

Release<Separator/>2026-04-10

:::caution

- Devices with firmware v1.5.1 or later cannot be downgraded to versions earlier than v1.5.1.

- The XP2-MAPB-H model cannot be downgraded to versions earlier than v1.4.3.

- XP2-GKDPB models with hardware v1.2.0 or later cannot be downgraded to versions earlier than v1.4.3. You can check the hardware version in the **Device** menu in BioStar 2.

- Devices with the new BLE chip cannot be downgraded to versions earlier than v1.3.1. This restriction does not apply to devices with the existing BLE chip.

To check whether your device can be downgraded and whether the new BLE chip is installed, verify the serial number and contact [Suprema](https://supremainc.com).

:::

## New Features and Improvements

---

1. Supported EV2 Secure Messaging authentication and issuance for DESFire EV2 and EV3 cards.

1. Supports the **Lock Override** feature, which allows you to enroll a card that can open a door to be opened using a specific card.
