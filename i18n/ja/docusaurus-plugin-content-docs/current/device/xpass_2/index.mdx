XPass 2は、Supremaの最新RFID技術とマルチハードウェアプラットフォームを基盤とする屋外用コンパクトRFID端末です。 マルチフォームファクター技術により、ドア枠に設置できるマリオン（Mullion）タイプ、スイッチボックスに設置できるギャングボックス（Gangbox）タイプ、キーパッドタイプのすべてをサポートし、さまざまな設置環境に完全に対応できます。

さらに、Dual-Frequency（125 kHz / 13.56 MHz）RFID技術を採用し、EM、MIFARE、DESFire EV1を含む多様な規格のカードと、最新のモバイル通信技術であるNFCおよびBLEを用いたモバイルカードにも対応します。IP67等級の防水防塵性能とIK08等級のバンダルプルーフ構造を備え、屋外設置に最適化されており、ユーザーが望むあらゆる場所に設置できます。
