## 各部の名称と機能

<Image src='/img/device/name-function-each-part-xp2.png' width='80%' className='none' />

* **LED状態表示灯**：製品の動作状態を表示します。

  * 緑：認証成功

  * 赤：認証失敗

  * 青色／水色が点滅：正常動作中

  * 青/緑が点滅：ネットワーク設定を初期化する時

  * 緑色が点滅：入力待機状態

  * 緑/白が点滅：Suprema AirFob Pass (BLE) アプリケーションと接続中

  * 白が点滅：Device Manager アプリケーションと接続中

* **RFカード・モバイル入退出カードリーダー**：入退出のためにRFカードまたはモバイル入退出カードを読み取る部分です。

* **モバイルアクセスカード読取部**：入退出用モバイルアクセスカードを認識する箇所です。

* **初期化ボタン**

  * ネットワーク設定を初期化します。 詳細については[ネットワーク設定の初期化](installation#reset-network-settings)を参照してください。

  * デバイスに保存されたすべての情報と証明書を削除し、設定を初期化します。 詳細については[工場出荷時リセット](installation#factory-reset)を参照してください。

* **ケーブル**

  電源 / RS-485 / イーサネット / Wiegand 入出力 / 入力 / リレー

* **キーパッド**：<Badge only>XP2-GKDPB</Badge> カードIDを直接入力して認証するときに使用します。 直接入力方式で認証するには、BioStar X で認証方法の設定を変更してください。

  * **0\~9**：数字を入力する際に使用します。

  * <Image src='/img/common/ico-star.svg' ico alone />：Wiegand カード ID を入力する際に FC コードと ID を区別するために使用します。

  * <Image src='/img/common/ico-pound.svg' ico alone />：入力を終了する際に使用します。
