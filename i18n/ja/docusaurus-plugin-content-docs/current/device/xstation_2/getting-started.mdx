## 各部の名称と機能

<Image src='/img/device/name-function-each-part-xs2.png' className='none' width='60%' />

* **カメラ**：ユーザーの写真を登録したり、イメージログを残すことができます。

* **ディスプレイ**：操作用のUIを提供します。

* **RFカード・モバイル入退出カードリーダー**：入退出のためにRFカードまたはモバイル入退出カードを読み取る部分です。

* **QR/バーコード認識部**：<Badge only>XS2-QDPB, XS2-QAPB</Badge> 入退室のためにQR/バーコードを読み取る部分です。

* **指紋認識部**：<Badge only>XS2-ODPB, XS2-OAPB</Badge> 入退室のために指紋を読み取る部分です。

* **ケーブルコネクタ**：ケーブルを接続します。

  電源 / RS-485 / イーサネット + PoE / リレー / Wiegand 入出力 / Input

* **スピーカー**：音を出力します。

* **USB（Type-C）メモリスロット**：USBメモリを接続します。

## QR/バーコードおよびデバイスライセンスに関する注意事項

### 正しいQR/バーコード認証方法 {#how-to-scan-qrbarcode}

ユーザーにQR/バーコードを発行して認証手段として利用できます。 認識率を高めるため、画面の案内に従い正しくQR/バーコードをスキャンします。

<Badge only>XS2-DPB, XS2-APB</Badge>

<Image src='/img/device/qr-scanning-xs2.svg' className='none' alone />

* デバイスのカメラとQR/バーコードの距離を5〜20 cmに保ちます。

* QR/バーコードをデバイスのスキャン画面中央に位置させます。

* モバイルデバイスでダウンロードしたQR/バーコードで認証する場合、画面が見えやすいようにモバイルデバイスの明るさを調整します。

* 画面の指示に従わないと、QR/バーコードの認識に時間がかかるか、失敗する場合があります。

<Badge only>XS2-QDPB, XS2-QAPB</Badge>

<Image src='/img/device/qr-scanning-xs2-q.svg' className='none' alone />

* デバイスのQR/バーコード認識部とQR/バーコードの距離を5〜20 cmに保ちます。

* デバイスのQR/バーコード認識部とスキャンするQR/バーコードが水平になるようにし、ガイドビームがQR/バーコードの中央に位置するようにします。

* モバイルデバイスでダウンロードしたQR/バーコードで認証する場合、画面が見えやすいようにモバイルデバイスの明るさを調整します。

* 画面の指示に従わないと、QR/バーコードの認識に時間がかかるか、失敗する場合があります。

<Badge only>XS2-ODPB, XS2-OAPB</Badge>

<Image src='/img/device/qr-scanning-xs2-finger.svg' className='none' alone />

* デバイスのカメラとQR/バーコードの距離を5〜20 cmに保ちます。

* QR/バーコードをデバイスのスキャン画面中央に位置させます。

* モバイルデバイスでダウンロードしたQR/バーコードで認証する場合、画面が見えやすいようにモバイルデバイスの明るさを調整します。

* 画面の指示に従わないと、QR/バーコードの認識に時間がかかるか、失敗する場合があります。

### デバイスライセンスに関する注意事項

* より多様なQR/バーコード認証を使用するには、別途デバイスライセンスが必要です。 デバイスライセンスを発行するには、購入先にお問い合わせください。

* デバイスライセンスファイルは暗号化されているため、任意に変更できません。

* デバイスライセンスはデバイスIDを基準に発行されます。 不正な方法でデバイスIDを変更した場合、ライセンスの保証サービスは提供されません。
