XPass D2 は Suprema の CoreStation と組み合わせて集中型入退室管理システムに最適化された屋外用コンパクト RFID リーダーです。 Suprema のマルチ RFID 技術と NFC・BLE を用いた特許取得済みの最新モバイルカード技術を採用し、さまざまな周波数の RFID カードとモバイルカードをサポートします。 IP67 等級の防水・防塵設計だけでなく、IK08 等級の耐衝撃設計により高い耐久性を備え、ドアフレームやスイッチボックスなど設置環境に応じて多様なオプションを提供します。
