デバイスで発生したイベントの履歴を確認する方法を説明します。

## イベントログを検索する

条件を設定してログを検索できます。

1. **ESC**キーを押してから、管理者権限の認証手段で認証してください。

2. <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_066' product='dev' /> を選択し、**6** を押します。

3. 変更する項目を選択し、**6** を押して設定を変更します。

   <Image src='/img/device/bln2-eventlog.png' />

4. 検索条件を設定して **OK** を押すと、条件に一致するログが画面に表示されます。

5. 前の画面に戻るには **ESC** を押してください。

## すべてのログを削除

保存されているログをすべて削除できます。

1. **ESC**キーを押してから、管理者権限の認証手段で認証してください。

2. <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_301' product='dev' /> を選択し、**6** を押します。

3. **OK** を押して全ログを削除します。 **ESC** を押してキャンセルします。

## ログの使用量を確認する

ログの使用状況を確認できます。

1. **ESC**キーを押してから、管理者権限の認証手段で認証してください。

2. <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_120' product='dev' /> を選択し、**6** を押します。

3. デバイスのログ使用量を確認します。

4. 前の画面に戻るには **ESC** を押してください。
