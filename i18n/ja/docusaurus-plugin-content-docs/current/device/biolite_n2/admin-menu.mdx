管理者権限でアクセスできる主なメニューと機能を案内します。

## すべてのメニュー

1. **ESC**キーを押してから、管理者権限の認証手段で認証してください。

2. 目的のメニューを選択してください。

   <Image src='/img/device/bln2-menu.png' />

:::info

管理者が指定されていない場合は、**ESC**キーを押すとメニュー画面が表示されます。

:::
