BioEntry W2 は、Suprema の最新の指紋認証技術とハードウェアプラットフォームを基盤とした屋外型指紋認証入退室管理装置です。 同クラス最高の 1.2 GHz クアッドコア CPU を搭載し、超高速の認証性能と迅速な動作速度を実現しています。さらに、Suprema の次世代指紋認証アルゴリズムと偽造指紋検知（Live Finger Detection）技術を採用し、より高いセキュリティ性能を提供します。 また、125 kHz と 13.56 MHz の両周波数に対応した RFID カードを同時にサポートし、EM、HID Prox、MIFARE、iCLASS、DESFire、FeliCa など多様なカードを読み取ることができます。IP67 等級の防水・防塵設計と IK09 等級の耐衝撃設計により、屋外環境への設置も容易です。
