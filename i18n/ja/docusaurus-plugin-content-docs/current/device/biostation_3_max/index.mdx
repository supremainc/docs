BioStation 3 Max は、本人確認中心のセキュリティのための Suprema のフラッグシップ AI ベース生体認証端末です。混雑したロビーや厳格なセキュア入退室環境に最適化した製品で、1 つの7インチタッチスクリーン画面で、高速かつ信頼性の高い多要素認証と幅広い認証情報のサポートを提供する、セキュリティ強化された高認証セキュリティプラットフォームです。
