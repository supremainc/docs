:::note

<Anno id="anno1">1)</Anno> 表示されている最小顔認証距離は製品のデフォルト設定であり、推奨顔認証距離は0.5\~1.3 mです。

<Anno id="anno2">2)</Anno> DESFire EV2/EV3カードはDESFire EV1カードとの下位互換性があるため、DESFire EV1カードの機能を含めてCSNおよびスマートカード機能がサポートされます。

<Anno id="anno3">3)</Anno> RF認識距離は設置環境によって異なる場合があります。

<Anno id="anno4">4)</Anno> 最大32文字のASCIIコード文字列で構成されるQR/バーコードをサポートします。

<Anno id="anno5">5)</Anno> より多様なQR/バーコード認証を利用するには、Camera QRライセンスが必要です。 デバイスライセンスの詳細については、[BioStar X 管理者ガイド](../../platform/biostar_x/settings-license-device-license)を参照してください。

<Anno id="anno6">6)</Anno> クレデンシャルなしで登録されたユーザーを基準としています。

<Anno id="anno7">7)</Anno> PoE+（IEEE 802.3at, 30 W）またはPoE++（IEEE 802.3bt, 60~100 W）の電源が必要です。 標準 PoE (IEEE 802.3af, 15.4 W) には対応していません。 標準 PoE を使用すると、デバイスが不安定に動作したり、永久的に損傷したりする可能性があります。 デバイスを接続する前に、ネットワークスイッチまたは PoE インジェクターが PoE+ または PoE++ に対応していることを必ず確認してください。

:::

:::info

製品仕様は性能向上のため予告なく変更されることがあります。

:::
