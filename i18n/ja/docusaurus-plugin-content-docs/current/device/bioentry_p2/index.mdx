BioEntry P2 は、Suprema の次世代バイオ認証技術とセキュリティプラットフォームに基づくコンパクトな入退室管理端末です。 同クラス最高の 1.0GHz CPU を搭載し、超高速の認証性能と迅速な動作を実現します。また、Suprema の最新指紋認識アルゴリズムを採用し、さらに優れた性能を提供します。 さらに、125 kHz と 13.56 MHz の両周波数を同時に読み取れるマルチ RFID カード技術により、MIFARE、HID iCLASS、DESFire、FeliCa、HID Prox、EM など多様な規格のカードを 1 台で認識できます。 Mullion タイプで設計された BioEntry P2 は、ドア枠などの狭いスペースにも完璧に設置できます。
