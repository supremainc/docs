## ダッシュボード

:::info

Suprema CLUe モードを使用する場合、ユーザーの登録、編集、削除は Suprema CLUe サーバーでのみ可能で、デバイスではできません。

:::

1. <Image src='/img/common/ico-menu.svg' ico alone />を押し、パスワードを入力してメニューに入ります。

   デバイスの使用状況と情報は <Cmd sid='ui_109' product='dev' /> で確認できます。 また、<Cmd sid='ui_174' product='dev' /> を押してすぐにユーザーを登録することもできます。

2. ユーザーを登録するかデバイスを設定するには、ダッシュボード下部で目的のメニューを選択します。

   <Image src='/img/device/bs3-menu.png' />

:::info

* <Cmd sid='ui_387' product='dev' />が設定されていない場合、<Image src='/img/common/ico-menu.svg' ico alone />を押すと<Cmd sid='ui_109' product='dev' />画面が表示されます。 詳しいパスワード設定方法は、[メニューパスワード](settings#menu-password)を参照してください。

* <Cmd sid='ui_142' product='dev' />項目は BS3-APWB モデルでのみ確認できます。

:::
