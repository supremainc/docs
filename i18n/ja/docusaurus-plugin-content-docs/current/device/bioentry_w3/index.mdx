BioEntry W3は、Supremaの先端AI技術を搭載した次世代顔認証入退室管理端末です。 ディープラーニングに最適化されたAIプロセッサ（NPU）を搭載し、高い認証精度と超高速の顔認証速度を誇ります。 砂漠のような過酷な環境や強い外部衝撃にも耐え、スリムかつコンパクトなデザインで優れた設置利便性を提供します。
