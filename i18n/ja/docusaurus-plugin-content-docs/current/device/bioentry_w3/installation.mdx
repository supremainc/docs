インストールに必要な全手順と接続例を提供します。

## ブラケットと製品の固定

1. 製品を取り付ける位置を決めてください。

   <Image src='/img/device/installation-recommended-bew3.svg' className='none' alone />

   <InstallationBracketNotes height={137} />

2. 製品を取り付ける位置にブラケットを固定ネジでしっかり固定してください。

   <Image src='/img/device/installation-device1-bew3.svg' className='none' alone />

   :::info

   * 製品をコンクリート壁に設置する場合は、ドリルで穴を開けてPVCアンカーを挿入し、固定ネジで固定してください。

   * RF信号干渉を回避するため、最小離隔距離を確認して設置してください。

   <Columns>
   <Column className='text--center'>

   <Image src='/img/device/bew3-installation_distance1.png' className='none' width='60%' />

   <div className='overflow-x'>

   |  壁の厚さ  |   距離   |
   | :----: | :----: |
   | 100 mm | 270 mm |
   | 120 mm | 250 mm |
   | 150 mm | 170 mm |

   </div>

   </Column>

   <Column className='text--center'>

   <Image src='/img/device/bew3-installation_distance.png' className='none' width='65%' />

   </Column>
   </Columns>

   * モバイルアクセスカードを使用する場合、BLE信号の干渉を回避するため、デバイス間に1 m以上の距離を空けて設置してください。

   :::

3. 固定したブラケット上に製品を取り付けます。

   <Image src='/img/device/installation-device2-bew3.svg' className='none' alone />

4. 製品固定ネジを締めて、製品とブラケットを組み付けてください。

   <Image src='/img/device/installation-device3-bew3.svg' className='none' alone />

   :::info

   ブラケットと製品を組み立てる際は、セキュリティを向上させるため、製品固定用ネジの代わりに同梱のブラケット固定用ネジ（星形）を使用できます。

   :::

## ファームウェアをアップグレードする

デバイスにUSBメモリを接続してファームウェアをアップグレードできます。

1. USBメモリにアップグレード用ファームウェアファイルを保存してください。

   :::info

   アップグレード時はUSBメモリにファームウェアファイルを1つだけ保存することを推奨します。

   :::

2. デバイスにUSBメモリを接続してください。

3. 「管理者認証が必要です」というメッセージが表示されたら、管理者権限の認証方法で認証してください。

   :::info

   BioStar Xでデバイスに管理者を登録してください。 デバイスに管理者が登録されていない場合、ファームウェアをアップグレードできません。 管理者の登録方法の詳細は [BioStar X 管理者ガイド](../../platform/biostar_x/set-permission) を参照してください。

   :::

4. 管理者認証に成功すると、ファームウェアファイルがデバイスに転送されます。

5. ファームウェアファイルの転送が完了すると、USBメモリを取り外すメッセージが画面に表示されます。 USBメモリを取り外してください。

   :::info

   USBメモリを取り外さない場合、デバイスの再起動後にファームウェアアップグレードが再度実行されます。

   :::

6. ファームウェアアップグレード完了後、デバイスが自動的に再起動します。

:::caution

ファームウェアのアップグレード中に電源が切れないよう注意してください。 故障の原因になります。

:::
