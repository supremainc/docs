## 各部の名称と機能

<Image src='/img/device/name-function-each-part-extm-sc.png' className='none' width='80%' />

* **USB拡張ポート（5ピン）**：デバイスと拡張モジュールを接続します。

* **RS-485（4ピン）**：RS-485ケーブルを接続します。

* **RS-232C（4ピン）**：RS-232Cケーブルを接続します。

## 設置例

:::caution

拡張モジュールの接続は一般ファームウェアではサポートされないため、別途カスタムファームウェアが必要です。

:::

Supremaデバイスに拡張モジュールを取り付けると、RS-485ポートとRS-232Cポートを追加で使用できます。

拡張モジュールを取り付け可能なSupremaデバイスは次のとおりです。

* [**BioStation 3**](../biostation_3/index.mdx)

* [**BioStation 2a**](../biostation_2a/index.mdx)
