ViOnyx Web Server に初めて接続してパスワードを設定し、ログインする方法を説明します。

## カメラのWebサーバーに接続する {#access-web-server}

:::info

* カメラと同じネットワークに接続されたコンピューターで操作してください。

* 工場出荷状態に初期化した後にIPアドレスが割り当てられない場合は、デフォルトIPアドレス（`https://192.168.1.100`）に変更されます。こちらのアドレスを使用して接続してください。

:::

1. [Suprema 営業所](https://www.supremainc.com/jp/support/main.asp)にインストールパッケージ (*VionyxTool.zip*) をリクエストしてください。

{/* 1. [Supremaダウンロードセンター](https://download.supremainc.com/)にアクセスしてログインし、インストールパッケージ(*VionyxTool.zip*)をダウンロードしてください。 */}

1. インストールファイルの圧縮を解凍した後、*vionyx_tool.exe* を実行してください。

{/* 2. ダウンロードしたインストールファイルの圧縮を解凍した後、*vionyx_tool.exe*を実行してください。 */}

   <Image src="/img/camera/web-server-settings1.png" width='80%' alone />

3. **Search for Cameras** をクリックしてください。

4. 検出されたカメラ一覧から接続するカメラのシリアル番号（**DEVICE ID**）を探し、IPアドレス（**IP ADDRESS**）を確認してください。

   <Image src="/img/camera/web-server-settings2.png" width='80%' alone />

5. Web ブラウザを起動します。

6. アドレスバーに`https://`を含むカメラのIPアドレス（**IP ADDRESS**）を入力して、**Enter**キーを押してください。

   > 例: `https://192.168.1.100/`

7. ViOnyx Web Server の画面が表示されます。

## パスワードの設定

初回接続時にパスワードを設定してください。

1. [カメラのWebサーバーへの接続](#access-web-server)を参照して、WebブラウザでViOnyx Web Serverの`https://IPアドレス`にアクセスしてください。

   > 例: `https://192.168.1.100/`

   <ProdImg src="/img/camera/password-setting1-vionyx.png" />

2. <Cmd sid='password' product='cam' /> 入力欄にパスワードを入力してください。

3. <Cmd sid='confirm_password' product='cam' /> 入力欄に同じパスワードをもう一度入力してください。

4. 入力したパスワードが以下の要件をすべて満たしていることを確認してください。

   :::note

   **パスワード要件**

   * **パスワード確認**: <Cmd sid='password' product='cam' /> と <Cmd sid='confirm_password' product='cam' /> 入力欄の値が正確に一致している必要があります。

   * **長さ**: 12文字以上64文字以下で入力してください。

   * **文字の組み合わせ**: 英大文字(A-Z)、英小文字(a-z)、数字(0-9)、記号(`"` `!` `@` `#` `%` `^` `&` `*` `(` `)` `-` `_` `=` `+` `[` `]` `{` `}`)をそれぞれ1文字以上含める必要があります。

     * 空白、ハングル文字、その他のUnicode文字は使用できません。

   * **セキュリティ規則**: セキュリティ上の脆弱性を防ぐため、次のパターンは使用できません。

     * 4文字以上の連続した文字または数字（例：abcd、1234、4321）

     * 同じ文字や数字が4回以上連続している（例：aaaa、1111）

     * ユーザーIDを含む（大文字と小文字は区別されません）

     * 推測されやすい一般的な単語やパターン（例：password、qwerty）

   :::

5. すべての条件が満たされると、<Cmd sid='submit' product='cam' /> ボタンが有効になります。 ボタンをクリックしてパスワードの設定を完了してください。

:::caution

パスワードを紛失しないようご注意ください。 パスワードを紛失した場合は、カメラを初期化する必要があります。

:::

## ログインする

パスワードの設定が完了すると、ログインページへ自動的に移動します。 管理者IDと設定したパスワードを入力して、ViOnyx Web Server に接続してください。

<ProdImg src="/img/camera/password-setting2-vionyx.png" />

1. <Cmd sid='id' product='cam' /> 入力欄に **admin** を入力してください。

   * 管理者ID（**admin**）は <Cmd sid='system' product='cam' /> → <Cmd sid='account_management' product='cam' /> で変更できます。

2. <Cmd sid='password' product='cam' /> 入力欄に設定したパスワードを入力してください。

3. <Cmd sid='sign_in' product='cam' /> ボタンをクリックしてください。 ログインに成功すると、ViOnyx Web Server のカメラのライブビューア画面が表示されます。

4. [ViOnyx Web Server ユーザーガイド](../vionyx_webserver)を参照してカメラを設定してください。
