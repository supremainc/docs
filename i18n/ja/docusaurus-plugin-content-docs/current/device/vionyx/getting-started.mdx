## 各部の名称と機能

### 内部ポートの構成

<Image src='/img/camera/name-function-each-part-vionyx.svg' className='none' />

* **アラーム入力・リレー（5ピン）**：アラーム入力およびリレー出力用のケーブルを接続します。

* **オーディオ入出力（4ピン）**：オーディオ入出力用のケーブルを接続します。

* **イーサネット（PoE+）**：イーサネットケーブルを接続します。 PoE+で給電できます。

* **電源・RS-485（5ピン）**：電源供給およびRS-485通信用のケーブルを接続します。

* **初期化ボタン**：すべての設定を工場出荷時の設定に戻します。 詳細については[工場初期化](installation#factory-reset)を参照してください。

* **microSDカードスロット**：microSDカードを挿入します。

* **USB（Type-A）**：Wi-FiやBLEなど、USB対応のアクセサリを接続します。

* **マイクスイッチ**：マイクを物理的に操作して音声録音を制御します。

### LED ステータス表示灯

カメラカバー前面のLEDステータス表示は、動作状態や認証状態を色や点滅パターンで示します。

<Image src='/img/camera/led-status-indicator-vionyx.svg' className='none' alone />

| LEDの色 | 状態 |
| :-: | --- |
| 緑 | 認証成功 |
| 赤 | 認証失敗 |
| 青 | 正常動作 |
| 青／緑で点滅 | <Cmd sid='ui_012' product='dev' /> を有効に設定すると、IP アドレスを取得できない状態 |

### マイクスイッチの設定

マイクスイッチを操作することで、カメラのマイクを物理的に遮断し、音声録音を防ぐことができます。

<Image src='/img/camera/microphone-switch-vionyx.svg' className='none' alone />

* 音声録音機能は初期設定で有効になっています。

* 音声録音を無効にするには、マイクスイッチを左にスライドしてマイクをオフにしてください。

* Commandアカウントでも音声録音をオフにできます。

:::caution

マイクスイッチが物理的にオフになっている場合、Commandアカウントで設定を変更しても音声は録音されません。

:::
