## 各部の名称と機能

### 内部ポートの構成

<ProdImg src='/img/camera/name-function-each-part-vionyx.svg' className='none' width={700} />

<div className='overflow-x col2-2-8'>
| 名称 | 機能説明 |
| -- | -- |
| **アラーム入力、リレー（5 ピン）** | アラーム入力およびリレー出力用のケーブルを接続します。 |
| **オーディオ入力/出力（4 ピン）** | オーディオ入出力用のケーブルを接続します。 |
| **Ethernet (PoE+)** | イーサネットケーブルを接続します。 PoE+で電源供給が可能です。 |
| **電源、RS-485（5 ピン）** | 電源供給およびRS-485通信用のケーブルを接続します。 |
| **リセットボタン** | すべての設定を工場出荷時の状態に初期化します。 詳細については[工場初期化](maintenance#factory-reset)を参照してください。 |
| **Micro SD カードスロット** | Micro SD カードを挿入します。 |
| **USB (Type-A)** | USB対応のアクセサリを接続します。 |
| **MIC スイッチ** | マイクのスイッチを操作して、音声録音を物理的に制御します。 |
</div>

### LED ステータス表示灯

カメラカバー前面のLEDステータス表示は、動作状態や認証状態を色や点滅パターンで示します。

<Image src='/img/camera/led-status-indicator-vionyx.svg' className='none' width={350} alone />

| LEDの色 | 状態 |
| :-: | --- |
| 緑 | 認証成功 |
| 赤 | 認証失敗 |
| 青 | 正常動作 |
| 青／緑で点滅 | <Cmd sid='ui_012' product='dev' /> を有効に設定すると、IP アドレスを取得できない状態 |

<PageBreak />

### マイクスイッチ設定（対応予定）

マイクスイッチを操作することで、カメラのマイクを物理的に遮断し、音声録音を防ぐことができます。

<Image src='/img/camera/microphone-switch-vionyx.svg' className='none' width={450} alone />

* 音声録音機能はデフォルトで無効になっています。

* 音声録音を使用するには、マイクスイッチを右にスライドして有効にしてください。
