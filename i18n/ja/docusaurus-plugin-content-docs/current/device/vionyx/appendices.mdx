## カメラ設置の注意事項

公共の場所にカメラを設置する際は、次の注意事項を守ってください。

:::caution

関連法令に違反した場合、違反の内容や重大性に応じて過料または罰金が科されることがあります。

:::

**許可された目的のみに設置**

* 犯罪防止、施設の安全確保、火災予防など、指定された目的のみに設置してください。

* 個人のプライバシーを侵害するおそれのある場所（入浴室、トイレ、サウナ、脱衣室など）には設置しないでください。

**案内板の掲示義務**

* カメラの設置目的、撮影場所および範囲、管理責任者の連絡先を含む案内板を、利用者が容易に確認できる場所に掲示してください。

**無断操作および録音の制限**

* 本来の設置目的と異なる目的でカメラの角度を変更したり、無断で操作したりしないでください。 公共の場所では、関連法令により録音機能を使用できません。

**映像情報の無断流出および公開の制限**

* 個人の映像情報を提供する際は、本人確認を行った上で、必要最小限の情報のみを提供してください。 他人の顔が含まれる映像は、モザイク処理などで保護してください。

**運用および管理方針の策定**

* 映像情報の管理責任者を指定し、ホームページなどアクセス可能なチャネルを通じてカメラの運用および管理方針を明示してください。

**映像情報保護のためのセキュリティ対策**

* 許可された管理者のみが映像にアクセスできるように制限してください。 各管理者に個別アカウントを付与し、物理的な施錠などのセキュリティ対策を講じてください。
