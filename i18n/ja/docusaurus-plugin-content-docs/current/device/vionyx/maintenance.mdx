カメラを工場出荷時の状態にリセットし、防湿モジュールを交換する方法を説明します。

## 工場初期化 {/* #factory-reset */}

カメラに保存されているすべての情報を削除し、設定を初期状態に戻します。

1. カメラに電源を接続した後、初期化ボタンの位置を確認してください。

   <Image src='/img/camera/factory-reset-vionyx.svg' className='none' width={350} alone />

2. 初期化ボタンを5秒以上押し続けてください。

3. 工場出荷時の初期化が完了すると、カメラは自動的に再起動します。

:::caution

工場出荷時の初期化を実行すると、ネットワーク設定を含むすべてのユーザーデータおよび設定が削除されます。

:::

:::info

* IPアドレスはデフォルトでDHCPに設定されています。

* 工場出荷状態に初期化した後にIPアドレスが割り当てられない場合は、デフォルトIPアドレス（`https://192.168.1.100`）に変更されます。こちらのアドレスを使用して接続してください。

:::

## 防湿モジュールの交換

レンズの結露を防ぐため、内部に防湿モジュールが取り付けられています。 防湿性能を維持するため、少なくとも1年ごとに防湿モジュールを交換することを推奨します。

1. 防湿モジュールの取り付け位置を確認してください。

2. ネジを反時計回りに緩めて、防湿モジュールをカバーから取り外してください。

   <Image src='/img/camera/change-silicagel-vionyx.svg' className='none' width={684} alone />

3. 新しい防湿モジュールをカバーに合わせ、ネジを時計回りに回して固定してください。

:::caution

交換周期は設置環境によって異なる場合があります。

:::

:::info

* 交換日を記録して、次回交換の時期を管理してください。

* 交換用の防湿モジュールは、製品の購入先でお求めいただけます。

:::
