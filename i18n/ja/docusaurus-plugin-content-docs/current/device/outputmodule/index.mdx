Output Moduleは、12個のリレー出力を使用できるマルチ出力制御モジュールです。 BioStar Xと連携してエレベーターを構成すると、フロアアクセス管理に利用できます。 1台のマスター装置にRS-485チェーンで最大31台まで接続でき、AUX入力も利用できます。 また、アンチパスバック区域や火災警報区域と連携させて多彩に活用できます。
