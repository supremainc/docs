CoreStationは、一般的なACU入退室管理システムの限界を超え、バイオ認証の利点まで提供するコントローラーです。 最大500,000人のユーザーを保存し、1秒あたり400,000回の指紋照合を実行するエンタープライズレベルのハードウェア性能を提供します。 さらに、BioStar Xプラットフォームのすべての入退室管理および勤怠管理機能をサポートします。
