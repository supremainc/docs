配線を完了した後、CoreStationの入力および出力状態をリアルタイムで確認できます。 さらに、接続されたWiegandリーダーでスキャンしたカードIDも確認できます。

:::info

CoreStationモデルによって接続可能なポート数は異なる場合があります。

:::

## 入力ポートの状態

入力およびAux入力ポートの接続状態が表示されます。 配線を完了した後、現場ですぐに入力ポートの接続状態を確認できます。

<Image src='/img/device/monitoring-input.png' alone />

## 出力ポートの状態

リレーおよび出力ポートの接続状態が表示されます。 対象を選択し、スイッチをクリックしてリレーおよび出力ポートを制御することもできます。

<Image src='/img/device/monitoring-output.png' alone />

## Wiegandポートの状態

接続されたWiegandリーダーでカードをスキャンすると、そのカードIDが**Status**列に表示されます。

<Image src='/img/device/monitoring-wiegand.png' alone />
