管理者アカウントと Web サーバーのネットワーク設定方法を案内します。

## 管理者アカウントを編集する

CoreStation Setup Manager の管理者アカウントを編集します。

1. **SETTINGS** をクリックします。

2. **Admin Account** → <IcEdit />**Edit** をクリックします。

   <Image src='/img/device/settings-admin-accounts.png' alone />

3. [デバイスの初期設定](initial-setup) を参照して各項目を編集したあと、**OK** をクリックします。

   <Image src='/img/device/settings-edit-user.png' alone />

4. **Apply** をクリックして設定を保存します。

## Web サーバーネットワーク

CoreStation Setup Manager のネットワーク情報を変更できます。

1. **SETTINGS** をクリックします。

2. 各項目を設定してください。

   <Image src='/img/device/settings-webserver-network.png' alone />

   <div className='overflow-x col2--20-80'>

   | 名称              | 機能説明                                                                                |
   | --------------- | ----------------------------------------------------------------------------------- |
   | **IP Address**  | CoreStation Setup Manager の IP アドレスを編集します。<br/>- 固定 IP のみ使用できます。 DHCP はサポートされていません。 |
   | **Gateway**     | CoreStation Setup Manager のゲートウェイを編集します。                                            |
   | **Subnet Mask** | CoreStation Setup Manager のサブネットマスクを編集します。                                          |

   </div>

3. **Apply** をクリックして設定を保存します。 デバイスが再起動されます。
