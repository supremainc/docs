CoreStation Setup Manager は Web ベースのサービスを提供します。 そのため、CoreStation を用いて集中型アクセス制御システムを構築する際、配線が完了したら、どこからでも Web ブラウザで CoreStation Setup Manager にアクセスし、CoreStation の接続状態を確認できます。

CoreStation Setup Manager を初めて使用する場合は、CoreStation を接続して初期設定を進めてください。

1. CoreStation にケーブルを接続します。

2. Web ブラウザを起動します。

   :::info

   Chrome 75 以降の Web ブラウザを使用します。

   :::

3. Web ブラウザのアドレス入力欄に既定の IP アドレス `https://169.254.0.1:3001` を入力します。

4. ログイン画面で **Download https certification** をクリックします。 CoreStation Setup Manager に初めて接続する場合、HTTPS を正しく使用するために証明書をインストールする必要があります。

   <Image src='/img/device/initial-main.png' width='80%' alone />

5. **cs_client.crt** ファイルを実行します。

6. **ファイルを開く** - **セキュリティの警告** ウィンドウが表示されたら **開く** をクリックします。

7. **証明書** ウィンドウで **証明書のインストール** をクリックします。 **証明書のインポート ウィザード** が起動します。

8. 続行するには **次へ** をクリックします。

9. 証明書ストアを選択し、**次へ** → **完了** → **OK** の順にクリックします。

10. ユーザー ID とログインパスワードを入力します。 初回接続時のユーザー ID とログインパスワードはいずれも "admin" です。

11. 管理者アカウントを登録するには、各項目を設定してから **OK** をクリックします。

    <Image src='/img/device/initial-setup-new-user.png' alone />

    | 名称                   | 機能説明                                                                                            |
    | -------------------- | ----------------------------------------------------------------------------------------------- |
    | **ID**               | ID は自動的に **0** に設定され、変更できません。                                                                   |
    | **Name**             | 管理者名を入力します。<br/>- 名前は最大 48 文字まで入力できます。                                                          |
    | **Operator Level**   | Operator Level は自動的に **Administrator** に設定され、変更できません。                                           |
    | **Login ID**         | ログイン ID を入力します。<br/>- ログイン ID は最大 32 文字まで入力できます。                                                |
    | **Password**         | ログインパスワードを入力します。<br/>- ログインパスワードは 7～32 文字の英数字と記号の組み合わせで入力します。                                   |
    | **Confirm Password** | 確認のため、ログインパスワードを再入力します。                                                                         |

12. ログイン画面に切り替わります。 登録した管理者アカウントでログインします。

:::info

* 管理者アカウントは CoreStation Setup Manager へ初回接続する際にのみ設定でき、1 名のユーザーしか指定できません。 登録が完了すると、初回接続時に使用した既定のユーザー ID とログインパスワードは使用できなくなります。 また、管理者アカウントは工場出荷時リセットを含むすべての設定を確認および編集できます。

* CoreStation Setup Manager バージョン 1.01 から **Level 1** 管理者ランクの名称が **Administrator** に変更され、**Level 2** ～ **Level 3** ランクの管理者アカウントはサポートされなくなりました。

:::
