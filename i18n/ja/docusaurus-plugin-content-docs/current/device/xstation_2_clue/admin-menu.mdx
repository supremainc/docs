管理者権限でアクセスできる主なメニューと機能を案内します。

## 上部アイコン

<Image src='/img/device/xs2-main.png' />

| 名称 | 機能説明 |
| --- | --- |
| <Image src='/img/common/ico-clue-connect.svg' ico alone /> (Connecting to Suprema CLUe) | デバイスが Suprema CLUe に接続中であることを示すアイコン |
| <Image src='/img/common/ico-clue.svg' ico alone /> (Suprema CLUe) | デバイスが Suprema CLUe に接続されていることを示すアイコン |
| <Image src='/img/common/ico-clue-rs485.svg' ico alone /> (RS-485) | RS-485 通信が接続されていることを示すアイコン |
| <Image src='/img/common/ico-clue-tcp-ip.svg' ico alone /> (TCP/IP) | TCP/IP 通信の接続状態を示すアイコン |

## すべてのメニュー

1. <Image src='/img/common/ico-menu2.svg' ico alone />を押した後、パスワードを入力してメニューに入ります。

2. 目的のメニューを選択してください。

   <Image src='/img/device/xs2-menu.png' />

:::info

<Cmd sid='ui_387' product='dev' />が設定されていない場合、<Image src='/img/common/ico-menu2.svg' ico alone />を押すとメニュー画面が表示されます。 詳しいパスワード設定方法は、[メニューパスワード](settings#menu-password)を参照してください。

:::
