管理者アカウントのパスワードを変更して、カメラへのアクセスのセキュリティを強化します。 パスワードは定期的に変更することがセキュリティ上の推奨です。

次の場合にパスワードを変更してください。

* 管理者アカウントを初めて設定する際、セキュリティを強化する場合

* 定期的なセキュリティポリシーに従ってパスワードを更新する場合

* パスワードが漏えいしたため、緊急に変更する必要がある場合

* 管理者交代時に新しいパスワードに設定する場合

## パスワード要件の確認

パスワードを設定する前に、次のルールを確認してください。 すべてのルールを同時に満たす必要があります。

* パスワードは12文字以上にしてください。 最大64文字以内です。

* 少なくとも1つの英小文字(`a-z`)と英大文字(`A-Z`)を含める必要があります。

* 少なくとも1つの数字(`0-9`)と特殊文字を含める必要があります。

  > 特殊文字の例: `\"` `!` `@` `#` `$` `%` `^` `&` `*` `(` `)` `-` `_` `=` `+` `[` `]` `{` `}`

* `1234`、`abcd` のような連続する4文字以上、または同一文字が4回以上含まれていてはいけません。

* 大文字小文字を問わず、IDをパスワードに含めることはできません。

  > admin, Admin, ADMIN, Admin123! など

* `password`、`qwerty`、`123456` など、よく使用されるパスワードパターンは使用できません。

## 管理者パスワードの変更

1. 画面左上の <Cmd sid='system' product='cam' /> メニューをクリックします。

2. 画面左のサイドバーで <Cmd sid='account_management' product='cam' /> をクリックします。

   <ProdImg src='/img/cam-server/cam-webserver-system-account-manage.png' />

3. <Cmd sid='administrator' product='cam' /> セクションで、<Cmd sid='update' product='cam' /> ボタンをクリックします。

4. <Cmd sid='change_password' product='cam' /> ウィンドウが表示されたら、<Cmd sid='current_password' product='cam' /> 入力フィールドに現在の管理者パスワードを入力します。

   <ProdImg src='/img/cam-server/cam-webserver-system-admin-change-pw.png' />

5. <Cmd sid='password' product='cam' /> と <Cmd sid='confirm_password' product='cam' /> 入力フィールドに新しいパスワードを順に入力します。

6. 新しいパスワードに変更するには、<Cmd sid='ok' product='cam' /> ボタンをクリックします。

成功メッセージが表示されたら、パスワードの変更は完了です。

:::info

* パスワード変更後、再ログインが必要になる場合があります。

* パスワード変更の履歴はシステムログに記録されます。

:::

## パスワードを忘れた場合

管理者パスワードを紛失した場合は、次の方法で対処できます。

* **工場出荷時設定へのリセット**: 工場出荷時設定へのリセットを実行すると、管理者アカウントはデフォルト値に復元されます。 ただし、すべてのデータが削除されます。 工場出荷時設定へのリセット方法の詳細は、<Linkto to='docs/device/vionyx/maintenance'>ViOnyx インストールガイド</Linkto>を参照してください。

* **技術サポートへの連絡**: <Support /> に連絡して支援を受けます。
