カメラの各種設定により、撮影環境に適した最適な映像品質を維持し、ネットワークを構成し、セキュリティおよび個人情報保護ポリシーを適用できます。 ビデオプロファイル、カメラ画像、プライバシーとオーバーレイ、日付と時刻、ネットワーク設定を構成し、カメラを効果的に運用してください。
