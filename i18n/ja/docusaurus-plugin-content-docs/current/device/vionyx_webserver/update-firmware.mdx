この機能でカメラのファームウェアを最新バージョンに更新できます。 ファームウェア更新で新機能、性能向上、セキュリティパッチを適用できます。

以下の場合はファームウェアを更新してください。

* 新機能を利用する場合

* セキュリティパッチを適用してカメラを保護する場合

* カメラの性能を改善する場合

* 技術サポートが推奨するファームウェアバージョンがある場合

:::info

* ファームウェア更新中はカメラが再起動し、この間カメラは使用できません。

* ファームウェア更新を開始する前に、機器に十分な電源が供給されていることを確認してください。

* シュプリマ技術サポートが提供する公式のファームウェアファイル(`.bin`)のみを使用してください。

:::

## 現在のファームウェアバージョンの確認

カメラにインストールされている現在のファームウェアバージョンを確認してください。

1. 画面左上の <Cmd sid='system' product='cam' /> メニューをクリックします。

2. 画面左のサイドバーで <Cmd sid='device_info' product='cam' /> をクリックします。

<ProdImg src='/img/cam-server/cam-webserver-system-firmware.png' />

<Cmd sid='firmware_reset' product='cam' /> セクションの <Cmd sid='version' product='cam' /> で、現在インストールされているファームウェアバージョンを確認できます。

## ファームウェアのアップデート

新しいファームウェアファイルを選択して機器を更新してください。

1. 画面左上の <Cmd sid='system' product='cam' /> メニューをクリックします。

2. 画面左のサイドバーで <Cmd sid='device_info' product='cam' /> をクリックします。

3. <Cmd sid='firmware_reset' product='cam' /> セクションで <Cmd sid='update' product='cam' /> ボタンをクリックしてください。

4. <Cmd sid='firmware_update' product='cam' /> ポップアップが表示されたら、シュプリマが提供するファームウェアファイル(`.bin`ファイル)を選択してください。

   <ProdImg src='/img/cam-server/cam-webserver-system-firmware-update.png' />

   * ファイル拡張子は必ず `.bin` にしてください。

   * アップロード領域にファイルをドラッグするか、<Cmd sid='choose_file' product='cam' /> ボタンをクリックしてファイルを選択できます。

5. ファイルを選択すると <Cmd sid='firmware_upload' product='cam' /> ボタンが有効になります。

6. ファームウェア更新を進めるには <Cmd sid='firmware_upload' product='cam' /> ボタンをクリックしてください。

7. アップロードが完了すると再起動確認のポップアップが表示されます。 <Cmd sid='ok' product='cam' /> ボタンをクリックしてカメラを再起動し、ファームウェア更新を完了してください。

:::note

* カメラが再起動した後、Web UI に再度接続してください。 新しいファームウェアバージョンを確認するには、<Cmd sid='device_info' product='cam' /> に移動してください。

* ファームウェア更新が開始された後はキャンセルできません。 更新が完了するまで待ってください。

:::
