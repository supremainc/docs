顔認証設定で、認証効率、セキュリティレベル、偽造検知を一度に管理できます。 組織のセキュリティポリシーと運用環境に合わせて、適切な値に調整します。

1. 画面左上の <Cmd sid='people' product='cam' /> メニューをクリックします.

2. 画面右上の <IcCamFace title='Face Setting' /> <Cmd sid='face_setting' product='cam' /> ボタンをクリックします。

3. <Cmd sid='face_setting' product='cam' /> ウィンドウが表示されたら、必要なオプションを設定します。

   <ProdImg src='/img/cam-server/cam-webserver-people-face-setting.png' />

   * <Cmd sid='face_settings_security_level' product='cam' />: セキュリティレベルを設定できます。 セキュリティレベルが高いほど、認証はより厳格になります。

   * <Cmd sid='face_settings_fake_detection' product='cam' />: 写真などの偽造顔を使ったユーザー認証を防止できます。

4. 設定が完了したら、<Cmd sid='apply' product='cam' /> ボタンをクリックします。

:::tip

* 高いセキュリティが必要なエリアでは、<Cmd sid='face_settings_security_level' product='cam' /> オプションを <Cmd>Most Secure</Cmd> に設定することを推奨します。

* <Cmd sid='face_settings_auth_timeout' product='cam' /> オプションはサポートされていない機能です。

:::
