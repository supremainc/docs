XPass D2 に適用する設定をあらかじめテンプレートとして作成し、個別デバイスに直接適用できます。 PC で管理プログラムに接続して設定を変更したりコマンドカードを使用する方法と比べて、はるかに速く設定を適用できます。

:::info

デバイスのファームウェアバージョンが低い場合、いくつかの設定が適用されないことがあります。 すべての設定を適用するには、[デバイスファームウェアをアップグレードする](#device-firmware-upgrade)を参照してファームウェアをアップグレードしてください。

:::

## テンプレートを追加する {#template-add}

RS-485 通信、カード形式、キーパッド設定、PIN、LED、ブザーなどを設定した後、テンプレートとして追加できます。 テンプレートを追加すると、複数の個別デバイスを毎回設定せずに、迅速かつ簡単に設定を適用できます。

1. モバイル端末で Suprema Device Manager を起動します。

2. メイン画面で XPass D2 を選択します。

3. <Cmd sid='ui_338' product='dev' /> → <Image src='/img/common/ico-add.svg' ico alone /> を押します。

   <Image src='/img/device/xpd2-template-management-add.png' className='none' width={540} />

4. テンプレートを作成するデバイスモデルを選択します。

5. 各項目を設定した後、テンプレートを保存するには <Cmd sid='ui_295' product='dev' /> を押します。

:::info

テンプレートは最大 100 個まで追加できます。

:::

### テンプレート名および管理者パスワードの設定

<Image src='/img/device/xpd2-template-name-pw.png' width={250} />

* <Cmd sid='ui_341' product='dev' />：希望するテンプレート名を入力します。

* <Cmd sid='ui_074' product='dev' />：管理者パスワードを入力します。

:::info

* 管理者パスワードは 6～32 桁の数字で設定できます。

* 管理者パスワードを紛失しないよう注意してください。 管理者パスワードを紛失した場合、テンプレートを適用したデバイスに接続するにはそのデバイスを初期化する必要があります。

* 管理者パスワードの変更について詳しくは、[管理者パスワードを変更する](#admin-password-change)を参照してください。

:::

:::

<Image src='/img/device/xpd2-template-keypad.png' width={250} />

## テンプレートを適用する

追加したテンプレートはBLE通信を利用して複数台のデバイスに同一に適用できます。

:::info

デバイスのファームウェアバージョンが低い場合、いくつかの設定が適用されないことがあります。 すべての設定を適用するには、[デバイスファームウェアをアップグレードする](#device-firmware-upgrade)を参照してファームウェアをアップグレードしてください。

:::

1. モバイル機器でBluetooth機能を有効にした後、Suprema Device Managerを起動してください。

2. メイン画面で XPass D2 を選択します。

3. <Cmd sid='ui_343' product='dev' />を押してください。 選択可能なテンプレートの一覧が表示されます。

   <Image src='/img/device/xpd2-template-choose-select.png' className='none' width={540} />

4. テンプレート一覧から目的のテンプレートを選択してください。

5. <Cmd sid='ui_212' product='dev' />を押してください。

   <Image src='/img/device/xpd2-template-choose-select-start.png' width={250} />

6. テンプレートを適用するデバイスにモバイル機器の背面を完全に接触させてください。

7. テンプレート適用が完了したら、<Cmd sid='ui_367' product='dev' />を押してください。

:::info

* Suprema Device Managerで変更した設定はデバイスにのみ適用され、サーバーには同期されません。

* デバイスがマスターデバイスと接続されているか、Wiegand出力設定が変更されている場合、Suprema Device Managerにデフォルトキーで接続できません。 Suprema Device Managerに接続するにはデバイスを初期化してください。

:::

## テンプレートを管理する

### テンプレートを編集する

1. モバイル端末で Suprema Device Manager を起動します。

2. メイン画面で XPass D2 を選択します。

3. <Cmd sid='ui_338' product='dev' />を押してください。

   <Image src='/img/device/xpd2-template-management-select.png' className='none' width={540} />

4. テンプレート一覧から編集するテンプレートを選択します。

5. 「[テンプレートを追加](#template-add)」を参照してテンプレートを編集します。

6. 変更した設定を保存するには <Cmd sid='ui_295' product='dev' /> を、新しいテンプレートとして保存するには <Cmd sid='ui_178' product='dev' /> を押します。

### テンプレートを削除する

1. モバイル端末で Suprema Device Manager を起動します。

2. メイン画面で XPass D2 を選択します。

3. <Cmd sid='ui_338' product='dev' />を押してください。

   <Image src='/img/device/xpd2-template-management-select.png' className='none' width={540} />

4. テンプレート一覧で削除するテンプレートを選択します。

5. テンプレートを削除するには <Image src='/img/common/ico-delete.svg' ico alone /> → <Cmd sid='ui_367' product='dev' /> を押します。

   <Image src='/img/device/xpd2-template-management-select-del.png' width={250} />

### テンプレートをエクスポートする

XPass D2 設定のテンプレートをアプリからエクスポートできます。

1. モバイル端末で Suprema Device Manager を起動します。

2. メイン画面で XPass D2 を選択します。

3. <Cmd sid='ui_338' product='dev' />を押してください。

   <Image src='/img/device/xpd2-template-management-export1.png' className='none' width={840} />

4. テンプレート一覧画面の上部で <Image src='/img/common/ico-menu1.svg' ico alone /> を押してから <Cmd sid='ui_339' product='dev' /> を選択します。

5. エクスポートするテンプレートを選択した後、<Image src='/img/common/ico-check.svg' ico alone /> を押します。

   <Image src='/img/device/xpd2-template-management-export2.png' width={250} />

:::info

テンプレートは Suprema Device Manager アプリケーションのインストールファイルと同じ場所に作成される *DeviceManager_TemplateFiles* フォルダーに保存されます。

:::

### テンプレートをインポートする

XPass D2 設定のテンプレートをアプリにインポートできます。

1. モバイル端末で Suprema Device Manager を起動します。

2. メイン画面で XPass D2 を選択します。

3. <Cmd sid='ui_338' product='dev' />を押してください。

   <Image src='/img/device/xpd2-template-management-import.png' className='none' width={840} />

4. テンプレート一覧画面の上部で <Image src='/img/common/ico-menu1.svg' ico alone /> を押してから <Cmd sid='ui_337' product='dev' /> を選択します。

5. テンプレートが保存されているフォルダーでインポートするテンプレートを選択します。

:::info

テンプレートは Suprema Device Manager アプリケーションのインストールファイルと同じ場所に作成される *DeviceManager_TemplateFiles* フォルダーに保存されます。

:::

## デバイスの検索と接続 {#device-search-connect}

周辺に設置された XPass D2 を検索して接続できます。 Suprema Device Manager と接続すると、デバイスのファームウェアのアップグレードやデバイスの再起動、テンプレートのパスワード変更など、さまざまな機能を利用できます。

1. モバイル機器でBluetooth機能を有効にした後、Suprema Device Managerを起動してください。

2. メイン画面で XPass D2 を選択します。

3. <Cmd sid='ui_285' product='dev' /> を押します。 接続可能なデバイスの一覧が表示されます。

   <Image src='/img/device/xpd2-template-search-device-select.png' className='none' width={540} />

4. デバイス一覧でデバイス ID を確認してデバイスを選択するか、接続するデバイスにモバイル機器を近づけます。

5. 接続パスワードを入力します。 入力したパスワードを画面に表示するには <Image src='/img/common/ico-password.svg' ico alone /> を押します。

6. デバイスの接続を完了するには <Cmd sid='ui_367' product='dev' /> を押します。

## デバイスのファームウェアをアップグレードする {#device-firmware-upgrade}

デバイスのファームウェアを簡単にアップグレードできます。

:::info

* ファームウェアをアップグレードするには、モバイル機器にファームウェアファイルをダウンロードする必要があります。 最新ファームウェアは[Suprema ダウンロードセンター](https://download.supremainc.com/)からダウンロードできます。

* ファームウェアのアップグレード中は、デバイスとモバイル機器の距離を 1 m 以内に保ってください。

:::

1. 「[デバイスの検索と接続](#device-search-connect)」を参照して、ファームウェアをアップグレードするデバイスと接続します。

2. <Image src='/img/common/ico-option.svg' ico alone /> → <Cmd sid='ui_018' product='dev' /> を押します。

   <Image src='/img/device/xpd2-template-management-fwupgrade.png' width={250} />

3. ファームウェアファイルが保存されているパスでファームウェアを選択します。 ファームウェアのアップグレードが実行されます。

4. ファームウェアのアップグレードを完了するには <Cmd sid='ui_367' product='dev' /> を押します。

## デバイスを再起動する

デバイスを再起動できます。

1. 「[デバイスの検索と接続](#device-search-connect)」を参照して、再起動するデバイスと接続します。

2. <Image src='/img/common/ico-option.svg' ico alone /> → <Cmd sid='ui_287' product='dev' /> を押します。

   <Image src='/img/device/xpd2-template-management-restart.png' width={250} />

3. デバイスを再起動すると、モバイル機器との BLE 通信が切断されます。 自動で再接続するには、<Cmd sid='ui_276' product='dev' /> 機能を有効にします。 詳細な設定方法は「[アプリを設定する](initial-setup#app-settings)」を参照してください。

## 管理者パスワードを変更する {#admin-password-change}

テンプレートの管理者パスワードを変更できます。

1. 「[デバイスの検索と接続](#device-search-connect)」を参照し、パスワードを変更するテンプレートが適用されたデバイスと接続します。

2. <Image src='/img/common/ico-option.svg' ico alone /> → <Cmd sid='ui_156' product='dev' /> を押します。

   <Image src='/img/device/xpd2-template-management-password.png' width={250} />

3. 現在のパスワードと新しいパスワードを入力します。

   <Image src='/img/device/device-manager-change-password.png' width={250} />

4. パスワードの変更を完了するには <Cmd sid='ui_367' product='dev' /> を押します。

:::info

* 管理者パスワードは 6～32 桁の数字で設定できます。

* 管理者パスワードを紛失しないよう注意してください。 管理者パスワードを紛失した場合、テンプレートを適用したデバイスに接続するにはそのデバイスを初期化する必要があります。

:::
