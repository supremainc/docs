# Application Version 1.1.8

Release<Separator/>2023-10-30

:::caution

Device Manager v1.1.8 supports Android 5.0 or higher, iOS 9.0 or higher.

:::

## New Features and Improvements

---

1. Supports **Custom Smart Card Layout**.

   * XPass D2 firmware v1.7.1 or higher.

1. Supports keypad **Backlight** configuration.

   * XPass D2 (XPD2-GKDB) hardware V02M, firmware v1.7.1 or higher.

1. Supports iOS.
