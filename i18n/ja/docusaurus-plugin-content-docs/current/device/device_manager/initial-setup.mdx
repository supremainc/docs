デバイスの選択とアプリ設定の管理方法を案内します。

## デバイスを選択する

Suprema Device Manager を使用して設定するデバイスを選択します。

IP デバイスまたは XPass D2 を選択でき、選択したデバイスによって設定できる項目が異なります。

1. デバイス管理者のモバイル端末に **Suprema Device Manager** アプリケーションをインストールします。 **Suprema Device Manager** は **App Store** と **Play ストア** からダウンロードできます。

2. モバイル端末で Suprema Device Manager を起動します。

3. 設定するデバイスを選択します。

   <Image src='/img/device/device-manager-main.png' width={250} />

## アプリを設定する {#app-settings}

デバイス自動接続機能を設定したり、アプリのバージョン情報を確認できます。

1. モバイル端末で Suprema Device Manager を起動します。

2. メイン画面で <IcSet /> を選択します。

   <Image src='/img/device/device-manager-app-setting.png' className='none' width={540} />

   * <Cmd sid='ui_276' product='dev' />：デバイス自動接続機能を有効化または無効化できます。

   * <Cmd sid='ui_226' product='dev' />：アプリの現在のバージョンとライセンス情報を確認できます。
