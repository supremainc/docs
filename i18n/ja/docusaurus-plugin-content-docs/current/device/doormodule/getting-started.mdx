## 各部の名称と機能

<Magnify src='/img/device/name-function-each-part-dm20.svg' width={800} zoomFactor={.4} mgWidth={500} mgHeight={500} />

:::info

* **INIT** ボタンを 2 秒以上押して、デバイスと連動している製品を初期化し、別のデバイスに接続します。

* **RESET** ボタンを押して、製品を再起動します。

* Supervised Inputを使用する際は、以下のLED状態を参照してください。

  * オレンジ：ショート

  * 赤：オープン

  * 緑：On

  * 消灯：Off

:::

### 設置例

本製品は多様な入出力とWiegand接続をサポートし、Suprema製デバイスおよび他社製デバイスと連携して拡張性を高めることができます。

Supervised Inputも使用でき、ショート、オープン、On、Offの状態を監視できます。
