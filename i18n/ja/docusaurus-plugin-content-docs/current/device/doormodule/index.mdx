Door Module は暗号化通信により最大 2 つのドアを制御し、サードパーティー製リーダーとの連携用に 2 つの Wiegand リーダーインターフェースを備えています。 多彩な I/O インターフェースにより、Door Module をさまざまなシステム構成で活用できます。
