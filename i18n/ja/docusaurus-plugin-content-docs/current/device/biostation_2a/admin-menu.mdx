管理者権限でアクセスできる主なメニューと機能を案内します。

## すべてのメニュー

1. <Image src='/img/common/ico-menu1.svg' ico alone />を押してから、管理者権限の認証手段で認証してください。

2. 目的のメニューを選択してください。

   <Image src='/img/device/bs2a-all-menu.png' />

:::info

管理者が設定されていない場合、<Image src='/img/common/ico-menu1.svg' ico alone />を押すとメニュー画面が表示されます。

:::

## クイックメニュー

1. <Image src='/img/common/ico-menu1.svg' ico alone />を1秒以上長押ししてから指を離します。 管理者権限の認証手段で認証してください。

2. 目的のメニューを選択してください。

   <Image src='/img/device/bs2a-quick-menu.png' />

:::info

管理者が設定されていない場合、<Image src='/img/common/ico-menu1.svg' ico alone />を長押ししてから指を離すとクイックメニュー画面が表示されます。

:::

## キーメニュー

**OK**ボタンを押すとキーメニュー画面が表示されます。 <Cmd sid='ui_334' product='dev' />またはホーム画面で目的のメニューに対応する方向ボタンを押してアクセスできます。

<Image src='/img/device/bs2a-key-menu.png' />

* <IcFw /> <Cmd sid='ui_020' product='dev' />：<Cmd sid='ui_023' product='dev' />で設定したとおりにID認証を実行できます。

* <IcDown/> <Cmd sid='ui_083_title' product='dev' />：勤怠イベントを選択して認証できます。

* <IcBw /> <Cmd sid='ui_279' product='dev' />：ユーザーの作業コードを選択できます。

* <IcUp/> <Cmd sid='ui_069' product='dev' />：警備を開始／解除できます。

:::info

<Cmd sid='ui_069' product='dev' />状態の場合、<Cmd sid='ui_334' product='dev' />にアクセスできません。 ホーム画面で <IcUp/> ボタンを押して警備を解除できます。

:::
