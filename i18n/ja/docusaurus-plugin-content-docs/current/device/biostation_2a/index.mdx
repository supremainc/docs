BioStation 2aはSupremaの一段階進化したセキュリティ技術を搭載した製品です。 世界で初めて指紋認証にディープラーニング技術を導入し、アクセス認証機能を強化してセキュリティ性能をさらに向上させました。 さらに、ディープラーニングで最適な性能を実現するAIプロセッサーのNPU（Neural Processing Unit）を搭載し、指紋認証性能を革新的に高め、優れた精度と速度を提供します。
