デバイスで発生したイベントの履歴を確認する方法を説明します。

## イベントログを検索する

条件を設定してログを検索できます。

1. <Image src='/img/common/ico-menu1.svg' ico alone />を押してから、管理者権限の認証手段で認証してください。

2. <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_066' product='dev' />を選択して **OK**をクリックします。

   <Image src='/img/device/bs2a-eventlog.png' />

3. 希望の項目を選択し、<IcFw />/<IcBw /> ボタンを押して設定を変更します。 **OK**をクリックすると、条件に合ったログが画面に表示されます。

4. 前の画面に戻るには **ESC** を押してください。

## すべてのログを削除

保存されているログをすべて削除できます。

1. <Image src='/img/common/ico-menu1.svg' ico alone />を押してから、管理者権限の認証手段で認証してください。

2. <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_303' product='dev' />を選択して **OK**をクリックします。

   <Image src='/img/device/bs2a-eventlog1.png' />

3. **OK** を押して全ログを削除します。 前の画面に戻るには **ESC** を押してください。

## ログの使用量を確認する

ログの使用状況を確認できます。

1. <Image src='/img/common/ico-menu1.svg' ico alone />を押してから、管理者権限の認証手段で認証してください。

2. <Cmd sid='ui_255' product='dev' /> → <Cmd sid='ui_120' product='dev' />を選択して **OK**をクリックします。 デバイスのログ使用量を確認します。

   <Image src='/img/device/bs2a-eventlog2.png' />
