FaceStation F2 は次世代フュージョン顔認証技術を搭載したマルチモーダル入退室管理端末で、ポストコロナ時代の非接触型セキュリティソリューションの要件に最適化された製品です。 Suprema 独自のフュージョンマッチング技術はビジュアル認識と赤外線（IR）認識を同時に活用し、業界最高レベルの認証精度を実現するとともに、誤認識率（FAR）100億分の1という卓越した性能を誇ります。
