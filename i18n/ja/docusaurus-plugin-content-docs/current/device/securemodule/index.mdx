Secure Moduleは、外部ドアリレー1系統の制御と入出力拡張のための分離型コントローラーです。 SupremaのIP入退室管理端末と暗号化通信を行うことでセキュリティを強化でき、利用環境に最適化されたソリューションを提供します。
