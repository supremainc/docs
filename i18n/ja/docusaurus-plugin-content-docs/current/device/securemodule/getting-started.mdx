:::info

* **INIT** ボタンを 2 秒以上押して、デバイスと連動している製品を初期化し、別のデバイスに接続します。

* **RESET** ボタンを押して、製品を再起動します。

### 設置例

本製品は RS-485 で接続し、コンパクトなためさまざまな場所に設置できます。

ジャンクションボックスと一緒に設置したり、既存のウォールコントロールボックスに取り付けたりできます。 また、退出ボタンの背面にも設置できます。
