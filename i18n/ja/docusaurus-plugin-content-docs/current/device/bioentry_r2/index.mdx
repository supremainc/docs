BioEntry R2はACUシステム向けに設計された小型指紋リーダーです。 最新のSuprema指紋認識アルゴリズムとセンサーにより高解像度の指紋画像を取得でき、125kHzと13.56MHzの周波数を同時に読み取るマルチRFIDカード技術によって、さまざまな規格のカードを1台のデバイスで認識できます。 また、Mullionタイプのデザインを採用しているためドア枠などの狭いスペースにも完璧に設置でき、Supremaのインテリジェント生体認証コントローラーであるCoreStationと組み合わせて、より高度なACUシステムを構築できます。
