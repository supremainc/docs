Input Moduleは、監視入力（12チャネル）、リレー（2チャネル）、AUX入力（2チャネル）、タンパー入力（1チャネル）をサポートし、1台のマスター装置にRS-485チェーンで最大31台まで接続できる多重入力制御モジュールです。 検知センサーや出入口などと連動して使用でき、他のシステムとも無電圧接点（ドライコンタクト）方式で容易に連携できます。
