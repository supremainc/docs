BioStation 3 は、新しい入退場体験を提供する次世代のアクセス制御ソリューションです。 従来比で小型化し、セキュリティ機能もさらに向上したため、小規模オフィスから大規模ビルまで、あらゆる出入口に強力なセキュリティシステムを構築できます。
