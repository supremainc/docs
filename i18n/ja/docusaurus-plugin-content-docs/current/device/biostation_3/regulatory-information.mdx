製品が準拠する規制および認証に関する情報を提供します。

## 規制認証およびライセンス

製品上でライセンス、規制認証、適合マークを確認できます。

* CE / FCC / IC / KC / UKCA / RCM / MIC / TELEC

1. <Image src='/img/common/ico-menu.svg' ico alone />を選択し、管理者権限の認証手段で認証します。

2. <Cmd sid='ui_109' product='dev' /> 右上の <Image src='/img/common/ico-regulatory.svg' ico alone /> をタップすると <Cmd sid='ui_077' product='dev' /> を確認できます。

   <Image src='/img/device/bs3-regulatory-licenses.png' />

:::info

管理者が指定されていない場合、<Image src='/img/common/ico-menu.svg' ico alone />を押すと<Cmd sid='ui_109' product='dev' />画面が表示されます。

:::
