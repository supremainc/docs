管理者権限でアクセスできる主なメニューと機能を案内します。

## ダッシュボード

1. <Image src='/img/common/ico-menu.svg' ico alone />を押した後、管理者権限の認証方法で認証してください。<br/>デバイスの使用状況と情報は<Cmd sid='ui_109' product='dev' />で確認できます。 また、<Cmd sid='ui_174' product='dev' />を押してすぐにユーザーを登録することもできます。

2. ユーザーを登録するかデバイスを設定するには、ダッシュボード下部で目的のメニューを選択します。

   <Image src='/img/device/bs3-menu.png' />

:::info

* 管理者が指定されていない場合、<Image src='/img/common/ico-menu.svg' ico alone />を押すと<Cmd sid='ui_109' product='dev' />画面が表示されます。

* <Cmd sid='ui_142' product='dev' />項目はBS3-APWBモデルでのみ確認できます。

:::
