XPass Q2 は、来訪者やチケットによる出入りが多い環境で、迅速な入退場のために設計されたリーダーです。 QR/バーコードを中心とした設計とマルチクレデンシャル対応により、通行の流れを途切れさせず維持し、コンパクトで耐久性に優れたフォームファクタにより容易に導入できます。
