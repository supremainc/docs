インストールに必要な全手順と接続例を提供します。

## ブラケットと製品の固定

1. 製品を取り付ける位置にブラケットを固定ネジでしっかり固定してください。

   <Image src='/img/device/installation-device1-xpq2.svg' className='none' width='20%' alone />

   :::info

   * 製品をコンクリート壁に設置する場合は、ドリルで穴を開けてPVCアンカーを挿入し、固定ネジで固定してください。

   * RF信号干渉を回避するため、最小離隔距離を確認して設置してください。

   <Columns>
   <Column className='text--center'>

   <Image src='/img/device/installation-wall1-xpq2.png' className='none' width='60%' />

   <div className='overflow-x'>

   |  壁の厚さ  |   距離   |
   | :----: | :----: |
   | 100 mm | 270 mm |
   | 120 mm | 250 mm |
   | 150 mm | 170 mm |

   </div>

   </Column>

   <Column className='text--center'>

   <Image src='/img/device/installation-wall2-xpq2.png' className='none' width='60%' />

   </Column>
   </Columns>

   * モバイルアクセスカードを使用する場合、BLE信号の干渉を回避するため、デバイス間に1 m以上の距離を空けて設置してください。

   :::

   <PageBreak />

2. ケーブルカバーに取り付けられている4本のネジを締めて、しっかり固定します。

   <Image src='/img/device/installation-device2-xpq2.svg' className='none' width='20%' alone />

   :::info

   IP65等級を満たすため、製品とケーブルを接続した後、必ずケーブルカバーを使用してください。

   :::

3. 固定したブラケット上に製品を取り付けます。

   <Image src='/img/device/installation-device3-xpq2.svg' className='none' width='35%' alone />

4. 製品固定ネジを締めて、製品とブラケットを組み付けてください。

   <Image src='/img/device/installation-device4-xpq2.svg' className='none' width='15%' alone />

   :::info

   ブラケットと製品を組み立てる際は、セキュリティを向上させるため、製品固定用ネジの代わりに同梱のブラケット固定用ネジ（星形）を使用できます。

   :::

### ケーブル規格別の最大延長距離

電源接続に使用するケーブルの規格と設置環境によって、接続できる距離が異なる場合があります。 正しく接続しないと装置が誤動作する恐れがあるため、注意してください。 本製品はDC 12V、DC 24V、PoE電源すべてに対応しているため、各ケーブル規格の最大延長長を確認し、正しく電源を接続します。

## ファームウェアをアップグレードする

デバイスにUSBメモリを接続してファームウェアをアップグレードできます。

1. USBメモリにアップグレード用ファームウェアファイルを保存してください。

   :::info

   アップグレード時はUSBメモリにファームウェアファイルを1つだけ保存することを推奨します。

   :::

2. デバイスにUSBメモリを接続してください。

   :::info

   USBメモリを読み込んでいる間は、入退室認証機能が一時的に停止します。

   :::

3. 認証要求を示すLED／ブザーが動作したら、管理者権限の認証手段で認証します。

   :::info

   * BioStar Xでデバイスに管理者を登録してください。 デバイスに管理者が登録されていない場合、ファームウェアをアップグレードできません。 管理者の登録方法の詳細は [BioStar X 管理者ガイド](../../platform/biostar_x/set-permission) を参照してください。

   * 認証の試行はUSBメモリの挿入につき1回のみ可能です。

   * 認証に失敗すると、失敗を知らせるLED／ブザーが動作した後、USB読み取り状態が終了します。 USBメモリを取り外してから、もう一度装置に接続します。

   :::

4. 管理者認証に成功すると、装置はファームウェアアップグレードを実行し、アップグレード中であることをLEDで示します。

   :::info

   ファームウェアファイルがない、または破損している場合、アップグレードは実行されず、失敗状態をLED／ブザーで示します。

   :::

5. ファームウェアアップグレードが完了すると、装置は自動的に再起動します。

:::caution

ファームウェアのアップグレード中に電源が切れないよう注意してください。 故障の原因になります。

:::
