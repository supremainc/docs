ユーザー一覧が多く、目的のユーザーを見つけにくい場合は、検索機能とフィルタリング機能を使ってすばやく探せます。

## ユーザー検索

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-user-list-search.png' alone />

</Column>
<Column className='padding--l40'>

ユーザーのID、名前、メールアドレスを基準に検索できます。

1. アプリのホーム画面で <Cmd>USER</Cmd> をタップします。

   :::note

   サイドメニューからも開けます。 <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>User</Cmd> をタップします。

   :::

2. ユーザー一覧の上部にある <IcSearch title='Search' width='20px' height='20px' /> をタップします。

3. 検索入力フィールドに検索語を入力します。 ID、名前、メールアドレスを組み合わせて検索できます。

4. 検索語の入力を完了したら、 <IcSearch title='Search' width='20px' height='20px' /> をタップします。

検索語と一致するユーザーが検索結果に表示されます。 検索結果から目的のユーザーをタップすると、ユーザー詳細情報に移動できます。

:::info

* 検索入力フィールドの下には、最近の検索履歴が表示されます。 最近の検索履歴をタップすると、その検索語で再度検索できます。

* 検索入力フィールドを閉じるには <IcClose title='Close' width='20px' height='20px' /> をタップします。

:::

## ユーザー一覧のフィルタリング

<Column className='width--23'>

</Column>
<Column className='padding--l40'>

ユーザー一覧をさまざまな条件でフィルタリングして、目的のユーザーをすばやく見つけられます。

1. アプリのホーム画面で <Cmd>USER</Cmd> をタップします。

   :::note

   サイドメニューからも開けます。 <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>User</Cmd> をタップします。

   :::

2. ユーザー一覧の上部にある <IcAppFilter title='Filter' width='20px' height='20px' /> をタップします。

3. <Cmd>User Filter</Cmd> で必要な条件を設定します。

   * <Cmd>Sort By</Cmd>: ユーザーID、名前、グループを基準に、昇順または降順の並べ替えを選択できます。

   * <Cmd>User Group</Cmd>: 特定のユーザーグループを選択して、そのグループに属するユーザーだけを表示できます。

   * <Cmd>Status</Cmd>: 有効なユーザーだけを表示するには <Cmd>Active</Cmd> を選択します。 無効なユーザーだけを表示するには <Cmd>Inactive</Cmd> を選択します。

4. 設定した条件でユーザー一覧をフィルタリングするには <Cmd>Apply</Cmd> をタップします。

:::info

フィルタリング条件を初期化するには、画面右上の <IcAppLogFiltered title='Filtered' width='20px' height='20px' /> → <IcAppReset title='Reset' width='20px' height='20px' /> をタップします。

:::

</Column>
</Columns>
