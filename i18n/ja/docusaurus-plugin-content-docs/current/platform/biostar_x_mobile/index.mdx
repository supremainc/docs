BioStar Xモバイルアプリを使用すると、モバイル端末からBioStar X入退室管理システムを管理できます。 ユーザー登録、ドアのリモート制御、リアルタイムイベント監視、アラーム処理などの操作を実行できます。

![BioStar X Mobile Connection Architecture](/img/common/biostarx-mobile-connection-architecture.png)

## 主な機能

### ユーザー管理

* ユーザー情報の追加、削除、編集

* クレデンシャル管理: PIN、指紋、IR Face、Face、カード（CSN、Wiegand、スマートカード: SCC、AoC）

### 出入口ドア制御

* ドアのロック、ロック解除、開閉をリモートで操作

* アンチパスバック（APB）の解除とドアアラームの初期化

### リアルタイム監視

* 継続的な管理のためにリアルタイムの入退室イベントログを確認

### アラーム管理

* ドアとデバイス、高度な入退室管理アラームの処理

* 新規イベントと確認済みイベントを別々のリストで効率的にアラームを管理

### 移動中の入退室管理

移動中でもBioStar Xモバイルアプリを使用して次の操作を実行できます。

* ユーザー管理とクレデンシャルの追加

* リアルタイムイベント監視

* アラームに迅速に対応

* ドア通知: 開放要求、強制開放、開放保持

* デバイス通知: 改ざん、再起動、RS485接続切断

* ゾーンアラーム: アンチパスバック、火災など

## 個人情報保護

Supremaは、BioStar Xモバイルアプリを通じて、いかなる個人情報も収集、保存、処理しません。 すべてのユーザー関連情報（顔画像やクレデンシャルなど）は、顧客の内部システムに直接送信され、Supremaサーバーには送信されません。
