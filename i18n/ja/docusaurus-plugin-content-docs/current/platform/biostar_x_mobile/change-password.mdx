BioStar Xモバイルアプリでアカウントのパスワードを変更できます。 アカウントのセキュリティを強化するため、パスワードを定期的に変更してください。

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-account-change-pw.png' alone />

</Column>
<Column className='padding--l40'>

1. アプリのホーム画面で、画面左上の <IcHamberg title='Hamburger icon' width={20} height={20} /> をタップします。

2. サイドメニューで <Cmd>Change Password</Cmd> をタップします。

3. <Cmd>Current Password</Cmd> 入力欄に現在のパスワードを入力します。

4. 新しいパスワードを <Cmd>New Password</Cmd>、<Cmd>Confirm password</Cmd> 入力欄に順に入力します。

5. パスワードの変更を完了するには、<Cmd>Change Password</Cmd> をタップします。

</Column>
</Columns>
