ドア一覧が多く、目的のドアを見つけにくい場合は、検索機能とフィルタリング機能で素早く見つけられます。

## ドアの検索

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-door-search.png' alone />

</Column>
<Column className='padding--l40'>

ドアの名前、グループ名、割り当てられたデバイス名で検索できます。

1. アプリのホーム画面で<Cmd>DOOR</Cmd>をタップしてください。

   :::note

   サイドメニューからも開けます。 <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>Door</Cmd>をタップしてください。

   :::

2. ドア一覧の上部にある <IcSearch title='Search' width='20px' height='20px' /> をタップします。

3. 検索入力フィールドに検索語を入力します。 ドア名、グループ名、割り当てられたデバイス名で検索できます。

4. 検索語の入力を完了したら、 <IcSearch title='Search' width='20px' height='20px' /> をタップします。

検索語句に一致するドアが検索結果に表示されます。 検索結果で目的のドアをタップすると、ドアの詳細情報に移動できます。

:::info

* 検索入力フィールドの下には、最近の検索履歴が表示されます。 最近の検索履歴をタップすると、その検索語で再度検索できます。

* 検索入力フィールドを閉じるには <IcClose title='Close' width='20px' height='20px' /> をタップします。

:::

</Column>
</Columns>

## ドア一覧のフィルタリング

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-door-filter.png' alone />

</Column>
<Column className='padding--l40'>

ドア一覧をドアグループでフィルタリングできます。

1. アプリのホーム画面で<Cmd>DOOR</Cmd>をタップしてください。

   :::note

   サイドメニューからも開けます。 <IcHamberg title='Hamburger icon' width={20} height={20} /> → <Cmd>Door</Cmd>をタップしてください。

   :::

2. ドア一覧の上部にある <IcAppFilter title='Filter' width='20px' height='20px' /> をタップします。

3. <Cmd>All Door Groups</Cmd> をタップします。

4. ドアグループ一覧から目的のグループを選択します。

   <Image src='/img/mobile/bsx-mobile-app-door-filter-select-group.png' width='252px' alone />

5. ドアグループの選択が完了したら、<Cmd>Yes</Cmd> をタップします。

6. 選択した条件でドア一覧をフィルタリングするには、<Cmd>Apply</Cmd> をタップします。

選択したドアグループに属するドアが一覧に表示されます。

:::info

フィルタリング条件を初期化するには、画面右上の <IcAppLogFiltered title='Filtered' width='20px' height='20px' /> → <IcAppReset title='Reset' width='20px' height='20px' /> をタップします。

:::

</Column>
</Columns>
