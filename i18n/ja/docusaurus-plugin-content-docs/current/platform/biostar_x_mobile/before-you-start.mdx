BioStar X モバイルアプリを使用する前に、次の項目を確認してからアプリをインストールしてください。

## サポート環境

次の条件で BioStar X モバイルアプリを使用できます。

* **対応システム**: BioStar X

* **サーバー要件**: API アクセスが可能な BioStar X サーバーが必要

* **ユーザー**: サポート対象の権限は、事前定義されたすべてのアカウントレベルです。将来的に、カスタムアカウントレベルのサポートを追加する予定です。

* **モバイルデバイス**: iOS 13.0 以上、または Android 7.0 以上

:::info

BioStar X モバイルアプリは単独では動作できません。社内で運用中の BioStar X サーバーに接続する必要があります。

* BioStar X インストールの最小仕様の詳細については <DocLink docId='platform/biostar_x/system-requirements' target='_blank' /> を参照してください。

* BioStar X のインストールの詳細については <DocLink docId='platform/biostar_x/express-install' target='_blank' /> を参照してください。

:::

## ライセンスとネットワーク要件

BioStar X モバイルアプリを使用するには、BioStar X ライセンスが必要です。 ライセンス追加オプションを購入して利用できます。 ライセンスポリシーの詳細については、<DocLink docId='platform/biostar_x/licensing#features-add-ons' target='_blank' />を参照してください。

| 接続環境 | 必要ライセンス |
| --- | --- |
| 同じネットワーク | モバイルアプリライセンス |
| 外部ネットワーク（公開 IP） | モバイルアプリライセンス |
| 外部ネットワーク（公開 IP なし） | モバイルアプリライセンス + リモートアクセスライセンス |

* BioStar X サーバーとモバイルアプリを連携して使用するには、**モバイルアプリ** ライセンスが必要です。

* 基本的に、同じネットワーク内から BioStar X サーバーに接続してモバイルアプリを使用できます。

* 外部ネットワークから接続するには公開 IP アドレスが必要です。ない場合は、**リモートアクセス** ライセンスを追加購入してください。

:::info

* BioStar X でのライセンス適用方法の詳細については <DocLink docId='platform/biostar_x/settings-license-biostar-x-license' target='_blank' /> を参照してください。

* BioStar X サーバー設定の詳細については <DocLink docId='platform/biostar_x/settings-server-server' target='_blank' /> を参照してください。

* BioStar X のリモートアクセス機能の詳細については <DocLink docId='platform/biostar_x/settings-remote-access' target='_blank' /> を参照してください。

:::
