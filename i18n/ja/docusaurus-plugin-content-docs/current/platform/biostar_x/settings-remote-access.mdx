リモートアクセス（Remote Access）は、ngrok サービスを通じて外部ネットワークから内部ネットワークへ安全に接続できる機能です。 この機能を使用すると、ファイアウォール設定やポートフォワーディングを行わなくても、遠隔地から **BioStar X** サーバーに接続できます。

:::info

* リモートアクセス（Remote Access）機能は、追加オプションで利用できます。 ライセンスポリシーの詳細は[次のドキュメント](licensing#features-add-ons)を参照してください。

* リモートアクセスライセンスを購入すると、Suprema はリモートアクセスライセンスを発行し、リモートアクセス利用用の ngrok Suprema サイトの bot アカウントとエンドポイントを作成して、ライセンス文書をユーザーに提供します。

:::

:::caution

* リモートアクセス（Remote Access）機能は、サードパーティサービスである ngrok のトンネリングサービスを通じて提供されます。 機能を使用する前に必ず [Remote Access 機能使用同意](../../legal/eula#remote-access-agreement) 規約を確認してください。

* BioStar X は単に ngrok API を呼び出して接続をサポートするだけであり、インターネット接続のセキュリティ、安全性、継続性は ngrok サービスに完全に依存します。 したがって、本機能の使用中に発生し得るセキュリティ事故、データ損失、システム侵害などについて、Suprema は一切の責任を負いません。

:::

## いつ使用しますか？

次のような状況でリモートアクセス機能を使用してください。

* 外部ネットワークから社内の **BioStar X** サーバーに接続する必要がある場合

* ファイアウォールや NAT 環境により直接接続が困難な場合

* 複雑なネットワーク設定なしで簡単にリモート接続を構成したい場合

* 一時的に外部接続の権限を提供する必要がある場合

## リモートアクセスの設定

<Steps>
<Step>

### リモートアクセスライセンスの有効化

1. **BioStar X** 管理者 ID でログインします。

2. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

3. 画面左のサイドバーで <Cmd sid='setting.menu.license.root' /> → <Cmd sid='setting.menu.license.biostarXLicense' /> をクリックします。

4. Suprema から提供されたリモートアクセスライセンスキーと管理者名を入力し、<Cmd sid='setting.server.license.button.activate' product='2' /> ボタンをクリックします。

:::info

ライセンス登録の詳細については[次のドキュメント](settings-license-biostar-x-license)を参照してください。

:::

</Step>
<Step>

### リモートアクセスの有効化

1. **BioStar X** 管理者 ID でログインします。

2. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

3. 画面左のサイドバーで <Cmd sid='setting.menu.remoteAccess.root' /> をクリックします。

4. <Cmd sid='setting.remoteAccess.section.title' /> セクションで <Cmd sid='setting.menu.remoteAccess.root' /> オプションを <Cmd sid='common.use' /> に変更します。

<Image src='/img/biostarx-settings-remote-access.png' />

</Step>
<Step>

### ngrok 設定情報の入力

Suprema から提供された ngrok の設定情報を入力します。

* <Cmd sid='setting.remoteAccess.field.ngrokUserId' />：ngrok アカウントを入力します。

* <Cmd sid='setting.remoteAccess.field.tunnelToken' />：**Authtoken** を入力します。

* <Cmd sid='setting.remoteAccess.field.hostURL' />：エンドポイント URL を入力します。 URL は `your-name.bsx.ngrok.app` 形式です。

すべての情報を入力したら、<Cmd sid='button.apply' product='2' /> ボタンをクリックします。 警告メッセージが表示されたら内容を確認し、<Cmd sid='setting.remoteAccess.button.agree' /> ボタンをクリックします。

</Step>

</Steps>

## エンドポイント URL への接続

設定を完了したら、<Cmd sid='setting.remoteAccess.field.hostURL' /> 項目に入力したエンドポイント URL から外部ネットワークで **BioStar X** サーバーに接続できるか確認します。

:::info

外部ネットワークからエンドポイント URL で接続できない場合は、ライセンスを発行した販売店または代理店にお問い合わせください。

:::
