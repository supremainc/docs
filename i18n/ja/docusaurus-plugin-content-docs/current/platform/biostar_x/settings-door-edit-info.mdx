登録済みのドアの設定情報を編集する方法を説明します。 個別のドアの詳細設定を変更するか、複数のドアを選択して共通項目を一括編集できます。

## 1つのドア情報を編集 {#edit-one-door-info}

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.door.root' /> をクリックします。

3. ドアリストで編集するドアをクリックします。

4. 各セクションの詳細項目を編集します。

5. すべての設定が完了したら、画面最下部の <Cmd sid='button.apply' product='2' /> ボタンをクリックします。

:::info

各セクションの詳細については[次のドキュメント](settings-door-add)を参照してください。

:::

## 複数のドアを一括編集 {#edit-multiple-door-info}

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.door.root' /> をクリックします。

3. ドアリストで編集するドアのチェックボックスをクリックします。 2つ以上のドアを選択します。

4. 画面右上にある<Cmd sid='button.batchEdit' product='2' />をクリックします。

5. <Cmd sid='button.batchEdit' product='2' />ウィンドウが表示されたら、必要なオプションを設定します。

   <Image src='/img/biostarx-setting-door-edit-batch.png' />

   * <Cmd sid='door.openTime' product='2' />：認証完了後にドアが開いたままになる時間を設定します。 この時間を過ぎると、ドアは自動的にロックされます。

6. 設定が完了したら<Cmd sid='button.apply' product='2' />をクリックします。

:::info

<Cmd sid='button.batchEdit' product='2' />は、ドアリストで2つ以上のドアを選択すると有効になります。

:::
