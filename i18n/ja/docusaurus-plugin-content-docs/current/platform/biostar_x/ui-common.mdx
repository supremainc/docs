**BioStar X** のユーザーインターフェース（User Interface、以下 UI）は、ユーザーがシステムを簡単に操作・利用できるように設計されています。 本ドキュメントでは、**BioStar X** の共通 UI 構成要素について説明します。 このドキュメントを通じて **BioStar X** の UI を理解し、システムをより効率的に利用できます。

画面のヘッダー領域はすべてのページで共通で、**BioStar X** のさまざまな機能へアクセスできるリンクボタンを提供します。

<Image src='/img/biostarx-header.png' />

<Calloutlists>

1. **BioStar X ロゴ**：ユーザーが **BioStar X** の <Cmd sid='shell.header.launcher' /> ページに戻るためのリンクボタンです。

2. **ショートカット**：**BioStar X** が提供する主要ページへ移動できるリンクを提供します。

3. <Cmd sid='shell.header.action' />：サーバーユーザーが設定した **アクション**（Action）機能を実行できます。 ボタンをクリックするとアクション一覧が表示されます。 詳細については[次のドキュメント](settings-custom-interface)を参照してください。

4. <IcSvTime width='26' height='26' /> **サーバー時刻**：稼働中の **BioStar X** サーバーの時刻を確認できます。 ボタンにマウスを乗せると現在のサーバー時刻が表示されます。

5. <IcInfo width='26' height='26' /> **情報**：インストールされている **BioStar X** のバージョン情報を確認できます。

6. <IcHelp width='26' height='26' /> **ヘルプ**：**BioStar X** のヘルプページへ移動するリンクボタンです。

7. <IcLogout width='26' height='26' /> **ログアウト**：現在ログインしているユーザーのセッションを終了してログアウトできます。

</Calloutlists>
