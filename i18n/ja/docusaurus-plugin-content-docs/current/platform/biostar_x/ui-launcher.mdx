<Cmd sid='shell.header.launcher' /> ページは、ログイン後に最初に表示される **BioStar X** のスタートページであり、ユーザーが **BioStar X** の主要機能にアクセスするための中心的なハブとして機能します。 このページは、ユーザーが **BioStar X** のすべての機能を簡単に探索して利用できるように設計されています。 ランチャーページの UI 構成要素は次のとおりです。

<Image src='/img/biostarx-launcher.png' className='none' />

<Cmd sid='shell.header.launcher' /> ページでは、<Cmd sid='shell.header.user' />, <Cmd sid='shell.header.arena' />, <Cmd sid='shell.header.data' />, <Cmd sid='shell.header.dashboard' />, <Cmd sid='shell.header.setting' /> ページへのリンクが提供されます。

* <Cmd sid='shell.header.user' />：ユーザーグループ管理、ユーザー管理、ユーザーテンプレート管理、入退室権限別ユーザー確認など、さまざまな機能を通じてユーザーを効率的に管理し、権限によってセキュリティを強化できます。 詳細については、[次のドキュメント](how-to-guide-user) を参照してください。

* <Cmd sid='shell.header.arena' />：ドア、デバイス、ビデオ、イベントを多様な方法で監視します。 関連機能を制御し、リアルタイム映像で監視できます。 詳細については、[次のドキュメント](how-to-guide-monitoring) を参照してください。

* <Cmd sid='shell.header.data' />：ユーザー情報や希望の条件のイベントを照会でき、レポートを一定の周期で生成できます。 詳細は[次のドキュメント](how-to-guide-data)を参照してください。

* <Cmd sid='shell.header.dashboard' />：**BioStar X** が提供する多様なデータを利用して、ユーザーが選択したデータに基づくウィジェットを追加・配置し、希望のダッシュボードを構築できます。 詳細については、[次のドキュメント](how-to-guide-dashboard) を参照してください。

* <Cmd sid='shell.header.setting' />：デバイスおよび入退室制御、ユーザーごとの権限、言語、時刻などの多様な設定を通じて、ユーザー環境に最適化された **BioStar X** を構成できます。 詳細は [次のドキュメント](how-to-guide-settings) を参照してください。

:::info
