title: レポート保存パスを設定する
description: 自動生成スケジュールを設定した場合は、レポート保存パスを設定します。
keywords:
  - レポート保存
  - 自動生成
---

<Cmd sid='reportApp.mainMenu.schedule' product='2' />を設定した場合は、レポート保存パスを設定します。

1. <Cmd sid='reportApp.mainMenu.settings' product='2' />をクリックします。

2. レポート保存パスを入力します。

   <Image src='/img/biostarx-data-settings.png' className='none' />

   * <Cmd sid='reportApp.exportReportPath' product='2' />：指定したパスにレポートが保存されます。パスを入力しない場合、ユーザー PC の *Documents\\BioStarX* フォルダーに自動保存されます。

3. 設定を保存するには、<Cmd sid="reportApp.apply" product='2' />をクリックします。
