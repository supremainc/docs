<Cmd sid='shell.header.visitor' /> 機能を使用すると、訪問申請用PCで訪問申請ページを作成し、訪問者管理用PCで訪問者の入退室を管理できます。

:::info

* <Cmd sid='shell.header.visitor' /> 機能は、**アドバンス**(Advanced) ライセンス以上で追加オプションとして利用できます。 ライセンスポリシーの詳細は[次のドキュメント](licensing#features-add-ons)を参照してください。

* 訪問者機能をライセンスで有効化した後、設定方法の詳細は[次のドキュメント](settings-visitor)を参照してください。

:::

## 訪問者ライセンスの適用

訪問者ライセンスの適用方法を案内します。 **BioStar X** の購入先から <Cmd sid='shell.header.visitor' className='normal' /> ライセンスを購入した後、次の手順に従ってライセンスを適用してください。

import LicenseApply from '../../common/_bsx-license-apply.mdx'

<LicenseApply />

:::tip

訪問者ライセンスを適用すると、<Cmd sid='shell.header.setting' /> → <Cmd sid="common.visitor" product='2' /> メニューが有効になり、画面左上のショートカットリストに <Cmd sid='shell.header.visitor' /> 管理ページへ移動できるリンクが追加されます。

:::

## 訪問申請ページの設定

訪問申請ページは、先に訪問者設定を行った後に使用できます。 訪問申請ページの設定方法の詳細は[次のドキュメント](settings-visitor)を参照してください。 

訪問申請ページには専用の訪問申請用PCからアクセスでき、訪問者が入退室を申請できます。 訪問申請の詳細は[次のドキュメント](visitor-register)を参照してください。

<Image src='/img/biostarx-settings-visitor-welcome-screen.png' />

## 訪問者入退室管理

訪問申請履歴を確認し、訪問者の入退室を管理できます。 訪問者入退室管理の詳細は[次のドキュメント](visitor-register-manage)を参照してください。
