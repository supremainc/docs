入退室管理のために **BioStar X** サーバーに登録されたユーザー情報をデバイスから削除する機能を提供します。 各デバイスでユーザー情報を削除できます。

1. <Cmd sid='shell.header.launcher' /> ページで <Cmd sid='shell.header.user' /> をクリックします。

2. ユーザーリストで、デバイスから削除するユーザーの一番左にあるチェックボックスをクリックします。

3. 画面右上の <IcMoreW/> ボタンを選択し、<Cmd sid='user.header.removeFromDevice' product='2' /> ボタンをクリックします。

4. <Cmd sid='user.header.removeFromDevice' product='2' /> ウィンドウが表示されたら、選択したユーザー情報を削除するデバイスを選択します。

5. <Cmd sid='common.delete' /> ボタンをクリックします。

:::info

* 検索アイコンボタンをクリックしてデバイスを検索します。

* <Cmd sid='setting.server.automaticUserSync' product='2' /> オプションを <Cmd sid='device.Input.unsupervised' product='2' /> に設定すると、<Cmd sid='user.header.removeFromDevice' product='2' /> ボタンが有効になります。

* ユーザー情報はデバイスからのみ削除され、**BioStar X** サーバーには保持されます。

:::
