**BioStar X** のビデオ設定機能により、入退室管理と映像セキュリティを統合管理できます。 **Video Management System**（VMS）と連携し、リアルタイム映像モニタリング、イベント連動録画、自動ブックマーク生成などの強力なセキュリティ管理機能を利用します。

:::info

ビデオ設定機能は **アドバンス**（Advanced）ライセンス以上で追加オプションとして利用できます。 ライセンスポリシーの詳細は[次のドキュメント](licensing#features-add-ons)を参照してください。

:::

## ビデオ設定の概要

ビデオ設定は次の 3 段階で構成されます。

<Steps>
<Step>

### VMS サーバーを連携する

VMS サーバーと **BioStar X** の接続を設定し、証明書を構成します。 統合ビデオセキュリティ管理システムを構築するための必須ステップです。 詳細は [以下](settings-video-integration) を参照してください。

</Step>
<Step>

### カメラを設定する

VMS に接続されているカメラの中から必要なカメラを選択し、効率的な管理のためにグループを構成します。 詳細は [以下](settings-videos-camera) を参照してください。

</Step>
<Step>

### ルールを設定する

入退室イベントと映像を連携するルールを作成し、自動ブックマークやイベント記録などの高度な機能を活用します。 詳細は [以下](settings-videos-rule) を参照してください。

</Step>

</Steps>

## 主な機能

* **統合モニタリング**：入退室管理と映像セキュリティを単一インターフェースで管理できます。

* **自動イベント記録**：入退室イベント発生時に関連映像を自動で記録し、タグを生成します。

* **効率的な検索**：ブックマークとイベントタグによる素早い映像検索と証拠収集が可能です。

* **リアルタイム対応**：セキュリティ事象発生時に関連カメラ映像を即時に確認できます。
