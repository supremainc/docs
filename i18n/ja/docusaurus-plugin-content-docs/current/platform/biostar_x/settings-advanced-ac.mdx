高度な入退室制御設定で入退室を制御します。 セキュリティ要件と運用環境に応じて、さまざまな高度な入退室制御機能を設定します。 アンチパスバックで不正な再入室を防止し、火災や侵入などの緊急事態に対応し、スケジュールに基づいてドアを自動制御し、特定エリアの人数を制限してモニタリングします。
