デバイス、ドア、ゾーンでイベントが発生した場合に画面に表示するアラームの種類とメッセージを設定します。 また、アラームが発生したときにアップロードしたサウンドファイルを再生するように設定します。

## アラームを設定する

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左のサイドバーで<Cmd sid='setting.menu.event.root' /> → <Cmd sid='setting.menu.event.alert' />をクリックします。

3. 各カテゴリーのイベント一覧で画面に表示するイベントを選択します。

4. すべての設定が完了したら、<Cmd sid='button.apply' product='2' /> ボタンをクリックしてください。

<Image src='/img/biostarx-settings-alert.png' />

## アラームメッセージを編集する {#edit-alert-message}

イベント一覧で<IcEdit2 />ボタンをクリックしてアラームメッセージを編集します。 <Cmd sid='setting.alert.alertMessage' product='2' />ウィンドウが表示されたら各項目を設定し、<Cmd sid='button.apply' product='2' />をクリックします。

<Image src='/img/biostarx-settings-alert-edit-message.png' />

* <Cmd sid='common.name' product='2' />：アラームメッセージの名前を変更します。

* <Cmd sid='common.message' product='2' />：画面に表示するメッセージを入力します。

* <Cmd sid='setting.alert.config.soundFile' product='2' />：イベント発生時に再生するサウンドファイルをアップロードしている場合は、一覧から選択します。

* <Cmd sid='setting.alert.config.playOption' product='2' />：サウンドファイルを選択した場合は再生回数を選択します。

:::info

再生するサウンドファイルがない場合は[次のドキュメント](settings-preferences#sound)を参照してアップロードします。

:::
