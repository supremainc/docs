個人情報の保存期間が過ぎた訪問者情報を削除します。

1. <Cmd sid='shell.header.launcher' /> ページで <Cmd sid='shell.header.visitor' /> をクリックします。

2. 画面上部の <Cmd sid='visitor.expired.personalData' product='2' /> タブをクリックします。

3. 個人情報の保存期間を超過した訪問者リストが表示されたら、削除する訪問者のチェックボックスをクリックします。 1人以上選択します。

   <Image src='/img/biostarx-visitor-expire-personal-data-lists.png' />

4. 訪問者リストの右上にある <Cmd sid='visitor.button.deletePersonalData' product='2' /> をクリックします。

5. 確認メッセージが表示されたら、<Cmd sid='button.yes' product='2' /> ボタンをクリックします。

:::info

* 個人情報の期限切れ訪問者一覧を確認できるのは、<Cmd sid='setting.account.administrator' product='2' className='bold' /> 権限を持つユーザーのみです。 ユーザー権限の詳細については[以下](add-user)を参照してください。

* 個人情報の保存期間設定の詳細は[以下](settings-visitor#settings-visitor-terms)を参照してください。

:::
