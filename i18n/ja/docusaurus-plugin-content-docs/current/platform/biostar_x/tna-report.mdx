システムで収集したユーザーの勤怠イベントから勤怠レポートを生成でき、タイムレコードを編集したり、CSV ファイルまたは PDF ファイルにエクスポートしたりできます。

あらかじめ設定されている 7 つのレポートフィルターを利用して簡単に使うことができ、管理者がフィルターを直接設定することもできます。

## 開始する前に

### 多言語レポートを使用する前に

**BioStar X** では韓国語と英語を基本サポートしています。 多言語レポートを使用するには、次の案内に従って希望の言語を設定してください。

1. 次のパスに移動します。 *C:\Program Files\BioStar X\plugin\ta\dist\setup\report_fonts*

2. 使用する言語名でフォルダーを作成します。 言語名は ISO 639-1 規格を参照してください。 例として、スペイン語を使用する場合はフォルダー名を "es" にします。

3. 作成したフォルダーにフォントをコピーして貼り付けます。 TrueType フォントは 1 つのみサポートされます。

### レポートを更新する前に

**BioStar X** は標準データベースとして MariaDB を使用します。 MS SQL Server データベースを使用する場合は、以下の事項を先に確認してください。

**BioStar X** を MS SQL Server データベースと併用すると、登録ユーザー数が多い場合、レポートを更新するたびに PC のメモリ使用量が蓄積されることがあります。 MS SQL Server データベースの最大サーバーメモリを再設定してください。

1. **Microsoft SQL Server Management Studio** を実行します。

2. オブジェクトエクスプローラーで **BioStar X** のデータベースを右クリックします。

3. ポップアップメニューで **プロパティ** をクリックします。

4. **メモリ** をクリックし、**最大サーバーメモリ** 項目の値を下げます。

:::info

**BioStar X** のインストール時の MariaDB および MS SQL Server の設定の詳細は[次のドキュメント](custom-install)を参照してください。

:::

## 勤怠記録の確認

### 勤怠レポートを作成する

登録されたユーザーの勤怠記録レポートを生成する方法を説明します。

1. <Cmd sid='shell.header.launcher' /> ページで <Cmd sid='shell.header.tna' /> をクリックします。

2. 画面左のサイドバーで <Cmd sid='ta.report.title' product='2' /> タブをクリックします。

   <Image src='/img/biostarx-tna-report.png' />

3. 画面左のサイドバーで目的のレポートフィルター項目をクリックします。

4. <Cmd sid='common.userGroup' product='2' /> または <Cmd sid='common.user' product='2' /> 項目で <IcSearch /> ボタンをクリックし、グループまたはユーザーを指定します。

5. レポートを生成するには <Cmd sid='button.updateReport' product='2' /> ボタンをクリックします。

6. 画面下部にレポート一覧が表示されたら、目的の項目をクリックして詳細を確認できます。

<Image src='/img/biostarx-tna-report-update-list.png' />

### フィルター条件の設定

フィルター条件を設定して新しい勤怠記録レポートを生成できます。

<Image src='/img/biostarx-tna-report-filter-setting.png' />

* <Cmd sid='common.name' product='2' />：レポート名を入力します。

* <Cmd sid='ta.report.condition.reportType' product='2' />：希望するレポート種類を選択します。 提供されているレポート種類は次のとおりです。

  * <Cmd sid='report.daily' product='2' />, <Cmd sid='report.dailySummary' product='2' />, <Cmd sid='report.individual' product='2' />, <Cmd sid='report.individualSummary' product='2' />, <Cmd sid='report.leave' product='2' />, <Cmd sid='report.exception' product='2' />, <Cmd sid='report.modifiedPunchLogHistory' product='2' />, <Cmd sid='report.workweek' product='2' />

  :::info

  <Cmd sid='ta.report.condition.reportType' product='2' /> を <Cmd sid='report.individual' product='2' /> に選択すると、ユーザーの入退室記録の出力有無を設定できます。

  <Image src='/img/biostarx-tna-report-individual-types.png' />

  * <Cmd sid='report.individual.optionSummary' product='2' />：レポートにはユーザーの出勤および退勤記録のみを出力します。

  * <Cmd sid='report.individual.optionDetail' product='2' />：レポートにはユーザーのすべての入退室記録を出力します。

  :::

* <Cmd sid='grid.columnSetting' product='2' />：レポートテーブルの列順を変更したり非表示にしたりできます。

  <Image src='/img/biostarx-tna-report-filter-setting-column.png' />

  :::info

  列順を変更した後に既定設定へ戻すには <Cmd sid='columnSetting.defaultColumn' product='2' /> ボタンをクリックします。

  :::

* <Cmd sid='report.filter' product='2' />：<Cmd sid='ta.report.condition.reportType' product='2' /> を <Cmd sid='report.leave' product='2' /> または <Cmd sid='report.exception' product='2' /> に設定した場合のみ有効になり、不在記録または例外記録の詳細条件を選択できます。

* <Cmd sid='common.userGroup' product='2' /> / <Cmd sid='common.user' product='2' />：レポートを生成するユーザーグループまたはユーザーを選択します。

* <Cmd sid='ta.report.condition.saveFilter.button' product='2' />：設定した勤怠レポートをフィルターとして保存できます。

  <Image src='/img/biostarx-tna-report-filter-saving.png' />

### レポート期間の設定

レポート期間の設定方法を説明します。

<Image src='/img/biostarx-tna-report-period-setting.png' />

レポート期間を <Cmd sid='timeCard.daily' product='2' />、<Cmd sid='timeCard.weekly' product='2' />、<Cmd sid='timeCard.monthly' product='2' /> に設定できます。 特定期間のレポートを生成するには <Cmd sid='timeCard.custom' product='2' /> を選択します。

### レポートのエクスポート

* <Cmd sid='button.downloadToCsv' product='2' />：生成したレポートを CSV ファイルとして保存できます。

* <Cmd sid='button.exportPdf' product='2' />：生成したレポートを PDF ファイルとして保存できます。

## 警告勤務時間レポートを作成する

指定された勤務時間に到達したユーザーの勤怠レポートを生成したり、メールで管理者に通知したりできます。 警告勤務時間レポートは週単位で生成されます。

1. <Cmd sid='shell.header.launcher' /> ページで <Cmd sid='shell.header.tna' /> をクリックします。

2. 画面左のサイドバーで <Cmd sid='ta.report.title' product='2' /> タブをクリックします。

3. 画面左サイドバーのフィルター一覧で <Cmd>警告勤務時間レポート{/* Working alarm time Report */}</Cmd> をクリックします。

   <Image src='/img/biostarx-tna-report-working-alarm.png' />

4. <Cmd sid='ta.report.condition.filterConditions' product='2' /> と <Cmd sid='ta.report.condition.reportPeriod' product='2' /> の各項目を設定します。

5. レポートを生成するには <Cmd sid='button.updateReport' product='2' /> ボタンをクリックします。

:::info

指定された勤務時間に達したユーザーの通知を管理者へメールで送信するには <Cmd sid='ta.report.condition.autoEmail' product='2' /> を設定します。 詳細については[次のドキュメント](#autoEmail)を参照してください。

:::

### メール自動送信の設定

指定された勤務時間に達したユーザーに関する警告メールを自動で管理者へ送信できます。

<Image src='/img/biostarx-tna-report-working-alarm-auto-email.png' />

* <Cmd sid='common.email' product='2' />：管理者へメールを自動送信する場合は、チェックボックスをクリックします。

* <Cmd sid='setting.dst.dayOfWeek' product='2' />：管理者へメールを送信する曜日を選択します。

* <Cmd sid='shift.time' product='2' />：管理者へメールを送信する時間を設定します。

* <Cmd sid='setting.triggerAction.recipient' product='2' />：メールを受信する管理者を指定できます。 <Cmd sid='button.edit' product='2' /> ボタンをクリックすると、<Cmd sid='setting.triggerAction.recipient' product='2' /> ポップアップが表示されます。 メールアドレスを入力します。 1人以上の管理者を指定できます。

  <Image src='/img/biostarx-tna-report-working-alarm-auto-email-receipt.png' />

:::info

* <Cmd sid='ta.report.condition.autoEmail' product='2' /> 機能を設定するには、フィルター条件を設定してからフィルターを保存する必要があります。

* 自動送信メールの送信者情報は、画面左のサイドバーの <Cmd sid='setting.setting' product='2' />(<IcSet />) → <Cmd sid='setting.triggerAction.senderInfor' product='2' /> で設定できます。 詳細は[次のドキュメント](tna-settings#senderInfo)を参照してください。

:::
