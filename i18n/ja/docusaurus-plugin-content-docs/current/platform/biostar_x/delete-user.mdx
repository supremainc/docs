退職、契約終了、セキュリティ違反などの理由で**BioStar X**に登録したユーザーを削除する場合は、次の手順に従って削除できます。

1. <Cmd sid='shell.header.launcher' /> ページで <Cmd sid='shell.header.user' /> をクリックします。

2. ユーザー一覧で削除するユーザーの左端にあるチェックボックスをクリックします。 1 人以上のユーザーを選択できます。

3. 画面右上の<Cmd sid='common.delete' />をクリックします。

   <Image src='/img/biostarx-remove-users.png' />

4. 確認メッセージが表示されたら<Cmd sid='common.delete' />をクリックします。

ユーザー一覧で選択したユーザーが削除されたことを確認します。

:::info

* `1`のIDを持つユーザーは削除できません。

* デバイスからユーザー情報を削除する方法の詳細は[次のドキュメント](settings-device-manage-device-users)を参照します。

:::
