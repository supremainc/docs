チェックイン状態の訪問者を確認し、訪問情報を編集するかチェックアウトできます。

1. <Cmd sid='shell.header.launcher' /> ページで <Cmd sid='shell.header.visitor' /> をクリックします。

2. 画面上部の <Cmd sid='visitor.status.checkIn' product='2' /> タブをクリックします。

   <Image src='/img/biostarx-visitor-register-checkin-list.png' />

3. 訪問者リストで、編集またはチェックアウトする訪問者をクリックします。

4. 訪問者の訪問申請情報を確認します。

   <Image src='/img/biostarx-visitor-register-checkin-edit.png' />

   * 個別項目を編集して保存するには、画面右下の <Cmd sid='common.edit' product='2' /> ボタンをクリックします。

   * チェックアウト処理を行うには、画面右下の <Cmd sid='button.checkout' product='2' /> ボタンをクリックします。

:::info

<Cmd sid='visitor.status.checkIn' product='2' /> リストから訪問者を選択してチェックアウトすることもできます。 リストでチェックアウトする訪問者のチェックボックスをクリックし、画面右上の <Cmd sid='button.checkout' product='2' /> ボタンをクリックします。

<Image src='/img/biostarx-visitor-register-checkin-list-checkout.png' />

:::
