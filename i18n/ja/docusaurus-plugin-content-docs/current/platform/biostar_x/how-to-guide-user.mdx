**BioStar X** の主要機能の 1 つである <Cmd sid='shell.header.user' /> ページで利用できるさまざまな機能について説明します。 このページでは、ユーザーグループ管理、ユーザー管理、ユーザーテンプレート管理、入退室権限別ユーザー確認など、多様な機能を活用してユーザーを効率的に管理し、権限によってセキュリティを強化する方法を紹介します。 各機能を活用して **BioStar X** をさらに効果的に利用しましょう。

<Cmd sid='shell.header.launcher' /> ページで <Cmd sid='shell.header.user' /> をクリックするか、画面左上のショートカットリストから <Cmd sid='shell.header.user' /> を選択します。
