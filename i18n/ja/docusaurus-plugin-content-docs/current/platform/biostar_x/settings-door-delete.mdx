登録されたドアを削除する方法を説明します。

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.door.root' /> をクリックします。

3. ドア一覧で、削除したいドアのチェックボックスをクリックします。

4. 画面右上の <Cmd sid='button.deleteDoor' product='2' /> ボタンをクリックします。

   <Image src='/img/biostarx-setting-door-delete.png' />

5. 確認メッセージが表示されたら、<Cmd sid='button.yes' product='2' /> ボタンをクリックします。

選択したドアが削除されます。 削除したドアは復元できません。

:::info

ドア一覧で1つ以上のドアを選択すると、<Cmd sid='button.deleteDoor' product='2' /> ボタンが有効になります。

:::
