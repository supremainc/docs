## BioStar Xユーザーガイドへようこそ

**BioStar X**は、AIベースの入退室管理とインテリジェント映像解析を融合したSupremaの次世代統合セキュリティプラットフォームです。 単なる入退室管理を超え、セキュリティ運用全体を効果的に統合管理できる革新的なプラットフォームです。 **BioStar X**はAIベースの入退室管理とインテリジェント映像解析を統合し、AI技術を活用してリアルタイムの入退室管理、映像モニタリング、異常行動検知を1つのプラットフォームで行えるよう設計されています。 入退室管理だけでなく、侵入検知、徘徊検知、転倒検知など多彩なAIベースの映像解析機能を備えており、事前対応が可能です。

本書では、**BioStar X**の多様な機能と設定方法を紹介します。 豊富なガイドにより、**BioStar X**のすべての機能を最大限に活用できるようサポートします。 ステップバイステップのガイドに従って、**BioStar X**を簡単かつ迅速に導入してみましょう。
