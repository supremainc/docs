ユーザー認証手段として QR コードまたはバーコードを登録する方法を案内します。 **BioStar X** で直接生成した QR コードと外部で発行された QR/バーコードを登録できます。

:::info

* QR/Barcode 認証を使用できるスキャナーを搭載したデバイスは下記を参照してください。

  * X-Station 2 (XS2-QDPB, XS2-QAPB)

* QR/Barcode 認証を使用できるカメラを搭載したデバイスは下記のリストを参照してください。

  * X-Station 2 (XS2-ODPB, XS2-OAPB, XS2-DPB, XS2-APB) ファームウェア 1.2.0 以上

  * BioStation 3 (BS3-DB, BS3-APWB) ファームウェア 1.1.0 以上

  * カメラで QR/Barcode 認証を使用するには、別途デバイスライセンスが必要です。 詳細については [次のドキュメント](settings-license-device-license) を参照してください。

:::

## BioStar X QR {#biostarxqr}

暗号化された PIN を含む QR コードをユーザーに発行して入退室を許可できます。

:::note

**始める前に**

* **BioStar QR** を使用する前に、SMTP 設定を含むメール情報の設定を完了してください。 詳細については [次のドキュメント](settings-email-setting) を参照してください。

* **BioStar QR** を発行するには、必ずユーザー情報にメールアドレスを登録する必要があります。 詳細については[次のドキュメント](add-user-basic-info)を参照してください。

:::

1. <Cmd sid='common.credential' product='2' /> セクションで <Cmd sid='button.addQr' product='2' /> ボタンをクリックします。

2. <Cmd sid='user.header.qr' product='2' /> ウィンドウが表示されたら、<Cmd sid='user.qrType' product='2' /> オプションで <Cmd sid='team.user.biostarQr' /> を選択します。

   <Image src='/img/biostarx-new-user-credential-biostarqr.png' />

3. <Cmd sid='zone.intrusion.inputType' product='2' /> オプションを選択します。

   * <Cmd sid='user.random.cardid' product='2' />：カード ID を自動生成します。

   * <Cmd sid='user.manual.input' product='2' />：カード ID を直接入力します。

4. <Cmd sid='button.enroll' product='2' /> ボタンをクリックします。

その後、ユーザーが入退室認証を行う際に QR コードを発行し、ユーザー情報に登録されたメールアドレスへ送信します。 ユーザーはメールに含まれる QR コードをデバイスでスキャンして入退室できます。

:::info

重複したカード ID の生成を防ぐため、<Cmd sid='zone.intrusion.inputType' product='2' /> オプションで <Cmd sid='user.random.cardid' product='2' /> を設定することを推奨します。

:::

## QR/バーコード {#qrbarcode}

外部の他の発行元で発行された QR/バーコードをユーザーに登録できます。

1. <Cmd sid='common.credential' product='2' /> セクションで <Cmd sid='button.addQr' product='2' /> ボタンをクリックします。

2. <Cmd sid='user.header.qr' product='2' /> ウィンドウが表示されたら、<Cmd sid='user.qrType' product='2' /> オプションで <Cmd sid='team.user.credential.qrBarcode.title' /> を選択します。

   <Image src='/img/biostarx-new-user-credential-qrbarcode.png' />

3. <Cmd sid='common.registrationOption' product='2' /> オプションで <Cmd sid='common.enterManually' product='2' /> を選択します。

4. <Cmd sid='user.cardId' product='2' /> オプションに既に発行された QR/バーコード ID を直接入力します。

5. <Cmd sid='button.enroll' product='2' /> ボタンをクリックします。

:::info

<Cmd sid='user.cardId' product='2' /> オプションには、英字、数字、特殊記号を含めて最大 32 文字まで入力できます。

:::
