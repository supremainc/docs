ユーザーのクレデンシャルを登録する方法を案内します。 ユーザーの指紋や顔、カード情報、パスワードなどをクレデンシャルとして登録できます。 クレデンシャルはユーザーを新規登録または編集する際に追加または変更できます。

<Cmd sid='shell.header.launcher' /> → <Cmd sid='shell.header.user' /> ページに移動し、以下の二つの方法でクレデンシャルを登録、追加、変更できます。

import EnterUserPage from '../../common/_enter-user-page.mdx';

<EnterUserPage />

<Cmd sid='team.user.credential.title' /> セクションで設定できる項目は以下のとおりです。 各項目の説明を参考に適切なユーザー権限を設定します。

<Image src='/img/biostarx-new-user-credential.png' />

クレデンシャルを登録し、<Cmd sid='user.fingerSecurityLevel' product='2' /> 項目で適切なセキュリティレベルを設定します。 セキュリティレベルが高いほど認証率が低下したり本人拒否率（FRR）が高くなる可能性があります。

:::
