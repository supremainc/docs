**BioStar X** は、大規模エンタープライズ環境向けに、安定して拡張可能な統合セキュリティ管理ソリューションを提供します。 システムはメインサーバーとコミュニケーションサーバーで構成される分散アーキテクチャにより、最大 1,000 台以上のデバイスを効率的に管理できます。

メインサーバーはシステムの中核管理機能を担当し、コミュニケーションサーバーはデバイス通信を専任することで負荷を分散し、パフォーマンスを最適化します。 管理者はクライアント PC のウェブブラウザーからシステムにアクセスし、すべての管理業務を実行できます。 以下のシステム要件を確認し、組織規模に合わせて最適な環境を構築してください。

:::info

ライセンス構成および最適化の詳細については、[Suprema テクニカルサポートチーム](https://support.supremainc.com) までお問い合わせください。

:::

## メインサーバー {#main-server}

メインサーバーは **BioStar X** の中央管理システムがインストールされるサーバーです。 ユーザーがウェブブラウザーで接続してシステムを管理し、ユーザー情報や入退室権限設定などのすべての業務を処理します。 さらに、システム全体の状態を監視し、イベントログやアラームの管理を担当します。

<div className='overflow-x col5--15-10-25'>

| \>     | 項目            | 小規模組織          | 中規模組織          | 大規模組織                                                                                                                                        |
| ------ | ------------- | -------------- | -------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| 使用環境   | デバイス数         | 1～50 台         | 51～100 台       | 101～1,000 台                                                                                                                                  |
| システム環境 | オペレーティング システム | \>             | \>             | Windows 10、Windows 11、Windows Server 2016、Windows Server 2019、Windows Server 2022、Windows Server 2025                                   |
| ^      | データベース        | \>             | \>             | MariaDB 11.4.4、MS SQL Server 2012 SP3、MS SQL Server 2014 SP2、MS SQL Server 2016 SP1、MS SQL Server 2017、MS SQL Server 2019、MS SQL Server 2022 |
| ^      | CPU           | 2.3 GHz 6 Core | 2.3 GHz 8 Core | 最小 2.3 GHz<br/>推奨 4.0 GHz 16 Core                                                                                                            |
| ^      | RAM           | 16GB           | 32GB           | 最小 64GB<br/>推奨 128GB                                                                                                                         |
| ^      | SSD           | 512GB          | 512GB          | 1TB                                                                                                                                          |
| ^      | HDD           | \>             | \>             | <Cmd sid='common.imageLog' product='2' /> 機能を使用する場合、1,000万件のイメージログにつき 200GB が必要です                                                            |

</div>

:::info

* **BioStar X** は 64 ビットのオペレーティングシステムにのみインストールできます。

* MS SQL Server のセキュア通信は TLS 1.2 をサポートします。

* MS SQL Server と **BioStar X** を異なる PC にインストールする場合、**BioStar X** がインストールされている PC に [Microsoft OLE DB Driver for SQL Server](https://learn.microsoft.com/sql/connect/oledb/download-oledb-driver-for-sql-server) をインストールする必要があります。

* 大規模組織では MS SQL Server データベースの導入を推奨します。

* データベースに MS SQL Server を使用し、**BioStar X T&A** を利用する場合は [Microsoft ODBC Driver 17 for SQL Server](https://learn.microsoft.com/ko-kr/sql/connect/odbc/download-odbc-driver-for-sql-server?view=sql-server-ver16#version-17) が必要となるため、環境に適したドライバーをインストールしてください。

* Mac オペレーティングシステムで提供される Boot Camp の Windows 仮想環境はサポートしていません。

* Ethernet (TCP/IP) で接続できるデバイスの最大数は 1,000 台です。 コミュニケーションサーバーを追加して、さらにデバイスを接続します。

:::

### SQL Server ライセンス要件 {#sqlserver-requriements}

データベースに SQL Server を使用する場合、**BioStar X** を安定して運用するには、接続されるデバイス数に応じた適切な SQL Server ライセンスが必要です。 各環境の推奨ライセンス構成を参考に、組織の規模とニーズに合ったライセンスを選択してください。

:::info

SQL Server のライセンスの詳細については [Microsoft SQL Server ライセンス](https://www.microsoft.com/sql-server/sql-server-2022-pricing) を参照してください。

:::

## コミュニケーションサーバー {#communication-server}

コミュニケーションサーバーは、入退室管理デバイスとの通信を専任で行うサーバーです。 メインサーバーとは別に設置し、多数のデバイスを安定して管理できるよう支援します。 1 台のコミュニケーションサーバーで最大 1,000 台のデバイスを接続でき、必要に応じて複数台を追加することもできます。

<div className='overflow-x'>

| \>     | 項目            | 仕様                                                                                                         |
| ------ | ------------- | ---------------------------------------------------------------------------------------------------------- |
| システム環境 | オペレーティング システム | Windows 10、Windows 11、Windows Server 2016、Windows Server 2019、Windows Server 2022、Windows Server 2025 |
| ^      | CPU           | 最小 2.3 GHz、推奨 4.0 GHz 16 Core                                                                              |
| ^      | RAM           | 最小 64GB、推奨 128GB                                                                                           |
| ^      | SSD           | 1TB                                                                                                        |

</div>

:::info

Ethernet (TCP/IP) で接続できるデバイスの最大数は 1,000 台です。

:::

## クライアント {#client}

クライアントは、管理者が Web ブラウザーを介して **BioStar X** にアクセスし、システムを運用する PC です。

<div className='overflow-x'>

| \>     | 項目       | 小規模組織   | 中規模組織   | 大規模組織                                    |
| ------ | -------- | ------- | ------- | ---------------------------------------- |
| システム環境 | CPU      | 2.5 GHz | 2.5 GHz | 最小 2.5 GHz                               |
| ^      | RAM      | 16GB    | 32GB    | 最小 32GB                                  |
| ^      | GPU      | \>      | \>      | VMS を使用する場合、最低でも NVIDIA GeForce RTX 4060 |
| ^      | ウェブブラウザー | \>      | \>      | Google Chrome バージョン 100 以上               |

</div>

:::info

* **BioStar X** は Google Chrome に最適化されています。

* **BioStar X** のウェブインターフェースはモバイルブラウザーをサポートしていません。

:::
