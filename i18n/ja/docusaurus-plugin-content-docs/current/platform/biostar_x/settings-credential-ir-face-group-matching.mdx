<Cmd sid="setting.menu.credential.irFaceGroupMatching" className="normal" /> は、ユーザーグループを基にマッチンググループを設定してユーザーを認証する機能です。 この機能を使用すると、特定のユーザーグループに属するユーザーだけがIRフェイス認証を通じて入退室できます。

1. <Cmd sid="shell.header.launcher" />ページで<Cmd sid="shell.header.setting" />をクリックします。

2. 画面左のサイドバーで <Cmd sid="setting.menu.credential.root" /> → <Cmd sid="setting.menu.credential.irFaceGroupMatching" /> をクリックします。

3. <Cmd sid="globalSetting.facegroup.detail.useOption" product="2" /> オプションを <Cmd sid="common.use" /> に変更します。

   <Image src="/img/biostarx-settings-credential-irface-group-match.png" />

4. <Cmd sid="globalSetting.facegroup.detail.deviceTap.title" product="2" /> で <Cmd sid="button.addplus" product="2" /> ボタンをクリックします。

5. <Cmd sid="globalSetting.facegroup.detail.deviceTap.title" product="2" /> ウィンドウが表示されたら、目的のデバイスを選択し、<Cmd sid="button.add" product="2" /> ボタンをクリックします。

   :::note

   FaceStation 2モデルのみ選択できます。

   :::

6. <Cmd sid="globalSetting.facegroup.detail.groupSettings" product="2" /> で <Cmd sid="button.addplus" product="2" /> ボタンをクリックします。

7. <Cmd sid="globalSetting.facegroup.detail.groupName" product="2" /> と <Cmd sid="common.userGroup" product="2" /> 項目を設定します。

8. 設定を保存するには、画面右下の<Cmd sid="button.apply" product="2" />ボタンをクリックしてください。

:::info

* 最大で10個のマッチンググループを作成できます。

* グループごとに最大3,000件のIRフェイステンプレートを含めることができます。

* 1つのマッチンググループに複数のユーザーグループを設定できます。

* マッチンググループ全体のIRフェイステンプレート数は5,000件を超えられません。

* ユーザーグループに含まれるIRフェイステンプレートが3,000件を超える場合、マッチンググループとして設定できません。

* この機能はFaceStation 2モデルのみ対応しています。

* グループマッチング機能を無効化するには、既に設定されているデバイスとグループ設定をすべて削除する必要があります。

:::
