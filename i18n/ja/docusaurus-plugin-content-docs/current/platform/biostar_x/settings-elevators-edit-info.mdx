登録済みエレベーターの設定情報を編集する方法を案内します。 個別エレベーターの詳細設定を変更するか、複数のエレベーターを選択して共通項目を一括編集します。

## 1台のエレベーター情報を編集する {#edit-one-elevator-info}

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.elevator.root' /> をクリックします。

3. エレベーター一覧で編集対象のエレベーターをクリックします。

4. 各セクションの詳細項目を編集します。

5. すべての設定が完了したら、画面最下部の <Cmd sid='button.apply' product='2' /> ボタンをクリックします。

:::info

詳細は[以下のドキュメント](settings-elevators-add)を参照します。

:::

## 複数エレベーターを一括編集する {#edit-multiple-elevator-info}

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.elevator.root' /> をクリックします。

3. エレベーター一覧で編集対象のエレベーターのチェックボックスをクリックします。 2台以上のエレベーターを選択します。

4. 画面右上にある<Cmd sid='button.batchEdit' product='2' />をクリックします。

5. <Cmd sid='button.batchEdit' product='2' />ウィンドウが表示されたら、必要なオプションを設定します。

6. 設定が完了したら<Cmd sid='button.apply' product='2' />をクリックします。

:::info

<Cmd sid='button.batchEdit' product='2' /> ボタンは、エレベーター一覧で2台以上のエレベーターを選択すると有効になります。

:::
