登録済みのエレベーターを削除する方法を説明します。

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.elevator.root' /> をクリックします。

3. エレベーター一覧で、削除するエレベーターのチェックボックスをクリックします。

4. 画面右上の <Cmd sid='button.delete' product='2' /> ボタンをクリックします。

選択したエレベーーターが削除されます。 削除したエレベーターは復元できません。

:::info

<Cmd sid='button.delete' product='2' /> ボタンは、エレベーター一覧で1台以上のエレベーターを選択すると有効になります。

:::
