ダッシュボードに追加したウィジェットの編集方法を説明します。

1. <Cmd sid='shell.header.launcher' /> ページで <Cmd sid='shell.header.dashboard' /> をクリックします。

2. 画面右上の <IcSet2 /> アイコンをクリックします。 

3. <Cmd sid='reportApp.dashboardSettings' product='2' />モードに入ったら、編集する**ウィジェット**の右上にある<IcEdit />アイコンをクリックします。

4. <Cmd sid='reportApp.editWidget' product='2' />ウィンドウが表示されたら、必要な項目を編集します。

   <Image src='/img/biostarx-widget-edit.png' className='none' caption />

5. 編集が完了したら、<Cmd sid='reportApp.apply' product='2' />ボタンをクリックします。

6. <Cmd sid='reportApp.dashboardSettings' product='2' />画面の右上にある<Cmd sid='reportApp.apply' product='2' />ボタンをクリックします。

ダッシュボード画面で変更内容が反映されていることを確認します。

:::info

* <Cmd sid='reportApp.editWidget' product='2' />では<Cmd sid='reportApp.widgetTypes' product='2' />を変更できません。

* 定義済み項目を使用する<Cmd sid='reportApp.widgetTypes.text' product='2' />, <Cmd sid='reportApp.widgetTypes.systemUsage' product='2' />, <Cmd sid='reportApp.widgetTypes.doorControl' product='2' />では<Cmd sid='reportApp.widgetName' product='2' />を変更できません。

:::
