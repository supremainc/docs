Suprema の Airfob Portal と連携してモバイルアクセスカードを使用すると、ユーザーにモバイルアクセスカードを発行できます。 モバイルアクセスカードは、各ユーザーが個別に登録することも、CSV インポート機能を使用して多数のユーザーを一括登録することもできます。

Airfob Portal で設定したモバイルアクセスカード送信方法に応じて、ユーザーのメールアドレスを入力する必要があります。

:::info

* モバイルアクセスカードは **CSN モバイルカード** と **テンプレート・オン・モバイル** のいずれか一方のみ使用できます。

* この機能を使用するには Airfob Portal と連携する必要があります。 Airfob Portal との連携およびモバイルアクセスカードの使用について詳しくは[以下のドキュメント](settings-credential-mobile-access-card)を参照してください。

:::

## CSN モバイル {#csnmobilecard}

ユーザーに CSN モバイルアクセスカードを発行できます。

### カード割り当て

1. <Cmd sid='common.credential' product='2' /> セクションで <Cmd sid='common.mobileCredential' product='2' /> ボタンをクリックしてください。

2. <Cmd sid='user.header.mobile' product='2' /> ウィンドウが表示されたら、<Cmd sid='user.cardType' product='2' /> オプションで <Cmd sid='user.cardEnrol.cardTypeCsnMobile' product='2' /> を選択してください。

3. <Cmd sid='common.registrationOption' product='2' /> オプションで <Cmd sid='common.assignCard' product='2' /> を選択してください。

   <Image src='/img/biostarx-new-user-credential-mobile-csn-assign.png' className='none' />

   <AirFobNote />

4. カード一覧から割り当てたいカードを選択するか検索してください。

5. <Cmd sid='common.info' product='2' /> と <Cmd sid='common.validPeriod' product='2' /> を設定してください。

6. <Cmd sid='button.enroll' product='2' /> ボタンをクリックします。

:::info

<MobileNotes />

:::

### 直接入力

1. <Cmd sid='common.credential' product='2' /> セクションで <Cmd sid='common.mobileCredential' product='2' /> ボタンをクリックしてください。

2. <Cmd sid='user.header.mobile' product='2' /> ウィンドウが表示されたら、<Cmd sid='user.cardType' product='2' /> オプションで <Cmd sid='user.cardEnrol.cardTypeCsnMobile' product='2' /> を選択してください。

3. <Cmd sid='common.registrationOption' product='2' /> オプションで <Cmd sid='common.enterManually' product='2' /> を選択します。

   <Image src='/img/biostarx-new-user-credential-mobile-csn.png' className='none' />

   <AirFobNote />

4. <Cmd sid='zone.intrusion.inputType' product='2' /> オプションを選択します。

   * <Cmd sid='user.random.cardid' product='2' />：カード ID を自動生成します。

   * <Cmd sid='common.useUserId' product='2' />：ユーザー ID をカード ID として使用します。

   * <Cmd sid='user.manual.input' product='2' />：カード ID を直接入力できます。

5. <Cmd sid='common.info' product='2' /> と <Cmd sid='common.validPeriod' product='2' /> を設定してください。

6. <Cmd sid='button.enroll' product='2' /> ボタンをクリックします。

:::info

<MobileNotes />

* メールで送信したアクティベーションコードをユーザーが紛失または削除し、アクティベーションリンクを再発行する場合は **再送信** ボタンをクリックしてください。

* CSN モバイルアプリケーションの詳細については[次のリンク](https://www.csn.edu/csn-mobile-apps)を参照してください。

:::

## <Cmd sid='team.user.tom' className='normal' /> {#template-on-mobile}

**テンプレート・オン・モバイル** は、ユーザーのバイオメトリックテンプレートをモバイルデバイスに保存するモバイルアクセスカードです。 **BioStar X** サーバー、Airfob Portal、およびデバイスにユーザーデータを保存しなくてもバイオメトリック認証が行えます。

この機能は、バイオメトリックをクレデンシャルとして使用したいが、プライバシー保護のためサーバーやデバイスに生体情報を保存できない環境で有用です。 **テンプレート・オン・モバイル** の発行とバイオメトリックテンプレートの登録が別々に行われるため、ユーザーは管理者と対面せずに **テンプレート・オン・モバイル** 対応デバイスで直接顔を登録できます。

1. <Cmd sid='common.credential' product='2' /> セクションで <Cmd sid='common.mobileCredential' product='2' /> ボタンをクリックしてください。

2. <Cmd sid='user.header.mobile' product='2' /> ウィンドウが表示されたら、<Cmd sid='user.cardType' product='2' /> オプションで <Cmd sid='user.cardEnroll.cardTypeTemplateOnMobile' product='2' /> を選択してください。

   <Image src='/img/biostarx-new-user-credential-mobile-tom.png' className='none' />

   <AirFobNote />

3. <Cmd sid='user.cardModeLong' product='2' /> オプションで希望のカードタイプを選択してください。

   * <Cmd sid='user.accessOnCard' product='2' />：ユーザーの情報（カード ID、PIN、アクセスグループ、期間、個人認証モード）をカードに保存できます。

   * <Cmd sid='user.secureCredentialCard' product='2' />：セキュアクレデンシャルカードです。 ユーザーのカード ID と PIN データをカードに保存できます。

4. <Cmd sid='common.info' product='2' /> と <Cmd sid='common.validPeriod' product='2' /> を設定してください。

5. <Cmd sid='button.enroll' product='2' /> ボタンをクリックします。

登録されたユーザーのメールアドレスに発行メールを送信します。 メールのリンクから Airfob Pass アプリケーションをインストールし、モバイルデバイスで **テンプレート・オン・モバイル** の発行を完了すると使用できます。

:::tip

**テンプレート・オン・モバイル認証方法**

ユーザーはテンプレート・オン・モバイルを発行したモバイルデバイスを装置にかざし、画面の案内に従って顔を認証してください。

:::

:::info

<MobileNotes />

* **テンプレート・オン・モバイル** が使用可能な装置とファームウェアバージョンについては以下の一覧を参照してください。

  * BioStation 3 ファームウェア 1.2.0 以上

  * BioEntry W3 ファームウェア 1.0.0 以上

  * FaceStation F2 ファームウェア 2.2.0 以上

* モバイルアクセスカードは **CSN モバイルカード** と **テンプレート・オン・モバイル** のいずれか一方のみ発行できます。

:::
