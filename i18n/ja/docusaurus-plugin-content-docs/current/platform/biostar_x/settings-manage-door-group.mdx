複数のドアを簡単に管理できるよう、グループ設定方法を案内します。 ドアグループは、複数のドアを 1 つのグループにまとめて管理できる機能です。 グループを設定すると、ドアを一括設定できます。

:::tip

ドアの位置やオフィス名などでグループ名を登録すると、管理が便利です。

:::

## ドアグループを追加 {#adding-door-group}

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.door.root' /> をクリックします。

3. ドア一覧で <Cmd sid='menu.door.allDoorGroups' product='2' /> を選択し、右クリックします。

   <Image src='/img/biostarx-setting-door-add-group.png' />

4. ポップアップメニューで <Cmd sid='door.addDoorGroup' product='2' /> をクリックします。

   <Image src='/img/biostarx-setting-door-add-group-naming.png' />

5. 希望するグループ名を入力します。

:::info

* ドアグループは最大 8 階層まで作成できます。

* ドアグループ名は最大 48 文字まで入力できます。

:::

## ドアグループ名を変更 {#renaming-door-group}

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.door.root' /> をクリックします。

3. ドア一覧で名前を変更するグループを選択し、右クリックします。

   <Image src='/img/biostarx-setting-door-rename-group.png' />

4. ポップアップメニューで <Cmd sid='door.renameDoorGroup' product='2' /> をクリックします。

5. 変更するグループ名を入力します。

:::info

ドアグループ名は最大 48 文字まで入力できます。

:::

## ドアグループを削除 {#deleting-door-group}

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.door.root' /> をクリックします。

3. ドア一覧で削除するグループを選択し、右クリックします。

   <Image src='/img/biostarx-setting-door-rename-group.png' />

4. ポップアップメニューで <Cmd sid='door.deleteDoorGroup' product='2' /> をクリックします。

:::caution

ドアグループを削除すると、グループに含まれるすべてのドアが削除されます。 グループ内のドアを削除したくない場合は、まずドアを別のグループに移動してからグループを削除してください。

:::

## グループにドアを追加 {#adding-door-to-group}

### ドア一覧からグループに追加する

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.door.root' /> をクリックします。

3. ドア一覧でグループに追加するドアを選択し、目的のグループにドラッグします。

   <Image src='/img/biostarx-setting-door-to-group.png' />

選択したドアがグループに移動します。

<Image src='/img/biostarx-setting-door-to-group-result.png' />

### ドア情報からグループに追加する

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.door.root' /> をクリックします。

3. 画面右側のドア一覧で、グループに追加するドアをクリックします。

   <Image src='/img/biostarx-setting-door-to-group-list.png' />

4. ドア情報の編集画面が表示されたら、<Cmd sid='common.information' product='2' /> セクションの <Cmd sid='common.group' product='2' /> オプションをクリックします。

   <Image src='/img/biostarx-setting-door-to-group-info-select.png' />

5. 目的のグループを選択します。

6. 画面最下部にある <Cmd sid='button.apply' product='2' /> をクリックします。
