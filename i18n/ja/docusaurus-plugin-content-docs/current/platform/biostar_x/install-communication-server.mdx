本ドキュメントでは、**BioStar X** の **Communication サーバー** のインストール方法を案内します。 

コミュニケーションサーバーは、入退室管理デバイスとの通信を専任で行うサーバーです。 メインサーバーとは別に設置し、多数のデバイスを安定して管理できるよう支援します。 1 台のコミュニケーションサーバーで最大 1,000 台のデバイスを接続でき、必要に応じて複数台を追加することもできます。

:::info

Ethernet (TCP/IP) で接続できるデバイスの最大数は 1,000 台です。

:::

## 開始前 {#start-before}

Communication サーバーをインストールする前に、次の事項を確認します。

* Communication サーバーは **BioStar X** サーバーと同じサーバーにインストールできません。

* Communication サーバーは 64 ビットオペレーティングシステムにのみインストールできます。 システムの最小要件の詳細は [次のドキュメント](system-requirements#communication-server) を参照します。

* Communication サーバーをインストールして接続するには、マルチ Communication サーバーライセンスが必要です。 ライセンスポリシーの詳細は[次のドキュメント](licensing#features-add-ons)を参照してください。

### インストール前の確認事項 {#check-before-install}

Communication サーバーをインストールする前に **BioStar X Service Manager** を実行し、インストール中に入力が必要な情報を確認します。

1. Windows で <Start/> → **BioStar X** → **BioStar X Service Manager** をクリックします。

2. <Cmd>BioStar X Service Manager</Cmd> ウィンドウが表示されます。

3. <Cmd>SERVICE SETTINGS</Cmd> をクリックします。

以下に、Communication サーバーのインストール時に入力が必要な情報を示します。 各項目を確認して準備します。

<Image src='/img/biostarx-cumm-server-matching.png' />

<div className='table-fixed'>

| BioStar X Service Manager                                         | Communication サーバーインストール時の入力情報                           |
| ----------------------------------------------------------------- | -------------------------------------------------------- |
| <Cmd>BioStar X Core Service</Cmd> → <Cmd>RPC Port</Cmd>           | **BioStar X Server gRPC Port for Communication**         |
| <Cmd>BioStar X Coordinator Service</Cmd> → <Cmd>Client Port</Cmd> | **BioStar X Coordinator Service Port for Communication** |
| <Cmd>BioStar X Cache Service</Cmd> → <Cmd>Client Port</Cmd>       | **BioStar X Cache Service Client for Communication**     |

</div>

:::info

<Cmd>SERVICE SETTINGS</Cmd> の詳細は [次のドキュメント](service-settings) を参照します。

:::

## インストールガイド {#install-guide}

Communication サーバーをインストールするには、次の手順に従います。

1. [Suprema ダウンロードセンター](https://download.supremainc.com/) にアクセスしてログインし、インストールパッケージ (*BioStar X Communications Server.X.Y.X.BB.exe*) をダウンロードします。

2. ダウンロードしたインストールファイルを実行してください。

3. 使用する言語を選択し、<Cmd code='biostar2.install.ok' className='bold'/> ボタンをクリックしてください。

   <Image src='/img/biostarx-install-select-language.png' />

4. インストールを続行するには、<Cmd code='biostar2.install.agree' className='bold'/> を選択し、<Cmd code='biostar2.install.next' className='bold'/> ボタンをクリックしてください。

   <Image src='/img/biostarx-install-comm-server-license-agree.png' />

5. Communication サーバーをインストールするパスを設定し、<Cmd code='biostar2.install.next' className='bold'/> をクリックします。

   <Image src='/img/biostarx-install-comm-server-set-path.png' />

6. 個人情報に関する権限と責任の内容を確認し、インストールを続行するには <Cmd code='biostar2.install.next' className='bold'/> をクリックします。

   <Image src='/img/biostarx-install-comm-server-private-policy.png' />

7. Communication サーバーのコンポーネントを選択し、<Cmd code='biostar2.install.next' className='bold'/> をクリックします。

   <Image src='/img/biostarx-install-comm-server-select-component.png' />

8. **BioStar X** サーバーの IP アドレスとポート番号を入力し、**BioStar X** サーバーと接続するために各項目を入力します。 設定を完了し、<Cmd code='biostar2.install.next' className='bold'/> をクリックします。

   <Image src='/img/biostarx-install-comm-server-settings.png' />

   :::note

   各入力項目の詳細は [次のドキュメント](#check-before-install) を参照します。

   :::

9. **BioStar X** の管理者 ID とパスワードを入力し、<Cmd code='biostar2.install.next' className='bold'/> ボタンをクリックします。

   <Image src='/img/biostarx-install-comm-server-login.png' />

10. 通信用ポートを設定し、<Cmd code='biostar2.install.next' className='bold'/> をクリックします。

    <Image src='/img/biostarx-install-comm-server-port-settings.png' />

11. すべてのインストール準備が完了したら、<Cmd code='biostar2.install.btnInstall' className='bold'/> ボタンをクリックしてください。 インストールを実行します。

    <Image src='/img/biostarx-install-comm-server-install-step.png' />

12. インストール完了メッセージを確認し、<Cmd code='biostar2.install.complete' className='bold'/> をクリックします。

    <Image src='/img/biostarx-install-comm-server-complete.png' />

Communication サーバーのインストールを完了します。 インストールが完了したら、**BioStar X** メインサーバーで Communication サーバーと接続されていることを確認します。

## インストール後の確認事項 {#check-after-install}

Communication サーバーのインストールを完了したら、**BioStar X Service Manager** を使用してメインサーバーと Communication サーバーを接続する必要があります。

1. Windows で <Start/> → **BioStar X** → **BioStar X Service Manager** をクリックします。 

2. <Cmd>BioStar X Service Manager</Cmd> ウィンドウが表示されます。

3. 画面左側のサイドバーで <Cmd>COMMUNICATIONS</Cmd> メニューをクリックします。

   <Image src='/img/common/biostarx-sm-cluster-main-01.png' alone />

4. 画面右上の <Cmd>Add Communication Server</Cmd> ボタンをクリックします。

5. <Cmd>Add Communication Server</Cmd> ウィンドウが表示されたら、インストールした Communication サーバーの IP アドレスと一致する項目のチェックボックスをクリックし、<Cmd>Database</Cmd> 列で使用するデータベースを選択します。

   <Image src='/img/common/biostarx-sm-cluster-main-03.png' alone />

6. <Cmd>Add</Cmd> をクリックします。

Communication サーバー一覧で、追加した Communication サーバーの <Cmd>Server Status</Cmd> 列が <Cmd>Connected</Cmd> 状態であることを確認します。

<Image src='/img/common/biostarx-sm-cluster-main-04.png' alone />

:::info

* <Cmd>COMMUNICATIONS</Cmd> メニューは、マルチ Communication サーバーライセンスを有効化したときに使用できます。 ライセンスポリシーの詳細は[次のドキュメント](licensing#features-add-ons)を参照してください。

* Communication サーバーを削除するには、一覧で削除するサーバーのチェックボックスをクリックし、画面右上の <Cmd>Delete Commnunication Server</Cmd> ボタンをクリックします。

* Communication サーバーを追加した場合、メインサーバーの負荷を軽減するために、Communication サーバーと連携する補助データベースを追加できます。 補助データベースの追加の詳細は [次のドキュメント](add-sub-db) を参照します。

:::
