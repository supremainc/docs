**BioStar X** は **BioStar 2** の後継バージョンとして開発された次世代入退室管理ソフトウェアです。 強化されたシステム アーキテクチャと拡張された機能、そしてより柔軟なライセンス モデルにより、現在および将来のさまざまな導入環境をサポートします。

**BioStar X** は **スターター**から **エリート**まで、合計 5 つの拡張可能なライセンス グレードを提供しており、小規模事業所から大規模企業環境まで、ニーズに合わせて選択できます。 モジュール構造で設計されているため、ビデオ連携、モバイルアクセス、マルチサーバーサポートなどのさまざまなオプション機能を追加できます。

## 基本ライセンス {#base-license}

* **デバイスマネージャー** ライセンスは、サードパーティーのソフトウェアまたはシステムを使用するお客様が **BioStar X** を通じて Suprema デバイスを効率的に管理する際に必要なライセンスです。

## ライセンスアップグレード {#capacity-upgrade}

ライセンスアップグレードは **エッセンシャル**、**アドバンス**、**エンタープライズ**、**エリート** グレードのライセンス保有者に提供されます。 **デバイスマネージャー** ライセンスの保有者は、ユーザー (User) と管理者 (Operator) のアップグレードのみ購入できます。

| &nbsp; |   数   |
| :----: | :---: |
|   ドア   |   32  |
|  ユーザー  | 5,000 |
|   管理者  |   10  |

## パッケージ (Package) {#package}

以下の項目は **アドバンス** ライセンスグレードでは追加オプションとして購入でき、**エンタープライズ** および **エリート** グレードでは基本で含まれています。

* **高度アクセス制御**：アンチパスバック（Global Anti-Passback）、火災警報（Global Fire Alarm）、警備（Intrusion Alarm）、マスタリング（Mustering）、人数制限（Occupancy Limit）、エレベーター、インターロック（Interlock）などの高度なアクセス制御機能を利用できるパッケージです。

## デバイスライセンス

デバイスライセンスは個別に購入でき、基本ライセンスに関係なく追加できます。 デバイスライセンスはシリアル番号（S/N）を基準に発行されます。 ライセンスが必要なデバイスでのみ、その機能を有効にできます。

| 機能             | ライセンス種類 | 対応モデル                                 | 備考                  |
| -------------- | ------- | ------------------------------------- | ------------------- |
| **顔サーバーマッチング** | デバイス    | -                                     | 高度なサポートには保守契約が必要です。 |
| **カメラ QR**     | デバイス    | X-Station 2（QR 読取センサーなし）、BioStation 3 | -                   |
| **ワイヤレスロック**   | デバイス    | CoreStation 40                        | -                   |
