ダッシュボードは、ユーザーごとに必要な情報を選択してウィジェットを追加し、ウィジェットを自由に設定・配置して好みに合わせて構成できます。

:::info

* 各ユーザーは個別に<Cmd sid='shell.header.dashboard' />を構成できます。

* <Cmd sid='shell.header.dashboard' />で確認できる情報は、<Cmd sid='user.bioStarOperator' product='2' />によって異なる場合があります。

:::
