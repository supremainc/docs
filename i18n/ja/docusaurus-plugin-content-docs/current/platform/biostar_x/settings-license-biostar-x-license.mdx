購入したBioStar Xライセンスを有効化できます。

ユーザーのネットワーク環境によって、<Cmd sid='setting.menu.license.biostarXLicense' /> を有効化する方法が異なります。 案内に従ってネットワーク状態を確認し、ライセンスを有効化してください。

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左のサイドバーで <Cmd sid='setting.menu.license.root' /> → <Cmd sid='setting.menu.license.biostarXLicense' /> をクリックします。

:::info

* ライセンスポリシーの詳細は [以下](licensing) を参照してください。

* ライセンスのエラーコードを確認するには[次のリンク](https://reprisesoftware.com/docs/isv/appendix/appendix-b-rlm-status-values.html)を参照してください。

:::
