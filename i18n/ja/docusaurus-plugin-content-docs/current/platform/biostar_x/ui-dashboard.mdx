<Cmd sid='shell.header.dashboard' /> ページのユーザーインターフェース(User Interface)について説明します。

<Image src='/img/biostarx-dashboard.png' className='none' caption />

<Calloutlists>

1. **ウィジェット表示エリア**：ユーザーが追加したウィジェットが表示されるエリアです。

   * 各ウィジェットのサイズは、ウィジェットごとに制限された範囲内で自由に調整できます。

   * ウィジェットは空いている領域にのみ配置でき、ウィジェット同士を重ねることはできません。

2. **再読み込み**：ダッシュボードページを再読み込みして、編集したウィジェットのデータを更新します。

   <Cmd sid='reportApp.dashboardSettings' product='2' /> の <Cmd sid='reportApp.autoRefreshInterval' product='2' /> で再読み込み間隔を設定できます。

3. <Cmd sid='reportApp.dashboardSettings' product='2' />：新しいウィジェットを追加するか、追加済みウィジェットのデータと外観を設定できます。

</Calloutlists>
