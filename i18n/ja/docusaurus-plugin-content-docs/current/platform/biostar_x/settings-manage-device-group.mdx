複数のデバイスを効率的に管理できるデバイスグループ機能について説明します。 デバイスグループを使用すると複数のデバイスを1つのグループにまとめて管理できるため、大規模システムでデバイス設定を一括適用したり監視したりする際に便利です。

:::tip

デバイスグループ名は、デバイスの設置場所や用途で指定すると管理がしやすくなります。 例えば、「1階出入口」「2階オフィス」「会議室」などでグループを設定できます。

:::

## デバイスグループの作成

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.device.root' /> をクリックしてください。

3. デバイスリストで <Cmd sid='menu.device.allDeviceGroups' product='2' /> を右クリックします。

   <Image src='/img/biostarx-setting-device-add-group.png' />

4. ポップアップメニューで <Cmd sid='device.menu.addDeviceGroup' product='2' /> をクリックします。

5. 希望するグループ名を入力します。

:::info

* デバイスグループは最大8階層までサブグループを作成できます。

* デバイスグループ名は最大48文字まで入力できます。

* デバイスリストでデバイスグループを選択すると、そのグループに属するデバイスのみがリストに表示されます。

:::

## デバイスグループ名の変更

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.device.root' /> をクリックしてください。

3. デバイスリストの <Cmd sid='menu.device.allDeviceGroups' product='2' /> 配下で、名前を変更するデバイスグループを選択して右クリックします。

   <Image src='/img/biostarx-setting-device-rename-group.png' />

4. ポップアップメニューで <Cmd sid='device.menu.renameDeviceGroup' product='2' /> をクリックします。

5. 変更するグループ名を入力します。

:::info

デバイスグループ名は最大48文字まで入力できます。

:::

## デバイスグループの削除

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左側のサイドバーで <Cmd sid='setting.menu.device.root' /> をクリックしてください。

3. デバイスリストの <Cmd sid='menu.device.allDeviceGroups' product='2' /> 配下で、削除するデバイスグループを選択して右クリックします。

   <Image src='/img/biostarx-setting-device-remove-group.png' />

4. ポップアップメニューで <Cmd sid='device.menu.deleteDeviceGroup' product='2' /> をクリックします。

:::caution

デバイスグループを削除すると、そのグループに属するすべてのデバイスも削除されます。

:::
