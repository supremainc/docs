VMS連携を完了した後、BioStar Xで使用するカメラを選択して設定します。 適切なカメラ設定により、次のセキュリティ管理機能を利用できます。

* **選択的モニタリング**：VMSのすべてのカメラの中から必要なカメラだけを**BioStar X**に追加できます。

* **効率的な管理**：カメラをグループ別に構成し、場所や用途に応じて体系的に管理できます。

:::info

VMSサーバーと連携する方法は[次のドキュメント](settings-video-integration)を参照してください。

:::

## カメラの設定と管理

VMS連携後、<Cmd sid="shell.header.setting" /> → <Cmd sid="setting.menu.video.root" /> → <Cmd sid="setting.menu.video.video" /> メニューでカメラを設定および管理できます。

### カメラの追加

VMSと連携している場合、VMSサーバーに接続されたカメラを**BioStar X**に追加し、<Cmd sid="shell.header.arena" /> ページでライブ映像と録画映像を再生できます。

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左のサイドバーで <Cmd sid='setting.menu.video.root' /> → <Cmd sid='setting.menu.video.video' /> をクリックします。

3. <Cmd sid="video.addCamera.button" product="2" /> ボタンをクリックします。

4. <Cmd sid="video.addCamera" product="2" /> ウィンドウが表示されたら、一覧から除外するカメラを選択します。

   目的のカメラが表示されない場合は、<Cmd sid="video.addCamera.refresh" product="2" /> ボタンをクリックしてカメラ一覧を更新します。

5. カメラをすべて選択したら、画面右下の<Cmd sid="button.add" product="2" /> ボタンをクリックします。

カメラ一覧で追加したカメラを確認できます。

<Image src='/img/biostarx-settings-video-add-camera.png' />

### カメラ情報の編集

追加したカメラの情報を編集できます。 カメラ名、グループ、説明などを変更できます。

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左のサイドバーで <Cmd sid='setting.menu.video.root' /> → <Cmd sid='setting.menu.video.video' /> をクリックします。

3. 画面左上の<Cmd sid="video.table.th.camera" product="2" /> タブをクリックします。

4. 一覧から編集するカメラをクリックします。

5. カメラ情報が表示されたら、必要な項目を編集します。

   <Image src='/img/biostarx-settings-video-edit-camera-info.png' />

6. 編集が完了したら、画面右下の<Cmd sid="button.apply" product="2" /> ボタンをクリックします。

:::info

カメラIDとIPアドレスは編集できません。

:::

### カメラの削除

追加したカメラを削除できます。

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左のサイドバーで <Cmd sid='setting.menu.video.root' /> → <Cmd sid='setting.menu.video.video' /> をクリックします。

3. 一覧で削除するカメラの左端にあるチェックボックスをクリックします。 1台以上のカメラを選択できます。

4. 画面右上の<IcTrash /> <Cmd sid="video.camera.delete" product="2" /> ボタンをクリックします。

5. メッセージを確認し、<Cmd sid='button.yes' product='2' /> ボタンをクリックします。

<Image src='/img/biostarx-settings-video-delete-camera.png' />

:::info

削除対象のカメラがいずれかのルールに含まれている場合、カメラを削除できません。 そのカメラをルールから削除してから再試行します。 ルールの作成と管理の詳細は[次のドキュメント](settings-videos-rule)を参照してください。

:::

## カメラグループの設定と管理

### カメラグループの追加

カメラグループを追加し、複数のカメラをグループ化して管理できます。

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左のサイドバーで <Cmd sid='setting.menu.video.root' /> → <Cmd sid='setting.menu.video.video' /> をクリックします。

3. 画面左上の<Cmd sid="video.table.th.camera" product="2" /> タブをクリックします。

4. <Cmd sid="video.camera.all.group" product="2" /> 項目を選択し、右クリックします。

   <Image src='/img/biostarx-settings-video-add-camera-group.png' />

5. ポップアップメニューで<Cmd sid="video.contextMenu.add.group" product="2" /> をクリックします。

6. 希望するグループ名を入力し、<Kbd>Enter</Kbd> キーを押します。

:::info

* カメラグループは8階層までのサブグループを作成できます。

* カメラグループ名は最大48文字まで入力できます。

* カメラ一覧でグループを選択すると、そのグループに属するカメラだけが表示されます。

:::

### グループへの複数カメラの追加

作成したグループに1台以上のカメラを追加できます。

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左のサイドバーで <Cmd sid='setting.menu.video.root' /> → <Cmd sid='setting.menu.video.video' /> をクリックします。

3. 一覧でグループに追加するカメラの左端にあるチェックボックスをクリックします。 1台以上のカメラを選択できます。

4. 1台以上を選択すると、画面右上の<IcEditUL /> <Cmd sid="button.batchEdit" product="2" /> ボタンが有効になります。 <Cmd sid="button.batchEdit" product="2" /> ボタンをクリックします。

   <Image src='/img/biostarx-settings-video-batch-edit-camera.png' />

5. <Cmd sid="button.batchEdit" product="2" /> ウィンドウが表示されたら、<Cmd sid="common.group" product="2" /> 項目の<IcEdit /> ボタンをクリックします。

6. グループ一覧から目的のグループを選択します。

7. <Cmd sid='button.apply' product='2' /> ボタンをクリックします。

:::info

<Cmd sid="video.table.th.camera" product="2" /> タブの一覧から項目を選択し、ドラッグ＆ドロップでもカメラをグループに追加できます。

<Image src='/img/biostarx-settings-video-move-camera-group.png' alone />

:::

### カメラグループ名の編集

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左のサイドバーで <Cmd sid='setting.menu.video.root' /> → <Cmd sid='setting.menu.video.video' /> をクリックします。

3. <Cmd sid="video.table.th.camera" product="2" /> タブをクリックします。

4. 名前を変更するグループを選択し、右クリックします。

5. ポップアップメニューで<Cmd sid="video.contextMenu.rename.group" product="2" /> をクリックします。

   <Image src='/img/biostarx-settings-video-rename-camera-group.png' />

6. 希望するグループ名を入力し、<Kbd>Enter</Kbd> キーを押します。

### カメラグループの削除

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左のサイドバーで <Cmd sid='setting.menu.video.root' /> → <Cmd sid='setting.menu.video.video' /> をクリックします。

3. <Cmd sid="video.table.th.camera" product="2" /> タブをクリックします。

4. 名前を変更するグループを選択し、右クリックします。

5. ポップアップメニューで<Cmd sid="video.contextMenu.delete.group" product="2" /> をクリックします。

6. 警告メッセージを確認し、<Cmd sid="button.yes" product="2" /> ボタンをクリックします。

:::info

カメラグループにカメラが含まれている場合、グループを削除できません。 グループ内のカメラを別のグループへ移動するか、グループから削除してから再試行します。

:::

## リスト列を設定する

リストで表示する列を追加、順序を変更、または非表示に設定できます。

1. <Cmd sid='shell.header.launcher' />ページで<Cmd sid='shell.header.setting' />をクリックします。

2. 画面左のサイドバーで <Cmd sid='setting.menu.video.root' /> → <Cmd sid='setting.menu.video.video' /> をクリックします。

3. <Cmd sid="video.header.all.camera" product="2" /> リストの右上で<IcMoreW /> → <Cmd sid="columnSetting.dialog.header" product="2" /> をクリックします。

4. <Cmd sid='columnSetting.dialog.header' product='2' /> ウィンドウが表示されたら、表示する列のチェックボックスをクリックして追加または非表示設定ができます。 ドラッグ＆ドロップで列の順序を変更することもできます。

   <Image src='/img/biostarx-settings-video-column-layout.png' />

5. 列設定をすべて変更したら、<Cmd sid='button.ok' product='2' /> ボタンをクリックします。

:::info

列設定を初期化するには、<Cmd sid='columnSetting.button.defaultColumn' product='2' /> をクリックします。

:::
