ダッシュボードに追加されたウィジェットを削除する方法を説明します。

1. <Cmd sid='shell.header.launcher' /> ページで <Cmd sid='shell.header.dashboard' /> をクリックします。

2. 画面右上の <IcSet2 /> アイコンをクリックします。

3. <Cmd sid='reportApp.dashboardSettings' product='2' /> モードに入ったら、削除したい **ウィジェット** の右上にある <IcTrash2 /> ボタンをクリックします。

4. 削除確認ポップアップを確認し、<Cmd sid='reportApp.yes' product='2' /> ボタンをクリックします。

   <Image src='/img/biostarx-dashboard-widget-del.png' className='none' />

5. <Cmd sid='reportApp.dashboardSettings' product='2' /> 画面右上の <Cmd sid='reportApp.apply' product='2' /> をクリックします。

ダッシュボード画面で変更内容が反映されていることを確認します。
