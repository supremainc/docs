<Cmd sid='shell.header.arena' /> ページのユーザーインターフェース (User Interface) を案内します。 <Cmd sid='shell.header.arena' /> ページでは、扉、マップ、デバイスを基準にセキュリティ状態をリアルタイムでモニタリングできます。 また、リアルタイムで発生するイベントを確認し、必要な対応を行えます。 <Cmd sid='shell.header.arena' /> ページの UI コンポーネントは次のとおりです。

<Image src='/img/biostarx-ui-monitoring.png' className='none' />

<Calloutlists>

1. 検索入力フィールドにキーワードを入力して、デバイス、扉、カメラなど必要な項目を検索できます。

2. <Cmd sid='arena.tree.door.title' />, <Cmd sid='arena.tree.map.title' />, <Cmd sid='arena.tree.device.title' /> をグループ単位のツリー構造リストで確認し、モニタリングできます。

   * <Cmd sid='arena.tree.door.title' /> のモニタリングの詳細は [以下](monitoring-door) を参照してください。

   * <Cmd sid='arena.tree.map.title' /> のモニタリングの詳細は [以下](monitoring-map) を参照してください。

   * <Cmd sid='arena.tree.device.title' /> のモニタリングの詳細は [以下](monitoring-device) を参照してください。

3. <Image src='/img/menus/ico-more-black.svg' ico alone /> ボタンをクリックすると、リストのツリー構造を展開または折りたたむことができます。

   * <Cmd sid='arena.tree.expandAll' />：サイドメニューでグループ内のすべてのリストを展開して表示します。

   * <Cmd sid='arena.tree.collapseAll' />：サイドメニューでグループ内のすべてのリストを折りたたんで非表示にします。

4. **ビデオタイル** では、扉、カメラ、エリアなどのリアルタイム映像またはマップを確認できます。 複数カメラの映像を同時に配置してモニタリングできます。

   * カメラ映像のモニタリングの詳細は [以下](monitoring-video) を参照してください。

   * マップモニタリングの詳細は [以下](monitoring-map) を参照してください。

5. **コントロールパネル** では、選択した扉またはデバイスの機能を制御できます。 詳細は [以下](control-door) を参照してください。

6. **イベントリスト** では、リアルタイムで発生するイベントを時系列で確認できます。 イベントごとに詳細情報、状態、関連ユーザーおよびデバイス情報を確認し、必要に応じてすぐに対処できます。 詳細は [以下](monitoring-event) を参照してください。

</Calloutlists>
