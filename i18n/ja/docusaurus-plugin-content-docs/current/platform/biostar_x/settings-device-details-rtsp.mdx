<Cmd sid='device.info.rtsp' product='2' />セクションでは、**Real Time Streaming Protocol**(RTSP) ストリーミング用の詳細項目を設定できます。 デバイスのカメラを通じてリアルタイムで映像を配信できます。

<Image src='/img/biostarx-setting-device-edit-rtsp.png' />

* <Cmd sid='device.info.rtsp' product='2' />：RTSP ストリーミングを使用するには、<Cmd sid='device.info.enabled' product='2' />に設定します。

* <Cmd sid='device.info.rtsp.address' product='2' />：RTSP サーバーアドレスは固定されています。 RTSP アドレスをコピーするには、<Cmd sid='device.info.rtsp.copy' product='2' />をクリックします。

* <Cmd sid='device.info.rtsp.port' product='2' />：RTSP ポートを設定します。

* <Cmd sid='device.info.rtsp.id' product='2' />：RTSP サーバーの ID を入力します。

* <Cmd sid='device.info.rtsp.password' product='2' />：RTSP サーバーのパスワードを入力します。

* <Cmd sid='device.info.rtsp.resolution' product='2' />：RTSP 使用時に出力される映像の解像度を選択します。 デフォルトは <Cmd>180 x 320</Cmd> です。

:::info

* この機能は BioStation 3 と BioEntry W3 モデルでサポートされています。

* <Cmd sid='device.info.rtsp.resolution' product='2' /> オプションは BioStation 3 のファームウェア 1.3.0 以降でサポートされています。

* ユーザーが任意で変更した内容がある場合は、<Cmd sid='button.apply' product='2' /> をクリックします。 変更内容を保存しないとデバイスに反映されません。

:::
