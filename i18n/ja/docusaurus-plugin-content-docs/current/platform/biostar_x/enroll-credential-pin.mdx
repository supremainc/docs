ユーザーがデバイスにPINを入力して入退室できるように、PINを登録する方法を説明します。 個々のユーザーにPINを登録することも、CSVインポートで多数のユーザーにPINを一括登録することもできます。

## PIN登録

1. <Cmd sid='common.credential' product='2' /> セクションで <Cmd sid='user.pin' product='2' /> ボタンをクリックします。

   <Image src='/img/biostarx-new-user-credential-pin.png' />

2. 入力フィールドが有効になったら、パスワードを入力します。

3. <Cmd sid='user.confirmPin' product='2' /> に同じパスワードを入力します。

4. <Cmd sid='button.enroll' product='2' /> ボタンをクリックします。

:::info

* PINは8桁以上で、数字のみ入力できます。

* ユーザーごとに登録できるPINクレデンシャルは最大1つです。

:::

## CSVインポートでPINを登録

CSVインポート機能を使用すると、多数のユーザーを一度に登録できます。

1. インポートするCSVファイルに`pin`列を追加します。

2. `pin`列にユーザーのPINを入力し、CSVファイルを保存します。

3. <Cmd sid='shell.header.user' /> ページに移動します。

4. 画面右上の<IcMoreW /> ボタンをクリックし、<Cmd sid='team.menu.import.title' /> → <Cmd sid='team.menu.import.csv' /> を選択します。

5. 画面の案内に従ってCSVファイルを選択し、インポートオプションを設定します。

:::info

* ユーザーごとに登録できるPINクレデンシャルは最大1つです。

* CSVインポート機能の詳細は[次のドキュメント](export-import-user-info#import-csv)を参照してください。

:::
