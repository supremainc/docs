生成されたレポートのテーブルをクリックすると勤怠記録を修正できます。

:::info

* 勤怠記録を修正するには、先にレポートを生成する必要があります。 レポートの生成について詳しくは、[以下のドキュメント](tna-report)を参照してください。

* 勤怠スケジュールが登録されていないユーザーの出退勤記録は修正できません。

:::

1. 生成されたレポート一覧で、修正する記録をクリックします。

2. 必要に応じて勤怠記録を修正するか、不在を追加します。

## 一覧で修正する

<Image src='/img/biostarx-tna-report-edit-list.png' />

<Calloutlists className='svg-none-style'>

1. <Cmd sid='timeCard.dateRange' product='2' />：一覧に表示する勤怠記録の期間を設定できます。

2. **日別勤怠記録**：日別の勤怠記録を確認できます。

   * 出勤／退勤時間をクリックすると勤怠記録を追加・修正・削除できます。 出勤／退勤時間をクリックしてから<IcEdit />ボタンをクリックすると、既に登録されている勤怠記録を修正できます。変更内容を保存するには<Cmd sid='button.apply' product='2' />ボタンをクリックします。

   * 不在を追加するには<IcEdit />ボタンをクリックします。 休暇を追加するには、不在として設定された時間ルールが必要です。 追加したユーザー不在の<IcTrash2 />をクリックすると削除できます。

3. **勤怠記録サマリー**：設定した期間の勤怠記録を確認できます。

4. **更新** / **カレンダー表示**

   * <IcRefresh />：勤怠記録一覧を更新します。

   * <IcCalc />：勤怠記録をカレンダー形式で確認できます。

</Calloutlists>

:::info

時間ルールの作成について詳しくは、[以下のドキュメント](tna-rule-shift-paycode)を参照してください。

:::

## カレンダーで修正する

<Image src='/img/biostarx-tna-report-edit-calendar.png' />

<Calloutlists className='svg-none-style'>

1. **イベント種別**：イベント種別を選択してカレンダーに表示または非表示にできます。

2. **月**：前月または翌月に移動するには<IcFw />または<IcBw />ボタンをクリックします。

3. **日別勤怠記録**：日別の勤怠記録を確認できます。

   * 勤務時間（白色）をクリックすると勤怠記録を追加・修正・削除できます。 <IcEdit />ボタンをクリックすると、既に登録されている勤怠記録を修正できます。変更内容を保存するには<Cmd sid='button.apply' product='2' />ボタンをクリックします。

   * 勤務ルール（灰色）をクリックするとユーザー不在管理を追加できます。 ユーザー不在を追加するには、不在として設定された時間ルールが必要です。 追加したユーザー不在の<IcTrash2 />をクリックすると削除できます。

4. **勤怠記録サマリー**：月別の勤怠記録を確認できます。

5. **更新** / **一覧表示**

   * <IcRefresh />：勤怠記録一覧を更新します。

   * <IcList />：勤怠記録を一覧形式で確認できます。

</Calloutlists>

:::info

時間ルールの作成について詳しくは、[以下のドキュメント](tna-rule-shift-paycode)を参照してください。

:::
