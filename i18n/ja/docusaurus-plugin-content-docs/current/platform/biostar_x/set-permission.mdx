組織のセキュリティポリシーに応じて、ユーザーごとに **BioStar X** の運用権限と入退室権限を区別して設定できます。 管理者グレードで主要メニューへのアクセスを制限し、入退室グループで物理的スペースへのアクセスを制御できます。 さらに、IP アドレス制限と多要素認証によりセキュリティを強化できます。

<Cmd sid='shell.header.launcher' /> → <Cmd sid='shell.header.user' /> ページに移動し、次の 2 つの方法で入退室権限を設定します。

import EnterUserPage from '../../common/_enter-user-page.mdx';

<EnterUserPage />

<Cmd sid='team.user.permission.title' /> セクションで設定できる項目は次のとおりです。 各項目の説明を参考に適切なユーザー権限を設定します。

<Image src='/img/biostarx-new-user-permission.png' />

## 管理者グレードを選択 {#operatorLevel}

管理者グレードによってアクセスできるメニューが制限されます。 <Cmd sid='team.user.permission.title' /> セクションの <Cmd sid='user.bioStarOperator' product='2' /> 項目で、ユーザーの管理者グレードを選択します。 ユーザーに割り当てられる管理者グレードは次の一覧を参照してください。

import AccountList from '../../common/_account-list.mdx';

<AccountList />

### ログイン ID とパスワードを設定

管理者グレードを選択したら、そのユーザーが **BioStar X** にログインする際に使用する ID とパスワードを設定する必要があります。

<Image src='/img/biostarx-new-user-permission-set-login.png' />

:::info

**ID とパスワードを設定する際の注意事項**

* ID は英字と数字の組み合わせで 8～32 文字以内に設定します。

* パスワードポリシー設定の詳細は[次のドキュメント](settings-system-security#password-policy-configuration)を参照してください。

* セキュリティ上脆弱なパスワードは使用しないでください。

  * 連続した文字または数字（例：abcd、1234）

  * 一般的な単語またはパターン（例：qwerty、password）

:::

### ログインユーザーのパスワード変更 {#changeLoginUserPassword}

ユーザーは **BioStar X** にログインした後、自分のパスワードを変更できます。

1. **BioStar X** にログインします。

2. 画面右上の <IcAccount width='26' height='26' /> ボタンをクリックします。

   <Image src='/img/biostarx-user-account-popup.png' />

3. ドロップダウンメニューで <Cmd sid='auth.passwordChange.title' /> をクリックします。

4. <Cmd sid='auth.passwordChange.title' /> ウィンドウが表示されたら、現在のパスワードと新しいパスワードを入力します。

   <Image src='/img/biostarx-user-change-pw.png' />

5. すべての項目を入力したら <Cmd sid='common.button.change' /> ボタンをクリックします。

## 多要素認証を設定 {#multiFactorAuth}

<Cmd sid='common.fingerprintLogin' product='2' /> を有効にすると、ユーザーは **BioStar X** にログインする際に指紋認証を追加してログインできます。

:::info

**多要素認証** の設定に関する詳細は[次のドキュメント](login-with-mfa)を参照してください。

:::

## ユーザー IP アドレスでログインを設定 {#userIp}

ユーザー IP アドレスを登録すると、アカウントに登録した IP 情報と一致する場合にのみ PC にログインできます。 これによりセキュリティが向上します。 <Cmd sid='team.user.permission.title' /> セクションの <Cmd sid='team.user.permission.userIp' /> 項目にユーザーの IP アドレスを入力します。

:::info

* IP アドレスは xxx.xxx.xxx.xxx 形式で入力でき、各オクテットには 0～255 の数値のみを入力できます。

* IP アドレスを登録しない場合は、IP 情報に関係なく PC にログインできます。

:::

## 入退室グループを選択 {#accessGroup}

ユーザーの入退室グループを設定して、組織や部署ごとに入退室を制限できます。 <Cmd sid='team.user.permission.title' /> セクションの <Cmd sid='team.user.permission.accessGroup' /> 項目で入退室グループを選択します。

:::info

入退室グループの追加および設定の詳細は[次のドキュメント](settings-access-control-manage-access-group)を参照してください。

:::
