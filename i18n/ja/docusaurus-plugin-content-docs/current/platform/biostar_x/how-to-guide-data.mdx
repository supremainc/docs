<Cmd sid='shell.header.data' /> メニューを使用すると、**BioStar X** に登録されたユーザー情報や希望条件のイベントを照会でき、レポートを一定周期で生成できます。 必要に応じて希望のレポートを一定周期で自動生成でき、生成されたレポートは CSV または PDF ファイルとしてエクスポートするか印刷できます。

<Cmd sid='shell.header.launcher' /> ページで <Cmd sid='shell.header.data' /> をクリックするか、画面左上のショートカットリストから <Cmd sid='shell.header.data' /> を選択してください。
