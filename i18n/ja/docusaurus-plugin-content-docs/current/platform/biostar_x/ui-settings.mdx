<Cmd sid='shell.header.setting' /> ページでは、**BioStar X** のさまざまな設定を管理できます。 このページの UI コンポーネントは次のとおりです。

<Image src='/img/biostarx-ui-settings.png' className='none' />

<Calloutlists className='svg-none-style'>

1. <Cmd sid='shell.header.setting' /> メニューから提供されるさまざまな設定メニューを確認できます。 詳細は [次のドキュメント](how-to-guide-settings) を参照してください。

2. ページリストをナビゲートできるツールです。 ページを移動したり、任意のページへジャンプできます。

   <Image src='/img/biostarx-settings-access-control-status-pagenation.png' />

   * <IcFirst2 />：最初のページに移動します。

   * <IcFw2 />：前のページに移動します。

   * <IcBw2 />：次のページに移動します。

   * <IcLast2 />：最後のページに移動します。

   * 目的のページに移動するには、数字入力フィールドにページ番号を入力します。

   * 1ページに表示する一覧の件数を設定するには、行選択ボックスをクリックします。

   :::info

   設定メニューによっては対応していない場合があります。

   :::

3. <IcMoreW /> ボタンをクリックすると追加機能を使用できます。

   :::info

   設定メニューによってサポートされる機能は異なる場合があります。

   :::

</Calloutlists>
