マップ設定機能は、マップと連携してフロア、施設、エリアを設定および管理する機能です。 この機能を使用すると、<Cmd sid='shell.header.arena' /> ページでエリア、施設、フロアごとに区分されたマップを確認できます。

<DocCardList />

:::info

* マップモニタリングの詳細は [以下](monitoring-map) を参照してください。

* マップ設定およびマップモニタリング機能は、**アドバンス**以上のライセンスでのみ利用できます。

:::
