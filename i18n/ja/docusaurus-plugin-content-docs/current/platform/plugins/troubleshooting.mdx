**BioStar X** プラグインの開発・登録・運用過程で発生し得る問題の解決方法を提供します。

## プラグイン登録に失敗する

* プラグインを登録する前にサーバーが稼働していることを確認してください。

* <Badge className='get'>GET</Badge> `/bsx` エンドポイントが正しく応答するか確認してください。 `curl` コマンドを使用してテストできます。

  ```bash
  curl -H "X-BioStar-Ping: test123" http://localhost:8000/bsx
  ```

* **BioStar X** サーバーからプラグインサーバーの URL にアクセスできることを確認してください。

## 認証コールバックに失敗する

* 証明書ファイルが存在し、有効であることを確認してください。

* **BioStar X Service Manager** の <Cmd>SERVICE SETTINGS</Cmd> で **BioStar X** サーバーのアドレスとポートを確認してください。

* プラグインサーバーと **BioStar X** サーバー間のネットワーク接続を確認してください。

## 復号化エラー

* AES キーの生成と暗号化プロセスを確認してください。

* IV が同じキーから派生しているか確認してください。

## デバッグのヒント

* リクエストおよびレスポンスの詳細を確認するには、ロギングを有効にしてください。

* 統合前にエンドポイントを個別にテストしてください。

* `curl` コマンドを使用して **BioStar X** サーバーへのリクエストをシミュレーションしてください。
