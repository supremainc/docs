## 主要な活用事例

**BioStar X** のプラグインシステムを使用して、さまざまな外部システムと連携できます。 代表的な活用事例は次のとおりです。

* **勤怠管理システム**：外部 T&A（Time & Attendance）ソリューションとの連携

* **来訪者管理システム**：別途の来訪者登録・管理システムとの連携

* **レポートソリューション**：カスタムレポートの作成および分析ツールとの連携

* **他社システム連携**：ERP、HRM など既存業務システムとのデータ連携

## プラグインの種類

### 外部プラグイン

* **開発主体**：顧客、パートナー、他社開発会社

* **登録方法**：**BioStar X Service Manager** の **PLUGINS** ページで手動登録

* **登録制限**：最大10件まで登録可能

* **管理**：ユーザーが直接追加/削除を管理

### Suprema 提供プラグイン

* **開発主体**：Suprema

* **提供方式**：インストールパッケージによる配布

* **自動連携**：インストール時に **BioStar X** と自動連携

* **管理**：プログラムのインストール/削除で管理

## システム制限事項

* **プラグイン総数の上限**：12件

* **外部プラグインの上限**：10件
