Suprema Passアプリを使用すると、プラスチックのRFIDカードの代わりにモバイルクレデンシャルでドアを開けることができます。 モバイルデバイスをBluetoothまたはNFCでタグ付けするか、カードを長押しして、QRコードをスキャンしてドアを開けることができます。

:::info

Suprema PassアプリはBioStar Airシステムのユーザーインターフェースとして機能し、主にモバイル認証情報用のデジタルウォレットとして使用されます。 ユーザーはアプリ内でモバイル入退室カードを発行・保存し、モバイル機器を使って安全な場所に出入りできます。

:::

## 必須要件と準備

モバイル認証情報で出入り口を開くには、次の要件を満たす必要があります。

* Suprema Passのモバイル認証情報に対応した入退室制御装置が設置されている必要があります。

* ユーザーのモバイルデバイスにSuprema Passアプリがインストールされている必要があります。

  * [iOS](https://apps.apple.com/kr/app/suprema-pass/id6745057639)

  * [Android](https://play.google.com/store/apps/details?id=com.supremainc.supremapass)

* モバイルデバイスでBluetoothまたはNFC通信が有効でなければなりません。

* 2段階認証を有効にすると<Cmd>バックグラウンドモード</Cmd>は無効になります。

* QR認証は管理者がBioStar Airポータルで有効にする必要があります。

* ハンズフリー入退室機能を使用するには、その入退室方法がサイトでサポートされているか確認してください。

:::info

* モバイル機器のSuprema Passアプリでモバイル入退室カードにより認証を行うには、位置情報サービスが必要です。 <DocLink docId='platform/biostar_air/location-services-issues' />で関連するFAQを確認してください。

* モバイルカードが機器と認証されているがドアが開かない場合は、案内に従って問題を解決してください。 <DocLink docId='platform/biostar_air/troubleshooting-suprema-pass-app' />で関連するFAQを確認してください。

:::

## 接続方法の設定 {#setCoonection}

Suprema Pass アプリで、ユーザーは2種類の動作モードから選択できます。 Suprema Passアプリの初期画面で、右上の<IcHamberg />アイコンをタップした後、<Cmd>Connection Method</Cmd>から希望のモードを選択してください。

* <Cmd>Only while App is on Screen</Cmd>: アプリが画面に表示され、モバイルカードが見えているときのみ出入口を開けることができます。

* <Cmd>Background Mode</Cmd>: アプリがバックグラウンドで実行されている場合、端末がロックされていても出入口を開けることができます。

:::info

モードに関係なく、ユーザーは常にモバイル機器をデバイスにかざして認証できます。

:::

## 出入口を開ける

Suprema Passアプリを使用して、ユーザーはさまざまな方法で出入口を開けることができます。

### モバイル機器をデバイスにかざす

モバイル機器をBioStar Airと互換性のあるデバイスに近づけ、BluetoothまたはNFCで認証して出入口を開けることができます。

1. Suprema Passアプリが実行中であることを確認してください。

2. モバイル機器のBluetoothまたはNFCアンテナ部分をデバイスに近づけてください。

3. モバイルクレデンシャルが認証されると出入口が開きます。

:::info

* ユーザーは2つの通信モードから選択できます。 詳細については <DocLink docId='platform/biostar_air/suprema-pass-app-advanced-features' /> を参照してください。

  * **Bluetoothモード**: モバイル機器がデバイスとワイヤレスで通信します。

  * **NFCモード**: Android端末でのみ使用でき、モバイル機器をデバイスにかざして認証できます。

* モバイル機器のBluetoothまたはNFCアンテナの位置はモデルによって異なる場合があります。 最適な接触ポイントはモバイル機器の仕様を参照してください。

:::

### モバイルカードを長押し

Suprema Passアプリでモバイルカードを長押しすることで出入口を開けることができます。

1. Suprema Passアプリの初期画面を表示してください。

2. BioStar Airと互換性のあるデバイスの近くで、モバイルカードを2-3秒間長押ししてください。

3. モバイルクレデンシャルが認証されると出入口が開きます。

:::info

* この機能を使用するには、<Cmd>接続方式</Cmd>設定で必ず<Cmd>Background Mode</Cmd>を選択してください。 詳細については <DocLink inner='#setCoonection' /> を参照してください。

* セキュリティ上の理由により、**サイト管理者がこの機能を無効にする場合があります**。

* **有効範囲は**モバイル機器のBluetooth強度によって異なります。

* 近隣の入退室制御リーダーが意図しない認証を引き起こす可能性があります。

:::

### QRコード認証

管理者がBioStar Airポータルで<Cmd sid='site_setting_qr_label' product='air' />オプションを有効にしている場合、Suprema PassアプリでQRコードをQRリーダーにかざして出入口を開けることができます。 Suprema Passアプリのホーム画面で、モバイルカードの下にある<IcSpqr title='QR Scan' />アイコンをタップしてください。

<Image src='/img/air/bsair-sp-app-contact-qrcode.png' width='50%' className='none' />

:::info

* <Cmd sid='site_setting_qr_label' product='air' />オプションの詳細は <DocLink docId='platform/biostar_air/site-additional-settings' /> を参照してください。

* QRコード認証はQRリーダーが設置された出入口でのみ使用できます。

:::
