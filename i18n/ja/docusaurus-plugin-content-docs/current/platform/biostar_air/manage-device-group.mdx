BioStar Airでは、複数のデバイスをグループにまとめて効率的に管理できます。 デバイスグループを使用すると、同じ設定を複数のデバイスに一括適用したり、特定のグループのデバイスの状態を簡単に監視できます。

## デバイスグループの追加

1. 画面左のサイドバーで <Cmd sid='device_menu' product='air' /> → <Cmd sid='device_groups_submenu' product='air' /> をクリックします。

   <Image src='/img/air/bsair-device-mng-group-init.png' />

2. 画面右上の <Cmd sid='device_group_btn_add' product='air' /> ボタンをクリックします。

3. <Cmd sid='device_group_title_add_popup_overlay' product='air' /> ウィンドウが表示されたら、グループ名を入力します。

   <Image src='/img/air/bsair-device-mng-group-add-group.png' />

   :::note

   特定グループのサブグループとしてデバイスグループを追加するには、<Cmd sid='add_sub_group_text' product='air' /> オプションをクリックし、親グループを選択します。

   :::

4. <Cmd sid='device_group_label_add_device' product='air' /> でグループに含めるデバイスを選択します。

5. すべてのデバイスを選択したら、<Cmd sid='btn_ok' product='air' /> ボタンをクリックします。

デバイスグループ一覧で、追加されたグループとそのグループに属するデバイスの数を確認できます。 デバイスグループをクリックすると、そのグループに属するデバイスを確認できます。

<Image src='/img/air/bsair-device-mng-group-list.png' />

:::info

デバイスをグループに追加したり既存のグループから削除したりする方法の詳細は、<DocLink docId='platform/biostar_air/managing-registered-devices#device_group_in_device_list' />をご確認ください。

:::

## デバイスグループ名の変更

1. 画面左のサイドバーで <Cmd sid='device_menu' product='air' /> → <Cmd sid='device_groups_submenu' product='air' /> をクリックします。

2. デバイスグループ一覧で、名前を変更するグループのチェックボックスをクリックします。

3. デバイスグループ一覧の上部にある<IcAirEdit title='Edit' />ボタンをクリックしてください。

4. <Cmd sid='device_group_title_edit_popup_overlay' product='air' /> ウィンドウが表示されたら、新しいグループ名を入力します。

   <Image src='/img/air/bsair-device-mng-edit-group.png' />

5. 変更したグループ名を保存するには、<Cmd sid='btn_ok' product='air' /> ボタンをクリックします。

## デバイスグループの削除

1. 画面左のサイドバーで <Cmd sid='device_menu' product='air' /> → <Cmd sid='device_groups_submenu' product='air' /> をクリックします。

2. デバイスグループ一覧で、削除するグループのチェックボックスをクリックします。

3. デバイスグループ一覧の上部にある <IcAirRemove /> ボタンをクリックします。

4. 警告メッセージのウィンドウが表示されたら、<Cmd sid='btn_yes' product='air' /> をクリックします。

デバイスグループ一覧で選択したグループが削除されます。 そのグループのサブグループもすべて削除されます。 グループを削除してもデバイスは削除されません。
