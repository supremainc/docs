The BioStar Air Partner Portal is a dedicated platform for value-added resellers (VARs) with distribution agreements. Create customer sites, allocate credits, or configure federation sites to centrally manage multiple locations. 

Install on-site hardware such as SIP intercoms, IP cameras, and QR scanners, and configure network settings.
