## 100% クラウド入退室管理プラットフォーム

BioStar Airは、生体認証セキュリティ分野の世界的リーダーであるSupremaが開発したクラウドネイティブな入退室管理ソリューションです。 オンプレミスインフラの必要性をなくすよう設計されたBioStar Airは、どこからでも管理できる安全で柔軟かつスケーラブルな入退室管理を提供します。 クラウドに直接接続されたスマートリーダーを通じて、企業は携帯電話やブラウザだけで複数の拠点の入退室を簡単に制御できます。

## 比類のないクレデンシャルの柔軟性

BioStar Airは、最も包括的な認証オプションを提供し、安全で便利なさまざまな方法で施設への入退室を管理できます。

* **顔認証**

* **iOSおよびAndroid向けカスタムモバイル認証**（無料）

* **各種RFIDフォーマット**

* **アプリ不要のDynamic QRおよびLinkPass**

## オンプレミスインフラ不要

BioStar Airはオンサイトサーバー、仮想マシン、VPN、またはドアコントローラーの必要性をなくしました。 IT保守は不要です。 **スマートリーダー**（Smart reader）を設置してクラウドに接続すれば、すぐに入退室管理を始められます。 リーダー（Reader）は、同じネットワークや物理的な場所に関係なく複数のサイトで動作します。

## 簡単なマルチサイト管理

BioStar Airは単一のシステムで、**オフィス**、**倉庫**、**小売店**、**遠隔地**などあらゆる拠点の入退室を制御できる包括的なソリューションを提供します。

## エッジアーキテクチャによる将来に備えた拡張性

BioStar Airの**エッジアーキテクチャ**は、入退室管理をスマートリーダーに直接内蔵しており、別途ドアコントローラーを必要としません。 これにより企業は**容易に拡張**できます。 現在必要な機能だけを購入し、複雑なアップグレードなしで簡単に拡張できます。

## クラウド中心の企業に最適化

BioStar Airは業界最高の生体認証技術を搭載した、クラウドネイティブな入退室管理ソリューションを求める**システムインテグレーター**および**顧客**のために設計されています。 クラウド中心の企業の入退室管理に対して、モダンで柔軟かつ高い安全性を備えた手段を提供します。

## BioStar Airの機能

| 機能 | 説明 |
| --- | --- |
| **包括的な入退室管理ソリューション** | 対応 |
| **Suprema ハードウェア互換** | XPass 2、X-Station 2、BioEntry W3、BioStation 3、Airfob Patch、Airfob Tag、Output Module（OM-120）、SIO2<br/>- 対応予定：XPass D2、CoreStation 20 |
| **高いセキュリティ** | ISO 27001 / 27701、TLS 1.2、AES-256 |
| **生体認証：顔および指紋** | Supremaリーダーに対する基本サポートを提供しており、生体認証データベースを内蔵しています。 |
| **RFIDカード** | MIFARE Classic、DesFire、Felica、EM4100、Custom Wiegand Formats、iClass、Seos。 1ユーザーあたり最大8枚のカードをサポートします。 |
| **モバイル認証情報** | 無料で無制限に提供されます。 セキュアなBLE/NFC認証情報（iOS、Android）、アプリ不要のLinkPassおよびQR認証を含みます。 |
| **展開** | クラウドネイティブ |
| **インターフェース** | ブラウザ、iOSアプリ、Androidアプリ |
| **価格モデル** | サブスクリプションベース |
| **統合** | 幅広いAPI、Webhook、およびSDK |
| **エッジアーキテクチャ** | リーダーには内蔵コントローラーがあり、イーサネットを介してクラウドに直接接続されます。 別途クラウドノードや入退室制御パネルは必要ありません。 |
| **ログイン** | どこからでも利用できます。 |
| **サーバーの所在地** | 韓国（グローバル）およびドイツ（ヨーロッパ） |
| **勤怠** | リーダーの勤怠キー（key）はイベントログに記録され、勤務時間レポートが作成されて統合できます。 |
| **ビデオ** | OnvifおよびUniviewをサポートする基本的なVMSです。 NVRは不要で、リーダーがゲートウェイの役割を果たします。 |
| **アラーム** | 侵入警報エリアに対する管理者への通知をサポートします。 |
| **データパーティショニング** | 標準 |
| **対象** | フランチャイズ、複数拠点、マルチテナント、遠隔地など |
