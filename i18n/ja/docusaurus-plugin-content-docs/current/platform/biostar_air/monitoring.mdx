BioStar Airのモニタリング機能により、システムの主要な活動をリアルタイムで追跡し、管理します。 イベントおよび変更ログを監視してシステム活動を分析し、ドアの状態をリアルタイムで確認してリモートで制御し、生成された通知とレポートを通じてセキュリティ活動を総合的に管理します。
