エレベーターを登録し、階レベルを定義して、ユーザーやグループがアクセスできる階を制限します。 各階レベルごとにアクセススケジュール（例：勤務時間内）と休日ポリシーを設定し、セキュリティと利便性を両立して管理できます。

1. 画面左のサイドバーで <Cmd sid='access_menu' product='air' /> → <Cmd sid='floor_levels_submenu' product='air' /> をクリックします。

2. 画面右上の <Cmd sid='floor_levels_title_add_button' product='air' /> をクリックします。

   <Image src='/img/air/bsair-elevator-mng-add-floor.png' />

3. <Cmd sid='add_floor_levels_add_floors_title' product='air' /> パネルが表示されたら、<Cmd sid='floor_levels_list_column_name' product='air' /> と <Cmd sid='add_floor_levels_description_column' product='air' />(オプション) を入力します。

   > 例：セキュリティチーム、清掃スタッフ

   <Image src='/img/air/bsair-elevator-mng-add-floor-config.png' />

4. <Cmd sid='add_floor_levels_add_floors' product='air' /> セクションで階レベルを適用する階を選択します。

5. <Cmd sid='schedules_label' product='air' /> セクションでアクセススケジュールを設定してください。

   > 例：清掃スタッフの場合は勤務時間外のみアクセスを許可します。

6. 必要に応じて、<Cmd sid='floor_levels_label_holiday_grp' product='air' /> を追加して特定の日のアクセスを制限します。

7. 階レベルを追加するには、<Cmd sid='save_btn' product='air' /> をクリックします。

:::info

階レベルの追加・編集・削除の詳細は <DocLink docId='platform/biostar_air/manage-floor-level' />を参照してください。

:::
