## BioStar Airアプリでのエアポップパッチ登録 {/* #registerApp */}

### 開始する前に

* モバイル端末にBioStar Airアプリがインストールされている必要があります。 モバイルアプリは次のリンクからインストールできます。

  * [iOS](https://apps.apple.com/us/app/airfob-pro/id1659959609)

  * [Android](https://play.google.com/store/apps/details?id=com.mocainc.afs2pro)

* BioStar Air管理者ポータルで使用するIDとパスワードでログインしてください。

* モバイル端末でBLEが有効になっていることを確認してください。

* エアポップパッチを入退室制御デバイスに恒久的または一時的に取り付けます。 詳細は<DocLink docId='platform/biostar_air/install-airfob-patch' />を参照してください。

:::info

BioStar Airモバイルアプリの詳細は、<DocLink docId='platform/biostar_air/managing-sites-with-app' />を参照してください。

:::

### エアポップパッチ登録

1. モバイル端末でBioStar Airアプリを起動してログインします。

2. BioStar Airアプリの初期画面で<Cmd sid='devices_card' product='air' />カードをタップします。

3. モバイル端末をエアポップパッチに10cm以内まで近づけ、画面右上の<IcAdd />ボタンをタップします。

4. デバイス検索画面で、登録するエアポップパッチの<Cmd sid='button.register' product='2' />ボタンをタップします。

5. デバイス登録ウィンドウが表示されたら <Cmd sid='device_btn_ok' product='air' /> ボタンをタップします。

デバイス一覧に戻り、追加されたエアポップパッチを確認します。

### カードID出力方法の設定 {/* #set-card-id-output-method */}

* **Card ID Output Method** までスクロールし、**Edit** をタップしてください。

* **User Card ID** の横にある **Select** をタップしてください。

### 出力方法を選択 {/* #choose-the-output-method */}

* **None**：カードIDを送信しない場合はこのオプションを選択してください。 バッテリー寿命を延ばすための勤怠管理(T&A)用途に有用です。

* **Fixed Device**：あらかじめプログラムされた固定カードIDを送信します。 個別にRFIDカードをプログラムする必要があるドアロックやインターホンに適しています。 機器ログにはこの固定カードIDのみが表示されます。

* **User Card ID**：各ユーザーごとに固有のカードIDを送信し、入退室管理システムの既存RFIDカード番号と一致します。 これにより、新しい物理カードを発行せずにシームレスに統合できます。

* **Direct Input**：送信するカードIDを手動で指定します。 複数のデバイスで1つのIDを使用する必要があるカスタム設定に有用です。

</Step>
<Step>

### 設定の保存 {/* #save-your-settings */}

* モバイル機器をデバイスの近くに置き、 <Cmd sid='save_btn' product='air' /> ボタンをタップして選択内容を確認してください。

* 更新完了のメッセージが表示されます。

</Step>
