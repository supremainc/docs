サイト管理者は BioStar Air サイトの運用全般を担当します。 ユーザー登録と認証情報の管理、デバイスや出入口、エレベーターの設定、アクセス権やスケジュールの設定、イベントの監視、システム設定やサービス構成など、必要なすべての管理作業を案内します。
