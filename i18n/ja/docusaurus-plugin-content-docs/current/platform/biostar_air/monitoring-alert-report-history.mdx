Learn how to check the history of alerts and reports generated in BioStar Air. Administrators can view past alerts and generated reports through the web portal and mobile app.

## Before start

The alert and report history shows only items generated by the alert and report rules configured in <Cmd sid='settings_menu' product='air' /> → <Cmd sid='alerts_history.alerts_history_submenu' product='air' />. For more information on setting rules for creating alerts and reports, refer to <DocLink docId='platform/biostar_air/settings-alert-report' />.

## Check alert and report history

On the left sidebar, click <Cmd sid='monitoring_menu' product='air' /> → <Cmd sid='alerts_history.alerts_history_submenu' product='air' />. The alert and report history appears.

<Image src='/img/air/bsair-monitoring-alert-report-history.png' />

* View the time, type, and details of each generated alert or report in the list.

* Click the date range area in the top-right corner of the screen, set the desired period, and filter the alert and report history within that period.

* Click <IcAirRef title='Refresh' /> at the top of the list to refresh the list.
