Requisitos previos​

Se requieren los siguientes requisitos previos para usar la integración de Suprema con Gallagher. Revise las condiciones de soporte antes de instalar la integración de Suprema con Gallagher.

Requisitos previos
Microsoft Windows 10 (x64) o posterior
Navegadores webNuevo Microsoft Edge
Gallagher Command CentreCommand Centre 8.80 o posterior
Suprema BioStar 2

  • 64 bits

  • Versión 2.9.5 o posterior

Software

  • Node v16.17.0

  • PostgreSQL v14.5

  • pgAdmin 4 v6.16

  • Win64OpenSSL-3.0.5

Requisitos de hardware del sistema middleware

  • 16 GB de RAM

  • Windows 10 o posterior

  • SSD de 256 GB/HDD de 1 TB

  • Intel Iris Xe Graphics (recomendado)

Instalar software de requisitos previos​

Instalación de la configuración de Gallagher Command Centre​

Información

  • No cambie ningún nombre ni contraseña durante la instalación y la configuración.

  • Para actualizar la integración de Suprema con la versión de Gallagher, haga clic aquí.

  • En concreto, puede consultar la 3C4643 Gallagher Command Centre vEL8.80 Release Note para instalaciones nuevas.

  • La 3E0068 Upgrade Procedure para fines de actualización contiene más información en la carpeta Release Notes de su directorio ISO de Command Centre.

  • Debe extraer Suprema integration With Gallagher.zip en C:\Program Files\.

  1. Dentro de la carpeta Gallagher Command Centre vEL.8.80 (o posterior) → Setup.exe, marque la casilla y presione el botón Siguiente.

  2. Haga clic en el botón "…" y ubique el archivo de licencia incluido en la configuración.

  3. Cuando seleccione el archivo de licencia correcto, el icono cambiará a una marca de verificación; después, presione el botón Siguiente.

  4. Desmarque solo la casilla de la subred y presione el botón Siguiente.

  5. Seleccione la opción Instalar nueva SQL y presione el botón Siguiente.

  6. Seleccione la opción No instalar una plantilla de base de datos.

  7. Sin cambiar ninguna opción, presione Siguiente hasta llegar a la pantalla del código de recuperación.

    Información

    Haga clic en el botón de exportación, guarde el código de recuperación y marque la casilla.

  8. Seleccione Sí y comenzará la instalación.

  9. Después de completar la instalación, presione el botón Finalizar.

Instalación de BioStar 2​

  1. Abra BioStar 2.exe, elija el idioma, acepte el acuerdo y presione Siguiente.

    Información

    Descargue la versión más reciente de BioStar 2: Haga clic aquí.

  2. Ingrese aquí su contraseña de BioStar 2 y presione el botón Siguiente.

  3. Seleccione la instalación Express e ingrese de nuevo la contraseña de BioStar 2.

  4. Seleccione la ubicación y continúe.

  5. Presione el botón Siguiente hasta llegar a la pantalla Seleccionar componentes. Marque ambas opciones y presione el botón Siguiente.

  6. Presione el botón Instalar para iniciar la instalación.

  7. Desmarque la opción Instalar USB Device Agent, presione el botón Finalizar y BioStar 2 se instalará en su PC.

    Información

    Si va a usar el dispositivo USB, debe marcar Instalar USB Device Agent.

Requisitos previos de instalación del middleware​

Información

  • Antes de extraer o ejecutar Suprema Integration with Gallagher2.0.zip, desactive el antivirus.

  • Si está actualizando desde una versión anterior, revise la solución en la guía de solución de problemas.

  1. Debe ejecutar el archivo Suprema Integration with Gallagher 2.0 como administrador dentro de la carpeta Suprema Integration with Gallagher 2.0. Esto instalará OpenSSL, PostgreSQL, PgAdmin4 y Node. JS. Además, esto configurará los certificados necesarios para nuestra integración.

  2. Presione Sí si solicita permisos para permitir desde un editor desconocido.

  3. Presione Aceptar.

  4. Ahora que está listo para instalar el software, presione Instalar.

  5. Comenzará la instalación de los requisitos previos y el software se instalará automáticamente.

  6. Presione Aceptar para la ventana emergente siguiente.

  7. Ya terminó con la instalación de los requisitos previos; presione Finalizar.

    Información

    Después de instalar el middleware, reinicie el sistema una vez.
