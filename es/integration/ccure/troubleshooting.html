Frequently Asked Questions (FAQs)

If the Suprema devices were acquired through the authorized Software House channel they should be already licensed and ready to add and use with this integration.

Please message us https://support.supremainc.com if you are a customer with an existing site:

  1. Who’s migrating from another integration.

  2. Who wants to implement this integration in your current site.

This integration is only for C•CURE or can it be used with Tyco/SH’s other access control solutions such as Kantech?

You can send an email to latam@supremainc.com and we will suggest some partner integrations you can work with.

If I already have two separate databases, one in Suprema and the other in C•CURE, how should the integration be done before installing this integration license?

The information in the C•CURE database is maintained, any replication of information in another database is handled automatically by the integration.

How many licenses does the integration system require?

You require 3 licenses: C•CURE, BioStar 2 and the Integration itself, for further information please contact your sales representative.

Are all Suprema readers supported by this Integration?

Please visit the "Supported and Unsupported Devices" of this manual.

Is the Suprema - C•CURE Integration service license permanent? i.e. does it not expire?

It’s permanent.

The license for Suprema devices, does it have a limited number of supported devices?

Up to 1,000 devices are supported.

Which versions of C•CURE 9000 does the integration support?

Version 2.9, 3.0 and 3.10.

Is C•CURE IQ Supported?

The support for C•CURE IQ is currently under development, you can expect to have a support for C•CURE IQ from Q2 2026.

Can enrollment be done from C•CURE 9000?

Exactly, the enrollment of users is done from C•CURE.

Massive enrollment features are available from the Integration CT (Advanced version).

Can I enroll from any Suprema device?

Yes, you can enroll from any supported and configured Suprema device.

Is there a way to enroll faces from an image database?

Yes, massive facial pre-enrollment is available in the Advanced version of this integration.

Can you double authenticate? Fingerprint and card, for example.

That’s correct, the authentication scheme can be adjusted for each device using BioStar 2 GUI.

When a face is added to personnel, is it saved as an image for this cardholder? So that I can use this face, for example to print a card or to appear in the Monitoring Window.

Yes, the image used for badging in C•CURE can be used to pre-enroll a facial template (biometric template), this can be done in the Advanced version only.

For facial pre-enrollment, what considerations should the photo file have, size, resolution, etc.?

An image for pre-enrollment, the picture area that covers the face of the person must have at least 250x250 pixels with a minimum resolution of 150 dpi. PNG format is preferred over JPG.

If I already have a C•CURE 9000 (v2.9, v3.0, v3.10) running and I have a Suprema apart, can I buy both licenses to integrate into an already working system?

Yes you can.

In a system running other integration, and you want to migrate to the Suprema - C•CURE Integration, should you enroll all users again?

No, ideally you can make use of your existing data, for any specific assessment please contact your sales representative.

Will C•CURE be able to accept mobile face pre-enrollment using Suprema Mobile?

Not at the moment, but it is already in the functions to be added in the next version.

If there is a customer with C•CURE and BioStar 2, could these systems be connected?

When installing the integration, it takes care to configure the devices and adjust the information for the data found in C•CURE.

In terms of data reports, whether from staff, timestamps, departments, etc., which ones are available?

The reports are available in C•CURE in the same way as they have been handled so far.

Does the current C•CURE license need to be upgraded in order for the integration to work?

Depending on the license class you hold, the specific license data needed for C•CURE 9000 will be provided by Tyco/SH.

Where can I get training to install and use this Integration and its devices?

We’re going to upload the recording to https://suprematraining.talentlms.com/.

If I have 100,000 users in C•CURE, will the same number of users be transferred to the Suprema devices or can I select the users I want in the Suprema devices?

With this integration you can deploy users in the Suprema devices according to C•CURE 9000 clearances.
