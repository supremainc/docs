Registrar puerta

Esta guía describe cómo registrar y configurar puertas. Conecte dispositivos de entrada y salida para cada puerta y configure funciones de seguridad como anti-passback y autenticación dual para establecer un entorno de gestión de acceso sistemático. Además, configure alarmas basadas en el estado de la puerta para prevenir incidentes de seguridad con anticipación.

Antes de comenzar​

  • Configure los niveles de acceso, los grupos de acceso y los niveles de piso para el control de acceso antes de registrar puertas. Para obtener más información, consulte lo siguiente.

  • Registre los dispositivos antes de registrar puertas. Para obtener más información sobre el registro de dispositivos, consulte lo siguiente.

    • Registrar dispositivo
    • Registrar credenciales Wiegand
    • Registrar esclavo

Registrar puerta​

  1. Haz clic en Settings en la página Launcher.

  2. Haz clic en Door en la barra lateral izquierda.

  3. Haga clic en ADD DOOR.

  4. Cuando aparezca la pantalla Add New Door, configure cada elemento de la sección en orden.

    • Information: Configure la información básica de la puerta. Para obtener más información, consulte lo siguiente.

    • Configuration: Configure los dispositivos conectados a la puerta, los botones de entrada y salida y los sensores de puerta. Para obtener más información, consulte lo siguiente.

    • Option: Configure opciones adicionales. Para obtener más información, consulte lo siguiente.

    • Anti PassBack: Configure la función anti-passback para gestionar el historial de acceso y mejorar la seguridad. Para obtener más información, consulte lo siguiente.

    • Timed Anti PassBack: Configure el tiempo de inicialización para la función anti-passback. Para obtener más información, consulte lo siguiente.

    • Alarm: Configure la alarma para activar o bloquear el uso del dispositivo cuando ocurra una violación de anti-passback. Para obtener más información, consulte lo siguiente.

  5. Una vez completadas todas las configuraciones, haga clic en Apply en la parte inferior de la pantalla.

Configurar información básica​

En la sección Information, ingrese o cambie el nombre, el grupo y la descripción de la puerta.

  • Name: Ingrese el nombre de la puerta. Ingrese un nombre que permita una gestión conveniente.

  • Group: Seleccione el grupo de la puerta.

  • Description: Ingrese una descripción breve de la puerta.

Información

  • El nombre de la puerta puede tener hasta 48 caracteres.

  • Para obtener más información sobre el registro de grupos de puertas, consulte lo siguiente.

Configurar la puerta​

En la sección Configuration, configure los dispositivos conectados a la puerta, los botones de salida y los sensores de puerta.

  • Entry Device: Seleccione el dispositivo que se utilizará para la entrada. Si el dispositivo no está en la lista, regístrelo primero.

  • Door Relay: Elija el relevador que funcionará como mecanismo de cierre de la puerta.

  • Exit Button: Seleccione el puerto que se utilizará como botón de salida.

    • Switch: Se puede configurar en N/C (Normalmente Cerrado) o N/O (Normalmente Abierto).

    • Does not activate relay: Configure para que una entrada del botón de salida genere un registro de solicitud de apertura de puerta sin activar el relevador.

  • Door Sensor: Elija el puerto para confirmar el estado de la puerta. Si se establece en None, se deshabilitará el uso de la sección Alarm.

    • Switch: Se puede configurar en N/C (Normalmente Cerrado) o N/O (Normalmente Abierto).

    • Use sensor when Entry Confirmed APB enabled: Configure si se utilizará el sensor de puerta al usar la opción Entry Confirmed APB.

    Si Timed Anti PassBack está habilitado, no se podrá usar la opción Use sensor when Entry Confirmed APB enabled.

  • Exit Device: Seleccione el dispositivo que se utilizará para la salida.

Información

  • Si seleccionó una cerradura de puerta inalámbrica para Entry Device, también debe seleccionar una cerradura de puerta inalámbrica para Exit Device.

  • Si seleccionó una cerradura de puerta inalámbrica para Entry Device, no podrá usar la opción Door Relay.

  • Los modelos CoreStation no se pueden usar como dispositivos de entrada ni de salida.

Configurar opciones adicionales​

En la sección Option, configure opciones adicionales para la puerta.

  • Open: Configure las opciones de apertura de la puerta.

    • Open Time: Configure la duración durante la cual la puerta permanece abierta después de completar la autenticación. La puerta se bloqueará automáticamente después de este tiempo.

    • Lock when door is closed: La puerta se bloqueará cuando el sensor detecte que se ha cerrado. Si se establece en On, no se podrá usar la opción Use Automatic Door.

    • Use Automatic Door: Si se usa una puerta automática, el relevador puede operar independientemente del estado del sensor de puerta. Si se establece en On, no se podrá usar la opción Lock when door is closed.

    Información

    El valor de Open Time puede variar según el tipo de dispositivo de cierre de puerta que se use.

  • Dual Authentication: Configura para requerir que dos personas (un usuario regular y un administrador) autentiquen credenciales para abrir la puerta.

    • Device: Selecciona el dispositivo que se usará para la autenticación dual. Device: Selecciona el dispositivo para la autenticación dual.

    • Schedule: Selecciona el horario para usar la autenticación dual. Si el horario deseado no está disponible, haz clic en + Add Schedule para agregar uno.

    • Approval Type: Define el orden de la autenticación del administrador.

      • None: Se requieren dos autenticaciones sin importar el grupo de autenticación.

      • Last: El usuario general debe autenticarse primero, seguido de un usuario autenticado incluido en el grupo de autenticación establecido.

    • Approval Group: Define el grupo al que pertenece el administrador. Esta opción está disponible cuando Approval Type se establece en Last.

    • Timeout: Define el tiempo de espera entre la primera y la segunda autenticación. Si la segunda autenticación no se completa dentro del tiempo establecido, la puerta no se abrirá.

    Información

    • Modifica la configuración en el siguiente menú para cambiar la autenticación dual del dispositivo configurado con el límite de ocupantes. Settings → Advanced AC → Occupancy Limit

      Para obtener más información sobre la configuración de Occupancy Limit, consulta lo siguiente.

    • Para obtener más información sobre la configuración de horarios, consulta lo siguiente.

  • Anti-Tailgating: Configura para detectar tailgating cuando una persona no autorizada sigue a una autorizada para entrar.

    • Sensor: Selecciona un sensor para detectar tailgating.

Configurar anti-passback​

Anti-passback se usa para gestionar el historial de acceso y mejorar la seguridad. Previene situaciones en las que un usuario entrega su tarjeta de entrada a otro después de ingresar y detiene a personas externas que siguen a usuarios con privilegios de acceso.

Consulta a continuación para completar la configuración en la sección Anti PassBack.

  • Type: Elige el tipo de anti-passback.

    • None: No utiliza la función de anti-passback.

    • Soft APB: Durante una violación de anti-passback, se permite la entrada pero se activa una alarma o se crea un registro en BioStar X.

    • Hard APB: Durante una violación de anti-passback, la entrada no está permitida y se activará una alarma o se creará un registro en BioStar X.

  • Reset Time: Define el tiempo hasta que se reinicie la función de anti-passback. Admite hasta 7 días (10080 minutos); si se establece en 0, no se reinicia.

Información

  • Funciona únicamente cuando se instalan dispositivos de entrada y salida; si en la sección Configuration la opción Exit Device se establece en None, quedará inutilizable. Para el método de configuración del dispositivo de salida, consulta lo siguiente.

  • La sección Anti PassBack requiere un dispositivo maestro configurado vía RS-485 y un dispositivo esclavo.

  • Si se habilita la sección Timed Anti PassBack, Anti PassBack no estará disponible.

Configurar anti-passback temporizado​

Cuando un usuario intenta volver a autenticarse en el mismo dispositivo después de la autenticación de ingreso, la función de anti-passback impide la reautenticación inmediata. La sección Timed Anti PassBack define el tiempo de inicialización de la función de anti-passback para limitar de forma efectiva las entradas frecuentes de los usuarios.

  • Timed APB: Selecciona el dispositivo que usará la función de anti-passback temporizado.

  • Reset Time: Define el tiempo hasta que se reinicie la función de anti-passback. La unidad de entrada es minutos (min) y el valor predeterminado es 10 minutos. Admite un máximo de 60 minutos.

  • Bypass Group: Selecciona los grupos de acceso que siempre pueden pasar sin estar sujetos al anti-passback temporizado.

Configurar alarmas​

En la sección Alarm, configura que se active una alarma o se bloquee el uso del dispositivo cuando la puerta se abra por la fuerza, quede abierta o ocurra una violación de anti-passback. Haz clic en + Add para agregar una alarma al elemento.

  • Held Open: Configura las acciones de alarma cuando la puerta quede abierta.

  • Held Open Time: Define el tiempo para que suene la alarma cuando la puerta haya estado abierta durante un periodo prolongado. Determina el tiempo máximo que la puerta puede permanecer abierta.

  • Forced Open: Configura las acciones de alarma cuando la puerta se abra por la fuerza.

  • Anti PassBack / Timed Anti PassBack: Configura las acciones de alarma para violaciones de anti-passback o anti-passback temporizado.

Información

La opción Anti PassBack está disponible únicamente si el dispositivo de salida está registrado.
