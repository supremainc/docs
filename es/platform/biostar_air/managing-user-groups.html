사용자 그룹 관리하기

Not translated
This page is currently being translated. We will complete the translation as soon as possible.

BioStar Air에서 관리자는 사용자 그룹을 통해 부서, 역할 또는 기타 분류를 기준으로 사용자를 분류할 수 있습니다. 이를 통해 출입 통제를 보다 효율적으로 구성하고 관리할 수 있습니다.

2025년 5월 이전에 BioStar Air 출입 통제 서비스를 사용하기 시작한 고객은 현재 Suprema Pass 앱으로 새로 설계된 Airfob Space 앱을 계속 사용할 수 있습니다. 두 앱 모두 동일한 기능을 제공하며 완벽하게 호환됩니다. 기존 사용자는 전환할 필요가 없으며, 모바일 인증 정보가 두 앱 모두에서 원활하게 작동합니다.

Consejo

사용자 그룹 관리에 대한 동영상 가이드를 시청하려면 다음 영상을 참고하세요.

그룹 만들기​

새로운 사용자 그룹을 만드세요.

  1. 화면 왼쪽 사이드바에서 User Management → User Groups을 클릭하세요.

  2. 화면 오른쪽 상단의 Add User Group 버튼을 클릭하세요.

  3. Add User Group 창이 나타나면 사용자 그룹 이름을 입력하세요.

  4. 그룹을 생성하려면 Save 버튼을 클릭하세요.

  5. 경고 메시지 창이 나타나면 Yes 버튼을 클릭하세요.

Información

그룹을 삭제하더라도 해당 그룹에 속한 사용자는 삭제되지 않습니다. 사용자는 그룹에서 해제될 뿐입니다.

하위 그룹 만들기​

하위 그룹을 사용해 사용자 그룹을 더 세부적으로 분류할 수 있습니다. 하지만 하위 그룹은 계층적이지 않습니다. 즉, 하위 그룹의 사용자는 상위 그룹에 자동으로 포함되지 않습니다.

  1. 화면 왼쪽 사이드바에서 User Management → User Groups을 클릭하세요.

  2. 화면 오른쪽 상단의 Add User Group 버튼을 클릭하세요.

  3. Add User Group 창이 나타나면 Add as a Subgroup 옵션을 클릭하세요.

  4. Select Parent Group에서 상위 그룹을 선택하세요.

  5. User Subgroup Name에 그룹 이름을 입력하세요.

  6. 하위 그룹을 생성하려면 Save 버튼을 클릭하세요.

생성된 하위 그룹은 사용자 그룹 목록에서 상위 그룹 아래에 들여쓰기되어 표시됩니다.

그룹에 사용자 할당하기​

사용자 그룹을 생성한 후 사용자를 추가하세요.

  1. 화면 왼쪽 사이드바에서 User Management → Users를 클릭하세요.

  2. 사용자 목록에서 앞서 생성한 그룹에 할당할 사용자의 체크박스를 클릭하세요.

  3. 사용자 목록 왼쪽 상단의 버튼을 클릭하세요.

  4. Manage User Groups 창이 나타나면 사용자를 할당할 상위 그룹 또는 하위 그룹을 선택하세요.

  5. 선택을 완료하고 그룹 설정을 저장하려면 Save 버튼을 클릭하세요.

사용자 목록의 User Group 열에서 사용자가 새 그룹에 할당된 것을 확인할 수 있습니다.

User Groups으로 돌아가면 해당 그룹의 Total Users가 업데이트된 것을 확인할 수 있습니다.

그룹 해제 또는 이동​

사용자를 그룹에서 해제하거나 다른 그룹으로 이동할 수 있습니다.

  1. 화면 왼쪽 사이드바에서 User Management → Users를 클릭하세요.

  2. 사용자 목록에서 그룹에서 해제하거나 이동할 사용자의 체크박스를 클릭하세요.

    그룹 이동 시에는 동일한 그룹의 사용자를 선택한 후 진행하세요.

  3. 사용자 목록 왼쪽 상단의 버튼을 클릭하세요.

  4. Manage User Groups 창이 나타나면 다음 중 하나를 수행하세요.

    • 그룹을 해제할 때는 선택된 그룹의 체크박스를 비활성화하세요.

    • 그룹을 이동할 때는 선택된 그룹의 체크박스를 활성화하고, 이동할 그룹을 선택하세요.

  5. 그룹 설정을 저장하려면 Save 버튼을 클릭하세요.

사용자 목록의 User Group 열에서 사용자의 그룹 변경 사항을 확인할 수 있습니다.

그룹 이름 수정​

이미 만들어진 사용자 그룹의 이름을 수정할 수 있습니다.

  1. 화면 왼쪽 사이드바에서 User Management → User Groups를 클릭하세요.

  2. 목록에서 수정할 사용자 그룹의 체크 박스를 클릭하세요.

  3. 사용자 그룹 목록 왼쪽 상단의 버튼을 클릭하세요.

  4. Edit User Group 창이 나타나면 그룹 이름을 수정하세요.

  5. 수정한 내용을 저장하려면 Save 버튼을 클릭하세요.

Información

그룹 이름 수정은 한 개의 그룹만 선택했을 때 수정할 수 있습니다.

그룹 삭제​

사용하지 않는 사용자 그룹이 있다면 삭제할 수 있습니다.

  1. 화면 왼쪽 사이드바에서 User Management → User Groups을 클릭하세요.

  2. 목록에서 삭제할 사용자 그룹의 체크박스를 클릭하세요.

  3. 사용자 그룹 목록 왼쪽 상단의 버튼을 클릭하세요.

사용자 그룹 이해하기​

사용자 그룹 구성​

사용자 그룹은 계층적으로 구성되어 있지 않습니다.

  • 하위 그룹의 사용자는 자동으로 상위 그룹에 나타나지 않습니다.

  • 상위 그룹의 총 사용자 수는 하위 그룹 구성원을 수동으로 추가하지 않는 한 포함되지 않습니다.

  • 하위 그룹을 선택하면 해당 하위 그룹에 할당된 사용자만 표시됩니다.

사용자는 여러 사용자 그룹에 포함할 수 있나요?​

BioStar Air의 사용자는 여러 사용자 그룹의 구성원이 될 수 있습니다.

BioStar 2에서 사용자 그룹은 일반적으로 단일 부서에 연결되어 있었습니다. BioStar Air는 이를 분리하여 보다 동적인 방법으로 사용할 수 있습니다.

  • 그룹별로 사용자 목록을 필터링하여 출입 레벨을 빠르게 관리할 수 있습니다.

    예, 인사부

  • 여러 부서를 아우르는 상황에 맞는 그룹을 만들 수 있습니다.

    예, 남자 탈의실

  • 사용자 그룹은 이벤트 로그 필터 및 보고서 생성에 사용됩니다.

  • 사용자 그룹을 소집 구역 장치 또는 장치 그룹에 할당하여 특정 안전 프로토콜을 적용할 수 있습니다.

이러한 유연성 덕분에 중복되는 사용자 역할 및 액세스 시나리오를 보다 쉽게 관리할 수 있습니다.
