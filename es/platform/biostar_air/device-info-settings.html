Device Information and Configuration Options

Administrators can select a registered device to view its basic information and configure operational settings such as network, authentication, and time and attendance management. Configuration options vary by device model. Expand the device entry to view the available options.

AirFob Edge​

Detalle​

The Detalle section provides basic information — such as the device name, model, serial number, and firmware version — and includes an action menu for controlling features and default settings.

Basic Information​

View the device's basic information. This information cannot be edited.

  • Producto: The product name.

  • Nombre del modelo: The device's model name.

  • Número de serie: The device's serial number.

  • Versión de Firmware: The firmware version installed on the device.

Actions​

The Action (액션) feature lets you control the device's functions.

  • Actualización de firmware: Update the device firmware to the latest version.

  • Reiniciar: Reboot the device remotely when an issue or anomaly occurs.

  • Dispositivo Reiniciado: Initialize all settings stored on the device. This action deletes all settings and data on the device.

  • Dispositivo Reiniciado Excluida la red: Initialize all settings except the network settings. This action deletes all settings and data but keeps the network settings.

Initial Settings​

Modify the device's default settings.

  • Apodo: The alias shown in the device list and logs. You can edit this field directly.

  • Zona horaria: Set the device time zone to keep logs consistent when the site and device clocks differ. Activate the Usar DST option to apply daylight saving time (DST).

  • Manos Libres: Extend the BLE range.

  • Lector QR: Confirm that an RS-485 QR reader is connected to the device.

    • When a QR reader is connected to the device, you cannot use another RS-485 device.

    • When using an SIO2, you cannot use a QR reader on the device connected to that door.

  • Tarjeta NFC/RF: Activate or deactivate card authentication. DESFire and Felica card formats are supported.
  • Configuración del pitido: Configure the success beep for access authentication.

Red​

Modify the device's network settings.

  • Configuración de la red: Configure the device to obtain an IP address automatically.

    To set the network manually, disable Configuración de la red and fill in the following fields.

  • Usar DHCP
  • Dirección IP
  • Entrada
  • Dirección DNS

T&A (Time & Attendance, T&A)​

When this feature is enabled, Evento T&A is logged in Event Log and used in Informe de tiempo y asistencia.

  • Usar T&A: Click the toggle switch to turn attendance management on or off.

  • Modo T&A: Select the desired attendance mode.

    • Por horario: Attendance events change automatically according to the set schedule. You can select a schedule to apply to attendance events.

    • Fijado: Only the selected attendance event is available.

    • Móvil: When presenting a mobile credential, the Suprema Pass app lets the user select the attendance event.

Información

For more information on configuring time and attendance, see Time & Attendance (T&A).

Salida Wiegand​

This feature integrates with external systems (e.g., alarm panels). Activate the Usar salida Wiegand option to use this feature.

  • Modo de salida: Configure the Wiegand signal output mode.

    • Regular: Scan cards using the configured Wiegand format.

      • Usar código de fallo: You can set error codes and choose the value to send when Wiegand card authentication fails.

    • Omitir: Send the CSN regardless of Wiegand authentication. Set this if you are using it as a device without door control functionality.

  • Información de salida: Select the information output during authentication.

  • Ancho de pulso: Set the pulse width for the Wiegand signal.

  • Intervalo de pulso: Set the pulse interval for the Wiegand signal.

  • Etiqueta móvil consecutiva: Tag mobile cards consecutively without wait time.

  • Límite de etiquetas consecutivas: Set the maximum number of consecutive mobile card tags.

Información

The Etiqueta móvil consecutiva feature is supported in Suprema Pass app 2.9.0 or later.

Sensibilidad​

Adjust the RSSI value to set how close the mobile device must be to unlock the door.

  • Lower values help block unintended activation.

  • Higher values improve detection in high-interference environments.

Información

To reset the settings, click Restablecer.

Módulos​

Option to detect an OM-120 module connected to the device. Click Agregar to detect the module.

Tarjeta​

In the Tarjeta section, you can modify card authentication settings.

Tarjeta NFC/RF​

  • Tarjeta NFC/RF: Click the toggle switch to enable or disable card access authentication. Select the card types supported for card authentication.

    • EM4100 (125 KHz)
    • Mifare Classic/Mifare DESFire/FeliCa
  • Orden de bytes

    • MSB: Processes card data in order from large byte units to small byte units, sequentially storing the card serial number (CSN) from left to right.

    • LSB: Processes card data in order from small byte units to large byte units, storing the card serial number (CSN).

  • Método de lectura de datos: Select CSN (default) or Wiegand format.

    Wiegand supports 5 default formats. Add more formats in Ajustes → Formato de tarjeta. For more information about custom card format settings, see Custom Wiegand card format.

Tarjetas propietarias​

For Wiegand cards, you can select iClass (SE / SEOS) and HID Prox. Click the toggle to enable the feature, then select the desired card format in Tarjetas propietarias.

Información

  • Support may vary by device.

  • Wiegand supports 5 default formats. Add more formats in Ajustes → Formato de tarjeta. For more information about custom card format settings, see Custom Wiegand card format.

  • Entrada
  • Dirección DNS

Autenticación​

  • Modo de autenticación: Add authentication methods. Click Agregar to add an authentication method.

    • XP2-MDPB / XP2-GDPB: Only RF cards can be selected as the authentication method.

    • XP2-GKDPB: You can use a combination of an RF card and a PIN as the authentication method.

  • Tiempo de espera de autenticación: Authentication fails if the credential is not authenticated within the set time.

    This feature is supported only on the XP2-GKDPB model.

Información

You can configure different authentication methods for the selected schedule. For example, you can allow card-only authentication during work hours and allow card-and-PIN authentication outside work hours.

Rostro​

Seguridad​

  • Detección de falsificaciones: Prevent user authentication with spoofed faces such as photos.

  • Comprobación de duplicados: Check for duplicate registrations when enrolling facea.

Detección​

  • Sensibilidad del Sensor de Movimiento: Set the sensitivity for detecting movement around the device.

  • Ángulo Máximo de la Postura de la Cabeza: Set the maximum rotation angle allowed for a face.

  • Búsqueda amplia: Expand the area in the camera view where faces are recognized.

  • Distancia de Detección (cm): Set the minimum and maximum recognition distance.

  • Tiempo de registro (seg): If a face is not enrolled within the set time during enrollment, the process is canceled.

Detección de mascarilla​

Activate the Detección de mascarilla option to configure mask detection.

  • Nivel de Detección de Mascarilla: Select mask detection sensitivity.

  • Si falla la detección de mascarilla

    • Denegar acceso: Users not wearing a mask are denied authentication, and a mask-non-compliance event log is recorded.

    • Permitir acceso después de iniciar sesión: Users not wearing a mask can still authenticate, but a mask-non-compliance event log is recorded.

  • Comprobación de mascarilla

    • Después del acceso: After authentication, check mask usage or measure temperature.

    • Antes del acceso: Check mask usage or measure temperature before authentication. In this case, users not wearing a mask or exceeding the threshold temperature cannot authenticate.

    • Sin acceso: Use the device only to check mask usage or measure temperature. Users wearing a mask or having a temperature below the threshold can enter regardless of authentication.

  • Autenticar QR o código de barras con la cámara del dispositivo: Set whether to allow QR/barcode authentication via the device's camera.

  • Autenticar el número de tarjeta con QR: Set authentication to accept a QR code containing the same data as the CSN card or Wiegand card issued to the user. This feature is available when the Autenticar QR o código de barras con la cámara del dispositivo option is enabled.

Información

When using non-face credentials, the timing of "before" and "after" is usually important.

    • Móvil: When presenting a mobile credential, the Suprema Pass app lets the user select the attendance event.

Información

For more information on configuring time and attendance, see Time & Attendance (T&A).

Pantalla y sonido​

Pantalla​

  • Usar protector de pantalla: Reduce LCD brightness when the device is not in use to lower unnecessary power consumption.

  • Retroiluminación siempre activa: Keep the backlight always on or dim it after inactivity.

  • Tiempo de espera de la retroiluminación: Set the time before the backlight turns off automatically.

  • Idioma: Set the language shown on the device screen.

  • Pantalla de inicio: Configure the items displayed on the device’s home screen background.

    • Normal: Display the default image on the home screen.

    • Logotipo: Display a user-registered image on the home screen.

    • Aviso: Display administrator input on the home screen.

    After clicking Actualizar and saving, the image or notice no longer appears in the edit screen but remains on the device.

Sonido​

  • Usar instrucciones de voz: Use voice guidance instead of notification tones.

  • Volumen del dispositivo: Adjust the volume of sounds generated by the device.

  • Volumen del altavoz del intercomunicador / Volumen del micrófono del intercomunicador: Adjust the output or input volume when using the IP intercom feature.

Salida Wiegand​

This feature integrates with external systems (e.g., alarm panels). Activate the Usar salida Wiegand option to use this feature.

  • Modo de salida: Configure the Wiegand signal output mode.

    • Regular: Scan cards using the configured Wiegand format.

      • Usar código de fallo: You can set error codes and choose the value to send when Wiegand card authentication fails.

    • Omitir: Send the CSN regardless of Wiegand authentication. Set this if you are using it as a device without door control functionality.

  • Información de salida: Select the information output during authentication.

  • Ancho de pulso: Set the pulse width for the Wiegand signal.

  • Intervalo de pulso: Set the pulse interval for the Wiegand signal.

  • Etiqueta móvil consecutiva: Tag mobile cards consecutively without wait time.

  • Límite de etiquetas consecutivas: Set the maximum number of consecutive mobile card tags.

Información

The Etiqueta móvil consecutiva feature is supported in Suprema Pass app 2.9.0 or later.

Intercomunicador​

  • Usar intercomunicador: Activate or deactivate the IP intercom feature.

  • Transporte del servidor SIP: Choose the SIP transport method when configuring the intercom’s SIP server.

  • Dirección del servidor SIP: Enter the SIP server address.

  • Puerto del servidor SIP: Enter the SIP server port.

  • Duración del registro: Enter the registration duration. The device attempts to register with the SIP server every time the registration duration expires.

  • Nombre de usuario SIP: Enter the SIP account username.

  • Clave: Enter the SIP account password.

  • ID de autorización: Enter the SIP account authorization ID.

  • Servidor proxy de salida: Use a separate outbound proxy server for SIP services if required.

  • Número de extensión: Add or edit extension numbers.

  • Resolución de vídeo: Select the video resolution when using the intercom.

  • Botón de apertura de puerta: Specify a button to unlock the door during a call.

Protección contra manipulaciones seguras​

If a tamper event occurs, configure the device to delete all user information, all logs, and security keys stored on the device.

RTSP​

Configure details for Real Time Streaming Protocol (RTSP) streaming. The device's camera can stream live video.

  • Usar RTSP: Turn RTSP streaming on or off.

  • Dirección: RTSP server address.

  • Puerto: Enter the RTSP server port.

  • ID / Clave: Enter the RTSP server ID and password.

  • Resolución de video RTSP: Select the video resolution when using RTSP.

Información

The RTSP streaming feature is supported only on BioStation 3.

Sensibilidad​

Adjust the RSSI value to set how close the mobile device must be to unlock the door.

  • Lower values help block unintended activation.

  • Higher values improve detection in high-interference environments.

Información

To reset the settings, click Restablecer.
