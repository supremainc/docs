Add Users

This document provides step-by-step instructions on adding and managing individual users in BioStar Air. It covers the entire process of selecting the user type (regular or temporary), issuing the appropriate credentials, and setting access levels and validity periods. It also includes how to edit or delete the added user’s information.

Consejo

Refer to the following video for step-by-step guidance on the entire process of adding a new user.

Add users​

  • Web
  • Mobile

  1. Click Gestión de usuarios → Usuarios in the left sidebar.

  2. A list of users appears.

  3. Click Añadir nuevo usuario → Añadir nuevo usuario at the top right of the screen.

When the Agregar un usuario window appears, enter and configure the required information in each section. Set the required fields in all sections and click Activar Credencial at the bottom right.

  1. Log in with the mobile app.

  2. Tap Administrar at the bottom of the home screen.

  3. Tap the Usuarios card.

  4. Tap at the top of the user list.

When the Agregar un usuario screen appears, enter and configure the required information in each section. Set the required fields in all sections and click Activar Credencial at the bottom right.

Enter user information​

Enter the new user's basic information in the Detalles del usuario section.

  • Web
  • Mobile

  • ID: Enter a specific ID manually or allow the system to generate it sequentially.

  • Idioma: Select the user's default language.

  • Grupo de usuario: Select the group to which the user belongs.

  • Nombre: Enter the user's full name.

  • Puesto: Enter the user's title.

  • Departamento: Enter the name of the user's department.

  • Foto de usuario: Upload the user's profile image. For more information, see #uploadProfileImage.

Información

  • * indicates a required field.

  • Enter the user's email and phone number in Enviar invitación por in the Detalles de la Credencial section.

Upload profile image​

  • Web
  • Mobile

  1. Hover over the profile image area and click the plus icon.

  2. Select the image file saved locally.

  3. When the Delimitar imagen window appears, adjust the photo to the desired size. Click Ajuste esta imagen to resize the image automatically.

  4. Click Guardar to save the uploaded image as the profile image.

  1. Tap the user profile image.

  2. Choose how to add an image from the menu that appears at the bottom.

    • Choose from library: Select an image file saved on your mobile device to upload.

    • Take photo: Use your mobile device's camera to take a photo and upload it.

  3. Resize the uploaded image to an appropriate size and save it.

Select user type​

  • Web
  • Mobile

Select whether the user you are adding in the Detalles de la Credencial section is a regular user or a visitor. The credential types that can be issued vary by user type.

  • Regular: A user who is a regular member of an organization or facility and requires long-term, ongoing access. All credential types can be issued.

  • Invitada: A visitor or non-regular member who needs short-term, limited access. This can include guests, external visitors, short-term contractors, delivery couriers, and so on. Only Tarjeta Web credentials, including QR code and LinkPass, can be issued.

Información

  • Regular users are automatically assigned the full access level, whereas Invitada users must manually select an access level.

  • Tipo de Usuario is supported in BioStar Air version 2.6 or higher.

Issue credentials​

Enter user information, and then set the credentials to use for access authentication in the Detalles de la Credencial section. Each credential has a toggle switch to enable or disable it.

  • Tarjeta Móvil: Use the Suprema Pass mobile app as an access authentication method. The administrator can edit the card ID number.

    • Web
    • Mobile

  • Tarjeta RF: Add up to eight RFID cards per user for access authentication. Click Agregar.

    • Web
    • Mobile

    Configure the card type (CSN or Wiegand), card ID, and card name. Select a reader in Escanear con el lector.. Alternatively, scan a physical RFID card with the connected reader.

    • Web
    • Mobile

  • Tarjeta Web: Use Web QR or Pase de enlace as access authentication methods.

    • Web
    • Mobile

    Web QR appears only after enabling Lector QR at Ajustes → Sitio → Adicionales.

  • Rostro: Add up to two face profile images for access authentication. You can upload an existing image or enroll it using a connected reader. Alternatively, email the user a link so they can enroll directly using their mobile device's camera.

    • Web
    • Mobile

    • Seleccionar Archivo: Select an image file saved locally to upload.

    • Registrar en el Dispositivo: Use the connected reader to enroll a face image.

    • Registrarse en el Móvil: Scan a QR code with a mobile device’s camera to open a link for face image enrollment.

    Tap to add a face photo. Choose how to add a face photo from the menu that appears at the bottom.

    • Choose from library: Select an image file saved on your mobile device to upload.

    • Take photo: Use your mobile device's camera to take a photo and upload it.

    • Usar como imagen de perfil: You can use the image currently set as your profile picture as your face credential.

    For more information on enrolling a face photo, see Ver Guía de Fotos.

  • PIN: Set a 4–16-digit PIN to use as part of two-factor authentication (2FA).

    • Web
    • Mobile

Send invitation link​

Email the Suprema Pass app download link to users who use Tarjeta Móvil or Tarjeta Web. Enter the email address (required) and phone number (optional).

  • Web
  • Mobile

Información

Request SMS credit recharge in Ajustes → Centro de crédito to use the Correo electrónico option.

Set access levels and validity​

After selecting credentials, configure the credential validity period, access levels, and floor levels in the Detalles de acceso section.

  • Web
  • Mobile

  • Período de validez de la credencial: Set the validity period of the credentials issued to the user. Click to leave the expiration date open-ended.

  • Nivel de acceso: Select the access levels for the user. Grant full access or choose a predefined access level.

  • Nivel del suelo: Select the floor levels the user can access.

  • Gerenciador de Dispositivos: Allow a user to manage access permissions on a specific reader.

  • Etiqueta móvil consecutiva: Allows consecutive mobile card tags without wait time. This feature is supported in Suprema Pass app version 2.9.0 or higher.

Información

  • Nivel de acceso can be assigned permanently (expires with the credential) or temporarily (expires before the credential). For more information on access-level settings, see Manage Access Levels.

  • For more information on floor-level settings, see Manage Floor Levels.

  • Gerenciador de Dispositivos option is available only to X-Station 2 users who have access to the X-Station 2 device's admin menu.

Select card template​

  • Web
  • Mobile

Click or under Card Template in the right panel to change the card template design.

Tab or on Vista previa de credenciales to change the card template design.

Información

For more information about creating new card templates, see Manage Card Templates.

Save or activate the user​

After configuring all user settings, choose one of the following options.

  • Click Guardar to save the user for later activation.

  • Click Activar Credencial to issue and activate the credentials immediately.
