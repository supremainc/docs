Primeros Pasos

La aplicación móvil BioStar X está disponible en iOS y Android. Instala la aplicación siguiendo las instrucciones y luego conéctala al servidor BioStar X.

Descargar la aplicación móvil BioStar X​

Descarga la aplicación usando los enlaces o códigos QR a continuación.

PlataformasEnlace de descargaCódigo QR
iOSApp Store
AndroidGoogle Play

Permisos de la aplicación​

La aplicación móvil BioStar X solicita los siguientes permisos cuando se usan ciertas funciones. No solicita permisos durante la instalación inicial.

PermisosObligatorioPropósito
AlmacenamientoOpcionalSe requiere para administrar y cargar imágenes de usuario, como imágenes de la galería o del almacenamiento del dispositivo.
CámaraOpcionalSe usa para capturar imágenes para el enrolamiento de credenciales faciales y la grabación de video.
BiométricoOpcionalSe requiere para iniciar sesión de forma segura con Fingerprint o autenticación facial en un dispositivo móvil.
Información

Todos los permisos son opcionales, y los usuarios pueden rechazarlos sin afectar las funciones básicas de la aplicación.

Inicio de la aplicación y configuración inicial​

Cuando se inicia la aplicación, aparece una pantalla donde ingresas la información para conectar con el servidor BioStar X.

  • Dirección del servidor BioStar X: Ingresa la dirección IP o de dominio del servidor BioStar X.

  • Puerto: Ingresa el número de puerto del servidor BioStar X. El valor predeterminado es 443.

  • ID / Contraseña: Ingresa tu ID y contraseña para iniciar sesión en el servidor BioStar X.

Después de ingresar toda la información requerida, toca Iniciar sesión para acceder.

Información

  • Para guardar el ID y la contraseña de la cuenta en los campos de entrada, selecciona la casilla Recordar nombre de usuario y contraseña. El ID y la contraseña ingresados se completan automáticamente en el siguiente inicio de sesión.

  • Para obtener información sobre la dirección IP y el número de puerto del servidor BioStar X, consulta Configuración Detallada del Servidor.

Solución de problemas de conexión​

Si el inicio de sesión falla, revisa lo siguiente:

  • Verifica que la dirección del servidor BioStar X y el número de puerto sean correctos.

  • Verifica que el ID y la contraseña de la cuenta sean correctos.

  • Verifica que el dispositivo móvil y BioStar X estén conectados a la red.

Información

El acceso desde una red externa requiere una dirección IP pública. De lo contrario, compre una licencia adicional de Acceso remoto. Para obtener más información, consulta Antes de Comenzar.

Uso de la aplicación​

Después de que el inicio de sesión se complete correctamente, aparece la pantalla principal de la aplicación móvil BioStar X. En la pantalla principal, consulta usuarios, puertas, registros en tiempo real, registros de eventos y alertas. También puedes acceder al menú lateral y al menú de ajustes.

  • : Abre el menú lateral. En el menú lateral, cambia la contraseña o cierra sesión, y accede a los menús principales.

  • : Abre el menú de ajustes. Para obtener más información sobre los ajustes de la aplicación, consulta Configurar la App.

  • USUARIO: Consulta la lista de usuarios y administra usuarios. Para obtener más información sobre la administración de usuarios, consulta Administrar Usuarios.

  • PUERTA: Consulta la lista de puertas y contrólalas de forma remota. Para obtener más información sobre la administración de puertas, consulta Administrar Puertas.

  • REGISTRO EN TIEMPO REAL: Consulta registros de eventos de acceso en tiempo real. Para obtener más información sobre la supervisión de registros en tiempo real, consulta Monitoreo de Registros en Tiempo Real.

  • REGISTRO DE EVENTOS: Consulta todos los registros de eventos almacenados en el servidor BioStar X. Para obtener más información sobre la búsqueda de registros, consulta Ver Registros de Eventos Pasados.

  • ROLLL CALL: En caso de emergencia, el usuario puede determinar si llegó al lugar de lista de asistencia al autenticarse a través de dispositivos de entrada o salida en una ubicación predesignada. La función de Roll Call está disponible mediante una opción adicional con una licencia Advanced o superior. Para obtener más información sobre la función Roll Call, consulta Primeros Pasos con Roll Call.

  • ALERTA: Consulta alertas de puertas, dispositivos y control de acceso avanzado. Para obtener más información sobre las alertas, consulta Configuración de Alertas.

  • Acción rápida: Esta función te permite controlar fácilmente varias operaciones de puerta durante una emergencia. Debes crear Acciones rápidas en la interfaz web de BioStar X antes de usarlas en la app móvil. Para obtener más información sobre la función Acción rápida, consulta Usar Acciones Rápidas.

Información

Todas las funciones también están disponibles a través del menú lateral ().
