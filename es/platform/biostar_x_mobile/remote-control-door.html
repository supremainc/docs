Control Remoto de Puertas

En la lista de puertas, revisa el estado de cada puerta en tiempo real y contrólala según sea necesario. Durante la operación normal, administra las puertas con funciones de control básicas. Usa opciones de control adicionales para mayor seguridad o emergencias.

Toca DOOR en la pantalla de inicio de la app. La lista de puertas registradas en BioStar X aparece.

Información

También puedes acceder a ella desde el menú lateral. Toca → Door.

Verifica el estado de la puerta​

En la lista de puertas, el icono de estado te permite ver de un vistazo las condiciones anómalas del dispositivo asignado a cada puerta. El icono de estado de la puerta indica si la puerta está abierta, cerrada o tiene una alarma.

Ícono de estadoDescripción
La puerta tiene un problema. La puerta está en un estado anómalo, como un dispositivo desconectado, un error del servidor de comunicación o un error de sincronización.
La puerta está abierta.
La puerta está cerrada.
Se ha producido una alarma en la puerta.
Se produjo una alarma APB en la puerta.

Abrir/Bloquear puerta​

La función de control remoto de puertas te permite abrir o bloquear la puerta para permitir o bloquear el acceso.

  • Abrir: Abre temporalmente la puerta para permitir el acceso. La puerta se cierra automáticamente después del tiempo configurado.

  • Liberar: Configura la puerta para permitir el acceso solo a usuarios autenticados. El estado de la puerta cambia según el horario configurado para la puerta.

Información

Para obtener más información sobre cómo abrir temporalmente una puerta y configurar el tiempo de cierre automático, consulta Abrir → Tiempo Apertura en Registrar Puerta.

Control adicional​

Cuando se necesita mayor seguridad o controlar el acceso durante mucho tiempo, bloquea o desbloquea la puerta manualmente sin límite de tiempo. También puedes borrar la alarma activada.

Toca en el extremo derecho de la puerta para usar opciones de control adicionales.

  • Bloqueo manual: Bloquea la puerta manualmente para impedir el acceso. Esta función bloquea la puerta sin límite de tiempo. En este estado, el acceso queda bloqueado para todos los usuarios.

  • Desbloqueo manual: Desbloquea la puerta manualmente para permitir el acceso. Esta función desbloquea la puerta sin límite de tiempo. En este estado, el acceso queda permitido para todos los usuarios.

  • Borrar alarma: Borra la alarma que se produjo en la puerta.

  • Borrar APB: Borra el estado APB para todos los usuarios en la puerta seleccionada.

Información

Después de bloquear o desbloquear una puerta manualmente, el estado de la puerta no cambia automáticamente. Por seguridad, usa Liberar para que la puerta funcione según el horario configurado.
