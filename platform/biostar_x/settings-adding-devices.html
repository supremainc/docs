장치 등록하기

BioStar X에 장치를 등록하는 방법을 안내합니다. 장치를 등록하면 BioStar X에서 해당 장치의 상태를 모니터링하고 관리할 수 있습니다. 장치를 검색하기 전에 장치가 올바르게 연결되어 있는지 확인하세요. 여러 개의 장치를 한번에 추가할 때 각 장치의 위치, ID, IP 주소 등을 미리 알아두면 편리합니다.

알아두기

  • 장치를 새로 등록할 때 장치의 키 값을 서버의 데이터 암호화 키 값으로 변경합니다. 키 변환 시 장치의 모든 사용자 데이터가 삭제됩니다.

  • 장치에 저장된 모든 설정 정보 및 사용자 정보를 삭제한 뒤 다시 전송하려면 데이터 삭제 후 장치 동기화를 클릭하세요. 데이터 삭제 후 장치 동기화 옵션은 장치 목록에서 장치를 선택하고 버튼을 클릭하면 확인할 수 있습니다.

  • 대기 장치 그룹의 모든 장치를 등록하려면 그룹 이름에서 마우스 오른쪽 버튼을 클릭하고 모든 등록 대기 장치 등록를 클릭하세요. 장치를 각각 등록하려면 장치 이름에서 마우스 오른쪽 버튼을 클릭하고 등록 대기 장치 추가를 클릭하세요.

  • BioStar X와 장치에 사용자 ID 종류가 다르게 설정되어 있다면 BioStar X의 사용자 ID 설정에 따라 장치 설정을 변경하세요.

  • BioStar X의 사용자 ID 종류가 영숫자로 설정된 경우 일부 장치를 사용할 수 없거나 제약 사항이 발생할 수 있습니다. 자세한 내용은 다음 문서를 참고하세요.

  • 장치를 등록하고 세부 정보를 설정하려면 다음 문서를 참고하세요.

시작하기 전에​

장치를 등록하기 전에 다음 사항을 확인하세요.

  • 출입 통제를 위한 출입 등급, 출입 그룹, 층 등급을 설정하세요. 자세한 내용은 다음 문서를 참고하세요.

간편 등록하기​

BioStar X와 같은 네트워크에 있는 장치를 자동으로 검색하고 등록할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 장치를 클릭하세요.

  3. 장치 목록에서 간편 검색 버튼을 클릭하세요.

  4. 간편 검색 창이 나타나면 등록할 수 있는 모든 장치 목록이 표시됩니다. 원하는 장치가 목록에 없다면 오른쪽 상단의 검색 버튼을 클릭하세요.

  5. 등록할 장치를 선택하거나 등록하지 않을 장치는 선택을 해제하세요.

  6. 등록할 장치의 이름을 변경하려면 장치 이름 열에서 버튼을 클릭한 뒤 변경하세요.

  7. 등록할 장치의 그룹을 설명하려면 장치 그룹 열에서 선택하세요.

  8. 선택한 장치에 대한 설정을 모두 완료했다면 추가 버튼을 클릭하세요.

장치 목록에 추가한 장치가 나타납니다. 추가한 장치를 선택하고 장치 동기화 버튼을 클릭하세요.

알아두기

IP 주소를 사용할 수 없거나 변경이 필요한 장치는 IP 설정 버튼을 클릭해 변경할 수 있습니다. 자세한 내용은 다음 문서를 참고하세요.

IP 주소 설정​

새로 등록하려는 장치의 IP 주소를 변경할 수 있습니다. 간편 검색 창의 왼쪽 하단에 IP 설정 버튼을 클릭하세요. IP 설정 창이 나타나면 장치 목록에서 IP 주소를 변경할 장치를 선택하세요.

  • DHCP 사용: 장치가 DHCP를 통해 IP 주소를 자동으로 할당받도록 설정합니다. 이 옵션을 선택하면 장치가 네트워크에 연결될 때마다 IP 주소가 변경될 수 있습니다.

    IP 주소와 서브넷 마스크, 게이트웨이 등을 수동으로 입력하려면 옵션을 선택 해제하세요.

  • 장치에서 서버 연결: 장치에서 BioStar X 서버로 연결하도록 설정하려면 이 옵션을 선택하세요. BioStar X를 설치한 서버의 IP 주소와 포트 번호를 입력해야 합니다.

알아두기

IP 설정을 모두 완료하고 변경 사항을 저장하려면 확인 버튼을 클릭하세요. 변경 사항을 취소하려면 취소 버튼을 클릭하세요.

장치 검색 옵션​

간편 검색 창 오른쪽 상단의 버튼을 클릭하면 장치 검색 옵션을 설정할 수 있습니다.

  • 새로 검색된 장치만 보이기: 새로 검색된 장치만 검색 목록에 표시하려면 이 옵션을 선택하세요.

  • 응답 대기시간(sec): 일정 시간 동안 응답하지 않는 장치를 검색 목록에서 제외하려면 원하는 시간을 설정하세요. 초 단위로 설정할 수 있으며, 기본값은 3초입니다.

고급 검색​

특정 장치의 IP 주소와 포트 번호를 지정하여 등록할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 장치를 클릭하세요.

  3. 장치 목록에서 고급 검색 버튼을 클릭하세요.

  4. 고급 검색 창이 나타나면 등록할 장치의 IP 주소와 포트 번호를 입력하세요.

  5. 검색 버튼을 클릭하면 입력한 조건과 일치하는 장치가 목록에 표시됩니다.

  6. 장치를 등록하려면 추가 버튼을 클릭하세요.

장치 목록에 추가한 장치가 나타납니다. 추가한 장치를 선택하고 장치 동기화 버튼을 클릭하세요.
