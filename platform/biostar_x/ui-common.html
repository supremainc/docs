공통 UI 이해하기

BioStar X의 사용자 인터페이스(User Interface, 이하 UI)는 사용자가 시스템을 쉽게 탐색하고 사용할 수 있도록 설계되었습니다. 이 문서에서는 BioStar X의 공통 UI 구성 요소를 안내합니다. 이 문서를 통해 BioStar X의 UI를 이해하고, 시스템을 보다 효율적으로 사용할 수 있습니다.

화면의 헤더 영역은 모든 페이지에서 공통으로 사용되며, BioStar X의 다양한 기능에 접근할 수 있는 링크 버튼을 제공합니다.

  1. BioStar X 로고: 사용자가 BioStar X의 런처 페이지로 돌아갈 수 있는 링크 버튼입니다.

  2. 바로가기: BioStar X에서 제공하는 주요 페이지로 이동할 수 있는 링크를 제공합니다.

  3. 액션: 서버 사용자가 설정한 액션(Action) 기능을 실행할 수 있습니다. 버튼을 클릭하면 액션 목록이 나타납니다. 자세한 내용은 퀵 액션 사용하기를 참고하세요.

  4. 서버 시간: 운영 중인 BioStar X 서버의 서버 시간을 확인할 수 있습니다. 마우스를 버튼 위로 오버하면 현재 서버 시간이 표시됩니다.

  5. 정보: 설치된 BioStar X의 버전 정보를 확인할 수 있습니다.

  6. 도움말: BioStar X의 도움말 페이지로 이동할 수 있는 링크 버튼입니다.

  7. 계정: 기본 설정 메뉴로 이동하거나 현재 로그인한 사용자의 비밀번호를 변경할 수 있습니다.

    • 기본 설정 메뉴에 대한 자세한 내용은 기본 설정 구성하기를 참고하세요.

    • 현재 로그인한 사용자의 비밀번호 변경에 대한 자세한 내용은 사용자 권한 설정하기를 참고하세요.

  8. 로그아웃: 현재 로그인한 사용자의 세션을 종료하고 로그아웃할 수 있습니다.
