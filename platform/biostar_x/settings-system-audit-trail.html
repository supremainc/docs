감사 추적 관리하기

감사 추적 기능은 시스템에서 발생하는 모든 활동을 기록하고 추적하는 기능입니다. 사용자의 로그인/로그아웃, 설정 변경, 시스템 관리 작업 등 모든 활동이 기록되어 보안 감사와 시스템 모니터링에 활용할 수 있습니다.

  • 보안 감사: 비인가 접속 시도나 권한 남용 탐지할 수 있습니다.

  • 문제 해결: 시스템 오류 발생 시점과 설정 변경 이력 추적할 수 있습니다.

  • 규정 준수: 접근 기록 보관 및 감사 보고서 작성할 수 있습니다.

감사 추적 사용하기​

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 시스템 → 감사 추적을 클릭하세요.

  3. 감사 추적 목록이 화면에 표시됩니다.

주요 추적 항목에 대한 자세한 내용은 다음 문서를 참고하세요.

추적되는 동작 유형​

필터 섹션의 동작 옵션을 클릭하면 다음과 같은 동작 유형으로 감사 추적을 분류할 수 있습니다.

  • 추가: 새로운 항목 생성

  • 수정: 기존 항목 변경

  • 삭제: 항목 제거

  • 동작: 시스템 기능 실행 (로그인, 장치 제어 등)

권한 등급별 분류​

관리자 등급에 따라 감사 추적을 분류할 수 있습니다. 필터 섹션의 관리자 등급 옵션을 클릭하면 다음과 같은 권한 등급으로 필터링할 수 있습니다.

  • 전체 관리자(1): 모든 시스템 기능 접근

  • 사용자 관리자(2): 사용자 관리 기능

  • 모니터링 관리자(3): 모니터링 기능

  • 비디오 관리자(253): 비디오 관련 기능

  • 근태 관리자(254): 근태 관리 기능

알아두기

비디오 관리자(253) 및 근태 관리자(254) 권한 등급은 어드밴스(Advanced) 라이선스 이상에서 추가 옵션을 통해 사용할 수 있습니다. 라이선스 정책에 대한 자세한 내용은 다음 문서를 참고하세요.

기간 설정 조회​

조회할 기간을 설정하여 특정 기간의 감사 추적 이력을 확인하세요.

사전 정의된 기간으로 조회​

화면 왼쪽 필터 목록에서 최근 1개월 또는 최근 3개월을 선택하세요.

사용자 정의 기간으로 조회​

상세한 기간 설정이 필요한 경우, 필터 섹션의 날짜 및 시간 옵션에서 시작 날짜와 종료 날짜를 설정하세요.

필터 설정​

감사 추적 목록을 효율적으로 탐색하기 위해 다양한 조건으로 필터링하세요.

필터 섹션에서 다음 옵션들을 설정할 수 있습니다. 필터 옵션은 감사 추적 목록의 컬럼 항목과 일치합니다. 단, 변경 내용 컬럼은 필터링할 수 없습니다.

  • 날짜 및 시간: 이벤트가 발생한 날짜와 시간

  • 사용자: 이벤트를 수행한 사용자 아이디

  • 관리자 등급: 사용자의 권한 등급

  • 접속 IP: 사용자가 접속한 IP 주소

  • 변경 항목: 변경된 항목의 메뉴 카테고리

  • 변경 대상: 변경된 대상

  • 동작: 수행된 동작

  • 변경 내용: 이벤트 발생 시의 상세 내용

필터 저장​

필터를 설정하고 필터 섹션의 필터 저장 버튼을 클릭하면 화면 왼쪽 필터 목록에 설정한 필터를 저장할 수 있습니다.

  • 필터 이름을 수정하려면 버튼을 클릭하세요. 필터 이름을 입력하고 Enter 키를 누르면 필터 이름이 변경됩니다.

  • 필터를 삭제하려면 버튼을 클릭하세요. 필터 목록에서 해당 필터가 삭제됩니다.

페이지 탐색​

페이지를 이동하거나 한 페이지에 표시할 목록 개수를 설정할 수 있습니다.

  • : 처음 페이지로 이동합니다.

  • : 이전 페이지로 이동합니다.

  • : 이동하려는 페이지 번호를 입력하고 Enter 키를 누르면 해당 페이지로 이동합니다.

  • : 다음 페이지로 이동합니다.

  • : 마지막 페이지로 이동합니다.

  • : 한 페이지에 표시할 목록 개수를 선택할 수 있습니다.

CSV 내보내기​

감사 추적 목록을 CSV 파일로 내보내 외부 분석 도구에서 활용하세요.

  1. 필터 섹션에서 필요한 경우 필터를 설정하세요.

  2. 섹션 오른쪽의 → CSV 내보내기를 클릭하세요.

  3. CSV 내보내기 창이 나타나면 내보내기 사유(선택 사항)을 입력하고 다운로드 버튼을 클릭하세요.

팁

필터링을 설정한 경우, 필터링된 결과만 CSV 파일로 내보내집니다.

열 설정​

감사 추적 목록의 컬럼의 위치를 변경하거나 숨길 수 있습니다. 필터 섹션의 가장 오른쪽에 → 열 설정을 클릭하세요.

  • 열 목록에서 숨기고 싶은 항목의 체크박스 선택을 해제하세요.

  • 열의 순서를 변경하려면 원하는 항목을 드래그하여 위치를 변경할 수 있습니다.

  • 열 설정을 초기화하려면 기본 열 버튼을 클릭하세요.

설정을 저장하려면 확인 버튼을 클릭하세요.
