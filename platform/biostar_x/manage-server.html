    서버 관리하기

    BioStar X 서버의 상태를 관리하는 방법을 안내합니다. 실행 중인 서비스를 종료하거나 다시 시작할 수 있습니다.

    BioStar X Setting​

    BioStar X 서버의 상태를 관리할 수 있는 도구를 제공합니다. 시작 → BioStar X → BioStar X Setting을 클릭하세요. BioStar X Setting 창이 나타납니다.

    알아두기

    BioStar X 서버의 시간 설정을 변경했다면 Core Web Server 서비스를 종료하고 다시 시작하세요. 다시 시작하지 않으면 BioStar X가 동작하지 않을 수 있습니다.

    서비스 종료하기​

    BioStar X를 종료하려면, Service Status 섹션에서 개별 서비스의 Stop 버튼을 클릭하세요.

    서버 시작하기​

    BioStar X를 다시 시작하려면, Service Status 섹션에서 Stopped 상태인 서비스의 Start 버튼을 클릭하세요.
