카드 인쇄하기

BioStar X와 cardPresso를 연동하여 BioStar X에서 사용자가 원하는 디자인의 카드를 인쇄할 수 있습니다.

cardPresso 설치 및 설정하기​

BioStar X의 카드 프린터 기능을 사용하려면 cardPresso 프로그램을 아래와 같이 설치 및 설정하세요.

  1. cardPreso를 이용하여 카드를 인쇄하려는 PC에 최신 버전의 cardPresso를 다운로드하세요. 다운로드 경로는 다음 링크를 참고하세요.

  2. 프로그램을 설치하고 cardPresso에서 발급한 라이선스를 활성화하세요.

    • 라이선스 유형: cardPresso XXL edition

    • BioStar X에서 지원하는 속성 ID는 아래와 같습니다. 카드 템플릿 생성 시, 속성 ID를 알맞은 XML 항목 ID 이름으로 입력하세요.

    목록BioStar X 사용자 데이터XML 항목 ID 이름
    1User IDID
    2User NameNAME
    3EmailEMAIL
    4DepartmentDEPARTMENT
    5TelephoneTELEPHONE
    6User GroupGROUP
    7Profile PhotoPHOTO
    8Custom Field 1CUSTOM1
    9Custom Field 2CUSTOM2
    10Custom Field 3CUSTOM3
    11Custom Field 4CUSTOM4
    12Custom Field 5CUSTOM5
    13Custom Field 6CUSTOM6
    14Custom Field 7CUSTOM7
    15Custom Field 8CUSTOM8
    16Custom Field 9CUSTOM9
    17Custom Field 10CUSTOM10

  3. cardPresso 웹 인쇄 서버에서 사용할 카드 템플릿의 저장 경로를 설정한 후 저장을 클릭하세요.

  4. 바탕 화면에 생성된 cardPresso 아이콘을 마우스 우클릭하여 바로 가기 만들기를 실행하세요.

  5. 생성된 바로가기 아이콘에 마우스 우클릭하여 속성을 선택하세요. 바로가기 탭의 대상 항목 끝부분에 한 칸 띄우고 /PRINTSERVER를 추가한 다음 확인을 클릭하세요. 이 바로가기가 웹 인쇄 서버 바로가기가 됩니다.

  6. 수정한 cardPresso 웹 인쇄 서버 바로가기 아이콘을 더블 클릭하여 실행한 뒤 각 항목을 설정하세요.

    • 주소: 웹 인쇄 서버로 동작하는 PC의 IP 주소를 입력하세요.

    • 포트: cardPresso에서 인쇄 동작을 수신하는데 사용하는 포트 번호입니다.

    • 허용된 사용자 그룹: 웹 인쇄 서버에 접근할 권한이 있는 사용자 그룹을 선택하세요.

  7. 설정이 완료되면 저장을 클릭하세요.

  8. 우측 하단부에 있는 을 클릭하여 서비스를 시작하세요. 알맞게 설정된 경우 아이콘이 로 변경되고 listen OK 메시지가 나타납니다.

    • 카드 인쇄가 완료되기 전까지 창을 닫지 마세요.

cardPresso를 BioStar X에 연결하기​

cardPresso를 BioStar X에 연결하려면 BioStar X를 아래와 같이 설정하세요.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 카드 프린터를 클릭하세요.

  3. cardPresso 설정을 사용으로 설정한 뒤 각 항목을 입력하세요.

    • ID: cardPresso 로그인 ID를 입력하세요.

    • 암호: cardPresso 로그인 암호를 입력하세요.

    • IP 주소: 웹 인쇄 서버로 동작하는 PC의 IP 주소를 입력하세요.

    • Port: cardPresso에서 인쇄 동작을 수신하는데 사용하는 포트 번호입니다.

    • 프린터 이름: 카드 프린터로 사용할 프린터 이름을 입력하세요. Windows의 제어판 → 장치 및 프린터에서 확인할 수 있습니다.

    • 카드 템플릿: + 추가를 클릭해 카드 템플릿을 추가할 수 있습니다. 이름은 카드 템플릿의 파일명과 확장자가 포함된 전체 경로를 입력하세요.

      • 카드 템플릿의 파일명은 영문, 숫자, 특수 기호로 설정할 수 있습니다.

      • 최대 20개의 카드 템플릿을 지원합니다.

      • 경로 예시: C:\template\example.card

    • 카드 템플릿 테스트 인쇄: 카드 템플릿을 테스트 인쇄할 수 있습니다.

  4. 설정이 완료되면 확인을 클릭하세요.

카드 템플릿 테스트 인쇄하기​

Administrator 정보를 가진 카드 템플릿을 테스트 인쇄할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 카드 프린터를 클릭하세요.

  3. 인쇄하려는 카드 템플릿을 선택한 후, 테스트 인쇄를 클릭하세요.

  4. Administrator 정보를 가진 카드 템플릿이 인쇄됩니다.

사용자별 카드 템플릿 인쇄하기​

사용자별 정보를 가진 카드 템플릿으로 카드를 인쇄할 수 있습니다.

  1. 런처 페이지에서 사용자를 클릭하세요.

  2. 화면 오른쪽 상단의 새 사용자 버튼을 클릭하세요. 새 사용자 창이 나타납니다.

  3. 양식을 입력하고 카드 인쇄를 클릭하세요.

    • 사용자 정보는 영문, 숫자, 특수 기호로 입력할 수 있습니다.

  4. 인쇄하려는 카드 템플릿을 선택하고 인쇄를 클릭하세요.

  5. 카드 인쇄 여부를 묻는 팝업창이 나타나면 확인을 클릭하세요.

  6. 새 사용자가 생성되고 사용자 정보를 가진 카드 템플릿으로 카드가 인쇄됩니다.

커스텀 사용자 필드 추가하기​

커스텀 사용자 필드를 추가한 카드 템플릿으로 카드를 인쇄할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 서버를 클릭하세요.

  3. 커스텀 사용자 필드 항목에서 + 추가를 클릭해 이름을 입력하고 유형을 선택하세요. 추가한 커스텀 사용자 필드의 순서대로 카드 템플릿의 속성 ID가 결정됩니다.

    • BioStar X 속성 ID 기준으로 순서 1은 CUSTOM1과 같습니다. BioStar X 속성 ID에 대한 자세한 내용은 다음 문서를 참고하세요.

  4. 설정이 완료되면 확인을 클릭하세요.

  5. cardPresso 프로그램에서 파일 → 템플릿 열기를 클릭해 카드 템플릿 파일을 여세요. 새로운 속성을 추가하고 BioStar X 속성 ID 순서에 알맞은 XML 항목 ID 이름으로 ID를 입력하세요.

  6. 설정이 완료되면 파일 → 저장을 클릭하세요.

  7. BioStar X 사용자 페이지에서 카드 템플릿을 적용할 사용자를 선택하세요. 커스텀 사용자 필드 항목에서 추가한 항목들이 나타납니다. 양식을 입력하고 카드 인쇄를 클릭하세요.

  8. 인쇄하려는 카드 템플릿을 선택하고 인쇄를 클릭하세요.

  9. 카드 인쇄 여부를 묻는 팝업창이 나타나면 확인을 클릭하세요.

  10. 사용자 정보가 저장되고 커스텀 사용자 필드를 추가한 카드 템플릿으로 카드가 인쇄됩니다.
