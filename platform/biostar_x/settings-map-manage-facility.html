시설 구성하기

시설을 구성하고 층을 배치하세요. 층을 순차적으로 배치하여 실제 시설의 구조를 표현할 수 있습니다. 시설을 구성하면 모니터링 페이지에서 지도에 배치된 시설을 선택하고 각 층으로 진입하여 모니터링할 수 있는 기능을 제공합니다.

알아두기

  • 시설을 구성하기 전에 층을 먼저 등록하세요. 층 등록에 대한 자세한 내용은 다음 문서를 참고하세요.

  • 맵 모니터링에 대한 자세한 내용은 다음 문서를 참고하세요.

  • 맵 설정 및 맵 모니터링 기능은 어드밴스 이상의 라이선스에서만 사용할 수 있습니다.

시설 추가​

시설에 층을 추가하여 구조를 설계하려면 다음 단계를 따르세요.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 맵 → 시설을 클릭하세요.

  3. 시설 목록 화면 오른쪽 상단의 New Facility 버튼을 클릭하세요.

  4. Create New Facility 화면의 정보 섹션에서 추가하려는 시설의 이름과 설명을 입력하세요.

  5. 시설 설정에서 추가 버튼을 클릭하세요.

  6. Add Floor 창이 나타나면 추가할 층의 체크박스를 클릭해 층을 선택하세요.

    • 입력 필드에 키워드를 입력해 층을 검색할 수도 있습니다.

    • 모든 층을 선택하려면 Select All 버튼을 클릭하세요.

  7. 층을 모두 선택했다면 Apply 버튼을 클릭하세요.

  8. 출입문 목록에 추가한 층을 확인하고 순서를 변경하려면 Order 열의 아이콘을 드래그하여 순서를 변경하세요.

  9. 시설 구성을 완료하고 저장하려면 적용 버튼을 클릭하세요.

시설의 층 추가​

이미 만들어진 시설에 층을 추가할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 맵 → 시설을 클릭하세요.

  3. 시설 목록에서 층을 추가할 시설을 선택하세요.

  4. 시설 설정에서 추가 버튼을 클릭하세요.

  5. Add Floor 창이 나타나면 추가할 층의 체크박스를 클릭해 층을 선택하세요.

  6. 층을 모두 선택했다면 Apply 버튼을 클릭하세요.

  7. 출입문 목록에 추가한 층을 확인하고 순서를 변경하려면 Order 열의 아이콘을 드래그하여 순서를 변경하세요.

  8. 시설 구성을 완료하고 저장하려면 적용 버튼을 클릭하세요.

시설의 층 삭제​

이미 만들어진 시설에 층을 삭제할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 맵 → 시설을 클릭하세요.

  3. 시설 목록에서 층을 삭제할 시설을 선택하세요.

  4. 시설 설정에서 삭제할 층의 버튼을 클릭하세요.

  5. 층 삭제를 완료하고 저장하려면 적용 버튼을 클릭하세요.

  3. 시설 목록에서 삭제할 시설의 체크박스를 클릭하세요.
