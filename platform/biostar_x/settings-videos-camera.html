카메라 설정하기

VMS 연동을 완료한 후에는 BioStar X에서 사용할 카메라를 선택하고 설정해야 합니다. 적절한 카메라 설정을 통해 다음과 같은 보안 관리 기능을 활용할 수 있습니다.

  • 선택적 모니터링: VMS의 모든 카메라 중 필요한 카메라만 BioStar X에 추가할 수 있습니다.

  • 효율적인 관리: 카메라를 그룹별로 구성하여 위치나 용도에 따른 체계적으로 관리할 수 있습니다.

알아두기

VMS 서버와 연동하는 방법은 다음 문서를 참고하세요.

카메라 설정 및 관리​

VMS 연동 후 설정 → 비디오 → 비디오 메뉴에서 카메라를 설정하고 관리할 수 있습니다.

카메라 추가​

VMS와 연동되어 있다면 VMS 서버에 연결된 카메라를 BioStar X에 추가하고 모니터링 페이지에서 실시간 영상 및 녹화 영상을 재생할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 비디오 → 비디오를 클릭하세요.

  3. 카메라 추가 버튼을 클릭하세요.

  4. 카메라 추가 창이 나타나면 카메라 목록에서 제외할 카메라를 선택하세요.

    원하는 카메라가 표시되지 않는다면 새로고침 버튼을 클릭해 카메라 목록을 새로고침하세요.

  5. 카메라를 모두 선택했으면 화면 오른쪽 하단의 추가 버튼을 클릭하세요.

카메라 목록에서 추가한 카메라를 확인할 수 있습니다.

카메라 정보 수정​

추가한 카메라의 정보를 수정할 수 있습니다. 카메라 이름 및 그룹, 설명 등을 변경할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 비디오 → 비디오를 클릭하세요.

  3. 화면 왼쪽 상단의 카메라 탭을 클릭하세요.

  4. 카메라 목록에서 정보를 수정할 카메라를 클릭하세요.

  5. 카메라 정보가 표시된 화면이 나타나면 원하는 항목을 수정하세요.

  6. 카메라 정보를 모두 수정했으면 화면 오른쪽 하단의 확인 버튼을 클릭하세요.

알아두기

카메라 아이디와 IP 주소는 수정할 수 없습니다.

카메라 삭제​

추가한 카메라를 삭제할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 비디오 → 비디오를 클릭하세요.

  3. 카메라 목록에서 삭제할 카메라의 가장 왼쪽에 체크박스를 클릭하세요. 1대 이상의 카메라를 선택할 수 있습니다.

  4. 화면 오른쪽 상단의 카메라 삭제 버튼을 클릭하세요.

  5. 메시지를 확인하고 네 버튼을 클릭하세요.

알아두기

삭제하려는 카메라가 하나의 규칙에 포함되어 있다면 카메라를 삭제할 수 없습니다. 해당 카메라를 규칙에서 제거한 후 다시 시도하세요. 규칙 생성 및 관리에 대한 자세한 내용은 다음 문서를 참고하세요.

카메라 그룹 설정 및 관리​

카메라 그룹 추가​

카메라 그룹을 추가해 여러 대의 카메라를 그룹으로 묶어 관리할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 비디오 → 비디오를 클릭하세요.

  3. 화면 왼쪽 상단의 카메라 탭을 클릭하세요.

  4. 모든 카메라 항목을 선택하고 마우스 오른쪽 버튼을 클릭하세요.

  5. 팝업 메뉴에서 카메라 그룹 추가를 클릭하세요.

  6. 원하는 그룹 이름을 입력하고 Enter 키를 누르세요.

알아두기

  • 카메라 그룹은 8단계의 하위 그룹까지 생성할 수 있습니다.

  • 카메라 그룹 이름은 최대 48자까지 입력할 수 있습니다.

  • 카메라 목록에서 카메라 그룹을 선택하면 해당 그룹에 속한 카메라만 목록에 표시됩니다.

그룹에 여러 대의 카메라 추가​

추가한 그룹에 1대 이상의 카메라를 추가할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 비디오 → 비디오를 클릭하세요.

  3. 카메라 목록에서 그룹에 추가할 카메라의 가장 왼쪽에 체크박스를 클릭하세요. 1대 이상의 카메라를 선택할 수 있습니다.

  4. 1대 이상의 카메라를 선택하면 화면 오른쪽 상단의 일괄 편집 버튼이 활성화됩니다. 일괄 편집 버튼을 클릭하세요.

  5. 일괄 편집 창이 나타나면 그룹 항목의 버튼을 클릭하세요.

  6. 그룹 목록에서 원하는 그룹을 선택하세요.

  7. 확인 버튼을 클릭하세요.

알아두기

카메라 탭의 카메라 목록에서 원하는 항목을 선택하고 드래그 앤 드롭으로도 카메라를 그룹에 추가할 수 있습니다.

카메라 그룹 이름 수정​

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 비디오 → 비디오를 클릭하세요.

  3. 카메라 탭을 클릭하세요.

  4. 이름을 변경할 그룹을 선택하고 마우스 오른쪽 버튼을 클릭하세요.

  5. 팝업 메뉴에서 카메라 그룹 이름 변경을 클릭하세요.

  6. 원하는 그룹 이름을 입력하고 Enter 키를 누르세요.

카메라 그룹 삭제​

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 비디오 → 비디오를 클릭하세요.

  3. 카메라 탭을 클릭하세요.

  4. 이름을 변경할 그룹을 선택하고 마우스 오른쪽 버튼을 클릭하세요.

  5. 팝업 메뉴에서 카메라 그룹 삭제를 클릭하세요.

  6. 경고 메시지를 확인하고 네 버튼을 클릭하세요.

알아두기

카메라 그룹에 카메라가 포함되어 있다면 그룹을 삭제할 수 없습니다. 그룹에 속한 카메라를 다른 그룹으로 이동하거나 그룹에서 제거한 후 다시 시도하세요.

목록 열 설정​

목록에서 표시할 열을 추가하거나 순서 변경, 숨김 설정을 할 수 있습니다.

  1. 런처 페이지에서 설정을 클릭하세요.

  2. 화면 왼쪽 사이드바에서 비디오 → 비디오를 클릭하세요.

  3. 모든 카메라 목록의 오른쪽 상단에서 → 열 설정을 클릭하세요.

  4. 열 설정 창이 나타나면 표시할 열의 체크박스를 클릭해 추가하거나 숨김 설정을 할 수 있습니다. 드래그 앤 드롭으로 열의 순서를 변경할 수도 있습니다.

  5. 열 설정을 모두 변경했으면 확인 버튼을 클릭하세요.

알아두기

열 설정을 초기화하려면 기본 열 버튼을 클릭하세요.
