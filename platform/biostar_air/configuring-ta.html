근태 관리(T&A) 구성하기

BioStar Air는 교대 근무 및 휴식 시간을 기록하여 기본적인 근태 관리(T&A) 추적 기능을 제공하며, 외부 근태 관리 시스템과 연동하거나 내보낼 수 있습니다. BioStar Air는 완전한 근태 관리 시스템을 제공하지는 않지만, 이벤트 로깅 및 급여, 근태 관리 시스템과의 API 연동을 지원합니다. 또한 기본적인 근무 시간 보고서도 생성할 수 있습니다.

팁

근태 관리 구성에 대한 동영상 가이드를 시청하려면 다음 영상을 참고하세요.

근태 이벤트​

사용자는 아래와 같은 근태 이벤트를 기록할 수 있습니다. 근태 이벤트는 이벤트 로그에 기록되며 CSV 파일로 내보내거나 API를 통해 검색할 수 있습니다.

  • 교대 시작

  • 교대 종료

  • 휴식 시작

  • 휴식 종료

근태 관리 설정​

근태 관리를 위한 설정을 활성화하고 세부 설정을 구성하려면 다음 단계를 따르세요.

  1. 화면 왼쪽 사이드바에서 설정 → 사이트를 클릭하세요.

  2. 화면 상단의 설정 탭을 클릭하세요.

  3. 모바일 근태관리 기능을 활성화하세요.

  4. 메시지 창이 나타나면 확인 버튼을 클릭하세요.

  5. 모바일 근태관리 기능에서 설정 버튼을 클릭하세요.

  6. 모바일 근태관리 창이 나타나면 근태 키에 대한 라벨을 설정하세요.

    예, 교대 시작, 교대 종료, 휴식 시작, 휴식 종료

  7. 근태 키에 대한 이벤트 종류가 Clock-In 또는 Clock-Out인지 선택하세요.

  8. 설정을 저장하려면 저장 버튼을 클릭하세요.

장치에서 근태 기능 설정​

모바일 근태관리 기능을 활성화하면 이를 사용할 장치를 지정할 수 있습니다.

  1. 화면 왼쪽 사이드바에서 장치 관리 → 등록된 장치를 클릭하세요.

  2. 장치 목록에서 근태 관리 기능을 설정할 장치를 클릭하세요.

  3. 장치 정보 수정 창이 나타나면 근태 섹션으로 이동하세요.

  4. 근태 사용 옵션의 토글 버튼을 클릭해 근태 관리 기능을 활성화하세요.

  5. 설정을 저장하려면 저장 버튼을 클릭하세요.

  • 근태 모드를 모바일로 설정하면, 출입 인증 시 근태를 필수로 선택할 수 있는 근태 필수 선택 옵션을 사용할 수 있습니다.

  • 근태 이벤트 라벨은 사이트 등급에서 관리되며 장치별로 변경할 수 없습니다.

근태 모드​

장치는 다양한 방법으로 근태 이벤트를 기록할 수 있습니다. 근태 모드에서 원하는 근태 관리 모드를 선택하세요.

  • 수동 변경: 화면에 근태 키 표시해 인증 시, 사용자가 근태 이벤트를 선택할 수 있습니다.

  • 자동 변경: 설정된 일정에 따라 출근 및 퇴근 시간을 기록합니다. 예를 들어, 8:00 AM - 12:00 PM 사이에 기록된 이벤트는 Clock-In으로, 1:00 PM - 5:00 PM 사이에 기록된 이벤트는 Clock-Out으로 기록됩니다.

  • 수동 고정: 사용자가 마지막으로 사용한 근태 이벤트를 계속 사용할 수 있습니다.

  • 고정: 고정된 근태 코드를 장치에 할당합니다. 해당 장치에서 인증할 때마다 동일한 근태 이벤트가 기록됩니다. 이 옵션을 통해 해당 장치를 출근용 또는 퇴근용 장치로 사용할 수 있습니다.

  • 모바일: 사용자가 인증하기 전에 근태 코드를 선택해야 합니다. 선택한 근태 이벤트는 출입 로그와 함께 기록됩니다.

알아두기

장치 모델에 따라 지원되는 근태 관리 모드가 다를 수 있습니다. 자세한 내용은 장치 및 패널 설정 옵션 안내에서 장치별 근태 모드를 참고하세요.

근태 로그 보기 및 내보내기​

근태 이벤트는 이벤트 로그에 기록되며, 이를 검토하거나 CSV 파일로 내보낼 수 있습니다.

  1. 화면 왼쪽 사이드바에서 모니터링 → 이벤트 로그를 클릭하세요.

  2. 로그 테이블에서 T&A 열(column)을 찾으세요.

  3. 로그 테이블 상단의 버튼을 클릭해 사용자 정의하여 T&A 열을 더 잘 보이도록 이동하세요.

  4. 데이터를 CSV 파일로 다운로드하려면 버튼을 클릭하고, CSV 다운로드를 선택하세요.
