개별 사용자 추가하기

서비스 개요​

관리자는 개별 사용자를 BioStar Air에 추가하고, 자격 증명과 액세스 등급을 지정하여 원활한 사이트 출입을 관리할 수 있습니다. 사용자는 유형에 따라 모바일 또는 RFID, 웹 기반 인증을 받을 수 있습니다.

2025년 5월 이전에 BioStar Air 출입 통제 서비스를 사용하기 시작한 고객은 현재 Suprema Pass 앱으로 새로 설계된 Airfob Space 앱을 계속 사용할 수 있습니다. 두 앱 모두 동일한 기능을 제공하며 완벽하게 호환됩니다. 기존 사용자는 전환할 필요가 없으며, 모바일 인증 정보가 두 앱 모두에서 원활하게 작동합니다.

User Management 메뉴로 이동​

개별 사용자를 추가하세요.

  1. BioStar Air 포털에서 User Management로 이동하세요.

  2. Users를 클릭하면 사용자 목록이 나타납니다.

  3. Add User 버튼을 클릭하세요.

사용자 정보 입력​

새로운 사용자를 추가할 때 다음 항목을 입력하세요.

  • User ID: 특정 ID를 할당하거나 시스템이 순차적으로 생성하도록 허용할 수 있습니다.

  • Language: 사용자의 기본 언어를 선택합니다 (English, Korean, Japanese, Dutch, Spanish, 또는 Portuguese).

  • User Group: 사용자를 관련 그룹에 할당합니다.

  • Name: 사용자의 전체 이름을 입력합니다 (필수).

  • Additional Information(optional): 직책, 부서 및 주소를 입력하세요.

  • Contact Information: Provide an email or mobile phone number for the invitation.

프로필 이미지 업로드​

  1. Upload를 클릭하고 이미지 파일을 선택하세요.

  2. 사용 가능한 도구를 사용하여 이미지를 조정하세요.

  3. Fit this image를 클릭한 다음 Save를 선택하세요.

사용자 유형 선택​

BioStar Air 버전 2.6부터 사용자는 다음과 같이 분류할 수 있습니다:

  • Regular Users: 모바일 카드, QR 코드, RFID 카드, 얼굴 인증, PIN 등 지원되는 모든 크리덴셜 유형을 받을 수 있습니다.

  • Temporary Users: Dynamic QR 코드와 LinkPass를 포함한 Web Card 크리덴셜만 받을 수 있습니다.

정규직 사용자는 자동으로 전체 액세스 등급이 할당될 수 있는 반면, 임시 사용자는 수동으로 액세스 등급을 선택해야 합니다.

인증 할당​

Credential Details에서 관리자는 사용자에게 할당할 크리덴셜 유형을 결정할 수 있습니다. 각 크리덴셜 유형에는 활성화 또는 비활성화할 수 있는 토글 스위치가 있습니다.

크리덴셜 유형​

  • Mobile Card: Suprema Pass 모바일 크리덴셜입니다. 관리자는 카드 ID 번호를 편집할 수 있습니다.

  • RF Card: 사용자당 최대 8개의 RFID 카드를 추가할 수 있습니다. 카드 유형(CSN / Wiegand), 카드 ID 번호 및 카드 이름을 구성하여 여러 카드를 쉽게 관리할 수 있습니다. Scan with Reader를 클릭하고 리더기를 선택하세요. 연결된 리더기를 사용하여 실물 RFID 카드를 스캔할 수도 있습니다.

  • Web Card: LinkPass 또는 Dynamic QR입니다.

    Dynamic QR은 사이트 설정에서 QR이 활성화된 경우에만 나타납니다.

  • Face: 얼굴 프로필 이미지를 최대 2개까지 추가할 수 있습니다. 기존 이미지를 업로드하거나 리더를 통해 등록할 수 있습니다. 또는 이메일로 링크를 보내서 사용자가 모바일 기기의 카메라를 사용해 직접 등록하는 방법도 있습니다.

  • PIN: 4~16자리 PIN을 설정하여 2단계 인증(2FA)의 일부로 사용할 수 있습니다.

액세스 등급 및 유효성 설정​

인증을 선택한 후, 사용자의 액세스 권한을 구성하세요.

  1. Access Level (전체 액세스 또는 미리 정의된 등급)을 선택하세요.

  2. 사용자가 할당된 Floor Levels를 선택하세요.

  3. Credential Validity Period를 설정하세요.

    • 시작 날짜(필수)와 종료 날짜(선택 사항)를 설정하세요.

    • 시작 날짜를 설정한 후에는 일반적인 종료 날짜에 대한 단축 버튼을 사용할 수 있습니다.

알아두기

출입 등급은 영구 할당(크리덴셜과 함께 만료)하거나 일시적으로 할당(크리덴셜 보다 일찍 만료)할 수 있습니다.

팁

Device Manager 옵션은 X-Station 2 사용자만 사용할 수 있으며, 관리자 메뉴를 사용할 수 있습니다.

카드 템플릿 선택​

카드 템플릿을 할당하려면, 드롭다운 메뉴를 클릭하고 Card Template을 선택하세요.

사용자 저장 또는 활성화​

모든 설정이 구성되면, 다음 중 하나를 선택할 수 있습니다.

  • 사용자를 나중에 활성화하려면 저장하세요.

  • Activate Card를 클릭하고 크리덴셜을 즉시 발급하세요.

사용자 세부 정보 수정​

사용자를 추가하고 세부 정보를 수정할 수 있습니다.

  1. 목록에서 사용자를 선택하세요.

  2. 세부 정보를 확인하려면 이름을 클릭하세요.

  3. 필요한 세부 정보를 수정하세요.
