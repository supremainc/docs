사용자 추가하기

이 문서는 BioStar Air에서 개별 사용자를 추가하고 관리하는 방법을 단계별로 안내합니다. 사용자 유형(일반/임시)을 선택하고, 각 유형에 맞는 크리덴셜을 발급하며, 출입 레벨과 유효 기간을 설정하는 전체 과정을 다룹니다. 추가한 사용자의 정보를 수정하거나 삭제하는 방법도 포함합니다.

팁

새로운 사용자 추가에 대한 전체 과정을 단계별로 안내하는 다음 동영상을 참고하세요.

사용자 추가 시작​

  • Web
  • Mobile

Web

  1. 화면 왼쪽 사이드바에서 사용자 관리 → 사용자를 클릭하세요.

  2. 사용자 목록이 나타납니다.

  3. 화면 오른쪽 상단의 사용자 추가 → 사용자 추가를 클릭하세요.

사용자 추가 창이 나타나면 각 섹션마다 필요한 정보를 입력하고 설정하세요. 모든 섹션의 필수 항목을 설정하고 오른쪽 하단의 크리덴셜 활성화 버튼을 클릭하세요.

Mobile

  1. 모바일 앱으로 로그인하세요.

  2. 초기 화면의 하단에서 관리를 탭하세요.

  3. 사용자 카드를 탭하세요.

  4. 사용자 목록 화면 상단의 버튼을 탭하세요.

사용자 추가 화면이 나타나면 각 섹션마다 필요한 정보를 입력하고 설정하세요. 모든 섹션의 필수 항목을 설정하고 오른쪽 하단의 크리덴셜 활성화 버튼을 클릭하세요.

사용자 정보 입력​

사용자 정보 섹션에서 새롭게 추가하는 사용자에 대한 기본 정보를 입력하세요.

  • Web
  • Mobile

Web

Mobile

  • ID: 특정 아이디를 직접 입력하거나 시스템이 순차적으로 생성하도록 허용할 수 있습니다.

  • 언어: 사용자의 기본 언어를 선택하세요.

  • 사용자 그룹: 사용자가 속할 그룹을 선택하세요.

  • 사용자 이름: 사용자의 전체 이름을 입력하세요.

  • 직함: 사용자의 직함을 입력하세요.

  • 부서: 사용자가 속한 부서 이름을 입력하세요.

  • 사용자 이미지: 사용자의 프로필 이미지를 업로드할 수 있습니다. 자세한 내용은 #uploadProfileImage를 참고하세요.

알아두기

  • * 표시는 필수 입력 항목입니다.

  • 사용자의 이메일과 전화번호는 크리덴셜 정보 섹션의 초대 링크 전송 방법 항목에서 입력할 수 있습니다.

프로필 이미지 업로드​

  • Web
  • Mobile

Web

  1. 프로필 이미지 영역에 마우스를 오버하고 플러스 버튼을 클릭하세요.

  2. 로컬에 저장된 이미지 파일을 선택하세요.

  3. 이미지 자르기 창이 나타나면 원하는 크기로 사진을 조정하세요. 자동으로 이미지 크기를 맞추려면 사이즈 맞춤 옵션을 클릭하세요.

  4. 업로드한 이미지를 프로필로 저장하려면 저장 버튼을 클릭하세요.

Mobile

  1. 사용자 프로필 이미지를 탭하세요.

  2. 하단에 나타나는 메뉴에서 원하는 방법으로 이미지를 추가하세요.

    • 사진 선택하기: 모바일 기기에 저장된 이미지 파일을 선택해 업로드할 수 있습니다.

    • 카메라로 촬영하기: 모바일 기기의 카메라로 사진을 찍어 업로드할 수 있습니다.

  3. 업로드한 이미지를 적당한 크기로 조정하고 저장하세요.

사용자 종류 선택​

  • Web
  • Mobile

Web

크리덴셜 정보 섹션에서 추가하는 사용자가 일반 사용자인지, 임시로 방문하는 사용자인지 선택하세요. 사용자 종류에 따라 발급할 수 있는 크리덴셜 종류가 달라집니다.

  • 일반: 조직 또는 시설의 정규 구성원으로서 장기적이고 지속적인 출입 권한이 필요한 사용자입니다. 모든 크리덴셜 유형을 발급할 수 있습니다.

  • 임시: 단기적이고 제한적인 출입 권한이 필요한 방문자 또는 비정규 구성원입니다. 게스트, 외부 방문객, 단기 계약자, 배송 기사 등이 해당될 수 있습니다. QR 코드와 링크 패스(LinkPass)를 포함한 웹 카드 크리덴셜만 발급할 수 있습니다.

Mobile

사용자 종류에서 추가하는 사용자가 일반 사용자인지, 임시로 방문하는 사용자인지 선택하세요. 사용자 종류에 따라 발급할 수 있는 크리덴셜 종류가 달라집니다.

  • 일반: 조직 또는 시설의 정규 구성원으로서 장기적이고 지속적인 출입 권한이 필요한 사용자입니다. 모든 크리덴셜 유형을 발급할 수 있습니다.

  • 임시: 단기적이고 제한적인 출입 권한이 필요한 방문자 또는 비정규 구성원입니다. 게스트, 외부 방문객, 단기 계약자, 배송 기사 등이 해당될 수 있습니다. QR 코드와 링크 패스(LinkPass)를 포함한 웹 카드 크리덴셜만 발급할 수 있습니다.

알아두기

  • 일반 유형의 사용자는 자동으로 전체 출입 레벨이 할당될 수 있는 반면, 임시 유형의 사용자는 수동으로 출입 레벨을 선택해야 합니다.

  • 사용자 종류는 BioStar Air 2.6 버전 이상에서 지원합니다.

크리덴셜 발급​

사용자 정보를 입력한 후, 크리덴셜 정보 섹션에서 출입 인증 수단으로 사용할 크리덴셜을 설정하세요. 각 크리덴셜에는 활성화 또는 비활성화할 수 있는 토글 스위치가 있습니다.

  • 모바일 카드: 슈프리마 패스 모바일 앱을 출입 인증 수단으로 사용할 수 있습니다. 관리자는 카드 ID 번호를 편집할 수 있습니다.

    • Web
    • Mobile

    Web

    Mobile

  • RF 카드: 사용자당 최대 8개의 RFID 카드를 출입 인증 수단으로 추가할 수 있습니다. 추가 버튼을 클릭하세요.

    • Web
    • Mobile

    Web

    Mobile

    카드 종류(CSN / Wiegand), 카드 ID 및 카드 이름을 구성할 수 있습니다. 장치로 스캔 항목에서 리더기를 선택하세요. 연결된 리더기를 사용하여 실물 RFID 카드를 스캔할 수도 있습니다.

    • Web
    • Mobile

    Web

    Mobile

  • 웹 카드: 웹 QR 또는 링크 패스을 출입 인증 수단으로 사용할 수 있습니다.

    • Web
    • Mobile

    Web

    Mobile

    웹 QR은 설정 → 사이트 → 추가 기능에서 QR 리더 사용 옵션을 활성화해야 나타납니다.

  • 얼굴: 출입 인증 수단으로 얼굴 프로필 이미지를 최대 2개까지 추가할 수 있습니다. 기존 이미지를 업로드하거나 장치를 통해 등록할 수 있습니다. 또는 이메일이나 문자 메시지로 링크를 사용자에게 보내서 사용자의 모바일 기기 카메라를 사용해 직접 등록하는 방법도 있습니다.

    • Web
    • Mobile

    Web

    • 사진 선택: 로컬에 저장된 이미지 파일을 선택하여 업로드할 수 있습니다.

    • 장치로 등록: 등록된 장치를 사용하여 얼굴 이미지를 등록할 수 있습니다.

    • 모바일로 등록: 모바일 기기의 카메라로 QR 코드를 스캔하면 얼굴 이미지를 등록할 수 있는 링크로 이동할 수 있습니다.

    • 얼굴 등록 링크 전송: 사용자의 이메일이나 문자 메시지로 얼굴 등록 링크를 보내 모바일 기기의 카메라로 직접 얼굴을 등록할 수 있습니다.

    Mobile

    얼굴 사진을 추가하려면 버튼을 탭하세요. 하단에 나타나는 메뉴에서 원하는 방법으로 얼굴 사진을 추가할 수 있습니다.

    • 사진 선택하기: 모바일 기기에 저장된 이미지 파일을 선택해 업로드할 수 있습니다.

    • 카메라로 촬영하기: 모바일 기기의 카메라로 사진을 찍어 업로드할 수 있습니다.

    • 얼굴 등록 링크 전송: 사용자의 이메일로 얼굴 등록 링크를 보내 모바일 기기의 카메라로 직접 얼굴을 등록할 수 있습니다.

    • 사용자 프로필 이미지로 사용: 프로필로 사용 중인 이미지를 얼굴 크리덴셜로 사용할 수 있습니다.

    얼굴 사진 등록에 대한 자세한 내용을 확인하려면 사진 가이드 보기 버튼을 클릭하세요.

  • 2FA PIN: 4~16자리 PIN을 설정하여 2단계 인증(2FA)의 일부로 사용할 수 있습니다.

    • Mobile

초대 링크 전송​

모바일 카드 또는 웹 카드를 사용하는 사용자에게 슈프리마 패스 앱 다운로드 링크를 이메일로 전송할 수 있습니다. 이메일 주소(필수)와 휴대폰 번호(선택 사항)를 입력하세요.

  • Web
  • Mobile

Web

Mobile

알아두기

문자메시지 옵션은 설정 → 크레딧 센터에서 SMS 크레딧 충전을 요청해야 사용할 수 있습니다.

출입 레벨 및 유효성 설정​

크리덴셜을 선택한 후 출입 정보 섹션에서 사용자의 크리덴셜 유효 기간, 출입 레벨 및 층 레벨 등을 구성하세요.

  • Web
  • Mobile

Web

Mobile

  • 크리덴셜 유효 기간: 사용자에게 발급한 크리덴셜의 유효 기간을 설정하세요. 종료일을 설정하지 않으려면 버튼을 클릭하세요.

  • 출입 레벨: 사용자가 출입할 수 있는 권한을 선택하세요. 전체 권한을 부여하거나 사전 정의된 출입 레벨을 선택하세요.

  • 층 레벨: 사용자가 출입할 수 있는 층 레벨을 선택하세요.

  • 장치 관리자: 사용자가 특정 장치에서 출입 권한을 관리할 수 있도록 허용할 수 있습니다.

  • 모바일 연속 태그: 대기 시간 없이 연속으로 모바일 카드를 태그할 수 있도록 허용할 수 있습니다. 이 기능은 슈프리마 패스 앱 2.9.0 버전 이상에서 지원합니다.

알아두기

  • 층 레벨 설정에 대한 자세한 내용은 층 레벨 관리하기를 참고하세요.

  • 장치 관리자 옵션은 BioStation 3, X-Station 2 사용자만 사용할 수 있습니다. 장치 관리자로 설정된 사용자는 BioStation 3, X-Station 2 장치의 관리자 메뉴를 사용할 수 있습니다.

카드 템플릿 선택​

  • Web
  • Mobile

Web

카드 템플릿의 디자인을 변경하려면 오른쪽 패널의 카드 템플릿 아래에서 또는 버튼을 클릭해 선택할 수 있습니다.

Mobile

카드 템플릿의 디자인을 변경하려면 크리덴셜 미리보기에서 또는 버튼을 탭하여 선택할 수 있습니다.

알아두기

새로운 카드 템플릿 생성 방법에 대한 자세한 내용은 카드 템플릿 관리하기를 참고하세요.

사용자 저장 또는 활성화​

사용자에 대한 모든 설정을 구성하면 아래 옵션 중 하나를 선택할 수 있습니다.

  • 사용자를 저장하고 나중에 활성화하려면 저장 버튼을 클릭하세요.

  • 크리덴셜을 즉시 발급하고 활성화하려면 크리덴셜 활성화 버튼을 클릭하세요.
