등록 출입문 관리하기

등록된 출입문에 입출입 장치를 할당하고, 릴레이를 구성하며, 출입 스케줄을 설정합니다. 보안, 화재 대피, 근태 관리(T&A)를 위해 체계적으로 출입을 통제할 수 있습니다.

팁

출입문 등록 및 관리에 대한 동영상 가이드를 시청하려면 다음 영상을 참고하세요.

출입문 목록 조회​

출입문 목록에서는 BioStar Air에 등록된 모든 출입문을 확인할 수 있습니다. 각 출입문의 이름, 할당된 장치, 그룹 등 주요 정보를 확인할 수 있습니다. 또한 개별 출입문을 클릭하여 세부 정보를 확인하고 설정을 수정할 수 있습니다.

등록된 출입문을 확인하려면 화면 왼쪽 사이드바에서 출입문 관리 → 등록된 출입문을 클릭하세요.

출입문 정보​

출입문 목록은 다음과 같은 정보를 제공합니다.

  • 출입문: 출입문 이름입니다. 일반적으로 장치 등록 시 자동 지정되며, 관리자는 출입문 이름을 변경할 수 있습니다.

  • 입구 장치: 출입문으로 들어오는 사용자를 인증하는 장치입니다.

  • 출구 장치: 출입문에서 나가는 사용자를 인증하는 장치입니다.

  • 출입문 그룹: 출입문에 할당된 그룹 이름입니다.

  • 설명: 출입문에 대한 설명입니다.

  • 카메라: 출입문과 연결된 카메라 정보가 표시됩니다. 카메라 열은 설정 → 사이트 → 추가 기능에서 비디오 관리 기능을 활성화해야 표시됩니다.

추가 기능​

목록에서 하나 또는 여러 개의 출입문을 선택하면 관리자는 제공하는 추가 기능으로 출입문을 관리할 수 있습니다.

  • : 출입문 정보를 수정하거나 할당된 장치의 설정을 변경할 수 있습니다. 한 개의 출입문만 선택해야 이 기능을 사용할 수 있습니다. 자세한 내용은 #editingDoors을 참고하세요.

  • : 출입문 목록을 새로고침할 수 있습니다.

  • : 목록에서 선택한 출입문을 제거합니다.

  • : 출입문 열림 및 잠김 상태를 위한 스케줄을 설정합니다. 자세한 내용은 출입문 스케줄 관리하기를 참고하세요.

  • : 목록에서 선택한 출입문을 그룹으로 구성할 수 있습니다. 그룹 관리에 대한 자세한 내용은 출입문 그룹 관리하기를 참고하세요.

  • : 출입문 리포트를 생성할 수 있습니다. 리포트 생성에 대한 자세한 내용은 #generatingDoorReports을 참고하세요.

필터링​

출입문 목록에서는 출입문 그룹을 기준으로 필터링하여 조회할 수 있습니다. 출입문 그룹 열의 버튼을 클릭하고 원하는 그룹을 선택하세요. 확인 버튼을 클릭하면 선택한 그룹에 속한 출입문만 표시됩니다.

알아두기

적용한 필터 조건을 초기화하려면 버튼을 클릭하고 초기화 버튼을 선택하세요.

출입문 추가​

장치와 함께 추가​

모바일 앱에서 장치 등록 시 마지막 단계에서 이 장치로 출입문 만들기 옵션을 선택하면 출입문을 생성할 수 있습니다. 모바일 앱으로 장치를 등록하는 방법에 대한 자세한 내용은 엣지 장치 등록 및 삭제하기를 참고하세요.

웹 포털에서 추가​

BioStar Air 포털에서 새 출입문을 추가하려면 다음 단계를 따르세요.

  1. 화면 왼쪽 사이드바에서 출입문 관리 → 등록된 출입문을 클릭하세요.

  2. 화면 오른쪽 상단의 출입문 추가 버튼을 클릭하세요.

  3. 출입문 추가 패널이 나타나면 출입문 이름을 입력하세요.

  4. 입구 장치와 출구 장치를 선택하세요.

  5. 출입문 설정을 저장하고 추가하려면 저장 버튼을 클릭하세요.

출입문의 장치 할당 규칙

  • 각 장치는 하나의 출입문에만 할당할 수 있습니다.

  • 장치가 이미 등록되어 있지만 사용할 수 있는 장치 목록에 나타나지 않는다면, 해당 장치가 다른 출입문에 할당되어 있는지 확인하세요. 다른 장치에 할당되어 있다면 해당 출입문에서 제거한 후 다시 시도하세요.

  • 한 개의 장치는 출입문과 엘리베이터에 동시 할당할 수 없습니다.

마스터/슬레이브 장치를 사용하는 출입문

  • 출입문은 마스터 장치(XPass 2, CoreStation 20)가 제어합니다.

  • 마스터 장치로 설정된 XPass 2는 할당된 보조 장치, 모듈 수에 관계 없이 한 개의 출입문에만 할당할 수 있습니다.

  • 마스터 장치로 설정된 XPass 2는 최대 31개의 보조 장치를 할당할 수 있으며, 보조 Secure Module(SIO2)로 릴레이를 제어하는 새로운 출입문을 추가할 수 있습니다.

  • CoreStation 20(CS-20)과 Secure Module(SIO2)을 사용하는 출입문은 생성할 수 없습니다.

출입문 구성 제한 사항​

출입문 구성에는 아래와 같은 제한 사항이 있습니다.

  • XPass D2 1개 + SIO2 (릴레이는 SIO2 소유)

  • XPass 2 1개 + XPass D2 (릴레이는 XPass 2 소유)

  • XPass D2 2개 (릴레이는 XPass 2 소유)

  • 1개 또는 2개의 OSDP/Wiegand 장치 (릴레이는 CS-20 소유)

  • 출입문 당 SIO2는 1개만 허용 (릴레이는 I/O 용): XPass D2와 동일한 XPass 2에 배선 필요

출입문 수정​

출입문의 세부 정보 및 설정을 수정하려면 다음 단계를 따르세요.

  1. 화면 왼쪽 사이드바에서 출입문 관리 → 등록된 출입문을 클릭하세요.

  2. 목록에서 수정할 출입문을 클릭하세요. 또는 출입문 왼쪽의 체크박스를 클릭한 후 상단의 화면 왼쪽 사이드바에서 버튼을 클릭하세요.

  3. 출입문 수정 패널이 나타나면 출입문 이름 및 설명을 수정하세요.

  4. 입출구 장치 섹션에서 출입문의 사용자 인증을 위한 장치를 선택하세요.

    • 입구 장치: 출입문으로 들어오는 사용자를 인증하는 장치를 선택하세요.

    • 출구 장치: 출입문에서 나가는 사용자를 인증하는 장치를 선택하세요.

  5. 보조 장치 섹션에서 릴레이 장치를 설정하세요.

    • 카메라: 이벤트 로그를 위한 CCTV 카메라를 연결할 수 있습니다.

    • 출입문 릴레이: 출입문 잠금을 제어하는 릴레이 장치를 선택하세요.

    • 도어 센서 무시 릴레이: 출입문 센서를 무시하도록 릴레이를 설정할 수 있습니다.

    카메라는 설정 → 사이트 → 추가 기능에서 비디오 관리 기능을 활성화해야 사용할 수 있습니다.

  6. 추가 설정을 구성하세요.

    • 설치된 경우 퇴실 버튼으로 설정할 포트를 선택하세요.

    • 도어 센서로 설정할 포트를 선택하세요.

    • 개방 시간에서 출입문이 열린 상태로 유지되는 시간을 설정하세요. (기본값: 3초)

  7. 수정 사항을 적용하려면 저장 버튼을 클릭하세요.

알아두기

  • 보안을 위해 릴레이는 항상 출입문의 보안 측에 있는 장치를 연결해야 합니다. 일반적으로 출구 장치의 릴레이를 사용하거나 SIO2 모듈을 사용하는 것을 권장합니다.

  • 클라우드에 직접 연결하여 구성한다면 입구 장치와 출구 장치에 각각 네트워크 케이블 연결이 필요합니다.

  • 마스터-보조 장치 구성에서, 마스터(보통 출구 장치) 장치는 네트워크 연결이 필요하고 보조 장치는 RS-485를 통해 마스터 장치에 연결됩니다.

  • SIO2 보안 모듈 구성에 대한 자세한 내용은 SIO2 보안 모듈 구성하기를 참고하세요.

출입문 리포트 생성​

출입문 리포트를 생성하여 관리자의 이메일로 전송할 수 있습니다. 출입문 리포트는 출입문을 통해 출입한 사용자에 대한 정보를 제공합니다.

  1. 화면 왼쪽 사이드바에서 출입문 관리 → 등록된 출입문을 클릭하세요.

  2. 출입문 목록 상단의 버튼을 클릭하세요.

  3. 리포트 설정 창이 나타나면 리포트 이름을 입력하세요.

  4. 리포트에 포함할 항목을 선택하세요.

  5. 생성한 리포트를 저장하려면 리포트 추가 버튼을 클릭하세요.

  6. 리포트 목록이 생성되며 관리자의 이메일로 다운로드 링크가 전송됩니다.

알아두기

리포트 목록에서 생성된 리포트를 검색하거나, 수정, 다운로드, 삭제할 수 있습니다.
