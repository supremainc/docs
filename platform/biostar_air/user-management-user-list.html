사용자 관리하기

사용자 목록은 조직의 출입 통제를 담당하는 관리자를 위한 필수 관리 도구입니다. 사용자 그룹화를 통한 조직 관리, 실시간 활성화/일시 정지로 신입 또는 퇴사 직원 즉시 대응, 다양한 크리덴셜 통합 관리, 유효 기간 모니터링을 통한 보안 강화, 필터링과 리포트 생성을 통한 보고 및 감사 대비까지 모든 사용자 관리 작업을 수행할 수 있습니다.

사용자 목록 조회​

사용자 목록에서는 BioStar Air에 등록된 모든 사용자를 한눈에 확인할 수 있습니다. 각 사용자의 세부 정보를 보고, 필터링 및 정렬 기능을 활용하여 특정 사용자를 쉽게 찾을 수 있습니다. 또한, 사용자 목록에서 직접 사용자 편집, 삭제, 그룹 할당 등의 작업을 수행할 수 있습니다.

  • Web
  • Mobile

Web

화면 왼쪽 사이드바에서 사용자 관리 → 사용자를 클릭하세요.

Mobile

BioStar Air 앱에서 사용자 관리 기능을 사용할 수 있습니다.

  1. 모바일 기기에서 BioStar Air 앱을 실행하세요.

  2. 화면 하단의 탐색바에서 관리를 탭하세요.

  3. 사용자 버튼을 탭하세요.

사용자 목록이 나타납니다. 새로운 사용자를 추가하거나 기존 사용자의 세부 정보를 편집할 수 있습니다.

사용자 정보​

사용자 목록의 테이블에서 제공하는 정보는 아래와 같습니다.

  • 사용자 종류: 사용자가 일반 사용자인지 임시 사용자인지 나타냅니다. 일반 및 임시 사용자에 대한 자세한 내용은 사용자 카드 및 크리덴셜 종류를 참고하세요.

  • 사용자 이름: 사용자의 이름입니다.

  • ID: 사용자의 고유 아이디입니다.

  • 크리덴셜 종류: 사용자에게 발급한 크리덴셜을 나타냅니다. 크리덴셜의 종류에 대한 자세한 내용은 사용자 카드 및 크리덴셜 종류를 참고하세요.

  • 유효 기간: 사용자에게 발급한 크리덴셜의 유효 기간입니다.

  • 사용자 상태: 해당 사용자의 활성화 또는 비활성화 상태를 나타냅니다.

  • 사용자 그룹: 사용자가 속한 그룹입니다. 사용자 그룹에 대한 자세한 내용은 사용자 그룹 관리하기를 참고하세요.

  • 출입 레벨: 사용자에게 할당한 출입 레벨입니다. 출입 레벨에 대한 자세한 내용은 출입 레벨 관리하기를 참고하세요.

  • 층 레벨: 사용자가 출입할 수 있는 층에 대한 권한입니다. 층 레벨에 대한 자세한 내용은 층 레벨 관리하기를 참고하세요.

필터링​

  • Web
  • Mobile

Web

사용자 목록의 상단 헤더 영역에서 필터링 기능을 이용할 수 있습니다. 헤더 영역의 버튼을 클릭하면 원하는 조건을 설정할 수 있습니다.

알아두기

적용한 필터 조건을 초기화하려면 사용자 목록 상단의 버튼을 클릭하세요.

Mobile

모바일 앱에서 사용자 목록의 사용자를 원하는 기준으로 필터링하여 볼 수 있습니다. 사용자 목록 오른쪽 상단의 필터를 탭하세요. 사용자는 아래 기준으로 필터링할 수 있습니다.

  • 사용자 종류: 일반 또는 임시 사용자로 필터링합니다.

  • 크리덴셜 종류: 크리덴셜 종류에 따라 필터링합니다.

  • 사용자 상태: 활성화 상태에 따라 필터링합니다.

  • 사용자 그룹: 사용자 그룹에 따라 필터링합니다.

추가 기능​

사용자 목록 상단에서 제공하는 기능은 아래와 같습니다.

  • : 선택한 사용자에게 그룹을 할당하거나 제외, 이동할 수 있습니다. 자세한 내용은 #manageUserGroups를 참고하세요.

  • : 선택한 사용자의 출입 레벨을 변경할 수 있습니다. 출입 레벨 관리에 대한 자세한 내용은 출입 레벨 관리하기를 참고하세요.

  • : 선택한 사용자에게 대기 시간 없이 연속으로 모바일 카드를 태그할 수 있도록 설정할 수 있습니다. 슈프리마 패스 앱은 2.9.0 버전 이상에서 지원합니다.

  • : 선택한 사용자에게 사용자 템플릿을 적용할 수 있습니다. 기본 설정된 사용자 정보 외에 필요한 다른 정보를 추가할 수 있습니다.

  • : 선택한 사용자 정보를 PDF 또는 CSV 파일로 내보낼 수 있습니다.

  • : 사용자 이미지를 일괄 업로드할 수 있습니다. 업로드하는 이미지 파일의 이름은 사용자 아이디와 일치해야 합니다.

    예, 601.jpg 파일은 사용자 아이디 601에 해당합니다.

    파일 형식 요구사항

    • 지원되는 형식: .jpg, .jpeg, .png

    • 파일 크기 제한: 이미지당 1MB

  • : 선택한 사용자에게 얼굴 등록 링크를 이메일 또는 문자 메시지로 전송할 수 있습니다. 사용자가 링크를 클릭해 웹 페이지에서 얼굴을 등록하면, 얼굴 인증을 통해 출입할 수 있는 권한이 부여됩니다.

    이메일과 휴대폰 번호가 등록되어 있지 않은 사용자에게는 링크를 전송할 수 없습니다.

  • : 사용자 목록에 적용된 필터를 초기화합니다. 이 기능은 필터링이 적용된 상태에서 사용할 수 있습니다. 필터링 적용 방법에 대한 자세한 내용은 #userFiltering를 참고하세요.

  • : 사용자 리포트를 설정할 수 있습니다. 리포트를 생성을 완료하면 관리자의 이메일로 전송됩니다. 리포트 설정에 대한 자세한 내용은 #userReport를 참고하세요.

  • : 사용자 목록에서 개인 정보 보호를 위해 이름, 아이디 등의 정보를 마스킹(***)할 수 있습니다. 이 기능은 개인 정보 보호가 필요한 환경에서 유용합니다. 토글 버튼을 선택해 기능을 활성화하고, 저장 버튼을 클릭하세요.

사용자 정보 수정​

  • Web
  • Mobile

Web

사용자의 세부 정보를 수정하거나 크리덴셜을 변경하려면 다음 단계를 따르세요.

  1. 화면 왼쪽 사이드바에서 사용자 관리 → 사용자를 클릭하세요.

  2. 목록에서 수정할 사용자를 클릭하세요.

  3. 사용자 세부 정보 오른쪽 상단의 버튼을 클릭하세요.

  4. 사용자 정보 수정 창이 나타나면 필요한 세부 정보를 수정하세요.

  5. 수정을 완료하면 오른쪽 하단의 저장 버튼을 클릭하세요.

Mobile

모바일 앱에서 사용자 세부 정보를 수정하려면 다음 단계를 따르세요.

  1. 사용자 목록에서 정보를 수정할 사용자를 탭하세요.

  2. 오른쪽 상단의 버튼을 탭하세요.

  3. 사용자 정보 수정 화면에서 필요한 정보를 수정하세요.

  4. 수정 사항을 저장하려면 저장 버튼을 탭하세요.

    크리덴셜을 발급하려면 크리덴셜 활성화 버튼을 사용하세요.

알아두기

사용자 정보에 대한 자세한 내용은 사용자 추가하기를 참고하세요.

사용자 그룹 관리​

사용자를 사용자 그룹에 할당하거나 이동, 제외하는 방법을 안내합니다. 사용자 그룹 생성 및 관리에 대한 자세한 내용은 사용자 그룹 관리하기를 참고하세요.

  • Web
  • Mobile

Web

그룹에 사용자 할당하기​

사용자 그룹을 생성한 후 사용자 목록에서 사용자를 추가할 수 있습니다.

  1. 화면 왼쪽 사이드바에서 사용자 관리 → 사용자를 클릭하세요.

  2. 사용자 목록에서 앞서 생성한 그룹에 할당할 사용자의 체크박스를 클릭하세요.

  3. 사용자 목록 왼쪽 상단의 버튼을 클릭하세요.

  4. 사용자 그룹 관리하기 창이 나타나면 사용자를 할당할 상위 그룹 또는 하위 그룹을 선택하세요.

  5. 선택을 완료하고 그룹 설정을 저장하려면 저장 버튼을 클릭하세요.

사용자 목록의 사용자 그룹 열에서 사용자가 새 그룹에 할당된 것을 확인할 수 있습니다.

사용자 그룹으로 돌아가면 해당 그룹의 전체 사용자 수가 업데이트된 것을 확인할 수 있습니다.

그룹 해제 또는 이동​

사용자를 그룹에서 제외하거나 다른 그룹으로 이동할 수 있습니다.

  1. 화면 왼쪽 사이드바에서 사용자 관리 → 사용자를 클릭하세요.

  2. 사용자 목록에서 그룹에서 해제하거나 이동할 사용자의 체크박스를 클릭하세요.

    그룹 이동 시에는 동일한 그룹의 사용자를 선택한 후 진행하세요.

  3. 사용자 목록 왼쪽 상단의 버튼을 클릭하세요.

  4. 사용자 그룹 관리하기 창이 나타나면 다음 중 하나를 수행하세요.

    • 그룹을 해제할 때는 선택된 그룹의 체크박스를 비활성화하세요.

    • 그룹을 이동할 때는 선택된 그룹의 체크박스를 활성화하고, 이동할 그룹을 선택하세요.

  5. 그룹 설정을 저장하려면 저장 버튼을 클릭하세요.

사용자 목록의 사용자 그룹 열에서 사용자의 그룹 변경 사항을 확인할 수 있습니다.

Mobile

모바일 앱에서 사용자 그룹을 관리할 수 있습니다. 사용자 목록 상단의 버튼을 탭하세요. 사용자 그룹 목록을 확인할 수 있습니다.

  • 그룹에 속한 사용자를 보려면 원하는 그룹의 오른쪽에 버튼을 탭하세요.

  • 새로운 그룹을 추가하려면 오른쪽 상단의 버튼을 탭하세요.

  • 그룹 이름을 수정하려면 원하는 그룹을 길게 터치하고 하단의 버튼을 탭하세요.

  • 그룹을 삭제하려면 원하는 그룹을 길게 터치한 후 하단의 버튼을 탭하세요. 하나 이상의 그룹을 선택해 삭제할 수 있습니다.

알아두기

사용자 그룹 관리에 대한 자세한 내용은 사용자 그룹 관리하기를 참고하세요.

사용자 상태 관리​

사용자의 출입 권한을 활성화하거나 일시 정지할 수 있습니다. 또한 크리덴셜을 재발급할 수도 있습니다.

알아두기

관리자가 BioStar Air에서 사용자의 크리덴셜을 재발급하면 이전에 발급된 모바일 크리덴셜, QR 코드 및 링크패스(LinkPass)는 모두 취소되고 새 크리덴셜로 교체됩니다. 사용자는 사이트 설정에 따라 이메일 또는 SMS로 새 크리덴셜 링크를 받게 됩니다. 사용자는 해당 링크를 통해 슈프리마 패스 앱에 새 모바일 크리덴셜을 다운로드하거나 새 QR 코드 또는 링크패스를 이용해야 합니다. 취소된 모바일 크리덴셜, QR 코드 또는 링크패스로 출입을 시도하면 실패합니다.

개별 사용자​

  • Web
  • Mobile

Web

  1. 화면 왼쪽 사이드바에서 사용자 관리 → 사용자를 클릭하세요.

  2. 사용자 목록에서 상태를 변경할 사용자를 클릭하세요.

  3. 사용자 세부 정보 상단에서 원하는 기능을 선택하세요.

  • : 사용자의 출입 권한을 활성화합니다. 새 사용자의 초기 활성화 또는 일시 정지된 사용자의 재활성화 시 사용하세요.

  • : 크리덴셜을 다시 발급합니다. 이 기능은 사용자의 크리덴셜이 만료되었거나, 크리덴셜을 사용할 수 없어 새로 발급할 때 사용할 수 있습니다.

  • : 사용자의 출입을 차단합니다. 이 기능은 사용자가 활성화된 상태일 때 사용할 수 있습니다.

Mobile

얼굴 등록 링크 전송​

이미 등록한 사용자에게 얼굴 등록 링크를 이메일 또는 문자 메시지로 전송하여 얼굴을 등록하도록 안내할 수 있습니다. 사용자가 링크를 클릭해 웹 페이지에서 얼굴을 등록하면, 얼굴 인증을 통해 출입할 수 있습니다.

알아두기

이메일과 휴대폰 번호가 등록되어 있지 않은 사용자에게는 링크를 전송할 수 없습니다.

사용자 삭제​

주의

삭제한 사용자는 복구할 수 없습니다. 삭제하기 전에 반드시 확인하세요.

개별 사용자​

  1. 화면 왼쪽 사이드바에서 사용자 관리 → 사용자를 클릭하세요.

  2. 목록에서 삭제할 사용자를 클릭하세요.

  3. 사용자 세부 정보에서 버튼을 클릭하세요.

  4. 경고 메시지 창이 나타나면 확인 버튼을 클릭하세요.

일괄 삭제​

  1. 화면 왼쪽 사이드바에서 사용자 관리 → 사용자를 클릭하세요.

  2. 목록에서 삭제할 사용자의 체크박스를 클릭하세요. 1명 이상의 사용자를 선택할 수 있습니다.

  3. 화면 왼쪽 상단의 → 삭제를 클릭하세요.

  4. 경고 메시지 창이 나타나면 확인 버튼을 클릭하세요.

사용자 리포트 생성​

사용자 리포트를 생성하여 관리자의 이메일로 전송할 수 있습니다.

  1. 사용자 목록 상단의 버튼을 클릭하세요.

  2. 리포트 설정 창이 나타나면 리포트 이름을 입력하세요.

  3. 각 필터 옵션에서 리포트에 포함할 항목을 선택하세요.

  4. 생성한 리포트를 저장하려면 리포트 추가 버튼을 클릭하세요.

  5. 리포트 목록이 생성되며 관리자의 이메일로 다운로드 링크가 전송됩니다.

알아두기

리포트 목록에서 생성된 리포트를 검색하거나, 수정, 다운로드, 삭제할 수 있습니다.

사용자 검색​

사용자 목록 오른쪽 상단의 검색 입력 필드를 통해 특정 사용자를 빠르게 찾을 수 있습니다. 사용자 이름, 아이디, 전화번호를 기준으로 검색할 수 있습니다.
