2026년 5월

2026년 5월 28일

이번 5월, Suprema Docs는 더 크게, 더 넓게 움직였습니다. 신규 장치 BioStation 3 Max가 새롭게 라인업에 합류했습니다. BioStar X와 BioStar Air의 기능도 한층 보강되었습니다. 특히 이번 달엔 타사 제품과의 연동을 다루는 인테그레이션 카테고리가 새롭게 문을 열었으니, 더욱 풍성해진 Suprema Docs를 지금 바로 만나보세요.

Platform​

BioStar X​

  • 1.0.2 버전 릴리스 노트 살펴보기

  • 메인 서버의 IP 주소 변경하기: BioStar X 메인 서버의 IP 주소 변경 안내 추가

  • 디렉토리 연동 설정하기: 디렉토리 서비스 연동 설정에 추가/업데이트/삭제 모드 추가

  • 시스템 복원하기: 복원 도구 사용에 대한 상세 과정 업데이트

  • 장치 내 이벤트 로그 삭제: 장치에 저장된 이벤트 로그 삭제 기능 추가

  • VMS 연동하기: Nx Witness v6.x.x에 대한 지원 추가

  • 카메라 설정하기: 카메라 그룹 권한 설정에 대한 안내 추가

  • BioStation 3 Max, XPass Q2 장치에 대한 지원 사양 추가

BioStar Air​

  • SW 2.12 버전 릴리스 노트 살펴보기

  • 에어팝 패치 설치: 에어팝 패치 설치 안내 추가

Devices​

신규 장치​

슈프리마 플래그십 AI 기반 바이오인식 제품 BioStation 3 Max의 매뉴얼을 오픈했습니다.

  • BioStation 3 Max

장치 업데이트​

  • BioEntry W3 1.0.2 버전 업데이트 적용

    • 벽걸이용 브래킷 이미지 변경

    • RS-485 사용 관련 주의 사항 추가

  • Device Manager 1.2.3 버전 업데이트 적용

    • XPass D2 설정하기: EV2/EV3 보안 메시징 강제 기능 추가

장치 릴리스 노트​

  • BioEntry R2: 1.4.2 버전 릴리스 노트 추가

  • CoreStation: 1.7.2 버전 릴리스 노트 추가

  • CoreStation 20: 1.0.4 버전 릴리스 노트 추가

  • BioEntry W3: 1.0.2 버전 릴리스 노트 추가

  • Device Manager: 1.2.3 버전 릴리스 노트 추가

  • FaceStation F2: 2.2.2 버전 릴리스 노트 추가

인테그레이션​

인테그레이션(Integration) 카테고리를 추가했습니다. 타사 제품과 슈프리마 장치를 연동하는 방법을 확인해보세요.
