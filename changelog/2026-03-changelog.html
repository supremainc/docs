2026년 3월

2026년 4월 8일

이번 3월 업데이트는 더욱 직관적인 인터페이스와 강화된 검색 기능으로 Suprema Docs 사용 경험을 개선했습니다. 동시에 BioStar Air 관리자 가이드를 완전히 재구성하여 사용자 관리부터 출입 통제, 비디오 감시까지 모든 기능을 한눈에 파악할 수 있도록 정리했습니다.

Docs​

  • 첫 페이지 디자인 변경

  • 검색 기능 강화

    제품별 검색 기능 추가, 단축키(ctrl + k)로 검색 창 호출

Platform​

BioStar X​

  • VMS 서버 연동 시 인증서 설치 관련 문서 업데이트

BioStar Air​

  • BioStar Air 가이드 문서 전체 재구성 및 최적화 완료

  • BioStar Air 다국어 지원 확대: 영문, 일본어, 스페인어 번역 추가

    • 사용자 관리, 출입 권한 관리, 비디오 관리 등 주요 기능 문서 추가

    • 영문 이미지 및 사용자 인터페이스 스크린샷 추가

  • BioStar Air 목차 구조 개선 및 섹션 재조정

Devices​

신규 릴리스 노트를 추가하고 기존 장치 문서를 업데이트했습니다.

장치 문서 업데이트​

  • XPass 2: 1.5.1 버전 릴리스 노트 추가

  • Device Manager: 앱 릴리스 노트 카테고리 신설 및 1.2.2 버전 릴리스 노트 추가

  • BioEntry P2: 1.6.0 버전 릴리스 노트 추가

  • 장치 공장 초기화 버튼 위치 안내 이미지 추가

Developer​

  • API 문서 업데이트: Create BioStar X QR

  • Device SDK: 2.9.12 버전 릴리스 노트 추가

다국어 지원​

  • 일본어, 스페인어 문서의 완성도 향상

    • 문서 제목 및 내용 일관성 개선

    • 오번역 수정 및 용어 통일

    • 띄어쓰기 및 문법 수정

  • 영문 번역 대폭 업데이트

    • 기술 용어 표준화

    • 문장 구조 및 표현 개선

오픈 소스 라이선스​

  • 라이선스 고지문 추가: BSD-2-Clause, BSD-3-Clause, EPL, EDL, LGPLv2.1​, GPLv2​, ISC, zlib
