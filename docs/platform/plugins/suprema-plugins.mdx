# 슈프리마 제공 플러그인

슈프리마에서 공식적으로 개발하여 제공하는 플러그인들의 설치 및 관리 방법을 설명합니다.

## 개요

슈프리마 제공 플러그인은 기존 BioStar 2에서 제공하던 주요 기능들을 BioStar X 환경에서 플러그인 형태로 제공하는 서비스입니다. 외부 플러그인과 달리 설치 패키지를 통해 제공되며, 자동으로 BioStar X와 연동됩니다.

### 외부 플러그인과의 차이점

| 구분 | 외부 플러그인 | 슈프리마 플러그인 |
|------|---------------|-------------------|
| **개발 주체** | 고객, 파트너, 타사 | 슈프리마 |
| **제공 방식** | URL 등록 | 설치 패키지 |
| **등록 방법** | 수동 등록 | 자동 등록 |
| **관리 방법** | Service Manager에서 삭제 | 프로그램 추가/제거 |
| **목록 표시** | 등록 순서대로 | 최상단 우선 표시 |
| **최대 개수** | 10개 | 2개 |

## T&A (Time and Attendance) 플러그인

### 기능 개요

T&A 플러그인은 기존 BioStar 2에서 제공하던 근태 관리 기능을 BioStar X 환경에서 플러그인 형태로 제공합니다.

**주요 기능:**
- **자동 근무시간 계산**: BioStar X의 출입 기록을 기반으로 한 근무시간 자동 산출
- **근태 상태 관리**: 출근, 퇴근, 지각, 조퇴, 결근 등 근태 상태 추적
- **휴가 관리**: 연차, 병가, 특별휴가 등 휴가 신청 및 승인 관리  
- **리포트 생성**: 개인별, 부서별, 기간별 근태 현황 리포트
- **급여 연동**: 급여 시스템과의 데이터 연동을 위한 인터페이스 제공

### 설치 방법

#### 1. 설치 패키지 다운로드
- 슈프리마 공식 웹사이트 또는 파트너를 통해 T&A 플러그인 설치 패키지 다운로드
- 파일명 예시: `BioStar-X-TA-Plugin-v1.0.0.msi`

#### 2. 설치 실행
```bash
# 관리자 권한으로 설치 실행
BioStar-X-TA-Plugin-v1.0.0.msi
```

#### 3. 설치 과정
1. **설치 마법사 시작**: 설치 프로그램 실행
2. **라이선스 동의**: 소프트웨어 사용권 계약 동의
3. **설치 경로 선택**: 기본 경로 또는 사용자 지정 경로 선택
4. **BioStar X 연동 설정**: BioStar X 서버 정보 입력
5. **설치 완료**: 자동으로 Service Manager에 등록

#### 4. 설치 확인
설치가 완료되면 다음과 같이 확인할 수 있습니다:

- **Service Manager**: Plugins 페이지에 "T&A Management" 항목 표시
- **Launcher**: T&A 아이콘이 상단에 표시
- **Windows 서비스**: T&A 관련 서비스가 실행 중

### 설정 및 관리

#### Service Manager에서 확인 가능한 정보

**기본 정보:**
- **Name**: T&A Management (변경 불가)
- **Description**: 근태 관리 시스템 (변경 불가)
- **URL**: 자동 생성된 내부 URL (변경 불가)
- **Session Bridge**: 자동으로 활성화 (변경 불가)

**상세 정보:**
- **Version**: 설치된 T&A 플러그인 버전 정보
- **T&A HTTPS Port**: T&A 서비스가 사용하는 HTTPS 포트 번호
- **Service Status**: T&A 서비스 실행 상태
- **Last Updated**: 마지막 업데이트 일시

#### 설정 변경 가능 항목

**T&A HTTPS Port 설정:**
```
기본값: 8443
변경 가능 범위: 1024-65535 (시스템 포트 제외)
적용 방법: 설정 변경 후 서비스 재시작 필요
```

**데이터베이스 연결 설정:**
- 연결 문자열 설정
- 백업 일정 설정
- 로그 보관 기간 설정

### 사용 방법

#### 1. 플러그인 접속
- BioStar X Launcher에서 T&A 아이콘 클릭
- Session Bridge를 통해 자동 로그인
- 사용자 권한에 따른 메뉴 표시

#### 2. 주요 메뉴 구성
- **Dashboard**: 전체 근태 현황 요약
- **Employee Management**: 직원 정보 및 근무 규칙 관리
- **Time Tracking**: 실시간 출입 현황 모니터링
- **Schedule Management**: 근무 일정 및 시프트 관리
- **Reports**: 각종 근태 리포트 생성
- **Settings**: 시스템 설정 및 정책 관리

### 제거 방법

T&A 플러그인은 Service Manager에서 직접 삭제할 수 없으며, Windows 프로그램 관리를 통해서만 제거 가능합니다.

#### 1. Windows 설정을 통한 제거
```
Windows 10/11:
설정 → 앱 → 앱 및 기능 → "BioStar X T&A Plugin" 선택 → 제거

Windows Server:
제어판 → 프로그램 → 프로그램 및 기능 → "BioStar X T&A Plugin" 선택 → 제거
```

#### 2. 제거 과정
1. **제거 확인**: 제거 여부 확인 다이얼로그
2. **서비스 중단**: T&A 관련 서비스 자동 중단
3. **파일 삭제**: 설치된 파일 및 폴더 삭제
4. **레지스트리 정리**: 관련 레지스트리 항목 삭제
5. **Service Manager 업데이트**: Plugins 목록에서 자동 제거

#### 3. 제거 후 확인
- Service Manager Plugins 페이지에서 T&A 항목 삭제 확인
- Launcher에서 T&A 아이콘 삭제 확인
- Windows 서비스에서 T&A 서비스 삭제 확인

## 향후 제공 예정 플러그인

슈프리마에서는 기존 BioStar 2의 주요 기능들을 순차적으로 플러그인 형태로 제공할 예정입니다.

### 계획 중인 플러그인

**Mobile Card Manager**
- 모바일 카드 발급 및 관리
- QR/바코드 기반 출입 관리
- 임시 카드 발급 시스템

**Advanced Reporting**
- 사용자 정의 리포트 생성
- 데이터 시각화 대시보드
- 자동 리포트 배포

**Third-party Integration Hub**
- 주요 ERP/HRM 시스템 연동
- API 통합 관리
- 데이터 동기화 도구

### 제공하지 않는 기능

**Visitor Management**
- 방문자 관리 기능은 BioStar X의 기본 출입 통제 서비스에 포함
- 별도 플러그인으로 제공하지 않음
- 이유: 출입 통제의 핵심 기능으로 분류되어 기본 서비스에 통합

## 문제 해결

### 일반적인 문제

**설치 실패**
- **관리자 권한**: 설치 시 관리자 권한으로 실행 확인
- **포트 충돌**: 사용하려는 포트가 다른 서비스에서 사용 중인지 확인
- **방화벽 설정**: Windows 방화벽에서 T&A 서비스 허용 설정
- **디스크 공간**: 충분한 디스크 공간 확보 (최소 500MB)

**서비스 시작 실패**
- **포트 설정**: T&A HTTPS Port 설정 확인
- **데이터베이스 연결**: 데이터베이스 연결 문자열 확인
- **인증서**: SSL 인증서 유효성 확인
- **로그 확인**: 서비스 로그에서 구체적인 오류 메시지 확인

**Session Bridge 오류**
- **인증서 문제**: BioStar X와의 인증서 동기화 확인
- **네트워크 연결**: BioStar X 서버와의 네트워크 연결 확인
- **권한 설정**: 사용자의 T&A 접근 권한 확인

### 로그 위치

**T&A 서비스 로그:**
```
기본 위치: C:\ProgramData\Suprema\BioStarX-TA\Logs\
주요 파일: 
- service.log: 서비스 관련 로그
- api.log: API 호출 로그
- error.log: 오류 로그
```

**Windows 이벤트 로그:**
```
위치: Windows 로그 → 응용 프로그램
원본: BioStarX-TA-Service
```
