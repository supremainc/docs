개발한 플러그인을 **BioStar X Service Manager**에 등록하고 관리하는 방법을 안내합니다.

## 등록 전 준비사항

1. **플러그인 서버 실행**

    플러그인을 등록하기 전에 반드시 플러그인 서버가 실행 중이어야 합니다. BioStar X는 등록 과정에서 플러그인 서버에 접근성을 확인합니다.

1. **네트워크 접근 확인**

    BioStar X 서버에서 플러그인 서버로 접근할 수 있는지 확인하세요.

    ```bash
    # BioStar X 서버에서 테스트
    curl -H "X-BioStar-Ping: test123" http://localhost:8000/bsx
    ```

## 접근성 확인

`curl` 명령어를 사용해 수동으로 엔드포인트를 테스트할 수 있습니다.

### GET 엔드포인트 테스트

```bash
curl -H "X-BioStar-Ping: test123" http://localhost:8000/bsx
```

### POST 엔드포인트 테스트

```bash
curl -X POST http://localhost:8000/bsx \
    -H "Content-Type: application/x-www-form-urlencoded" \
    -d "user_id={user_id}&plugin_id={your_plugin_id}"
```


## 설정 요구사항

서버 설정을 위한 설정 파일을 생성하세요.

```python title="config.py"
SERVER_CONFIG = {
    "address": "192.168.1.2", # BioStar server IP address
    "port": 443, # BioStar server port
    "ssl_verify": False # SSL verification
}
FILE_CONFIG = {
    "public_key_path": "public_key.pem" # RSA public key file
}
```
