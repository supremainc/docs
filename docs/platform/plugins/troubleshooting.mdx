BioStar X 플러그인 개발, 등록, 운영 과정에서 발생할 수 있는 문제들의 해결 방법을 제공합니다.

## 플러그인 등록 실패

* 플러그인을 등록하기 전에 서버가 실행 중인지 확인하세요.

* <Badge className='get'>GET</Badge> `/bsx` 엔드포인트가 올바르게 응답하는지 확인하세요. `curl` 명령어를 사용해 테스트할 수 있습니다.

  ```bash
  curl -H "X-BioStar-Ping: test123" http://localhost:8000/bsx
  ```

* **BioStar X** 서버에서 플러그인 서버 URL에 접근할 수 있는지 확인하세요.


## 인증 콜백 실패

* *public_key.pem* 파일이 존재하고 유효한지 확인하세요.

* **BioStar X Service Manager**의 <Cmd>SERVICE SETTINGS</Cmd>에서 **BioStar X** 서버 주소와 포트를 확인하세요.

* 플러그인 서버와 **BioStar X** 서버 간에 네트워크가 연결되어 있는지 확인하세요.


## 복호화 오류

* BioStar X에서 올바른 공개키를 사용하고 있는지 확인하세요.

* AES 키 생성 및 암호화 과정을 확인하세요.

* IV가 같은 키에서 파생되는지 확인하세요.


## 디버그 팁

* 요청 및 응답의 세부 사항을 확인하려면 로깅을 활성화하세요.

* 전체 통합 전에 엔드포인트를 개별적으로 테스트하세요.

* `curl` 명령어를 사용하여 BioStar X 서버에 요청 시뮬레이션하세요.
