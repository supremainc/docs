## 플러그인(Plugins)이란?

BioStar X 플러그인(Plugins)은 BioStar X 플랫폼의 확장 기능으로, 고객의 특별한 요구사항이나 기존 시스템과의 연동을 위해 추가 기능을 제공합니다.

:::info

플러그인 기능을 사용하려면 플러그인 라이선스가 필요합니다. 라이선스 정책에 대한 자세한 내용은 [다음 문서](../biostar_x/licensing)를 참고하세요.

:::

## 주요 활용 사례

BioStar X 플러그인(Plugins) 시스템을 통해 다양한 외부 시스템과 연동할 수 있습니다. 대표적인 활용 사례는 다음과 같습니다.

- **근태 관리 시스템**: 외부 T&A(Time & Attendance) 솔루션과 연동

- **방문자 관리 시스템**: 별도의 방문자 등록 및 관리 시스템과 연동

- **보고서 솔루션**: 사용자 정의 보고서 생성 및 분석 도구 연동

- **타사 시스템 연동**: ERP, HRM 등 기존 업무 시스템과의 데이터 연동


## 플러그인 유형

### 외부 플러그인

- **개발 주체**: 고객, 파트너, 타사 개발업체

- **등록 방식**: **BioStar X Service Manager**의 **PLUGINS** 페이지에서 수동 등록

- **등록 제한**: 최대 10개까지 등록 가능

- **관리**: 사용자가 직접 추가/삭제 관리

### 슈프리마 제공 플러그인  

- **개발 주체**: 슈프리마

- **제공 방식**: 설치 패키지를 통한 배포

- **자동 연동**: 설치 시 BioStar X와 자동 연동

- **관리**: 프로그램 설치/제거를 통해 관리


## 시스템 제한사항

- **전체 플러그인 최대 개수**: 12개

- **외부 플러그인 최대 개수**: 10개


## 세션 브리지 기능

플러그인 실행 시 별도의 로그인 절차 없이 BioStar X에 자동 인증하는 기능입니다.

- **자동 인증**: 사용자가 플러그인 클릭 시 별도 로그인 없이 접근할 수 있습니다.

- **권한 상속**: BioStar X에 로그인된 관리자의 권한을 그대로 적용합니다.

- **보안 통신**: RSA 공개키 암호화와 AES 대칭키 암호화를 통한 안전하게 세션을 전달합니다.
