BioStar X 모바일 앱에서 관리자 계정의 비밀번호를 변경할 수 있습니다. 관리자 계정의 비밀번호를 정기적으로 변경하여 계정 보안을 강화할 수 있습니다.

<Columns>
<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-app-account-change-pw.png' />

</Column>
<Column className='padding--l40'>

1. 앱 홈 화면에서 화면 왼쪽 상단의 <IcHamberg title='Hamburger icon' width={20} height={20} />을 탭하세요.

1. 사이드 메뉴에서 <Cmd>Change Password</Cmd>를 탭하세요.

1. <Cmd>Current Password</Cmd> 입력 필드에 현재 비밀번호를 입력하세요.

1. 새로운 비밀번호를 <Cmd>New Password</Cmd>, <Cmd>Confirm password</Cmd> 입력 필드에 차례로 입력하세요.

1. 비밀번호 변경을 완료하려면 <Cmd>Change Password</Cmd>를 탭하세요.

</Column>
</Columns>
