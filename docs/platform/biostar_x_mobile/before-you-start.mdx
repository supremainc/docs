BioStar X 모바일 앱을 사용하기 전에 다음 항목을 확인한 후 앱을 설치하세요.


## 지원 환경

다음 조건에서 BioStar X 모바일 앱을 사용할 수 있습니다.

* **지원 시스템**: BioStar X

* **서버 요구사항**: API 액세스가 가능한 BioStar X 서버 필요

* **사용자**: BioStar X 시스템에 사전 등록된 관리자 권한이 있는 사용자만 사용 가능

* **모바일 기기**: iOS 13.0 이상 또는 Android 7.0 이상

:::info

BioStar X 모바일 앱은 독립적으로 작동할 수 없으며, 고객사 내부에 운영 중인 BioStar X 서버에 연결해야 합니다.

* BioStar X를 설치하기 위한 <DocLink docId='platform/biostar_x/system-requirements' target='_blank' />을 참고하세요.

* BioStar X 설치에 대한 자세한 내용은 <DocLink docId='platform/biostar_x/express-install' target='_blank' />을 참고하세요.

:::


## 라이선스 및 네트워크 요구사항

BioStar X 모바일 앱을 사용하려면 BioStar X 라이선스가 필요합니다. 라이선스 추가 옵션을 구매해 사용할 수 있습니다. 라이선스 정책에 대한 자세한 내용은 <DocLink docId='platform/biostar_x/licensing#features-add-ons' target='_blank' />을 참고하세요.

| 접속 환경 | 필요 라이선스 |
| -- | -- |
| 같은 네트워크 | 모바일 앱 라이선스 |
| 외부 네트워크(공용 IP) | 모바일 앱 라이선스 |
| 외부 네트워크(공용 IP 없음) | 모바일 앱 라이선스 + 리모트 액세스 라이선스 |

* BioStar X 서버와 모바일 앱을 연동해 사용하려면 **모바일 앱** 라이선스가 필요합니다.

* 기본적으로 같은 네트워크 내에서 BioStar X 서버에 연결하여 모바일 앱을 사용할 수 있습니다.

* 외부 네트워크에서 접속하려면 공용 IP 주소가 필요하며, 그렇지 않은 경우 **리모트 액세스** 라이선스를 추가 구매해야 합니다.

:::info

* BioStar X에서 라이선스 적용 방법에 대한 자세한 내용은 <DocLink docId='platform/biostar_x/settings-license-biostar-x-license' target='_blank' />을 참고하세요.

* BioStar X 서버 설정에 대한 자세한 내용은 <DocLink docId='platform/biostar_x/settings-server-server' target='_blank' />을 참고하세요.

* BioStar X의 리모트 액세스 기능에 대한 자세한 내용은 <DocLink docId='platform/biostar_x/settings-remote-access' target='_blank' />을 참고하세요.

:::
