<Cmd>USER</Cmd> 메뉴에서는 BioStar X에 등록된 사용자를 관리할 수 있습니다. 사용자 추가, 수정, 삭제 등 다양한 작업을 수행할 수 있습니다. 앱 홈 화면에서 <Cmd>USER</Cmd>를 탭하세요.

<Image src='/img/mobile/bsx-mobile-app-all-users.png' width='500px' className='none' alone />

## 도구 아이콘

사용자 화면에서 제공하는 도구 아이콘을 사용해 아래와 같은 작업을 수행할 수 있습니다.

* <IcAdd title='Add user' width='24px' height='24px' />: 새로운 사용자를 추가할 수 있습니다.

* <IcTrash title='Delete user' width='24px' height='24px' />: 선택한 사용자를 삭제할 수 있습니다.

* <IcFilter title='Filter' width='24px' height='24px' />: 사용자 목록을 검색하거나 필터링할 수 있습니다.

* <IcSearch title='Search' width='24px' height='24px' />: 사용자 목록에서 특정 사용자를 검색할 수 있습니다.

* <IcUp title='Scroll to top' width='24px' height='24px' /> / <IcDown title='Scroll to bottom' width='24px' height='24px' />: 사용자 목록을 위 또는 아래로 스크롤할 수 있습니다.

## 사용자 정보 확인

사용자 목록의 각 사용자는 아래와 같은 정보를 포함하고 있습니다.

<Image src='/img/mobile/bsx-mobile-app-all-users-info.png' width='500px' className='none' alone />

* 사용자 이름

* 사용자 ID | 사용자 그룹

* 등록된 크리덴셜 정보: 지문 / IR 페이스 / 얼굴 / 카드 / PIN
