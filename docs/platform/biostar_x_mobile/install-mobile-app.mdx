## 앱 권한

BioStar X 모바일 앱은 특정 기능 사용 시 다음 권한을 요청합니다. 최초 설치 시에는 권한을 요청하지 않습니다.

| 권한 | 필수 여부 | 용도 |
| -- | -- | -- |
| 저장공간 | 선택 사항 | 사용자 이미지(예: 갤러리 또는 기기 저장소)를 관리하고 업로드하는 데 필요합니다. |
| 카메라 | 선택 사항 | 얼굴 이미지 등록 및 비디오 녹화를 위한 이미지 캡처에 사용합니다. |
| 생체 인식 | 선택 사항 | 모바일 기기 지문 또는 얼굴 인식을 사용한 보안 로그인에 필요합니다. |

:::info

모든 권한은 필수가 아니며, 사용자는 앱의 기본 기능에 영향을 미치지 않고 권한을 선택적으로 거부할 수 있습니다.

:::

## 앱 시작 및 초기 설정

앱을 시작하면 BioStar X 서버 연결을 위한 정보를 입력하는 화면이 표시됩니다.

<Columns>

<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-init-setting.png' alone />

</Column>
<Column className='padding--l40'>

* <Cmd>BioStar X Server Address</Cmd>: BioStar X 서버의 IP 주소 또는 도메인 주소를 입력하세요.

* <Cmd>Port</Cmd>: BioStar X 서버의 포트 번호를 입력하세요. 기본값은 `443`입니다.

* <Cmd>ID</Cmd> / <Cmd>Password</Cmd>: BioStar X 서버에 로그인하기 위한 관리자 등급의 아이디와 비밀번호를 입력하세요.

필요한 정보를 모두 입력한 후 로그인하려면 <Cmd>Login</Cmd> 버튼을 탭하세요.

:::info

* BioStar X 서버의 IP 주소, 포트 번호를 확인하려면 <DocLink docId='platform/biostar_x/settings-server-server' target='_blank' />를 참고하세요.

* 관리자 아이디와 비밀번호를 입력 필드에 저장하려면 <Cmd>Remember username and password</Cmd> 옵션의 체크박스를 선택하세요. 다음 로그인 시 입력한 아이디와 비밀번호가 자동으로 채워집니다.

:::

</Column>
</Columns>


## 연결 문제 해결

로그인에 실패했다면 아래 사항을 확인하세요.

* BioStar X 서버 주소와 포트 번호가 올바른지 확인하세요.

* 관리자 등급의 아이디와 비밀번호가 올바른지 확인하세요.

* 네트워크 연결 상태를 확인하세요. 모바일 기기가 BioStar X를 설치한 서버와 같은 네트워크에 연결되어 있는지 확인하세요.

:::info

외부 네트워크에서 접속하려면 공용 IP 주소가 필요하며, 그렇지 않은 경우 **리모트 액세스** 라이선스를 추가 구매해야 합니다. 자세한 내용은 <DocLink docId='platform/biostar_x_mobile/before-you-start' />를 참고하세요.

:::


## 앱 사용

로그인에 성공하면 BioStar X 모바일 앱의 홈 화면이 표시됩니다. 홈 화면에서는 등록된 사용자, 출입문, 실시간 로그, 이벤트 로그 및 경보를 확인할 수 있습니다. 또한 사이드 메뉴와 설정 메뉴에 접근할 수 있습니다.

<Columns>

<Column className='width--23'>

<Image src='/img/mobile/bsx-mobile-home-screen.png' alone />

</Column>
<Column className='padding--l40'>

* <IcHamberg title='Hamburger icon' width='24px' height='24px' />: 사이드 메뉴를 열 수 있습니다. 사이드 메뉴에서는 비밀 번호를 변경하거나 로그아웃할 수 있으며, 주요 메뉴에 접근할 수 있습니다.

* <IcSet2 title='Setting icon' width='24px' height='24px' />: 설정 메뉴를 열 수 있습니다.

* <Cmd>USER</Cmd>: 등록된 사용자 목록을 확인하고, 사용자를 관리할 수 있습니다.

* <Cmd>DOOR</Cmd>: 등록된 출입문 목록을 확인하고, 원격으로 제어할 수 있습니다.

* <Cmd>REAL-TIME LOG</Cmd>: 실시간 출입 이벤트 로그를 확인할 수 있습니다.

* <Cmd>EVENT LOG</Cmd>: BioStar X 서버에 저장된 모든 이벤트 로그를 확인할 수 있습니다.

* <Cmd>ALERT</Cmd>: 출입문 및 장치, 고급 출입 통제 경보를 확인할 수 있습니다. 

</Column>
</Columns>
