## 100% 클라우드 출입 통제 플랫폼 {#biostar-air}

BioStar Air는 생체 인식 보안 분야의 세계적인 선도 기업인 슈프리마가 개발한 클라우드 네이티브 출입 통제 솔루션입니다. 온프레미스 인프라의 필요성을 없애도록 설계된 BioStar Air는 어디서나 관리할 수 있는 안전하고 유연하며 확장 가능한 출입 통제를 제공합니다. 클라우드에 직접 연결된 스마트 리더기를 통해 기업은 휴대폰이나 브라우저만으로 여러 위치의 출입을 손쉽게 제어할 수 있습니다.

## 비교할 수 없는 크리덴셜 유연성 {#unmatched-credential-flexibility}

BioStar Air는 가장 완전한 범위의 인증 옵션을 제공하며, 안전하고 편리한 여러가지 방법을 통해 사용자의 시설 출입을 관리할 수 있습니다.

* **얼굴 인증**

* **지문**(출시 예정)

* **iOS 및 안드로이드를 위한 맞춤형 모바일 인증**(무료)

* **Apple Wallet**(출시 예정)

* **다양한 형식의 RFID**

* **앱 없이도 사용할 수 있는 Dynamic QR 및 LinkPass**


## 온프레미스 인프라 불필요 {#zero-on-prem-infrastructure}

BioStar Air는 온사이트(on-site) 서버, 가상 머신, VPN 또는 출입문 컨트롤러의 필요성을 없앴습니다. IT 유지 보수가 필요하지 않습니다. **스마트 리더기**(Smart reader)를 설치하고 클라우드에 연결하면, 즉시 출입 관리를 시작할 수 있습니다. 리더기(Reader)는 동일한 네트워크나 물리적 위치와 관계 없이 여러 사이트에서 작동합니다.


## 손쉬운 다중 사이트 관리 {#effortless-multi-site-management}

BioStar Air는 단일 시스템 하에서 **사무실**, **창고**, **소매점** 또는 **원격 지역** 등 모든 위치의 출입을 제어할 수 있는 완벽한 솔루션을 제공합니다.


## 엣지 아키텍처를 통한 미래 지향적 확장성 {#future-proof-scalability-with-edge-architecture}

BioStar Air의 **엣지 아키텍처**는 출입 통제를 스마트 리더기에 직접 내장하여 별도의 출입문 컨트롤러가 필요하지 않습니다. 이를 통해 기업은 **손쉽게 확장**할 수 있습니다. 현재 필요한 기능만 구매하고 복잡한 업그레이드 없이 간편하게 확장할 수 있습니다.


## 클라우드 중심 기업에 최적화 {#tailored-for-cloud-first-organizations}

BioStar Air는 업계 최고의 생체 인식 기술을 탑재한 완벽한 클라우드 네이티브 출입 통제 솔루션을 원하는 **시스템 통합 업체**와 **고객**을 위해 설계되었습니다. 클라우드 중심 기업의 출입 관리를 위한 현대적이고 유연하며 매우 안전한 방법을 제공합니다.

## BioStar Air 기능

| 기능 | BioStar Air |
| -- | -- |
| **완벽한 출입 통제 솔루션** | 지원 |
| **Suprema 하드웨어와 호환** | XPass 2, XStation 2, BioEntry W3, BioStation 3, BioLite N2 (예정), SIO2, OM-120, CoreStation-20(예정) |
| **높은 보안성** | ISO 27001 / 27701, TLS 1.2, AES-256 |
| **생체 인식: 얼굴 및 지문** | Suprema 리더기에 대한 기본 지원하며, 생체 인식 데이터베이스가 내장되어 있습니다. |
| **RFID 카드** | MIFARE Classic, DesFire, Felica, EM4100, Custom Wiegand Formats, iClass, Seos. 사용자당 최대 8장의 카드를 지원합니다. |
| **모바일 자격 증명** | 무료이며 무제한 제공합니다. 보안 BLE/NFC 자격 증명(iOS, Android), 앱 없는 LinkPass 및 QR 인증을 포함합니다. |
| **배포** | 클라우드 네이티브 |
| **인터페이스** | 브라우저, iOS 앱, Android 앱 |
| **가격 모델** | 구독 기반 |
| **Integration** | 광범위한 API, 웹후크 및 SDK |
| **엣지 아키텍처** | 리더기에는 내장 컨트롤러가 있으며 이더넷을 통해 클라우드에 직접 연결됩니다. 별도의 클라우드 노드나 출입 통제 패널이 필요하지 않습니다. |
| **로그인** | 어디서나 가능합니다. |
| **서버 위치** | 한국(글로벌) 및 독일(유럽) |
| **근태** | 리더기의 근태 키(key)는 이벤트 로그에 기록되며, 근무 시간 보고서가 작성되어 통합 가능합니다. |
| **비디오** | Onvif 및 Uniview를 지원하는 기본 VMS입니다. NVR 필요 없으며, 리더기가 게이트웨이 역할을 합니다. |
| **알람** | 침입 경보 구역에 대한 관리자 알림을 지원합니다. |
| **데이터 파티셔닝** | 지원 |
| **대상** | 프랜차이즈, 다중 지점, 다중 세대, 원격 지역 등 |
