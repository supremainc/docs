## 개요 {#overview}

BioStar Air는 Airfob Edge 리더기와 동일한 네트워크에 연결된 IP 카메라를 자동 감지합니다. 카메라는 리더기를 게이트웨이로 사용하므로 네트워크 비디오 레코더(Network Video Recorder, NVR)가 필요하지 않습니다. 대신 카메라는 내장 SD 카드 저장 장치를 사용합니다.

Airfob Edge 리더기 또는 XPass 2를 각각 설치했다면 BioStar Air는 아래와 같은 동시 라이브 뷰 스트림을 표시할 수 있습니다.

* 최대 2대의 카메라에서 Full HD 해상도(1080p)

* 최대 4대의 카메라에서 HD 해상도(720p)

이 사양은 XStation 2를 사용할 때 두 배로 증가합니다.


## 호환성 {#compatibility}

* **Onvif Profiles**: IP 카메라는 Onvif 프로필 S 및/또는 G와 호환되어야 합니다.

  * **Onvif Profile S**: 라이브 비디오 스트리밍

  * **Onvif Profile G**: 녹화된 비디오를 SD 카드에 저장

* **SD 카드**: IP 카메라는 나중에 녹화한 영상을 재생할 수 있도록 내장 SD 카드가 있어야 합니다. 

  :::note

  SanDisk microSDXC Class10 Max Endurance 256GB 또는 512GB를 권장합니다.

  :::

* **인코딩**: 카메라는 H.264(AVC)를 지원해야 하며 활성 CODEC으로 설정되어 있어야 합니다. BioStar Air는 현재 H.265를 지원하지 않습니다.

* **추천 브랜드**: Hanwha, Hikvision 및 Dahua 카메라와의 호환성이 뛰어납니다. 다른 브랜드와의 호환성은 Onvif 프로필 준수 여부에 따라 달라질 수 있습니다.

* **새 카메라 테스트**: 테스트하고 싶은 다른 카메라 브랜드가 있다면 서비스 팀에 문의하세요.


## 기능 소개 {#functionality}

BioStar Air 앱에서 아래와 같은 작업을 수행할 수 있습니다.

* 실시간 카메라 피드를 볼 수 있습니다.

* 녹화된 카메라 영상을 볼 수 있습니다.

* 출근 이벤트와 관련된 카메라 영상에 접근합니다.


## 사용 안내 {#instructions}

<Steps>
<Step>

### IP 카메라 연결

* **네트워크 연결**: IP 카메라가 Airfob Edge 리더기 또는 Airfob Edge 리더기 Ultimate와 동일한 네트워크에 연결되어 있는지 확인하세요.

* **Onvif 활성화**: 카메라에서 Onvif가 활성화되어 있는지 확인하세요.

* **리더기 등록**: 리더기가 사이트에 등록되어 있는지 확인하세요.

</Step>
<Step>

### BioStar Air 로그인

**마스터 관리자**만 비디오 관리 활성화와 같은 사이트 설정을 변경할 수 있습니다. 마스터 권한으로 로그인하세요.

</Step>
<Step>

### 비디오 관리 활성화

**Settings** → **Site** → **Additional Features** → **Video Management**로 이동하여 비디오 관리를 활성화하세요.

:::info

이 기능은 마스터 레벨의 관리자만 활성화할 수 있습니다.

:::

</Step>
<Step>

### 카메라 추가

* **Device Management** → **Cameras** → **+ Add Camera**으로 이동하세요.

* **Select Device (Reader)**: 게이트웨이 역할을 할 리더기를 선택하세요. 리더기는 펌웨어 2.5.1 버전 이상이 설치되어 있어야 하며, 카메라와 동일한 네트워크에 있어야 합니다.

* **Automatic Search**: BioStar Air는 자동으로 네트워크에서 카메라를 검색합니다.

* **Camera Registration**: 카메라를 등록하려면 ID와 비밀번호를 입력해야 할 수 있습니다.

</Step>
<Step>

### 출입문과 연결

* **Door Management** → **Registered Doors**으로 이동하세요.

* **Assign Devices**: 출입문을 선택한 다음, **Assign Devices** 섹션에서 카메라를 선택하세요. 카메라가 출입문에 연결되어 이벤트 로그에서 관련 영상을 볼 수 있습니다.

</Step>
<Step>

### Live Camera 영상 보기

* **Monitoring** → **Real-Time Video**로 이동하여 실시간 비디오 스트림을 시청하세요.

* **동시 스트림**: 설치된 각 Airfob Edge Reader에 대해 최대 2개의 Full HD (1080p) 실시간 스트림을 볼 수 있습니다.

* **Additional Streams**: 더 많은 실시간 스트림을 보려면 더 많은 리더기를 추가하거나 카메라 해상도를 줄이세요. (**Device Management** → **Cameras** → 카메라 클릭 → **Profile**)

</Step>
<Step>

### 녹화된 카메라 영상 보기

**Monitoring** → **Recorded Video**로 이동하여 녹화된 비디오 스트림을 시청하세요.

</Step>
<Step>

### 이벤트 영상 보기

**Monitoring** → **Event Logs**로 이동하세요. 관련 비디오 클립이 있는 이벤트 로그를 찾아 재생 버튼을 클릭하면 영상을 시청할 수 있습니다.

</Step>
<Step>

### 추가 설정 살펴보기

앱 내에서 탐색할 수 있는 다양한 설정이 있습니다. 이 문서는 공식 사용 설명서가 출시될 때까지 임시로 참고할 수 있도록 작성되었습니다.

</Step>
</Steps>

## IP 카메라에서 ONVIF를 활성화하기 위한 일반 단계

1. **카메라의 웹 인터페이스에 로그인**: 웹 브라우저를 열고, 카메라의 IP 주소를 입력하세요. 관리자 크리덴셜로 인증하여 로그인하세요.

1. **네트워크 또는 구성 설정으로 이동**: **Network** 또는 **Configuration** 섹션을 찾으세요.

1. **ONVIF 프로토콜 활성화**: ONVIF 또는 개방형 네트워크 비디오 인터페이스를 활성화하세요.

1. **ONVIF 프로필 또는 사용자 설정**: 필요한 경우 ONVIF 전용 사용자 프로필을 생성하세요.

1. **저장 및 연결 테스트**: 설정을 저장하고 Airfob Space에서 ONVIF 연결을 테스트하세요.


## 추가 팁

* **펌웨어 업데이트**: 카메라와 리더기를 최신 펌웨어로 업데이트하세요.

* **포트 포워딩**: 필요한 경우 ONVIF에 대한 포트 포워딩을 구성하세요.

* **Third-Party 도구**: ONVIF Device Manager를 사용하여 카메라 설정 및 기능을 확인하세요.
