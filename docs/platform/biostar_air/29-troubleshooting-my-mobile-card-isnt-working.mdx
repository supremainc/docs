# 문제 해결 - 내 모바일 카드가 작동하지 않아요.

모바일 카드가 리더기에 인증되었지만 출입문이 열리지 않는다면 다음 단계를 따라 문제를 해결하세요.

<CustomerNote />


## BioStar Air 관리자에게 문의 {#contact-your-biostar-air-administrator}

귀하의 스페이스 관리자는 모바일 카드 재발급, 일시적인 모바일 기기 문제 및 설정 문제를 도와줄 수 있습니다.

### 스페이스 관리자에게 연락하기 {#how-to-contact-your-spaces-admin}

* 모바일 기기에서 **Suprema Pass** 앱을 실행하세요.

* 사이트 관리자에게 직접 문의하려면 모바일 카드 아래의 정사각형 아이콘을 클릭하세요.


## 모바일 카드가 삭제되었나요? {#was-your-mobile-card-deleted}

모바일 기기를 변경하거나 Suprema Pass 앱을 재설치했다면 보안상의 이유로 모든 모바일 카드가 삭제됩니다.

### 모바일 카드 재발급을 위한 연락처 {#who-to-contact-for-mobile-card-reissuance}

* **Company Spaces**: HR, IT, 보안 또는 운영팀

* **Shared Spaces**: 지역 관리자 또는 고객 서비스 팀


## 단계별 문제 해결

<Steps>
<Step>

### Suprema Pass 앱에서 오류 메시지가 나타났나요?

출입문이 열리지 않는다면 Suprema Pass 앱에서 다시 인증을 시도하세요.

에러 메시지가 나타나면 스크린샷을 찍어 관리자에게 공유하세요.

</Step>
<Step>

### 모바일 기기의 블루투스를 켰나요?

모바일 카드는 모바일 기기와 리더기 간의 블루투스 통신에 의존합니다.

* 블루투스가 켜져 있는지 확인하세요.

* 블루투스를 껐다가 다시 켜보세요.

* 필요하다면 모바일 기기를 재시작하세요.

</Step>
<Step>

### Suprema Pass 앱의 위치 권한을 허용하세요.

블루투스 및 NFC 인증에는 위치 권한이 필요합니다.

* 모바일 기기의 설정으로 이동하세요.

* **Suprema Pass** 앱을 실행하세요.

* 위치 접근을 **허용** 또는 **항상 허용**으로 설정하세요.

</Step>
<Step>

### 무선 네트워크와 Wi-Fi 연결을 확인하세요.

모바일 카드 인증을 위해서는 안정적인 무선 네트워크 또는 Wi-Fi 연결이 필요합니다.

</Step>
<Step>

### 선택한 연결 방법을 사용했나요?

Suprema Pass 앱에서 연결 설정을 확인하세요:

* 오른쪽 상단 모서리에 있는 **≡** 아이콘을 탭하세요.

* 인증 모드를 확인하려면 **Connection Method**를 선택하세요.

* **Background Mode**를 사용한다면 앱이 백그라운드에서 열려 있어야 합니다.

</Step>
<Step>

### 인증 시 모바일 기기에 진동이 발생했나요?

모바일 기기가 진동했지만 출입문이 열리지 않았다면 Suprema Pass 앱을 열고 다시 인증을 시도하세요.

에러 메시지가 나타나면 스크린샷을 찍어 관리자에게 공유하세요.

</Step>
</Steps>
