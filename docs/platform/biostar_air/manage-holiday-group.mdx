## 휴일 그룹 추가

휴일 그룹은 매년 반복되는 공휴일과 특정 기간의 휴일을 정의하여 해당 기간 동안 출입 권한을 제한하거나 변경하는 데 사용됩니다.

import AddHolidayGroup from '../../common/_add-holiday-group.mdx';

<AddHolidayGroup />

:::info

* 휴일 그룹은 스케줄 보다 우선합니다. 항상 출입이 필요한 사용자에게는 제한 없는 출입 권한을 부여하세요.

* 매년 같은 날짜에 반복되는 공휴일(예, 신정, 어린이날)과 매년 날짜가 달라지는 휴일(예, 추석, 설날)을 별도의 그룹으로 관리하는 것을 권장합니다.

:::


## 휴일 그룹 수정

1. 화면 왼쪽 사이드바에서 <Cmd sid='access_menu' product='air' /> → <Cmd sid='holiday_groups_submenu' product='air' />를 클릭하세요.

1. 목록에서 수정할 휴일 그룹을 클릭하세요. 또는 휴일 그룹의 체크박스를 클릭해 선택한 후 목록 상단의 <IcAirEdit /> 버튼을 클릭하세요.

1. <Cmd sid='edit_holiday_group_title' product='air' /> 패널이 나타나면 원하는 항목을 수정하세요.

   <Image src='/img/air/bsair-access-mng-holiday-edit.png' />

1. 수정 사항을 저장하려면 <Cmd sid='save_btn' product='air' /> 버튼을 클릭하세요.


## 휴일 그룹 삭제

1. 화면 왼쪽 사이드바에서 <Cmd sid='access_menu' product='air' /> → <Cmd sid='holiday_groups_submenu' product='air' />를 클릭하세요.

1. 목록에서 휴일 그룹의 체크박스를 클릭하세요. 1개 이상의 휴일 그룹을 선택할 수 있습니다.

1. 목록 상단의 <IcAirRemove /> 버튼을 클릭하세요.

1. 경고 메시지 창이 나타나면 <Cmd sid='btn_yes' product='air' /> 버튼을 클릭하세요.
