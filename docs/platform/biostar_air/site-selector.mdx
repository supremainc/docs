BioStar Air 포털에서 여러 사이트를 관리하려면 각 사이트에 대해 별도의 이메일 주소와 비밀번호가 필요했습니다. 이로 인해 과정이 번거롭고 시간이 많이 소요되었습니다. 새로운 **사이트 선택**(**Site Selector**) 기능을 사용하면 이제 하나의 이메일 주소로 여러 사이트를 관리할 수 있어 프로세스가 간소화되고 여러 번 로그인할 필요가 줄어듭니다.

<CustomerNote />

## 사이트 선택(Site Selector)이란? {#siteSelector}

**사이트 선택** 기능을 사용하면 여러 사이트를 관리할 때에도 하나의 통합된 이메일 주소로 BioStar Air 포털에 로그인할 수 있습니다. 각 사이트에 대해 서로 다른 이메일 주소로 로그인하지 않아도 됩니다. 하나의 이메일로 로그인하여 **사이트 선택** 메뉴를 이용하면 관리하는 사이트 간에 쉽게 전환할 수 있습니다.

<Image src='/img/air/bsair-site-selector.png' />


## 새로운 사이트 생성 {#newSite}

파트너 포털을 통해 새로운 사이트를 생성할 때, 다른 사이트의 관리자 계정으로 등록한 기존 이메일 주소를 사용할 수 있습니다. 시스템에서 새로운 사이트를 기존 이메일에 자동으로 연결하여 하나의 계정으로 여러 사이트를 관리할 수 있습니다.

* [파트너 포털](https://partner.biostarair.com/)을 통해 새로운 사이트를 생성하세요.

* 관리자의 이메일을 입력하라는 메시지가 표시되면, 다른 사이트와 이미 연결된 이메일 주소를 사용하세요.

* 새로운 사이트를 기존 계정에 자동 연결됩니다.

:::info

파트너 포털 가입 및 사용에 대한 자세한 내용은 [다음 문서](intro-to-partner-portal)를 참고하세요.

:::

## 기존 사이트에서 사용 {#existingSites}

이미 여러 사이트를 관리하고 있지만 각 사이트마다 별도의 로그인 인증을 사용하고 있다면, **슈프리마**에 수동 마이그레이션을 요청해야 합니다. 이 프로세스를 통해 이메일 주소를 통합하여 모든 사이트를 단일 로그인으로 관리할 수 있습니다.

* 모카(MOCA) 지원팀에 연락하여 사이트에 대한 수동 마이그레이션을 요청하세요.

* bob+site1@example.com 및 bob+site2@example.com과 같은 별칭 주소를 사용한다면 루트 이메일 주소(bob@example.com)를 제공하세요.

* 각 사이트에 대해 완전히 고유한 이메일 주소를 사용한다면 전체 이메일 목록을 제공해야 합니다.

* 마이그레이션이 완료되면 **사이트 선택** 기능을 통해 단일 이메일 주소로 모든 사이트에 액세스할 수 있습니다.


## 사이트 선택(Site Selector) 사용 단계 {#howToLogInAndUseTheSiteSelector}

통합된 이메일 주소가 있다면 로그인하고 사이트 간에 전환하는 것은 간단합니다.

<Image src='/img/air/bsair-site-selector-top.png' />

1. **로그인**: 통합된 이메일 주소와 비밀번호를 입력하세요.

2. **사이트 선택**: 로그인하면 **사이트 선택** 기능이 자동으로 나타나며, 이메일 주소와 연결된 모든 사이트 목록을 확인할 수 있습니다.

3. **검색 및 탐색**: 여러 사이트를 관리한다면 검색창을 통해 원하는 사이트를 빠르게 찾을 수 있습니다. 사이트를 선택하면 대시보드로 이동합니다.


:::info

* **보안**: 통합된 이메일을 사용하더라도 각 사이트의 권한 및 설정은 독립적으로 유지됩니다. 관리자 역할과 접근 권한은 사이트마다 고유하게 관리됩니다.

* **문제 해결**: **사이트 선택** 기능이 보이지 않거나 잘못된 사이트가 표시된다면, 마이그레이션 요청이 완료되었는지 확인하거나 모카(MOCA) 지원팀에 추가 지원을 요청하세요.

**사이트 선택** 기능은 로그인 과정을 간소화하고 여러 사이트를 하나의 계정으로 통합함으로써 BioStar Air에서 여러 사이트를 관리하는 시간과 노력을 절약할 수 있습니다.

:::
