## 펌웨어 버전 2.11.2 (빌드 번호 2.11.2_260112)

릴리스<Separator/>2025-01-12

이번 릴리스는 BioStar Air 환경의 사용자 동기화 개선, Suprema 펌웨어 표준에 맞춘 장치 동작 일치 및 네트워크 설정 안정성 향상을 포함합니다.

### 주요 기능 및 개선 사항

#### 인증

1. 장치에 저장된 사용자 및 출입 권한을 기반으로 하는 인증 지원

1. 이번 버전에서 서버 기반 인증 미지원(정상적인 작동에 불필요)

#### 사용자 관리

1. 사용자 동기화 기능 추가

1. 동기화를 통해 장치에 사용자 및 출입 권한 직접 저장

#### 시스템 및 펌웨어 동작

1. Suprema XPass 2 기본 펌웨어 v1.5.0 기반

1. LED 표시등을 포함한 장치 동작을 표준 Suprema 펌웨어와 일치하게 적용

   **LED 표시등**

   | 색상 | 설명 |
   | --- | --- |
   | 초록색 | 인증 성공 |
   | 빨간색 | 인증 실패 |
   | 파란색/하늘색 점멸 | 정상 동작 상태 |
   | 파란색/초록색 점멸 | 네트워크 초기화 진행 중 |
   | 초록색 점멸 | 입력 대기 상태 |
   | 초록색/흰색 점멸 | Suprema Pass(BLE) 연결 중 |
   | 흰색 점멸 | Device Manager 연결 중 |

1. 기존 Airfob Space 전용 LED 표시등 제거

#### 호환성 및 마이그레이션

1. BioStar 2 환경에서 사용 가능하도록 지원

1. v2.9.0 및 이전 버전에서 v2.11.2로의 펌웨어 업그레이드 지원

   * 이전 버전(예: v2.9.0, v2.8.1)으로 다운그레이드 미지원

#### 펌웨어 전환 관련 안내

1. Suprema 펌웨어 v1.5.0에서 AIR v2.11.2로 펌웨어 전환 지원

   * 기존 모델(AEM, AE-MC, AE-MU)의 경우 Suprema 펌웨어로 업그레이드 미지원(전환 시 사용 불가 상태가 될 수 있음)

     해당 모델은 BioStar Air 펌웨어 v2.9.0 또는 이전 버전 유지 필요


### 버그 수정

#### 네트워크

1. 고정 IP 사용 시 장치의 네트워크 설정이 의도치 않게 변경되는 문제

## 펌웨어 버전 2.8.0 (빌드 번호 2.8.0_250115)

릴리스<Separator/>2025-01-15

이 릴리스에서는 새로운 **이벤트 로그** 기능이 도입되었으며, 장치 구성 동작 개선과 그외 알려진 문제를 해결하였습니다.

### 주요 기능 및 개선 사항

#### 공통

1. 다음 상황을 확인할 수 있는 **이벤트 로그** 기능 추가

   * 강제 출입문 열림

   * 출입문이 열린 채로 유지됨

1. 모바일 앱으로 장치 등록 시 장치 시간대가 사이트 시간대로 자동 설정하도록 개선

1. 모바일 기기가 오프라인일 때 장치가 이벤트 로그를 로컬에 저장하도록 개선

1. 탬퍼 상태 On/Off 보고 문제 개선


### 버그 수정

#### 에어팝 에지(Airfob Edge) 리더기 멀리온(Mullion) 관련 수정

1. 경고음을 짧게 설정한 경우 링크 패스(LinkPass) 크리덴셜로 출입문을 열 때 소리가 재생되지 않는 문제 

1. 일광 절약 시간(DST) 구성을 해제할 때 장치에서 발생하는 시간 오류 문제

## 펌웨어 버전 2.4.0 (빌드 번호 2.4.0_240603)

릴리스<Separator/>2024-06-03

새로운 인증 기능을 도입했으며, 초기화 또는 API 처리 중 시스템 동작을 개선하였습니다.

### 주요 기능 및 개선 사항

1. 비디오 게이트웨이 기능 추가

1. 사용자가 XPass 2 키패드를 통해 RFID 카드 값을 입력해 인증할 수 있도록 지원

   > 예시: `123456789#` → 인증 / `*` → 입력 취소

1. API 시간 초과가 발생할 때 오류 카운터가 증가하지 않고, 실제 API 응답 오류만 카운트하도록 개선

1. 초기화 중 장치 시간 및 언어 설정을 재설정하도록 개선
