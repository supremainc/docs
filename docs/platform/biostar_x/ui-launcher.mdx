<Cmd sid='shell.header.launcher' /> 페이지는 **BioStar X**의 시작 페이지로, 사용자가 **BioStar X**의 주요 기능에 접근할 수 있는 중심 허브 역할을 합니다. 이 페이지는 사용자가 **BioStar X**의 모든 기능을 쉽게 탐색하고 사용할 수 있도록 설계되었습니다. 런처 페이지의 UI 구성 요소를 살펴보면 다음과 같습니다.

<Image src='/img/biostarx-launcher.png' className='none' />

<Cmd sid='shell.header.launcher' /> 페이지에서는 <Cmd sid='shell.header.user' />, <Cmd sid='shell.header.arena' />, <Cmd sid='shell.header.data' />, <Cmd sid='shell.header.dashboard' />, <Cmd sid='shell.header.setting' /> 페이지로 이동할 수 있는 링크를 제공합니다.

* <Cmd sid='shell.header.user' />: 사용자 그룹 관리, 사용자 관리, 사용자 템플릿 관리, 출입 권한별 사용자 확인 등 다양한 기능을 통해 사용자를 효율적으로 관리하고, 권한을 통해 보안을 강화할 수 있습니다. 자세한 내용은 [다음 문서](how-to-guide-user)를 참고하세요.

* <Cmd sid='shell.header.arena' />: 출입문과 관련한 장치, 엘리베이터, 구역을 모니터링합니다. 관련한 기능들을 제어하고 실시간 영상을 통해 모니터링할 수 있습니다. 자세한 내용은 [다음 문서](how-to-guide-monitoring)를 참고하세요.

* <Cmd sid='shell.header.data' />: 사용자와 관련한 다양한 데이터 및 통계 정보를 확인할 수 있습니다. 자세한 내용은 [다음 문서](#)를 참고하세요.

* <Cmd sid='shell.header.dashboard' />: **BioStar X**에서 제공하는 다양한 데이터를 통해 사용자가 선택한 데이터 기반의 위젯을 추가하고, 배치하여 원하는 대시보드를 구성할 수 있습니다. 자세한 내용은 [다음 문서](#)를 참고하세요.

* <Cmd sid='shell.header.setting' />: 장치 및 구역, 사용자에 따른 권한, 언어, 시간 등 다양한 설정을 통해 사용자의 환경에 최적화된 **BioStar X**를 구성할 수 있습니다. 자세한 내용은 [다음 문서](#)를 참고하세요.

:::info

화면 상단 헤더 영역에 대한 자세한 내용은 [다음 문서](ui-common)를 참고하세요.

:::
