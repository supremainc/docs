**BioStar X**의 비디오 설정 기능을 통해 출입 통제와 영상 보안을 통합 관리할 수 있습니다. **Video Management System**(VMS)과 연동하여 실시간 영상 모니터링, 이벤트 연계 녹화, 자동 북마크 생성 등 강력한 보안 관리 기능을 활용하세요.

:::info

비디오 설정 기능은 **어드밴스**(Advanced) 라이선스 이상에서 추가 옵션을 통해 사용할 수 있습니다. 라이선스 정책에 대한 자세한 내용은 [다음 문서](licensing#features-add-ons)를 참고하세요.

:::

## 비디오 설정 개요

비디오 설정은 다음 3단계로 구성됩니다.

<Steps>
<Step>

### VMS 서버 연동하기

VMS 서버와 **BioStar X** 간의 연결을 설정하고 인증서를 구성합니다. 통합 영상 보안 관리 시스템을 구축하는 필수 단계입니다. 자세한 내용은 [다음 문서](settings-video-integration)를 참고하세요.

</Step>
<Step>

### 카메라 설정하기

VMS에 연결된 카메라 중 필요한 카메라를 선택하고, 효율적인 관리를 위한 그룹을 구성하세요. 자세한 내용은 [다음 문서](settings-videos-camera)를 참고하세요.

</Step>
<Step>

### 규칙 설정하기

출입 이벤트와 영상을 연계하는 규칙을 생성하여 자동 북마크, 이벤트 기록 등의 고급 기능을 활용합니다. 자세한 내용은 [다음 문서](settings-videos-rule)를 참고하세요.

</Step>

</Steps>


## 주요 기능

- **통합 모니터링**: 출입 통제와 영상 보안을 하나의 인터페이스에서 관리할 수 있습니다.

- **자동 이벤트 기록**: 출입 이벤트 발생 시 관련 영상을 자동으로 기록하고 태그를 생성합니다.

- **효율적인 검색**: 북마크와 이벤트 태그를 통한 빠른 영상 검색 및 증거 수집이 가능합니다.

- **실시간 대응**: 보안 상황 발생 시 즉시 관련 카메라 영상 확인할 수 있습니다.
