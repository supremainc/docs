**BioStar X**의 핵심 기능 중 하나인 <Cmd>사용자</Cmd>(User) 페이지에서 활용할 수 있는 다양한 기능에 대해 안내합니다. 이 페이지에서는 사용자 그룹 관리, 사용자 관리, 사용자 템플릿 관리, 출입 권한별 사용자 확인 등 다양한 기능을 통해 사용자를 효율적으로 관리하고, 권한을 통해 보안을 강화하는 방법을 소개합니다. 각 기능을 통해 **BioStar X**를 더욱 효과적으로 활용해 보세요.
