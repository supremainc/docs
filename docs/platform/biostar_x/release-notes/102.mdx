# 버전 1.0.2 (빌드 번호 1.0.2.168)

릴리스<Separator/>2026-06-10

## 신규 기능 및 개선

---

1. 신규 장치 연결 지원

   - BioStation 3 MAX

   - XPass Q2

1. BioStar X에서 장치의 이벤트 로그를 삭제할 수 있는 기능 추가

1. 메인 서버의 IP 주소를 변경할 수 있는 기능 지원

1. 디렉토리 서비스 연동 시 사용자 동기화 방식을 2가지 옵션으로 선택할 수 있도록 지원

   - **추가/업데이트만** (기본값)

   - **추가/업데이트/삭제**

1. 백업 및 복원 기능 개선

   - 서버 IP 주소가 변경된 환경에서도 정상적으로 복원되도록 개선

   - 암호화 키, 이미지 로그 등 사용자 지정 경로의 데이터도 백업 항목에 포함되도록 개선

   - 백업 및 복원 과정의 안정성 개선

1. 관리자별로 카메라 그룹 단위의 접근 권한을 할당하여 지정된 그룹의 카메라만 접근할 수 있도록 개선

1. 시스템 보안 취약점 개선

   - 웹 서버 보안 헤더(HTTP Security Headers) 적용

   - 데이터베이스 접속 비밀번호의 암호화 저장 방식 강화

   - XSS 보안 취약점 개선

   - SQL Injection 보안 취약점 개선

   - 사용자 목록 페이지의 접근 권한 관련 보안 취약점 개선

   - 장치 보안 통신 암호화 키 관련 보안 취약점 개선

   - PDF 출력 관련 보안 취약점 개선

1. 근태 관리 보안 취약점 개선

   - Minimist 라이브러리 보안 취약점에 따른 버전 업데이트

   - 스크립트 파일 내 보안 취약점 개선

   - 데이터베이스 비밀번호 암호화 방식 강화

   - AngularJS, jQuery, Moment.js 라이브러리 버전 업그레이드

1. Open JDK 버전 업데이트

1. VMS 연동 시스템으로 **Nx Witness v6** 버전 지원

1. BioStar X로 업그레이드 시, 불필요한 Redis가 설치되어 있을 경우 삭제를 권고하는 팝업 메시지 추가
