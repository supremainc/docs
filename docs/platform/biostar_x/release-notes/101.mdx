# 버전 1.0.1 (빌드 번호 1.0.1.354)

릴리스<Separator/>2025-12-29

## 신규 기능 및 개선

---

1. 상단 메뉴바에서 접속 중인 계정의 기본 설정 및 비밀번호를 변경 할 수 있는 메뉴 추가

1. 일부 메뉴에서 인쇄 또는 내보내기 작업 시 사유를 입력하고, 입력한 사유를 **감사 추적**에서 확인할 수 있도록 개선

    - **사용자**: **인쇄** / **CSV 내보내기** / **데이터 파일로 내보내기**

    - **데이터**: **인쇄** / **CSV 내보내기**

1. 로그인 실패 시 감사 추적에서 이력을 확인할 수 있도록 개선

1. BioStar X **근태 관리(T&A) 라이선스**를 모든 기본 라이선스(디바이스 매니저 이상)에서 활성화할 수 있도록 변경

1. BioStar X **운용 규모(Capacity) 업그레이드**를 **에센셜** 라이선스부터 지원하도록 개선

1. **얼굴** 인증수단 표기 변경

    - **얼굴** → **IR 페이스**


## 버그 수정

---

<BugLists>

1. 시스템 안정성 향상을 위한 버그 수정 및 서비스 최적화 작업 진행 <Badge>1.0.0</Badge>

1. **BioStar X Core Web Service**의 기동 시간을 개선하고, 서비스가 중지되었거나 시작 중인 상태에서 로그인 시도 시 표시되는 팝업 메시지를 더 명확하게 개선 <Badge>1.0.0</Badge>

1. 서로 다른 장치에서 같은 사용자의 인증 이벤트가 동일한 시간대에 발생한 경우, **인원 점검**을 실행하면 서버가 종료되던 문제 <Badge>1.0.0</Badge>

</BugLists>
