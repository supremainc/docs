## 신규 기능 및 개선

---

<Jira num='BX-4306' /> [Header] 내 계정 설정 1.0.0.310, 1.0.0.319

1. 상단 메뉴바에서 접속 중인 계정의 기본 설정 및 비밀번호를 변경 할 수 있는 메뉴 추가

1. Added the menu in the top navigation bar to change the signed-in account’s Preference settings and password.

<Jira num='BX-4698' /> [User][User List][More] Print - 사유 기록 및 감사 추적 사유 기록 확인 1.0.0.313
<Jira num='BX-4719' /> [Setting] Audit Trail에서 사용자 정보를 파일로 내보내기 시 사유 기록 기능 추가 1.0.0.316, 1.0.0.323
<Jira num='BX-4742' /> [Data] Export / Print Audit Trail 동작 확인 1.0.0.316, 1.0.0.323
<Jira num='BX-4696' /> [User][User List][More][Export] CSV - 사유 기록 및 감사 추적 사유 기록 확인 1.0.0.313
<Jira num='BX-4697' /> [User][User List][More][Export] Data File - 사유 기록 및 감사 추적 사유 기록 확인 1.0.0.313

1. 일부 메뉴에서 인쇄 또는 내보내기 작업 시 사유를 입력하고, 입력한 사유를 **감사 추적**에서 확인할 수 있도록 개선

    - **사용자**: **인쇄** / **CSV 내보내기** / **데이터 파일로 내보내기**

    - **데이터**: **인쇄** / **CSV 내보내기**

1. Improved to allow entering a reason when printing or exporting in certain menus, and to view the entered reason in the **Audit Trail**.

    - **User**: **Print** / **CSV Export** / **Data File Export**

    - **Data**: **Print** / **CSV Export**

<Jira num='BX-4744' /> Login 실패 시 이력 확인

1. 로그인 실패 시 **감사 추적**에서 이력을 확인할 수 있도록 개선

1. Improved to allow viewing the history in the **Audit Trail** upon login failure.

<Jira num='BX-4846' /> [Setting] BioStar X TA 라이선스를 모든 Base 라이선스에서 활성화할 수 있도록 변경

1. BioStar X **근태 관리(T&A) 라이선스**를 모든 기본 라이선스(디바이스 매니저 이상)에서 활성화할 수 있도록 변경

1. Improved to allow the BioStar X **Attendance Management (T&A) license** to be activated on all base licenses (Device Manager or higher).

<Jira num='BX-4692' /> [Setting] Essential Base License 부터 Capacity 라이선스 지원

1. BioStar X **운용 규모(Capacity) 업그레이드**를 **에센셜** 라이선스부터 지원하도록 개선

1. Improved to support the BioStar X **Capacity upgrade** starting from the **Essential** license.

<Jira num='BX-4760' /> Face -> IR Face 메세지 변경

1. **얼굴** 인증수단 표기 변경

    - **얼굴** → **IR 페이스**

1. Rename the **Face** credential

    - **Face** → **IR Face**
