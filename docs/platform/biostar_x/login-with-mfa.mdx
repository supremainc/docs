**BioStar X**에 로그인 시 보안 강화를 위해 **멀티팩터 인증** 기능을 사용할 수 있습니다. <Cmd>Multi-Factor Auth</Cmd> 기능을 활성화하면 사용자 아이디와 비밀번호 조합에 지문 스캐너를 통한 지문 인증 단계를 추가합니다.

:::info

* **멀티팩터 인증**(Multi-factor authentication, MFA)이란, 2회 이상의 인증 절차를 통해 사용자를 인증하는 방식입니다.

* <Cmd>Multi-Factor Auth</Cmd> 기능을 사용하려면 **BioStar X** 클라이언트 PC에 다중 인증 로그인을 지원하는 지문 스캐너를 연결해야 합니다. 지원되는 지문 스캐너 장치는 아래 목록을 참고하세요.

  * BioMini

  * BioMini Plus 2

* **BioStar X** 클라우드에서는 지문 인증 로그인을 지원하지 않습니다. <Cmd>Multi-Factor Auth</Cmd> 항목을 활성화한 사용자는 로그인할 수 없습니다.

* 액티브 디렉토리 서버 계정으로 <Cmd>BioStar X 로그인에 사용</Cmd> 옵션을 사용한다면 <Cmd>Multi-Factor Auth</Cmd> 기능을 사용할 수 없습니다.

* <Cmd>Multi-Factor Auth</Cmd>을 설정한 사용자는 BioStar X AC 이외의 BioStar X 서비스에 등록할 수 없습니다.

:::

:::caution

* 전체 관리자(ID 1번) 계정에 멀티팩터 인증 기능을 설정한 후 지문 인증을 사용할 수 없게 되면, 해당 계정에 영구적으로 로그인할 수 없게 됩니다. 주의하세요.

* 지문 인증을 사용해 로그인할 수 없다면 슈프리마 기술지원팀(https://support.supremainc.com)으로 문의하세요.

:::


## 멀티팩터 인증 설정

<Cmd>Multi-Factor Auth</Cmd>을 설정하려면 다음 단계를 따르세요.

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 사용자 목록에서 <Cmd>Multi-Factor Auth</Cmd>를 설정하려는 사용자를 더블 클릭하세요.

    또는 사용자를 선택하고 오른쪽 화면에 표시되는 프로필에서 <Cmd>See More</Cmd> 버튼을 클릭하세요.

1. 사용자 프로필 화면이 나타나면 <Cmd>Permission</Cmd> 섹션에서 <Cmd>Multi-Factor Auth</Cmd> 항목을 클릭해 활성화하세요.

1. <Cmd>Save</Cmd> 버튼을 클릭해 설정 사항을 저장하세요.


## 멀티팩터 인증 일괄 설정

일괄 편집 기능을 사용해 여러 명의 사용자에게 <Cmd>Multi-Factor Auth</Cmd> 기능을 적용할 수 있습니다.

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 사용자 목록에서 <Cmd>Multi-Factor Auth</Cmd>을 설정하려는 사용자의 가장 왼쪽에 체크 박스를 클릭하세요.

1. 화면 오른쪽 상단의 <Cmd>Batch Edit</Cmd> 버튼을 클릭하세요.

1. 일괄 편집 창이 나타나면 <Cmd>Permission</Cmd> 섹션에서 <Cmd>Multi-Factor Auth</Cmd> 항목을 클릭해 활성화하세요.

1. <Cmd>Save</Cmd> 버튼을 클릭해 설정 사항을 저장하세요.

:::note

* 선택한 사용자 중 <Cmd>Multi-Factor Auth</Cmd> 기능을 활성화하는 데 필요한 조건을 만족하지 않는 사용자는 팝업 메시지에 <Cmd>적용할 수 없는 사용자 목록</Cmd>을 통해 확인할 수 있습니다. 설정에 필요한 조건을 확인하고 다시 시도하세요.

* 이 기능을 사용하려면 해당 사용자에게 지문 정보를 등록하고, 아이디 및 비밀번호, 권한 등급을 모두 설정해야 합니다.

:::

## 멀티팩터 인증으로 로그인

1. 웹 브라우저에서 **BioStar X** 로그인 화면으로 접속하세요.

1. 사용자 아이디와 비밀번호를 입력해 로그인하세요.

1. 지문 입력 화면이 나타나면 지문 스캐너를 통해 지문을 스캔하세요.

로그인을 완료합니다.

:::note

* 지문 스캔 제한 시간은 18초로 고정되어 있으며, 변경할 수 없습니다.

* 최대 3회까지 연속으로 지문을 스캔할 수 있으며, 3회 동안 지문이 정확히 스캔되지 않으면 인증 실패 처리됩니다.

* 인증에 실패했다면 <Cmd>재시도</Cmd> 버튼을 클릭해 지문 인증을 재시도하세요. 최대 2회까지 재시도할 수 있습니다. 재시도 횟수 동안 인증에 실패한다면 로그인 초기 화면으로 전환됩니다.

:::
