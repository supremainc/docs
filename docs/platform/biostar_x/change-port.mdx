**BioStar X**의 기본 포트(443)를 사용할 수 없을 때 포트를 변경하는 방법을 안내합니다.

1. <Start/> → **BioStar X** → **BioStar X Service Manager**를 클릭하세요.

1. 웹브라우저를 통해 **BioStar X Service Manager** 창이 나타나면 <Cmd>SERVICE</Cmd> 메뉴를 클릭하세요.

1. 모든 서비스의 <Cmd>Stop</Cmd> 버튼을 클릭하세요

    <Image src='/img/common/biostarx-tutoria-service-manager-services-stop.png' alone />

1. 화면 왼쪽 사이드바에서 <Cmd>SERVICE SETTINGS</Cmd> 메뉴를 클릭하세요.

1. <Cmd>BioStar X Core Web Service</Cmd> 섹션의 <Cmd>HTTPS Port</Cmd> 입력 필드에서 변경할 포트 번호를 입력하세요.

    <Image src='/img/common/biostarx-tutoria-service-manager-services-change-port.png' alone />

1. 화면 오른쪽 상단의 <Cmd>Apply</Cmd> 버튼을 클릭하세요.

1. 확인 메시지가 나타나면 <Cmd>OK</Cmd> 버튼을 클릭하세요.

1. <Cmd>SERVICE</Cmd> 메뉴로 이동하여 모든 서비스의 **Start** 버튼을 클릭하세요.

1. 웹 브라우저를 통해 **BioStar X**로 접속하세요.

:::info

* 포트 번호를 450으로 변경했다면 주소 표시줄에 `https://{ip_address}:450`을 입력하세요.

* **BioStar X**에 접속해 포트를 변경하는 방법에 대한 자세한 내용은 [다음 문서]를 참고하세요.

:::
