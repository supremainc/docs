사용자 템플릿은 조직 및 부서별 기본 정보를 표준화하여, 신규 사용자 등록 프로세스를 간소화하고 관리 효율을 높일 수 있습니다. 템플릿을 사용해 모든 신규 사용자에 대해 일관된 정보와 권한을 빠르고 정확하게 적용할 수 있습니다.

## 사용자 템플릿 생성 {#create-a-user-template}

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd>Template</Cmd>를 클릭하세요.

1. 화면 오른쪽 위에 <Cmd>New User Template</Cmd> 버튼을 클릭하세요.

1. 템플릿 생성 창이 나타나면 <Cmd>Information</Cmd>에서 템플릿 이름, 이메일, 부서, 직책 등을 입력하세요.

1. <Cmd>Permission</Cmd> 섹션에서 출입 권한을 선택하세요.

1. 입력을 완료하고 <Cmd>Save</Cmd> 버튼을 클릭하면 템플릿이 저장됩니다.

템플릿 목록에서 추가한 새 템플릿을 확인하세요.

:::note

* 부서는 조직 구조를 반영하여 사용자를 구분하는 데 사용됩니다. 자세한 내용은 [다음 문서](add-user#information)를 참조하세요.

* 그룹은 여러 사용자를 하나로 묶어 출입 권한을 한 번에 적용하기 위해 사용됩니다. 자세한 내용은 [다음 문서](user-group-manage)를 참조하세요.

* <Cmd>Access Group</Cmd>은 특정 출입 구역에 대한 사용자 권한을 정의하며 사용자에게 일괄적으로 출입 권한을 부여하는 데 사용됩니다. 자세한 내용은 [다음 문서](#)를 참조하세요.

:::


## 템플릿으로 사용자 추가 {#add-user-as-a-template}

추가한 템플릿으로 신규 사용자를 추가하려면 다음 단계를 따르세요.

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 화면 오른쪽 상단의 <Cmd>New User</Cmd> 버튼오른쪽에 <IcDown/> 버튼을 클릭하세요.

1. 원하는 템플릿을 선택하세요.

1. 템플릿에 설정된 정보를 확인하고, 사용자 ID, 이름, 연락처 등의 나머지 항목을 입력하세요.

1. <Cmd>Save</Cmd> 버튼을 클릭하세요.

사용자 목록에서 템플릿으로 추가한 신규 사용자를 확인하세요.

:::note

사용자 추가에 대한 자세한 내용은 [다음 문서](add-user)를 참조하세요.

:::


## 템플릿 수정 {#template-modification}

이미 만들어진 사용자 템플릿을 수정할 수 있습니다.

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd>Template</Cmd>를 클릭하세요.

1. 템플릿 목록에서 수정하려는 항목을 더블 클릭하세요.

1. 템플릿 수정 창이 나타나면 원하는 항목을 수정하세요.

1. 수정을 완료하고 <Cmd>Save</Cmd> 버튼을 클릭하세요.

수정한 내용이 템플릿에 반영됩니다.


## 템플릿 삭제 {#delete-template}

불필요한 사용자 템플릿을 삭제할 수 있습니다.

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd>Template</Cmd>를 클릭하세요.

1. 템플릿 목록에서 삭제하려는 항목의 가장 왼쪽의 체크박스를 클릭하세요.

1. 화면 오른쪽 상단의 <Cmd>Delete</Cmd> 버튼을 클릭하세요.

1. 확인 메시지가 나타나면 <Cmd>확인</Cmd> 버튼을 클릭하세요.

템플릿 목록에서 선택한 템플릿이 삭제됩니다.

:::note

목록에서 모든 템플릿을 선택하려면 화면 왼쪽 상단의 <Cmd>Select All</Cmd> 버튼을 클릭하세요.

:::
