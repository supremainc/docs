title: 사용자 등록하기
description: 새로운 사용자를 등록하는 방법을 안내합니다.
keywords: [ 사용자, 사용자 그룹 ]
---

새로운 사용자를 등록하는 방법을 안내합니다. 사용자의 사진 및 이름, 이메일, 전화번호 등과 같은 정보를 입력하고 사용자의 출입 권한 및 생체 정보를 설정할 수 있습니다.

사용자 정보 입력 및 출입 권한 설정, 크리덴셜 등록순으로 진행합니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.user' />를 클릭하세요.

1. <Cmd sid='shell.header.user' /> 페이지의 오른쪽 상단의 <Cmd sid='team.user.newUser' /> 버튼을 클릭하세요.

1. <Cmd sid='user.addNewUser' product='2' /> 화면에서 사용자 정보를 입력하세요.

1. <Cmd sid='team.user.information' /> 섹션에서 사용자의 기본 정보를 입력하세요.

1. <Cmd sid='team.user.credential.title' /> 섹션에서 사용자의 생체 인식 정보를 등록하세요.

1. <Cmd sid='team.user.permission.title' /> 섹션에서 사용자의 출입 권한을 설정하세요.

1. <Cmd sid='common.advanced' product='2' /> 섹션에서 사용자가 출입 시 장치에 표시할 메시지를 입력할 수 있습니다.

1. 모든 정보를 입력하고 화면 오른쪽 상단의 <Cmd sid='button.apply' product='2' /> 버튼을 클릭하세요.

사용자 목록에서 등록한 사용자를 확인할 수 있습니다. 섹션별 입력 및 설정에 대한 자세한 내용은 아래를 참고하세요.

<NextStep notitle>
<NextItem to='add-user-basic-info'>

<Cmd sid='team.user.information' /> 섹션에서 사용자의 기본 정보를 입력하세요. 각 항목에 대한 자세한 내용을 확인할 수 있습니다.

</NextItem>
<NextItem to='enroll-credential'>

<Cmd sid='team.user.credential.title' /> 섹션에서 사용자 크리덴셜을 등록하고 출입 인증 방식을 설정하세요.

</NextItem>
<NextItem to='set-permission'>

<Cmd sid='team.user.permission.title' /> 섹션에서 사용자의 출입 권한 설정 방법을 확인하세요.

</NextItem>
<NextItem to='set-user-advanced'>

<Cmd sid='common.advanced' product='2' /> 섹션에서 사용자 출입 시 장치에 표시할 메시지를 설정하거나 액티브 디렉토리 연동 시 해당 사용자를 동기화에서 제외할 수 있습니다.

</NextItem>
</NextStep>

:::info

* 출입 권한 및 생체 인식 정보는 선택 사항입니다. 기본 정보를 입력하고 저장한 후에 수정하거나 추가할 수 있습니다.

* 사용자 정보를 카드 템플릿을 통해 카드로 인쇄하려면 화면 오른쪽 상단의 <Cmd sid="button.printCard" product='2' /> 버튼을 클릭하세요. 자세한 내용은 [다음 문서](settings-card-printer)를 참고하세요.

* 사용자 등록을 취소하려면 화면 오른쪽 상단의 <Cmd sid="button.cancel" product='2' /> 버튼을 클릭하세요.

:::
