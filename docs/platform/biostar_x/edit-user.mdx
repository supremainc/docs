사용자의 정보를 수정하고 권한을 변경하는 방법을 안내합니다. 사용자의 기본 정보를 수정하고, 출입 권한 및 크리덴셜을 변경할 수 있습니다.

:::info

최초 만들어진 아이디가 `1`인 사용자는 수정할 수 없습니다.

:::

## 사용자 정보 수정 {#user-information-modification}

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.user' />를 클릭하세요. 

1. 사용자 목록에서 수정할 사용자를 선택하세요.

1. 화면 오른쪽에 사용자의 프로필 정보가 표시되면 <Cmd>See More</Cmd> 버튼을 클릭하세요.

1. 사용자 정보를 수정할 수 있는 창이 나타납니다.

1. 원하는 항목을 수정하고 화면 오른쪽 상단의 <Cmd sid='common.save' /> 버튼을 클릭하세요.

:::info

* <Cmd sid='team.user.information' /> 섹션의 각 항목에 대한 자세한 내용은 [다음 문서](add-user-basic-info)를 참고하세요.

* <Cmd sid='team.user.credential.title' /> 섹션의 각 항목에 대한 자세한 내용은 [다음 문서](enroll-credential)를 참고하세요.

* <Cmd sid='team.user.permission.title' /> 섹션의 각 항목에 대한 자세한 내용은 [다음 문서](set-permission)를 참고하세요.

* <Cmd sid='common.advanced' product='2' /> 섹션에서 사용자와 관련한 개인 메시지를 입력할 수 있습니다.
:::

## 사용자 정보 일괄 수정 {#user-information-collective-modification}

한 번에 여러 사용자의 정보나 출입 권한을 수정할 수 있습니다. 이 기능은 다수의 사용자에게 동일한 출입 권한이나 그룹을 설정할 때 유용합니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.user' />를 클릭하세요. 

1. 사용자 목록에서 수정할 사용자의 가장 왼쪽에 체크박스를 클릭하세요. 1명 이상의 사용자를 선택할 수 있습니다.

1. 화면 오른쪽 상단의 <Cmd sid='common.batchEdit' /> 버튼을 클릭하세요.

1. <Cmd sid='common.batchEdit' /> 화면이 나타나면 원하는 정보를 수정하세요.

    <Image src='/img/biostarx-edit-user-batch-edit.png' />

1. 수정한 정보를 저장하려면 화면 오른쪽 상단의 <Cmd sid='common.save' /> 버튼을 클릭하세요.

:::info

* 사용자의 ID 및 이름, 이메일, 전화번호, 크리덴셜과 같은 개인정보는 일괄 수정할 수 없습니다.

* 수정한 정보를 취소하려면 화면 오른쪽 상단의 <Cmd sid='common.cancel' /> 버튼을 클릭하세요.

:::
