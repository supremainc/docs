description: 엘리베이터의 상태를 확인하고 상황에 따라 대처하는 방법을 안내합니다.

각 엘리베이터의 상태를 확인하세요. 안내된 아이콘과 상세 설명을 통해 관리자들이 각 엘리베이터의 작동 상태를 한눈에 파악하고, 문제 발생 시 신속하게 대응할 수 있도록 돕습니다.

<Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.arena' />을 클릭하거나 화면 왼쪽 상단의 바로 가기 목록에서 <Cmd sid='shell.header.arena' />을 선택하세요. 화면 왼쪽 사이드바의 <Cmd sid='arena.tree.door.title' /> 탭에서 <Cmd sid='arena.tree.door.allElevatorGroups' />으로 이동하세요.

:::info

엘리베이터 설정 방법에 대한 자세한 내용은 [다음 문서](#)를 참고하세요.

:::

## 엘리베이터 알람 확인 {#check-the-elevator-alarm}

각 엘리베이터에서 이벤트가 발생하면 <IcErr width='20px' height='20px' /> 아이콘이 표시됩니다. 알람 아이콘을 클릭하면 발생한 이벤트를 확인할 수 있습니다.

## 엘리베이터 상태 확인 {#check-the-elevator-status}

<div className='col2--20-80 overflow-x'>

| 상태 아이콘 | 설명 |
| :--: | -- |
| <StatusOK/> | 정상 상태입니다. |
| <StatusFail /> | 이상 상태입니다. 아이콘을 클릭하면 툴팁을 통해서 상세 정보를 확인할 수 있습니다.  |

</div>
