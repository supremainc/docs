title: 장치에서 사용자 정보 삭제하기
description: 장치에 등록된 사용자 정보를 삭제하는 방법을 안내합니다.
keywords: [ 사용자, 장치 ]
---

출입 통제를 위해 **BioStar X** 서버에 등록한 사용자 정보를 장치에서 삭제하는 기능을 제공합니다. 각각의 장치에서 사용자 정보를 삭제할 수 있습니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.user' />를 클릭하세요.

1. 사용자 목록에서 장치에서 삭제할 사용자의 가장 왼쪽에 체크박스를 클릭하세요.

1. 화면 오른쪽 상단의 <IcMoreW/> 버튼을 선택하고 <Cmd sid='user.header.removeFromDevice' product='2' /> 버튼을 클릭하세요.

1. <Cmd sid='user.header.removeFromDevice' product='2' /> 창이 나타나면 선택한 사용자 정보를 삭제할 장치를 선택하세요.

1. <Cmd sid='common.delete' /> 버튼을 클릭하세요.

:::info

* 검색 아이콘 버튼을 클릭해 장치를 검색할 수도 있습니다.

* <Cmd sid='setting.server.automaticUserSync' product='2' /> 옵션을 <Cmd sid='device.Input.unsupervised' product='2' />으로 설정해야 <Cmd sid='user.header.removeFromDevice' product='2' /> 버튼이 활성화됩니다.

* 사용자 정보는 장치에서만 삭제되고, **BioStar X** 서버에 유지됩니다.

:::
