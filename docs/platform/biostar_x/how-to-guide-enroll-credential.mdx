사용자의 크리덴셜을 등록하는 방법을 안내합니다. 사용자의 지문 및 얼굴, 카드 정보, 비밀번호 등을 크리덴셜로 등록할 수 있습니다. 크리덴셜은 사용자를 새로 추가하거나 수정할 때 등록할 수 있습니다.

<Cmd>Launcher</Cmd> > <Cmd>User</Cmd> 페이지로 이동해, 아래 두 가지 방식으로 크리덴션을 등록하거나 추가, 변경할 수 있습니다.

* **새로운 사용자**: 화면 오른쪽 상단의 <Cmd>New User</Cmd> 버튼을 클릭하세요. New User 창의 Credential 섹션에서 크리덴셜을 등록할 수 있습니다.

* **기존 사용자**: 사용자 목록에서 사용자를 선택하면 화면 오른쪽에 표시되는 프로필 화면에서 See More 버튼을 클릭하세요. 사용자 정보를 수정할 수 있는 창이 나타나면 Credential 섹션에서 크리덴셜을 등록할 수 있습니다.

:::info

**크리덴셜이란?**

사용자의 신원을 확인하기 위한 데이터입니다. 일반적으로 디지털 서명, 스마트카드, 바이오 정보, 사용자 이름과 비밀번호 등이 있습니다.

:::

:::note

크리덴셜을 등록하고 <Cmd>1:1 Security Level</Cmd> 항목에서 적절한 보안 등급을 설정하세요. 보안 등급이 높을 수록 인증률이 낮거나 본인 거부율(FRR)이 높을 수 있습니다.

:::

## 지문 등록 {#enroll-fingerprint}

지문 인증을 지원하는 장치를 통해 사용자의 지문 정보를 등록할 수 있습니다. 지문은 지문 스캐너가 설치된 장치에서 스캔할 수 있습니다.

:::note

**지문을 등록하기 전에...**

* 사용자의 지문이 깨끗하고 물기가 없는지 확인하세요.

* 상처가 있거나 지문이 희미한 손가락은 등록하지 마세요.

:::

<Cmd>Credential</Cmd> 섹션의 Fingerprint 항목에서 <IcAdd/> 버튼을 클릭하세요. 지문 등록 창이 나타나면 각 항목을 설정하고 <Cmd>Enroll</Cmd> 버튼을 클릭하세요. 

* <Cmd>Enrollment Device</Cmd>: 지문을 등록할 장치를 선택하세요.

* <Cmd>FingerPrint Quality</Cmd>: 지문 등록 품질을 조절할 수 있습니다. 품질 수준에 미치지 않으면 지문 정보를 등록할 수 없습니다.

* <Cmd>View FingerPrint Image</Cmd>: 토글 버튼을 클릭하면, 지문을 스캔했을 때 원본 이미지를 확인할 수 있습니다.

* 지문은 최대 10개까지 등록할 수 있습니다. 각 번호에 한 개의 지문을 등록할 수 있습니다. 원하는 번호를 선택하고 <Cmd>Scan Fingerprint</Cmd> 버튼을 클릭하세요.

  ![](/img/common/biostarx-enroll-fingerprint-count.png)

  :::note

  번호 버튼을 선택해야 <Cmd>Scan Fingerprint</Cmd> 버튼이 활성화됩니다. 지문을 장치에 스캔하면, 지문 정보가 등록됩니다.

  :::

* <Cmd>Duress</Cmd>: 협박 지문으로 등록하려면 이 옵션을 선택하고 지문을 스캔하세요. 누군가에게 협박을 당하거나 강제로 출입문을 열어야 할 때, 해당 지문을 사용하면 알림을 보낼 수 있습니다.

* <Cmd>Clear</Cmd>: 등록한 지문을 삭제할 수 있습니다. 지문을 삭제하려면 해당 지문 번호를 선택하고 버튼을 클릭하세요.

:::info

* 일상에서 출입을 위해 사용하는 지문을 협박 지문으로 사용하지 마세요.

* <Cmd>View FingerPrint Image</Cmd> 옵션을 활성화할 때, 지문 이미지를 확인할 수 있지만 **BioStar**에 저장하지 않습니다.

* 지문 인증률이 낮다면 지문 정보를 삭제하고 새로운 지문 정보를 등록하세요.

* 좋은 품질의 지문 정보를 얻으려면 지문 인식 센서 표면을 모두 덮을 수 있도록 손가락을 대세요. 검지나 중지의 지문을 등록하세요.

  ![](/img/common/biostarx-enroll-fingerprint-caution.png)

:::


## 얼굴 등록 {#enroll-face}

얼굴 인증을 지원하는 장치를 통해 사용자의 얼굴 정보를 등록할 수 있습니다.

:::note

**얼굴을 등록하기 전에...**

* 얼굴을 등록할 때 장치와 얼굴의 거리를 40 ~ 80cm로 유지하세요.

* 표정이 변하지 않도록 주의하세요. (웃는 얼굴, 찡그린 얼굴, 윙크 등)

* 장치 화면의 지시에 따르지 않으면 얼굴 등록이 오래 걸리거나 실패할 수 있습니다.

* 눈이나 눈썹이 가려지지 않도록 주의하세요.

* 모자, 마스크, 선글라스, 안대를 착용하지 마세요.

* 두 명의 얼굴이 화면에 동시에 나타나지 않도록 주의하세요. 한 번에 한 사람씩 등록하세요.

* 안경을 착용한 사용자는 안경 착용 전/후 얼굴을 모두 등록할 것을 권장합니다.

:::

<Cmd>Credential</Cmd> 섹션의 <Cmd>Face</Cmd> 항목에서 <IcAdd/> 버튼을 클릭하세요. 얼굴 등록 창이 나타나면 각 항목을 설정하고 <Cmd>Enroll</Cmd> 버튼을 클릭하세요.

* <Cmd>Enrollment Device</Cmd>: 얼굴을 등록할 장치를 선택하세요.

* <Cmd>Face Pose Variation</Cmd>: 얼굴을 등록할 때 얼굴의 위치, 각도, 거리에 대한 민감도를 설정할 수 있습니다. 정밀한 얼굴 템플릿을 얻으려면 높은 값으로 설정하세요.

* <Cmd>Add</Cmd>: 얼굴을 추가하려면 버튼을 클릭하세요. 최대 5개의 얼굴을 추가할 수 있습니다.

* <Cmd>Scan Face</Cmd>: 얼굴을 스캔하려면 버튼을 클릭하세요. 버튼을 클릭하고 장치 화면의 지시에 따라 얼굴을 스캔하세요.

* <Cmd>Use as Profile Image</Cmd>: 스캔한 얼굴을 프로필 이미지로 사용하려면 옵션을 선택하세요.

* <Cmd>Clear</Cmd>: 등록한 얼굴을 삭제할 수 있습니다. 삭제하려는 순번 버튼을 선택하고 버튼을 클릭하세요.

:::info

얼굴 인증률이 낮다면 얼굴 정보를 삭제하고 새로운 얼굴 정보를 등록하세요

:::

## 비주얼 페이스 등록

비주얼 페이스(Visual Face)는 사용자의 얼굴을 비주얼 카메라로 촬영한 인증 수단입니다. 적외선 카메라로 촬영한 얼굴 정보와 구분되며, 비주얼 페이스를 지원하는 장치에서만 사용할 수 있습니다. 비주얼 페이스는 사용자의 모바일 기기에서 비대면으로 등록할 수도 있습니다.

:::info

**비주얼 페이스를 등록하기 전에...**

* 장치로 등록할 때 장치와 얼굴의 거리는 60 ~ 100cm로 유지하세요.

* 장치로 등록할 때 움직이지 말고 정면으로 서서 얼굴을 등록하세요.

* 장치 또는 모바일 기기로 등록할 때 화면의 지시에 따르지 않으면 얼굴 등록이 오래 걸리거나 실패할 수 있습니다.

* 표정이 변하지 않도록 주의하세요.

* 마스크, 모자, 안대를 착용하지 마세요.

* 고개를 들거나 숙이지 마세요.

* 진한 화장을 하지 마세요.

* 눈을 감지 마세요.

* 양쪽 어깨가 모두 나타나도록 하세요.

* 두 명의 얼굴이 동시에 나타나지 않도록 주의하세요. 한 번에 한 사람씩 등록하세요.

:::

<Cmd>Credential</Cmd> 섹션의 <Cmd>Visual Face</Cmd> 항목에서 <IcAdd/> 버튼을 클릭하세요. 비주얼 페이스 등록 창이 나타나면 각 항목을 설정하고 <Cmd>Enroll</Cmd> 버튼을 클릭하세요.

* <Cmd>Enrollment Device</Cmd>: 비주얼 페이스를 등록할 장치를 선택하세요.

* <Cmd>Add</Cmd>: 비주얼 페이스를 추가하려면 버튼을 클릭하세요. 최대 2개의 비주얼 페이스를 추가할 수 있습니다.

* <Cmd>Scan</Cmd>: 비주얼 페이스를 스캔하려면 버튼을 클릭하세요. 버튼을 클릭하고 장치 화면의 지시에 따라 비주얼 페이스를 스캔하세요.

* <Cmd>Upload Image</Cmd>: 비주얼 페이스로 사용할 이미지를 업로드할 수 있습니다.

  :::note

  * 지원하는 이미지 파일의 크기는 최대 10Mb입니다.

  * 지원하는 이미지 파일은 JPG, JPEG, PNG 형식입니다.

  :::

* <Cmd>Use as Profile Image</Cmd>: 스캔한 비주얼 페이스를 프로필 이미지로 사용하려면 옵션을 선택하세요.

* <Cmd>Delete</Cmd>: 등록한 비주얼 페이스를 삭제할 수 있습니다. 삭제하려는 순번 버튼을 선택하고 버튼을 클릭하세요.

:::note

* 비주얼 페이스를 사용할 수 있는 장치는 아래와 같습니다.

  > FaceStation F2, BioStation 3, BioEntry W3

* 비주얼 페이스를 다른 방식으로 불러오거나 모바일 기기 또는 웹캠으로 등록할 수 있습니다. 자세한 내용은 다음 문서를 참고하세요.

:::


## 출입 카드 등록

사용자에게 출입 카드를 할당할 수 있습니다. 장치가 지원하는 카드 종류는 장치의 매뉴얼을 참고하세요.

:::info

**USB 에이전트를 이용한 카드 등록**

<div className='table-fixed'>

| 카드 종류 | CSN | Wiegand | 스마트 카드 |
| --- | :--: | :--: | :--: |
| EM | ❌ | ❌ | ❌ |
| MIFARE | ✅ | ❌ | ✅ |
| DESFire | ✅ | ❌ | ✅ |
| FeliCa | ✅ | ❌ | ❌ |
| HID Prox | ❌ | ❌ | ❌ |
| HID iCLASS | ❌ | ❌ | ❌ |

</div>

:::

### CSN 카드

1. <Cmd>Credential</Cmd> 섹션의 <Cmd>Card</Cmd> 항목에서 <IcAdd/> 버튼을 클릭하세요. 

1. 카드 등록 창이 나타나면 <Cmd>Card Type</Cmd> 목록에서 <Cmd>CSN</Cmd>을 선택하세요.

1. <Cmd>Registration Option</Cmd> 항목에서 원하는 카드 등록 방식을 선택하세요.

    * <Cmd>Register by Card Reader</Cmd>: <Cmd>Device</Cmd> 목록에서 카드를 스캔할 장치를 선택하고 <Cmd>Read Card</Cmd> 버튼을 클릭하세요.

    * <Cmd>Assign Card</Cmd>: 목록에서 사용자에게 할당하려는 카드를 선택하세요. 입력 필드를 통해 검색할 수도 있습니다.

    * <Cmd>Enter Manually</Cmd>: 카드 번호를 직접 입력하거나 <Cmd>Use User ID</Cmd> 버튼을 클릭하세요.

1. <Cmd>Enroll</Cmd> 버튼을 클릭하세요.

### Wiegand 카드

1. <Cmd>Credential</Cmd> 섹션의 <Cmd>Card</Cmd> 항목에서 <IcAdd/> 버튼을 클릭하세요. 

1. 카드 등록 창이 나타나면 <Cmd>Card Type</Cmd> 목록에서 <Cmd>Wiegand</Cmd>를 선택하세요.

1. <Cmd>Card Data Format</Cmd> 항목에서 카드 데이터 형식을 선택하세요. 

1. <Cmd>Registration Option</Cmd> 항목에서 원하는 카드 등록 방식을 선택하세요.

    * <Cmd>Register by Card Reader</Cmd>: <Cmd>Device</Cmd> 목록에서 카드를 스캔할 장치를 선택하고 <Cmd>Read Card</Cmd> 버튼을 클릭하세요.

    * <Cmd>Assign Card</Cmd>: 목록에서 사용자에게 할당하려는 카드를 선택하세요. 입력 필드를 통해 검색할 수도 있습니다.

    * <Cmd>Enter Manually</Cmd>: 식별 코드(Facility Code)와 카드 ID를 직접 입력하세요.

1. <Cmd>Enroll</Cmd> 버튼을 클릭하세요.

:::note

<Cmd>Card Data Format</Cmd> 항목에 원하는 카드 데이터 형식이 없다면 [다음 문서](#)를 참조하세요.

:::
