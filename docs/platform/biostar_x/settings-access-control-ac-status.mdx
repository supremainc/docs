사용자가 출입할 수 있는 출입문 목록을 확인하는 방법을 안내합니다. 출입 권한 상태를 확인하면 사용자가 출입할 수 있는 출입문과 해당 출입문에 대한 권한을 관리할 수 있습니다. 필터 기능을 사용해 원하는 결과를 정렬해 볼 수 있으며, CSV 파일로 목록을 저장할 수 있습니다. 

## 출입 권한 상태 확인

출입 권한은 출입문 또는 층, 사용자를 기준으로 볼 수 있습니다.

<Image src='/img/biostarx-settings-access-control-status.png' />

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.setting' />을 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd>Access Control</Cmd>을 클릭하세요.

1. <Cmd sid='common.status' product='2' /> 탭을 클릭하세요.

1. <Cmd sid='common.status' product='2' /> 탭 아래의 상태 목록에서 원하는 항목을 클릭하세요.

    * <Cmd sid='access_control.status.access_control_status_by_door.list_name' product='2' />

    * <Cmd sid='access_control.status.access_control_status_by_user.list_name' product='2' />

    * <Cmd sid='ac.status.elevatorbyFloor.title' product='2' />

    * <Cmd sid='ac.status.elevatorbyUser.title' product='2' />

선택한 항목에 대한 출입 권한 상태를 확인할 수 있습니다.


## 필터 저장

원하는 결과를 정렬하려면 출입 권한 목록 테이블의 헤더에서 <IcFilter2 /> 버튼을 클릭하세요. 필터 조건을 추가할 수 있습니다.

필터를 설정한 후에는 필터를 저장할 수 있습니다. <Cmd sid='monitoring.saveFilter' product='2' /> 버튼을 클릭하세요. 저장된 필터는 <Cmd sid='common.status' product='2' /> 탭 아래의 상태 목록에서 확인할 수 있습니다.

<Image src='/img/biostarx-settings-access-control-status-filter.png' />

* 필터 이름을 변경하려면 <IcEditUL /> 아이콘을 클릭하세요.

* 필터를 삭제하려면 <IcTrash2 /> 아이콘을 클릭하세요.


## 목록 페이지 탐색

페이지를 이동하거나 한 페이지에 표시할 목록 개수를 설정할 수 있습니다. 화면 오른쪽 상단의 페이지 이동 도구를 사용하세요.

<Image src='/img/biostarx-settings-access-control-status-pagenation.png' />

* <IcFirst2 />: 첫 페이지로 이동합니다.

* <IcFw2 />: 이전 페이지로 이동합니다.

* <IcBw2 />: 다음 페이지로 이동합니다.

* <IcLast2 />: 마지막 페이지로 이동합니다.

* 원하는 페이지로 이동하려면 숫자 입력 필드에 페이지 번호를 입력하세요.

* 한 페이지에 표시할 목록 개수를 설정하려면 행 선택 상자를 클릭하세요.


## CSV 내보내기

현재 보고 있는 출입 권한 상태 목록을 CSV 파일로 내보낼 수 있습니다. 화면 오른쪽 상단의 <IcMoreW /> 버튼을 클릭하고 <Cmd sid='button.downloadToCsv' product='2' />를 선택하세요.


## 열 설정

현재 보고 있는 출입 권한 상태 목록 테이블에서 원하는 열을 숨기거나 표시할 수 있습니다. 화면 오른쪽 상단의 <IcMoreW /> 버튼을 클릭하고 <Cmd sid='columnSetting.dialog.header' product='2' />을 선택하세요.

<Image src='/img/biostarx-settings-access-control-status-column-setting.png' />

* <Cmd sid='columnSetting.columnList' product='2' />에서 표시할 열을 선택하세요. 선택한 열은 목록 테이블에 표시됩니다. 선택하지 않은 열은 목록 테이블에서 숨겨집니다.

* 설정을 초기화하려면 <Cmd sid='columnSetting.defaultColumn' product='2' /> 버튼을 클릭하세요.

모든 설정을 완료하면 <Cmd sid='button.apply' product='2' /> 버튼을 클릭하세요.
