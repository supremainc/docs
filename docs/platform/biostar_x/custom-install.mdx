이 문서에서는 <Cmd code='biostar2.install.customInstall' className='bold'/> 방식으로 **BioStar X**를 설치하는 절차를 단계별로 안내합니다. <Cmd code='biostar2.install.customInstall' className='bold'/>는 로컬 서버 또는 원격 서버에 기존에 설치된 데이터베이스(MariaDB, MS SQL Server)와 연동하여 데이터베이스 스키마를 생성합니다. 설치를 시작하기 전에 연동할 데이터베이스와 사용자 계정을 미리 준비하세요.

## 시작하기 전에 {#start-before}

* **BioStar X**와 연동하여 사용할 수 있는 데이터베이스는 아래와 같습니다.

  * MariaDB 10.1.10

  * MS SQL Server 2012 SP3

  * MS SQL Server 2014 SP2

  * MS SQL Server 2016 SP1

  * MS SQL Server 2017

  * MS SQL Server 2019

  * MS SQL Server 2022

* MS SQL Server의 보안 통신은 TLS 1.2를 지원합니다.

* MS SQL Server를 데이터베이스로 설정하고, 데이터베이스 테이블 생성에 실패했다면 *C:\Program Files\BioStar X\dbscript\mssql* 경로의 스크립트를 실행하여 테이블을 생성할 수 있습니다.

* 설치 옵션으로 <Cmd code='biostar2.install.customInstall' className='bold'/>를 선택할 때 **AC DB name**, **TA DB name**, **VE DB name**은 동일하게 설정할 수 없습니다.

* Windows 인증을 사용하려면 [Microsoft ODBC Driver 17 for SQL Server](https://learn.microsoft.com/ko-kr/sql/connect/odbc/download-odbc-driver-for-sql-server?view=sql-server-ver16#version-17)가 필요합니다.
