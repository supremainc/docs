설치 패키지를 통해 **BioStar X** 설치를 모두 완료했다면 웹 브라우저로 접속해 정상 동작하는지 확인하세요. **BioStar X**는 웹 기반 서비스를 제공하여 언제 어디서든 접속할 수 있습니다.

## 웹 브라우저에서 로그인하기 {#log-in-from-a-web-browser}

1. Windows에서 <Start/> → **BioStar X** → **BioStar X**를 클릭하세요. 

1. 웹 브라우저를 통해 **BioStar X**로 연결됩니다.

    <Image src='/img/common/biostarx-login-web-browser-ip-address.png' />

    :::note

    * 웹 브라우저의 주소 표시줄에서 **BioStar X**의 접속 주소를 확인하세요.

    * **BioStar X**를 다른 PC에 설치했다면 해당 PC의 IP 주소를 입력하세요. 예, `https://192.168.0.1`

    * 'localhost'를 **BioStar X** 접속 주소로 사용하지 마세요.

    * **BioStar X**는 통합 게이트웨이(Unified Gateway)를 통해 모든 요청을 통합 처리합니다. 따라서 웹 접속 및 API 호출은 통합 게이트웨이 포트(기본 443)를 통해 이루어집니다.

    :::

1. 로그인 화면이 나타나면 관리자 계정으로 로그인하세요.

    <Image src='/img/biostarx-login.png' />

    :::note

    * 관리자 아이디는 **admin**입니다. 

    * 만약 웹 브라우저의 주소 표시줄에 **안전하지 않음** 경고가 표시된다면 HTTPS 인증서를 설치해야 합니다. 인증서 설치에 대한 자세한 내용은 [다음 문서](settings-server-https)를 참고하세요.

    :::

로그인하면 <Cmd sid='shell.header.launcher' /> 화면을 확인할 수 있습니다.

<Image src='/img/biostarx-main-launcher.png' alone />

:::info

* Chrome 100 버전 이상의 브라우저를 사용하기를 권장합니다.

* **BioStar X**를 설치한 PC의 외부 IP 주소는 [다음 링크](https://whatismyipaddress.com/)로 접속해 확인할 수 있습니다.

* **BioStar X**는 **443** 포트를 기본으로 사용합니다. **443** 포트를 사용하는 프로그램이 있다면 종료하고, 다시 접속을 시도하세요. 해당 프로그램을 종료할 수 없다면 **BioStar X Service Manager**를 실행하여 포트 번호를 변경하세요. 자세한 내용은 [다음 문서](change-port)를 참고하세요.

:::
