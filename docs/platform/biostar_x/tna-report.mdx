시스템을 통해 수집된 사용자의 근태 이벤트로 근태 보고서를 생성할 수 있으며, 시간 기록을 편집하거나 CSV 파일이나 PDF파일로 내보내기할 수 있습니다.

미리 설정된 7 개의 보고서 필터를 이용하여 간편하게 사용할 수 있으며, 관리자가 직접 필터를 설정할 수도 있습니다.

## 시작하기 전에

### 다국어 보고서 사용하기 전에

**BioStar X**에서는 한국어와 영어를 기본 지원합니다. 다국어 보고서를 사용하려면 다음 안내에 따라 원하는 언어를 설정하세요.

1. 다음 경로로 이동하세요. *C:\Program Files\BioStar X\plugin\ta\dist\setup\report_fonts*

1. 사용하려는 언어 이름으로 폴더를 생성하세요. 언어 이름은 ISO 639-1 규격을 참고하세요. 예를 들어, 스페인어를 사용하려면 폴더 이름을 "es"로 생성하세요.

1. 생성한 폴더로 글꼴을 복사하여 붙여넣기를 하세요. 하나의 트루 타입 글꼴만 지원합니다.


### 보고서 업데이트 전에

**BioStar X**는 MariaDB를 기본 데이터베이스로 사용합니다. MS SQL Server 데이터베이스를 사용한다면 아래 사항을 먼저 확인하세요.

**BioStar X**를 MS SQL Server 데이터베이스와 사용할 때, 등록된 사용자의 수가 많은 경우 보고서를 업데이트할 때마다 PC의 메모리 사용량이 누적될 수 있습니다. MS SQL Server 데이터베이스의 최대 서버 메모리를 다시 설정하세요.

1. **Microsoft SQL Server Management Studio**를 실행하세요.

1. 개체 탐색기에서 **BioStar X**의 데이터베이스를 마우스 오른쪽 버튼을 클릭하세요.

1. 팝업 메뉴에서 **속성**을 클릭하세요.

1. **메모리**를 클릭한 후 **최대 서버 메모리** 항목의 값을 줄이세요.

:::info

**BioStar X** 설치 시 MariaDB 및 MS SQL Server 설정에 대한 자세한 내용은 [다음 문서](custom-install)를 참고하세요.

:::


## 근태 기록 확인

### 근태 보고서 생성

등록된 사용자의 근태 기록 보고서를 생성하는 방법을 안내합니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.tna' />를 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid='ta.report.title' product='2' /> 탭을 클릭하세요.

    <Image src='/img/biostarx-tna-report.png' />

1. 화면 왼쪽 사이드바에서 원하는 보고서 필터 항목을 클릭하세요.

1. <Cmd sid='common.userGroup' product='2' /> 또는 <Cmd sid='common.userGroup' product='2' /> 항목에서 <IcSearch /> 버튼을 클릭한 후 그룹 또는 사용자를 지정하세요.

1. 보고서를 생성하려면 <Cmd sid='button.updateReport' product='2' /> 버튼을 클릭하세요.

1. 화면 하단에 보고서 목록이 나타나면 원하는 항목을 클릭하여 세부 정보를 확인할 수 있습니다.

<Image src='/img/biostarx-tna-report-update-list.png' />

### 필터 조건 설정

필터 조건을 설정하여 새로운 근태 기록 보고서를 생성할 수 있습니다.

<Image src='/img/biostarx-tna-report-filter-setting.png' />

* <Cmd sid='common.name' product='2' />: 보고서 이름을 입력하세요.

* <Cmd sid='ta.report.condition.reportType' product='2' />: 원하는 보고서 종류를 선택하세요. 제공하는  보고서 종류는 아래와 같습니다.

  * <Cmd sid='report.daily' product='2' />, <Cmd sid='report.dailySummary' product='2' />, <Cmd sid='report.individual' product='2' />, <Cmd sid='report.individualSummary' product='2' />, <Cmd sid='report.leave' product='2' />, <Cmd sid='report.exception' product='2' />, <Cmd sid='report.editHistory' product='2' />, <Cmd sid='report.workweek' product='2' />

  :::info

  <Cmd sid='ta.report.condition.reportType' product='2' />를 <Cmd sid='report.individual' product='2' />로 선택하면 사용자의 출입 기록 출력 여부를 설정할 수 있습니다.

  <Image src='/img/biostarx-tna-report-individual-types.png' />

  * <Cmd sid='report.individual.optionSummary' product='2' />: 보고서에 사용자의 출근 및 퇴근 기록만 출력합니다.

  * <Cmd sid='report.individual.optionDetail' product='2' />: 보고서에 사용자의 모든 출입 기록을 출력합니다.

  :::

* <Cmd sid='grid.columnSetting' product='2' />: 보고서 테이블의 열 순서를 변경하거나 숨길 수 있습니다.

  <Image src='/img/biostarx-tna-report-filter-setting-column.png' />

  :::info

  열 순서를 변경한 후 기본 설정으로 복원하려면 <Cmd sid='columnSetting.defaultColumn' product='2' /> 버튼을 클릭하세요.

  :::

* <Cmd sid='report.filter' product='2' />: <Cmd sid='ta.report.condition.reportType' product='2' />를 <Cmd sid='report.leave' product='2' /> 또는 <Cmd sid='report.exception' product='2' />로 설정했을 때만 활성화되며, 부재 기록이나 예외 기록의 세부 조건을 선택할 수 있습니다.

* <Cmd sid='common.userGroup' product='2' /> / <Cmd sid='common.user' product='2' />: 보고서를 생성할 사용자 그룹이나 사용자를 선택하세요.

* <Cmd sid='ta.report.condition.saveFilter.button' product='2' />: 설정한 근태 보고서를 필터로 저장할 수 있습니다.

  <Image src='/img/biostarx-tna-report-filter-saving.png' />


### 보고서 기간 설정

보고서 기간 설정 방법을 안내합니다.

<Image src='/img/biostarx-tna-report-period-setting.png' />

보고서 기간을 일별, 월별, 주별로 설정할 수 있습니다. 특정 기간의 보고서를 생성하려면 <Cmd sid='timeCard.custom' product='2' />를 선택하세요.


### 보고서 내보내기

* <Cmd sid='button.downloadToCsv' product='2' />: 생성한 보고서를 CSV 파일로 저장할 수 있습니다.

* <Cmd sid='button.exportPdf' product='2' />: 생성한 보고서를 PDF 파일로 저장할 수 있습니다.


## 경고 근무 시간 보고서 생성

지정된 근무 시간에 도달한 사용자의 근태 보고서를 생성하거나, 이메일로 관리자에게 알려줄 수 있습니다. 경고 근무 시간 보고서는 주별로 생성됩니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.tna' />를 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid='ta.report.title' product='2' /> 탭을 클릭하세요.

1. 화면 왼쪽 사이드바의 필터 목록에서 <Cmd>경고 근무 시간 보고서{/* Working alarm time Report */}</Cmd>를 클릭하세요.

    <Image src='/img/biostarx-tna-report-working-alarm.png' />

1. <Cmd sid='ta.report.condition.filterConditions' product='2' />과 <Cmd sid='ta.report.condition.reportPeriod' product='2' />의 각 항목을 설정하세요.

1. 보고서를 생성하려면 <Cmd sid='button.updateReport' product='2' /> 버튼을 클릭하세요.

:::info

지정된 근무 시간에 도달한 사용자에 대한 알림을 관리자에게 이메일로 보내려면 <Cmd sid='ta.report.condition.autoEmail' product='2' />을 설정하세요. 자세한 내용은 [다음 문서](#autoEmail)를 참고하세요.

:::


### 이메일 자동 발신 설정

지정된 근무 시간에 도달한 사용자에 대한 알람 이메일을 자동으로 관리자에게 보낼 수 있습니다.

<Image src='/img/biostarx-tna-report-working-alarm-auto-email.png' />

* <Cmd sid='common.email' product='2' />: 관리자에게 이메일을 자동으로 보내려면 체크박스를 클릭하세요.

* <Cmd sid='setting.dst.dayOfWeek' product='2' />: 관리자에게 이메일을 보낼 요일을 선택하세요.

* <Cmd sid='shift.time' product='2' />: 관리자에게 이메일을 보낼 시간을 설정하세요.

* <Cmd sid='setting.triggerAction.recipient' product='2' />: 이메일을 받을 관리자를 지정할 수 있습니다. <Cmd sid='button.edit' product='2' /> 버튼을 클릭하면 <Cmd sid='setting.triggerAction.recipient' product='2' /> 팝업창이 나타납니다. 이메일 주소를 입력하세요. 1명 이상의 관리자를 지정할 수 있습니다.

  <Image src='/img/biostarx-tna-report-working-alarm-auto-email-receipt.png' />

:::info

* <Cmd sid='ta.report.condition.autoEmail' product='2' /> 기능을 설정하려면 필터 조건을 설정한 후 필터를 저장해야 합니다.

* 자동 발신한 이메일의 보낸 사람 정보는 <Cmd sid='setting.setting' product='2' /> → <Cmd sid='setting.triggerAction.senderInfor' product='2' />에서 설정할 수 있습니다. 자세한 내용은 [다음 문서](tna-settings#senderInfo)를 참고하세요.

:::
