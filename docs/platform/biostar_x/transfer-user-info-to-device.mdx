description: 사용자 정보를 등록된 장치로 전송하는 방법을 안내합니다.
keywords: [ 사용자, 장치 ]
---

출입 통제를 위해 **BioStar X** 서버에 등록한 사용자 정보를 장치로 전송하는 기능을 제공합니다. 장치로 사용자 정보를 전송하면, 장치는 사용자 데이터를 활용하여 출입 권한을 판단하고, 오프라인 환경에서도 인증을 수행할 수 있습니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.user' />를 클릭하세요.

1. 사용자 목록에서 사용자 정보를 전송하려는 사용자의 가장 왼쪽에 체크 박스를 클릭하세요. 1명 이상의 사용자를 선택할 수 있습니다.

1. 화면 오른쪽 상단의 <IcMoreW/> 버튼을 선택하고 <Cmd sid='team.menu.transferToDevice' /> 버튼을 클릭하세요.

1. <Cmd sid='team.menu.transferToDevice' /> 창이 나타나면 장치 목록에서 사용자 정보를 전송할 장치를 선택하세요.

1. <Cmd>Send</Cmd> 버튼을 클릭하세요.

선택한 장치로 사용자 정보를 전송합니다.

:::info

* 중복되는 사용자 정보를 장치에 덮어쓰게 하려면 <Cmd sid='user.overwriteWhenTransfer' product='2' />를 선택하세요.

* 검색 아이콘 버튼을 클릭해 장치를 검색할 수도 있습니다.

* 사용자 정보를 전송하려면 장치가 **BioStar X** 서버에 연결되어 있어야 합니다.

* 장치에 이미 등록된 사용자의 정보를 변경한 경우, 변경 사항을 장치에 반영하려면 사용자 정보를 다시 전송해야 합니다. 사용자 정보를 장치와 동기화하는 <Cmd sid='setting.server.automaticUserSync' product='2' /> 옵션을 이용하려면 [다음 문서](#)를 참고하세요.

* 장치마다 등록할 수 있는 사용자 수에 제한이 있습니다. 장치에 등록할 수 있는 사용자 수를 확인하세요.

:::
