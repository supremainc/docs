**BioStar X**의 데이터베이스, 각종 설정, 키 등 주요 정보를 백업할 수 있습니다. <Cmd sid="common.backup" product='2' /> 기능을 사용하면 서버의 데이터 손실을 방지하거나 새로운 PC로 이전할 때 사용합니다.

지원하는 백업 방식은 아래와 같습니다.

* **수동 백업**: 사용자가 수동으로 백업 파일을 생성합니다.

* **자동 백업**: 시스템에서 정해진 주기마다 백업 파일을 자동 생성합니다.

:::info

* 이전 PC에서 백업한 **BioStar X** 백업 파일로 새 PC에서 **BioStar X** 복원을 진행하면 라이선스를 재발급 받아야 합니다.

* SQL Server 데이터베이스와 **BioStar X**가 서로 다른 PC에 설치되어 있다면 **BioStar X** 시스템 백업 및 복원을 사용할 수 없습니다.

* 시스템 복원에 대한 자세한 내용은 [다음 문서](settings-system-system-restore)를 참고하세요.

:::

## 수동 백업

사용자가 직접 백업 파일을 생성할 수 있습니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.setting' />을 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid='setting.menu.system.root' /> → <Cmd sid="common.backup" product='2' />을 클릭하세요.

    <Image src='/img/biostarx-settings-system-system-backup-general.png' className='none' />

1. <Cmd sid='common.backup.general' product='2' /> 섹션에서 백업 파일을 저장할 경로와 최대 백업 파일 수를 설정하세요.

    * <Cmd sid='backup.filePath' product='2' />: 백업 파일을 저장할 경로를 입력하세요. 저장 경로는 이미 생성되어 있는 디렉토리 경로를 입력하세요.

    * <Cmd sid='backup.numberFileToKeep' product='2' />: 최대 백업 파일 수를 설정하세요.

1. 백업 파일을 생성하려면 <Cmd sid="backup.backupNow" product='2' /> 버튼을 클릭하세요.

백업 파일은 <Cmd sid='backup.filePath' product='2' /> 옵션에 지정된 경로에 *zip* 파일 형식으로 저장됩니다. 파일 이름은 `BioStar_X_Backup_YYYYMMDD_HHMMSS.zip` 형식으로 생성됩니다. 여기서 `YYYYMMDD`는 백업 날짜, `HHMMSS`는 백업 시간입니다.

:::info

* 최대 백업 수를 초과할 경우 가장 오래된 백업 파일부터 자동으로 삭제됩니다.

* <Cmd sid="backup.numberFileToKeep" product='2' />는 1 ~ 100 사이의 숫자만 입력할 수 있습니다.

:::


## 자동 백업 설정

자동 백업을 설정하면 시스템에서 정해진 주기마다 백업 파일을 생성합니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.setting' />을 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid='setting.menu.system.root' /> → <Cmd sid="common.backup" product='2' />을 클릭하세요.

    <Image src='/img/biostarx-settings-system-system-backup-automatic.png' className='none' />

1. <Cmd sid='common.backup.autoBackup' product='2' /> 섹션에서 <Cmd sid='common.backup.frequency' product='2' /> 옵션을 선택하세요.

    * <Cmd sid="backup.daily" product='2' />: 매일 자동으로 백업합니다. <Cmd sid='backup.time' product='2' /> 옵션에서 백업 시간을 설정하세요.

    * <Cmd sid="backup.weekly" product='2' />: 매주 자동으로 백업합니다. <Cmd sid='backup.day' product='2' /> 옵션에서 자동으로 백업할 요일을 설정하고, <Cmd sid='backup.time' product='2' /> 옵션에서 백업 시간을 설정하세요.

    * <Cmd sid="backup.monthly" product='2' />: 매월 자동으로 백업합니다. <Cmd sid='backup.date' product='2' /> 옵션에서 자동으로 백업할 날짜를 설정하고, <Cmd sid='backup.time' product='2' /> 옵션에서 백업 시간을 설정하세요.

1. 설정을 저장하려면 <Cmd sid="button.apply" product='2' />을 클릭하세요.

백업 파일은 <Cmd sid='common.backup.general' product='2' /> 섹션의 <Cmd sid='backup.filePath' product='2' /> 옵션에 지정된 경로에 *zip* 파일 형식으로 저장됩니다. 파일 이름은 `BioStar_X_Backup_YYYYMMDD_HHMMSS.zip` 형식으로 생성됩니다. 여기서 `YYYYMMDD`는 백업 날짜, `HHMMSS`는 백업 시간입니다.

:::info

* 자동 백업 빈도를 <Cmd sid="backup.monthly" product='2' />으로 선택하고, 날짜를 29, 30, 31일로 설정하면 해당 날짜가 없는 달에는 자동 백업을 실행하지 않습니다.

* 시간은 <Cmd sid='setting.menu.preferences.root' />에서 설정한 표준 시간대를 기준으로 합니다. 표준 시간대 설정에 대한 자세한 내용은 [다음 문서](settings-preferences#language-timezone)를 참고하세요.

:::
