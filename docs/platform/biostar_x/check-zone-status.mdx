각 구역의 상태를 확인하세요. 안내된 아이콘과 상세 설명을 통해 관리자들이 각 구역의 작동 상태를 한눈에 파악하고, 문제 발생 시 신속하게 대응할 수 있도록 돕습니다.

<Cmd sid='common.header.launcher' /> 페이지에서 <Cmd sid='common.header.arena' />을 클릭하거나 화면 왼쪽 상단의 바로가기 목록에서 <Cmd sid='common.header.arena' />을 선택하세요. 화면 왼쪽 사이드바의 <Cmd sid='arena.tree.door.title' /> 탭에서 <Cmd sid='arena.tree.door.allZones' />로 이동하세요.

:::info

구역 설정에 대한 자세한 내용은 [다음 문서](#)를 참조하세요.

:::

## 구역 알람 확인

각 구역에서 이벤트가 발생하면 <IcErr width='20px' height='20px' /> 아이콘이 표시됩니다. 알람 아이콘을 클릭하면 발생한 이벤트를 확인할 수 있습니다.

## 구역 상태 확인

| 상태 아이콘 | 설명 |
| :--: | -- |
| <IcDisable width='auto' height='30px' /> | 비활성화된 구역의 아이콘입니다. 비활성화된 구역은 알람이 발생하지 않습니다. |
| <IcArm width='auto' height='35px'/> | Intrusion Alarm 구역의 경비가 활성화된 상태입니다. |

:::info

활성화된 구역은 아이콘이 표시되지 않습니다.

:::
