**BioStar X**의 사용자 인터페이스(User Interface, 이하 UI)는 사용자가 시스템을 쉽게 탐색하고 사용할 수 있도록 설계되었습니다. 이 문서에서는 **BioStar X**의 공통 UI 구성 요소를 안내합니다. 이 문서를 통해 **BioStar X**의 UI를 이해하고, 시스템을 보다 효율적으로 사용할 수 있습니다.

화면의 헤더 영역은 모든 페이지에서 공통으로 사용되며, **BioStar X**의 다양한 기능에 접근할 수 있는 링크 버튼을 제공합니다.

<Image src='/img/biostarx-header.png' />

* <Num num='1' /> **BioStar X 로고**: 사용자가 **BioStar X**의 <Cmd sid='shell.header.launcher' /> 페이지로 돌아갈 수 있는 링크 버튼입니다.

* <Num num='2' /> **바로가기**: **BioStar X**에서 제공하는 주요 페이지로 이동할 수 있는 링크를 제공합니다.

* <Num num='3' /> <Cmd sid='shell.header.action' />: 서버 사용자가 설정한 **액션**(Action) 기능을 실행할 수 있습니다. 버튼을 클릭하면 액션 목록이 나타납니다. 자세한 내용은 [다음 문서](#)를 참고하세요.

* <Num num='4' /> <IcSvTime width='26' height='26' /> **서버 시간**: 운영 중인 **BioStar X** 서버의 서버 시간을 확인할 수 있습니다. 마우스를 버튼 위로 오버하면 현재 서버 시간이 표시됩니다.

* <Num num='5' /> <IcAccout width='26' height='26' /> **사용자 계정**: 현재 로그인한 사용자의 계정 정보를 확인할 수 있습니다.

* <Num num='6' /> <IcHelp width='26' height='26' /> **도움말**: **BioStar X**의 도움말 페이지로 이동할 수 있는 링크 버튼입니다.

* <Num num='7' /> <IcAibtn width='26' height='26' /> **AI 프롬프트**: **BioStar X**의 AI 프롬프트 기능을 사용할 수 있는 버튼입니다. 버튼을 클릭하면 AI 프롬프트 창이 나타납니다. 사용자의 질문에 답변하고 원하는 기능을 실행할 수 있습니다. 자세한 내용은 [다음 문서](#)를 참고하세요.
