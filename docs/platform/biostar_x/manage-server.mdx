**BioStar X** 서버의 상태를 관리하는 방법을 안내합니다. 실행 중인 서비스를 종료하거나 다시 시작할 수 있습니다.

## BioStar X Service Manager {#biostar-x-setting}

**BioStar X** 서버의 상태를 관리할 수 있는 도구를 제공합니다. <Start/> → **BioStar X** → **BioStar X Service Manager**를 클릭하세요. <Cmd>BioStar X Service Manager</Cmd> 창이 나타납니다.

<Image src='/img/common/biostarx-tutoria-service-manager-main.png' alone />

:::info

**BioStar X** 서버의 시간 설정을 변경했다면 **BioStar X Core Service**를 종료하고 다시 시작하세요. 다시 시작하지 않으면 **BioStar X**가 동작하지 않을 수 있습니다.

:::

### 서비스 종료하기 {#end-service}

**BioStar X**를 종료하려면, <Cmd>SERVICES</Cmd> 메뉴에서 개별 서비스의 <Cmd>Stop</Cmd> 버튼을 클릭하세요.

<Image src='/img/common/biostarx-tutoria-service-manager-main.png' alone />

{/* 모든 서비스를 종료하려면 화면 오른쪽 상단의 <Cmd>Stop All</Cmd> 버튼을 클릭하세요. */}


### 서버 시작하기 {#start-the-server}

**BioStar X**를 다시 시작하려면, <Cmd>SERVICES</Cmd> 메뉴의 <Cmd>Status</Cmd> 열에서 <Cmd>Stopped</Cmd> 상태인 서비스의 <Cmd>Start</Cmd> 버튼을 클릭하세요.

<Image src='/img/common/biostarx-tutoria-service-manager-main-restart.png' alone />

{/* 모든 서비스를 시작하려면 화면 오른쪽 상단의 <Cmd>Start All</Cmd> 버튼을 클릭하세요. */}
