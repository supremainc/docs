출입문 및 관련 장비의 상태를 정확하게 파악하는 것은 보안 시스템의 핵심입니다. 이 문서에서는 장치의 상태를 확인하는 방법과 오류 발생 시 대처 방안을 안내합니다. 안내된 아이콘과 상세 설명을 통해 관리자들이 각각 장치의 작동 상태를 한눈에 파악하고, 문제 발생 시 신속하게 대응할 수 있도록 돕습니다.

<Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.arena' />을 클릭하거나 화면 왼쪽 상단의 바로 가기 목록에서 <Cmd sid='shell.header.arena' />을 선택하세요. 화면 왼쪽 사이드바의 <Cmd sid='arena.tree.door.title' /> 탭에서 각 출입문과 연결된 슬레이브 장치의 상태를 확인할 수 있습니다.

## 출입문 상태 확인 {#check-the-door-status}

출입문 상태를 확인하여 출입문이 정상적으로 작동하는지 확인할 수 있습니다. 출입문 상태는 아래를 참고하세요.

<div className='col2--20-80 overflow-x'>

| 상태 아이콘 | 설명 |
| :--: | -- |
| <IcErr width='35px' height='35px' /> | 커뮤니케이션 서버 오류이거나 연결이 끊긴 상태입니다. |
| <IcNotsync width='35px' height='35px' /> | 동기화 오류가 발생한 상태입니다. |

</div>

:::tip

상태 아이콘을 클릭하면 툴팁을 통해서 상세 정보를 확인할 수 있습니다. 툴팁에 표시된 오류 메시지의 링크를 클릭하면 출입문과 연결된 장치 목록으로 이동할 수 있습니다.

<Image src='/img/monitoring-door-status-tooltip.png' alt='툴팁 메시지'/>

해당 장치에서 마우스 오른쪽 버튼을 클릭하고, <Cmd>재연결</Cmd> 또는 <Cmd>재시작</Cmd> 옵션을 선택하세요.

<Image src='/img/monitoring-door-status-tooltip-device.png' alt='장치 목록' />

:::

## 출입문 열림/잠금 확인 {#open-the-doorlock-confirmation}

출입문 센서를 통해 출입문이 열려 있는지 확인할 수 있습니다. 출입문 센서의 상태는 아래를 참고하세요.

<div className='col2--20-80 overflow-x'>

| 상태 아이콘 | 설명 |
| :--: | -- |
| <IcDoorClose width='35px' height='35px' /> | 출입문이 닫힌 상태입니다. |
| <IcDoorOpen width='auto' height='40px' /> | 출입문이 열린 상태입니다. |

</div>


## 출입문 잠금 확인 {#check-the-door-lock}

릴레이 상태를 통해 출입문의 잠금 상태를 확인할 수 있습니다. 출입문 릴레이의 상태는 아래를 참고하세요.

<div className='col2--20-80 overflow-x'>

| 상태 아이콘 | 설명 |
| :--: | -- |
| <IcRelayLock width='auto' height='35px' /> | 출입문이 닫힌 상태입니다. |
| <IcRelayUnlock width='auto' height='40px' /> | 출입문이 열린 상태입니다. |

</div>


## 카메라 상태 확인 {#check-the-camera-status}

카메라의 상태를 확인하여 카메라가 정상적으로 작동하는지 확인할 수 있습니다. 카메라의 상태는 아래를 참고하세요.

<div className='col2--20-80 overflow-x'>

| 상태 아이콘 | 설명 |
| :--: | -- |
| <StatusOK/> | 카메라가 연결된 상태입니다. |
| <StatusFail /> | 카메라가 꺼져 있거나 연결이 해제된 상태입니다. |

</div>


## 경비 상태 확인 {#check-the-security-status}

출입문의 경비 상태를 확인할 수 있습니다. 

<div className='col2--20-80 overflow-x'>

| 상태 아이콘 | 설명 |
| :--: | -- |
| <IcArm width='auto' height='35px'/> | 경비가 활성화된 상태입니다. |
| <IcDisarm width='auto' height='35px'/> | 경비가 비활성화된 상태입니다. |

</div>

:::info

경비 상태는 출입문을 Intrusion Alarm Zone으로 설정했을 때만 표시됩니다.

:::
