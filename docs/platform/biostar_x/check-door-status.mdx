출입문의 상태를 실시간으로 모니터링하여 보안 시스템을 안정적으로 운영할 수 있습니다. <Cmd sid='shell.header.arena' />에서 제공하는 상태 아이콘을 통해 출입문에 할당된 장치의 이상 상태를 한눈에 확인하고, 문제 발생 시 즉시 대응하세요.

<Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.arena' />을 클릭하거나 화면 왼쪽 상단의 바로 가기 목록에서 <Cmd sid='shell.header.arena' />을 선택하세요. 화면 왼쪽 사이드바의 <Cmd sid='arena.tree.door.title' /> 탭에서 각 출입문의 상태를 확인할 수 있습니다.

## 시작하기 전에

출입문과 연결된 장치를 올바르게 구성했는지 확인하세요. 장치가 네트워크에 연결되어 있고, 최신 펌웨어로 업데이트되어 있는지 점검하는 것이 중요합니다. 문제가 발생할 경우, 장치의 전원 상태와 네트워크 연결 상태를 먼저 확인하는 것이 좋습니다.

:::info

* 장치 추가 및 구성에 대한 자세한 내용은 [다음 문서](settings-manage-devices)를 참고하세요.

* 출입문 추가 및 구성에 대한 자세한 내용은 [다음 문서](settings-manage-door)를 참고하세요.

* 출입문의 상태를 모니터링하거나 제어하려면 <Cmd sid='setting.server.logUploadType' product='2' /> 기능이 활성화되어 있어야 합니다. 이 기능은 <Cmd sid='shell.header.setting' /> → <Cmd sid='setting.menu.server.root' /> → <Cmd sid='setting.menu.server.server' /> 메뉴 경로의 <Cmd sid='common.general' product='2' /> 섹션에서 활성화할 수 있습니다.

* VMS 서버와 연동해 카메라를 출입문과 연결하는 기능은 **어드밴스**(Advanced) 라이선스 이상에서 추가 옵션을 통해 사용할 수 있습니다. 라이선스 정책에 대한 자세한 내용은 [다음 문서](licensing#features-add-ons)를 참고하세요.

:::

## 상태 확인하기

### 연결 오류 및 알람 확인 {#check-the-error-status}

출입문 또는 엘리베이터 상태를 확인하여 정상 작동 여부를 확인할 수 있습니다. 각 아이콘별 상태는 아래를 참고하세요.

<div className='col2--20-80 overflow-x'>

| 상태 아이콘 | 설명 |
| :--: | -- |
| <IcErr width='35px' height='35px' /> | 장치 연결 끊김, 커뮤니케이션 서버 오류, 동기화 오류 등 장치 이상 상태를 표시합니다. |
| <IcAlert width='40px' height='40px' /> | 출입문 알람(강제 문열림, 장시간 문열림, APB, 화재 알람, 비상 개방) 또는 AI 영상 분석 알람(배회 감지, 침입 감지 등)이 발생한 상태를 표시합니다. |

</div>

:::tip

상태 아이콘을 클릭하면 툴팁을 통해서 상세 정보를 확인할 수 있습니다. 툴팁에 표시된 오류 메시지의 링크를 클릭하면 출입문 또는 엘리베이터와 연결된 장치 목록으로 이동할 수 있습니다.

<Image src='/img/monitoring-door-status-tooltip.png' />

해당 장치에서 마우스 오른쪽 버튼을 클릭하고, <Cmd sid='arena.tree.contextMenu.reconnect' /> 또는 <Cmd sid='arena.tree.contextMenu.reboot' /> 옵션을 선택하세요.

<Image src='/img/monitoring-door-status-tooltip-device.png' />

:::

:::info

* 출입문 또는 엘리베이터에서 발생하는 알람은 다음 설정을 통해 구성할 수 있습니다.

  * [출입문 알람 설정](settings-door-add#alarm)

  * [엘리베이터 알람 설정](settings-elevators-add#alarm)

  * [고급 출입 통제 설정](settings-advanced-ac)

* 비디오 규칙에서 출입문과 카메라를 연결하여 알람을 설정할 수 있습니다. 비디오 규칙 설정에 대한 자세한 내용은 [다음 문서](settings-videos-rule)를 참고하세요.

* VMS 서버와 연동 시 영상에 대한 AI 이벤트 알람을 설정하면 **BioStar X**로 알람을 전송할 수도 있습니다. 자세한 내용은 VMS 서버의 매뉴얼을 참고하세요.

* 알람 해제에 대한 자세한 내용은 [다음 문서](monitoring-event#acknowledge-alarm-and-record-action)를 참고하세요.

:::


### 출입문 잠금 확인 {#check-the-door-lock}

출입문 릴레이 상태를 통해 출입문의 잠금 상태를 확인할 수 있습니다. 출입문 릴레이의 상태는 아래를 참고하세요.

<div className='col2--20-80 overflow-x'>

| 상태 아이콘 | 설명 |
| :--: | -- |
| <IcRelayLock width='auto' height='35px' /> | 출입문이 잠긴 상태입니다. |
| <IcRelayUnlock width='auto' height='40px' /> | 출입문이 열린 상태입니다. |

</div>


### 카메라 상태 확인 {#check-the-camera-status}

카메라의 상태를 확인하여 카메라가 정상적으로 작동하는지 확인할 수 있습니다. 카메라의 상태는 아래를 참고하세요.

<div className='col2--20-80 overflow-x'>

| 상태 아이콘 | 설명 |
| :--: | -- |
| <IcCamOk width='auto' height='35px'/> | 카메라가 연결된 상태입니다. |
| <IcCamFail width='auto' height='35px'/> | 카메라가 꺼져 있거나 연결이 해제된 상태입니다. |

</div>


### 경비 상태 확인 {#check-the-security-status}

출입문 또는 고급 출입 통제 설정의 경비 상태를 확인할 수 있습니다. 

<div className='col2--20-80 overflow-x'>

| 상태 아이콘 | 설명 |
| :--: | -- |
| <IcDisable width='auto' height='30px' /> | 비활성화된 설정의 아이콘입니다. 비활성화된 설정은 알람이 발생하지 않습니다. |
| <IcArm width='auto' height='35px'/> | 경비가 활성화된 상태입니다. |
| <IcDisarm width='auto' height='35px'/> | 경비가 비활성화된 상태입니다. |

</div>

:::info

경비 상태는 출입문을 **경비**으로 설정했을 때만 표시됩니다. 경비 설정에 대한 자세한 내용은 [다음 문서](settings-advanced-ac-intrusion-alarm)를 참고하세요.

:::

## 그룹 목록 확장/축소

최상위 그룹 또는 하위 그룹을 통해서 목록을 확장하거나 축소할 수 있습니다. 

<Image src='/img/biostarx-monitoring-group-expand-collapse.png' />

### 최상위 그룹

1. <Cmd sid='arena.tree.door.title' /> 탭에서 <Cmd sid='arena.tree.door.allDoorGroups' /> 또는 <Cmd sid='arena.tree.door.allElevatorGroups' />, <Cmd sid='arena.tree.door.allZones' />를 선택하고 마우스 오른쪽 버튼을 클릭하세요. 

1. 팝업 메뉴에서 <Cmd sid='arena.tree.expandAll' /> 또는 <Cmd sid='arena.tree.collapseAll' /> 옵션을 선택하세요. 

선택한 최상위 그룹을 기준으로 모든 하위 목록을 확장하거나 축소할 수 있습니다.

### 하위 그룹

1. 각 최상의 그룹의 하위 그룹 목록을 확장하거나 축소하려면 하위 그룹을 선택하고 마우스 오른쪽 버튼을 클릭하세요. 

1. 팝업 메뉴에서 <Cmd sid='arena.tree.contextMenu.expandBelow' /> 또는 <Cmd sid='arena.tree.contextMenu.collapseBelow' /> 옵션을 선택하세요. 

선택한 하위 그룹의 목록을 확장하거나 축소할 수 있습니다.
