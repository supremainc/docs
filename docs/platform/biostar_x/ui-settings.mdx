<Cmd sid='shell.header.setting' /> 페이지에서는 **BioStar X**의 다양한 설정을 관리할 수 있습니다. 이 페이지의 UI 구성 요소를 살펴보면 다음과 같습니다.

<Image src='/img/biostarx-ui-settings.png' className='none' />

<Calloutlists className='svg-none-style'>

1. <Cmd sid='shell.header.setting' /> 메뉴에서 제공하는 다양한 설정 메뉴를 확인할 수 있습니다. 자세한 내용은 [다음 문서](how-to-guide-settings)를 참고하세요.

1. 페이지 목록을 탐색할 수 있는 도구입니다. 페이지를 이동하거나 원하는 페이지로 이동할 수 있습니다.

    <Image src='/img/biostarx-settings-access-control-status-pagenation.png' />

    * <IcFirst2 />: 첫 페이지로 이동합니다.

    * <IcFw2 />: 이전 페이지로 이동합니다.

    * <IcBw2 />: 다음 페이지로 이동합니다.

    * <IcLast2 />: 마지막 페이지로 이동합니다.

    * 원하는 페이지로 이동하려면 숫자 입력 필드에 페이지 번호를 입력하세요.

    * 한 페이지에 표시할 목록 개수를 설정하려면 행 선택 상자를 클릭하세요.

    :::info

    설정 메뉴에 따라 지원하지 않을 수 있습니다.

    :::

1. <IcMoreW /> 버튼을 클릭하면 추가 기능을 사용할 수 있습니다.

    :::info

    설정 메뉴에 따라 지원하는 기능은 다를 수 있습니다.

    :::

</Calloutlists>
