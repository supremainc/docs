BioStar X의 이중화(클러스터) 설정을 통해 시스템의 가용성과 안정성을 높일 수 있습니다. 이중화 설정은 두 대의 서버가 서로 백업 역할을 하여 하나의 서버에 장애가 발생하더라도 다른 서버가 서비스를 계속 제공할 수 있도록 합니다.

## 시작하기 전에 {#start-before}

이중화 설정을 시작하기 전에 다음 사항을 확인하세요.
* 두 대의 서버가 동일한 네트워크에 연결되어 있어야 합니다.
* 각 서버에 BioStar X Service Manager가 설치되어 있어야 합니다.
* 각 서버의 방화벽 설정이 BioStar X 서비스에 필요한 포트를 허용하도록 구성되어 있어야 합니다.
* 각 서버의 시간 동기화가 정확하게 설정되어 있어야 합니다.
* 각 서버의 IP 주소와 호스트 이름이 고유해야 합니다.
* 이중화 설정을 위해 관리자 권한이 필요합니다.
