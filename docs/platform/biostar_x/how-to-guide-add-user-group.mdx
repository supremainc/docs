여러 명의 사용자를 효율적으로 관리하기 위해 그룹을 생성하세요. 사용자가 속한 조직 등으로 이름을 등록하면 편리하게 관리할 수 있습니다.

:::note

* 그룹 이름은 실제 조직에서 관리하는 부서 이름을 사용하면 관리가 편리합니다.

* 그룹의 하위 그룹을 최대 8단계까지 순차적으로 생성할 수 있습니다.

* 사용자 그룹 이름은 최대 48자까지 입력할 수 있습니다.

* 사용자 그룹 목록에서 그룹을 선택하면 해당 그룹에 속한 사용자만 사용자 목록에 표시할 수 있습니다.

:::

## 상위 그룹 생성하기

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 왼쪽 사이드바의 <Cmd>User Group</Cmd> 탭에서 <Cmd>All Users</Cmd>를 선택하고 마우스 오른쪽 버튼을 클릭하세요.

1. 팝업 메뉴에서 <Cmd>Add New Group</Cmd>을 클릭하세요.

1. 그룹이 생성되면 원하는 그룹 이름을 입력하세요.


## 하위 그룹 생성하기

1. 왼쪽 사이드바의 <Cmd>User Group</Cmd> 탭에서 최상위 그룹을 선택하고 마우스 오른쪽 버튼을 클릭하세요.

1. 팝업 메뉴에서 <Cmd>Add New Group</Cmd>을 클릭하세요.

1. 그룹이 생성되면 원하는 그룹 이름을 입력하세요.
