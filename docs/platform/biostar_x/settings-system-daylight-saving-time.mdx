<Cmd sid='common.dst' product='2' />(DST)은 자연 일광을보다 잘 활용하기 위해 시간을 조정하는 기능입니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.setting' />을 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid='common.dst' product='2' />을 클릭하세요.

1. <Cmd sid="button.plusAdd" product='2' />를 클릭하세요.

1. 각 항목을 설정한 뒤 <Cmd sid="button.add" product='2' />를 클릭하세요.

    <Image src='/img/biostarx-settings-system-daylight-saving-time.png' className='none' />

1. 설정을 저장하려면 <Cmd sid="button.apply" product='2' />을 클릭하세요.

:::info

이미 사용 중인 일광 절약 시간은 편집하거나 삭제할 수 없습니다.

:::
