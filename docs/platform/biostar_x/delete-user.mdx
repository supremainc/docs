---
id: delete-user
title: 사용자 삭제하기
description: 등록한 사용자를 삭제하는 방법을 안내합니다.
keywords: [ 사용자, 삭제 ]
---

**BioStar X**에 등록한 사용자를 퇴사, 계약 종료, 보안 위반 등의 사유로 삭제해야 하는 경우, 다음의 안내에 따라 삭제할 수 있습니다.

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 사용자 목록에서 삭제할 사용자를 선택하세요. 1명 이상 선택할 수 있습니다.

1. 화면 오른쪽 상단의 <Cmd>Delete</Cmd> 버튼을 클릭하세요.

1. 확인 메시지가 나타나면 <Cmd>확인</Cmd> 버튼을 클릭하세요.

사용자 목록에서 선택한 사용자가 삭제된 것을 확인하세요.

:::note

사용자 정보를 삭제하기 전에 장치에서 사용자 정보를 우선 삭제하세요. 장치에서 사용자 정보를 삭제하는 방법에 대한 자세한 내용은 [다음 문서](user-info-with-device#delete-from-device)를 참조하세요.

:::
