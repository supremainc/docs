사용자의 정보를 수정하고 권한을 변경하는 방법을 안내합니다. 사용자의 기본 정보를 수정하고 사용자의 출입 권한 및 크리덴셜을 변경할 수 있습니다.

## 사용자 정보 수정

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요. 

1. 사용자 목록에서 수정할 사용자를 선택하세요.

1. 화면 오른쪽에 사용자의 프로필 정보가 표시되면 <Cmd>See More</Cmd> 버튼을 클릭하세요.

1. 사용자 정보를 수정할 수 있는 창이 나타납니다.

1. 원하는 항목을 수정하고 <Cmd>Save</Cmd> 버튼을 클릭하세요.

:::info

* <Cmd>Information</Cmd> 섹션의 각 항목에 대한 자세한 내용은 [다음 문서](how-to-guide-add-user#information)를 참고하세요.

* <Cmd>Permission</Cmd> 섹션의 각 항목에 대한 자세한 내용은 [다음 문서](how-to-guide-add-user#permission)를 참고하세요.

* <Cmd>Credential</Cmd> 섹션의 각 항목에 대한 자세한 내용은 [다음 문서](how-to-guide-add-user#credential)를 참고하세요.

:::

## 사용자 정보 일괄 수정

한 개 이상의 사용자 정보를 일괄 수정할 수 있습니다. 이 기능은 다수의 사용자에게 동일한 출입 권한이나 그룹을 설정할 때 유용합니다.

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요. 

1. 사용자 목록에서 1명 이상의 사용자를 선택하세요.

1. 화면 오른쪽 상단의 <Cmd>Batch Edit</Cmd> 버튼을 클릭하세요.

1. 일관 편집 화면 창이 나타나면 원하는 정보를 수정하세요.

1. 수정한 정보를 저장하려면 <Cmd>Save</Cmd> 버튼을 클릭하세요.

:::info

사용자의 ID 및 이름, 이메일, 전화번호, 크리덴셜과 같은 개인정보는 일괄 수정할 수 없습니다.

:::
