BioStar X가 정상적으로 동작하지 않을 경우 BioStar X Restore 프로그램을 이용하여 백업해 둔 파일로 복원 지점을 선택하여 복원할 수 있습니다.

:::info

MSSQL 데이터베이스와 BioStar X가 서로 다른 PC에 설치되어 있는 경우 BioStar X 시스템 백업 및 복원을 사용할 수 없습니다.

:::

1. <Start/> → **BioStar X** → **BioStar X Restore**를 실행하세요.

    * 프로그램 경로: C:\Program Files\BioStar X\biostar-restore.exe

1. **Select File**을 클릭해 저장한 백업 파일을 선택한 뒤 **Restore**를 클릭하면 복원이 시작됩니다.

    <Image src='/img/common/setting-system-restore.png' className='none' alone />

1. **Starting Biostar services...Done** 문구가 출력되며 복원이 완료됩니다.

1. BioStar X에 재접속하세요.

:::info

백업 시점의 BioStar X 버전과 현재 버전이 다를 경우 복원이 진행되지 않습니다.

:::
