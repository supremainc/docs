## 출입 권한별 사용자 확인

출입 권한 그룹별로 등록한 사용자를 확인하는 방법을 안내합니다. 사용자의 출입 권한을 확인하고 사용자의 정보를 수정할 수 있습니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.user' />를 클릭하세요.

1. 트리 구조 메뉴에서 <Cmd sid='team.tree.accessExplorer' /> 탭을 클릭하세요.

    <Image src='/img/biostarx-user-access-explorer.png' />

1. <Cmd>All Access Groups</Cmd>에서 원하는 출입 권한 그룹을 클릭하세요.

선택한 출입 권한 그룹에 속한 사용자를 사용자 목록에서 확인할 수 있습니다.

:::info

트리 구조 메뉴는 아래와 같은 구조로 구성되어 있습니다.

<Columns>
<Column><Image src='/img/biostarx-user-access-explorer-tree-level.png' /></Column>

<Column>

**All Access Groups**

* Access Group: 출입 권한 그룹

  * Access Level: 출입 레벨

    * Door: 출입문

    * Schedule: 스케줄

  * Floor Level: 층 레벨

      * Elevator: 엘리베이터

      * Schedule: 스케줄

</Column>
</Columns>

:::

## 출입 그룹 목록 보기

출입 그룹 목록을 그룹별, 레벨별, 출입문과 스케쥴별로 확인할 수 있습니다. <Cmd sid='team.tree.accessExplorer' /> 탭의 상위에 <IcMoreB /> 버튼을 클릭하고 팝업 메뉴에서 <Cmd>View</Cmd>를 클릭하세요. 원하는 항목을 선택해 목록을 확인하세요.

<Image src='/img/biostarx-user-access-explorer-view.png' />

* <Cmd>Parent Access Group</Cmd>: 최상위 출입 권한 그룹을 기준으로 목록을 확인할 수 있습니다.

* <Cmd>Access Group</Cmd>: 모든 출입 권한 기준으로 목록을 확인할 수 있습니다.

* <Cmd>Access Level</Cmd>: 출입 레벨 기준으로 목록을 확인할 수 있습니다.

* <Cmd>Door &amp; Schedule</Cmd>: 출입문과 스케줄 기준으로 목록을 확인할 수 있습니다.
