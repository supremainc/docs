description: 출입 권한별로 등록된 사용자를 확인하는 방법을 안내합니다.
keywords: [ 사용자, 출입 권한 ]
---

출입 권한 그룹별로 등록한 사용자를 확인하는 방법을 안내합니다. 사용자의 출입 권한을 확인하고 사용자의 정보를 수정할 수 있습니다.

1. <Cmd>런처</Cmd> 페이지에서 <Cmd>사용자</Cmd>를 클릭하세요.

1. 트리 구조 메뉴에서 <Cmd>액세스 탐색기</Cmd> 탭을 클릭하세요.

1. <Cmd>All Access Groups</Cmd>에서 원하는 출입 권한 그룹을 클릭하세요.

사용자 목록에서 선택한 출입 권한 그룹에 속한 사용자를 확인할 수 있습니다.

:::note

트리 구조 메뉴는 아래와 같은 구조로 구성되어 있습니다.

* All Access Groups

  * Access Group: 출입 권한 그룹

    * Access Level: 출입 레벨

      * Door & Schedule: 출입문 및 스케줄

:::
