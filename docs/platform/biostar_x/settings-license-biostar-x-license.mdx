구매한 BioStar X 라이선스를 활성화할 수 있습니다.

1. <Cmd sid='shell.header.launcher' /> 페이지에서 <Cmd sid='shell.header.setting' />을 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid="common.license" product='2' />를 클릭하세요.

<Image src='/img/biostarx-settings-license-license.png' className='none' />

:::info

* 온라인으로 BioStar X 라이선스를 활성화하려면 <Cmd sid="setting.server.license.customerName" product='2' />과 전달받은 <Cmd sid="setting.server.license.activationKey" product='2' />를 입력한 뒤 <Cmd sid="setting.server.license.button.activate" product='2' />를 클릭하세요.

* 오프라인으로 BioStar X 라이선스를 활성화하려면 <Cmd sid="setting.server.license.button.requestOfflineKey" product='2' />을 클릭한 뒤 대화창의 지시에 따라 활성화하세요.

:::
