사용자 정보를 내보내거나 불러오는 방법에 대해 안내합니다. 이 기능을 통해 이전 버전 또는 다른 서버의 사용자 정보를 활용할 수 있습니다.

- **데이터 백업 및 복원**: 사용자 정보를 안전하게 백업할 수 있으며, 필요 시 데이터를 복원할 수 있습니다.

- **서버 간 데이터 이전**: 다른 서버로 사용자 정보를 쉽게 이전할 수 있어, 서버 교체나 업그레이드 시 유용합니다.

- **데이터 통합**: 여러 소스에서 사용자 정보를 수집하여 하나의 서버에 통합할 수 있습니다.

- **효율적인 관리**: 대량의 사용자 정보를 일괄적으로 관리할 수 있어, 시간과 노력을 절약할 수 있습니다.

사용자 정보는 CSV 파일과 데이터 파일(*.tgz*) 형식으로 내보내거나 불러올 수 있습니다. 사용자의 환경에 따라 선호하는 방식을 선택해 진행하세요.

:::tip

**CSV 파일 불러오기와 데이터 파일 불러오기 차이점**

* **CSV 파일 불러오기**

  - 단순 텍스트 기반의 CSV 형식으로, 사용자 목록의 선택된 정보만 포함합니다.

  - 사용자 정보를 백업 또는 복원을 위한 소규모 업데이트나 단순 데이터 관리에 적합합니다.

* **데이터 파일 불러오기**

  - 프로필 사진, 인증 모드, 지문, 얼굴 등 다양한 인증 정보를 포함하는 *\*.tgz* 압축 파일 형식입니다.

  - 대량의 사용자 정보를 장치로 직접 전송하거나, 복합 인증 데이터를 포함할 때 사용하세요.

:::

## CSV 파일 활용

사용자 목록에서 선택한 사용자 정보를 CSV 파일로 저장하거나, CSV 파일에서 사용자 정보를 불러올 수 있습니다. CSV 파일을 이용해 간편하게 사용자 정보를 관리할 수 있습니다.

:::note

* CSV 파일을 불러오기 전에, **BioStar X**에 추가되지 않은 커스텀 사용자 필드 정보가 있다면 CSV 파일을 올바르게 가져올 수 없습니다. 커스텀 사용자 필드 추가에 대한 자세한 방법은 [다음 문서](#)를 참고하세요.

* 영어와 한국어를 제외한 다른 언어로 사용자 정보를 입력했다면 CSV 파일을 UTF-8 형식으로 저장하세요.

:::

### CSV 파일로 내보내기

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 사용자 목록에서 내보내려는 사용자의 가장 왼쪽에 체크 박스를 클릭하세요. 모든 사용자를 선택하려면 사용자 목록 왼쪽 상단의 <Cmd>Select All</Cmd> 체크 박스를 클릭하세요.

1. 화면 오른쪽 상단의 더보기(<IcMore/>) 버튼을 선택하고 <Cmd>Export</Cmd> > <Cmd>CSV</Cmd>를 클릭하세요.

사용자의 로컬 드라이브에 CSV 파일이 다운로드됩니다.

### CSV 파일로 불러오기

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 화면 오른쪽 상단의 더보기(<IcMore/>) 버튼을 선택하고 <Cmd>Import</Cmd> > <Cmd>CSV</Cmd>를 클릭하세요.

1. CSV 파일을 선택하고 <Cmd>열기</Cmd> 버튼을 클릭하세요.

1. 불러오기 시작 행을 설정하고 <Cmd sid='user.list.csvImport.next' /> 버튼을 클릭하세요.

1. CSV 파일의 사용자 데이터 필드와 BioStar X의 사용자 데이터 필드가 자동 매핑되어 표시됩니다.

    <Cmd sid='user.list.csvImport.reMapping' /> 버튼을 클릭하면 이름이 동일한 필드까지 재매핑할 수 있습니다.

1. 이미 사용자 아이디가 등록된 사용자 데이터를 유지할지 CSV 파일의 정보로 덮어쓸지 선택하세요.

1. <Cmd sid='user.list.csvImport.next' /> 버튼을 클릭하세요.

:::note

* 불러올 CSV 파일에 새로운 열을 추가하면 사용자 정보를 불러올 수 없습니다.

* CSV 불러오기를 통해 모바일 출입 카드를 발급할 수 있습니다. 레귤러 사이트에서 불러오기를 완료하면 에어팝 포털(Airfob Portal)에서 모바일 출입 카드당 1개의 크레딧이 차감됩니다. 모바일 출입 카드를 발급하지 않으려면 매핑을 해제하세요.

* CSV 파일에 이미 등록된 사용자에게 발급된 모바일 출입 카드와 동일한 데이터가 있다면, 데이터를 유지하거나 덮어쓸 수 있습니다. 데이터를 유지하면 기존 모바일 출입 카드의 데이터가 유지됩니다.

* CSV 파일에 이미 등록된 사용자에게 발급된 모바일 출입 카드와 다른 데이터가 있다면, 데이터를 유지하면 기존 모바일 출입 카드가 유지되고, 덮어쓸 경우 해당 사용자에게 새 모바일 출입 카드를 발급됩니다.

* CSV 불러오기를 통해 모바일 출입 카드를 발급할 때 다이나믹 사이트를 사용한다면, `mobile_start_datetime` 및 `mobile_expiry_datetime` 필드의 값을 반드시 입력해야 합니다.

* **BioStar 2 QR**은 CSV 불러오기를 통해 발급할 수 없습니다.

* CSV 불러오기를 통해 사용자의 비주얼 페이스를 등록할 수 있습니다. 비주얼 페이스 등록에 대한 자세한 내용은 [다음 문서](enroll-credential#enroll-visualface)를 참고하세요.

* CSV 불러오기를 통해 사용자의 PIN을 등록할 수 있습니다. PIN 등록에 대한 자세한 내용은 [다음 문서](enroll-credential#enroll-accesspin)를 참고하세요.

:::

## 데이터 파일 활용

외부 저장매체(USB)에 사용자 정보를 저장해 **BioStar X** 또는 장치로 불러올 수 있습니다. 최대 500,000명의 사용자 정보를 서버에서 장치로 옮기거나 장치에서 장치로 옮길 수 있습니다.

:::note

* 오래된 펌웨어 버전을 사용하는 장치에서 내보내기한 데이터는 **BioStar X**로 불러올 수 없습니다. 항상 최신 버전의 펌웨어를 사용하세요.

* **BioStar X** 최신 버전을 사용한다면 오래된 펌웨어 버전을 사용하는 장치에서 데이터를 불러올 수 없습니다. 장치의 펌웨어를 호환하는 버전으로 업그레이드하세요. 호환하는 장치와 펌웨어 버전은 아래를 참고하세요.

  - BioStation 2 펌웨어 1.9.0 이상

  - BioStation A2 펌웨어 1.8.0 이상

  - FaceStation 2 펌웨어 1.4.0 이상

  - FaceStation F2 펌웨어 2.2.0 이상

  - X-Station 2 펌웨어 1.0.0 이상

  - BioStation 3 펌웨어 1.3.1 이상

  - BioStation 2a 펌웨어 1.0.0 이상

* 서로 다른 지문 템플릿 형식을 사용한다면 데이터를 가져올 수 없습니다. 예를 들어, 슈프리마 지문 템플릿 형식을 사용하는 장치에서 내보내기한 데이터를 ISO 지문 템플릿 형식을 사용하는 장치에서 가져올 수 없습니다.

* FaceStation F2에서 비주얼 페이스가 등록된 사용자 데이터를 가져올 때, **BioStar X** 서버나 모바일 장치를 통해 이미 등록된 비주얼 페이스 데이터가 있다면 기존 데이터가 새 데이터로 대체될 수 있습니다.

:::

### 데이터 파일로 내보내기

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 사용자 목록에서 내보내려는 사용자의 가장 왼쪽에 체크 박스를 클릭하세요. 모든 사용자를 선택하려면 사용자 목록 왼쪽 상단의 <Cmd>Select All</Cmd> 체크 박스를 클릭하세요.

1. 화면 오른쪽 상단의 더보기(<IcMore/>) 버튼을 선택하고 <Cmd>Export</Cmd> > <Cmd>Data File</Cmd>를 클릭하세요.

1. 내보내기 창이 나타나면 데이터 파일을 적용할 장치를 선택하세요.

1. <Cmd sid='button.acknowledge' /> 버튼을 클릭하세요.

사용자의 로컬 드라이브에 데이터 파일(*\*.tgz*)이 다운로드됩니다.

:::note

* 데이터 파일을 불러올 장치는 USB 포트를 지원하는 장치만 선택할 수 있습니다. 

* 내보내기 창에서 선택한 장치와 데이터 파일을 적용할 장치가 일치해야 합니다. 그렇지 않으면 장치에서 데이터 파일을 인식할 수 없습니다.

* 데이터 파일을 내보낼 때, 아래 사용자 정보를 파일 형태로 저장합니다.

  - 프로필 사진

  - ID

  - 이름

  - 유효 기간

  - 출입 그룹별

  - PIN

  - 개인인증모드

  - 크리덴셜(얼굴, 지문, 카드, 모바일 카드, 비주얼 페이스, BioStar 2 QR, QR/Barcode)

  - 1:1 보안 등급

:::

### 데이터 파일로 불러오기

1. <Cmd>Launcher</Cmd> 페이지에서 <Cmd>User</Cmd>를 클릭하세요.

1. 화면 오른쪽 상단의 더보기(<IcMore/>) 버튼을 선택하고 <Cmd>Import</Cmd> > <Cmd>Data File</Cmd>를 클릭하세요.

1. 데이터 파일(*\*.tgz*)을 선택하고 <Cmd>열기</Cmd> 버튼을 클릭하세요.

데이터 파일 불러오기를 성공하면 완료 메시지가 표시됩니다.
