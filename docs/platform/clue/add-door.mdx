:::info

In CLUe, you can configure access doors in advance through the web portal without directly connecting to a device. These settings will be automatically applied to the device once it connects to the server.

:::

## How to Use Slave Device Relay(SIO2, DM-20)

You need to check the necessary conditions for door configuration in advance. It is essential to verify and set up the relay and door sensor input ports beforehand.

:::info

When using an RS485 slave device such as SIO2 and DM-20, it must be assigned to a space like a regular device. Additionally, the slave device must be added to the master device.

:::

Please check the slave device from the device list.

{/* ![image.png](attachment:b449b29e-98a5-4c95-8df3-c9daaebe3132:image.png) */}

Please select the master device and add a connected slave device. Then, you can use a slavce device relay and input ports.

## How to Set a Door

Enter door name and select door relay and exit button and necessary setttings.

![image.png](attachment:5405c593-e75e-4f7f-b193-029991664a23:image.png)

:::info

What is Sensor Ignore Option?

When it’s set to “Use”, door relay switches its status after a lock time regardless of door sensor status. When it’s “Not in Use”, the relay does not switch back when the open door sensor status is detected. This option is necessary to be used to configure a automatic door.

:::

## How to Set a Schedule (Unlock/Lock Time)

In order to set a door schedule, it’s necessary to set weekly time segement first from the schedule menu.

1. Go to Schedule Tap → Click “Pencil” icon to add schedule.

   {/* ![image.png](attachment:2ee4922a-5137-4d9c-8721-89c9aec27144:image.png) */}

1. Enter its name and select weekly day and time range to apply. (Up to three time segments are supported.)

   {/* ![image.png](attachment:80d107e7-ab58-4377-b9b0-cfa18c22c4cd:image.png) */}

:::info
💡

Please note that only "\_" and "-" special characters are supported for the name field input.

:::

1. Click “Apply” and Verify and Edit the time segments and click “Add” to create.

   {/* ![image.png](attachment:c4b1c531-be9b-488f-bf4a-144c91f82bd3:image.png) */}

1. Once it’s created, the schedule can be seen from the list.

   {/* ![image.png](attachment:9ad9ae21-98be-42c7-ac13-dab095b1e667:image.png) */}

1. Go to “Door Settings” and select a door schedule to apply. 

   Unlock time and lock time cannot be overlapped.
