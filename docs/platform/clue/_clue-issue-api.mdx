## API 키 발급

1. [CLUe 웹 포털](https://clue.moon.supremainc.com)에 로그인하세요.

1. API 연동을 위한 공간 그룹을 선택하세요. 

1. 화면 오른쪽 상단의 <IcAirMore title='more' /> → <Cmd sid='settings_2' product='clue' />을 클릭하세요.

   <Image src='/img/clue/clue-place-group-settings.png' />

1. 화면 왼쪽 사이드바에서 <Cmd sid='admin_2' product='clue' />를 클릭하세요.

   <Image src='/img/clue/clue-place-group-settings-admin.png' />

1. <Cmd sid='manage_api_keys' product='clue' /> 섹션에서 <Cmd sid='add_new_api_key' product='clue' />을 클릭하세요.

1. 원하는 유효 기간을 선택하세요.

   <Image src='/img/clue/clue-place-group-settings-admin-new-api.png' />

   :::note

   <Cmd sid='custom' product='clue' />을 선택하면 유효 기간을 최대 365,000일까지 설정할 수 있습니다.

   :::


1. 새 API 키를 발급하려면 <Cmd sid='add_2' product='clue' /> 버튼을 클릭하세요.

1. 새 API 키가 생성되면 <Cmd sid='copy' product='clue' /> 버튼을 클릭해 CLUe API 호출 시 Bearer 토큰으로 사용하세요.

   <Image src='/img/clue/clue-place-group-settings-admin-api-copy.png' />

:::caution

API 키는 CLUe API에 대한 접근 권한을 그대로 부여하므로 외부에 노출되지 않도록 안전하게 보관하세요. 오남용으로 발생하는 문제에 대해 슈프리마는 책임지지 않습니다.

:::
