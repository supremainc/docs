**CLUe**는 국내 최초의 클라우드 출입인증 매니지드 서비스입니다. 웹 서비스 기업이 출입 인증에 필요한 단말기 설치부터 플랫폼 운영까지 신경 쓰지 않고, 자사 서비스 개발에만 집중할 수 있도록 지원합니다.

<Image src='/img/clue/suprema-clue-architecture.png' />

공유오피스, 피트니스, 스터디 카페, 회의실 관리, 방문객 관리, 근태 관리 등 공간 기반 서비스를 운영하는 기업이라면, CLUe를 통해 출입 인증 기능을 손쉽게 도입할 수 있습니다.

## CLUe의 주요 특징

* **간편한 개발**: SDK 방식이 아닌 웹 서비스 간 API 연동 방식을 사용하므로, 별도의 개발 환경 구축 없이 출입 인증 기능을 빠르게 연동할 수 있습니다.

* **간편한 설치**: 별도의 서버가 필요 없는 Plug & Play 방식으로 동작하므로, 복잡한 설치 과정 없이 바로 사용할 수 있습니다.

* **효율적인 유지보수**: 지속적인 업그레이드를 지원하며, 상시 시스템 모니터링을 통해 장애에 신속하게 대응합니다.

* **다양한 인증 수단**: QR 코드, 얼굴 인증, RFID, 지문 등 다양한 인증 수단을 지원합니다.

* **맞춤형 개발**: 고객사의 서비스 환경에 맞춰 커스텀 개발을 지원합니다.

* **데이터 보안**: CSA STAR Level 2 인증을 획득했으며, ISO27001 및 ISO27701 기준을 준수하여 안전하게 데이터를 관리합니다.
