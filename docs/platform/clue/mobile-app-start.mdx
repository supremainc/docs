This guide provides the steps to invite CLUe mobile app admin and the basic use of its features.

## Before Start

### How to Invite CLUe Mobile App admin?

In order to invite CLUe mobile app admin, the place group admin needs to access to the place and invite a place admin with email address.

{/* ![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/e9e3b079-acb5-4a48-9649-a73379adf827/99ddc429-ed22-4379-a9b5-9f52fdc9adfb/image.png) */}

Once it’s invited, the invited admin will receive an invitation email, and he or she needs to sign up through the link provided by the email.

After signup, the place admin status change and it can be found from the admin list.

![image.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/e9e3b079-acb5-4a48-9649-a73379adf827/3ed65eac-01fb-4b89-9b9c-ed9de0aa3f80/image.png)

:::info

Can I sign up directly through CLUe mobile app?

Yes, if the admin user is unable to access to the CLUe web portal, he or she can download CLUe app and sign up with the mobile app. However, it’s always necessary to be invited to a place for actual use.

:::

## Suprema CLUe App

### Where Can I Download CLUe App ?

CLUe APP is available from both iOS and Android markets.

- iOS: [https://apps.apple.com/kr/app/suprema-clue/id1639027796](https://apps.apple.com/kr/app/suprema-clue/id1639027796)

- Android: [https://play.google.com/store/apps/details?id=com.suprema.moon](https://play.google.com/store/apps/details?id=com.suprema.moon)

If you can’t find, you can search by “suprema clue” from the app store.

### How To Start?

Once you install the app, now you can login with the invited email address.

Please select “Login with Email”

Auth code will be sent your email address, it needs to be entered and verified to login. Once it’s logged in, the login status will continue untill logout.

## What Can I Do with CLUe App?

1. Remote Door Control

1. Invite and Manage Users. You can enroll face and fingerprint and RF card by using CLUe mobile app and Suprema devices.
