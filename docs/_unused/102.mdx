# 버전 1.0.2 (빌드 번호 1.0.2.111)

릴리스<Separator/>2026-04-15

## 신규 기능 및 개선

---

1. 커스텀 관리자 계정에 접근 가능한 카메라 그룹을 지정할 수 있는 권한 설정 기능 추가

1. 디렉토리 연동(AD/Entra ID) 동기화 기능 개선

    - 동기화 모드(**Add/Update/Delete**) 변경 시 경고 메시지 명확화

    - 자동 동기화 설정 UI 및 기본값 개선

1. 모니터링 영상 재생 기능 개선

    - Seekbar의 녹화 구간 표시, Knob 이동, 일시정지/재생 동작 안정성 향상

    - **No Signal** 및 **No Recording** 상태 표시 개선

1. 시스템 백업/복원 기능 개선

    - 백업 파일 저장 경로 변경 지원

    - 시스템 백업 관련 **감사 추적** 항목 추가 및 오류 문구 개선

1. 사용자 데이터 내보내기 권한 보안 강화

    - 접근 권한이 없는 사용자/출입 그룹이 **CSV 내보내기** 및 **데이터 파일 내보내기**에 포함되던 문제 개선

    - 일괄 편집(**Batch Edit**)을 통해 권한 없는 사용자 그룹·출입 그룹을 설정할 수 있던 문제 개선

1. MCS(멀티 커뮤니케이션 서버) 환경 개선

    - 커뮤니케이션 서버 추가·삭제 및 상태 표시 안정성 향상

    - 서비스 매니저의 데이터베이스 연결 정보 표시 개선
