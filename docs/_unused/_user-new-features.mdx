## 출입 예외 설정 {#accessException}

사용자가 모든 출입문을 자유롭게 출입하도록 허용하거나, 특정 출입문에 대해서만 출입을 제한하도록 예외를 설정할 수 있습니다. <Cmd sid='team.user.permission.title' /> 섹션의 <Cmd sid='team.user.permission.accessException' /> 항목에서 사용자가 출입할 수 있는 출입문(<Cmd sid='team.user.allow' />)과 출입을 제한할 출입문(<Cmd sid='team.user.deny' />)을 선택하세요.

:::info

추가한 출입문을 삭제하려면 <IcClose/> 버튼을 클릭하세요.

:::


## 연장 시간에 출입 설정 {#extendedDoorOpenTime}

연장된 출입 시간 동안에도 출입을 허용할지 설정할 수 있습니다. 연장 출입 시간에 출입을 허용하려면 <Cmd sid='team.user.permission.title' /> 섹션의 <Cmd sid='team.user.permission.extendedDoorOpenTime' /> 항목을 클릭해 <Cmd sid='common.use' />으로 설정하세요.
