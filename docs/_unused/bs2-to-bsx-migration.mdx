**BioStar 2**에서 **BioStar X**로의 마이그레이션은 시스템의 성능과 보안을 향상시키고, 최신 기능을 활용할 수 있는 기회를 제공합니다. 이 문서에서는 **BioStar 2**에서 **BioStar X**로 마이그레이션할 때의 주요 변경 사항과 고려해야 할 점을 안내합니다.

업그레이드를 통해 여러 기능을 향상시키고 시스템 확장성을 확장시킬 수 있지만, 라이선스, 사용자 제한, 가격 구조 등에 변화가 있습니다. **BioStar 2**와 **BioStar X**의 동등한 라이선스 등급을 비교하고, 차이점에 대한 이해를 통해 고객의 원활한 도입과 운영을 지원합니다.

:::info

* **BioStar X** 라이선스에 대한 자세한 내용은 [다음 문서](bsx-licensing)를 참고하세요.

* **BioStar X**의 라이선스별 가격에 대한 자세한 내용은 [다음 링크](#)를 참고하세요.

:::


## 스타터(Starter) 라이선스 비교

**BioStar 2**에서 **BioStar X**로 전환할 때 기능과 라이선스의 차이를 이해하는 것이 중요합니다. **BioStar X** 스타터 라이선스는 기본 기능만 필요로 하는 사용자를 위해 설계되었으며, 일회성 구매 비용이 발생합니다. **BioStar 2**에서 일부 기능을 무료로 제공했습니다. 아래 표를 통해 스타터 라이선스의 주요 기능의 변화를 확인할 수 있습니다.

<div className='table--licensing overflow-x'>

| 항목 | BioStar 2 | BioStar X | 비고 |
| --- | :--: | :--: | --- |
| **사용자** | 무제한 | 100 | 사용자 수 감소 |
| **출입문** | 5 | 5 | 출입문 수 제한 동일 |
| **구역**<br/>*(Global APB, Global Fire Alarm, Intrusion Alarm, Mustering, Occupancy Limit, Interlock)* | ❌ | ❌ | &nbsp; |
| **엘리베이터** | ❌ | ❌ | &nbsp; |
| **그래픽 맵** | ❌ | ❌ | &nbsp; |
| **서버 매칭** | ❌ | ❌ | &nbsp; |
| **클라우드** | ❌ | ❌ | &nbsp; |
| **Active Directory** | ❌ | ❌ | &nbsp; |
| **기본 비용** | 무료 | 일회성 구매 | &nbsp; |

</div>

주요 차이점은 사용자 수 제한과 일회성 구매 비용입니다. 주요 고급 기능이 필요한 고객은 BioStar X 어드밴스 라이선스로 업그레이드 해야 합니다.


## 베이직(Basic) VS 에센셜(Essential)


**BioStar 2**의 베이직 라이선스와 **BioStar X**의 에센셜 라이선스로 전환할 때 기능면에서 변화는 적지만, 비용 구조와 사용자 수 제한이 변경되었습니다. 두 버전 모두 유사한 기능을 제공하지만 BioStar X의 에센셜 라이선스는 사용자 수 제한이 있으며 비용이 크게 증가합니다. 아래 표를 통해 베이직 라이선스와 에센셜 라이선스의 주요 기능의 변화를 확인할 수 있습니다.

<div className='table--licensing overflow-x'>

| 항목 | BioStar 2 베이직 | BioStar X 에센셜 | 비고 |
| --- | :--: | :--: | --- |
| **사용자** | 무제한 | 1,000 | 사용자 수 감소 |
| **출입문** | 20 | 32 | 출입문 수 제한 증가 |
| **구역**<br/>*(Global APB, Global Fire Alarm, Intrusion Alarm, Mustering, Occupancy Limit, Interlock)* | ❌ | ❌ | &nbsp; |
| **엘리베이터** | ❌ | ❌ | &nbsp; |
| **그래픽 맵** | ❌ | ❌ | &nbsp; |
| **서버 매칭** | ❌ | ❌ | &nbsp; |
| **클라우드** | ❌ | ❌ | &nbsp; |
| **Active Directory** | ❌ | ❌ | &nbsp; |

</div>

:::note

고급 기능이 필요한 고객은 **BioStar X** 어드밴스(Advanced) 라이선스로 업그레이드할 것을 권장합니다.

:::


## 스탠다드(Standard) VS 어드밴스(Advanced)

**BioStar 2**에서는 스탠다드 라이선스가 비용 효율적인 옵션을 제공해 소규모에서 중규모 시스템에 적합한 견고한 세트를 갖추고 있습니다. **BioStar X**의 어드밴스 라이선스는 **BioStar 2**의 스탠다드 라이선스와 동등한 등급으로, 시스템 확장성과 모듈형 기능면에서 크게 확장됩니다. 대부분의 환경에서는 전체 고급 기능 세트가 필요하지 않을 수 있지만 실시간 모니터링, 더 많은 사용자와 출입문 수, 비디오, 서버 매칭, 클라우드 서비스와 같은 기능을 선택할 수 있습니다.

특히 제한된 기능이었던 출입문 잠금 일정 기능을 **BioStar X** 라이선스에 무료로 제공되어 필수 일정 기능에 대한 장벽을 낮췄습니다. 반면에 **BioStar X**의 어드밴스 라이선스는 보다 모듈화된 라이선스 접근 방식을 채택하여 특정 고급 기능을 유료로 추가할 수 있습니다.

<div className='table--licensing overflow-x'>

| 항목 | BioStar 2 스탠다드 | BioStar X 어드밴스 | 비고 |
| --- | :--: | :--: | --- |
| **사용자** | 무제한 | 50,000 | 대규모 환경에 적합한 증가 |
| **출입문** | 50 | 128 | 출입문 수 증가 |
| **구역**<br/>*(Global APB, Global Fire Alarm, Intrusion Alarm, Mustering, Occupancy Limit, Interlock)* | ✔️ | 💲 | **고급 출입 통제** 기능을 추가해 사용 가능 |
| **엘리베이터** | ❌ | 💲 | ^ |
| **그래픽 맵** | ❌ | ✔️ | &nbsp; |
| **서버 매칭** | ❌ | 💲 | 선택 추가 기능으로 제공 |
| **클라우드** | ✔️ | 💲 | \^ |
| **Active Directory** | ❌ | 💲 | \^ |

</div>

**BioStar X** 어드밴스 라이선스는 중간 규모 시스템의 요구 사항을 충족하며, 플랫폼의 확장성과 유연성은 고객 요구 사항에 따라 보다 맞춤화된 솔루션을 제공합니다.


## 어드밴스(Advanced) 라이선스 비교

**BioStar X**의 어드밴스와 **BioStar 2**의 어드밴스는 동등한 등급의 라이선스이지만 주요 기능 및 가격 구조에서 차이가 있습니다. **BioStar X**의 어드밴스 라이선스는 대규모 시스템을 지원하지만, 이전에 번들로 제공했던 여러 기능을 모듈식 라이선스로 분리해 도입합니다. 이러한 변경 사항은 더 큰 유연성을 제공하지만 시스템의 특정 요구 사항에 따라 전체 비용이 달라질 수 있습니다.

**BioStar 2**의 어드밴스 라이선스에서 기본 포함된 구역, 엘리베이터 제어 및 Active Directory 통합과 같은 고급 기능은 **BioStar X**에서는 **고급 출입 통제**(Advanced Access Control System) 모듈 또는 기타 유료 추가 기능으로 제공받을 수 있습니다. 추가 기능이 필요한 조직은 해당 모듈을 구매하거나 엔터프라이즈 라이선스로 업그레이드를 고려해야 합니다.

<div className='table--licensing overflow-x'>

| 항목 | BioStar 2 | BioStar X | 비고 |
| --- | :--: | :--: | --- |
| **사용자** | 무제한 | 50,000 | &nbsp; |
| **출입문** | 100 | 128 | 출입문 수 소폭 증가 |
| **구역**<br/>*(Global APB, Global Fire Alarm, Intrusion Alarm, Mustering, Occupancy Limit, Interlock)* | ✔️ | 💲 | **고급 출입 통제** 기능을 추가해 사용 가능 |
| **엘리베이터** | ✔️ | 💲 | ^ |
| **그래픽 맵** | ✔️ | ✔️ | 기본 패키지에 포함 |
| **서버 매칭** | ✔️ | 💲 | 선택 추가 기능으로 제공 |
| **클라우드** | ✔️ | 💲 | \^ |
| **Active Directory** | ✔️ | 💲 | \^ |

</div>

**BioStar X**의 어드밴스 라이선스는 강력한 기능을 제공하지만 엔터프라이즈 수준의 기능이 필요하지 않은 조직에 적합합니다. 기능 추가에 대한 의존도가 높아짐에 따라 프로젝트 범위 및 가격 책정은 정확한 계획을 필요로 합니다.


## 프로페셔널(Professional) VS 엔터프라이즈(Enterprise)

**BioStar 2**의 프로페셔널 라이선스를 사용 중인 조직은 **BioStar X**의 엔터프라이즈 라이선스로 전환하는 것을 권장합니다. 엔터프라이즈 라이선스는 대규모 시스템에 적합하게 설계되었으며 **고급 출입 통제**(Advanced Access Control) 기능을 기본 제공하며 별도의 추가 기능이 필요하지 않습니다. 기본 비용이 크게 증가하지만, 대규모 환경에 적합한 확장성과 유연성을 제공합니다.

엔터프라이즈 라이선스에서는 최대 **200,000**명의 사용자와 **500**개의 출입문을 지원합니다. 고급 보안 환경 및 다중  사이트 관리, 복잡한 통합 요구가 있는 고객에게 적합합니다. 그러나 서버 매칭, 클라우드 및 Active Directory 통합과 같은 기능은 **BioStar X**의 모듈식 설계 철학의 일환으로 별도 라이선스가 필요합니다.

| **사용자** | 무제한 | 100,000 | &nbsp; |
| **출입문** | 300 | 500 | 기업 규모의 환경에 적합 |
| **구역**<br/>*(Global APB, Global Fire Alarm, Intrusion Alarm, Mustering, Occupancy Limit, Interlock)* | ✔️ | ✔️ | &nbsp; |
| **엘리베이터** | ✔️ | ✔️ | &nbsp; |
| **그래픽 맵** | ✔️ | ✔️ | &nbsp; |
| **서버 매칭** | ✔️ | 💲 | 선택 추가 기능으로 제공 |
| **클라우드** | ✔️ | 💲 | \^ |
| **Active Directory** | ✔️ | 💲 | \^ |

</div>

엔터프라이즈 라이선스는 최대 확장성과 운영 제어가 필요한 대규모 조직에 적합합니다. 가격 인상이 상당하지만 번들로 제공하는 **고급 출입 통제** 기능과 높은 용량 한도를 감안하면 투자 가치가 있습니다. 엔터프라이즈 라이선스는 대규모 고객의 요구를 충족하는지 확인하기 위해 신중한 계획과 평가가 필요합니다.


## 엔터프라이즈(Enterprise) 라이선스 비교

**BioStar 2**의 엔터프라이즈 라이선스를 사용 중인 고객이라면 **BioStar X**의 엔터프라이즈 라이선스로 전환하는 것이 비용 절감의 효과를 제공할 수 있습니다. **BioStar X**의 엔터프라이즈 라이선스는 **BioStar 2**보다 낮은 비용으로 사용할 수 있고, **고급 출입 통제** 기능을 포함하고 있습니다.

다만, 사용자 수는 200,000명으로 제한되지만 출입문 수는 1,000개에서 500개로 감소합니다. 출입문이 많고 밀집된 환경의 고객은 신중하게 검토해야 합니다. 또한 서버 매칭, 클라우드 및 Active Directory 통합과 같은 기능은 별도의 라이선스가 필요합니다.

**BioStar X**의 엔터프라이즈 라이선스는 비용 효율적인 업그레이로 전환할 수 있는 기획를 제공합니다. 500개 이상의 출입문이나 번들로 제공되던 추가 서비스가 필요하지 않은 고객에게 적합합니다. 고객의 환경에 맞는 라이선스 선택을 통해 비용을 절감하고, 최신 기능을 활용할 수 있습니다.


## 엔터프라이즈(Enterprise) VS 엘리트(Elite)

**BioStar 2** 엔터프라이즈 라이선스를 사용하는 대규모 고객 중 더욱 향상된 확장성이 필요하다면 **BioStar X**의 엘리트 라이선스로 전환하는 것을 고려할 수 있습니다. 기본 비용이 크게 증가하지만 최대 500,000명의 사용자와 2,000개의 출입문을 지원합니다. **BioStar 2** 엔터프라이즈 라이선스보다 지원하는 출입문 수가 두 배입니다.

**BioStar X**의 엘리트 라이선스는 **고급 출입 통제** 기능을 기본으로 제공하며, 중앙 집중식, 고급 보안 및 다중 캠퍼스 환경에 적합합니다. 그러나 서버 매칭, 클라우드 및 Active Directory 통합과 같은 기능은 더 이상 번들로 제공하지 않으며 별도의 라이선스가 필요합니다. 이러한 기능은 대규모 환경에서 중요한 역할을 하므로, 엘리트 라이선스를 선택할 때 고려해야 합니다.

| **구역**<br/>*(Global APB, Global Fire Alarm, Intrusion Alarm, Mustering, Occupancy Limit, Interlock)* | ✔️ | ✔️ | 기본 패키지 포함 |
| **엘리베이터** | ✔️ | ✔️ | \^ |
| **그래픽 맵** | ✔️ | ✔️ | \^ |
| **서버 매칭** | ✔️ | 💲 | 선택 추가 기능으로 제공 |

| **Active Directory** | ✔️ | 💲 | \^ |

엘리트 라이선스는 미션 크리티컬(mission-critical) 엔터프라이즈 환경을 위해 설계되었습니다. 높은 처리량, 방대한 사용자 기반 및 복잡한 액세스 제어가 필수입니다. 이 라이선스가 필요한 고객은 **BioStar X**의 엘리트 라이선스를 통해 최신 기능과 확장성을 활용할 수 있습니다. 그러나 가격이 상당히 높기 때문에, 실제로 필요한 기능과 용량을 신중하게 검토해야 합니다.
