## 사용자 템플릿 {#user-template}

<Cmd sid='shell.header.user' /> 페이지의 <Cmd sid='team.sidebar.template' />은 조직 및 부서별 기본 정보를 표준화하여, 신규 사용자 등록 프로세스를 간소화하고 관리 효율을 높일 수 있는 기능입니다.

<Image src='/img/biostarx-ui-user-template.png' className='none' />

<Calloutlists>

1. 검색 입력 필드에 키워드를 입력하면 사이드 바에서 제공하는 메뉴를 검색하여 목록에 필터링할 수 있습니다.

1. 목록에 표시된 사용자 템플릿의 수를 확인할 수 있습니다.

1. 모든 사용자 템플릿을 선택하려면 <Cmd sid='common.selectAll' /> 버튼을 클릭하세요. 선택한 사용자 템플릿의 개수가 버튼의 오른쪽에 표시됩니다.

1. 화면 오른쪽 상단에 제공하는 페이지 이동 버튼을 클릭해 다음 또는 이전 사용자 목록을 확인할 수 있습니다. 또는 원하는 페이지를 지정해 이동할 수도 있습니다. 자세한 내용은 [다음 문서](user-template#select-view-options)를 참고하세요.

1. 키워드를 입력해 사용자 템플릿을 검색할 수 있습니다.

1. <Image src='/img/menus/ico-more-white.svg' ico alone /> 버튼을 클릭하면 <Cmd sid='team.menu.columnLayout' /> 기능을 사용할 수 있습니다. 사용자 템플릿 목록에서 열(column)의 헤더 항목을 변경할 수 있습니다. 자세한 내용은 [다음 문서](columnlayout)를 참고하세요.

1. 새로운 사용자 템플릿을 등록하려면 <Cmd sid='team.user.dialog.newTemplate' /> 버튼을 클릭하세요. 사용자 템플릿을 삭제하려면 <Cmd sid='common.delete' /> 버튼을 클릭하세요. 자세한 내용은 [다음 문서](user-template)를 참고하세요.

</Calloutlists>
