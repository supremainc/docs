Suprema Docs 사이트([https://docs.supremainc.com](https://docs.supremainc.com))는 서비스 제공과 이용 경험 개선을 위해 쿠키를 사용합니다. 이 페이지에서는 사이트에서 사용하는 쿠키의 종류와 목적을 안내합니다.

## 쿠키란

쿠키는 웹사이트를 방문할 때 브라우저에 저장되는 작은 데이터 파일입니다. 사이트는 쿠키를 이용해 사용자의 언어 설정을 기억하거나, 방문 통계를 분석하는 등의 기능을 제공합니다.

## 사용 중인 쿠키 유형

| 구분 | 목적 | 비활성화 가능 여부 |
| --- | --- | --- |
| 필수 쿠키 | 언어 설정, 보안 등 사이트의 기본 기능 제공 | 불가능 |
| 분석 쿠키 | Google Analytics(GA4)를 통한 방문 통계 분석 및 콘텐츠 개선 | 가능 |

최초 방문 시에는 저장된 쿠키가 없으므로, 브라우저에 설정된 언어를 참고해 사이트가 자동으로 해당 언어 페이지로 안내합니다. 이후 언어를 직접 선택하면 그 선택이 쿠키에 저장되어 재방문 시 우선 적용됩니다.

분석 쿠키에 동의하지 않으면 Google Analytics는 방문자의 활동을 수집하지 않습니다.

## 쿠키 설정 변경하기

사이트 하단에 표시되는 쿠키 동의 배너에서 **모두 동의**, **필수만 허용**, **모두 거부** 중 원하는 항목을 선택할 수 있습니다. 선택한 설정은 브라우저에 저장되며, 브라우저의 저장된 데이터(로컬 스토리지)를 삭제하면 다음 방문 시 배너가 다시 표시됩니다.
