## 안내 사항

</Xclude>

* **BioStar 소프트웨어를 통한 PIN 등록 시**

   * 마스터 관리자를 지원하지 않는 글로벌향 (Non-CE) 제품은 4자리 이상의 PIN을 장치에서 직접 등록할 수 있습니다. 단, BioStar X(또는 BioStar 2)를 통해 사용자를 등록하는 경우 아래 버전에서는 최소 8자리 PIN이 필요합니다.

      - BioStar 2 v2.9.11 이상

      - BioStar X v1.0.0 ~ v1.0.2

   * 위 버전에서 4자리 PIN으로 등록된 사용자는 장치 관리자로 지정할 수 없습니다.

* **장치 해시키 수동 변경 시**

   * 장치 해시키를 수동으로 변경할 경우 마스터 관리자의 PIN이 삭제된다는 경고 메시지가 표시되므로 메시지를 확인한 후 변경하세요.

* **RS-485 통신구간 얼굴 이미지 전송 제한**

   * RS-485 통신구간에서는 바이오메트릭 크리덴셜 이미지가 전송되지 않으므로, 슬레이브 장치에서 등록된 마스터 관리자 얼굴 조회 시 대체 이미지를 표시합니다.
