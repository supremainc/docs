## BioStar 2.1.0

### 새로운 기능 및 향상된 기능

* BioStar 2 API 지원

* DM-20, Wiegand 리더 설정 지원

* 글로벌 화재 경보 구역, 로컬 화재 경보 구역, 로컬 안티패스백 구역 지원

* 로그인 암호 조합 방법 변경

* 라이선스 정책 적용

* BioStar 2 클라우드 지원

* BioStar 2 모바일 지원

* 구역 상태 모니터링 지원
