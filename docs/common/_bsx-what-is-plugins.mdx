## 플러그인(Plugins)이란?

**BioStar X** 플러그인(Plugins)은 **BioStar X** 플랫폼의 확장 기능으로, 고객의 특별한 요구사항이나 기존 시스템과의 연동을 위해 추가 기능을 제공합니다.

:::info

플러그인 기능을 사용하려면 플러그인 라이선스가 필요합니다. 라이선스 정책에 대한 자세한 내용은 [다음 문서](../biostar_x/licensing)를 참고하세요.

:::

</Xclude>
