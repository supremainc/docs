## BioStar 2.2.1

### 새로운 기능 및 향상된 기능

* BioStation L2 지원

* 장기 미출입 사용자 관리 지원

* 사용자별 출입 권한 상태 관리 지원

* 데이터베이스 백업 지원

* 사용자 정보 입력 필드 커스텀 지원
