#### LAN 연결 (PC와 직접 연결)

<Include product='bioentry_w3,facestation_f2,bioentry_w2,biolite_n2,biostation_2a,biostation_3,facestation_2,xstation_2'>

이 제품은 자동 MDI/MDIX 기능을 가지고 있으므로 크로스 케이블이나 일반 스트레이트 타입의 CAT-5 케이블로 PC와 직접 연결할 수 있습니다.

</Include>

<Include product='xpass_s2'>

이 제품은 자동 MDI/MDIX 기능을 가지고 있으므로 일반 타입의 CAT-5 케이블로 PC와 직접 연결할 수 있습니다.

</Include>

<Include product='bioentry_p2,xpass_2,corestation_40,corestation_20'>

이 제품은 자동 MDI/MDIX 기능을 가지고 있으므로 일반 타입의 CAT-5e 케이블로 PC와 직접 연결할 수 있습니다.

</Include>

<Include product='corestation_20'>

<Badge only>CS-20P</Badge> 이더넷 케이블을 통해 전원을 공급하는 PoE+를 지원합니다. PoE+ 사용 시에는 Lock 연결을 하나만 지원합니다.

</Include>

<Include product='xpass_q2'>

* 이 제품은 자동 MDI/MDIX 기능을 가지고 있으므로 크로스 케이블이나 일반 스트레이트 타입의 CAT-5e 케이블로 PC와 직접 연결할 수 있습니다.

  :::caution

  일부 비표준 허브/스위치에서는 자동 MDI/MDIX 기능이 지원되지 않을 수 있습니다.

  :::

* <Badge only>XPQ2-APB</Badge> 이더넷 케이블을 통해 전원을 공급하는 PoE를 지원합니다.

</Include>
