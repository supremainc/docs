## 마스터 관리자 설정하기

<Cmd sid='ui_128' product='dev' /> 기능은 장치의 전체 관리자 권한을 강화하여, 장치의 보안을 향상시키고 무단 접근 및 설정 변경을 방지하기 위한 기능입니다. 새 장치에는 이 기능을 통해 반드시 **마스터 관리자를 등록**해야 하며, 등록된 마스터 관리자만 관리자 메뉴에 진입하여 장치 설정을 변경할 수 있습니다.

:::note

**마스터 관리자란?**

* 장치에 관리자를 설정하지 않고 사용하는 경우, 누구나 관리자 메뉴에 진입하여 장치 설정을 변경할 수 있습니다. 마스터 관리자 기능은 이러한 보안 취약점을 해결하기 위해 전체 관리자 설정을 강제하는 기능입니다.

* 마스터 관리자는 직접 삭제할 수 없으며 <Cmd sid='ui_072' product='dev' /> 시에만 삭제됩니다.

:::

### 새 장치에서 마스터 관리자 등록하기

새 장치에서는 처음부터 <Cmd sid='ui_128' product='dev' />를 필수적으로 등록해야하며, 등록하지 않을 경우 장치 사용이 제한됩니다.

:::info

* 새 장치에서는 최초 부팅 시 <Cmd sid='ui_128' product='dev' /> 등록 화면이 나타납니다. <Cmd sid='ui_128' product='dev' />를 등록하지 않으면 다음 화면으로 이동할 수 없습니다.

* BioStar 2와 연결하여 장치 전체 관리자를 할당하더라도 <Cmd sid='ui_128' product='dev' />를 등록해야만 모든 기능을 사용할 수 있습니다.

* 새 장치는 펌웨어 다운그레이드를 할 수 없습니다.

:::

1. 새 장치를 최초 부팅하면 <Cmd sid='ui_128' product='dev' /> 등록 화면이 나타납니다.

### 안내 사항

* **장치 해시키 수동 변경 시**

  장치 해시키를 수동으로 변경할 경우 마스터 관리자의 PIN이 삭제된다는 경고 메시지가 표시되므로 메시지를 확인한 후 변경하세요.

* **RS-485 통신구간 얼굴 이미지 전송 제한**

  RS-485 통신구간에서는 바이오메트릭 크리덴셜 이미지가 전송되지 않으므로, 슬레이브 장치에서 등록된 마스터 관리자 얼굴 조회 시 대체 이미지를 표시합니다.
