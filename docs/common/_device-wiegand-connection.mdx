Wiegand 장치를 연결하여 사용할 경우 제품과 12 V, 3.1 A 전원을 연결하세요.

<Image src='/img/device/installation-wiegand-in-connection-dm20.png' width='65%' className='none' />

:::info

제품에 Wiegand 장치를 연결하면 12 V, 최대 1.5 A의 전원을 사용할 수 있습니다. **WIEGAND 0**에 1.0 A가 필요한 장치를 연결하면 **WIEGAND 1**에서는 500 mA만 사용할 수 있습니다.

:::
