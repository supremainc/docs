# 애플리케이션 버전 1.2.1

릴리즈<Separator/>2025-05-19

:::caution

Device Manager 1.1.8 버전부터 Android 5.0 운영 체제 이상, iOS 9.0 이상에서 모두 지원됩니다.

:::

## 신규 기능 및 개선 (공통)

---

1. 신규 장치 지원

   * XPass 2 (XP2-MAPB-H)

1. 스마트 카드 레이아웃 설정에서 MIFARE Plus 스마트 카드(슈프리마/커스텀)의 **암호화 종류**(Crypto-1, AES) 선택 옵션 추가

## 버그 수정 (iOS)

---

<BugLists>

1. 애플리케이션에서 장치 재시작 시 ‘장치 재시작’ 팝업이 나타나도록 개선 <Badge>1.1.0</Badge>

1. Wiegand 포맷 생성 시 **종료 비트**를 **시작 비트**보다 낮게 설정하면 애플리케이션이 멈추는 문제 <Badge>1.1.0</Badge>

1. **커스텀 스마트 카드 레이아웃** 설정에서 **바이트 건너뛰기** 옵션이 255까지 설정되는 문제 <Badge>1.1.8</Badge>

1. 스마트 카드 레이아웃 설정에서 **DESFire 고급 설정**의 **파일 읽기 접속 키 인덱스** 옵션이 31까지 설정되는 문제 <Badge>1.1.0</Badge>

</BugLists>
