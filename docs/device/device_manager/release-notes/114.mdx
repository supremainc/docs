# 애플리케이션 버전 1.1.4

릴리즈<Separator/>2022-02-09

:::caution

Device Manager v1.1.4에서 개선된 사항은 Android 5.0 Lollipop 운영 체제 이상에서 지원되며, iOS에서는 추후에 지원될 예정입니다. Device Manager v1.1.3 이하 버전은 Android 5.0 Lollipop 운영 체제 이상, iOS 9.0 이상에서 모두 지원됩니다.

:::

## 신규 기능 및 개선

---

1. XPass D2 장치 설정 옵션 추가

   * OSDP ID 설정 범위를 0~126으로 제한

   * 카드 종류별 설정 옵션 지원

   * CSN 카드, Smart 카드에 바이트 순서 설정 지원

   * Wiegand 카드 기본 포맷 추가

   * DESFire 고급 설정 지원

   * DESFire 암호화 종류 설정 지원

   * 보안 탬퍼 설정 지원

   * 장치 검색 시 모델명 하단에 장치 ID를 표시하도록 개선

   * 펌웨어 업그레이드 시 펌웨어 파일 전송 속도 개선

1. XPass D2 키패드 (XPD2-GKDB) 장치에 키패드 설정 옵션 추가

   * 키패드 사용 여부 옵션을 추가: 키패드를 사용하여 카드 ID를 입력할 수 있도록 지원

   * 장치가 Wiegand 리더이고, 키패드로 카드 ID를 입력할 때 데이터 전달 방식을 싱글 키 또는 전체 키 중 선택할 수 있도록 지원

   * 인증 모드 동작에 따른 장치 LED 동작 및 앱 UI 추가

1. XPass 2 FW 업그레이드 UI 추가

   :::info
   
   Suprema Device Manager를 이용한 펌웨어 업그레이드는 XPass 2의 추후 버전 펌웨어에서 지원 예정입니다.

   :::
