# 애플리케이션 버전 1.1.7

:::caution

Device Manager v1.1.7에서 개선된 사항은 Android 5.0 Lollipop 운영 체제 이상에서 지원되며, iOS에서는 추후에 지원될 예정입니다. Device Manager v1.1.3 이하 버전은 Android 5.0 Lollipop 운영 체제 이상, iOS 9.0 이상에서 모두 지원됩니다.

:::
