# 펌웨어 버전 1.7.3 (빌드 번호 1.7.3_260610)

릴리즈<Separator/>2026-06-12

## 신규 기능 및 개선

---

1. CoreStation Setup Manager 기능 개선

   - SCB 키 수동 입력 설정 지원

   - 네트워크 정보(Device ID, SCB 키 등) 파일 백업 및 복원 기능 지원

   - CoreStation 교체 시 기존 설정을 이전할 수 있도록 개선

   - 전반적인 폰트 크기 및 UI 개선

1. 슬레이브 장치의 전송 속도를 설정할 수 있는 기능 지원

1. **보안 탬퍼** 기능을 지원하는 장치를 구분하도록 개선

1. **장치 해시키 수동 관리** 사용 여부와 상관없이 **보안 탬퍼** 기능을 사용할 수 있도록 개선
