# 펌웨어 버전 1.7.1 (빌드 번호 1.7.1_240130)

릴리즈<Separator/>2024-02-06

## 신규 기능 및 개선

---

1. **무선 도어락** 지원

1. BioStar 2에서 다수의 사용자를 장치에 동기화할 때 사용자 등록에 실패해도 동기화가 완료되도록 개선

1. 사용자 정보 업데이트 시 원하는 정보만 선택하여 업데이트할 수 있도록 개선 (BioStar 2.9.0 버전 이상에서 호환)

1. 신규 장치 연결 지원

    - BioStation 2a

## 버그 수정

---

<BugLists>

1. **서버 매칭**을 사용하는 경우 슬레이브로 연결된 FaceStation 2에서 **카드 + 얼굴** 또는 **ID + 얼굴** 인증에 실패하는 문제 <Badge>1.4.0</Badge>

1. 슬레이브로 연결된 장치에서 리소스 파일이 업데이트되지 않는 문제 <Badge>1.5.2</Badge>

1. 슬레이브로 연결된 장치에 **ID + 얼굴**로 인증한 후 **ID** 아이콘을 터치했을 때 **ID 입력** 화면으로 진입되지 않는 문제 <Badge>1.0.0</Badge>

1. 마스터 장치에서 **보안 탬퍼**가 발생한 경우 잠기도록 설정된 슬레이브 장치가 잠기지 않는 문제 <Badge>1.4.0</Badge>

1. 연결된 IM-120의 설정이 적용되지 않는 문제 <Badge>1.6.1</Badge>

1. 장치가 슬레이브로 연결된 상태에서 DESFire AoC로 인증할 경우 인증이 실패하는 문제 <Badge>1.0.0</Badge>

1. BioStar 2에서 장치의 **네트워크** 설정을 변경하면 장치가 재시작되는 문제 <Badge>1.0.0</Badge>

1. 지원하지 않는 설정을 SDK를 통해 설정한 경우 장치의 부팅이 정상적으로 완료되지 않는 문제 <Badge>1.0.0</Badge>

</BugLists>
