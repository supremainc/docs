# 펌웨어 버전 1.4.1 (빌드 번호 1.4.1_200902)

릴리즈<Separator/>2020-09-25

## 신규 기능 및 개선

---

1. 모바일 출입카드와 일반 RFID 카드의 이벤트 로그 분리

1. CoreStation SETUP Manager 사용자 인터페이스에서 Device ID 변경 관련 내용 삭제

    - CoreStation SETUP Manager를 통한 Device ID 변경 미지원

## 버그 수정

---

<BugLists>

1. 슬레이브 장치 추가 시 슬레이브 장치의 연결이 끊어지는 문제

1. Hash 키가 암호화되지 않은 상태로 저장되는 문제

1. 버전 1.3.1에서 1.4.0으로 펌웨어 업그레이드 중 보안 탬퍼가 발생할 때 오류가 발생하는 문제

1. 고정 IP를 사용하는 장치에 DNS 서버 주소가 설정되어 있을 때 장치를 재부팅하면 서버와의 통신에 오류가 발생하는 문제

</BugLists>
