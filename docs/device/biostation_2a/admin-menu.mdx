관리자 권한으로 접근 가능한 주요 메뉴와 기능을 안내합니다.

## 전체 메뉴

1. <Image src='/img/common/ico-menu1.svg' ico alone />을 누른 뒤 관리자 권한 인증 수단으로 인증하세요.

1. 원하는 메뉴를 선택하세요.

    <Image src='/img/device/bs2a-all-menu.png' />

:::info

관리자가 지정되어 있지 않을 경우 <Image src='/img/common/ico-menu1.svg' ico alone />을 누르면 메뉴 화면이 나타납니다.

:::

## 퀵 메뉴

1. <Image src='/img/common/ico-menu1.svg' ico alone />을 1초 이상 길게 누른 뒤 손을 떼세요. 관리자 권한 인증 수단으로 인증하세요.

1. 원하는 메뉴를 선택하세요.

    <Image src='/img/device/bs2a-quick-menu.png' />

:::info

관리자가 지정되어 있지 않을 경우 <Image src='/img/common/ico-menu1.svg' ico alone />을 길게 누른 뒤 손을 떼면 퀵 메뉴 화면이 나타납니다.

:::

## 키 메뉴

**OK** 버튼을 누르면 키 메뉴 화면이 나타납니다. <Cmd sid='ui_334' product='dev' /> 또는 홈 화면에서 원하는 메뉴에 해당하는 방향 버튼을 눌러 진입할 수 있습니다.

<Image src='/img/device/bs2a-key-menu.png' />

* <IcFw /> <Cmd sid='ui_020' product='dev' />: <Cmd sid='ui_023' product='dev' />에서 설정한 대로 ID 인증을 할 수 있습니다.

* <IcDown/> <Cmd sid='ui_083_title' product='dev' />: 근태 이벤트를 선택해 인증할 수 있습니다.

* <IcBw /> <Cmd sid='ui_279' product='dev' />: 사용자의 작업 코드를 선택할 수 있습니다.

* <IcUp/> <Cmd sid='ui_069' product='dev' />: 경비를 시작/해제할 수 있습니다.

:::info

<Cmd sid='ui_069' product='dev' /> 상태일 경우 <Cmd sid='ui_334' product='dev' />에 진입할 수 없습니다. 홈 화면에서 <IcUp/> 버튼을 눌러 경비를 해제할 수 있습니다.

:::
