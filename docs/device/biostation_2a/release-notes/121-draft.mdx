# 펌웨어 버전 1.2.1 (빌드 번호 1.2.1_260406)

릴리즈<Separator/>2026-04-17

:::caution

- 펌웨어를 v1.2.0 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

## 신규 기능 및 개선

---

<Jira num='BS2AFW-275' /> [BDP-805] Intelligent Slave 사용 시 CP 로부터 전송받은 인증 결과(osdp_LED)를 장치 화면에 표시하기

1. **인텔리전트 슬레이브**를 사용할 경우 타사 컨트롤러의 인증 성공/실패 결과를 장치 화면에 표시하는 **컨트롤러 결과 표시** 기능 지원

1. Supports the **Display Result from Controller** feature for displaying authentication success or failure results from a 3rd party controller on the device screen when using **Intelligent Slave**.

<Jira num='BS2AFW-582' /> [BDP-805 관련] osdp_LED 커맨드로 정의되지 않은 값이 내려올 경우 무시하도록 수정필요

1. 컨트롤러로부터 정의되지 않은 LED 제어 명령을 수신할 경우 장치에서 이를 무시하도록 개선

2. Improved the device to ignore undefined LED control commands received from a controller.

<Jira num='BS2AFW-388' /> [BDP-910] 장치 인증 결과에 ID, 이름을 가리거나 표시하지 않는 기능

1. 장치의 인증 결과 화면에서 사용자 ID, 이름이 표시되는 방법을 선택할 수 있는 **인증 결과 표시 옵션** 기능 지원

   - **사용자 ID 표시**

   - **사용자 이름 표시**

1. Supports the **Auth Result Display Option** that allows you to select how the user ID and name are displayed on the authentication result screen of the device.

   - **User ID Display**

   - **User Name Display**

<Jira num='BS2AFW-589' /> [BDP-1343] Desfire EV2 and EV3 features supported by Readers and Software

1. DESFire EV2/EV3 카드에 대한 EV2 Secure Messaging 인증 및 발급 지원

1. Supported EV2 Secure Messaging authentication and issuance for DESFire EV2 and EV3 cards.

<Jira num='BS2AFW-284' /> [BDP-856] Wiegand Out 후 일정 시간 내 반복적인 인풋 신호를 무시하기

1. 장치를 타사 컨트롤러와 Wiegand로 연결했을 때 컨트롤러로부터 반복적으로 입력되는 인증에 대한 신호를 무시하는 **반복 신호 무시 시간** 기능 지원

1. Supports the **Ignore Repeated Signals Duration** feature, which ignores repetitive authentication signals from the controller when the device is connected to a 3rd party controller via Wiegand.

<Jira num='BS2AFW-423' /> [BDP-788] [BDP-1066] Wiegand Out 후 일정 시간 내 반복적인 인풋 신호를 무시할 때 이벤트도 남기지 않기

1. **반복 신호 무시 시간** 기능이 작동할 때 이벤트를 기록하지 않도록 지원

1. Supports not logging events when the **Ignore Repeated Signals Duration** feature is activated.

<Jira num='BS2AFW-586' /> 장치에서 지원하지 않는 이벤트에 대해 "Undefined Event"로 로그가 남도록 개선

1. 장치에서 지원하지 않는 이벤트 및 에러에 대한 로그 표시 개선

1. Improved log display for unsupported events and errors on the device.

<Jira num='BS2AFW-587' /> CE Red 관련 개선

1. CE(유럽) 외 지역용 **글로벌** 펌웨어 제공

   - **마스터 관리자** 및 **관리자 2단계 인증** 기능 비활성화

   - PIN 등록 시 최소 길이 4자리로 완화

1. Provided **global** firmware for non-CE(EU) regions

   - Disabled the **Master Admin** and **Admin 2-step Authentication** features

   - Reduced the minimum length for PIN enrollment to 4 digits
