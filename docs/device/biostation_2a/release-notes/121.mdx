# 펌웨어 버전 1.2.1 (빌드 번호 1.2.1_260416)

릴리즈<Separator/>2026-04-17

:::caution

- 펌웨어를 v1.2.0 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

- 장치의 펌웨어 다운그레이드 가능 여부 및 신규 BLE 칩 적용 여부는 장치의 시리얼 번호를 확인한 뒤 [슈프리마 홈페이지](https://supremainc.com)에 문의하세요.

## 신규 기능 및 개선

---

1. **인텔리전트 슬레이브**를 사용할 경우 타사 컨트롤러의 인증 성공/실패 결과를 장치 화면에 표시하는 **컨트롤러 결과 표시** 기능 지원

1. 컨트롤러로부터 정의되지 않은 LED 제어 명령을 수신할 경우 장치에서 이를 무시하도록 개선

1. 장치의 인증 결과 화면에서 사용자 ID, 이름이 표시되는 방법을 선택할 수 있는 **인증 결과 표시 옵션** 기능 지원

   - **사용자 ID 표시**

   - **사용자 이름 표시**

1. DESFire EV2/EV3 카드에 대한 EV2 Secure Messaging 인증 및 발급 지원

1. 장치를 타사 컨트롤러와 Wiegand로 연결했을 때 컨트롤러로부터 반복적으로 입력되는 인증에 대한 신호를 무시하는 **반복 신호 무시 시간** 기능 지원

1. **반복 신호 무시 시간** 기능이 작동할 때 이벤트를 기록하지 않도록 지원

1. 장치에서 지원하지 않는 이벤트 및 에러에 대한 로그 표시 개선

1. CE(유럽) 외 지역용 **글로벌** 펌웨어 제공

   - **마스터 관리자** 및 **관리자 2단계 인증** 기능 비활성화

   - PIN 등록 시 최소 길이 4자리로 완화
