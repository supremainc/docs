# 펌웨어 버전 1.1.1 (빌드 번호 1.1.1_240319)

릴리즈<Separator/>2024-03-20

## 신규 기능 및 개선

---

1. **사용자 검색** 시 등록되지 않은 정보로 검색할 경우 표시되는 팝업 메시지 개선

1. **시간 제한 안티패스백** 발생 시 표시되는 팝업 메시지 개선

1. 중복 지문 검출 성능 개선

1. 지문 캡처 품질 개선

## 버그 수정

---

<BugLists>

1. 사용자에 **협박 지문**을 포함한 2개 이하의 지문이 저장되어 있을 때 **협박 지문**을 삭제할 수 없는 문제 <Badge>1.0.0</Badge>

1. 장치에 연결된 USB에 펌웨어 파일이 없을 때 **펌웨어 업그레이드**의 **파일 목록**에서 방향/OK 버튼을 연속으로 누르면 장치가 재시작되는 문제 <Badge>1.0.0</Badge>

1. 무선 네트워크를 사용하는 환경에서 무선 네트워크 연결이 반복적으로 끊어질 경우 RF카드 인증에 실패하는 문제 <Badge>1.0.0</Badge>

1. 32자로 저장된 사용자 ID가 특정 이벤트 로그에서 31자까지만 표기되는 문제 <Badge>1.0.0</Badge>

1. 사용자 ID, 이름이 포함된 인증 메시지가 일정 글자 수를 넘어가면 장치가 비정상적으로 종료되는 문제 <Badge>1.0.0</Badge>

1. 장치에 저장된 지문 수가 1:N 최대치를 넘겨 추가되었을 경우 **사용자 등록현황**의 **지문 수**가 비정상적으로 표시되는 문제 <Badge>1.0.0</Badge>

1. SDK에서 BS2_GetDeviceCapabilities 함수를 이용해 장치에서 지원하는 정보를 불러올 때 커스텀 스마트 카드를 지원하는 장치임에도 CustomClassicPlus와 CustomDesFireEV1 항목이 False로 표시되는 문제 <Badge>1.1.0</Badge>

1. 장치 메뉴에서 **릴레이**를 설정한 경우 모바일 카드 + 지문으로 인증할 때 출입문이 열리지 않는 문제 <Badge>1.0.0</Badge>

</BugLists>
