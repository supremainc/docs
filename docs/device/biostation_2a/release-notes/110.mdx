# 펌웨어 버전 1.1.0 (빌드 번호 1.1.0_240104)

릴리즈<Separator/>2024-01-04

## 신규 기능 및 개선

---

1. **커스텀 스마트 카드 레이아웃** 지원

1. Open SSL 3.0.10 적용

    - 펌웨어를 v1.1.0 이상으로 업그레이드 한 뒤 하위 버전으로 다운그레이드 할 수 없습니다.

1. 사용자 유효기간을 최대 '**2037-12-31**'까지 설정할 수 있도록 개선

1. **시간 제한 안티패스백** 발생 시 팝업 메시지가 표시되도록 개선

1. **키 메뉴** 화면 UI 개선

1. 지문 알고리즘 개선

## 버그 수정

---

<BugLists>

1. 장치 메뉴에서 **릴레이**를 설정한 경우 모바일 카드로 출입문이 열리지 않는 문제 <Badge>1.0.0</Badge>

1. BioStar 2에서 장치의 네트워크 설정을 변경하면 장치가 재시작되는 문제 <Badge>1.0.0</Badge>

1. **무선랜**에서 수동으로 SSID를 입력하여 무선 네트워크에 연결할 수 없는 문제 <Badge>1.0.0</Badge>

1. 장치가 슬레이브로 연결된 상태에서 DESFire AoC로 인증할 경우 인증이 실패하는 문제 <Badge>1.0.0</Badge>

1. 슬레이브로 연결된 장치에서 ID + PIN으로 인증할 경우 사용자 ID만 표시되고 이름은 출력되지 않는 문제 <Badge>1.0.0</Badge>

1. BioStar 2에서 사용자의 iClass AoC를 포맷한 후 재발급할 때 장치가 재시작되는 문제 <Badge>1.0.0</Badge>

1. OK 버튼을 연속으로 누른 후 방향키를 누르면 신호음이 출력되면서 인증을 할 수 없는 문제 <Badge>1.0.0</Badge>

</BugLists>
