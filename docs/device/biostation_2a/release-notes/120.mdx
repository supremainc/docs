# 펌웨어 버전 1.2.0 (빌드 번호 1.2.0_251014)

릴리즈<Separator/>2025-10-21

:::caution

- 펌웨어를 v1.2.0 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

- 장치의 펌웨어 다운그레이드 가능 여부 및 신규 BLE 칩 적용 여부는 장치의 시리얼 번호를 확인한 뒤 [슈프리마 홈페이지](https://supremainc.com)에 문의하세요.

:::

## 신규 기능 및 개선

---

1. 전체 관리자 권한을 갖는 **마스터 관리자** 설정 기능 지원

1. 글로벌 사이버 보안 표준 및 최신 보안 요구사항 반영

1. 최신 카드를 인식할 수 있도록 구분 코드 추가

    - DESFire EV3

1. 서버와 장치 간 시간 동기화 관련 동작 개선

1. 사용자 이름 또는 ID 입력 시 키패드 UI 개선

## 버그 수정

---

<BugLists>

1. **장치 → 서버** 모드를 사용 중일 때 DHCP 서버에 연결이 실패할 경우 서버 IP 주소가 초기화되는 문제 <Badge>1.0.0</Badge>

1. 카드가 2개 이상 등록되어있는 사용자가 Wiegand 출력 장치에 카드 인증 시 인증한 카드가 아닌 다른 카드의 ID가 출력되는 문제 <Badge>1.0.0</Badge>

1. 슬레이브로 연결된 DM-20의 특정 Supervised Input 포트에서 경비 시작/해제 시 비정상적으로 동작하는 문제 <Badge>1.0.0</Badge>

1. **장치 정보** 메뉴에 진입할 때마다 **무선 MAC** 주소가 다르게 나타나는 문제 <Badge>1.1.0</Badge>

1. Wiegand 포맷 25비트 이하의 카드 데이터를 인식하지 못하는 문제 <Badge>1.0.0</Badge>

1. 타사 컨트롤 패널에 다수의 장치가 연결된 상태에서 특정 장치를 재시작하면 다른 장치들의 RS-485 연결이 끊기는 문제 <Badge>1.1.0</Badge>

1. 펌웨어 업그레이드 후 기존에 사용 중이던 카드가 인증에 실패하는 문제 <Badge>1.1.0</Badge>

</BugLists>
