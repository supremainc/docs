# 펌웨어 버전 1.4.0 (빌드 번호 1.4.0_210617)

릴리즈<Separator/>2021-06-25

## 신규 기능 및 개선

---

1. 기본 해시 키를 설정해도 보안 탬퍼를 수동으로 켜거나 끌 수 있도록 개선

1. Open SSL 1.1.1i 버전 적용

1. 모바일 출입카드와 일반 RFID 카드의 이벤트 로그 분리

## 버그 수정

---

<BugLists>

1. 스케줄 개방 구역에 구성된 슬레이브 장치의 연결이 끊어졌다가 재연결될 경우 릴레이 상태가 끊기기 전 상태로 유지되지 않는 문제 <Badge>1.3.2 이하</Badge>

1. 이중 인증이 설정된 슬레이브 장치에서 두 번째 사용자 인증 성공 시 화면이 비정상적으로 표시되는 문제 <Badge>1.4.0 이하</Badge>

1. BioStar 2와의 연결이 끊어진 뒤 다시 연결될 때 상황에 맞지 않는 팝업 메시지가 표시되는 문제 <Badge>1.3.2 이하</Badge>

1. 버전 1.2.0 펌웨어가 설치된 슬레이브 장치를 마스터로 변경한 뒤 펌웨어를 버전 1.3.2로 업그레이드할 경우 장치가 비정상적으로 재부팅되는 문제 <Badge>1.3.2 이하</Badge>

1. 엘리베이터에 스케줄 개방 구역을 설정한 뒤 마스터 장치를 재부팅할 경우 릴레이가 Off(Lock)로 동작하는 문제 <Badge>1.3.2 이하</Badge>

1. 슬레이브로 연결된 장치가 비정상적으로 재부팅되는 문제 <Badge>1.3.2 이하</Badge>

1. 장치를 Secure I/O 2와 연결하여 사용할 때 연결 끊김 로그가 빈번하게 발생하는 문제 <Badge>1.3.2 이하</Badge>

1. 장치를 DM-20과 연결하여 사용할 때 DM-20에 연결된 Wiegand 리더를 Lock 설정한 뒤 장치를 재부팅하면 Wiegand 리더가 Unlock으로 동작하는 문제 <Badge>1.3.2 이하</Badge>

1. 출입문 수동 개방 상태에서 장치를 재부팅하면 출입문이 잠긴 채로 열리지 않는 문제 <Badge>1.3.2 이하</Badge>

1. 장치 관리자를 1,000명 등록한 상태에서 초기화를 진행해도 10명의 장치 관리자가 삭제되지 않고 유지되는 문제<Badge>1.3.2 이하</Badge>

1. 서버 URL 항목에 입력 가능해야 하는 특수 기호(., ?, /, *)를 입력할 수 없는 문제 <Badge>1.3.2 이하</Badge>

1. 장치 펌웨어 업그레이드 후 키 입력에 반응이 없거나 사용자 인터페이스 동작이 느려지는 문제 <Badge>1.3.2 이하</Badge>

1. AoC에 발급한 사용자 정보 중 지문을 신규로 등록하였을 때 기존 AoC에 발급한 지문으로 인증에 성공하는 문제 <Badge>1.3.2 이하</Badge>

1. Wiegand 리더에 대해 Card Only 이외의 인증 모드도 설정이 가능한 문제 <Badge>1.3.2 이하</Badge>

</BugLists>
