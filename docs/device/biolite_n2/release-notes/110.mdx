# 펌웨어 버전 1.1.0 (빌드 번호 1.1.0_181210)

릴리즈<Separator/>2018-12-11

## 주요 버그 수정

---

1. Micom 초기화 시 출력이 이전 상태로 복구되지 않고, 카드를 인식하지 못하는 문제

1. AES 암호화 옵션으로 발급한 DESFire EV1 카드를 CSN 카드로 인식하는 문제

1. 블랙 리스트 카드로 인증할 때 Wiegand Out 출력되지 않는 문제

1. 항시 통과를 활성화한 상태에서 미등록 ID 인증 시 인증 실패 메시지가 표시되지 않는 문제

1. DESFire 카드를 발급할 때 생성된 App과 File 중 File을 신규로 다시 발급할 수 없는 문제

1. 일부 표준 시간대가 누락되어 장치가 비정상적으로 재부팅되는 문제

1. 카드 인증 시 캐시 메모리가 깨져 인증에 실패하는 것을 방지하기 위한 코드 적용

1. 장치를 재부팅한 뒤 지문을 인증하면 센서가 동작하지 않는 문제

## 신규 기능 및 개선

---

1. DESFire 카드에 AES 암호화 옵션 지원

1. DESFire/DESFire EV1 고급 설정 지원

1. 출입 등급 및 출입 그룹을 2048개까지 생성할 수 있도록 지원

1. 사용자를 등록, 수정, 삭제한 경우 이벤트 로그에 해당 동작이 이루어진 서버 또는 장치가 표시되도록 개선

1. OSDP 통신 시 데이터 전송에 실패하면 다시 전송하도록 개선

1. 데이터 암호화 보안 수준 향상

    - 데이터 암호화 대상 확대

    - 개인 정보 저장 기간 설정 지원

    - 보안 탬퍼에 추가 기능 지원: 보안 탬퍼 발생 시 사용자, 로그, 데이터 암호화 키, SSL 인증서, 스마트 카드 레이아웃 삭제

1. Wiegand 입력에 대한 너비 최대 값 변경

1. 장치의 사용자 관리에 사용자, 지문, 얼굴, 카드 개수 표시 지원

1. 개인별 인증 성공 메시지 및 경고 시간 근무 알림 지원

1. 항시 통과를 사용할 때 AoC로 인증해도 카드 ID를 Wiegand 로 출력하도록 개선

## 버그 수정

---

<BugLists>

1. 연결된 슬레이브 장치를 재연결하면 릴레이가 이전 상태와 다르게 동작하는 문제

1. 엘리베이터를 화재 경보 구역으로 구성했을 때 화재 경보 발생 후 층 상태에서 화재 경보를 해제할 수 있는 문제

1. 마스터 장치에서 지원하지 않는 장치를 슬레이브로 연결했을 때 해당 슬레이브 장치의 펌웨어를 업그레이드할 수 없도록 수정

1. 언어를 일본어로 설정했을 때 일본어 리소스가 정상적으로 적용되지 않는 문제

1. 새 이벤트 로그가 누락되어 표시되지 않는 문제

1. 사용자 이름을 설정할 때 일부 특수 문자(\\, /, :, *, ?, ", ', `, \<, >, |, .)를 사용할 수 없도록 수정

1. 관리자 권한을 가진 사용자가 다른 관리자 권한을 가진 사용자의 권한을 설정할 수 없는 문제

1. 브래킷을 제거하고 장치를 재부팅하면 탬퍼 꺼짐 이벤트가 발생하는 문제

1. 장치를 재부팅할 때 탬퍼 켜짐/꺼짐 알림이 바로 표시되지 않는 문제

1. 작업 조건 및 동작에서 인증 성공 또는 실패 설정이 정상적으로 동작하지 않는 문제

1. 슬레이브 장치로 연결한 XPass D2에 카드를 인증하면 인증 결과 비프음이 2회 발생하는 문제

1. BS_GetLogBlob 명령어로 출입문 ID를 호출할 경우 출입문 ID가 정상적으로 출력되지 않는 문제

1. 지문을 등록할 때 스캔한 지문 이미지가 한쪽으로 치우쳐 나타나는 문제

</BugLists>
