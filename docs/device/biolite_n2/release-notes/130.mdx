# 펌웨어 버전 1.3.0 (빌드 번호 1.3.0_200609)

릴리즈<Separator/>2020-06-11

## 주요 버그 수정

---

1. 장치 연결이 해제된 동안 발생한 로그가 장치 재연결 이후에도 BioStar 2에 전송되지 않는 문제

## 신규 기능 및 개선

---

1. 장치 ID 변경 기능 지원

1. 장치 보안 강화

    - 암호화되지 않은 통신 연결 제한

    - 암호화 키 보안 강화

    - 사용자 정보 암호화 및 마이그레이션 지원

1. 안티패스백 구역 규칙이 실제 출입문의 상태에 따라 동작하도록 개선

1. 엘리베이터 제어 시 층별 스케줄 개방 구역 기능 지원

1. 신규 장치 연결 지원

    - XPass D2 (Rev 2)

1. Mobile Access V1.1 지원

## 버그 수정

---

<BugLists>

1. 스케줄 개방 및 수동 잠금이 설정된 상태에서 마스터 장치가 재부팅될 경우 스케줄 개방으로 동작하는 문제

1. 1세대 장치에서 발급한 iCLASS 카드를 정상적으로 인식하지 못하는 문제

1. Wiegand 출력 장치에서 카드를 등록한 뒤 다른 사용자에게 등록된 지문으로 Wiegand 출력 시 장치가 정상적으로 동작하지 않는 문제

1. 바이트 순서를 LSB로, Wiegand 출력 정보를 사용자 ID로 설정한 뒤 지문 인증으로 인증하면 장치가 재부팅되는 문제

1. 슬레이브 장치에서 스마트 카드 및 얼굴으로 인증하면 마스터 장치가 재부팅되는 문제

1. 사용자 ID 종류를 영숫자로 설정했을 때 장치에서 공백을 포함된 사용자 ID를 등록할 수 있는 문제

1. BioStation 2를 슬레이브 장치로 연결하였을 때 작업 조건 및 동작에서 출력 포트를 설정할 수 없는 문제

1. BioEntry R2(BER2-OD)와의 RS-485 통신이 정상적으로 동작하지 않는 문제

1. 동작 신호 출력에 지연이 발생하는 문제

1. 인증 모드를 편집해도 편집한 내용이 적용되지 않는 문제

1. SSL 보안 통신 중 펌웨어를 업그레이드하거나 사용자를 전송할 때 장치가 재부팅되거나 시간 초과되는 문제

1. 지문 템플릿 업데이트 시 오류가 발생하는 문제

1. 협박 지문으로 등록한 지문 인증을 실패하는 문제

1. XPass 2를 슬레이브 장치로 연결하였을 때 인증 모드 설정이 정상적으로 동작하지 않는 문제

1. 슬레이브 장치로 연결한 BioStation 2를 공장초기화해도 일부 리소스가 초기화되지 않는 문제

1. 슬레이브 장치에서 '잘못된 페이로드 정보입니다.' 메시지가 발생한 뒤 비정상적으로 슬레이브 장치의 연결이 끊기고 재연결이 불가능한 문제

1. 카드 형식별 선택 옵션에서 SEOS를 선택하지 않고, 스마트 카드 레이아웃을 설정한 뒤 iCLASS SEOS 카드로 인증할 경우 오류가 발생하는 문제

1. DES/AES 암호화 모드에서 발급된 AoC, SCC로 인증 시 오류가 발생하는 문제

1. 글로벌 안티패스백 구역에서 안티패스백 위반 시 인증 성공 로그가 발생하는 문제

1. RS-485 통신 시 장치에서 연속으로 지문 인증을 시도할 경우 오인증이 발생하거나 인증을 실패하는 문제

1. 간헐적으로 iCLASS 카드를 인식하지 못하는 문제

</BugLists>
