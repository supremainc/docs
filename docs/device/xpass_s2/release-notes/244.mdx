---

1. MIFARE Plus 카드를 스마트 카드로 사용하기 위해 포맷할 때, 장치에서 해당 카드의 사이트 키 값이 설정한 값과 다르게 적용되는 문제

1. 슬레이브 장치로 사용 시 장치 설정을 변경하고 재시작하면 RS-485 연결이 끊기는 문제

1. 보안 칩 인증 실패가 여러 번 반복되어 장치 부팅에 실패하는 문제
