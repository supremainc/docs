1. <Image src='/img/common/ico-menu.svg' ico alone />을 누른 뒤 관리자 권한 인증 수단으로 인증하세요.<br/><Cmd sid='ui_109' product='dev' />에서 장치 사용 현황 및 정보를 확인할 수 있습니다. 또는 <Cmd sid='ui_174' product='dev' />를 눌러 바로 사용자를 등록할 수도 있습니다.
