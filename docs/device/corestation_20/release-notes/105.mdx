# 펌웨어 버전 1.0.5 (빌드 번호 1.0.5_260616)

릴리즈<Separator/>2026-06-24

:::caution

- 펌웨어를 v1.0.3 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

- 장치의 펌웨어 다운그레이드 가능 여부 및 신규 BLE 칩 적용 여부는 장치의 시리얼 번호를 확인한 뒤 [슈프리마 홈페이지](https://supremainc.com)에 문의하세요.

:::

## 신규 기능 및 개선

---

1. 신규 장치 연결 지원

   - XPass Q2

1. CoreStation Setup Manager 기능 개선

   - SCB 키 수동 입력 설정 지원

   - 네트워크 정보(Device ID, SCB 키 등) 파일 백업 및 복원 기능 지원

   - CoreStation 교체 시 기존 설정을 이전할 수 있도록 개선

   - 전반적인 폰트 크기 및 UI 개선

1. 카메라 QR 라이선스 적용 시 '활성화된 수'가 표시되지 않도록 UI 개선
