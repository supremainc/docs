# 펌웨어 버전 1.0.1 (빌드 번호 1.0.1_250604)

릴리즈<Separator/>2025-06-05

## 신규 기능 및 개선

---

1. 슬레이브로 연결된 장치의 **커스텀 스마트 카드** 지원

1. CoreStation Setup Manager에서 신규 장치 연결 지원

    - Door Interface (DI-24)

1. 신규 장치 연결 지원

    - Door Interface (DI-24)

1. Fire Aux 입력 포트를 통해 엘리베이터 제어가 가능하도록 지원

## 버그 수정

---

<BugLists>

1. LED 상태 표시등이 빨간색으로만 점멸되는 현상 <Badge>1.0.0</Badge>

1. CoreStation Setup Manager에서 RS-485 포트 1개에 3개 이상의 슬레이브 장치가 연결되는 문제 <Badge>1.0.0</Badge>

1. CoreStation Setup Manager에서 탬퍼가 연결된 AUX 포트의 **Use Port** 설정이 체크되어 있는 상태에서 **보안 탬퍼**가 동작하지 않는 문제 <Badge>1.0.0</Badge>

1. 장치에서 탬퍼가 동작하면 LED 불빛이 이전 점등 상태로 되돌아가는 문제 <Badge>1.0.0</Badge>

1. CoreStation Setup Manager에서 AUX 포트(AC Fail, Tamper, Fire)의 기본 설정이 비활성화 되어있는 문제 <Badge>1.0.0</Badge>

1. 인증모드에 비주얼 페이스가 포함되어 발급된 스마트 카드로 인증 시, '유효기간 만료' 메시지가 표시되며 인증에 실패하는 문제 <Badge>1.0.0</Badge>

1. 슬레이브로 연결된 장치에서 복잡한 구조의 QR 코드로 인증할 때, 카드 값이 표시되지 않고 이벤트 로그가 '1:1 인증 실패(QR/Barcode)'로 잘못 표시되는 문제 <Badge>1.0.0</Badge>

1. 슬레이브로 구성된 장치에서 출입문을 생성한 후, 슬레이브 장치의 릴레이를 출입문 제어에 사용할 때 릴레이 상태가 잘못 표시되는 문제 <Badge>1.0.0</Badge>

1. 화재 경보로 출입문이 개방된 상태에서 컨트롤러 재부팅 후, 알람 해제 시 화재 경보가 정상적으로 해제되지 않는 문제 <Badge>1.0.0</Badge>

1. Wiegand 리더로 구성된 출입문에 이중 인증을 설정한 후 인증 시, 마스터 장치가 재시작되는 문제 <Badge>1.0.0</Badge>

1. 엘리베이터에 **스케줄 개방**이 설정된 상태에서 슬레이브 장치의 RS-485 연결이 해제되고, **스케줄 개방**이 종료된 후 슬레이브 장치가 다시 연결되었을 때 릴레이가 잠금으로 동작하지 않는 문제 <Badge>1.0.0</Badge>

1. 시간을 변경한 후 **서버와 시간 동기화** 옵션을 활성화할 경우, 동기화가 즉시 이루어지지 않고 일정 시간이 지난 후에 동기화되는 문제 <Badge>1.0.0</Badge>

1. **네트워크 제외** **설정 초기화**를 실행한 후 슬레이브 장치를 다시 추가하면, UI에서는 추가된 슬레이브 장치들이 끊어진 상태로 표시되지만 실제 장치 로그에서는 정상적으로 연결되어 있는 문제 <Badge>1.0.0</Badge>

1. Wiegand 포맷 25비트 이하의 카드 데이터를 인식하지 못하는 문제 <Badge>1.0.0</Badge>

1. OSDP 리더가 설치된 출입문에 **이중 인증**을 설정했을 때, 첫 번째 인증 후 입력 대기 상태에서 LED 및 버저가 잘못 동작하는 문제 <Badge>1.0.0</Badge>

1. CoreStation 20 및 Door Interface의 RS-485 포트 전송 속도 변경 후, Door Interface 펌웨어 업그레이드 시 CoreStation 20이 무한 재부팅되는 문제 <Badge>1.0.0</Badge>

1. 특정 장치를 슬레이브로 연결했을 때, 설정 변경 사항이 정상적으로 반영되지 않는 문제 <Badge>1.0.0</Badge>

</BugLists>
