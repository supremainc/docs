### CoreStation 20 상태 LED

제품의 상태는 제품 전면부의 오른쪽에 위치한 STATUS로 확인할 수 있으며, 이더넷, RS-485, RS-485 0, RS-485 1, 시스템의 상태를 1초씩 순서대로 보여줍니다.

| 항목 | LED | 상태 |
|---|---|---|
| 이더넷 | 빨간색/자홍색 점멸 | 고정 IP 연결 실패 |
| ^ | 빨간색/파란색 점멸 | 유동 IP 연결 성공 |
| ^ | 빨간색/하늘색 점멸 | 고정 IP 연결 성공 |
| ^ | 빨간색/노란색 점멸 | 유동 IP 연결 대기 중 |
| ^ | 빨간색/빨간색 점멸 | 유동 IP 연결 실패 |
| RS-485 | 자홍색/파란색 점멸 | 슬레이브 장치 정상 |
| ^ | 자홍색/노란색 점멸 | 슬레이브 장치 연결 해제 |
| ^ | 자홍색/빨간색 점멸 | 슬레이브 장치 없음 |
| RS-485 0 | 노란색/파란색 점멸 | 슬레이브 장치 정상 |
| ^ | 노란색/노란색 점멸 | 슬레이브 장치 연결 해제 |
| ^ | 노란색/빨간색 점멸 | 슬레이브 장치 없음 |
| RS-485 1 | 초록색/파란색 점멸 | 슬레이브 장치 정상 |
| ^ | 초록색/노란색 점멸 | 슬레이브 장치 연결 해제 |
| ^ | 초록색/빨간색 점멸 | 슬레이브 장치 없음 |
| 시스템 | 흰색/회색 점멸 | 시스템 잠금 (BioStar 2의 작업 조건 및 동작) + 탬퍼 켜짐 |
| ^ | 흰색/자홍색 점멸 | 시스템 잠금 (장치의 작업 조건 및 동작) + 탬퍼 켜짐 |
| ^ | 흰색/파란색 점멸 | 탬퍼 켜짐 |
| ^ | 흰색/하늘색 점멸 | BioStar 2 연결 성공 + 탬퍼 켜짐 |
| ^ | 흰색/초록색 점멸 | BioStar 2 연결 성공 |
| ^ | 흰색/노란색 점멸 | 시스템 잠금 (BioStar 2의 작업 조건 및 동작) |

### 설치 예제

이 제품은 중앙 집중식 출입 통제 시스템으로 엔터프라이즈급 환경을 위해 설계되었으며 최대 500,000명의 사용자 정보를 저장할 수 있습니다.

BioStar 2와 연동하여 출입통제와 근태관리를 모두 사용할 수 있으며, RS-485(OSDP), Wiegand, Suprevised input, AUX와 같은 다양한 인터페이스를 제공합니다.

### 시스템 구성도

이 제품은 3개의 RS-485 포트를 지원하며, Door Interface를 데이지 체인으로 연결하면 최대 34개의 출입문을 구성할 수 있습니다.

<Image src='/img/device/system-configuration-cs20.svg' width="70%" className='none' />

:::info

* RS-485 0 및 RS-485 1 포트에는 포트당 최대 2대의 장치를 연결할 수 있습니다.

* RS-485 포트 하나에 3대 이상의 장치가 연결된 경우 슬레이브 장치 검색 시 원하는 장치가 연결 가능 목록에 표시되지 않을 수 있습니다.

* CS-20P 모델에 SAM 카드를 장착하여 사용 중인 경우 CS-20P의 리더 포트에 직접 연결된 슬레이브 장치만 사용할 수 있습니다.

* Door Interface를 호스트 포트에 슬레이브로 연결하면 CoreStation 20과의 연결이 끊어지더라도 Door Interface에 연결된 슬레이브 장치에서 오프라인 모드 전용 시설 코드(Facility Code for Offline Mode)를 통해 카드의 시설 코드(Facility Code)를 식별하여 출입문을 열 수 있습니다.

:::
