# 펌웨어 버전 1.5.0 (빌드 번호 1.5.0_200722)

릴리즈<Separator/>2020-07-29

## 신규 기능 및 개선

---

1. 장치 ID 변경 기능 지원

1. 장치 보안 강화

    - 암호화되지 않은 연결 제한

    - 암호화 키 보안 강화

    - 사용자 정보 암호화 및 마이그레이션

1. 안티패스백 구역 규칙이 실제 출입문의 상태에 따라 동작하도록 개선

1. 엘리베이터 제어 시 층별 스케줄 개방 기능 지원

1. 신규 장치 연결 지원

    - XPass D2 (Rev 2)

1. LED 상태 표시등 색상 변경

## 버그 수정

---

<BugLists>

1. 카드 형식별 옵션을 설정하여 장치가 알 수 없는 카드를 인식할 수 있도록 수정 (BEW2-OAP)

1. 슬레이브 장치로 전송된 값이 정의된 값보다 클 경우 OSDP 통신이 정상적으로 동작하지 않는 문제

1. SSL 보안 통신 중 펌웨어를 업그레이드하거나 사용자를 전송할 때 장치가 재부팅되거나 시간 초과되는 문제

1. 장치의 이벤트 로그가 발생한 순서대로 전송되지 않는 문제

1. 공장 초기화 상태인 장치를 슬레이브로 연결 시 카드 형식 옵션이 비활성화되는 문제

1. 슬레이브 장치 추가 시 슬레이브 장치의 연결이 끊어지는 문제

1. 전원 연결 후 아무 동작이 없을 경우 장치가 재시작되는 문제

1. 유효 인식 거리 내에서도 LF 카드에 대한 인식 성능이 떨어지는 문제

</BugLists>
