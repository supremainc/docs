# 펌웨어 버전 1.4.0 (빌드 번호 1.4.0_190708)

릴리즈<Separator/>2019-07-12

## 주요 버그 수정

1. 스케줄 개방 구역에 구성된 출입문이 스케줄 개방에 의해 열려 있을 때 해당 구역을 삭제하면 출입문이 잠기지 않는 문제

1. 카드 인증 시 캐시 메모리가 깨져 인증에 실패하는 것을 방지하기 위한 코드 적용

## 신규 기능 및 개선

---

1. OSDP 표준화

    - 타사 컨트롤러와 연결할 때 OSDP V2.1.7 프로토콜을 준수하도록 개선

1. 안티 테일게이팅(따라 들어가기 방지) 지원

1. Wiegand 인증 결과 출력 옵션 설정 지원

    - 사용자 ID 및 카드 ID

1. 장치 일괄 편집을 이용해 관리자를 추가할 때 새로운 설정이 적용되는 방식 변경

    - 기존: 기존 설정에 새로 설정한 내용을 덮어쓰는 방식

    - 변경: 기존 설정에 새로 설정한 내용을 추가하는 방식

1. 추가 가능한 관리자 수 확대

1. 최대 층 등급 수 확대

1. 카드 형식별 선택 옵션 지원

1. 사용자별 APB 해제 지원

1. 모듈 펌웨어 버전을 확인할 수 있도록 지원

1. 최신 버전 I/O 모듈 Micom 지원 (V1.3.1)

1. 신규 장치 연결 지원

    - XPass 2(XP2-MDPB, XP2-GDPB, XP2-GKDPB)

## 버그 수정

---

<BugLists>

1. iCLASS Seos 카드를 CSN 카드로 인식하는 문제

1. 위조 지문 검출(LFD)로 인한 인증 실패에 대해 버저음이나 LED 상태 표시등의 출력을 지원하지 않는 문제

1. 마스터 장치를 부팅할 때 슬레이브 장치에서 발생하는 입력에 대해 반응하지 않는 문제

1. 오인증 개선 알고리즘 적용

1. SDK를 이용해 로그를 가져올 때 시작 시간이 UTC 기준으로 적용되지 않는 문제

1. 최신 펌웨어를 사용할 때 BioStar 1.93에 연결할 수 없는 문제

1. 1세대 장치에서 발급한 iCLASS 카드를 인식하지 못하는 문제

1. 미지원 장치(FaceStation 2, FaceLite)를 지원하는 문제

1. HID Prox 카드를 연속적으로 인식하는 문제

1. 다중 인증 모드를 사용할 때 마스터 장치와 슬레이브 장치에서 다음 크리덴셜 입력 화면의 타이틀이 다르게 표시되는 문제

    - 기존: 마스터 장치(사용자 ID, 사용자 이름), 슬레이브 장치(사용자 ID)

    - 변경: 마스터 장치와 슬레이브 장치(사용자 ID, 사용자 이름)

1. 엘리베이터 연결이 끊긴 상태에서 발생한 인증에 대해 재연결한 뒤 릴레이가 동작하는 문제

</BugLists>
