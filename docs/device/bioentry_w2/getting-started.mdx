## 각 부분의 명칭 및 기능

<Image src='/img/device/name-function-each-part-bew2.png' className='none' width='60%' />

* **LED 상태 표시등**: 제품의 작동 상태가 표시됩니다.

  * 초록색: 인증 성공

  * 빨간색: 인증 실패

  * 파란색/하늘색 점멸: 정상 동작 상태

  * 파란색/초록색 점멸:

    * <Cmd>DHCP</Cmd>를 사용으로 설정 시 IP 주소를 받아오지 못하는 상태

    * <Cmd>슬레이브</Cmd> 모드일 때 마스터 장치와 연결이 끊어진 상태

    * 네트워크 설정 초기화 시

  * 초록색 점멸: 입력 대기 상태

  * 초록색/흰색 점멸: 슈프리마 Airfob Pass (BLE) 애플리케이션과 연결 중인 상태

* **RF 카드 및 모바일 출입카드 인식부**: 출입을 위해 RF 카드 또는 모바일 출입카드를 인식시키는 부분입니다.

* **지문 인식부**: 출입을 위해 지문을 인식시키는 부분입니다.

* **초기화 버튼**

  * 네트워크 설정을 초기화합니다. 자세한 내용은 [네트워크 설정 초기화](installation#reset-network-settings)를 참고하세요.

  * 장치에 저장된 모든 정보와 인증서를 삭제하고 설정을 초기화합니다. 자세한 내용은 [공장 초기화](installation#factory-reset)를 참고하세요.

* **네트워크 상태 LED**: 네트워크 연결 상태를 표시합니다.

* **케이블**

  전원 / 이더넷 + PoE / RS-485 / Wiegand 입출력 / Input / 릴레이
