# 펌웨어 버전 1.1.0 (빌드 번호 1.1.0_210813)

릴리즈<Separator/>2021-08-13

## 신규 기능 및 개선

---

1. 지문 모델 추가

    - X-Station 2 Finger (XS2-ODPB, XS2-OAPB)

1. 인텔리전트 슬레이브 지원

    - 인텔리전트 슬레이브: 슈프리마 장치와 타사 컨트롤러를 연결하였을 때 슈프리마 장치에서 직접 1:1 또는 1:N 매칭을 수행하고 인증 결과를 OSDP 카드 데이터로 타사 컨트롤러에 전송할 수 있는 기능

1. 수동 해시키를 설정해도 보안 탬퍼를 수동으로 켜거나 끌 수 있도록 개선

## 버그 수정

---

<BugLists>

1. 엘리베이터에 스케줄 개방 구역을 설정했을 때 관련 이벤트 로그 문구가 정상적으로 표시되지 않는 문제 <Badge>1.0.0</Badge>

1. 데이터를 최대(사용자 500,000명, 텍스트 로그 5,000,000개)로 저장한 장치에서 전체 데이터를 선택해 USB 내보내기한 뒤 사용자 데이터를 가져오기 할 경우 장치가 비정상적으로 재부팅되는 문제 <Badge>1.0.0</Badge>

1. 개인 인증 모드를 카드+PIN으로 설정했을 때 슬레이브로 연결된 XPass D2(XPD2-GDB, XPD2-MDB)에서 카드를 스캔할 경우 바로 실패되지 않고 입력 대기 상태로 동작하는 문제 <Badge>1.0.0</Badge>

1. 슬레이브로 연결된 다른 모델 장치의 펌웨어 업그레이드에 실패하는 문제 <Badge>1.0.0</Badge>

1. 슬레이브로 연결된 일부 지문 장치에서 사용자 지문을 등록할 수 없는 문제 <Badge>1.0.0</Badge>

1. 슬레이브로 연결된 장치의 인증 모드가 카드+지문일 때 비정상적으로 인증에 실패하는 문제 <Badge>1.0.0</Badge>

1. 장치를 타사 컨트롤러와 OSDP 로 연결했을 때 컨트롤러의 Secure Communication 모드를 활성화하면 RS-485 통신이 정상적으로 동작하지 않는 문제 <Badge>1.0.0</Badge>

1. 펌웨어를 1.0.3 이하 버전에서 1.1.0 버전으로 업그레이드 시 실패하는 문제 <Badge>1.0.0</Badge>

    - 1.0.3 이하 버전에서 1.1.0 이후 버전으로 펌웨어를 업그레이드할 경우 먼저 1.1.0 펌웨어로 업그레이드를 진행한 뒤 1.1.0 이후 버전 펌웨어로 다시 업그레이드하십시오.

1. 장치에서 이더넷 페이지에 진입할 경우 서버와의 연결이 끊어지는 문제 <Badge>1.0.0</Badge>

1. 근태 모드가 설정된 장치에서 ID 인증을 통해 메뉴에 진입했다가 다시 홈 화면으로 이동하면 근태 캡션이 표시되지 않는 문제 <Badge>1.0.0</Badge>

1. 슬레이브로 연결된 OM-120의 특정 포트를 출입문 릴레이로 설정한 뒤 장치를 재부팅하면 릴레이가 비정상적으로 동작하는 문제 <Badge>1.0.0</Badge>

1. DHCP를 사용할 때 비정상적인 IP가 출력되는 문제 <Badge>1.0.0</Badge>

1. BioStar 2와 연결이 끊어졌을 때 상황과 무관한 '펌웨어 업그레이드 실패' 메시지가 표시되는 문제 <Badge>1.0.0</Badge>

1. 장치에서 이벤트 로그를 검색할 때 메뉴 표시 시간이 초과되어도 날짜 선택 팝업 화면이 계속 표시되는 문제 <Badge>1.0.0</Badge>
