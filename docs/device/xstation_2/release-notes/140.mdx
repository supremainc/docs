# 펌웨어 버전 1.4.0 (빌드 번호 1.4.0_251128)

릴리즈<Separator/>2025-12-10

:::caution

- 펌웨어를 v1.4.0 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

- X-Station 2 모델 중 XS2-APB, XS2-QAPB, XS2-OAPB 타입은 SE 프로세서를 통해 HID iClass 카드 타입을 지원하고 있습니다.<br/>신규 SE 프로세서가 적용된 장치의 경우 펌웨어를 v1.1.4 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없으며, 하위 버전에서 커스터마이징한 펌웨어도 다운그레이드할 수 없습니다.<br/>기존 SE 프로세서가 적용된 장치는 하위 버전으로 다운그레이드할 수 있습니다.

- 장치의 신규 SE 프로세서 적용 여부 및 펌웨어 다운그레이드 가능 여부는 장치의 시리얼 번호를 확인한 뒤 [슈프리마 홈페이지](https://supremainc.com)에 문의하세요.

:::

## 신규 기능 및 개선

---

1. 전체 관리자 권한을 갖는 **마스터 관리자** 설정 기능 지원

1. 글로벌 사이버 보안 표준 및 최신 보안 요구사항 반영

1. **서버 개인 메시지** 기능을 지원하는 장치를 구분하도록 지원

1. 최신 카드를 인식할 수 있도록 구분 코드 추가

    - DESFire EV3

1. **보안 탬퍼** 기능을 지원하는 장치를 구분하도록 개선

1. 서버와 장치 간 시간 동기화 관련 동작 개선

1. **반복 신호 무시 시간** 기능이 작동할 때 이벤트를 기록하지 않도록 지원
