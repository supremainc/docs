# 펌웨어 버전 1.4.1 (빌드 번호 1.4.1_260408)

릴리즈<Separator/>2026-04-15

:::caution

- 펌웨어를 v1.4.0 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

- X-Station 2 모델 중 XS2-APB, XS2-QAPB, XS2-OAPB 타입은 SE 프로세서를 통해 HID iClass 카드 타입을 지원하고 있습니다.<br/>신규 SE 프로세서가 적용된 장치의 경우 펌웨어를 v1.1.4 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없으며, 하위 버전에서 커스터마이징한 펌웨어도 다운그레이드할 수 없습니다.<br/>기존 SE 프로세서가 적용된 장치는 하위 버전으로 다운그레이드할 수 있습니다.

- 장치의 신규 SE 프로세서 적용 여부 및 펌웨어 다운그레이드 가능 여부는 장치의 시리얼 번호를 확인한 뒤 [슈프리마 홈페이지](https://supremainc.com)에 문의하세요.

:::

## 신규 기능 및 개선

---

<Jira num='ONYXFW-836' /> XS2-Q모델에서 Camera QR 지원

1. XS2-QDPB, XS2-QAPB 모델에서 **카메라 QR** 라이선스 활성화 시 장치 카메라를 활용한 QR 인증 방식을 선택할 수 있도록 지원

1. Supported an option to select QR authentication using the device camera upon **Camera QR** license activation on XS2-QDPB and XS2-QAPB models.

<Jira num='ONYXFW-982' /> (BDP-1343) Desfire EV2 and EV3 features supported by Readers and Software

1. DESFire EV2/EV3 카드에 대한 EV2 Secure Messaging 인증 및 발급 지원

1. Supported EV2 Secure Messaging authentication and issuance for DESFire EV2 and EV3 cards.

<Jira num='ONYXFW-979' /> 장치에서 지원하지 않는 이벤트에 대해 "Undefined Event"로 로그가 남도록 개선

1. 장치에서 지원하지 않는 이벤트 및 에러에 대한 로그 표시 개선

1. Improved log display for unsupported events and errors on the device.

<Jira num='ONYXFW-980' /> CE RED 관련 개선

1. CE(유럽) 외 지역용 **글로벌** 펌웨어 제공

   - **마스터 관리자** 및 **관리자 2단계 인증** 기능 비활성화

   - PIN 등록 시 최소 길이 4자리로 완화

1. Provided **global** firmware for non-CE(EU) regions

   - Disabled the **Master Admin** and **Admin 2-step Authentication** features

   - Reduced the minimum length for PIN enrollment to 4 digits
