# 펌웨어 버전 1.2.1 (빌드 번호 1.2.1_200612)

릴리즈<Separator/>2020-06-26

## 신규 기능 및 개선

---

1. 사용자 정보 마이그레이션 시 블랙리스트와 관리자로 설정한 사용자 ID에 대해서도 암호화 지원

1. 장치 ID 변경 시 네트워크 및 SSL 설정은 초기화되지 않도록 개선

## 버그 수정

---

<BugLists>

1. 값이 깨진 로그가 발생하여 서버로 전송하지 못하는 문제

1. 슬레이브 장치 추가 시 슬레이브 장치의 연결이 끊어지는 문제

1. 공장 초기화 상태인 장치를 슬레이브로 연결 시 카드 형식 옵션이 비활성화되는 문제

1. 공장 초기화 이후 장치의 로그가 서버로 전송되지 않는 문제

</BugLists>
