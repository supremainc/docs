# 펌웨어 버전 1.2.0 (빌드 번호 1.2.0_200409)

릴리즈<Separator/>2020-04-13

## 신규 기능 및 개선

---

1. 장치 ID 변경 기능 지원

1. 장치 보안 강화

    - 암호화되지 않은 연결 제한

    - 슬레이브 장치에 보안 탬퍼 지원

    - 암호화 키 보안 강화

    - 사용자 정보 암호화

    - TCP/IP, RS-232, USB와 같은 통신을 통한 장치 접근 제한

1. 신규 장치 연결 지원

    - XPass D2 (Rev 2)

1. 안티패스백 구역 규칙이 실제 출입문의 상태에 따라 동작하도록 개선

1. BioStar 2 Mobile Access 기능 지원

1. Mobile Access V1.1 지원

1. 엘리베이터 제어 시 층별 스케줄 개방 기능 지원

## 버그 수정

---

<BugLists>

1. 장치를 공장 초기화해도 Device Manager 앱에서 설정한 비밀번호가 초기화되지 않는 문제

1. LF(저주파수) 카드에 대한 인식률이 떨어지는 문제 (XP2-MDPB)

1. 슬레이브 장치로 전송된 값이 정의된 값보다 클 경우 OSDP 통신이 정상적으로 동작하지 않는 문제

1. 이미 보안 탬퍼가 발생한 상태에서 스마트 카드 레이아웃을 설정한 뒤 장치를 재부팅하면 보안 탬퍼가 다시 발생하는 문제

1. 장치의 이벤트 로그가 발생한 순서대로 전송되지 않는 문제

</BugLists>
