# 펌웨어 버전 1.5.0 (빌드 번호 1.5.0_250811)

릴리즈<Separator/>2025-08-14

:::caution

- 펌웨어를 v1.5.0 이상으로 업그레이드 한 뒤 하위 버전으로 다운그레이드 할 수 없습니다.

- XP2-MAPB-H 모델의 경우 펌웨어를 v1.4.3 이하로 다운그레이드 할 수 없습니다.

- 하드웨어 v1.2.0 이상의 XP2-GKDPB 모델은 펌웨어를 v1.4.3 이하 버전으로 다운그레이드 할 수 없습니다. 하드웨어 버전은 BioStar 2의 **장치** 메뉴에서 확인할 수 있습니다.

- 신규 BLE 칩이 적용된 장치의 경우 펌웨어를 v1.3.1 이상으로 업그레이드 한 뒤 하위 버전으로 다운그레이드 할 수 없습니다. 기존 BLE 칩이 적용된 장치는 하위 버전으로 다운그레이드 할 수 있습니다.

장치의 펌웨어 다운그레이드 가능 여부 및 신규 BLE 칩 적용 여부는 장치의 시리얼 번호를 확인한 뒤 [슈프리마 홈페이지](https://supremainc.com)에 문의하십시오.

:::
