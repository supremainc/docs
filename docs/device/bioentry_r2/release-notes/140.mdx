# 펌웨어 버전 1.4.0 (빌드 번호 1.4.0_200521)

릴리즈<Separator/>2020-05-27

## 신규 기능 및 개선

1. 장치 ID 변경 기능 지원

1. 장치 보안 강화

    - 보안 탬퍼 지원

    - 암호화 키 보안 강화

    - TCP/IP, RS-232, USB와 같은 통신을 통한 장치 접근 제한

    - 사용자 정보 암호화 및 마이그레이션 지원

1. LED 상태 표시등 색상 변경
