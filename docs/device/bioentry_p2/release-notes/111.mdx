# 펌웨어 버전 1.1.1 (빌드 번호 1.1.1_180720)

릴리즈<Separator/>2018-07-25

## 신규 기능 및 개선

---

1. LCD가 있는 장치에서, 슬레이브 모드로 연결된 경우에도 인증에 성공하면 사용자 이름이 표시되도록 개선

1. 신규 장치 연결 지원

    - XPass D2(XPD2-GDB, XPD2-GKDB)

## 버그 수정

---

<BugLists>

1. Wiegand 출력에서 AoC 인증 시 항시 통과가 미동작하는 문제

1. BioStar 2에 이벤트 로그 및 실시간 로그가 정상적으로 업로드되지 않는 문제

1. RS-485로 연결된 장치를 재연결할 경우 릴레이의 상태가 유지되지 않는 문제

1. 통신 보안 및 데이터 암호화 키를 사용 중인 상태에서는 공장 초기화 시에도 표준 시간대가 초기화되지 않는 문제

1. 출입문 릴레이로 설정된 장치가 연결 해제된 상태에서 인증에 성공할 경우, 해당 장치 재연결 후에 이전 값에 대한 릴레이가 발생하는 문제

1. 인증 실패 시 장치가 재시작되는 문제

1. 13.56 MHz 카드를 사용할 때 장치의 RF 모듈이 손상되는 문제

</BugLists>
