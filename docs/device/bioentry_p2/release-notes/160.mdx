# 펌웨어 버전 1.6.0 (빌드 번호 1.6.0_260212)

릴리즈<Separator/>2026-03-13

:::caution

- 펌웨어를 v1.6.0 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

- SW 기반 **위조 지문 검출** 기능이 추가된 장치의 경우 펌웨어를 v1.5.1 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

- BioEntry P2 모델 중 BEP2-OA는 SE프로세서를 통해 HID iCLASS 카드 타입을 지원합니다.<br/>신규 SE 프로세서가 적용된 장치의 경우 펌웨어를 v1.4.4 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없으며, 하위 버전에서 커스터마이징한 펌웨어도 다운그레이드할 수 없습니다. 기존 SE 프로세서가 적용된 장치는 하위 버전으로 다운그레이드할 수 있습니다.

- 장치의 펌웨어 다운그레이드 가능 여부는 장치의 시리얼 번호를 확인한 뒤 [슈프리마 홈페이지](https://supremainc.com)에 문의하세요.

:::
