# 펌웨어 버전 1.6.0 (빌드 번호 1.6.0_260212)

릴리즈<Separator/>2026-03-13

:::caution

- 펌웨어를 v1.6.0 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

- SW 기반 **위조 지문 검출** 기능이 추가된 장치의 경우 펌웨어를 v1.5.1 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

- BioEntry P2 모델 중 BEP2-OA는 SE프로세서를 통해 HID iCLASS 카드 타입을 지원합니다.<br/>신규 SE 프로세서가 적용된 장치의 경우 펌웨어를 v1.4.4 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없으며, 하위 버전에서 커스터마이징한 펌웨어도 다운그레이드할 수 없습니다. 기존 SE 프로세서가 적용된 장치는 하위 버전으로 다운그레이드할 수 있습니다.

- 장치의 펌웨어 다운그레이드 가능 여부는 장치의 시리얼 번호를 확인한 뒤 [슈프리마 홈페이지](https://supremainc.com)에 문의하세요.

:::

## 신규 기능 및 개선

---

 (BPIIFW-298) BSF-2469 고객의 DESFire EV3 (EV1으로 발급) 카드가 BioStation 3에서 읽히지 않는 증상 관련해 Desfire 구분 코드(ATQA 0x0304, SAK 0x20) 추가 - 발생버전: V1.5.1

1. 최신 카드를 인식할 수 있도록 구분 코드 추가
   * DESFire EV3

1. Added identification code to support recognition of the latest cards.
   * DESFire EV3

 (BPIIFW-320) BSF-2503 서버와 장치 간 시간 동기화 관련 동작 개선 - 발생버전: V1.4.2

1. 서버와 장치 간 시간 동기화 관련 동작 개선

1. Improved time synchronization behavior between server and devices.

 (BPIIFW-256) [BDP-853] Slave 장치의 Baudrate 설정 기능 지원

1. 슬레이브 장치의 전송 속도를 설정할 수 있는 기능 지원

1. Supports feature to set the baud rate of slave devices.

 (BPIIFW-311) Secure Tamper Capability 지원

1. **보안 탬퍼** 기능을 지원하는 장치를 구분하도록 개선

1. Improved to distinguish devices that support the **Secure Tamper** feature.

 (BPIIFW-182) [BDP-856] Wiegand Out 후 일정 시간 내 반복적인 인풋 신호를 무시하기

1. 장치를 타사 컨트롤러와 Wiegand로 연결했을 때 컨트롤러로부터 반복적으로 입력되는 인증에 대한 신호를 무시하는 **반복 신호 무시 시간** 기능 지원

1. Supports the **Ignore Repeated Signals Duration** feature, which ignores repetitive authentication signals from the controller when the device is connected to a 3rd party controller via Wiegand.

 (BPIIFW-277) [BDP-1066] Wiegand Out 후 일정 시간 내 반복적인 인풋 신호를 무시할 때 이벤트도 남기지 않기

1. **반복 신호 무시 시간** 기능이 작동할 때 이벤트를 기록하지 않도록 지원

1. Supports not logging events when the **Ignore Repeated Signals Duration** feature is activated.

 (BPIIFW-291) [BDP-1067] Mifare Plus EV1 SL1, SL3, SL1SL3 Mix Mode 지원[PFE 281]

1. MIFARE Plus EV1 카드의 보안 수준 호환성을 위해 SL1, SL3, SL1/SL3 혼합 모드 지원
   * SL1: MIFARE Classic 호환 모드
   * SL3: AES 기반 고급 보안 모드

Supports SL1, SL3, and SL1/SL3 Mix Mode for security level compatibility with MIFARE Plus EV1 cards.
   * SL1: Compatible mode with MIFARE Classic
   * SL3: Advanced security mode based on AES

 (BPIIFW-132) SEOS Config Card 처리시간 동안 unknown-CSN인식 무시

1. 장치 부팅 후 1분 동안 SEOS 설정 카드를 CSN으로 인식하지 않도록 개선

1. Improved to not recognize SEOS configuration cards as CSN for 1 minute after device booting.
