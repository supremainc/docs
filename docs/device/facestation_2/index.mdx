FaceStation 2는 세계 최고 성능의 얼굴 인증 알고리즘을 탑재한 슈프리마의 차세대 얼굴 인증 장치입니다. 강력한 쿼드코어 CPU를 이용하여 초당 최대 4,000명까지 매칭하는 초고속 인증 성능을 제공합니다.
