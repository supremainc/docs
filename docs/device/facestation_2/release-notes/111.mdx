# 펌웨어 버전 1.1.1 (빌드 번호1.1.1_180705)

릴리즈<Separator/>2018-07-23

## 신규 기능 및 개선

---

1. 스마트 카드에도 Wiegand 출력을 통한 항시 통과 모드 지원

    - 항시 통과 모드: 장치가 인증 절차 없이 타사 장치에 카드 데이터를 내보내는 모드. 이 모드에서는 장치가 더미 장치 역할을 하며, 카드 데이터만 Wiegand를 통해 전송합니다.

1. iCLASS 2K 카드를 등록할 때 지문 없이 보안 크리덴셜 카드의 발급이 가능하도록 개선

1. RFID 카드 인식 거리 개선

1. 신규 장치 연결 지원

    - FaceStation 2(FS2-DS), BioLite N2(BLN2-PAB), XPass D2(XPD2-GDB, XPD2-GKDB)

## 버그 수정

---

<BugLists>

1. Wiegand 출력에서 AoC 인증 시 항시 통과가 미동작하는 문제

1. BioStar 2에 이벤트 로그 및 실시간 로그가 정상적으로 업로드되지 않는 문제

1. 설정한 일광 절약 시간 종료 이후에도 장치의 시간이 일광 절약 시간으로 유지되는 문제

1. Secure I/O 2를 슬레이브 장치로 연결하면 릴레이 알람이 발생했을 때 설정한 대로 동작하지 않는 문제

1. 저장된 사용자가 없는 경우에도 USB로 사용자 내보내기가 가능한 문제

1. 경비 구역에 포함되지 않은 장치에 경비 개시/해제 키가 표시되는 문제

1. Wiegand Reader의 잠금을 해제할 수 없는 문제

1. 슬레이브 장치에서 미등록 지문 인증 시 장치가 재시작되는 문제

1. 저장된 데이터가 있는 경우에도 USB로 데이터 내보내기를 실행하면 빈 파일이 생성되는 문제

1. 장치가 카드를 NFC 태그로 인식하기 때문에 CSN을 읽을 수 없는 문제

1. BioStar 2에 사용자가 존재하는 상태에서 USB 불러오기를 실행하면 모든 사용자가 삭제되는 문제

1. 데이터 가져오기를 수행할 때 암호화 키가 일치하지 않을 경우 "USB Connection Fail" 메시지가 표시되는 문제

1. 스마트 카드의 기본 사이트 키를 보조 사이트 키로 변경할 경우, 장치가 해당 스마트 카드를 CSN 카드로 인식하는 문제

</BugLists>
