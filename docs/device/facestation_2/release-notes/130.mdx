# 펌웨어 버전 1.3.0 (빌드 번호1.3.0_190805)

릴리즈<Separator/>2019-09-25

## 주요 버그 수정

1. 연결이 끊긴 슬레이브 장치를 다시 연결하면 마스터 장치가 비정상적으로 종료되는 문제

1. 슬레이브 장치에서 연속으로 얼굴을 인증할 경우 마스터 장치가 비정상적으로 재시작되는 문제

1. 연결되어 있던 31대의 슬레이브 장치의 연결을 해제한 뒤 마스터 장치의 RS-485 모드를 변경할 경우 마스터 장치가 비정상적으로 종료되는 문제

## 신규 기능 및 개선

---

1. OSDP 표준화

    - 타사 컨트롤러와 연결할 때 OSDP V2.1.7 프로토콜을 준수하도록 개선

1. 추가 가능한 관리자 수 확대

1. 사용자별 APB 해제 지원

1. 카드 형식별 선택 옵션 지원

1. 최대 층 등급 수 2048개로 확대

1. 이미지 로그 설정에서 지원하는 이벤트 종류 추가

1. 장치 일괄 편집을 이용해 관리자를 추가할 때 새로운 설정이 적용되는 방식 변경

    - 기존: 기존 설정에 새로 설정한 내용을 덮어쓰는 방식

    - 변경: 기존 설정에 새로 설정한 내용을 추가하는 방식

1. 중복 얼굴 검사 지원

1. Wiegand 인증 결과값 출력 옵션을 선택할 수 있도록 지원

    - 사용자 ID 및 카드 ID

1. 출입문에서 안티 테일게이팅(따라 들어가기 방지) 지원

1. OSDP 통신 시 데이터 전송에 실패하면 다시 전송하도록 개선

1. 위조 얼굴 등록 검사 강화 기능 개선

1. 사용성 개선

    - 메뉴 버튼 영역 확대

1. 신규 장치에 대한 슬레이브 연결 지원

    - XPass 2(XP2-MDPB, XP2-GDPB, XP2-GKDPB)

## 버그 수정

---

<BugLists>

1. USB를 통한 펌웨어 업그레이드 시 시간 제한으로 인해 업그레이드를 실패하는 문제

    - 시간 제한을 90초에서 5분으로 변경

1. 삭제한 카드를 장치에 다시 스캔하면 장치가 비정상적으로 재시작되는 문제

1. iCLASS Seos 카드를 CSN 카드로 인식하는 문제

1. BioStation A2에서 내보낸 사용자 데이터를 가져올 경우 장치가 비정상적으로 재시작되는 문제

1. SDK를 이용해 필터링된 로그를 가져올 때 시작 시간이 UTC 기준으로 적용되지 않는 문제

1. 엘리베이터 연결이 끊긴 상태에서 발생한 인증에 대해 재연결한 뒤 릴레이가 동작하는 문제

1. One Device Mode에서 스마트 카드 또는 지문 인증 시 출입 거부되며 사용자 ID가 비정상적으로 출력되는 문제

1. Wiegand 출력 데이터가 중복으로 전송되는 문제

1. 인증 성공 시 출입문 개방에 지연이 발생하며, 사용자 인터페이스의 반응 속도가 전반적으로 느린 문제

</BugLists>
