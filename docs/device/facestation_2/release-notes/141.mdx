# 펌웨어 버전 1.4.1 (빌드 번호1.4.1_210203)

릴리즈<Separator/>2021-02-16

## 신규 기능 및 개선

---

1. 슈프리마 열화상 카메라(TCM10-FS2) SDK Revision

    - FPS 8.3 적용

    - 장치가 비정상적으로 동작이 멈출 때 안전한 복구를 위해 관리용 서비스 분리

    - 열화상 카메라 사용 중 적외선 이미지 동작이 멈추거나 장치가 재부팅되는 문제 수정

    - 캡처 이미지 버퍼 사양 변경

1. 열화상 카메라 일부 기능에 대한 영문 이름 변경

    - 기존: Emission Rate

    - 변경: Emissivity
