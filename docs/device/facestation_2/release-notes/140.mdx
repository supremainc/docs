# 펌웨어 버전 1.4.0 (빌드 번호1.4.0_200918)

릴리즈<Separator/>2020-09-28

## 주요 버그 수정

1. 장치 펌웨어를 업그레이드한 뒤 전체 또는 사용자 데이터를 USB로 내보내고 다시 사용자를 가져오기 할 경우 이벤트 로그가 삭제되는 문제

1. 장치 연결이 해제된 동안 발생한 로그가 장치 재연결 이후에도 BioStar 2에 전송되지 않는 문제

## 신규 기능 및 개선

---

1. 장치 ID 변경 기능 지원

1. 장치 보안 강화

    - 암호화되지 않은 통신 연결 제한

    - 암호화 키 보안 강화

    - 사용자 정보 암호화 및 마이그레이션 지원

1. 안티패스백 구역 규칙이 실제 출입문의 상태에 따라 동작하도록 개선

1. 엘리베이터 제어 시 층별 스케줄 개방 구역 기능 지원

1. 서버 매칭 지원

1. 모바일 출입카드 V1.1 지원

1. 열화상 카메라 지원

1. 화면 보호 기능(Screen Saver) 지원

1. 안경 착용자에 대한 얼굴 인식 알고리즘 개선

1. 신규 얼굴 템플릿 지원

    - 버전 1.4.0 이상의 펌웨어가 설치된 장치에서 스캔하거나 등록한 얼굴 템플릿은 이전 버전 펌웨어를 사용하는 장치에서 호환되지 않아 인증에 실패할 수 있습니다.

    - 슬레이브로 연결한 얼굴 인증 장치의 펌웨어가 최신 버전(FaceStation 2: 1.4.0 이상, FaceLite: 1.2.0 이상)일 경우 기존 템플릿과 호환하지 않아 인증에 실패할 수 있습니다.

1. 최대 사용자 수 (1:N) 확대

    - 기존: 3,000명

    - 변경: 4,000명

1. 신규 장치 연결 지원

    - XPass D2 (Rev 2)

## 버그 수정

---

<BugLists>

1. 슬레이브 장치의 인증 모드가 마스터 장치의 인증 모드에 따라 설정되도록 수정

1. 슬레이브 장치로 연결한 BioStation A2에서 협박 지문으로 인증할 경우 장치가 비정상적으로 종료되는 문제

1. 장치를 타사 시스템과 OSDP로 연결했을 때 카드 데이터가 잘못된 BitCount로 출력되는 문제

1. Wi-Fi 자동 연결이 안 되는 문제 (FS2-AWB)

1. 이미지 로그 색상의 일부 또는 전체가 깨지는 문제

1. 사용자에게 할당된 카드로 작업 코드에 접근할 경우 미등록 사용자로 잘못 인식하는 문제

1. 등록된 사용자 수가 최대일 때 기존에 등록된 사용자의 얼굴을 변경할 경우 장치가 비정상적으로 재부팅되는 문제

1. OM-120, XPass D2를 슬레이브 장치로 연결하면 OSDP 보안 세션 오류가 발생하는 문제

1. 장치에 등록된 얼굴이 최대 개수를 초과하지 않았음에도 신규 사용자 얼굴 등록 또는 기존 사용자 편집 시 최대 개수 초과 오류가 발생하는 문제

1. 등록된 사용자 수가 최대일 때 카드 등록을 시도하면 사용자가 사라지고 장치가 재부팅되는 문제

1. 사용자가 5,000명이고 사용자 그룹을 5개 사용할 때 그룹 매칭을 수행할 경우 오류가 발생하는 문제

1. 마스터 장치에서 그룹 매칭을 설정해도 슬레이브 장치가 그룹 매칭으로 동작하지 않는 문제

1. 신규 펌웨어가 적용된 장치에서 관리자로 지정된 사용자를 재전송할 경우 등록되지 않은 사용자로 인식하여 관리자 메뉴에 진입할 수 없는 문제

1. RS-485 통신 키로 인해 마스터-슬레이브 연동 오류가 나타나는 문제

1. 글로벌 안티패스백 위반 시 인증 성공 로그가 2회 발생하는 문제

1. 슬레이브 장치에서 '잘못된 페이로드 정보입니다.'메시지가 발생한 뒤 비정상적으로 슬레이브 장치의 연결이 끊기고 재연결이 불가능한 문제

1. RS-485 연결 시 잠김 상태의 장치를 재부팅하면 슬레이브 장치가 잠금 해제로 동작하는 문제

1. SDK를 이용해 공장 초기화를 실행할 경우 장치 리소스(로고 이미지)가 초기화되지 않는 문제

1. SSL 보안 통신 중 펌웨어를 업그레이드하거나 사용자를 전송할 때 장치가 재부팅되거나 시간 초과되는 문제

1. 인증 모드 편집, 삭제, 추가가 동작하지 않는 문제

1. 출력 신호 지연이 반복적으로 발생할 경우 정상적으로 동작하지 않는 문제

1. 작업 조건 및 동작에서 연결된 BioStation 2의 출력 포트를 설정할 수 없는 문제

1. 장치가 NTP 서버에 연결을 시도하는 문제

1. 고정 IP 및 서머 모드를 설정했음에도 장치가 DHCP로 동작하는 문제

1. 저조도 및 인증 모드를 설정했을 때 IR 카메라가 비정상적으로 계속 활성화되는 문제

1. 슬레이브 장치에서 스마트 카드 및 얼굴로 인증하면 마스터 장치가 재부팅되는 문제

1. 바이트 순서를 LSB로, Wiegand 출력 정보를 사용자 ID로 설정한 뒤 지문 인증으로 인증하면 장치가 재부팅되는 문제

1. 사용자 얼굴을 등록할 때 장치가 재부팅되는 문제

1. 카드 형식별 옵션을 설정하여 장치가 알 수 없는 카드를 인식할 수 있도록 수정

1. Wiegand 출력 장치에서 카드를 등록한 뒤 다른 사용자에게 등록된 지문으로 Wiegand 출력 시 장치가 정상적으로 동작하지 않는 문제
