1. 지갑 등에서 여러 장의 카드가 겹쳐진 채로 장치에서 스캔하면 장치가 비정상적으로 재부팅되는 문제 <Badge>1.0.0</Badge>

1. 사용자 ID 종류가 영숫자로 설정되어 있을 때 블랙리스트 카드로 설정한 AoC로 인증할 경우 이벤트 로그에 사용자 ID가 비정상적으로 표시되는 문제 <Badge>1.0.0</Badge>

1. 이중 인증을 설정한 장치에서 인증할 때 두 번째 사용자가 인증 실패할 경우 실패 메시지가 2회 표시되는 문제 <Badge>1.5.1</Badge>

1. 이벤트 로그 검색 시 날짜 선택 팝업창이 메뉴 표시 시간이 지나도 사라지지 않고 유지되는 문제 <Badge>1.0.0</Badge>

1. 엘리베이터를 스케줄 개방 구역으로 설정하고 '사용자 인증에 의한 시작' 옵션을 활성화했을 때 출입 그룹에 속한 사용자가 인증하면 층은 정상적으로 활성화되지만, 층 해제 로그가 출력되는 문제 <Badge>1.0.0</Badge>

1. LAN 케이블을 제거하였다가 다시 연결할 경우 DHCP IP를 할당 받는 데 시간이 오래 걸리는 문제 <Badge>1.0.0</Badge>

1. 지속적으로 오인식(FA, False Acceptance)이 발생하는 문제 <Badge>1.5.0</Badge>
