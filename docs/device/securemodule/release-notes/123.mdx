# 펌웨어 버전 1.2.3 (빌드 번호 1.2.3_171229)

릴리즈<Separator/>2018-04-13


## 신규 기능 및 개선

---

1. OSDP Secure Session으로 연결된 상태에서 응답하는 데이터가 존재할 경우 SEC_BLK_TYPE 값을 0x16으로 응답하고, 데이터가 없을 경우 0x18으로 응답

1. 최신 버전의 마스터 디바이스와 OSDP 호환성 검증

    - FS2-AWB/D(V1.1.0)

    - BSA2-OEPW/OHPW/OIPW/OMPW(V1.5.0)

    - BS2-OEPW/OHPW/OIPW/OMPW(V1.6.0)

    - CS40(V1.1.0)

    - BEW2-OAP/ODP/OHP/OAP(UL)(V1.2.0)

    - BSL2-OE/OM(V1.3.0)

    - BEP2-OA/OD/ODS(V1.1.0)
