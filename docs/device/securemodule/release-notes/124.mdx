# 펌웨어 버전 1.2.4 (빌드 번호 1.2.4_181031)

릴리즈<Separator/>2018-11-29


## 신규 기능 및 개선

---

1. OSDP 통신 시 데이터 전송에 실패하면 다시 전송하도록 개선

1. OSDP Secure Session으로 연결된 상태에서 응답하는 데이터가 존재할 경우 SEC_BLK_TYPE 값을 0x16으로 응답하고, 데이터가 없을 경우 0x18으로 응답
