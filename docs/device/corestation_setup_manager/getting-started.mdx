CoreStation Setup Manager는 CoreStation의 네트워크 설정을 관리하거나 슬레이브 장치, 입력 및 출력 포트, Wiegand 포트의 연결 상태를 확인할 수 있는 웹 서버입니다. CoreStation Setup Manager를 사용하면 CoreStation을 이용한 출입통제 시스템을 구성할 때 BioStar X 서버의 위치로 이동하지 않고 현장에서 네트워크 및 배선의 상태를 확인할 수 있습니다.

CoreStation Setup Manager에서 다음과 같은 기능을 사용할 수 있습니다.

* IP 주소 설정

* CoreStation 장치 정보 확인

* 관리자 설정

* 슬레이브 장치 연결 상태 확인

* 입력 및 출력, Wiegand 포트 상태 모니터링

* 장치 초기화, 네트워크 설정을 제외한 설정 초기화

* 공장 초기화

## 최소 요구 사항

호환되는 장치와 펌웨어 버전을 확인하세요.

* CoreStation 펌웨어 버전 1.3.1 이상

* CoreStation 20 펌웨어 버전 1.0.0 이상

:::info

* CoreStation Setup Manager는 CoreStation 펌웨어 버전 1.3.1 이상, CoreStation 20 펌웨어 버전 1.0.0 이상에서 지원됩니다. 낮은 버전의 펌웨어를 사용할 경우 BioStar X에 접속하여 펌웨어를 최신 버전으로 업그레이드하세요.

* 각 장치를 연결하고 설치하는 방법에 대한 자세한 내용은 해당 제품의 가이드를 참고하세요.

  * [**CoreStation**](../corestation_40/index.mdx)

  * [**CoreStation 20**](../corestation_20/index.mdx)

* CoreStation 및 CoreStation Setup Manager에 대한 더 자세한 내용은 슈프리마 기술 지원팀(CS@suprema.co.kr)에 문의하세요.

:::
