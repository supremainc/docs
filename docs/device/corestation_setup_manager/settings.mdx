관리자 계정 및 웹 서버 네트워크 설정 방법을 안내합니다.

## 관리자 계정 편집하기

CoreStation Setup Manager의 관리자 계정을 편집할 수 있습니다.

1. **SETTINGS**를 클릭하세요.

1. **Admin Account** → <IcEdit />**Edit**를 클릭하세요.

    <Image src='/img/device/settings-admin-accounts.png' alone />

1. [장치 초기 설정](initial-setup)을 참고하여 각 항목을 편집한 뒤 **OK**를 클릭하세요.

    <Image src='/img/device/settings-edit-user.png' alone />

1. 설정을 저장하려면 **Apply**를 클릭하세요.

## 웹 서버 네트워크

CoreStation Setup Manager의 네트워크 정보를 변경할 수 있습니다.

1. **SETTINGS**를 클릭하세요.

1. 각 항목을 설정하세요.

    <Image src='/img/device/settings-webserver-network.png' alone />

    <div className='overflow-x col2--20-80'>

    | 명칭 | 기능 설명 |
    | -- | -- |
    | **IP Address** | CoreStation Setup Manager의 IP 주소를 편집하세요.<br/>- 고정 IP만 사용할 수 있습니다. DHCP는 지원되지 않습니다. |
    | **Gateway** | CoreStation Setup Manager의 게이트웨이를 편집하세요. |
    | **Subnet Mask** | CoreStation Setup Manager의 서브넷 마스크를 편집하세요. |

    </div>

1. 설정을 저장하려면 **Apply**를 클릭하세요. 장치가 재시작됩니다.
