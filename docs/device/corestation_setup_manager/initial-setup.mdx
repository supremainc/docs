CoreStation Setup Manager는 웹 기반 서비스를 제공합니다. 따라서 CoreStation을 이용해 중앙 집중형 출입통제 시스템을 구축할 때 배선을 완료한 뒤 어디서나 웹 브라우저를 통해 CoreStation Setup Manager에 접속하여 CoreStation의 연결 상태를 확인할 수 있습니다.

CoreStation Setup Manager를 처음 사용하는 경우, CoreStation을 연결하고 초기 설정을 진행하세요.

1. CoreStation에 케이블 연결을 완료하세요.

1. 웹 브라우저를 실행하세요.

    :::info

    크롬 75 버전 이상인 웹 브라우저를 사용하세요.

    :::

1. 웹 브라우저의 주소 입력 필드에 기본 IP 주소 `https://169.254.0.1:3001`를 입력하세요.

1. 로그인 화면에서 **Download https certification**을 클릭하세요. CoreStation Setup Manager에 처음 접속할 경우 HTTPS를 올바르게 사용하기 위해 인증서를 설치해야 합니다.

    <Image src='/img/device/initial-main.png' width='80%' alone />

1. **cs_client.crt** 파일을 실행하세요.

1. **파일 열기** - **보안 경고** 창이 나타나면 **열기**를 클릭하세요.

1. **인증서** 창에서 **인증서 설치**를 클릭하세요. **인증서 가져오기 마법사**가 실행됩니다.

1. 계속하려면 **다음**을 클릭하세요.

1. 인증서 저장소를 선택한 뒤 **다음** → **마침** → **확인**을 클릭하세요.

1. 사용자 ID와 로그인 비밀번호를 입력하세요. 최초 접속 시의 사용자 ID와 로그인 비밀번호는 모두 'admin'입니다.

1. 관리자 계정을 등록하려면 각 항목을 설정한 후 **OK**를 클릭하세요.

    <Image src='/img/device/initial-setup-new-user.png' alone />

    | 명칭                 | 기능 설명                                                                                            |
    | -------------------- | ---------------------------------------------------------------------------------------------------- |
    | **ID**               | ID는 자동으로 **0**으로 설정되며 변경할 수 없습니다.                                                   |
    | **Name**             | 관리자 이름을 입력하세요.<br/>- 이름은 최대 48자까지 입력할 수 있습니다.                             |
    | **Operator Level**   | Operator Level은 자동으로 **Administrator**로 설정되며 변경할 수 없습니다.                             |
    | **Login ID**         | 로그인 ID를 입력하세요.<br/>- 로그인 ID는 최대 32자까지 입력할 수 있습니다.                          |
    | **Password**         | 로그인 비밀번호를 입력하세요.<br/>- 로그인 비밀번호는 7\~32자의 영숫자와 기호의 조합으로 입력하세요. |
    | **Confirm Password** | 확인을 위해 로그인 비밀번호를 다시 입력하세요.                                                       |

1. 로그인 화면으로 전환됩니다. 등록한 관리자 계정으로 로그인하세요.

:::info

* 관리자 계정은 CoreStation Setup Manager에 최초 접속할 때만 설정할 수 있으며, 한 명의 사용자만 지정할 수 있습니다. 등록이 완료되면 최초 접속 시 사용한 기본 사용자 ID와 로그인 비밀번호는 더 이상 사용할 수 없습니다. 또한, 관리자 계정은 공장 초기화를 포함한 모든 설정을 확인하고 편집할 수 있습니다.

* CoreStation Setup Manager 1.01 버전부터는 **Level 1** 관리자 등급의 명칭이 **Administrator**로 변경되었으며, **Level 2** ~ **Level 3** 등급의 관리자 계정은 더 이상 지원하지 않습니다.

:::
