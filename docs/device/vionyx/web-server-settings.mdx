ViOnyx Web Server에 처음 접속하여 비밀번호를 설정하고 로그인하는 방법을 안내합니다.

## 카메라 웹 서버 접속하기 {#access-web-server}

:::info

카메라와 동일한 네트워크에 연결된 컴퓨터에서 진행하세요.

:::

1. 슈프리마 기술 지원팀(CS@suprema.co.kr)에 문의하여 패키지 파일(*ViOnyxTool_V1_0_1 4.zip*)을 다운로드하세요.

1. 압축을 해제한 후 *ViOnyxTool.exe*를 실행하세요.

   <Image src="/img/camera/web-server-settings1.png" width='80%' alone />

   :::info
   
   1. 필수 프로그램이 설치되어 있지 않은 경우, 해당 프로그램을 다운로드할 수 있는 페이지로 연결됩니다.

      <Image src="/img/camera/web-server-settings2.png" width='50%' alone />

   1. 필요한 프로그램을 설치한 후 *ViOnyxTool.exe*를 다시 실행하세요.

   :::

1. **Search for Cameras**를 클릭하세요.

   <Image src="/img/camera/web-server-settings3.png" width='80%' alone />

1. 검색된 카메라 목록에서 접속할 카메라의 시리얼 넘버(**DeviceID**)를 찾아 IP 주소(**IP Address**)를 확인하세요.

   <Image src="/img/camera/web-server-settings4.png" width='80%' alone />

1. 웹 브라우저를 실행하세요.

1. 주소 표시줄에 카메라의 IP 주소(**IP Address**)를 입력한 후 **Enter**를 누르세요.

1. ViOnyx Web Server 화면이 나타납니다.

## 비밀번호 설정하기

ViOnyx Web Server에 최초 접속 시 비밀번호를 설정하세요.

1. [카메라 웹 서버 접속하기](#access-web-server)를 참고하여 웹 브라우저에서 ViOnyx Web Server의 IP 주소로 접속하세요.

   <Image src="/img/camera/password-setting1-vionyx.png" />

1. <Cmd>비밀번호</Cmd> 입력란에 비밀번호를 입력하세요.

1. <Cmd>비밀번호 확인</Cmd> 입력란에 동일한 비밀번호를 다시 입력하세요.

1. 입력한 비밀번호가 다음 요구 사항을 모두 충족하는지 확인하세요.

   :::note

   **비밀번호 요구 사항**

   * **길이**: 12자 이상 64자 이하로 입력하세요.

   * **문자 조합**: 영문 대문자(A-Z), 영문 소문자(a-z), 숫자(0-9), 특수문자(`"` `!` `@` `#` `$` `%` `^` `&` `*` `(` `)` `-` `_` `=` `+` `[` `]` `{` `}`)를 각각 1개 이상 포함해야 합니다.

      * 공백, 한글 및 기타 유니코드 문자는 사용할 수 없습니다.

   * **보안 규칙**: 보안 취약성을 방지하기 위해 다음 패턴은 사용할 수 없습니다.

      * 4자 이상의 연속된 문자나 숫자 (예: abcd, 1234, 4321)

      * 4회 이상 반복되는 동일한 문자나 숫자 (예: aaaa, 1111)

      * 사용자 ID 포함 (대소문자 구분 없음)

      * 유추하기 쉬운 일반적인 단어나 패턴 (예: password, qwerty)

   * **비밀번호 확인**: <Cmd>비밀번호</Cmd>와 <Cmd>비밀번호 확인</Cmd> 입력란의 값이 정확히 일치해야 합니다.

   :::

1. 모든 조건이 충족되면 <Cmd>제출</Cmd> 버튼이 활성화됩니다. 버튼을 클릭하여 비밀번호 설정을 완료하세요.

:::caution

비밀번호를 분실하지 않도록 주의하세요. 비밀번호를 분실한 경우, 카메라를 초기화해야 합니다.

:::

## 로그인하기

비밀번호 설정이 완료되면 로그인 페이지로 자동 이동합니다. 관리자 아이디와 설정한 비밀번호를 입력하여 ViOnyx Web Server에 접속하세요.

<Image src="/img/camera/password-setting2-vionyx.png" />

1. <Cmd>아이디</Cmd> 입력란에 **admin**을 입력하세요.

   * 관리자 아이디(admin)는 <Cmd>시스템</Cmd> → <Cmd>계정 관리</Cmd>에서 변경할 수 있습니다.

1. <Cmd>비밀번호</Cmd> 입력란에 설정한 비밀번호를 입력하세요.

1. <Cmd>로그인</Cmd> 버튼을 클릭하세요. 로그인에 성공하면 ViOnyx Web Server의 카메라 라이브 뷰어 화면이 나타납니다.

1. [ViOnyx Web Server 사용자 설명서](#)를 참고하여 카메라를 설정하세요.
