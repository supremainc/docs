카메라 웹 서버 접속 방법과 관리자 로그인·비밀번호 설정 및 보안 주의사항을 안내합니다.

:::info

최초 관리자 ID는 “admin”이며, 비밀번호는 최초 로그인 시 설정해야 합니다. 안전하게 개인정보를 보호하고, 개인정보도용 피해를 예방하기 위해 3개월 마다 주기적으로 비밀번호를 변경하세요. 부주의한 비밀번호 관리로 인한 보안 및 기타 문제의 책임은 사용자에게 있으므로 각별히 주의하세요.

:::

## 카메라에 접속하기

:::info

카메라와 동일한 네트워크에 연결된 컴퓨터에서 진행하세요.

:::

1. 슈프리마 기술 지원팀(CS@suprema.co.kr)에 문의하여 패키지 파일(*ViOnyxTool_V1_0_1 4.zip*)을 다운로드하세요.

1. 압축을 해제한 후 *ViOnyxTool.exe*를 실행하세요.

   <Image src="/img/camera/web-server-settings1.png" width='80%' alone />

   :::info
   
   1. 필수 프로그램이 설치되어 있지 않은 경우, 해당 프로그램을 다운로드할 수 있는 페이지로 연결됩니다.

      <Image src="/img/camera/web-server-settings2.png" width='50%' alone />

   1. 필요한 프로그램을 설치한 후 *ViOnyxTool.exe*를 다시 실행하세요.

   :::

1. **Search for Cameras**를 클릭하세요.

   <Image src="/img/camera/web-server-settings3.png" width='80%' alone />

1. 검색된 카메라 목록에서 접속할 카메라의 시리얼 넘버(**DeviceID**)를 찾아 IP 주소(**IP Address**)를 확인하세요.

   <Image src="/img/camera/web-server-settings4.png" width='80%' alone />

1. 웹 브라우저를 실행하세요.

1. 주소 표시줄에 카메라의 IP 주소(**IP Address**)를 입력한 후 **Enter**를 누르세요.

1. 로그인 화면이 나타납니다.

## 비밀번호 설정하기

최초 제품 접속 시 반드시 로그인 비밀번호를 등록해야 합니다.

* **길이**: 12자 이상 64자 이하로 입력하세요.

* **문자 조합**: 영문 대문자(A-Z), 영문 소문자(a-z), 숫자(0-9), 특수문자(`"` `!` `@` `#` `$` `%` `^` `&` `*` `(` `)` `-` `_` `=` `+` `[` `]` `{` `}`)를 각각 1개 이상 포함하세요.

   :::info
   
   공백, 한글 및 기타 유니코드 문자는 사용할 수 없습니다.

   :::

* **보안 규칙**: 보안 취약성을 방지하기 위해 다음 패턴은 사용할 수 없습니다.

   * 4자 이상의 연속된 문자나 숫자 (예: abcd, 1234, 4321)

   * 4회 이상 반복되는 동일한 문자나 숫자 (예: aaaa, 1111)

   * 사용자 ID 포함 (대소문자 구분 없음)

   * 유추하기 쉬운 일반적인 단어나 패턴 (예: password, qwerty)

* **비밀번호 확인**: **Password**와 **Confirm Password** 입력란에 대소문자를 구분하여 동일한 비밀번호를 입력하세요.

:::caution

* 보안강화를 위하여 동일한 문자 반복 사용이나 연속된 키보드 배열을 비밀번호로 사용하지 않는 것을 권장합니다.

* 장치 비밀번호를 분실하지 않도록 주의하세요. 비밀번호를 분실한 경우, 카메라를 초기화해야 합니다.

:::

## 로그인하기

카메라에 접속할때마다 로그인 창이 뜹니다. 설정된 사용자 이름과 암호를 입력해야 카메라에 접속할 수 있습니다.

1. **사용자 이름** 입력란에 **admin**을 입력하세요.

   * 관리자 아이디 'admin'은 웹 서버에서 변경할 수 있습니다.

1. **비밀번호** 입력란에 설정한 암호를 입력하세요.

1. **로그인** 버튼을 클릭하세요. 로그인이 완료되면 카메라 라이브 뷰어 화면이 나타납니다.

1. [웹 뷰어 사용자 설명서](#)를 참고하여 카메라를 설정하세요.

:::info

* 카메라 웹뷰어 접속시 영상정보가 암호화 되어있지 않으니 보안에 각별히 주의하시길 바랍니다.

* 화면비율이 100% 일 때 가장 좋은 화면을 볼 수 있습니다. 화면비율을 줄일 경우 화면이 잘려져서 보일 수 있습니다.

:::
