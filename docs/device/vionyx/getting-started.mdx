## 각 부분의 명칭 및 기능

### 내부 포트 구성

<Image src='/img/camera/name-function-each-part-vionyx.svg' className='none' />

* **알람 입력, 릴레이 (5핀)**: 알람 입력과 릴레이 출력 기능이 결합된 5핀 케이블을 연결하세요.

* **오디오 입출력 (4핀)**: 오디오 입출력 케이블을 연결하세요.

* **이더넷 (PoE+)**: 이더넷 케이블을 연결하세요. PoE 전원 공급을 지원합니다.

* **전원, RS-485 (5핀)**: 전원 공급과 RS-485 통신 기능이 결합된 5핀 케이블을 연결하세요.

* **초기화 버튼**: 모든 설정을 공장 기본값으로 초기화합니다. 자세한 내용은 [공장 초기화](installation#factory-reset)를 참고하세요.

* **Micro SD 카드 슬롯**: Micro SD 카드를 삽입하세요.

* **USB (Type-A)**: Wi-Fi, BLE 등 USB (Type-A) 기반 액세서리를 삽입하세요.

* **마이크 스위치**: 오디오 녹음 기능을 제어하세요.

### LED 상태 표시등

카메라 커버의 LED 상태 표시등은 색상에 따라 제품의 작동 상태를 나타냅니다.

<Image src='/img/camera/led-status-indicator-vionyx.svg' className='none' alone />

| LED | 상태 |
| :--: | -- |
| 초록색 | 인증 성공 |
| 빨간색 | 인증 실패 |
| 파란색 | 정상 동작 상태 |
| 파란색/초록색 점멸 | <Cmd sid='ui_012' product='dev' />를 사용으로 설정 시 IP 주소를 받아오지 못하는 상태 |

### 마이크 스위치 설정

마이크 스위치를 조작하여 카메라에서 마이크를 물리적으로 차단하고 오디오가 녹음되지 않도록 합니다.

<Image src='/img/camera/microphone-switch-vionyx.svg' className='none' alone />

* 오디오 녹음 기능은 기본적으로 활성화되어 있습니다.

* 오디오 녹음을 차단하려면 마이크 스위치를 왼쪽으로 밀어 마이크를 비활성화하세요.

* Command 계정에서도 오디오 녹음을 비활성화할 수 있습니다.
