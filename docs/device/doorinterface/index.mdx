Door Interface는 안정적인 출입 통제를 위해 설계된 강력한 2도어, 4리더 인터페이스입니다. CoreStation 시리즈와 간편하게 연동되며, 최소한의 배선으로 RS-485 통신을 통한 유연한 도어 제어가 가능합니다. 또한, 오프라인 상태에서도 안정적인 출입 통제를 제공합니다.
