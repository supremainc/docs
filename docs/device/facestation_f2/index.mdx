FaceStation F2는 차세대 퓨전 얼굴 인증 기술을 탑재한 멀티모달 출입통제 단말기로, 포스트 코로나 시대의 비접촉 보안 솔루션 요구에 최적화된 제품입니다. 슈프리마의 독자적인 퓨전 매칭 기술은 비주얼 인식과 적외선(IR) 인식을 동시에 활용하여 업계 최고 수준의 인증 정확도를 실현하며, 오인식률(FAR) 100억분의 1이라는 탁월한 성능을 자랑합니다.
