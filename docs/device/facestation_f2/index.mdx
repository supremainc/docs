FaceStation F2는 차세대 퓨전 얼굴 인증 기술을 탑재한 멀티모달 출입통제 단말기로, 포스트 코로나 시대의 비접촉 보안 솔루션 요구에 최적화된 제품입니다. 슈프리마의 독자적인 퓨전 매칭 기술은 비주얼 인식과 적외선(IR) 인식을 동시에 활용하여 업계 최고 수준의 인증 정확도를 실현하며, 오인식률(FAR) 100억분의 1이라는 탁월한 성능을 자랑합니다.

## 핵심 특징

**퓨전 얼굴 인증 기술**

- 비주얼 카메라와 적외선 카메라의 융합 매칭으로 최강의 얼굴 인증 성능 구현

- 딥러닝 알고리즘 기반으로 인종, 표정 변화에 상관없이 원활한 얼굴 인증 지원

- 0.5초 이내의 빠른 인증 속도와 강력한 위조 얼굴 검출 기능

**대규모 인증 환경 지원**

- 최대 100,000명의 사용자 등록 및 대규모 인증 처리 가능(지문 인증 시 최대 100,000명, 얼굴 인증 시 최대 50,000명)

- 최대 5,000,000개의 텍스트 로그와 50,000개의 이미지 로그 저장

- 1.8GHz 듀얼코어 + 1.4GHz 쿼드코어 CPU와 16GB 플래시 메모리로 고성능 보장

**포스트 코로나 대응 솔루션**

- 비접촉 얼굴 인증 방식으로 감염 위험 최소화

- 마스크 착용 여부 감지 및 미착용자 출입 제한

- 마스크 착용 상태에서도 정확한 얼굴 인증 가능

- 열화상 카메라 연동을 통한 발열 감지 기능으로 방역과 보안 동시 수행

**강력한 보안 및 암호화**

- 바이오인식 정보를 포함한 모든 사용자 정보 암호화

- Secure Booting 기능으로 향상된 장치 보안성 제공

- 크립토칩 탑재로 최고 수준의 데이터 보호

**다양한 인증 수단 지원**

- 얼굴, 지문, 카드, 모바일 출입카드 등 멀티모달 인증 지원

- Dual-frequency 실물 카드, NFC/BLE 기반 모바일 카드, ToC(Template-on-Card) 카드 호환

- 125kHz EM, 13.56MHz MIFARE, DESFire, FeliCa, HID Prox, iCLASS 등 다양한 카드 포맷 지원

**향상된 사용 편의성**

- 7인치 IPS 컬러 LCD 화면으로 직관적인 사용자 인터페이스 제공

- 워크스루(walkthrough) 인증 기능으로 자연스러운 통과 방식 인증

- 인증 범위 130cm로 확대하여 사용 편의성 극대화

- 적외선 인식 기술로 어두운 환경에서도 안정적인 얼굴 인증

**견고한 설계 및 환경 적응성**

- IP65 등급의 방진, 방수 기능으로 다양한 환경에서 안정적 운영

- -20°C ~ 50°C 동작 온도 범위로 극한 환경 대응

- 세련된 디자인과 컴팩트한 크기(119.8 x 233 x 23.5mm)로 설치 공간 최적화
