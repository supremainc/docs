# 펌웨어 버전 2.0.2 (빌드 번호 2.0.2_220502)

릴리즈<Separator/>2022-05-10

## 신규 기능 및 개선

---

1. 비주얼 페이스 알고리즘 v2 적용

1. 얼굴 등록 시 마스크를 감지하여 마스크를 착용하거나 입을 가린 상태로 얼굴을 등록할 수 없도록 개선

1. 열화상 카메라 사용 시 동적 ROI (동적 발열 측정 영역) 성능 개선

1. 마이그레이션 시 화면에 진행률 표시줄(Progress bar) 추가

1. 서버에서 장치로 사용자 데이터를 전송할 때 사용자에게 등록된 카드에 대해 중복 여부를 체크하도록 개선

1. 장치의 전원이 끊기더라도 경비/해제 상태가 유지되도록 개선
