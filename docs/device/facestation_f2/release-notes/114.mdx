# 펌웨어 버전 1.1.4 (빌드 번호 1.1.4_220315)

릴리즈<Separator/>2022-03-22

:::caution

장치 부팅이 안 되는 문제를 방지하기 위해 펌웨어를 최신 버전으로 업그레이드하기 전 반드시 '데이터 삭제 후 장치 동기화'를 진행하세요.

:::

## 신규 기능 및 개선

---

1. 전면 커버 (LCD 디스플레이 및 터치패드) 이원화에 따른 펌웨어 업그레이드 방식 변경

1. BioStar 2 또는 BioStar 2 Device SDK에서 비주얼 페이스 템플릿 전송 시 템플릿 버전(알고리즘 호환성)을 확인하여 필요할 경우에만 warp 영상에서 템플릿을 새로 추출하도록 개선

## 버그 수정

---

<BugLists>

1. 장치의 플래시 메모리에서 구조적 문제 확인 <Badge>1.1.2 이하</Badge>

    - 디스크 비교 용량 차이로 인해 장치 부팅이 불가한 문제

1. 펌웨어를 업그레이드한 뒤 사용자가 터치패드에 실제 터치한 위치와 다른 부분이 인식되는 문제 <Badge>1.1.3</Badge>

1. 마스터 장치를 재부팅하면 슬레이브 장치의 연결이 끊어지는 문제 <Badge>1.1.2 이하</Badge>

1. 간헐적으로 iCLASS Seos 카드 인식에 실패하고, 장치를 재시작해야 해당 카드를 인식하는 문제 (FSF2-AB) <Badge>1.1.2 이하</Badge>

1. 장치에서 BioStar 2 서버로 사용자 데이터를 동기화한 뒤 장치로 재전송할 때 일부 사용자만 전송되거나, 전송에 시간이 오래 걸리는 문제 <Badge>1.1.2 이하</Badge>

    - 품질이 낮은 비주얼 페이스 warp 이미지가 다른 장치로 동기화될 때 해당 장치에서 비주얼 페이스 템플릿 추출에 실패하고, 지속적으로 동기화를 재시도하여 위와 같은 문제 발생

1. 작업 조건 및 동작에서 '출입 성공' 이벤트를 설정한 경우 인증 시 온도가 'NaN'으로 출력되는 문제 <Badge>1.1.0 이상, 1.1.2 이하</Badge>

</BugLists>
