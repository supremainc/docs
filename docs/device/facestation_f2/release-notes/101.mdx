# 펌웨어 버전 1.0.1 (빌드 번호 1.0.1_201110)

릴리즈<Separator/>2020-11-13

## 신규 기능 및 개선

---

1. 비주얼 페이스 매칭 알고리즘 개선

    - 사진으로 얼굴을 등록한 뒤 빠른 인증 모드를 설정하고 얼굴을 인증할 경우 첫 번째 템플릿만 업데이트하는 문제 개선

    - 사진으로 얼굴을 등록한 뒤 퓨전 매칭 모드를 설정하고 얼굴을 인증할 경우 적외선 템플릿의 품질 문제로 인해 인증 실패가 발생하는 문제 개선

    - 사진으로 얼굴을 등록한 뒤 역광이 있는 어두운 환경에서 얼굴을 인증할 경우 인증 실패가 발생하는 문제 개선

    - 마스크 검출 알고리즘 개선 및 사용성 향상

1. 화면 보호기 사용 여부 선택 옵션 지원

1. 슈프리마 열화상 카메라 ROI 기본 설정 값 변경

    - ROI X(%) : 30

    - ROI Y(%) : 25

    - ROI 폭(%) : 50

    - ROI 높이(%) : 55

    - 펌웨어 1.0.0 버전에서 업그레이드한 펌웨어에서는 변경된 ROI 기본 설정 값이 자동으로 적용되지 않으며 위 기본 값을 수동으로 설정해야 합니다.

## 버그 수정

---

<BugLists>

1. 슈프리마 열화상 카메라를 사용할 때 작업 조건 및 동작에서 설정한 입력 신호에 대한 동작 내용과 다른 동작이 발생하는 문제

1. BioStar 2 Device SDK를 이용해 시간을 설정한 경우 장치의 시간이 비정상적으로 표시되는 문제

1. BioStar 2 Device SDK를 이용해 FaceStation 2 또는 FaceLite의 얼굴 템플릿을 장치로 전송할 경우 장치가 비정상적으로 종료되는 문제

1. 보안 탬퍼 발생 등의 이유로 장치 데이터베이스가 초기화 된 뒤 BioStar 2의 장치 상세 페이지 진입 시 고급 탭의 텍스트가 비정상적으로 출력되는 문제

1. Airfob Pass에서 직접 장치를 등록할 때 다이나믹 사이트에서는 등록이 불가능한 문제

1. 이미지 로그 기능이 정상적으로 동작하지 않는 문제

1. 슈프리마 열화상 카메라를 사용하고 Wiegand 출력 모드를 항시 통과로 설정한 경우, 열화상 카메라의 확인 모드 설정에 따라 항시 통과가 동작하도록 수정

1. 슬레이브로 연결한 XPass D2 (Rev 2)의 펌웨어 업그레이드가 정상적으로 동작하지 않는 문제

</BugLists>
