:::caution

* 펌웨어를 v2.x.x로 업그레이드한 뒤 하위 버전(v1.x.x) 펌웨어로 다운그레이드 할 수 없습니다.

* 펌웨어를 v1.x.x에서 v2.x.x로 업그레이드할 경우 BioStar 2 또는 BioStar Device SDK를 이용할 것을 권장합니다.

* 부득이 USB 메모리를 이용하여 펌웨어를 업그레이드할 경우 먼저 **화면과 소리** 메뉴에서 **메뉴 표시 시간**을 **항상 켜짐**으로 설정한 뒤 업그레이드를 진행하세요. 또한 업그레이드 진행 중 실패 메시지가 출력되더라도 USB 메모리를 제거하지 말고 성공 메시지가 출력될 때까지 대기하세요. 업그레이드가 완료되면 장치가 자동으로 다시 시작됩니다.

:::
