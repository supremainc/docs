# 펌웨어 버전 1.0.2 (빌드 번호 1.0.2_210104)

릴리즈<Separator/>2021-01-05

## 신규 기능 및 개선

---

1. 열화상 카메라 사용성 향상

    - 최저 기준 온도 설정 기능 추가

    - ROI 영역에 대한 사용자 인터페이스 개선

    - 실온 및 저온 환경의 온도 편차 개선

    - 온도 확인 화면에서 온도 바 제거

    - 적외선 방사율 영문 기능 이름 변경

1. 열화상 카메라 및 마스크 검출 시 사용자 인터페이스 개선

    - 안내 문구, 신호음, 음성 가이드 변경

1. 모바일 출입카드와 일반 RFID 카드의 이벤트 로그 분리

1. 슬레이브 장치에 화면 보호기 사용 여부 선택 옵션 지원

1. 비주얼 페이스 매칭 알고리즘 개선

    - 열화상 카메라 기능의 확인 모드를 '확인만'으로 설정했을 때 얼굴 인증 화면에 감지 UI가 표시되지 않는 문제 개선

    - 마스크 착용 시 인증 성능 개선

    - 역광, 외부광, 저조도 사용 환경에 대한 인증 성능 개선

    - 위조 얼굴 검출 성능 향상

## 버그 수정

---

<BugLists>

1. BioStar 2 Device SDK를 이용해 장치에 BS2AuthConfig를 호출할 경우 장치가 동작하지 않는 문제

1. 마스터-슬레이브로 연결했을 때 슬레이브 장치가 비정상적으로 재시작되는 문제

1. Secure I/O 2를 연결했을 때 RS-485 연결 끊김 로그가 지속적으로 발생하는 문제

1. 마스크 검출 및 열화상 카메라 기능의 확인 모드를 '확인만'으로 설정했을 때 동적 ROI를 설정하면 마스크 검출 및 온도 측정 동작이 원활하지 않는 문제

1. 열화상 카메라를 '사용'에서 '사용 안 함'으로 변경해도 열화상 카메라 화면이 유지되는 문제

1. '장치 → 서버'로 통신 연결을 설정할 때 장치가 비정상적으로 종료되는 문제

1. 장치와 DM-20을 마스터-슬레이브로 연결했을 때 장치를 재시작할 경우 DM-20에 연결된 Wiegand 리더가 Lock에서 Unlock으로 동작하는 문제

1. 출입문에 수동 개방이 설정된 상태에서 장치를 재시작할 경우 재시작이 완료되어도 출입문이 잠겨 열리지 않는 문제

1. 경비 구역을 설정했을 때 장치의 위조 얼굴 검출 화면에 의해 알람 해제 버튼이 가려지는 문제

1. 기본 해시 키에 의해 보안 탬퍼를 수동으로 끌 수 없는 문제

1. 보안 취약점 개선을 위한 USB 키보드 연결 제한

1. AoC에 등록한 지문을 변경한 뒤 장치에 전송했을 때 AoC 지문에 대해 카드 없이 지문으로 인증이 지속적으로 가능한 문제

1. 이벤트 로그에 이미지 로그 및 개수가 정상적으로 표시되지 않는 문제

1. 인증 유효 거리보다 근접하여 얼굴 인증을 시도할 경우 장치가 비정상적으로 재시작되는 문제

</BugLists>
