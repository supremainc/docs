# 펌웨어 버전 1.0.4 (빌드 번호 1.0.4_210303)

릴리즈<Separator/>2021-03-10

## 신규 기능 및 개선

---

1. 신규 LCD 디스플레이 적용

    - 버전 1.0.3 이하 펌웨어가 설치된 장치의 경우 반드시 펌웨어를 최신 버전으로 업그레이드한 뒤 신규 LCD 디스플레이 교체를 진행하세요. LCD 디스플레이를 먼저 교체한 뒤 펌웨어를 버전 1.0.4 이상으로 업그레이드할 경우 터치 오동작이 발생할 수 있습니다.

1. 슈프리마 열화상 카메라(TCM10-FSF2) SDK Revision

    - FPS 8.3 적용

    - 장치가 비정상적으로 동작이 멈출 때 안전한 복구를 위해 관리용 서비스 분리

    - 열화상 카메라 사용 중 적외선 이미지 동작이 멈추거나 장치가 재부팅되는 문제 수정

    - 캡처 이미지 버퍼 사양 변경

## 버그 수정

---

<BugLists>

1. 열화상 카메라의 펌웨어 버전이 표시되지 않는 문제

1. 근태 코드 설정을 변경해도 이전 설정이 유지되는 문제

1. 인증 모드를 카드+PIN으로 설정한 슬레이브 장치에서 장치 기본 인증 모드 미포함으로 설정한 사용자가 카드 인증을 시도할 경우 시간 초과가 발생하는 문제

1. BioStar 2를 통해 리소스를 업데이트하면 열화상 카메라 동작이 멈춘 상태로 유지되는 문제

1. 비주얼 페이스 매칭 알고리즘 개선

    - 안경 착용자에 대한 얼굴 인식 알고리즘 개선

    - 마스크 착용 시 인증 성능 개선

</BugLists>
