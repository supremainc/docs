# 펌웨어 버전 1.1.1 (빌드 번호 1.1.1_210623)

릴리즈<Separator/>2021-06-25

## 신규 기능 및 개선

---

1. 비주얼 페이스 알고리즘 개선

    - 신규 비주얼 페이스 알고리즘은 펌웨어 버전 1.0.5 이하에서 등록한 비주얼 페이스 템플릿과 호환하지 않습니다. 펌웨어를 버전1.0.5 이하에서 버전 1.1.0 이상으로 업그레이드할 경우 기존에 등록된 얼굴 템플릿을 추출하기 위한 마이그레이션이 진행되며, 이 작업은 등록된 사용자 수에 따라 최대 10시간 정도 걸릴 수 있습니다.

    - 버전 1.0.5 이하에서 발급한 얼굴 데이터가 저장된 AoC는 버전 1.1.0 이상 펌웨어가 적용된 장치에서 호환되지 않습니다.

    - 펌웨어를 버전 1.1.0 이상으로 업그레이드할 경우 버전 1.0.5 이하로 다운그레이드할 수 없습니다.

1. 재실 인원 제한 구역 지원

1. 열화상 카메라 사용성 향상

    - 온도 측정 거리 및 사용자의 움직임에 대한 음성 안내 추가

    - ROI 가이드 영역에 대한 사용자 인터페이스 개선

    - ROI 설정에 따른 가이드 영역 표시 구분

    - ROI 설정 기본 값을 동적 ROI에서 기본 ROI로 변경

    - 온도 측정 실패 시 안내 메시지 추가

1. 조명 밝기 조절 기능 명칭 변경 및 '사용 안 함' 옵션 추가

    - 기능 이름 변경: '주변 밝기(Ambient Brightness)'에서 '조명 밝기(LED Brightness)'로 변경

    - '사용 안 함' 옵션을 추가하여 장치의 조명을 끌 수 있도록 개선

    - '자동' 옵션 삭제

1. 기본 해시 키를 설정해도 보안 탬퍼를 수동으로 켜거나 끌 수 있도록 개선

1. 얼굴 인증 시간 간격을 2.5초로 변경

1. USB 내보내기 시 2 GB 이상의 크기가 큰 파일은 분할 후 압축하여 압축 해제 시 오류가 발생하지 않도록 개선

## 버그 수정

---

<BugLists>

1. 장치를 타사 컨트롤러와 OSDP 로 연결했을 때 컨트롤러의 Secure Communication 모드를 활성화하면 RS-485 통신이 정상적으로 동작하지 않는 문제 <Badge>1.0.5 이하</Badge>

1. 너무 밝거나 어둡거나 역광이 비치는 환경으로 인해 사용자 얼굴 인증에 실패할 경우 화면의 밝기가 밝아지고, 이 상태가 지속되는 문제 <Badge>1.0.4 이하</Badge>

1. 펌웨어를 버전 1.0.4 이상으로 업그레이드할 경우 열화상 카메라가 정상적으로 동작하지 않는 문제 <Badge>1.0.4 이하</Badge>

1. USB 내보내기로 로그를 내보낼 때 사용자 등록 여부에 따라 소요 시간에 차이가 발생하는 문제 <Badge>1.1.0 이하</Badge>

1. 인증 모드를 ID+PIN으로 설정하였을 때 간헐적으로 관리자 메뉴가 표시되지 않는 문제 <Badge>1.0.5 이하</Badge>

1. 이중 인증이 설정된 슬레이브 장치에서 두 번째 사용자 인증 성공 시 화면이 비정상적으로 표시되는 문제 <Badge>1.1.0 이하</Badge>

1. BioStar 2와의 연결이 끊어진 뒤 다시 연결될 때 상황에 맞지 않는 팝업 메시지가 표시되는 문제 <Badge>1.0.5 이하</Badge>

1. 설정된 인증 모드의 순서에 따라 협박 지문 인증 로그가 발생되지 않는 문제 <Badge>1.0.5 이하</Badge>

1. 엘리베이터에 스케줄 개방 구역을 설정한 뒤 마스터 장치를 재부팅할 경우 릴레이가 Off(Lock)로 동작하는 문제 <Badge>1.1.0 이하</Badge>

1. USB 내보내기 시 데이터베이스의 전체 파일이 삭제되는 문제 <Badge>1.0.5 이하</Badge>

1. 비주얼 페이스 매칭 알고리즘 개선 <Badge>1.1.0 이하</Badge>

    - 마스크 착용 시 인증 성능 개선

    - 안경 착용자에 대한 얼굴 인식 알고리즘 개선
