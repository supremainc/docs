## 각 부분의 명칭 및 기능

<Magnify src='/img/device/name-function-each-part-dm20.png' width={800} zoomFactor={.4} mgWidth={500} mgHeight={500} />

:::info

* 장치와 연동된 제품을 초기화한 뒤 다른 장치에 연결하려면 **INIT** 버튼을 2초 이상 누르세요.

* 제품을 재부팅하려면 **RESET** 버튼을 누르세요.

* Supervised Input을 사용할 때 아래 LED 상태를 참고하세요.

  * 주황색: 단락

  * 빨간색: 합선

  * 초록색: On

  * 꺼짐: Off

:::

### 설치 예제

이 제품은 다양한 입출력과 Wiegand 연결을 지원하며, 슈프리마 장치와 타사 장치를 연동하여 확장성을 높일 수 있습니다.

Supervised Input도 사용할 수 있으며 Supervised Input은 단락, 합선, On, Off 상태를 확인할 수 있습니다.
