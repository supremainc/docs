ViOnyx Web Server에 로그인하면 상단에 주요 메뉴가 표시됩니다. 각 메뉴를 통해 원하는 기능에 빠르게 접근할 수 있습니다. 이 페이지에서는 웹 서버 인터페이스의 구조와 각 메뉴의 역할을 안내합니다.

<Image src='/img/cam-server/vionyx-home-screen.png' />

## 메인 메뉴 바로가기

화면 왼쪽 상단에는 ViOnyx Web Server의 로고와 함께 주요 메뉴 아이콘이 있습니다. 각 메뉴에서 제공하는 기능은 아래와 같습니다.

<div className='menu-icon-table'>

| 메뉴 아이콘 | 메뉴 이름 | 설명 |
| :-: | :-: | --- |
| <Image src='/img/cam-server/ico-header-live.svg' width='30px' alone className='none' /> | <Cmd sid='live' product='cam' /> | 실시간으로 카메라에서 촬영 중인 영상을 확인할 수 있습니다. |
| <Image src='/img/cam-server/ico-header-settings.svg' width='30px' alone className='none' /> | <Cmd sid='settings' product='cam' /> | 카메라와 웹 서버의 설정값을 변경할 수 있습니다. |
| <Image src='/img/cam-server/ico-header-people.svg' width='30px' alone className='none' /> | <Cmd sid='people' product='cam' /> | 카메라 시스템의 사용자를 추가하고 관리할 수 있습니다. |
| <Image src='/img/cam-server/ico-header-system.svg' width='30px' alone className='none' /> | <Cmd sid='system' product='cam' /> | 장치 정보, 계정, 오디오, 로그 등 전체 시스템을 관리할 수 있습니다. |

</div>

화면 오른쪽 상단에는 카메라 정보와 언어 선택, 사용자 가이드 바로가기, 로그아웃 버튼이 있습니다.

<div className='menu-icon-table'>

| 메뉴 아이콘 | 메뉴 이름 | 설명 |
| :-: | :-: | --- |
| <Image src='/img/cam-server/ico-header-camera.svg' width='30px' alone className='none' /> | 카메라 정보 | 현재 접속한 카메라의 이름과 상태, IP 주소를 확인할 수 있습니다. |
| <span style={{fontSize: '18px'}}>🌐</span> | 언어 선택 | 웹 인터페이스의 언어를 변경할 수 있습니다. |

</div>
