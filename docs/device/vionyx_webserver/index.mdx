**ViOnyx Web Server**는 ViOnyx 카메라에 내장된 웹 기반 관리 서버입니다. 별도의 VMS 없이 웹 브라우저에서 실시간 영상 확인, AI 이벤트 운영, 사용자 관리, 시스템 설정을 수행할 수 있습니다.

360도 Fisheye 기반 영상과 AI 분석을 결합해, 사람의 이동과 이벤트를 현장에서 빠르게 파악하고 대응할 수 있도록 지원합니다.

### 신원 인식 기반 카메라 내 추적

일반 Re-ID는 회전, 앉기, 재킷 교체에 피사체를 놓칩니다. ViOnyx는 모든트랙을 얼굴 인증된 ID에 고정해, 동일 인물을 화면 전반에서 일관되게 유지합니다.

### 프라이버시 우선, 엣지 전용 처리

얼굴은 장치 안에서 매칭되며 원본 데이터는 외부로 나가지 않습니다. 다운스트림에는 신원 및 이벤트 메타데이터만 전달됩니다. 생체 정보는 기본 비활성화되어 있으며, 관리자가 사전 동의(Opt-in)해야 하며, 완전한 감사 로그를 제공합니다.
