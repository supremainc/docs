ViOnyx 카메라를 처음 설치한 후, 웹 브라우저에서 카메라의 IP 주소로 접속할 수 있습니다. 초기 로그인 시 보안을 위해 비밀번호를 설정해야 하며, 설정 완료 후 로그인할 수 있습니다.

:::info

공장 초기화 후 IP 주소를 할당받지 못했다면 기본 IP 주소(`https://192.168.1.100:8443`)로 변경되므로 웹 브라우저에서 해당 주소로 접속하세요.

:::

## 비밀번호 설정

처음 로그인할 때는 보안을 위해 강력한 비밀번호를 설정해야 합니다. ViOnyx는 강력한 비밀번호 정책을 적용하여 시스템 보안을 보장합니다.

1. 웹 브라우저에서 카메라에 할당한 IP 주소(`https://{IP_address}`)로 접속하세요.

   <Image src='/img/cam-server/password-setting1-vionyx.png' />

1. 비밀번호 설정 페이지에 <Cmd sid='password' product='cam' /> 입력란에 새로운 비밀번호를 입력하세요.

1. <Cmd sid='confirm_password' product='cam' /> 필드에 동일한 비밀번호를 다시 입력하세요.

1. 입력한 비밀번호들이 아래의 모든 요구 사항을 만족하는지 확인하세요.

1. 모든 조건이 충족되면 <Cmd sid='submit' product='cam' /> 버튼이 활성화됩니다. 버튼을 클릭해 비밀번호 설정을 완료하세요.

:::note

**비밀번호 요구 사항**

비밀번호는 다음의 모든 조건을 만족해야 합니다:

* <Cmd sid='password' product='cam' /> 필드와 <Cmd sid='confirm_password' product='cam' /> 필드의 값이 정확히 일치해야 합니다.

* 비밀번호는 최소 12자 이상이어야 합니다. 최대 64자 이하로 제한합니다.

* 최소 1개의 영어 소문자(`a-z`)와 대문자(`A-Z`)를 포함해야 합니다.

* 최소 1개의 숫자(`0-9`)와 특수문자를 포함해야 합니다.

  > 특수문자 예시: `"` `!` `@` `#` `$` `%` `^` `&` `*` `(` `)` `-` `_` `=` `+` `[` `]` `{` `}`

* `1234`, `abcd` 같이 연속 문자 4개 이상, 동일 문자 4개 이상을 포함할 수 없습니다.

* 대소문자와 무관하게 아이디는 비밀번호에 포함할 수 없습니다.

  > admin, Admin, ADMIN, Admin123! 등

* `password`, `qwerty`, `123456` 등 흔히 사용되는 비밀번호 패턴은 사용할 수 없습니다.

:::

:::caution

설정한 비밀번호는 안전한 장소에 보관하세요. 비밀번호를 분실한 경우 카메라를 초기화해야 합니다.

:::


## 로그인

비밀번호 설정이 완료되면 로그인 페이지로 자동 이동합니다. 관리자 아이디와 설정한 비밀번호를 입력하여 ViOnyx Web Server에 접속하세요.

<Image src='/img/cam-server/password-setting2-vionyx.png' />

1. <Cmd sid='id' product='cam' /> 입력란에 **admin**을 입력하세요.

1. <Cmd sid='password' product='cam' /> 입력란에 설정한 비밀번호를 입력하세요.

1. <Cmd>로그인</Cmd> 버튼을 클릭하세요

로그인에 성공하면 ViOnyx Web Server의 <Cmd sid='live' product='cam' /> 페이지로 이동합니다.

:::info

관리자 아이디(admin)는 <Cmd sid='system' product='cam' /> → <Cmd sid='account_management' product='cam' />에서 변경할 수 있습니다.

:::
