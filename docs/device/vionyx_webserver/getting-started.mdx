ViOnyx 카메라를 처음 설치한 후, 웹 브라우저에서 카메라의 IP 주소로 접속할 수 있습니다. 
초기 로그인 시 보안을 위해 비밀번호를 설정해야 하며, 설정 완료 후 로그인할 수 있습니다.


## 비밀번호 설정

처음 로그인할 때는 보안을 위해 강력한 비밀번호를 설정해야 합니다. ViOnyx는 강력한 비밀번호 정책을 적용하여 시스템 보안을 보장합니다.

1. 웹 브라우저에서 ViOnyx Web Server의 IP 주소로 접속하세요.

   <Image src='/img/cam-server/Vionyx-set-password.png' alone />

1. 비밀번호 설정 페이지에 <Cmd>Password</Cmd> 필드에 새로운 비밀번호를 입력하세요.

1. <Cmd>Confirm Password</Cmd> 필드에 동일한 비밀번호를 다시 입력하세요.

1. 입력한 비밀번호들이 아래의 모든 요구 사항을 만족하는지 확인하세요.

1. 모든 조건이 충족되면 <Cmd>Submit</Cmd> 버튼이 활성화됩니다. 버튼을 클릭해 비밀번호 설정을 완료하세요.

:::note

**비밀번호 요구 사항**

비밀번호는 다음의 모든 조건을 만족해야 합니다:

* <Cmd>Password</Cmd> 필드와 <Cmd>Confirm Password</Cmd> 필드의 값이 정확히 일치해야 합니다.

* 비밀번호는 최소 12자 이상이어야 합니다. 최대 64자 이하로 제한합니다.

* 최소 1개의 소문자(`a-z`)와 대문자(`A-Z`)를 포함해야 합니다.

* 최소 1개의 숫자(`0-9`)와 특수문자(`!@#$%^&*` 등)를 포함해야 합니다.

* `1234`, `abcd` 같이 연속 문자 4개 이상, 동일 문자 4개 이상을 포함할 수 없습니다.

* 대소문자와 무관하게 아이디는 비밀번호에 포함할 수 없습니다.

  > admin, Admin, ADMIN, Admin123! 등

* `password`, `qwerty`, `123456` 등 흔히 사용되는 비밀번호 패턴은 사용할 수 없습니다.

:::

:::info

설정한 비밀번호는 안전한 장소에 보관하세요. 비밀번호를 분실한 경우 관리자에게 문의하세요.

:::

## 로그인

비밀번호 설정이 완료되면 로그인 페이지로 자동 이동합니다. 아이디와 비밀번호를 입력하고 <Cmd>Sign-In</Cmd> 버튼을 클릭하세요. 로그인에 성공하면 ViOnyx Web Server의 <Cmd sid='live' product='cam' /> 페이지로 이동합니다.

<Image src='/img/cam-server/Vionyx-login.png' />

:::info

최초 로그인하는 관리자 아이디는 `admin`입니다.

:::
