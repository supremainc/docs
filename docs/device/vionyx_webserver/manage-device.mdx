카메라를 재부팅하거나 설정을 초기화하여 카메라를 효과적으로 관리할 수 있습니다. 재부팅, 설정 초기화, 공장 초기화 세 가지 방법을 안내합니다.

다음과 같은 상황에서 카메라 유지 관리 기능을 사용하세요.

* 네트워크 설정 변경 후 새로운 설정을 적용하고 싶을 때

* 카메라의 응답이 느려지거나 성능 저하가 발생했을 때

* 시스템이 정상 작동하지 않을 때

* 카메라의 설정을 모두 기본값으로 돌리고 싶을 때

* 카메라를 다른 사용자에게 인수인계하기 전에 모든 정보를 삭제하고 싶을 때

* 시스템에 문제가 발생하여 공장 기본값으로 복원해야 할 때

:::warning

* 재부팅 또는 초기화 중에 카메라의 전원을 끄지 마세요.

* **공장 초기화**는 모든 사용자 데이터, 이벤트 기록, 로그를 영구 삭제합니다. 복구할 수 없으므로 매우 신중하게 진행하세요.

* 초기화하기 전에 필요한 데이터를 백업하세요.

:::


## 카메라 재부팅

카메라를 재시작하여 설정 변경사항을 적용하거나 시스템 성능을 회복시킵니다. 재부팅은 현재 설정을 유지하면서 시스템을 재초기화하는 방법입니다.

1. 화면 왼쪽 상단의 <Cmd sid='system' product='cam' /> 메뉴를 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid='device_info' product='cam' />를 클릭하세요.

1. <Cmd sid='firmware_reset' product='cam' /> 섹션의 <Cmd sid='reboot' product='cam' />에서 <Cmd sid='apply' product='cam' /> 버튼을 클릭하세요.

1. 재부팅 확인 팝업이 나타나면 <Cmd sid='ok' product='cam' /> 버튼을 클릭하세요.

:::info

* 재부팅 중에는 카메라가 일시적으로 사용할 수 없습니다. 재부팅은 최소 1-2분 소요됩니다.

* 재부팅 중에 카메라의 전원을 끄지 마세요. 시스템 손상이 발생할 수 있습니다.

* 재부팅 후 카메라의 IP 주소가 변경되었다면 새로운 IP 주소로 접속해야 합니다.

* 재부팅이 시작되면 웹 UI 연결이 자동으로 끊깁니다. 이는 정상 동작입니다.

:::


## 초기화

카메라의 설정을 초기화하거나 공장 기본값으로 완전히 복원할 수 있습니다. 두 가지 초기화 방식이 제공되며, 삭제되는 데이터가 다르므로 신중하게 선택해야 합니다.

### 초기화 방식 비교

| 방식 | 설명 | 삭제 대상 | 유지 대상 |
|---|---|---|---|
| <Cmd sid='config_reset' product='cam' /> | 설정값만 초기화 | 카메라 설정, 네트워크 설정(선택 사항), 비디오 프로필, 규칙, 계정 | 사용자 정보, 얼굴 데이터, 이벤트/로그 |
| <Cmd sid='factory_reset' product='cam' /> | 모든 데이터 삭제 후 공장 기본값 복원 | 모든 설정, 사용자 정보, 이벤트, 로그, 계정 | - |

### <Cmd sid='config_reset' product='cam' className='normal' />

설정값만 초기화하고 사용자 정보 및 로그 데이터는 유지합니다.

1. 화면 왼쪽 상단의 <Cmd sid='system' product='cam' /> 메뉴를 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid='device_info' product='cam' />를 클릭하세요.

1. <Cmd sid='firmware_reset' product='cam' /> 섹션의 <Cmd sid='config_reset' product='cam' />에서 <Cmd sid='reset' product='cam' /> 버튼을 클릭하세요.

   네트워크 설정 초기화를 제외하려면 <Cmd sid='config_reset_excluding_network' product='cam' /> 옵션의 체크박스를 클릭하세요.

1. 설정 초기화 확인 팝업이 나타나면 <Cmd sid='ok' product='cam' /> 버튼을 클릭하세요.

설정 초기화가 진행되고 카메라가 자동으로 재부팅됩니다.


### <Cmd sid='factory_reset' product='cam' className='normal' />

카메라를 완전히 초기화하여 공장 초기상태로 복원합니다. 이 작업은 되돌릴 수 없습니다.

:::warning

공장 초기화는 모든 데이터를 완전히 삭제합니다. 이 작업은 복구할 수 없으므로 매우 신중하게 진행하세요.

:::

1. 화면 왼쪽 상단의 <Cmd sid='system' product='cam' /> 메뉴를 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid='device_info' product='cam' />를 클릭하세요.

1. <Cmd sid='firmware_reset' product='cam' /> 섹션의 <Cmd sid='factory_reset' product='cam' />에서 <Cmd sid='reset' product='cam' /> 버튼을 클릭하세요.

   네트워크 설정 초기화를 제외하려면 <Cmd sid='config_reset_excluding_network' product='cam' /> 옵션의 체크박스를 클릭하세요.

1. 공장 초기화 확인 팝업이 나타나면 <Cmd sid='ok' product='cam' /> 버튼을 클릭하세요.

1. 최종 확인 팝업이 나타나면 관리자 비밀번호를 입력하세요.

1. <Cmd sid='ok' product='cam' /> 버튼을 클릭하세요.

:::info

* 공장 초기화 완료 후 아래 항목이 기본값으로 복원됩니다.

  * 관리자 ID: **admin**

  * 네트워크 설정: DHCP

  * 모든 사용자 데이터, 이벤트, 로그: 삭제됨

* 공장 초기화 후 웹 UI에 접속할 때 새로운 기본 비밀번호를 설정해야 합니다.

:::
