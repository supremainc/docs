ViOnyx Web Server의 <Cmd sid='live' product='cam' /> 메뉴에서는 카메라가 촬영 중인 영상을 실시간으로 확인할 수 있습니다. 원하는 채널과 뷰 모드를 선택하고, 화면을 확대하거나 전체 화면으로 볼 수 있습니다.

## Live 뷰 기본 구성

<Cmd sid='live' product='cam' /> 메뉴에 처음 진입하면 다음과 같은 기본 상태로 영상이 표시됩니다.

<Image src='/img/cam-server/vionyx-live-basic.png' />

<div className='menu-icon-table'>

| 항목 | 구성 요소 | 설명 |
| :-: | :-: | --- |
| <Num num='1' /> | <Image src='/img/cam-server/ico-content-ai-hide.svg' width='100%' alone className='none' /> | AI 기능이 비활성화된 상태입니다. 아이콘을 클릭해 AI 기능을 활성화할 수 있습니다. |
| <Num num='2' /> | <IcPanelFace width='26px' height='26px' /> | 페이스 샷(Face Shot) 패널을 열 수 있습니다. |

| <Num num='4' /> | <IcKnob width='28px' height='28px' /> | 라이브로 보고 있는 영상을 회전할 수 있습니다. |

## 채널과 뷰 모드 선택하기

ViOnyx Web Server는 여러 채널과 다양한 뷰 모드를 제공합니다. 화면 아래의 <Cmd>CH</Cmd> 드롭다운 메뉴에서 원하는 채널과 뷰 모드를 선택하세요.

<Image src='/img/cam-server/vionyx-live-basic-select-ch.png' alone />

**지원하는 채널과 뷰 모드**

<div className='table--viewmode'>

| 채널 | 뷰 모드 | 설명 | 예시 |
| :--: | -- | -- | :--: |
| **1ch** | Fisheye View (Main/Sub) | 기본 채널의 원본 피시아이 렌즈 영상 | <Image src='/img/cam-server/vionyx-live-basic-fisheye-view-mode.png' width='50%' alone /> |
| **2ch** | Quad View (Main) | 파노라마 뷰 채널의 4분할 뷰 | <Image src='/img/cam-server/vionyx-live-basic-quad-view-mode.png' width='50%' alone /> |

</div>

:::info

* 각 채널의 Main과 Sub는 서로 다른 해상도의 스트림을 제공합니다. Main은 고화질, Sub는 저화질 스트림입니다.

* 위 이미지는 예시 화면입니다. 실제 화면과 다를 수 있습니다.

:::


## 화면 확대 및 축소

마우스를 사용해 화면을 확대하고 축소할 수 있습니다.

* 스크롤 업(위로): 화면 확대(줌 인)

* 스크롤 다운(아래로): 화면 축소(줌 아웃)

* 더블클릭: 원래 크기로 복귀(1배)

:::info

* 1배 ~ 8배까지 화면을 확대할 수 있습니다.

* 화면 확대는 디지털 줌이므로, 높은 배율로 확대하면 영상이 흐릿해질 수 있습니다.

:::


## 전체 화면 모드

더 큰 영상을 확인하려면 전체 화면 모드를 사용하세요. 화면 오른쪽 하단의 <IcFull /> 버튼을 클릭하세요.

<Image src='/img/cam-server/vionyx-live-basic-full-screen.png' />

* 전체 화면 모드에서도 뷰 모드 변경, 줌 기능을 계속 사용할 수 있습니다.

* 전체 화면 모드를 종료하려면 <Kbd>ESC</Kbd> 키를 누르거나 <IcMin /> 버튼을 다시 클릭하세요.


## 화면 회전하기

촬영 중인 영상을 회전하려면 <IcKnob /> 버튼을 클릭한 상태에서 드래그하세요. 또는 <IcKnob /> 버튼 오른쪽에 입력 필드에 회전 각도를 직접 입력할 수도 있습니다.

<Image src='/img/cam-server/vionyx-live-basic-rotate-screen.png' />


## 얼굴 촬영 패널 열기

AI 이벤트 기능을 활성화했다면 카메라를 통해서 감지된 얼굴을 확인할 수 있습니다. <Cmd sid='face_shot' product='cam' /> 패널을 열려면 화면 오른쪽 상단의 <IcPanelFace /> 버튼을 클릭하세요.

<Image src='/img/cam-server/vionyx-live-basic-ai-face-shot.png' />

<Cmd sid='face_shot' product='cam' /> 패널 상단에서 제공하는 도구 아이콘을 클릭해 다음과 같은 기능을 사용할 수 있습니다.

* <Image src='/img/menus/ico-faceshot-pause.svg' ico alone className='none' /> / <Image src='/img/menus/ico-faceshot-play.svg' ico alone className='none' />: 얼굴 촬영을 일시 정지하거나 재게할 수 있습니다.

* <Image src='/img/menus/ico-faceshot-filter.svg' ico alone className='none' />: 얼굴 촬영 결과를 필터링할 수 있습니다.

* <Image src='/img/menus/ico-faceshot-max.svg' ico alone className='none' />: <Cmd sid='face_shot' product='cam' /> 패널을 전체 화면으로 확대할 수 있습니다.

:::info

AI 이벤트 및 규칙 설정에 대한 자세한 내용은 <DocLink docId='device/vionyx_webserver/configure-ai-events-and-rules' />를 참조하세요.

:::


## 영상이 재생되지 않을 때

카메라의 영상이 로드되지 않는다면 다음 단계를 따라 네트워크 연결 상태를 확인하세요.

1. 카메라와 네트워크 연결 상태를 확인하세요.

1. 웹 브라우저에서 **새로고침(F5)** 버튼을 누르세요.

1. 시스템이 자동으로 재연결을 시도합니다.
