h1. ViOnyx Network 기능 명세서
----
h2. 1. 기능 개요
||항목||내용||
|기능명|Network Settings|
|위치|Settings > Network|
|역할|IPv4 네트워크 설정, 포트 설정, CLUe 클라우드 연결|
|1차 스코프|IPv4 / Port 설정 (IPv6는 추후 지원)|
----
h2. 2. 기능 구성
||기능||설명||
|IPv4|IP 주소, 서브넷, 게이트웨이, DNS 설정|
|Port|HTTP, HTTPS, RTSP 포트 설정|
|Cloud (CLUe)|Suprema CLUe 클라우드 연결 설정|
----
h2. 3. IPv4 설정
----
h3. 3.1 IP Type
||모드||설명||기본값||
|DHCP|자동 IP 할당|*기본값*|
|Static|수동 IP 설정|-|
----
h3. 3.2 IPv4 설정 항목
||항목||DHCP 모드||Static 모드||형식||
|MAC|읽기 전용|읽기 전용|XX:XX:XX:XX:XX:XX|
|IP Address|자동 (읽기 전용)|수동 입력|IPv4 형식|
|Subnet Mask|자동 (읽기 전용)|수동 입력|IPv4 형식|
|Gateway|자동 (읽기 전용)|수동 입력|IPv4 형식|
|DNS 1|수동 입력|수동 입력|IPv4 형식|
|DNS 2|수동 입력 |수동 입력|IPv4 형식|
|MTU|수동 입력|수동 입력|1280 ~ 1500|
----
h3. 3.3 MTU 설정
||항목||사양||
|범위|1280 ~ 1500|
|기본값|1500|
----
h3. 3.4 입력 유효성 검사
||항목||검사 내용||
|IP Address|IPv4 형식 (0-255.0-255.0-255.0-255)|
|Subnet Mask|유효한 서브넷 마스크 값|
|Gateway|IPv4 형식|
|DNS|IPv4 형식|
|MTU|1280 ~ 1500 범위|
||유효성 실패 시||동작||
|입력 필드|빨간색 테두리(Stroke) 표시|
|Apply 버튼|비활성화|
----
h2. 4. Port 설정
----
h3. 4.1 포트 항목

h3. 4.3 포트 충돌 검사

|동일 포트 중복 사용|에러 표시, Apply 비활성화|

h2. 6. CLUe 클라우드 연결
----
h3. 6.1 기능 설명
||항목||내용||
|목적|Suprema CLUe 클라우드 서비스 연결|
|기본값|*OFF (연결 안함)*|
----
h3. 6.2 CLUe 연결 상태
||상태||설명||
|OFF|CLUe 연결 비활성화|
|ON (연결 가능)|CLUe 연결 성공, 장치 등록/할당 가능|
|ON (인증서 없음)|인증서 미발급 상태|
|ON (연결 불가)|인터넷 연결 실패 또는 서버 연결 실패|
----
h3. 6.3 CLUe Enable 플로우

 {{[CLUe OFF → ON 전환 시도]}}
 \{{ │}}
 \{{ ▼}}
 {{[인터넷 연결 확인]}}
 \{{ │}}
 {{ ├── 실패 ──▶ [Enable 불가] ──▶ [에러 표시: 연결 불가]}}
 \{{ │}}
 {{ └── 성공 ──▶ [CLUe 서버 연결 확인]}}
 \{{ │}}
 {{ ├── 실패 ──▶ [Enable 불가] ──▶ [에러 표시]}}
 \{{ │}}
 {{ └── 성공 ──▶ [Enable 완료]}}
 \{{ │}}
 \{{ ▼}}
 {{ [인증서 확인]}}
 \{{ │}}
 \{{ ├── 있음 ──▶ "연결 가능" 표시}}
 \{{ │}}
 \{{ └── 없음 ──▶ "인증서 없음" 표시}}
 \{{ (OFF→ON 시 인증서 재시도)}}
----
h3. 6.4 CLUe Disable 동작
||동작||설명||
|Disable 시|CLUe에 등록되어 있더라도 장치 연결 불가|
|등록 상태|CLUe 서버 측 등록 정보는 유지|
----
h3. 6.5 CLUe 상태별 메시지
||상태||메시지 (영어)||
|연결 가능|"Can be connected"|
|인증서 없음|"No certification status"|
|연결 불가|"Unable to connect to the CLUe server. Please check your internet connection and try again."|
----
h3. 6.6 CLUe 로그 기록
||이벤트||로그 기록||
|Enable 성공|✓ 필수|
|Enable 실패|✓ 필수 (실패 사유 포함)|
|Disable|✓ 필수|
{quote}*⚠️ 개발 참고:* CLUe Enable/Disable 여부는 시스템 로그에 반드시 기록
{quote}
----
h2. 7. 설정 저장 및 반영
----
h3. 7.1 버튼 동작
||버튼||동작||
|Apply|변경 사항 저장 및 적용|
|Cancel|변경 입력 내용 버리기 (원래 내용으로 복구)|
----
h3. 7.2 Apply 버튼 상태
||조건||버튼 상태||
|유효성 검사 통과|활성화|
|유효성 검사 실패|비활성화|
|변경 사항 없음|비활성화|
----
h3. 7.3 네트워크 설정 변경 시 주의
||항목||동작||
|IP 변경 시|연결 끊김 경고 필요 여부|
|적용 후|새 IP로 재접속 필요 안내|
{quote}*� [개발팀 확인 필요]* IP 변경 시 사용자 경고 팝업 필요 여부
{quote}
----
