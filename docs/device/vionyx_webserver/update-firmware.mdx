펌웨어 업데이트 기능을 통해 카메라의 펌웨어를 최신 버전으로 업데이트할 수 있습니다. 펌웨어 업데이트로 새로운 기능, 성능 개선, 보안 패치를 적용할 수 있습니다.

아래와 같은 상황에서 펌웨어를 업데이트하세요.

* 새로운 기능을 사용하고 싶을 때

* 보안 패치를 적용하여 카메라를 보호하고 싶을 때

* 카메라의 성능을 개선하고 싶을 때

* 기술 지원팀에서 권장하는 펌웨어 버전이 있을 때

:::info

* 펌웨어 업데이트 중에는 카메라가 재시작되며, 이 기간 동안 카메라는 사용할 수 없습니다.

* 펌웨어 업데이트를 시작하기 전에 장치에 충분한 전원이 공급되고 있는지 확인하세요.

* 슈프리마 기술 지원팀에서 제공하는 공식 펌웨어 파일(`.bin`)만 사용하세요.

:::


## 현재 펌웨어 버전 확인

카메라에 설치된 현재 펌웨어 버전을 확인하세요.

1. 화면 왼쪽 상단의 <Cmd sid='system' product='cam' /> 메뉴를 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid='device_info' product='cam' />를 클릭하세요.

<Image src='/img/cam-server/cam-webserver-system-firmware.png' />

<Cmd sid='firmware_reset' product='cam' /> 섹션의 <Cmd sid='version' product='cam' />에서 현재 설치된 펌웨어 버전을 확인할 수 있습니다.


## 펌웨어 업데이트

새로운 펌웨어 파일을 선택하여 장치를 업데이트하세요.

1. 화면 왼쪽 상단의 <Cmd sid='system' product='cam' /> 메뉴를 클릭하세요.

1. 화면 왼쪽 사이드바에서 <Cmd sid='device_info' product='cam' />를 클릭하세요.

1. <Cmd sid='firmware_reset' product='cam' /> 섹션에서 <Cmd sid='update' product='cam' /> 버튼을 클릭하세요.

1. <Cmd sid='firmware_update' product='cam' /> 팝업이 나타나면 슈프리마에서 제공하는 펌웨어 파일(`.bin` 파일)을 선택하세요.

   <Image src='/img/cam-server/cam-webserver-system-firmware-update.png' />

   * 파일 확장자는 반드시 `.bin`이어야 합니다.

   * 업로드 영역에 파일을 드래그하거나 <Cmd sid='choose_file' product='cam' /> 버튼을 클릭해 파일을 선택할 수 있습니다.

1. 파일을 선택하면 <Cmd sid='firmware_upload' product='cam' /> 버튼이 활성화됩니다.

1. 펌웨어 업데이트를 진행하려면 <Cmd sid='firmware_upload' product='cam' /> 버튼을 클릭하세요.

1. 업로드가 완료되면 재부팅 확인 팝업이 나타납니다. <Cmd sid='ok' product='cam' /> 버튼을 클릭해 카메라를 재시작하고 펌웨어 업데이트를 완료하세요.

:::note

* 카메라가 재부팅한 후 웹 UI에 다시 접속하세요. 새로운 펌웨어 버전을 확인하려면 <Cmd sid='device_info' product='cam' />로 이동하세요.

* 펌웨어 업데이트가 시작된 후에는 취소할 수 없습니다. 업데이트가 완료될 때까지 기다려야 합니다.

:::
