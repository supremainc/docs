사용자 목록에서 이름이나 아이디를 입력하여 원하는 사용자를 신속하게 찾을 수 있습니다. 필요에 따라 테이블 또는 카드 형태로 사용자 정보를 표시할 수 있습니다.

아래와 같은 상황에서 사용자 검색 및 관리 기능을 사용하세요.

* 많은 사용자 중에서 특정 사용자를 빠르게 찾아야 할 때

* 카드 형태로 사용자 프로필을 한눈에 확인하고 싶을 때

:::info

검색과 필터를 함께 사용하면 대규모 사용자 목록을 더 효율적으로 관리할 수 있습니다.

:::


## 사용자 검색

1. 화면 왼쪽 상단의 <Cmd sid='people' product='cam' /> 메뉴를 클릭하세요.

1. 화면 오른쪽 상단의 검색 입력란에 사용자 이름 또는 사용자 ID의 일부를 입력하세요.

   <Image src='/img/cam-server/cam-webserver-people-list-search.png' />

1. 입력한 검색어와 일치하는 사용자가 목록에 표시됩니다.

:::info

* 부분 검색을 지원합니다. 예를 들어, 'Kim'이라고 입력하면 이름에 'Kim'을 포함한 모든 사용자가 표시됩니다.

* 검색 결과는 현재 선택된 필터 기준에 따라 표시됩니다. 필터를 변경하면 검색 결과도 함께 업데이트됩니다.

:::


## 뷰 모드 변경

사용자 목록을 테이블 형태 또는 카드 형태로 표시할 수 있습니다.

### 테이블 뷰

사용자 목록을 테이블 형태로 표시하려면 목록 오른쪽 상단의 <Image src='/img/cam-server/ico-people-list-mode.svg' alone ico /> 클릭하세요.

<Image src='/img/cam-server/cam-webserver-people-list-mode.png' />

### 카드 뷰

사용자 목록을 카드 형태로 표시하려면 목록 오른쪽 상단의 <Image src='/img/cam-server/ico-people-card-mode.svg' alone ico /> 클릭하세요.

<Image src='/img/cam-server/cam-webserver-people-card-mode.png' />

:::info

리스트 뷰와 카드 뷰는 모두 동일한 기능을 제공합니다. 표시 방식만 다르며, 검색 및 필터, 정렬 결과는 동일합니다.

:::
