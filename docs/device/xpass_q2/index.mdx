XPass Q2는 방문객과 티켓 출입이 많은 환경에서 빠른 출입을 위해 설계된 리더입니다. QR/바코드 중심 설계와 멀티 크리덴셜 지원으로 통행의 흐름을 끊김 없이 유지하며, 콤팩트하고 내구성이 뛰어난 폼팩터로 손쉽게 구축할 수 있습니다.
