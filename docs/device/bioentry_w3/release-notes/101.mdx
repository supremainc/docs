# 펌웨어 버전 1.0.1 (빌드 번호 1.0.1_241024)

릴리즈<Separator/>2024-11-07

## 신규 기능 및 개선

---

1. **RTSP** 사용 시 장치에서 송출되는 영상의 해상도를 선택할 수 있는 **RTSP 비디오 해상도** 기능 지원

1. **커스텀 스마트 카드 레이아웃**에 FeliCa 카드 추가 지원

1. 장치의 인증 결과 화면에서 사용자 ID, 이름이 표시되는 방법을 선택할 수 있는 **인증 결과 표시 옵션** 기능 지원

    - **사용자 ID 표시**

    - **사용자 이름 표시**

1. BLE 통신을 이용하여 장치를 설정할 수 있는 모바일 애플리케이션 Device Manager 지원

    - Device Manager 1.2.0 버전 이상

1. 위조 얼굴 검출 성능 개선

    :::info

    햇빛이 강한 환경에서는 **위조 얼굴 검출** 기능을 **보통**으로 사용하는 것을 권장합니다.

    :::

1. 얼굴 인증 시 마스크를 착용한 사용자는 인증을 거부하도록 설정할 수 있는 **마스크 미착용** 옵션 지원

## 버그 수정

---

<BugLists>

1. **시간 제한 안티패스백**의 **초기화 시간**을 0으로 설정하면 안티패스백이 동작하지 않는 문제 <Badge>1.0.0</Badge>

1. 일부 RF 카드 인식에 실패하는 문제 <Badge>1.0.0</Badge>

1. 장치가 BioStar 2의 **대기 장치** 목록에 있을 때 장치의 **장치** → **서버** 설정을 꺼도 BioStar 2의 **대기 장치** 목록에 장치가 표시되고, 장치 상단에 BioStar 2 아이콘이 표시되는 문제 <Badge>1.0.0</Badge>

1. SDK를 사용하여 장치의 RS-485 모드를 **사용안함**으로 설정했을 때 장치가 멈추는 문제 <Badge>1.0.0</Badge>

1. 장치가 슬레이브로 연결되어있고 **인증 대기 시간**이 20초로 설정되어있는 상태에서 카드 인증 후 얼굴 인증 대기 상태에서 카드를 반복적으로 인증했을 경우 인증 오류가 발생하는 문제 <Badge>1.0.0</Badge>

1. 신규 SE 칩 펌웨어를 사용하는 환경에서 일부 스마트 카드 인식에 실패하는 문제 <Badge>1.0.0</Badge>

</BugLists>
