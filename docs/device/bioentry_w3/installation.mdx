## 브래킷 및 제품 고정

1. 제품을 장착할 위치를 정하세요.

    <InstallationBracketNotes height={137} />

    <Image src='/img/device/bew3-bracket-fix.png' className='none' alone />

    * RF 신호 간섭을 피하기 위해 최소 이격 거리를 확인하여 설치하세요.

## 릴레이 연결

### Fail Safe Lock

Fail Safe Lock을 사용하려면 아래 그림과 같이 N/C 릴레이를 연결하세요. Fail Safe Lock은 평상 시 릴레이 회로에 전류가 흐르며, 릴레이가 동작하여 전류를 일시적으로 차단하면 출입문이 열립니다. 정전이 되거나 외부 요인으로 장치의 전원이 끊길 경우 출입문이 열립니다.

<Image src='/img/device/installation-relay-fail-safe-lock-bew3.png' className='none' />

### Fail Secure Lock

Fail Secure Lock을 사용하려면 아래 그림과 같이 N/O 릴레이를 연결하세요. Fail Secure Lock은 평상 시 릴레이 회로에 전류가 흐르지 않으며, 릴레이가 동작하여 전류가 흐르면 출입문이 열립니다. 정전이 되거나 외부 요인으로 장치의 전원이 끊길 경우 출입문이 잠깁니다.

## Secure I/O 2 연결

제품에 I/O 장치인 Secure I/O 2를 RS-485로 연결할 수 있습니다. 제품과 Secure I/O 2의 연결이 끊기거나 외부 요인에 의해 제품의 전원이 차단되더라도 보안을 유지할 수 있습니다.

* RS-485 케이블은 AWG24, 꼬임선(Twisted Pair), 최대 길이 1.2 km를 사용하세요.

* RS-485 데이지 체인으로 연결할 경우 종단 저항(120Ω)을 데이지 체인 연결의 양 끝에 연결하세요. 중간 선로에 연결할 경우 신호 레벨이 작아져 통신 성능이 떨어지므로 반드시 데이지 체인 연결의 양 끝에 연결하세요.

## 네트워크 설정 초기화 {#reset-network-settings}

1. 전원을 켜세요.

1. 장치가 자동으로 재부팅될 때까지 초기화 버튼을 누르세요.

1. 초기화된 네트워크 정보를 이용해 장치를 연결하세요.

    * **TCP/IP 주소**: DHCP 주소 할당 (DHCP 주소 할당을 실패하면 169.254.x.x로 설정됩니다.)

    * **서버 모드**: 사용 해제

    * **RS-485**: 기본값, 115200 bps

1. TCP/IP나 RS-485 정보를 변경하세요.

1. 전원을 끄고 켠 뒤 네트워크 정보가 올바르게 설정되었는지 확인하세요.


## 공장 초기화 {#factory-reset}

장치에 저장된 모든 정보와 루트 인증서를 삭제하고 설정을 초기화합니다.

1. 전원을 켜세요.

1. 초기화 버튼을 빠르게 세 번 누르세요.

1. 장치의 LED가 초록색으로 깜박거릴 때 초기화 버튼을 한 번 더 누르세요.

:::info

장치에 저장된 루트 인증서가 존재하지 않을 경우 공장 초기화를 진행할 수 없습니다.

:::


## 펌웨어 업그레이드

장치에 USB 메모리를 연결하여 펌웨어를 업그레이드할 수 있습니다.

1. USB 메모리에 업그레이드할 펌웨어 파일을 저장하세요.

    :::info

    USB 메모리에는 펌웨어 파일을 하나만 저장하여 업그레이드하는 것을 권장합니다.

    :::

1. 장치에 USB 메모리를 연결하세요.

1. '관리자 인증이 필요합니다' 메시지가 나타나면 관리자 권한 인증 수단으로 인증하세요.

    :::info

    BioStar 2에서 장치에 관리자를 등록하세요. 장치에 관리자가 등록되어 있지 않을 경우 펌웨어 업그레이드를 진행할 수 없습니다. 관리자 등록 방법에 대한 자세한 내용은 BioStar 2 관리자 설명서를 참고하세요.

    :::

1. 관리자 인증에 성공하면 펌웨어 파일이 장치로 전송됩니다.

1. 펌웨어 파일 전송이 완료되면 USB 메모리 제거 메시지가 화면에 나타납니다. USB 메모리를 제거하세요.

    :::info

    USB 메모리를 제거하지 않으면 장치 재시작 후 펌웨어 업그레이드가 다시 실행됩니다.

    :::

1. 펌웨어 업그레이드 진행 후 장치가 자동으로 재시작됩니다.

:::caution

펌웨어 업그레이드 진행 중 전원 공급이 중단되지 않도록 주의하세요. 고장의 원인이 됩니다.

:::
