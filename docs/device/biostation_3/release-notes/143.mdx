# 펌웨어 버전 1.4.3 (빌드 번호 1.4.3_20260406)

릴리즈<Separator/>2026-04-10

:::caution

- 펌웨어가 v1.4.0인 제품을 사용하는 경우, 반드시 v1.4.1로 업그레이드한 후 사용하세요.<br/>중요한 보안 및 안정성 개선이 포함되어 있으므로, 반드시 최신 버전으로 업그레이드한 후 사용해야 합니다.

- 펌웨어를 v1.4.0 이상으로 업그레이드한 뒤 하위 버전으로 다운그레이드할 수 없습니다.

- 장치의 펌웨어 다운그레이드 가능 여부 및 신규 BLE 칩 적용 여부는 장치의 시리얼 번호를 확인한 뒤 [슈프리마 홈페이지](https://supremainc.com)에 문의하세요.

:::
