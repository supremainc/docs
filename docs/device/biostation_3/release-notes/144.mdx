릴리즈<Separator/>2026-04-22

:::caution

- 펌웨어가 v1.4.0인 제품을 사용하는 경우, 반드시 v1.4.1로 업그레이드한 후 사용하세요.<br/>중요한 보안 및 안정성 개선이 포함되어 있으므로, 반드시 최신 버전으로 업그레이드한 후 사용해야 합니다.
