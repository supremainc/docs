## 각 부분의 명칭 및 기능

<Image src='/img/device/name-function-each-part-xpd2.png' width='80%' className='none' />

* **LED 상태 표시등**: 제품의 작동 상태가 표시됩니다.

  * 초록색: 인증 성공

  * 빨간색: 인증 실패

  * 파란색/하늘색 점멸: 정상 동작 상태

  * 파란색/초록색 점멸: 네트워크 설정 초기화 시

  * 초록색 점멸: 입력 대기 상태

  * 초록색/흰색 점멸: 슈프리마 AirFob Pass (BLE) 애플리케이션과 연결 중인 상태

  * 흰색 점멸: Device Manager 애플리케이션과 연결 중인 상태

* **RF 카드 및 모바일 출입카드 인식부**: 출입을 위해 RF 카드 또는 모바일 출입카드를 인식시키는 부분입니다.

* **모바일 출입카드 인식부**: 출입을 위해 모바일 출입카드를 인식시키는 부분입니다.

* **초기화 버튼**: 장치에 저장된 모든 정보와 인증서를 삭제하고 설정을 초기화합니다. 자세한 내용은 [공장 초기화](installation#factory-reset)를 참고하세요.

* **케이블**

  전원 / RS-485 / Wiegand 출력 / LED 입력 / Beep 입력 / 탬퍼

* **키패드**: <Badge only>XPD2-GKDB</Badge> 카드 ID를 직접 입력하여 인증할 때 사용합니다. 직접 입력 방식으로 인증하려면 BioStar 2에서 인증 방법 설정을 변경하세요.

  * **0\~9**: 숫자를 입력할 때 사용합니다.

  * <Image src='/img/common/ico-star.svg' ico alone />: Wiegand 카드 ID를 입력할 때 FC 코드와 ID를 구분하기 위해 사용합니다.

  * <Image src='/img/common/ico-pound.svg' ico alone />: 입력을 종료할 때 사용합니다.
