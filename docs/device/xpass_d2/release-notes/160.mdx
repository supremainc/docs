# 펌웨어 버전 1.6.0 (빌드 번호 1.6.0_220223)

릴리즈<Separator/>2022-03-02

## 신규 기능 및 개선

---

1. 장치를 타사 컨트롤러와 연결해 Wiegand 리더 또는 OSDP 리더로 사용 시 CSN 카드의 출력되는 데이터 값에 대해 바이트 순서를 설정할 수 있도록 지원

1. 스마트 카드 바이트 순서 설정 지원

    - Wiegand 또는 OSDP로 출력되는 데이터 값에 대해 바이트 순서를 설정할 수 있도록 지원

1. Device Manager 애플리케이션(v1.1.4 이상)에 적용된 신규 기능 및 개선 사항 지원

    - CSN 카드, 스마트 카드에 바이트 순서 설정 지원

    - 인증 모드 동작에 따른 장치 LED 동작 추가 (XPD2-GKDB)

1. 신규 BLE (Bluetooth Low Energy) 칩 지원

    - 장치의 BLE 칩이 변경됨에 따라 펌웨어가 기존 장치 및 신규 장치의 BLE 칩과 모두 호환되도록 지원
