**마스터 관리자** 기능은 장치의 **전체 관리자 권한을 강화**하여, **장치의 보안을 향상**시키고 **무단 접근 및 설정 변경을 방지**하기 위한 기능입니다.<br/>이 기능을 통해, 장치에는 **반드시 마스터 관리자를 등록**해야 하며, 등록된 마스터 관리자만 관리자 메뉴에 진입하여 장치 설정을 변경할 수 있습니다.

**마스터 관리자** 기능은 **LCD 화면을 통한 사용자 인터페이스가 지원되는 슈프리마 제품**에서만 사용할 수 있습니다.

:::info

마스터 관리자는 여러 명 등록할 수 있으며, 공장 초기화 시에만 삭제됩니다.

:::
