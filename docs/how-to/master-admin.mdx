<Cmd sid="ui_128" product='dev' /> 기능은 **LCD 화면을 통한 사용자 인터페이스가 지원되는 슈프리마 제품** 중 **CE향 제품**에서만 사용할 수 있습니다.

:::

<Cmd sid="ui_128" product='dev' /> 기능은 장치의 **전체 관리자 권한을 강화**하여, **장치의 보안을 향상**시키고 **무단 접근 및 설정 변경을 방지**하기 위한 기능입니다.<br/>이 기능은 EU 지역용 **CE향 제품**에만 적용되며, **CE향 제품**에는 **반드시 마스터 관리자를 등록**해야 합니다. 등록된 마스터 관리자만 관리자 메뉴에 진입하여 장치 설정을 변경할 수 있습니다.

:::info

* 이 기능은 EU 지역용 **CE향 제품**의 보안성을 강화하기 위해 도입된 필수 정책이며, **글로벌향 (Non-CE)** 장치에서는 지원되지 않습니다.

* 마스터 관리자는 직접 삭제할 수 없으며 **공장 초기화 시에만 삭제됩니다**.

:::

## 펌웨어 업그레이드 시 동작 로직 {#firmware-upgrade-logic}

기존에 사용 중인 제품의 펌웨어를 업그레이드할 경우, **업그레이드 이전 버전**과 **업그레이드 대상 버전**의 조합에 따라 <Cmd sid="ui_128" product='dev' /> 및 <Cmd sid="ui_073" product='dev' /> 지원 여부가 달라질 수 있습니다.

:::note

자세한 버전별 지원 여부는 슈프리마 기술 지원팀(CS@suprema.co.kr)에 문의하세요.

:::

## 새 장치 <Badge only>CE향 제품</Badge>

CE향 새 장치에서는 처음부터 <Cmd sid="ui_128" product='dev' />를 필수적으로 등록해야 하며, 등록하지 않을 경우 장치 사용이 제한됩니다.

:::info

* 최초 부팅 시 <Cmd sid="ui_128" product='dev' /> 등록 화면이 나타나며, 등록을 완료하지 않으면 다음 화면으로 이동할 수 없습니다.

* BioStar X와 연결하여 장치 전체 관리자를 할당하더라도 <Cmd sid="ui_128" product='dev' />를 장치에서 직접 등록해야만 모든 기능을 사용할 수 있습니다.

* 새 장치는 펌웨어 다운그레이드를 할 수 없습니다.

:::

## 펌웨어 업그레이드 장치 <Badge only>CE향 제품</Badge>

펌웨어를 업그레이드한 CE향 제품은 <Cmd sid="ui_128" product='dev' /> 설정을 제공하지 않지만, <Cmd sid="ui_073" product='dev' /> 옵션을 통해 전체 관리자의 권한을 강화하여 장치의 보안을 향상시킬 수 있습니다.

### 관리자 2단계 인증 설정하기 {#admin-two-step-authentication}

:::info

* 장치에 등록된 관리자의 크리덴셜이 두 종류 이상인 경우에만 <Cmd sid="ui_073" product='dev' />을 활성화할 수 있습니다.

  * PIN은 단독 크리덴셜로 사용할 수 없으므로 ID + PIN 조합은 지원하지 않습니다.

* 업그레이드 장치는 펌웨어 다운그레이드를 할 수 없습니다.

:::

* **설정** → **장치** → <Cmd sid="ui_073" product='dev' /> 메뉴에서 관리자 2단계 인증을 활성화할 수 있습니다.

* <Cmd sid="ui_073" product='dev' /> 옵션은 기존에 사용하던 장치에서 펌웨어를 최신 버전으로 업그레이드 한 경우에만 표시됩니다.

<Image src='/img/how-to/setting-two-step-authentication.png' caption width='300px' />

:::info

* 모든 관리자에게 두 종류 이상의 크리덴셜이 등록되어 있지 않을 때 활성화 시 설정이 실패하며 에러 메시지가 표시됩니다.<br/>모든 관리자에게 두 종류 이상의 크리덴셜을 등록한 다음 다시 시도하세요.

  * PIN은 단독 크리덴셜로 사용할 수 없으므로 ID + PIN 조합은 지원하지 않습니다.

<Image src='/img/how-to/error-two-step-authentication.png' caption width='300px' />

:::

:::warning

* <Cmd sid="ui_073" product='dev' />을 활성화한 후 모든 관리자의 크리덴셜을 2종 미만으로 삭제하면, BioStar X(또는 BioStar 2)에 연결할 수 없는 상황에서 관리자 메뉴에 접근할 수 없게 됩니다. 따라서 관리자 크리덴셜을 삭제할 때는 각별한 주의가 필요합니다.

:::
