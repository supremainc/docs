#### 3.2.1. 최소 권한 시행

- Active Directory 또는 로컬 사용자 및 그룹에서 별도의 사용자 그룹 생성
- 그룹 기반으로 파일 시스템 및 애플리케이션 권한 할당
- OS 수준 최소 권한 모델을 반영하도록 BioStar X 내 역할 구성

#### 3.2.2. 로컬 관리 권한 제한

- `lusrmgr.msc` 사용하여 Administrators 그룹에서 불필요한 계정 제거
- 일반 작업에는 표준 사용자 계정 사용, 필요할 때만 관리자 계정 사용
- 도메인에 가입된 경우 그룹 정책 제한 그룹을 적용하여 일관된 권한 시행

#### 3.2.3. 강력한 인증 시행

- 그룹 정책에서 복잡한 암호 요구(최소 12–15자, 대/소문자, 숫자, 기호 혼합)
- 암호 만료(예: 90일마다) 설정 및 재사용 제한
- 여러 번 실패한 시도 후 계정을 잠그는 계정 잠금 정책 구성
- 원격 접근 또는 특권 계정에 MFA 시행

#### 3.2.4. 기본 제공 계정 관리

- `lusrmgr.msc`에서 Administrator 계정을 마우스 오른쪽 단추로 클릭하여 이름 변경
- 명시적으로 필요하지 않은 경우 Guest 계정 비활성화
- PowerShell에서 `net user`로 정기적으로 계정 감사

#### 3.2.5. 세션 관리

- 15분 비활동 후 자동 잠금 구성
- 가능한 경우 동일 계정으로 동시 로그인 방지
- 유휴 원격 세션 제한

그룹 정책 설정 경로: `컴퓨터 구성 → Windows 설정 → 보안 설정 → 로컬 정책 → 보안 옵션 → 대화형 로그온: 컴퓨터 비활동 제한`(예: 900초 = 15분)

#### 3.2.6. 감사 및 모니터링

- 로그인 시도, 권한 사용, 계정 관리에 대한 감사 활성화
- 로그를 **SIEM** 또는 중앙 로그 수집기로 전달
- 이상 여부를 위해 보안 로그 정기적으로 검토

그룹 정책 경로: `컴퓨터 구성 → Windows 설정 → 고급 감사 정책 구성`
활성화 범주: 로그온/로그오프, 계정 관리, 권한 사용
