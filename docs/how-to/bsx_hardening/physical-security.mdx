#### 3.8.1. 서버 위치 보안

- 물리적으로 보안된 데이터 센터 또는 잠긴 서버실에 BioStar X 서버 호스팅
- 승인된 IT 직원만 접근 가능
- 서버실에 CCTV 감시 및 접근 로그 사용
- 방문자 로그 유지

---

#### 3.8.2. 이동식 및 백업 미디어 보호

- 접근이 통제된 안전한 공간에 백업 드라이브 또는 테이프 보관
- BioStar X 데이터가 포함된 모든 이동식 저장소에 암호화 사용
- 클라우드 백업 사용 시 스토리지 공급자가 컴플라이언스 기준 충족 확인

💡 **모범 사례**: 물리적 미디어 의존도를 줄이기 위해 **클라우드 기반 암호화 백업** 사용.

---

#### 3.8.3. BIOS/UEFI 설정 보안

- BIOS/UEFI 암호 설정하여 무단 구성 변경 방지
- 외부 장치(USB, DVD)에서 부팅 비활성화
- 지원되는 경우 Secure Boot 활성화

> ⚠️ BIOS/UEFI 암호는 보안 볼트에 문서화하십시오. 분실하면 정당한 유지 관리를 막을 수 있습니다.

---

#### 3.8.4. 장치 도난 방지

서버나 저장 장치가 도난당하면 공격자가 다른 시스템에서 디스크를 꺼내 내용에 직접 접근할 수 있습니다.

- 모든 시스템 및 데이터 드라이브에 BitLocker를 사용한 전체 디스크 암호화 활성화
- 추가 물리적 제어: 비데이터센터 배포에는 케이블 잠금 또는 보안 랙 사용

---

#### 3.8.5. 환경 제어

- 서버실에 안정적인 온도, 습도, 전원 공급(UPS) 보장
- 화재 소화 시스템(예: FM200, 불활성 가스) 설치
- 온도/습도 센서로 모니터링
