title: BioStar X 강화
description: 강력한 인증, 역할 기반 접근 제어, 암호화, 정기 백업으로 BioStar X 시스템과 개인 데이터를 보호하는 방법을 안내합니다.
---

BioStar X를 안전하게 운영하려면 두 가지 보안 계층을 함께 강화해야 합니다.

* **시스템 보안**은 기본 관리자 계정 변경, 역할 기반 접근 제어, 강력한 인증, 장치 암호화 통신을 통해 애플리케이션 자체를 보호합니다.

* **개인 데이터 보안**은 사용자 정보와 바이오메트릭 데이터를 데이터베이스 암호화와 최소 저장 원칙으로 보호합니다.

이 두 영역을 모두 구성해야 규정 준수(GDPR, ISO 27001 등)를 충족하고, 데이터 침해 위험을 최소화할 수 있습니다.
