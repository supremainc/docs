## Features

Suprema integration with Gallagher communication between Gallagher Command Centre and Suprema BioStar 2. So, you can generate biometric templates for the suprema Devices from the Gallagher Command Centre, this then provides a futuristic solution for all end users.

**Following are the features provided by Suprema integration with Gallagher:**

* User Management from Gallagher Command Centre Will be reﬂected in Suprema BioStar 2.

* From the Suprema Integration, users can register Biometric Visual Face and Fingerprint data through scanning from various Suprema Biometric devices.

* CSN detection and integration with Biostar for automatic data enrichment.

* From the Suprema Integration, users can enroll card data through scanning from biometric devices like X-Station 2, BioLite N2, BioStation 3, and FaceStation F2 (These are tested devices).

* Allows up to 1,000 Biometric Devices to Manage

* Sip Server Support and BioStation 3 intercom feature is a perfect match for any corporation.

* All Suprema Device configurations can be done within the same window of the Command Centre.
