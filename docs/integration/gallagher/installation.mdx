## Install Prerequisite Software

### Installation of Gallagher Command Centre Setup

:::info

* Please don’t change any names and passwords while installing and setting up the configuration.

* To Upgrade the Suprema Integration with the Gallagher version Click here.

* Specifically, you can refer to either the **3C4643 Gallagher Command Centre vEL8.80 Release Note** for new installations.

* The **3E0068 Upgrade Procedure** for upgrading purposes for more information in the Release Notes Folder of your Command Centre ISO Directory.

* You must have to extract `Suprema integration With Gallagher.zip` to `C:\Program Files\`.

:::

1. Inside the Gallagher Command Centre folder vEL.8.80 (or Later) → Setup.exe, **tick the checkbox** and press the **Next** button.

   <Image src='/img/integration/gallagher/installation-gallagher1.png' alone />

1. **Click** the *…* button and locate the License file provided in the setup.

   <Image src='/img/integration/gallagher/installation-gallagher2.png' alone />

   <Image src='/img/integration/gallagher/installation-gallagher3.png' width={800} alone />

1. When you select the correct license file the icon will change to a tick mark then **press** the **Next** button.

   <Image src='/img/integration/gallagher/installation-gallagher4.png' alone />

1. **Uncheck** the checkbox of the subnet only and press the **Next** button.

   <Image src='/img/integration/gallagher/installation-gallagher5.png' alone />

1. **Select** the Install new SQL option and press the **Next** button.

   <Image src='/img/integration/gallagher/installation-gallagher6.png' alone />

1. **Select** the do not install a database template.

1. Without changing any settings press **Next** until you are recovering the code screen.

   :::info
   
   Click the export button and store the Recovery code, and tick the checkbox.

   :::

   <Image src='/img/integration/gallagher/installation-gallagher7.png' alone />

1. Select **Yes** and the installation will start.

   <Image src='/img/integration/gallagher/installation-gallagher8.png' alone />

1. After the Installation is complete, press the **Finish** button.

   <Image src='/img/integration/gallagher/installation-gallagher9.png' alone />

### Installation of BioStar 2

1. Open the **BioStar 2.exe** and choose the language & Accept the Agreement press **Next**.

   :::info
   
   Download the latest version of BioStar 2 - [Click Here](https://www.supremainc.com/download-center/pages/login.asp).

   :::

   <Image src='/img/integration/gallagher/installation-biostar1.png' alone />

1. Enter your BioStar 2 **Password** here and press the **Next** button.

   <Image src='/img/integration/gallagher/installation-biostar2.png' alone />

1. Select **Express** Installation and enter the BioStar 2 password again.

1. Select the location and proceed.

   <Image src='/img/integration/gallagher/installation-biostar3.png' alone />

1. Press the **Next** button until you reach the Select the Components screen. Check both options and press the **Next** button.

   <Image src='/img/integration/gallagher/installation-biostar4.png' alone />

1. Press the **Install** button to start the installation.

   <Image src='/img/integration/gallagher/installation-biostar5.png' alone />

1. **Uncheck** the Install USB Device Agent option and Press the **Finish** button and BioStar 2 is installed on your PC.

   :::info
   
   If you want to use the USB device, you must **check** the Install USB Device Agent.

   :::

   <Image src='/img/integration/gallagher/installation-biostar6.png' alone />

### Installation middleware prerequisites

:::info

* Before extracting/running `Suprema Integration with Gallagher2.0.zip` please turn off the antivirus.

* If you are upgrading from an older version, please find the solution in the troubleshooting guide.

:::

1. You need to **run** the Suprema Integration with Gallagher 1.8.6.1 file as administrator inside the Suprema Integration with Gallagher 1.8.6.1 folder. This will install OpenSSL, PostgreSQL, PgAdmin4, and Node.js. Also, this will set up the certifications required for our integration.

   <Image src='/img/integration/gallagher/installation-middleware-prerequisite1.png' alone />

1. Press **Yes** when if it asks for allow from an unknown publisher permissions.

1. Press **OK**.

   <Image src='/img/integration/gallagher/installation-middleware-prerequisite2.png' alone />

1. Now that you are ready to install the software, press **Install**.

   <Image src='/img/integration/gallagher/installation-middleware-prerequisite3.png' alone />

1. It will start the installation for prerequisites, and the software will get installed automatically.

   <Image src='/img/integration/gallagher/installation-middleware-prerequisite4.png' width={500} alone />

1. Press **OK**, for below pop-up.

   <Image src='/img/integration/gallagher/installation-middleware-prerequisite5.png' alone />

1. We are done with the prerequisite installation press **Finish**.

   :::info

   After installing the middleware, must restart the system once.

   :::

   <Image src='/img/integration/gallagher/installation-middleware-prerequisite6.png' alone />
