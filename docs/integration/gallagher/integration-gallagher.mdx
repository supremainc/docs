**Smart card(AOC/TOC):-**

:::info

You must have a 2.9.4 or above version of BioStar 2 to use the smartcard feature.

:::

1. If you want to use the Smart Card (AOC/TOC) option go to settings in the device, **press** the **settings** button, and inside Device Management **select** the device you want to configure and **select** the Smart Card option.

   <Image src='/img/integration/gallagher/smart-card1.png' alone />

1. Now **enable** the Smart Card toggle and do the following settings as below screen.

   <Image src='/img/integration/gallagher/smart-card2.png' alone />

1. After changing all the settings **press** the Apply button. After that **press** the Exit Settings button from the bottom left corner to exit from the setting tab.

   <Image src='/img/integration/gallagher/smart-card3.png' alone />

1. Now search for the user on the user search screen, you want to apply the smart card and **press** the edit button to view user details.

   <Image src='/img/integration/gallagher/smart-card4.png' alone />

1. Now to assign the smart card you need to assign a card from the encoder from the Gallagher inside cardholder cards tab.

   <Image src='/img/integration/gallagher/smart-card5.png' alone />

1. After pressing the button you can see the card number now you have to **press** save to inside Gallagher and you can see the newly assigned card.

   <Image src='/img/integration/gallagher/smart-card6.png' alone />

1. Now inside the user details table **go** inside the card section and **press** the Use AOC button to assign a smart card to the user.

   :::info

   To assign a Smart Card (AOC) for the user, the user must need the biometrics.

   :::

   <Image src='/img/integration/gallagher/smart-card7.png' alone />

1. Now **select** the device you want to use for AOC, and **use** the same number for the card ID we get while assigning from the encoder. **Select** the face template and **choose an** image of the visual face tab. **Press** the write smart card button.

   <Image src='/img/integration/gallagher/smart-card8.png' alone />

1. Now you have to **scan** the same card against the selected device to assign the card to the user. If you are not scanning the same card you will get an error message, else it will write the card for the user and clear all the biometric information for the user.

   :::info

   You can only assign one biometric to the Smart Card (AOC) for the user.

   :::

1. To assign the smart card with a fingerprint user must need a fingerprint, the user can assign up to 2 fingerprints per card. Inside the user card section **press** the Use AOC button and select the device, **use** the same number for the card ID we get, while assigning a card from the encoder. **Select** the fingerprints that you want to use for the smart card. **Press** the write smart card button.

   <Image src='/img/integration/gallagher/smart-card9.png' alone />

1. Now you have to **scan** the same card against the selected device to assign the card to the user. If you are not scanning the same card you will get an error message, else it will write the card for the user and clear all the biometric information for the user.

**HID ICLASS card:-**

1. If you want to use the HID ICLASS card option go to settings in the device, **press** the **settings** button, and inside Device Management **select** the device you want to configure and **select** the Wiegand Settings option then scroll down and find the Wiegand settings.

   <Image src='/img/integration/gallagher/hid-iclass-card1.png' alone />

1. Now enable the Wiegand Settings option, and **select** ICLASS AND HID PROX Options. The Wiegand format must be the same as the selected default in the dropdown. **Press** Apply the button to save the changes.

   :::info

   To use the Wiegand card you must need at least one biometric for users.

   :::

   <Image src='/img/integration/gallagher/hid-iclass-card2.png' alone />

1. To use the feature, you have to assign the card to the user. Go to the Gallagher command center, press the assign card button, **select** the HID Seos Corporate, **enter** the number, **Press the** OK button, and **Press** save.

   <Image src='/img/integration/gallagher/hid-iclass-card3.png' alone />

**QR Code Card:**

1. If you want to use the QR Code card option, go to settings by **pressing** the **settings** button, and inside Device Management, **select** the device you want to configure. Then, **select** the QR Code Settings option, and **enable** the use of QR/Barcode through the camera option settings. **Press** the Apply button to save the changes.

   <Image src='/img/integration/gallagher/qr-code-card.png' alone />
